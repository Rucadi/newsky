# Heels Sound Volume
- Author: Yewido6031
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/122633


![](https://i.postimg.cc/sD44KWfh/headercropped.jpg)﻿
  
  
This simple mod is add volume sliders to the Heels Sound mod. The sliders is available in the game Settings menu.
  
The mod is not affect other Footstep sounds only the Heels!
  
  
Two versions available.
  
  
**Default version:**
  
The Default version is create two sliders. Player and NPC slider. This two slider is affect all Heels Sound volume All In One.

![](https://i.postimg.cc/GtSSpsBt/01.jpg)﻿
  
  
**Extended version:**
  
The Extended version is create 8-9 sliders. The NPC is All In One but the Player Heels volume is seperated.
  
  
![](https://i.postimg.cc/2jHtGgHY/02.jpg)﻿
  
  
**Choose only one Main File or use the FOMOD version!
  
  
Heels Sound 2025 Edition now supported!**Use the new 2025.003 version!
  
  
**Installation:**
  
Extract the file to the Data folder and activate in Mod Manager.
  
Or use the FOMOD version with Mod Manager installation method.
  
  
**Uninstall:**
  
Delete the installed file from the Data folder.
  
Or delete with the Mod Manager.
  
  
**Informations:**
  
The mod is not editing the sound files or the Main mod so its working with all Heels Sound version and with all Heels Sound replacer!
  
I create this mod in CK so its an ESP file but its ESL flagged so not using slot.
