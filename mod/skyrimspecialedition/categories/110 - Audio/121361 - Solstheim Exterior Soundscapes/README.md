**[font=Georgia]THE ALL-MAKER STONES[/font]**
  
  

  
* [font=Georgia]This optional plugin adds sound loops to the All-Maker Stones by adding them directly to the vanilla activator. This a separate plugin to keep the main plugin free from any vanilla edits. [/font]
[font=Georgia]These sounds loops cease once you have completed the quest [/font][[font=Georgia]"At The Summit of Apocrypha."[/font]](https://en.uesp.net/wiki/Skyrim:At_the_Summit_of_Apocrypha)
  
* [font=Georgia][size=5] Compatible with everything that doesn't edit the All-Maker Stones activators[/font][/size]
  
  
* [font=Georgia]V1.1 Adds patches for [Serana Dialogue Add-On](https://www.nexusmods.com/skyrimspecialedition/mods/32161)﻿ and [Pilgrim](https://www.nexusmods.com/skyrimspecialedition/mods/54099)﻿. FOMOD will auto-select correct plugin.﻿[/font]
