# The Black Door Revoiced
- Author: LeoMaximus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/91124


![](https://i.imgur.com/mWpcN7n.png)
  
I hired two voice actors to revoice The Black Door for the Dark Brotherhood's Sanctuary entrances. The original was quite harsh on the ears and could be difficult to understand, but Andrew Helbig (for the male version) and Isabel Salazar (for the female version) came to the rescue!
  
Please consider hiring them if you have any voice acting needs for your projects.
  
  
  
![](https://i.imgur.com/Ty9hl9v.png)
  
No ESP, loose audio files.
  
Safe to add or remove at any time.
  
Will conflict with any other mods that change the same audio files.
  
Should be compatible with all versions of Skyrim.
