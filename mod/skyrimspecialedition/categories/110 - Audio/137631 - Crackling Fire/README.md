# Crackling Fire
- Author: Clofas
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/137631


![](https://i.imgur.com/Quz7rd7.png)
  
[font=Georgia]25 [/font][font=Georgia]New Sound Loops to revamp audio for fire hazards, campfires, fireplaces, torches, sconces, and embers. 
  
  
![](https://i.imgur.com/iQpyU47.gif)
  
﻿
  
**No Magic sounds included.** 
  
  
It will, however, replace some sounds found in **effect shaders** as multiple shaders use the fxfirebody loop. Highly recommended to use [/font][font=Georgia][Unique Shaders Sound FX](https://www.nexusmods.com/skyrimspecialedition/mods/128556) for some much-needed variety and to replace those effect shader sounds.
  
[/font]﻿
  
[font=Georgia]
  
  
Crackling Fire replaces vanilla audio files and thus is **not compatible** with anything that touches the same audio files. Campfire and Fireplace audio come from [Murmurs and Mead](https://www.nexusmods.com/skyrimspecialedition/mods/114716), so don't be alarmed if one is overwriting the other.
  
﻿
  
  
Should be 100% compatible with magic sound replacers. 
  
FOMOD included to allow players to pick-and-choose or to download all files.
