# Unused Sounds Restored
- Author: Superfuzz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/33187


This mod restores a number of unused vanilla sound effects.
  
  
ambientsoundsrestored.esp
  

  
* Restored various insect sounds in Skyrim's tundra region.
New wind loop for the frozen marsh region (was using Reach wind loop before).
  
  
* Restored an unused music track.
Restored Solstheim's unused cricket sounds.
  

  
betamapsound.esl
  

  
* Changes the sound that plays when you hover over a map marker in the world map to the sound that can be heard in the E3 demo.

  
newquestsound.esl
  

  
* Changes the new quest sound to an unused variant (new quest and quest update are using the same sound in vanilla).
