# Male Argonian Shouts - Revoiced
- Author: JakeTheSilent
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/44124


Ever been annoyed that Argonians use the default human voice for shouts, instead of having their more distinct, gruff-sounding voice?
  
Well, I decided to see if I could remedy that.
  
  
Every shout, including the ones from the DLCs, have been re-dubbed by me, doing my best to get close to the vanilla male Argonian voice.

A demonstration of every shout in V1 can be heard here:
  
  
  
  
To install, I recommend Mod Organizer, otherwise you can install manually by unzipping the "sound" folder and placing it in steam/steamapps/common/Skyrim Special Edition/Data
