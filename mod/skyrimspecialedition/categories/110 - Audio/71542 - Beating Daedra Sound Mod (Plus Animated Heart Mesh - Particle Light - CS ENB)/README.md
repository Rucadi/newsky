# Beating Daedra Sound Mod (Plus Animated Heart Mesh - Particle Light - CS ENB)
- Author: Ashen - Dylan James - powerofthree - wankingSkeever - Everglaid - KhrysINXS - OsmosisWrench - Fuzzles - Xanza
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/71542


**DESCRIPTION**
  
  
**Daedra Hearts** produce a very subtle beating heart sound as you approach.
  
Additionally, the heart will animate in sync with the new audio and emit a faint light.
  
[ENB Series](http://enbdev.com/) is required for screenspace lighting seen in pictures. **ESL-Flagged.**
  
New mesh provided by **[wankingSkeever](https://www.nexusmods.com/users/7064860)** from [Alchemy Ingredients ENB Light](https://www.nexusmods.com/skyrimspecialedition/mods/65509).
  
  
  

![](https://i.imgur.com/fCG5z0y.gif)

  
  
**INSTALLATION**
  
  
 ■  Download this mod, [Beating Daedra Sound Mod](https://www.nexusmods.com/skyrimspecialedition/mods/71542).
  
 ■  Ensure that no files are overwriting

```
meshes/clutter/ingredients/daedraheart.nif
```

.
  
  
Incompatible with any mod that edits

```
daedraheart.nif
```

. No patches are planned.
  
The provided plugin is required for the new Daedra Heart-beat sounds to play.
  
Please remember to **endorse** the original mods if you haven't already.
