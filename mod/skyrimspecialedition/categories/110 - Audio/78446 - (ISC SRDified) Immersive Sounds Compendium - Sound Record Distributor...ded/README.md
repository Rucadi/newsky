# (ISC SRDified) Immersive Sounds Compendium - Sound Record Distributor...ded
- Author: puddles54
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78446
This is just .yaml files for [Sound Record Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/77815?tab=description#) that will apply [Immersive Sound Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/523)'s sounds to various items at runtime, meaning patches won't be needed for any other mods that modify those things.
  
  
**BOTH OF THE ABOVE MODS ARE REQUIRED**
  
**It supports the following from the original ISC mod:**
  

  
* Armor (pick up & drop sounds)
Armor Addons (footstep/Foley sounds based on armor when moving)
  
* Effect Shaders
Explosions
  
* Magic Effects
Misc Items (pick up & drop sounds for lock picks and gems)
  
* Projectiles
Soul Gems
  
* Weapons

  
**It extends support for the following official content:**

  
* Dawnguard (some was missing in ISC)
Dragonborn (some was missing in ISC)
  
* Creation Club Creations
