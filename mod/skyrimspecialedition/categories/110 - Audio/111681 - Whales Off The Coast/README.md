# Whales Off The Coast
- Author: Clofas
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111681


![](https://imgur.com/kFdI6mh.png)
  
  
[font=Georgia]Requires [Sound Record Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/77815)﻿[/font][font=Georgia]
  
  
Adds distant whale calls while exploring coastal regions. 
  
Includes 9 different whale sounds to play at different times of day. 
  
  
  
  
FOMOD Included in V1.2
  
  
Select How often Whale sounds trigger
  
  
Original - Chance set to .01
  
Less Often - Chance set to .007
  
More Often - Chance set to .03[/font]
