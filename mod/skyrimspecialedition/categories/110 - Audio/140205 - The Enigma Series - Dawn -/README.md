The High quality remake of the old [Enigma Series: Dragon Voices](https://www.nexusmods.com/skyrimspecialedition/mods/34620)﻿
  
Over 725+ Voice files has been remade using High quality PS4 voicefiles
  
  
The old enigma series uses vanilla pc release LQ horribly compressed which caused alot of issues during/after the rework clipping voices LQ lines got even lower, and with the new remake i made with newer tools and mindset instead of going the same route with all dragons, now each dragon voice is unique on it's own way, i love Durnehviir so much i made two versions of his voice!
  
  
Shouts are included as well as missing files from the old release
