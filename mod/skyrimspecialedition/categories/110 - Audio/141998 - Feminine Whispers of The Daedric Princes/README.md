# Feminine Whispers of The Daedric Princes
- Author: LeoMaximus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141998


![](https://i.imgur.com/mWpcN7n.png)
  
Pitch-adjusted audio from [Clofas](https://next.nexusmods.com/profile/Clofas)' [Whispers of the Daedric Princes](https://www.nexusmods.com/skyrimspecialedition/mods/141931) to sound more feminine.
  
  
  
![](https://i.imgur.com/Ty9hl9v.png)﻿
  
Requires Clofas' original mod to work! Install with the included FOMOD or manually replace the original audio files. If installing with a FOMOD or into a new mod folder, make sure it loads after the original mod so it overwrites the appropriate files.
