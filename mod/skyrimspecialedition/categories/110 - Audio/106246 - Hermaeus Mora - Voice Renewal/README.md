# Hermaeus Mora - Voice Renewal
- Author: LeoMaximus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106246


![](https://i.imgur.com/mWpcN7n.png)
  
Cleaned up many bad takes of audio removing doubling of letter pronunciations, audio distortion, and unnecessary pauses. Then cleaned up the voice, sped it by 20% (any faster sounded weird), and added filters I thought better fit Hermaeus Mora.﻿
  
  
  
![](https://i.imgur.com/Ty9hl9v.png)
  
No ESP, just loose audio files. Safe to add or remove at any time. Use the mod manager of your choice or drop the contents of the zip file into your Data folder.
  
Comes in WAV format by default, but there is an optional FUZ for download too. Make sure it loads after [Unofficial High Definition Audio Project](https://www.nexusmods.com/skyrimspecialedition/mods/18115) and [Phoenix Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/55221) if you have those mods installed. Conflicts or overwrites will only show if using the FUZ files.﻿﻿
  
NO ESP! 64 loose audio files in FUZ format. Safe to add or remove at any time. Ensure it loads after any other mods that alter Hermaeus Mora's voice. Let me know if there are any other mods that may need a patch to go with this. AFAIK there aren't any.
