# Nocturnal Revoiced
- Author: LeoMaximus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/166332


![](https://i.imgur.com/mWpcN7n.png)﻿
  
Nocturnal is the Daedric Prince of the night, darkness, and mystery. Yet her voice in Skyrim is simply the female dark elf with a reverb effect. This new voice gives Nocturnal an ethereal, but familiar sound. One better fitting the enigma in shadows.
  
  
**Patches are provided for the following mods**

Zero install requirements. This mod is simple loose audio files in .fuz or .wav format. Provided patches come in matching formats to ensure maximum compatibility.
  
Download the main mod by itself. Patches are provided as optional downloads.
  
Ensure these files load AFTER any existing audio mod that changes how Nocturnal sounds.
