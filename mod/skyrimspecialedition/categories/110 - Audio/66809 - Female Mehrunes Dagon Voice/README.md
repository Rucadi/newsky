# Female Mehrunes Dagon Voice
- Author: LeoMaximus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/66809


![](https://i.imgur.com/mWpcN7n.png)
  
I hired an awesome Fiverr voice actress to change Mehrunes Dagon lines so they better match PSBoss's New Statue of Mehrunes Dagon SE mod.
  
  
  
![](https://i.imgur.com/Ty9hl9v.png)
  
Zero plugins, just simple loose audio files in four variants. Install/uninstall at any time.
  
Tagged as not lore friendly. I don't keep up with it so I assume it's not.
  
Any other references to Mehrunes Dagon were not changed. Some NPCs and literature may still refer to the character as a male Daedric prince.
