**Quest Walkthrough**Spoiler

1. You must be at least level 20.
Trigger courier by going to a city. If you're in a city already, go to a different city. Courier will give you a letter.
  
2. Read the letter.
Go to bandit camp indicated by the letter.
  
3. Kill Isabile Frascel
Go back to a city to trigger Courier. Courier will give you a second letter.
  
4. Read the letter.
Read the armory plans to unlock crafting. Crafting requires plans to be in inventory.
  
**FAQ**
> Q: How to get the armor?
>   
> ﻿A: Either by finishing the introductory quest, using something like [AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563), or using console commands.﻿
>   
>   
> Q: Courier won't show up
>   
> A: You need to be level 20. Travel to a different city than where you are now. If quest still doesn't start, you might have reloaded a save causing a Skyrim bug; restarting the game will resolve that.
>   
>   
> Q: The quest won't end.
>   
> A: Be patient. It is not over yet. SpoilerAnother courier is coming.
>   
> Q: Helmet plume is broken
>   
> A: You need to install [HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿ properly. Don't ask us about how to use [HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿. If you don't want to use [HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿, build the non-HDT version of the helmets in [Bodyslide](https:////www.nexusmods.com/skyrimspecialedition/mods/201)Q: Helmet visor is down
>   
> A: You need to select the [visor up option](https://i.imgur.com/ByWKcpT.png)﻿ in [Bodyslide](https:////www.nexusmods.com/skyrimspecialedition/mods/201).
>   
>   
> Q: How to use [Bodyslide](https:////www.nexusmods.com/skyrimspecialedition/mods/201)﻿, [BHUNP](https://www.nexusmods.com/skyrimspecialedition/mods/31126)﻿, [UUNP](https://www.nexusmods.com/skyrimspecialedition/mods/43773)﻿, [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/198)﻿, [CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174) and [HIMBO](https:////www.nexusmods.com/skyrimspecialedition/mods/46311)?
>   
> A: This mod supports [BHUNP](https://www.nexusmods.com/skyrimspecialedition/mods/31126)﻿, [UUNP](https://www.nexusmods.com/skyrimspecialedition/mods/43773)﻿, [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/198)﻿, [CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174), and [HIMBO](https://://www.nexusmods.com/skyrimspecialedition/mods/46311)﻿ bodyslide. Don't ask us about how to use [Bodyslide](https://://www.nexusmods.com/skyrimspecialedition/mods/201)﻿, [BHUNP](https:////www.nexusmods.com/skyrimspecialedition/mods/46311)﻿, [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/198)﻿, [CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174), or [HIMBO](https:////www.nexusmods.com/skyrimspecialedition/mods/46311)﻿.
