**Introduction**
  
  
[font=Arial]After the success and outpour of great comments for my vigilant SE retexture, I decided to start working on a similar mod, for Unslaad. I learned a lot while working on these mods. For vigilant, I mostly modified the original textures but for Unslaad, I decided to redo most of the textures from scratch. I am very happy with the results and I hope you will be too![/font]
  
  
**About**
  
  
This mod retextures most weapons and armors from [Unslaad SE](https://www.nexusmods.com/skyrimspecialedition/mods/11789). Most textures are in 4k, while a few are 2k and some non-square textures are 2048x4096. I tried to stay true to Vicn's original designs, while adding some more detail and life to his great models.

**[font=Trebuchet MS]Installation[/font]**
  
  
Simply install using your favorite mod manager. If you are using the [Unofficial Unslaad LOTD patch](https://www.nexusmods.com/skyrimspecialedition/mods/56647), **YOU NEED TO UPDATE TO VERSION 1.2 OR NEWER!!** or at the very least load my mod after it. 
  
**[font=Trebuchet MS]Compatibility[/font]**[font=Arial]
  
[/font]
  
This mod is simply a texture and meshes replacer, so it's safe to install or uninstall at any time. The only incompatibility is pre 1.2 versions of the unofficial Unslaad LOTD patch, due to some leftover unused meshes in older version that could overwrite some of my meshes.
  
  
If you are using a mesh replacer for the Alt Mora armor, you will need a very simple patch. Simply contact me and I will see what I can do.
