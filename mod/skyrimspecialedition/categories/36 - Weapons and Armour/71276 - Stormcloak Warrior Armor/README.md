[font=Lucida Sans Unicode]**Stormcloak Warrior Armor**Made by [**NordwarUA**](https://www.nexusmods.com/skyrimspecialedition/users/53732701)
  
Heavy Armor
  
  
New set of craftable Stormcloak Warrior Armor and Weapons, beast races and both genders supported.
  
You can craft the set without perks, but the upgrade benefits from Advanced Armors.
  
Compatible with Survival Mode, and both **1.5.97** and **1.6+** versions of the game.
  
  
If you're looking for a replacer check out [**Realistic Armor Overhaul**](https://www.nexusmods.com/skyrimspecialedition/mods/91542).
