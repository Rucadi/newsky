# Armor of Mehrunes Dagon
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/113809


[Patreon](https://www.patreon.com/chakrasse)﻿      [Ko-Fi](https://ko-fi.com/chakrasse)﻿      [Discord](https://discord.gg/sMxZ6chT8c)
  
  
This mod adds a standalone heavy armor set and mace inspired by Dark Fantasy.
  
  
The armor all share almost the exact same stats and recipes as vanilla Ebony armor, which means you craft them at a forge under the Ebony section! (Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Dagon's [Armor Piece]''. Weapon is ''Dagon's Mace''.
  
  
[HDT-SMP version can be downloaded here!](https://www.nexusmods.com/skyrimspecialedition/mods/118199) Males and Females have custom meshes. Textures are all 4K Resolution! 
  
  
There is 1 main file for this mod.
