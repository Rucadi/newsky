**BALFIERAN KNIGHT**

**My patches by Xtudo**

**Description**
  
**My patches for this amazing mod
  
  
Fixes and tweaks**
  

  
* Full beast races support for helmets and armor.
Added "Warm" keyword for the Survival Mode.
  
* Added missing PerkFistsSteelPlate keyword to the gauntlets, as it is a heavy armor.
Fixed clipping with bare hands and feet.
  
* Added proper blood effects to the sword.
Removed some black lines from the sword texture.
  
* Added sword missing temper recipe.
Improved collisions of all ground items.
  
* ESL file that works with any Skyrim version. It will replace the original BalfieranKnightSet.esl file.
It's fine to install/update mid-game.
  

  
**Optional files**
  

  
* **1K optimized textures.** The original mod is 2K.
**Non-HDT version**. All fixes already included, install only one Main file.
  
* **Estormo SPID**. Requires one of the Main files installed first.
**Agent Lorcalin SPID**. Requires one of the Main files installed first.
  

  
**Requirements**
  

  
* [Balfieran Knight Set](https://www.nexusmods.com/skyrimspecialedition/mods/110420)﻿v1.0.
Optional: [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿.
