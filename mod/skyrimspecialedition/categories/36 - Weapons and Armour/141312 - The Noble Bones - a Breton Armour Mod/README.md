# The Noble Bones - a Breton Armour Mod
- Author: Marijn191
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141312


This mod adds a new Breton armour set along with a sword and shield and a unique hood for necromancers. All of this can be found in a new dilapidated fort on the border of High Rock.
  
  
The armour is unisex, and is a tiny bit better than steel plate armour. The quest is meant to be played around level 15. Also, the sword swings just a tiny bit faster than a normal sword.
  
  
You can start the (short) quest by reading the note added to the counter of the Silver-Blood Inn in Markarth.
  
  
This is my first foray into quests, worldspaces etc. So I'm curious what you think!
