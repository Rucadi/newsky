# Pig Iron Set - SPID and CID
- Author: Nite
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136087


Gives the Pig Iron Armor and Mace to Orc Bandits and some Vendors via SPID and CID
