# Arsenal of Twisted Faith
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/53481


This mod adds a standalone custom made armor set and two handed greatsword inspired by Dark Fantasy.
  
  
The armor and greatsword all share the exact same stats and recipes as vanilla Ebony armor, you can find everything in a chest in Bleak Falls Sanctum! (Check Screenshots, Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Twisted Faith [Armor Piece]''. Weapon is ''Twisted Faith Scimitar''.
  
  
No HDT SMP on this mod, Males and Females have custom meshes. Textures are all 4K Resolution! If anyone wants there to be a HDT SMP version, **message me and we can discuss!**
