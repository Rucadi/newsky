# Snazzy Common Wardrobes - Base Object Swapper
- Author: gutmaw
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/92809


[font=Times New Roman]

***Snazzy Common Wardrobes - Base Object Swapper*****Using Base Object Swapper, randomly distribute 3 different custom, [b]3D object filled** Common Wardrobes to all vanilla and mod-added Common Wardrobes.
  
  
![](https://staticdelivery.nexusmods.com/images/1704/9671587-1737902721.gif)﻿
  
  
[/b]

**﻿**[/font]
  
[font=Times New Roman]Mod Features:
  
~ 3 custom Common Wardrobes filled with actual clothing and items that would normally be lootable from them.  
  
~ Wardrobe doors swing all the way open prior to taking you to the loot menu, revealing the clothes and items inside!
  
~ Using Base Object Swapper, randomly replaces any vanilla or other mod-added Commonwardrobe01 with one of these three new varieties.~ Uses any SMIM or SMIM-based Common Furniture texture replacer of your choice (SMIM, Rustic Skyland, Noble Skyrim, etc.)
  
~ Esl-flagged plugin
  
  
  
Compatibility:
  
[/font]

  
* [font=Times New Roman]Must use with [/font][font=Times New Roman][Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) to get the distributed variety to all Noblewardrobes01﻿[/font]
[font=Times New Roman]May conflict with other mods that add CommonWardrobe01 models in a funky way (inside walls, scaled real small/large, etc).  Clipping/size issues can be fixed by adding the offending references as exceptions to the BOS SWAP.ini[/font]
  
* [font=Times New Roman]Requires [SMIM](https://www.nexusmods.com/skyrimspecialedition/mods/659)﻿ or SMIM-based mesh/texture replacer [/font]
[font=Times New Roman]Patch for [Weapons Armor Clothing and Clutter Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/18994)﻿ included in Optional Files[/font]
  
* [font=Times New Roman]Patch for [﻿Skyrim 3D Furniture](https://www.nexusmods.com/skyrimspecialedition/mods/23552)﻿ included in Optional Files[/font]
[font=Times New Roman]Patch for [/font][font=Times New Roman][Rally's Common Furniture](https://www.nexusmods.com/skyrimspecialedition/mods/110370)﻿ included in Optional Files[/font]
  
* [font=Times New Roman]Optional HQ Versions requiring textures from [Basic Dining Set Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/30055)﻿,  [JS Purses and Septims SE](https://www.nexusmods.com/skyrimspecialedition/mods/37306)﻿, and [Realistic HD Baskets Remastered](https://www.nexusmods.com/skyrimspecialedition/mods/22199) or [Forgotten Retex Project](https://www.nexusmods.com/skyrimspecialedition/mods/7849) [/font]
