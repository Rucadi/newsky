STATUE OF MARA****My Interior patches by Xtudo**
  
**Description**
  
**My Interior patches for the amazing lore-friendly statue of Divine Mother, Mara.**The "STANDALONE version" means that they only require the original meshes and textures. So you can replace the statues in LOTD and not in the Temple of Mara, for example.
  

  
* **USSEP** version. The original plugin with the USSEP fixes. All other patches/versions have the USSEP fixes too.
**2K textures**.
  
* **JK's Temple of Mara** patch.
**Lux** patch.
  
* **Lux - JK's Temple of Mara** patch.
**Lux - Variants - Temple of Mara: ﻿Valentine Day** patch.
  
* **Lux - Variants - Temple of Mara: ﻿Valentine Day - JK's Temple of Mara** patch.
**Lux - Distinct Interiors** patch.
  
* **Lux - Distinct Interiors - JK's Temple of Mara** patch.
**Lux - Dawnguard DLC** patch.
  
* **Enhanced Lights and FX - ELFX** patch.
**Enhanced Lights and FX - ELFX** - **JK's Temple of Mara**patch.
  
* **Alternate Start - Live Another Life.** Standalone version. Original size version available in the Optional files.
**Alternate Start - Live Another Life - Lux**. Standalone version. Lux edits to the prison cell are included too.
  
* **Alternate Start - Live Another Life - ELFX**. Standalone version. ELFX edits to the prison cell are included too.
**VIGILANT** patch. Standalone version. **Fully compatible with v1.8.0.**
  
* **Legacy of the Dragonborn - LOTD** patch. Standalone version. 100% compatible with LOTD v5 and v6.
**Distinct Interiors** patch.
  
* **Loading Screen** patch. Miscellaneous files. Including a new text. I hope you like it.
**Leaf Rest**patch. Standalone version. **Statue Turned Around version** available, for if your statue is in the wrong position.
  
* **Hjertesten Hall - Player Home and Settlement** patch. Standalone version.
**Gods And Worship** patch. Standalone version. Lux version available.
  
* **T'Skyrim - Riverwood  - Leaf Rest patch**. Two versions available, Statue default position and turned around. It's safe to install/switch versions mid-game.
**Lakeview. Manor - As It Should Be** patch.
  
* **Mara's Embrace - Riften Brothel** patch. Load it before "Lux - Mara's Embrace patch.esp" if you use it.
**Aemer's Refuge SE** patch.
  
* **Dawnguard DLC** patch.
**Castle Volkihar Rebuilt** patch. Only install it if you are joining the Dawnguard faction in the main DLC quest. Don't use it together with the Dawnguard DLC patch. [Daedric Shrines - All in One version here](https://www.nexusmods.com/skyrimspecialedition/mods/78809)﻿.
  
* **Alternate Start Reborn** patch. Do not use the original **ASLAL Reborn - Vanilla Statue of Azura Patch** with it.
**HS Riften - Temple of Mara.** Install it AFTER **HSRiften - Temple of Mara.esp**, but BEFORE the LUX or ELFX patch (if you use them).
