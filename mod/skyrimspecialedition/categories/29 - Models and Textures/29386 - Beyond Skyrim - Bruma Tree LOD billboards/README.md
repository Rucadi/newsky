Tree LOD Billboards for Beyond Skyrim Bruma

![](https://68.media.tumblr.com/a7fc36e249ccbabd03b545a878cb2ee7/tumblr_oshvykLnhu1s2ptqso6_540.jpg)
  
  
  
  
GENERAL INFO:  
  
These billboards will allow the tree LOD for landscape overhauls, that change vanilla trees, to be fully compatible with [Beyond Skyrim Bruma](http://www.nexusmods.com/skyrim/mods/84946/?)
  
  
  
All large trees and some small shrubs are included  
  
  
  
  
![](https://68.media.tumblr.com/bf62d75f71fcd057ee675b2fda2f16de/tumblr_oshvykLnhu1s2ptqso3_540.jpg)
  
  
  
  
  
  
Installation:  
  
  
1. Install vanilla billboards and then the billboards for all other mods with tree LOD (include Bruma billboards).  Make sure to install the billboards in the **same order** that you installed your various tree mods
  
  
  
2. Run SSELODGen or DynDOLOD.  Only generate **static tree LOD** with either program and **do not generate object and or dynamic LOD** (you will get errors...see [this post](http://forum.step-project.com/topic/11997-dynamic-distant-objects-lod-dyndolod-226/?p=204312) by Sheson for more information).
  
Please let me know in the comment section if this information has changed
  
  
Make sure to click the worldspace "BSHeartland" and generate LOD
  
  
3.  Enjoy (and post pics when you get a chance!)
