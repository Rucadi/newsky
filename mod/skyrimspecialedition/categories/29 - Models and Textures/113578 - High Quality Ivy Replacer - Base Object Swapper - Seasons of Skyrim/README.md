# High Quality Ivy Replacer - Base Object Swapper - Seasons of Skyrim
- Author: fxxkthisworld
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/113578


**DESCRIPTION:**﻿
  
Replaces ivy meshes and textures. Also replaces the meshes and textures of the reach fern, because it's quite similar to ivy, so I gave it a different look. Textures in resolution 4096x2048. The largest meshes have 1000 tris, so this will not impact performance.
  
  
﻿The mod replaces the following meshes:
  

  
* ﻿ivy01.nif (has 2 additional meshes when using base object swapper)
ivy02.nif (has 2 additional meshes when using base object swapper)
  
* ﻿ivy03.nif (has 2 additional meshes when using base object swapper)
stonewall01ivy.nif
  
* stonewallendl01ivy.nif
stonewallendr01ivy.nif
  

  
**REQUIREMENTS:**[﻿Base Objest Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) - soft requirement for static version, necessary if you want more ivy meshes. Hard requirement for animated version.
  
  
**INSTALLATION:**Just install the mod in the Data directory or use any mod manager.
  
  
**RECOMMENDED MODS:**
  
[Nature of the Wild Lands - forest and trees improvement mod](https://www.nexusmods.com/skyrimspecialedition/mods/63604)
  
[Wildlands Renewal - grass and groundcover improvement mod](https://www.nexusmods.com/skyrimspecialedition/mods/139896)
  
[Reimagined Mountain Flowers](https://www.nexusmods.com/skyrimspecialedition/mods/135460)
  
  
**COMPATIBILITY:﻿**﻿
  
This mod replaces the ivy stonewall meshes (stonewall01ivy.nif, stonewallendl01ivy, stonewallendr01ivy), so may be incompatible with mods that replace these meshes.
  
Includes patches for [Nordic Stonewalls](https://www.nexusmods.com/skyrimspecialedition/mods/57686), [High Poly Project](https://www.nexusmods.com/skyrimspecialedition/mods/12029)﻿, [3D stonewall](https://www.nexusmods.com/skyrimspecialedition/mods/86479)﻿, so this mods are fully compatible.
  
The patch for the [Skyrim 3D StoneWalls](https://www.nexusmods.com/skyrimspecialedition/mods/14406) is available [here](https://www.nexusmods.com/skyrimspecialedition/mods/113624).
  
When using [3D Trees](https://www.nexusmods.com/skyrimspecialedition/mods/12371), my mod should overwrite the ESP plugin of this mod.
  
The mod is compatible with [Seasonal Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/66903), but must overwrite it.
  
The mod includes ivy meshes for [JK's Fort Dawnguard](https://www.nexusmods.com/skyrimspecialedition/mods/110645) and [Lux Via](https://www.nexusmods.com/skyrimspecialedition/mods/63588). [Seasons of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/62861) are also supported.
