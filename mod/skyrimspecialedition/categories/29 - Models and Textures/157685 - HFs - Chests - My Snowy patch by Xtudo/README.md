# HFs - Chests - My Snowy patch by Xtudo
- Author: Author Halffaces - Snowy patch by Xtudo

**![](https://staticdelivery.nexusmods.com/mods/1704/images/headers/157685_1756078077.jpg)﻿****HFs - CHESTS - SNOWY PATCH**
  
**By Xtudo**

**Description**
  

  
* Adds HD snowy versions of the exterior vanilla snowy chests, like the one outside High Hrothgar.
It will replace **HFs\_Chests\_remodel.esp** form the original **HFs Chests remodel - DLC03 hotfix Update file**. ESPE that will work with any Skyrim version.
  
* **Default** and **Legacy of the Dragonborn** versions available. The LOTD version carries on edits to the loot of some chests.
**4K**, **2K** and **1K** versions available.
  
* Install only one of the Main files from here. It's fine to install/update mid-game.

  
**Requirements**
  

  
* [HFs - Chests - remodel.](https://www.nexusmods.com/skyrimspecialedition/mods/153839) Just one of the Main files is required, only meshes and textures. The Update file is not required.﻿﻿﻿
Optional: [Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802)﻿.
