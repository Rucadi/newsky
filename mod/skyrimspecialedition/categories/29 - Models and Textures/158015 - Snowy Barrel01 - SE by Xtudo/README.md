**Snowy Barrels**
  
**by Xtudo**

  
**Description (Small details that make big differences)**
  
**Adds snow to supposedly snowy barrels, used 68 times in-game.**
  

  
* **Barrels edited:**BarrelFood01\_Snow, BarrelEmpty01\_Snow, BarrelIngredientCommon01\_Snow and BarrelIngredientUncommon01\_Snow.

  
**Versions available (install only one)**

  
* **Vanilla.** With and without animations versions available.
**SMIM.** Vanila and Rustic textures, with and without animations versions available. 
  
* **FYX-SMIM.** With and without animations versions available.
**ALT-Barrels.** With and without animations versions available. 
  
* **ALT-Barrels - Diversity - BOS**. With and without animations versions available. It can be used together with other patches, like SIMM, Real 3D Barrels, etc.
**Renthal311 - Real 3D Barrels. Default** and **Less Color** versions available.
  

  
**Requirements**
  

  
* Optional: [Static Mesh Improvement Mod - SMIM](https://www.nexusmods.com/skyrimspecialedition/mods/659)﻿.
Optional: [FYX - SMIM Barrel](https://www.nexusmods.com/skyrimspecialedition/mods/76148)﻿.
  
* Optional: [ALT - Barrels](https://www.nexusmods.com/skyrimspecialedition/mods/95475).
Optional: [Renthal311 - Real 3D Barrels](https://www.nexusmods.com/skyrimspecialedition/mods/140611)﻿.
