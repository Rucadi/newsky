# Enhanced Rocks and Mountains - Fix and Addon
- Author: Cl3mus - CaptainCockerel
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/142493


[font=Georgia]Enhanced Rocks and Mountains – Fix and Addon
  
  
Enhanced Rocks and Mountains – Fix and Addon is a complement to the main Enhanced Rocks and Mountains mod. It extends its improvements to underground environments and adds a specific fix for the Skyforge meshes in Whiterun.
  
  
  
Features
  
  
  
  
ERM Addon
  
  
  
  
Replaces mine and cave meshes with ERM versions to improve detail and maintain visual consistency with outdoor landscapes.
  
  
Underground environments benefit from:
  
  
- Corrected UVs (reduces parallax seams)
  
- Improved collisions
  
- Better parallax behavior with Complex Material setups
  
  
Includes full support for Icy Caves, compatible with:
  
  
- Vanilla version
  
- Icy Cave Remaster
  
- Icy Cave Remaster – Complex Parallax Material
  
- Skyrim Remastered
  
  
  
  
  
 Skyforge Fix
  
  
  
  
- Updated UVs
  
- Collision reworked
  
- Footstep sounds corrected
  
- A small patch of dirt was added to mask the seam between the rock and the surrounding pavement
  
- Uses the same mesh settings as ERM, ensuring perfect visual consistency
  
- Includes a new LOD mesh, should be compatible with DynDOLOD
  
  
  
  
  
StoneRessource Fix
  
  
  
  
- Collision Reworked
  
- New blending method for the StoneQuarry01
  
- Uses the same mesh settings as ERM, ensuring perfect visual consistency
  
  
  
  
  
**Installation****- Download and install [Enhanced Rocks and Mountains](https://www.nexusmods.com/skyrimspecialedition/mods/121336)﻿** (required mod).
  
- Install **this mod** using your preferred mod manager
  
- If you're using *Icy Cave Remaster*, choose the appropriate version during installation.
  
- Make sure this mod loads after ERM and ICR in your load order. ( and other requirements.. )
  
- Run [PGPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/120946)﻿ !
