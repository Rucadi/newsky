# Edmond's Official Unique Flowers and Plants SSE
- Author: Edmond Noir
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/29154


![](https://i.imgur.com/VPAyeES.png)

  
  
  
  
**![](https://i.imgur.com/1ULElp4.png)**DescriptionHi Guys Edmond Here - I ran the meshes through nif optimizer, and edited the .esp so that its patched for Skyrim SE and works with all the DLCs. There also was a orange flower in the original version which had a broken normal map making it glow in shaded areas - I removed this flower to increase immersion. There were only 60 instances of this flower in game. I was given permission by the original skyrim LE mod author to publish this fixed version updated for Skyrim SE. Additionally the mod is now also offered in both ESL and ESM. This mod aims to add new Flowers and Plants across Skyrim. All the Flowers and Plants are handplaced. Currently this mod covers the Pine Forest, Reach, and Fall Forest regions.

* Only Skyrim is Required (Any Version)

**![](https://i.imgur.com/1ULElp4.png)**Installation

  
* Please use a mod manager to install the mod

**![](https://i.imgur.com/1ULElp4.png)**CompatibilityLoad this at the top of your load order below only Unofficial Skyrim Patch.  Should be compatible with  pretty much every mod since this only adds handplaced Flowers and Plants.  You might  see weird  placement if any mod drastically edits the same cell. Other than that there shouldn't be much of a problem.
  
  
Please note also that this is compatible with Skyrim Flora Overhaul and all Grass mods like Unique Grasses and  Groundcovers. The mod adds full 3D Plants and Flowers and not Grass which is spawned randomly on the  attached Texture. After through testing I have also noted that there is no performance impact.![](https://i.imgur.com/1ULElp4.png)Credits
