# Imperial Tents Animated
- Author: StoneSpiralGaming
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/101500


This Base Object Swapper made mod replaces the Imperial Large and Small tents Meshes with animated ones (using a vanilla mesh that I've split) making it look like the wind is blowing though them.
