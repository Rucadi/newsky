**WHAT IS INCLUDED?**
  
  
This mod retextures the iron, steel and ebony variants of the Spell Knight Armors
  
  
  
  
Chest textures are 4k, everything else is 2k
  
  
Half-res version available in Optional files (chest/robes are 2k, everything else 1k)
  
  
Low-res version also available in Optional Files (chest/robes are 1k, everything else is 1k)
