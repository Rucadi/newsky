# HFs - Archery target remodel - Diverse BOS or Model Swapper
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/138492


My new meshes with HD textures for the archery target.
  
There are 3 different models available, which will randomly replace the original vanilla model in the game. The mod also contains my new model of the tripod (only one variant), since in the game the tripod for the archery target also exists as a standalone mesh, as an element of decoration (clutter).

[font=Georgia]**Compatability:**[/font]

  
Should be fully compatible with anything except mods from other author's that modify *archerytargettripod01.nif* and *archerytarget01.nif*.
  
  
Not compatible with mods that make any changes to the *archerytarget01* and *archerytargettripod01* NIFs, models, IDs, etc., included in this mod, even if it is a simple name or folder structure change. Not compatible with any other *archerytargettripod01.nif* and *archerytarget01.nif**.* meshes replacers and/or retextures.
  
  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
