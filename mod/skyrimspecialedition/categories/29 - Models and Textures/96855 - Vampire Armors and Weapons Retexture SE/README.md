This mod retextures the Vampire Royal armor, all 3 variants of the Vampire armor (black, grey and red), Valerica's unique version of the Vampire royal armor and Harkon's sword. Also contains an optional retexture for Yamanotaka's [Volkihar Knight - Vampire Armor](https://www.nexusmods.com/skyrimspecialedition/mods/4806)'s 3 variants of the volkihar armor set.
  
  
**Serana's hood shares textures with the thieves guild set, so it will be included in my upcoming Thieves Guild Retexture**
  
This mod contains textures only. Meaning it is compatible with every mod that works with vanilla textures. 
  
  
  
  
  
**FOMOD OPTIONS**
  
  
Fomod lets you pick and choose what you want. 
  
  
Textures are 2k except Chest armor textures, which are 4k.
  
  
Half-res version available in Optional files (Chest textures are 2k, everything else is 1k)
