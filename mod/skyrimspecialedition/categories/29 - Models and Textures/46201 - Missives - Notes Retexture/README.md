# Missives - Notes Retexture
- Author: JaySerpa
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/46201
[![](https://i.imgur.com/Pvfeiqq.png)](https://www.nexusmods.com/users/5201727?tab=user+files&BH=2)﻿  [![](https://i.imgur.com/NQIZcnZ.png)](https://www.twitch.tv/jayserpa)﻿  [![](https://i.imgur.com/uGltkW7.png)](https://www.patreon.com/JaySerpa)
  
  
  
This is a small addition to [Missives](https://www.nexusmods.com/skyrimspecialedition/mods/17576?tab=images) by the legendary IronDusk33. The blank notes will now have images. This only affects the notes on the board.
  
  
As for the notes:
  
- Notes display some of the common request available in Missives: Gather ingredients, defeat bandits, hunt beasts...
  
- I added no actual readable text, so it will fit your game no matter the language.
  
  
Installation:
  
- This mod only includes a mesh and the new textures, so you will need the original Missives on top.
  
- Make sure you endorse Missives and give IronDusk33 some love, so he never leaves the community and keeps on giving us gems like Missives, Spell Research or Wounds.
