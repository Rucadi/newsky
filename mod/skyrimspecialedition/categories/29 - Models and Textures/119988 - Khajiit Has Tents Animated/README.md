# Khajiit Has Tents Animated
- Author: StoneSpiralgaming
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119988


﻿This Base Object Swapper made mod replaces the tents [Khajiit Has Tents](https://www.nexusmods.com/skyrimspecialedition/mods/119847) in with animated ones (using meshes that I've edited and vertex painted) making it look like they are rustling in the breeze.
  
  
Khajiit Has Tents is a Hard Requirement for this mod, as is Base Object Swapper
  
  
![](https://img.bugthesda.net/i/96cad4a2-de3e-438e-8c2a-057738e85315.gif)
  
  
﻿Enjoy!
