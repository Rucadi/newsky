# Creation Club Arrows - ENB Light Addon
- Author: monkeyangie
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68571


**![](https://pixelsafari.neocities.org/dividers/vine.gif)
  
Description
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)**

Simple mod that adds ENB Light to the Creation Club Arrows from Saints and Seducers, Arcane Archer Pack and Rare Curios
  
  

**Installation
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)**

  
Drop everything on your Data folder or use a Mod Manager, of preference use MO2
