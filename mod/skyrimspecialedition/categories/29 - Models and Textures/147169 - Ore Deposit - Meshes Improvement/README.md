# Ore Deposit - Meshes Improvement
- Author: Neutrall9
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/147169


[font=Comic Sans MS]What does this mod do?
  
This mod removes small rocks and unnecessary planes and replaces some meshes with high-polygonal ones with improved UV maps, which I took by [Janissrim](https://next.nexusmods.com/profile/Janissrim?gameId=1704) from the [Cave Rocks Improved](https://www.nexusmods.com/skyrimspecialedition/mods/120229) mod.
  
  
You can see all the changes in the screenshots.
