# Vitruvia - Skin texture overhaul for males
- Author: mandragorasprouts
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/9112
*![](https://staticdelivery.nexusmods.com/mods/1704/images/9112-0-1490475025.png)*

*This mod adds new detailed body and face textures for not-beast race males.
  
Men won't be too muscled or hairy as if they are on steroids. 
  

  
* [i]New hand-painted diffuse textures.*
  
*Reworked face normal maps, not too ridged, but still close to vanilla look.*
  
- *Nude or never nude versions (used body meshes from "better males" by Sundracon and FavoredSoul).*
*Options -  hairless or hairy chest, body scars.*
  
- In all-in-one optional installer you can choose between 2k, 4k and compressed texture options.
Version for "Schlongs of Skyrim -Light SE" also available
  
*Textures are also compatible with [4uDIKs Real body](http://www.nexusmods.com/skyrim/mods/6667/?) , download main file manually, extract  body mesh option you want to use in 
  
Data\Meshes\actors\character\character assets\**Do not use feet meshes ,because it has different UV mapping and not compatible with my textures, only body mesh.
  
If you choose "less hairy" or "more hairy" options, you need body hair textures too. Grab these texture files - [i]4rb\_m\_ph01.dds, 4rb\_m\_ph01\_n.dds, 4rb\_m\_ph02.dds, 4rb\_m\_ph02\_n.dds* from 4uDIKs mod and copy in:
  
Data\Textures\actors\character\male\
  
  
  
[/i]P.S. I will never add muscled,hairy and extreme vein options.
  
  
[/i]

**SOS version:**
  

  
1. Uninstall regular version of Vitruvia  completely.
Download  "Schlongs Of Skyrim Light SE" from loverslab - [Here](http://www.loverslab.com/topic/73132-schlongs-of-skyrim-light-se/) and install it with mod manager.
  
2. Download "Vitruvia - SOS version" , install with NMM, ***overwrite all files when asked***.
