# Iconic Statues
- Author: Iconic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/46708


**Overview**
  
  
  
  
  
**High resolution retexture covering every single statue in Skyrim; with the option to pick and choose which statues are installed, and Stunning Statues of Skyrim patches.
  
Optional Wuuthrad retexture that matches the Ysgramor statue.**
  
**Saved using BC7 compression for maximum quality.**
  
**Available in 8K, 4K, and 2K.**

**Notes**
  
The meshes from Stunning Statues were only used to make patches. 
  
Stunning Statues of Skyrim textures are still required for some of the patches to work properly﻿.
  
  
Textures are *not* done from scratch; so while miles better than vanilla, some of them will not look amazing.
  
You'd need a full mesh rework and high quality textures done from scratch for that.
