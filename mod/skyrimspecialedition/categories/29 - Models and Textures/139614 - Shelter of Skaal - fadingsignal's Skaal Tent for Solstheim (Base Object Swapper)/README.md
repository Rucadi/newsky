# Shelter of Skaal - fadingsignal's Skaal Tent for Solstheim (Base Object Swapper)
- Author: h495
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/139614


Description
  

  
* Uses [Base Object Swapper](http://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿
Replaces several tents near Skaal Village on Solstheim with the tent from [Solstheim - Skaal Fishing Camp](https://www.nexusmods.com/skyrimspecialedition/mods/14450)
  
* Some tents were scaled or rotated to avoid clipping without adjusting other objects, but results may depend on your mod setup
Standalone; Does not require the [original mod](https://www.nexusmods.com/skyrimspecialedition/mods/14450) but you should go download and endorse it
  

  
Requirements
  

  
* [Base Object Swapper](http://www.nexusmods.com/skyrimspecialedition/mods/60805)
