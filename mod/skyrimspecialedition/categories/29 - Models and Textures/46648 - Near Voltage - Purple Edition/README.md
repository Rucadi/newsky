# Near Voltage - Purple Edition
- Author: SRW0 and NearDaniel
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/46648


**Near Voltage - Purple Edition**
  
  
[font=Lucida Sans Unicode]This mod is a re-texture of SRW0's [Voltage](https://www.nexusmods.com/skyrimspecialedition/mods/15565)*,* it's an amazing mod but the blue color pallet always bothered me a little, specially after I started using [Strange Runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456)﻿. Strange Runes has purple runes and the voltage blue color threw me off a bit so I decided to recolor it to match Strange Runes. 
  
  
  
What I changed/tweaked to purple:
  

  
* Impact Decals
Gradients
  
* Effects
[/font][font=Lucida Sans Unicode]
  
All the textures are 2K at the highest.[/font]
  
  
[font=Lucida Sans Unicode]
  
Installation MO2
  
  
  
Download [Strange Runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456) (not mandatory but works perfectly)
  
Download the main file to MO2 and simply install it.
