# Particle Lights for ENB - Luminous Ground Cover
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72652


These plants have super cool animated glow in vanilla Skyrim... but of course they cast no light in vanilla. And I didn't add ENB light to them before!!! WHAT'S WRONG WITH ME??!! FLARK!!!...
  
  
So, now I did add ENB light... It can be a subtle effect sometimes and some other times not so subtle depending on their position :)
  
  
I hope you like it ^-^
  
  
**Check the images!! :O
  
  
  
INSTALLATION:**
  
1) Install the main file manually or using any mod manager.
  
  
2) Be sure that you have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
[size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
  
  
**COMPATIBILITY:**
  
-Compatible with everything that doesn't include the same meshes.
