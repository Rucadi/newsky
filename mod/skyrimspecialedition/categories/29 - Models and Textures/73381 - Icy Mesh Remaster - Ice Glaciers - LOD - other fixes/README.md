Installation
  
  
This mod must overwrite most conflicting mods unless stated otherwise!
  
This includes Lux, ELFX, Arctic, HQ cubemaps, Fixed Mesh Lighting, Assorted mesh fixes, SLaWF and etc.
  
  
Load order:
  
[other mods like lux, ELFX, UM, SLaWF, quality cubemaps, arctic]
  
Icy Mesh Remaster
  
Ice retexture
  
(Unreal Ice and Vanilla Ice for ENB have their own meshes, so let them overwrite)
  
  
Load IMR after DynDoLOD resources pack.
  
Make sure to run TexGen to generate the LOD textures.
  
Lod Textures will be whatever ice textures you have installed.
