This is a Winter version for [Bent Pines](https://www.nexusmods.com/skyrimspecialedition/mods/8306) by Vurt.
  
It is intended to be used with [Seasons of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/62861) that provides swap functionality In winter season.
  
Texgen and [Dyndolod](https://dyndolod.info/) 3.0alpha is required to build its lod for winter time. Ultra tree Ready.

  
  
  
[font=Trebuchet MS]**INSTALLATION**[/font]
  
Install with mod manager, rebuild texgen and dyndolod with winter option.
  
  
**INSTALL ONLY ONE MAIN FILE**
