# HFs - Bruma Bedroll - remodel
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141192


A simple mod that brings my new model and HD BC7 textures to replace the original bedroll from "[Beyond Skyrim - Bruma SE](https://www.nexusmods.com/skyrimspecialedition/mods/10917)" mod. Namely ***floorbed02.nif***.
  
  

[font=Georgia]**Compatability:**[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the floorbed02.nif.
  
  
**Not compatible** with mods that make any changes to the the ***floorbed02.nif***, included in this mod, even if it is a simple name or folder structure change. Not compatible with any other meshes replacers and/or retextures for vanilla badroll from "BS Bruma".
  
  
No ESP, no scripts. Mesh and textures only. So it absolutely safe to install/uninstall at mid-game (at any time).
