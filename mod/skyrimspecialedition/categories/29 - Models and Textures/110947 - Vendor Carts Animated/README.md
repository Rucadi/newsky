# Vendor Carts Animated
- Author: StoneSpiralGaming
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/110947
This Base Object Swapper made mod replaces the Vendor Carts and Canopies with animated ones making it look like they are rustling in the breeze.
