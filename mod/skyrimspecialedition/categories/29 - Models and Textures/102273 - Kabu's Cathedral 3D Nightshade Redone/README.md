# Kabu's Cathedral 3D Nightshade Redone
- Author: Kabu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/102273


[![](https://i.gyazo.com/e6ace15c7c6e0e40f6aee7506c0de0c4.png)](https://www.patreon.com/kabunouveau)
  
[![](https://cdn.ko-fi.com/cdn/kofi5.png)](https://ko-fi.com/V7V85BOQ)
  
  
A mystical, moonlit retexture and mesh tweak of the [Cathedral 3D Nightshade](https://www.nexusmods.com/skyrimspecialedition/mods/81284/) plant. Two color variants and new 3D stamen!
  
  
**TECHNICAL DETAILS**
  
As of 1.1, the original Cathedral 3D Nightshade mod is no longer required. Nightshades now have lightweight 3D stamen with some UV fixes as well. Please do still go give the original mod a download and an endorsement as this one would not have been possible without it and without the very generous open permissions.
