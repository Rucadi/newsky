# Spider Egg and Sacs Retexture
- Author: IWantATardis
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/48802


Done in Substance Painter, Photoshop and Nifskope. Seamless spider eggs and spider sacs, i've altered the UV on the webs so they look better. I had to create more diffuse and normals than in vanilla to wrap around each egg mesh. Should be compatible with other stuff out there.
  
  
Meshes are SSE optimized from Renegade79's [Agence Smooth Mesh Mod](https://www.nexusmods.com/skyrimspecialedition/mods/1383)
