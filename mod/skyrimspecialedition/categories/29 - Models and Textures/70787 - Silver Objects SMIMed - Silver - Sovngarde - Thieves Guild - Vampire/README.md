**Description**
  

  
* Makes silver objects smoother with approximately 2x the polygon count
Changed Sovngarde and Thieves Guild objects shaders to be more specular and less diffuse yellow.
  
* Changed Thieves Guild objects gems to be more specular and glass-like.
Fixes some issues with the vanilla assets
