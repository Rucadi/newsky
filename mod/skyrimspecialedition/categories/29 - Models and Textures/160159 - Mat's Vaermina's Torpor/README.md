# Mat's Vaermina's Torpor
- Author: Mat149
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/160159


Custom model and texture replacer for Vaermina's Torpor.
  
Textures are 2K. 
  
Includes a plugin to change the potion's model. As such, it edits one vanilla record, so if the new model does not appear in game, you may have a conflicting plugin.
