# Gemling Queen Jewelry SE
- Author: Saerileth
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/4294
This is a faithful port of [Saerileth](http://www.nexusmods.com/skyrim/users/4522213/?) excellent Jewelry mod [Gemling Queen Jewelry](http://www.nexusmods.com/skyrim/mods/52266/?) with amazing textures made by [Gamwich](http://www.nexusmods.com/skyrimspecialedition/users/5712421). I have kept all features intact, converted all meshes to Skyrim SE new format using [ousnius](http://www.nexusmods.com/skyrimspecialedition/users/4291759/?) excellent tool [SSE NIF Optimizer](http://www.nexusmods.com/skyrimspecialedition/mods/4089/?) & saved the Dawnguard vampire amulet fix esp in the new CK.
