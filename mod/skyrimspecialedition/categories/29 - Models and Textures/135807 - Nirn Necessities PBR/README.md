# Nirn Necessities PBR
- Author: Everglaid and Jiaye
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/135807


**Nirn Necessities PBR**PBR patch for Nirn Necessities, to be used with Community Shaders.
  
  
Install the original mod, run any bodyslide as you want, then patch the meshes with ParallaxGen.
  
Let the output overwrite everything.
