# HFs - Chests - remodel
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153839


My new 3D model for4 vanilla Chests(*chest01.nif*, *ruins\_largechest.nif, chestopen01.nif* and *upperchest01.nif*). And of course my new HD BC7 hand-made textures for this models (**not** *vanilla **retexture***).
  
  

[font=Georgia]Compatability:[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the above chests NIFs.
  
  
**Not compatible** with mods that make any changes to the above ***chests NIFs***, included in this mod, their IDs, names, etc., even if it is a simple name or folder structure change. Not compatible with any other mesh replacers and/or retexture for these three vanilla Chests.
  
  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
