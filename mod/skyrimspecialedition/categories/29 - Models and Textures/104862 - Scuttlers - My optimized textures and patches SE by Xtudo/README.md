**SCUTTLERS**
  
**My patches by Xtudo**

**Description**
  
**My optimized textures and patches for this amazing lovely mod.**
  

  
* Three options available, 4K (original size), 2K and 1K.
4K for example (the same pixel size than the original files), **has the same image quality**, with half the VRAM usage and half the file size. Thus reducing CTD chances, loading times and stuttering. Remember that the game load thousands of textures at the same time, so everything counts.
  
* No ESP,  so it's fine to install or uninstall it at anytime.

  
**Optional files**
  

  
* **Skyrim Addon**. Adds 9 creatures to lore-friendly places, like Hlaalu Farm, New Gnisis Cornerclub, Sarethi Farm, etc. 5 to interiors and 4 to exteriors. Install it BEFORE any mods that edits cells, like Lux, JK's Skyrim, etc. ESL.
**Pet Addon.** Allows to pet the lovely things. Requires [Immersive Interactions - Animated Actions](https://www.nexusmods.com/skyrimspecialedition/mods/47670)﻿ installed first. ESL.
  
* **Pet Addon for****Scrolls for Scuttlers Companions SE**. Allows to pet the lovely things. Requires [Scrolls for Scuttlers Companions SE](https://www.nexusmods.com/skyrimspecialedition/mods/104875) and [Immersive Interactions - Animated Actions](https://www.nexusmods.com/skyrimspecialedition/mods/47670)﻿ installed first. ESL.
