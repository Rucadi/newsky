# HFs - Ysgramor's Soup Spoon - remodel
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141244


***"Ah, now here is an item out of legend. This is Ysgramor's Soup Spoon. Now, I know what you're thinking - this is no spoon, it's a fork! Nobody can eat soup with a fork! Well, my friend, you did not know Ysgramor."***
  
© Calixto Corrium
  
  
My new mesh and HD textures for *Ysgramor's Soup Spoon*. 
  
  
This is a very simple mod that replaces the vanilla Ysgramor's Soup Spoon (which is no different from an ordinary dwemer fork) with my model and unique textures. Bethesda is so lazy that it has absolutely don't wish to create authentic models for unique items in the game. I thought I could fix this oversight. As with my "[HFs - Aretino Family Heirloom - remodel](https://www.nexusmods.com/skyrimspecialedition/mods/140723?tab=description)" mod, I made sure this legendary item had its own original look that differed from its vanilla counterparts.
  
Now this item has its own unique look and I proudly decorated my home with it. Perhaps you will like it too?
  
  
[font=Georgia]**Compatability:**[/font]
  
Should be **fully compatible** with anything except mods from other author's that modify the Ysgramor's Soup Spoon.
  
  
**Not compatible** with mods that make any changes to the the *Ysgramor's Soup Spoon.* So this is not compatible with any other meshes replacers and/or retextures for vanilla Ysgramor's Soup Spoon.
  
  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
