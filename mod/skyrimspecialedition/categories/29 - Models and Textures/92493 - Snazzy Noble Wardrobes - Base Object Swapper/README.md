# Snazzy Noble Wardrobes - Base Object Swapper
- Author: gutmaw
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/92493


[font=Times New Roman]

***Snazzy Noble Wardrobes - Base Object Swapper*****Using Base Object Swapper, randomly distribute 3 different custom, [b]3D object filled** Noble Wardrobes to all vanilla Noble Wardrobes, as well as any other mod-added Noble Wardrobes! [/b]

**![](https://staticdelivery.nexusmods.com/images/1704/9671587-1737872238.gif)

  

﻿**
  
Mod Features:
  
~ 3 custom Noble Wardrobes filled with actual clothing and items (at the NIF level) that would normally be lootable from them 
  
~ Wardrobe doors swing all the way open prior to taking you to the loot menu, revealing the clothes and items inside!
  
~ Using Base Object Swapper, randomly replaces any other mod-added NobleWardrobe01 with one of these three new varieties.
  
~ Used the improved hardware assets from [Skyrim Objects SMIMed](https://www.nexusmods.com/skyrimspecialedition/mods/77499)~ Uses any Noble Furniture texture replacer of your choice (Rustic Furniture, Noble Skyrim, etc.)
  
~ Configurable BOS SWAP.ini to help avoid clipping with the opening doors and contents inside
  
~ Esl-flagged plugin
  
  
  
Compatibility:
  
[/font]

  
* [font=Times New Roman]Must use with [/font][font=Times New Roman][Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) to get the distributed variety to all Noblewardrobes01﻿[/font]

[font=Times New Roman]

  
* [size=4]Checked every vanilla cell, and tested with all JK and Snazzy Interiors mods.  Any issues in vanilla or these mods are dealt with in the BOS SWAP.ini to avoid clipping issues
  

  
* May conflict with other mods that add NobleChestofDrawers01, NobleChestofDrawers02, NobleCupboard01, NobleCupboard02, or NobleNightTable01 models in a funky way (inside walls, scaled real small/large, etc).  Clipping/size issues can be fixed by adding the offending references to the BOS SWAP.ini

  
* Requires [SMIM](https://www.nexusmods.com/skyrimspecialedition/mods/659)﻿ or SMIM-based mesh/texture replacer
