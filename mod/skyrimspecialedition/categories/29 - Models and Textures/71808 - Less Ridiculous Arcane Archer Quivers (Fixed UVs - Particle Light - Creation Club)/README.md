**DESCRIPTION**
  
  
Replaces the quiver of the **Fire Arrow**, **Ice Arrow**, **Lightning Arrow**, and **Telekinesis Arrow**.
  
Changes include removing the elemental orb, leaving the gold buckles, and fixed UV maps.
  
Requires **Anniversary Edition** or the [Arcane Archer](https://en.uesp.net/wiki/Skyrim:Arcane_Archer_Pack) Creation.
  
  
**UPDATE 2.0.0**There is now an ENB Light option available, along with a patch for [Ammo Enchanting](https://www.nexusmods.com/skyrimspecialedition/mods/79764).
  
  
**INSTALLATION**
  
     ■  Download this mod, [Less Ridiculous Arcane Archer Quivers](https://www.nexusmods.com/skyrimspecialedition/mods/71808).
  
     ■  Ensure that no files are overwriting any files in

```
meshes/creationclub/bgssse002/weapons/magicarrows
```
