# Statue of Shalidor - Mage Statue Replacer
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/52483
Adds spell casting hand effects with ENB light to the Winterhold Mage statue.
  
  
Comes in two flavors - Fire/Frost and Fire/Shock.
  
  
This is a mesh replacer, so it will conflict with any other mod that changes the mage statue nif.
  
Texture replacers are fine.
