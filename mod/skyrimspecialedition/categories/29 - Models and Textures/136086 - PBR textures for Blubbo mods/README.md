# PBR textures for Blubbo mods
- Author: jepster
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136086


I understood that most people prefer to have those texture packs combined in a mod, instead of separate ones. So here it is.
  
  
PBR (physically based rendering) approximates how real materials react to light and is used in most modern games. PBR is a part of Community Shaders 1.0 that has several benefits which includes  more realistic lighting than in vanilla and complex material.
  
  
There are two ways (maybe more) to create PBR textures:
  
1. Using raw assets which have all the maps needed for perfect PBR textures.
  
2. Using non-PBR textures and create missing maps using AI models based on what is available.
  
  
I have used the 2nd approach as I do not have the raw assets. To achieve this I have unbaked shadows from diffuse maps, upscaled them using an AI
  
 model, created separate Ambient Occlusion maps, and roughness maps.
  
  
Please consider that some materials benefit more from PBR than others. Especially those textures which make heavy use of additional metallness, roughness maps or other features like glint or multilayer. Trees and shrubs do not make use of this.
  
So this may knock you off your feet like some other PBR mods. But it still looks better in various light conditions than vanilla textures imho.
  
  
**Includes PBR textures for the following tree mods:**
  

  
* Blubbos Gildergreen 2023
Blubbos Sleeping Tree
  
* Blubbos Aspen Replacer Base Object Variation Mix
Blubbos Shrub Replacer
  

**And PBR textures for the following city mods:**
  

  
* Blubbos Whiterun 2022
Blubbos Riverwood 2023
  
* Blubbos Markarth 2022

  
**Installation:**
  
1. Install like any other mod
  
2. Check requirements:
  
 ﻿- Make sure that you have version 1.0 or higher of Community Shaders installed
  
 ﻿- Make sure you have the original Blubbo mods installed
  
3. Run ParallaxGen: This mod does not incldue meshes, so you need to have the original mod installed and run ParallaxGen
