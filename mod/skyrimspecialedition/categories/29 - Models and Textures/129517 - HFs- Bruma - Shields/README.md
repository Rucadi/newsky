# HFs- Bruma - Shields
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129517


My new meshes and HD BC7 textures for 3 shields from "Beyond Skyrim - Bruma" mod. Namely: **Bruma guard shield** (*shield.nif*), **Iron Shield** (*cyrironshield.nif*) and **Leather Shield** (*shieldleather.nif*).
  
  
Only meshes and textures, no ESP or scripts. Safe to install/unistall at any time.
