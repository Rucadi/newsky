# FYX - 3D Stockades
- Author: Yuril
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65104


Simply replace the fake 2D planks on the Stockade with a 3D planks. No ESP
  
Compatible Stockades of Skyrim 3D, BDS, SoS, SMIM...
  
  

![](https://i.imgur.com/mvihAnw.jpg)

  
Enjoy !
  
  
*> For a full experience, it is highly recommended to use "[SMIM](https://www.nexusmods.com/skyrimspecialedition/mods/659)" and "[Stockades of Skyrim 3D](https://www.nexusmods.com/skyrimspecialedition/mods/43227)" if you don't already have them.
>   
> Place this mod after them.*
