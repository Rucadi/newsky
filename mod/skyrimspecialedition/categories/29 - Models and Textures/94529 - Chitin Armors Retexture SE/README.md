This mod retextures the Chitin Heavy Armor, Chitin Armor, Chitin Shield and Morag Tong Armor.
  
  
This mod contains textures only. Meaning it is compatible with every mod that works with vanilla Chitin textures.
  
  
  
  
  
**FOMOD OPTIONS**
  
  
Fomod lets you pick and choose what you want. 
  
  
Textures are 2k except Chest armor textures, which are 4k.
  
  
Half-res version available in Optional files (Chest textures are 2k, everything else is 1k)
