# Vanaheimr - Mines and Caves
- Author: Cl3mus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/137033


[font=Comic Sans MS]A complete retexture of Skyrim’s Mines & Caves.
  
  
  
![](https://iili.io/2Of9Vt4.png)﻿
  
  
  
Available in 4k and 2k.
  
  
  
This mod includes :
  
  
- Meshes with Complex Parallax Enabled for the mines.( With blending effect enabled )
  
- Works with Terrain Helper for Enb.
  
- Support of Icy Cave Remaster for the Icy Caves.
  
- Textures for Smim included.
  
- Patch for Lux ( Splitted Meshes )
  
  
  
![](https://iili.io/2O61jJp.png)﻿﻿
  
  
  
Required
  
  
[Icy Cave Remaster](https://www.nexusmods.com/skyrimspecialedition/mods/90476)﻿HIGHLY RECOMMENDED
  
[Icy Cave Remaster - Ice Material Patch](https://www.nexusmods.com/skyrimspecialedition/mods/139158)[Enhanced Rocks and Mountains - Addon﻿](https://www.nexusmods.com/skyrimspecialedition/mods/142493)﻿
  
  
  
Installation
  
  
  
Download the version you prefer (4k, 2k) and install it using your preferred mod manager. ( Load after ICR )
