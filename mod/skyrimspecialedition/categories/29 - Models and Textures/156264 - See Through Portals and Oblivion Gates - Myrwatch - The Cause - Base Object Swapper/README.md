**Description**

  
* Replaces the incorrect [Dark Fall Cave](https://en.uesp.net/wiki/Skyrim:Darkfall_Cave)﻿ cubemaps used by [The Cause](https://en.uesp.net/wiki/Skyrim:The_Cause)﻿'s Oblivion Gates with accurate images of [Red Scar Cavern](https://en.uesp.net/wiki/Skyrim:Red_Scar_Cavern)﻿ (Mythic Dawn Temple) and [Deadlands](https://en.uesp.net/wiki/Skyrim:Deadlands)﻿.
Replaces the blurry silver cubemaps used by [Myrwatch](https://en.uesp.net/wiki/Skyrim:Myrwatch)﻿ with accurate images of [Myrwatch](https://en.uesp.net/wiki/Skyrim:Myrwatch_(place)#Myrwatch)﻿ and [Myrwatch Gallery](https://en.uesp.net/wiki/Skyrim:Myrwatch_(place)#Myrwatch_Gallery)﻿.
  
* Uses [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿ to swap the models - no cell edits.
ESL.
