# Realistic PBR Footprints
- Author: jepster
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140780


The mod uses latest rendering techniques of Community Shaders and a new approach to create more realistic footprints.
  
Instead of a 'black stamp' on the ground this mod uses displacement and ambient occlusion maps to generate quite realistic footprints.
  
Like in reality the ground does not turn black when you trample over it, but is deformed causing some minor shadows. Just like taking screenshots of mammoths doing footprints may lead to deformation, as I can tell you.
  
  
  
**Features**

  
* Physically Based Rendering textures for footprints allows for better visiuals especially in different lighting conditions
Support for different PBR landscape mods for best matching textures
  
* Different displacement levels for different creatures, e.g. a fox creates less displacement than a human and much less than a mammoth
Support for SandPatch (footprints on sand and mud) and Footprints Ultimate-Fix (footprints for kids and more)
  
* Different wetness levels (implemented via roughness scale) for different ground textures, e.g. footprints on mud look rather wet, on sand a bit, on ash not at all

  
**Installation**
  

  
* Make sure that you have the requirements (listed in requirements section) installed
Install like any other mod using your favorite mod manager
  
* Make sure that you have a PBR Landscape texture pack installed. It probably does not crash if you do not have one, but surely will not look too nice
The default pack comes with matching textures for [Faultier's PBR Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/125308)﻿ because it is the most widely-used PBR Landscape texture pack. If you use a different landscape texture pack check the optional files here
