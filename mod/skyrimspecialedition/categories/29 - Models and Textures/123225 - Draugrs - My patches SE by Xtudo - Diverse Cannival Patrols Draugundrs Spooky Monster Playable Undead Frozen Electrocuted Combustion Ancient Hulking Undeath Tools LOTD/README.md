**Description**
  
**My patches for this amazing mod. (They're fine to install/uninstall at any time)**
  

  
* **4K, 2K, 1K optimized textures**. The 4K file is smaller because it has less optimized textures.
**Animated Clutter version.** Based on the [Animated Clutter LE](https://www.nexusmods.com/skyrim/mods/48287) ﻿mod by [jackmorris21](https://next.nexusmods.com/profile/jackmorris21)﻿. [Animated Clutter SE version here](https://www.nexusmods.com/skyrimspecialedition/mods/14398)﻿.
  
* **Skyrim Immersive Creatures (SIC) patch**. Also includes **Hulking Draugr (Dragonborn) patch.** 130 files edited/created, please let me know if you find some problem in-game.
**Hulking Draugr (Dragonborn) patch.** Model edited to look like a stronger Draugr, following Bethesda intentions, with new reddish textures as well.
  
* **Cannibal Draugr on Solstheim (Bloodmoon Creature Restoration Project) patch**. It also includes **Hulking Draugr (Dragonborn) patch**.
**The Dragon Cult - A Draugr Overhaul patch**. It also includes **Cannibal Draugr on Solstheim** and **Hulking Draugr (Dragonborn)** patch.
  
* **Glowing Eyes patch**. Restores and adjusts the Draugr glowing eyes to the new models. Vanilla, Skyrim Immersive Creatures, Cannibal Draugr and The Dragon Cult versions included. Install it last of all.
**Males only version**. Standalone, it doesn't required the original mod. 4K textures. Also includes **Hulking Draugr (Dragonborn) patch**.
  
* **Armored Draugr Corpses.** New 45 static models used by this mod. With fixed collisions. Also includes **Hulking Draugr (Dragonborn) patch**.
**Draugr Patrols and Cavalry - Mihail Monsters and Animals patch.**18 models created or edited. Now they will use your modded textures, like the recommended [Ancient Nord Armors and Weapons Retexture](https://www.nexusmods.com/skyrimspecialedition/mods/91136)﻿. Horses and Draugulf textures edited to fit the ones from the new Draugrs. **Optional Glowing Eyes patch**available, install it last of all.
  
* **Draugr Horse Mount- Mihail Monsters and Animals patch**. Textures edited to fit the ones from the new Draugrs.**Optional Glowing Eyes patch** available, install it last of all.
**Haugbui patches.** **Default** and **Cannibal** versions available, install only one. The Cannibal version requires one of the Cannibal Draur patches installed first. **Optional Glowing Eyes patch** available, works with both Default and Cannibal, install it last of all.
  
* **Beyond Reach patch**.
**Infant Draugrs- Mihail Monsters and Animals. Default** and **Glowing Eyes** versions available. Install only one.

**Draugundrs- Mihail Monsters and Animals.** It will conflict with **Draugr Patrols and Cavalry** if you use it, so if you want to use this Draugundrs model, install this patch last. **Optional Glowing Eyes patch** available, install it last of all.
  
* **Honored Dead patches.** **Default**and **Cannibal** versions available, install only one. The Cannibal version requires one of the Cannibal Draur patches installed first. Also includes **Hulking Draugr (Dragonborn) patch**.

**DRAGON PRIESTS - Optional Files section**
  

  
* All files related to Dragon Priests have been moved to the Optional Files section.
There are two versions: **Disintegration**and **Compatible** versions. Install only one type of them.
  

  
**Disintegration version**
  
Their body will disintegrate after killed. It is not compatible with mods that adds new Dragon Priest models.
  
Here are the mods compatible with this version:
  

  
* **Vanilla Dragon Priest - My version**. **Default**and **Glowing Eyes (GE)**versions available.
**Creation The Cause - Disintegration version**.
  
* **Xavbio's Dragon Priests Retexture - Unique Dragon Priests patch**. **Vanilla Dragon Priest - My version** already included. Install **Dragon Priest Vanilla GE** after Xavbio's patch if you want them to have glowing eyes.
**Legacy of the Dragonborn (LOTD).** [Zurin Arctus, the Underking](https://legacy-of-the-dragonborn.fandom.com/wiki/Zurin_Arctus,_the_Underking)﻿ replacer. Works for both LOTD v5 and v6.
  
* **Xavbio's Elven Armors and Weapons Retexture - CC The Cause - Disintegration version**.

  
**Compatible version**
  
Their body won't disintegrate after killed. 100% compatible with mods like Diverse Dragon Priests.
  
Here are the mods compatible with this version:
  

  
* **Vanilla Dragon Priest - Compatible version. Default** and **Glowing Eyes (GE)** versions available. 100% compatible with mods like Diverse Dragon Priests.
**Creation The Cause - Compatible version.**
  
* **Diverse Dragon Priests - Pixie's Creature Series**. Fully compatible with my[Diverse Dragon Priests - My optimized textures](https://www.nexusmods.com/skyrimspecialedition/mods/105713)﻿ (recommended).
**Xavbio's Dragon Priests Retexture - Unique Dragon Priests patch**. **Vanilla Dragon Priest - My version** already included. Install **Dragon Priest VComp GE** after Xavbio's patch if you want them to have glowing eyes.
  
* **Legacy of the Dragonborn (LOTD).** [Zurin Arctus, the Underking](https://legacy-of-the-dragonborn.fandom.com/wiki/Zurin_Arctus,_the_Underking)﻿ replacer. Works for both LOTD v5 and v6.
**Xavbio's Elven Armors and Weapons Retexture - CC The Cause - Compatible version.**
