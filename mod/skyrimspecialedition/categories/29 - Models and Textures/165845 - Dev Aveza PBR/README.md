# Dev Aveza PBR
- Author: dirigibalistic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/165845


Fancy PBR textures for the Dev Aveza. Made for LOTD, but should work fine with the standalone version as far as I know. This is my first texture mod, so please bear with me if something looks weird and I'll do my best to fix it.
  
  
**Requirements:**

  
* [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492)﻿
Either [Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802)﻿ or the original [Dev Aveza](https://www.nexusmods.com/skyrimspecialedition/mods/24587)﻿
  
* After installing, run [PGPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/120946?)﻿
