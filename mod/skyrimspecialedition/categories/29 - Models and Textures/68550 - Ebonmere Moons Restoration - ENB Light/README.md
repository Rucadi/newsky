# Ebonmere Moons Restoration - ENB Light
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68550
[color=#ff0000][color=#ffffff]-

Mod alternative which doesn't replace the original textures can be found [here](https://www.nexusmods.com/skyrimspecialedition/mods/68580)﻿-﻿[/color]
  
An artist at Bethesda made some interesting moon drawings/textures with glowmaps and everything. [/color]They were included in a vanilla texture to be used on The Ebonmere moons... but you never get to see them!! o\_O
  
Instead, you get some boring dwemer metal textures with no drawings at all .\_.
  
  
**So, I made some tweaks to vanilla meshes to be able to see those textures in-game and remastered said textures!! :)**
  
  
After that I made another version of the main file with ENB Light.
  
  
And finally... if you use [JS Lockpicking UI SE](https://www.nexusmods.com/skyrimspecialedition/mods/22160)﻿ I included a **fix** for you as an optional patch (last couple of screenshots) ^-^
  
  
**Check the images!! :O**[size=4]**[size=3][size=4][b]INSTALLATION:**
  
[/size][/b][size=4]1) Choose one main file and install it manually or using any mod manager.
  
[/size]
  
[size=4][size=3][size=4]2) IF you chose the main file with ENB Light... Be sure to have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
3) Check and install some optional patches if you want :)
  
  
  
[size=3][size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
