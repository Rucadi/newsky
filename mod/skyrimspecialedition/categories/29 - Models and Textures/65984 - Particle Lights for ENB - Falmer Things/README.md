# Particle Lights for ENB - Falmer Things
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65984
**Falmer Totems:**
  
These things are supposed to emit light from the glowing "eyes" they have. But in vanilla this is faked by adding glow to other parts of the totem... and of course that doesn't look realistic... IT LOOKS STUPID!! \*cough\*
  
So, I had to modify the glow map first to avoid the dumb look and finally I added ENB Light :)
  
  
**Falmer Huts (from Dawnguard):**
  
I added ENB light to the eyes of the chaurus because those are the biggest glowing spots :)
  
  
**Falmer pull-chain (from Dawnguard):**
  
The original glow looked completely sad... so I fixed it... lol
  
  
**Eyes of the Falmer:**
  
These already had some glow but it was very weak. So I added ENB light to them.
  
Since this is not related to the same kind of falmer I added the files as optional downloads (another mod just for these wasn't worth it xD).

**[size=4][b]INSTALLATION:**
  
[/size][/b][size=4]1) Install the main file manually or using any mod manager.
  
  
2) Remember to choose and install one of the optional files if you want the "Eyes of the Falmer" because this isn't included in the main file.
  
[/size]
  
[size=4][size=3][size=4]3) Be sure that you have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
[size=3][size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
