# Skyland Complex Material
- Author: Leo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/124865


**Description**
  
  
Complex Material is a new way to introduce more complex glossy-spec based shading and parallax.
  
The environment map mask texture was repurposed to make use of each channel, to include any necessary maps for glossy spec shading.

**Installation**
  
  
Install the requirements
  
Install this mod and let it overwrite the original mod
  
Run ParallaxGen
  
Install the output
  
  
[size=4]**ENB**
  
  
Open your enbseries.ini
  
EnableComplexMaterial=true
  
EnableDynamicCubemap=true
  
Open your enbcache folder and delete everything[/size]
  
  
**FAQ**
  
Q : Why is it no longer compatible with CS?
  
A : Because of the terrain helper and CS users can use the PBR version instead.
  
Q : What's so great about the terrain helper?
  
A : Because with it, you dont need Terrain Parallax Blending Fix anymore. Less textures loaded, less vram usage.
