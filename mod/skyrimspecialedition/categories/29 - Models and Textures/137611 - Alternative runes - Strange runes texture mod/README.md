# Alternative runes - Strange runes texture mod
- Author: TrolledBartholomew
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/137611


**REQUIREMENTS**
  
[Strange runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456?tab=description) and all its requirements
  
[SkyrimVR Strange Runes - Spellcasting Enhanced VR Edition](https://www.nexusmods.com/skyrimspecialedition/mods/30558) and all its requirements for the vr version
  
  
**INSTALLATION**

  
1. Select only 1 main file
Install via mod manager
  
2. Let it overwrite Strange runes

  
**UPDATING**

  
1. Uninstall alternative runes
Follow installation again
  

[size=3]**[size=3][size=3]ADDITIONAL NOTE - VR**[/size]The vr version should work as textures have been renamed to the matching runes but I'm not sure how to test for skyrim vr so be warned and due to the implementation of the [SkrimVR Strange runes](https://www.nexusmods.com/skyrimspecialedition/mods/30558) the following runes are not in the vr version:
