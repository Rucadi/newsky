# FYX - Windhelm Stable Roof
- Author: Yuril
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/66416


Simply add a roof to Windhelm's stable.
  
This is something I had done for me personally, because I thought a stone roof was weird for a house...
  
I asked if it interested people, there were some. he is here now.
  
  
Based on USSEP mesh
  
  
enjoy !
  
  
> If you don't want the roof, but just want the mesh to fix:
>   
> - Go to [Nifskope](https://github.com/niftools/nifskope/releases)ï»¿
>   
> - Right click on the added roof.
>   
> - "Block" => "Remove Branch"
>   
> - Save
>   
> You can also use [Outfit Studio](https://www.nexusmods.com/skyrimspecialedition/mods/201) to do this.
