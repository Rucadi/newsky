**CARROTS****Author****iimlenny - My version by Xtudo**

**Description**
  
**An ultra realistic HD carrot replacer.**
  

  
* 2K and 1K optimized textures versions available.
Improved textures, and new normal map, for a more realistic feeling.
  
* Adjusted size to Vanilla standards.
Custom texture filenames, to avoid conflicts with other mods that use the Vanilla carrot texture.
  
* Optional Eating Animations and Sound patch.
No ESP.
  

  
**Optional files**

  
* [Eating Animations and Sounds](https://www.nexusmods.com/skyrimspecialedition/mods/42602)﻿﻿ patch.﻿ Only meshes. I recommend to use it together with my[Eating animations - My HD version](https://www.nexusmods.com/skyrimspecialedition/mods/44010)﻿ mod﻿.
