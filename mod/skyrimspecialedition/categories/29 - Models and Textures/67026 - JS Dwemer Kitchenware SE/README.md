**What this mod includes:**
  
  
• New Meshes & Textures for Dwemer Kitchenware (Pan, Dish, Bowl, Knife, Fork, Spoon & Cup Variants)
  
• Updated Collisions that so objects can be placed inside Pans, Bowls & Dishes
  
  

![](https://i.imgur.com/ZM71deY.png)

  
**Installation**
  
  
• Mod Manager / Manual Extraction to the Skyrim Data folder.
  
 • Overwrite any existing files. No plugin requirement.
  
  

![](https://i.imgur.com/ZM71deY.png)
