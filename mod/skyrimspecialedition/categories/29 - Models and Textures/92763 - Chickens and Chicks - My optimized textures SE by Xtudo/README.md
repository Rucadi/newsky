**CHICKEN AND CHICKS**
  
**Optimized textures by Xtudo**

**Description**
  
**My optimized textures for this lovely mod.**
  

  
* Three options available, 4K (optimized original size), 2K and 1K.
The 2K file is larger than the 4K because it includes more optimized textures.
  
* No ESP,  so it's fine to install or uninstall it at any time.

  
**Optional files**
  

  
* Optimized [Chicks - Mihail Animal Fluffy](https://www.nexusmods.com/skyrimspecialedition/mods/84181)﻿﻿﻿ mod by [6Four9](https://www.nexusmods.com/skyrimspecialedition/users/32419020). 2K and 1K versions included. Only textures. Overwrite if asked.

  
**I made it for my game and now I'm sharing it, I hope you like it.
