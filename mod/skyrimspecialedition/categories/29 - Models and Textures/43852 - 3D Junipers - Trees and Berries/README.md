**Description**

  
* Higher polygon Juniper trees
Round Juniper berries
  
* 3D branches
Includes Hearthfire planter trees
  
* Fixed broken UV maps on Bethesda's tree trunk
Added brand new wind animations
  
* Compatible with texture replacers (if you let them overwrite)

  
**Installation**
  

  
* Install and uninstall whenever
