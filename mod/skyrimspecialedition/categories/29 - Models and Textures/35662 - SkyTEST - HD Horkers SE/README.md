Description
  
===========
  
An improved version of the horkers added by the SkyTEST mod. See comparison screenshots.
  
The new textures are 4k, 2k for the cub.
  
  
There are two versions included (use only one):
  
- HD textures
  
- HD textures and better meshes (requires ESP file)
  
  
**I did it for my game and now I'm sharing it, I hope you like it.**
