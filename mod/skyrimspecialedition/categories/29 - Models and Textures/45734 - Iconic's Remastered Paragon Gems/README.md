# Iconic's Remastered Paragon Gems
- Author: Iconic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/45734


**Overview**
  
  
  
  
  
  
**Another one where I was surprised nobody has retextured these unique items.**
  
**Created and assigned the meshes new and unique textures for more bling.**
  
**Multiple color choices for the metal casing, Gold, Silver, Bronze, and Black Iron. Customize your loot!**
  
**Saved using BC7 compression for maximum quality.**
  
**Available in 4K and 2K.**
  
  
  
**Be sure to head over to the**[Particle Lights For ENB SE - Paragon Gems](https://www.nexusmods.com/skyrimspecialedition/mods/45761)**﻿ page to get a patch for ENB Light!!**
  
  
  
  
Credits to [cALAMIN](https://www.nexusmods.com/skyrim/users/49221608)﻿ for the cubemaps from [HD CubeMap Collection](https://www.nexusmods.com/skyrim/mods/95466)﻿, cause I don't know crap about cubemaps.
