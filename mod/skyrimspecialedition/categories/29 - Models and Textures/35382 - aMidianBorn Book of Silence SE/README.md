# aMidianBorn Book of Silence SE
- Author: CaBaL - the AMB team
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/35382


![](https://staticdelivery.nexusmods.com/mods/110/images/102566/102566-1588186933-947735585.png)
  
  
This project aims to provide high quality replacements for Skyrim textures.  This huge work that offers high resolution textures for many armors and weapons, as well as textures for some of the creatures you'll find in Skyrim.  This is a port of the original [aMidianBorn Book of Silence](https://www.nexusmods.com/skyrim/mods/24909).
  
  
Textures have been crafted from scratch, and the use of original game assets is minimal, limited to normal maps for some equipment.
  
Textures have the same resolution as the high-res DLC pack.

**COMPLETE:** The COMPLETE version of this mod includes a single plugin (ESL flagged) and a BSA archive that contains all of the standard vanilla texture replacements from the Armor, Creatures, Dragonborn, Unique Items, and Weapons files. If you want a quick and easy installation that includes the complete set of textures from Book of Silence, then this is the file for you. If you download the COMPLETE option, you do not need to download any of the other options. However, the COMPLETE version does not include an installer and the texture selection cannot be customized. The only options included in the COMPLETE version are the optional plugins to add the different Glass Variants, Ebony Variants, or the unique Skyforge Weapons. These optional plugins need to be copied into your mod or data folder and enabled in addition to the aMidianBorn\_Book of Silence.esp
  
  
**CCOR CONTENT ADDON:** For those using CCOR's [aMidianBorn Content Addon](https://www.nexusmods.com/skyrimspecialedition/mods/35390), there is also a version specifically designed for use with the Content Addon. This version is similar to the COMPLETE version, and includes all of the Book of Silence textures packaged into a single BSA, along with the additional textures needed for the Content Addon. If you are using the Content Addon, this is the only file you need to download.
  
  
**INDIVIDUAL OPTIONS:** For better compatibility and customization, this work can also be downloaded as individual parts.  Each part features an installer to allow the user to decide and customize which content they wish to install.
