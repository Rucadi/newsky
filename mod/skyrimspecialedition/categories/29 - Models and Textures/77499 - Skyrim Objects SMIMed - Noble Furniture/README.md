**Description**
  

  
* Makes handles on noble furniture nicer
Some minor UV tweaks
  
* Some minor geometry fixes
Can be used with your own noble furniture textures
  
* Doesn't make noble furniture set perfect. Mod is open permissions, so anyone can build on it (provided they follow [CC BY-SA](https://creativecommons.org/licenses/by-sa/4.0/)﻿ license)
Very happy handles :D

**FAQ**
  
> ﻿Q: Does this require **noble** skyrim?
>   
> A: No. "Noble" is a common word in English and, in this case, refers to the noble furniture set found in the base game. Just be because another mod uses this common word in its name does not make it related.
