# Better Dyndolod Red Mountain Plume
- Author: NeverLost
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/43863


This mod tries to improve the 3D Red Mountain Plume mesh provided by DyndoLod, making it look less like a tornado and more like a proper eruption cloud.
  
Included are:
  
  
- New plume textures
  
- New plume model
  
  
**Requirements**:
  
[DyndoLod](https://www.nexusmods.com/skyrimspecialedition/mods/32382)
  
  
**Installation:**
  
Plug 'N Run DyndoLod
