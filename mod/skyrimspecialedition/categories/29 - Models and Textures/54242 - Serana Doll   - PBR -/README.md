【Overview】
  
Add a "Soft Serana Doll" for display use.
  
It swings a bit as it uses bones.
  
  
Version 1.2.0　　 PBR version added.
  
Please choose from the following two types.
  
・Default version
  
・PBR version (CommunityShaders required)
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/54242/54242-1737121372-1978341818.png)
  
  
If the [Description Framework](https://www.nexusmods.com/skyrimspecialedition/mods/105799) is installed, add a description to the doll.
  
  
【How to use】
  
Can be made from leather on a tanning rack.
  
  
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/54242/54242-1701606467-218146292.png)

【概要】
  
ディスプレイ用途のセラーナ人形 "Soft Serana Doll"（くたくたセラーナちゃん） を追加します。
  
ボーンが入っており、多少揺れます。　握ると"くたくた"します
  
  
バージョン1.2.0　新しくPBRバージョンを追加しました。
  
以下の2種類から選んでください。
  
・デフォルトバージョン
  
・PBRバージョン　（CommunityShadersが必須）
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/54242/54242-1737121372-1978341818.png)
  
[Description Framework](https://www.nexusmods.com/skyrimspecialedition/mods/105799)がインストールされている場合、人形に説明文を追加します。
  
  
【使い方】
  
皮なめしの棚で革から作成します。
  
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/54242/54242-1701606486-1595538600.png)
