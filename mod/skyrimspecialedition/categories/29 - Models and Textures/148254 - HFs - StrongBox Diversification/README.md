# HFs - StrongBox Diversification
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148254


My new models forStrongBox(*strongbox01.nif*)*.* And, of course, my new HD BC7 hand-made textures for this models (**not** *vanilla **retexture***). 
  
There are 2 different models available, iron and wooden, which will randomly replace the original vanilla model in the game using [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) or [Model Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/136461) (for your choice).
  
  
![](https://i.postimg.cc/RVTRjJLf/0-Trending-Mods.jpg)
  
  

[font=Georgia]**Compatability:**[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the *[i]strongbox01.nif*[/i].
  
  
**Not compatible**with mods that make any changes to the *strongbox01.nif*, included in this mod, even if it is a simple name or folder structure change. Not compatible with any other mesh replacers and/or retexture for vanilla **StrongBox**.
  
  
[font=Georgia]**Video by [Heavy Burns](https://www.youtube.com/@HeavyBurns)﻿:**[/font]  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
