[font=Courier New]*For all the people and fans who asked for a male version of my mod, well here it is!! converted from "Feminine Khajiit Textures" just to complete the Khajiit race with both genders, also comes with two variations (Grey Cat and Leopard) with 3 options each:
  
- Abs: Humanoid blood runs in your veins, your khajiit will be able to do more than 1000 crunches!
  
- Smooth Belly: Your humanoid genes are strong enough and nothing can stop that smooth belly you got!
  
- Furry: purrrrrr!! :3

  

![](https://staticdelivery.nexusmods.com/mods/110/images/75078/75078-1632019660-935493795.png)
  
**EXTRA Package:** ﻿It's the additional content for more chest variations including:
  
**Diffuse Maps:**
  
1. Milky Nipples
  
2. No Nipples and Belly Button
  
**Normal Maps:**
  
1. Furry Abs
  
2. No Belly Button
  
**Specular Maps:**
  
1. Abs and Nipples
  
2. Nipples Only
  
  
These options are mixed with the previous chest variations (Abs, Furry, etc.)
  
You can choose whatever chest shape you want with any nipple shape you like whether it's diffuse maps, normal maps or specular maps thus creating hundreds of possible outcomes.
  
Example: you can mix diffuse map (Smooth Belly - Milky Nipples) with normal map (Furry - Abs) with specular map (Abs and Nipples).
  
The EXTRA package is optional so you still need the main file to use it, I have excluded the options that already exist in the main files.*[/font]
  
![](https://staticdelivery.nexusmods.com/mods/110/images/75078-3-1477914290.png)
  
*![](https://staticdelivery.nexusmods.com/mods/110/images/75078-2-1477914290.png)*[font=Courier New]*- Install normally via Vortex, NMM or Mod Manager.
  
- Vanilla textures option doesn't require any other mod, it's for base-game Khajiit male.
  
- SOS Lite textures option requires SOS Lite installed first then overwrite with this mod.
  
- SOS Full textures option requires SOS Full (Smurf Average).
  
- Better Males textures option requires Better Males body (Cut, Uncut and Erect).
  
- SAM Light textures option requires SAM Light body (Cut, Uncut and Underwear).
  
Use the patch for (Uncut version) and load it after the main file.
  
- If installing manually, you need to extract to Skyrim inside "Data" folder.*[/font]*![](https://staticdelivery.nexusmods.com/mods/110/images/75078-1-1477914290.png)
