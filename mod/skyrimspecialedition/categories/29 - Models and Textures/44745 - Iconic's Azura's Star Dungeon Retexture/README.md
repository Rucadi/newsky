# Iconic's Azura's Star Dungeon Retexture
- Author: Iconic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/44745


**Overview**
  
  
  
My personal take on the dungeon inside Azura's Star. Azura is the Deadric Prince of Dusk and Dawn, so I personally thought the inside of the Star was overly bright, and completely the wrong color. I tried to give it more of a spooky and twilight feel, especially because of the intruding necromancer busy corrupting it. The eldritch effect is somewhat diminished if you're not running [Window Shadows](https://www.nexusmods.com/skyrimspecialedition/mods/37831)﻿ with the enhancer, so I highly recommend it. Includes an optional plugin that provides the same lighting tweak as Window Shadows, for people who aren't running it. Compatible with ELFX and flagged as ESL.
  
  
Pair this with [Iconic's Blackreach Crystals Retexture](https://www.nexusmods.com/skyrimspecialedition/mods/45917)﻿ so you don't get weird dark Blackreach Crystals.
