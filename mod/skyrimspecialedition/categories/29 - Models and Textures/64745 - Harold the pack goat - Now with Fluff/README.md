# Harold the pack goat - Now with Fluff
- Author: Macxhiin
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64745


A patch to the original Harold the pack goat that gives Harold a very fluffy look using Fluffworks (Fluffy Animals).
  
  
Requirements:
  

  
* [Harold the pack goat](https://www.nexusmods.com/skyrimspecialedition/mods/5712)﻿
[Fluffworks (Fluffy Animals)](https://www.nexusmods.com/skyrimspecialedition/mods/56361)﻿
  

  
Installation:
  

  
1. install "Fluffworks (Fluffy Animals)"
install "Harold the pack goat"
  
2. install "Harold the pack goat - Now with Fluff" (let this mod overwrite the Harold nif)
