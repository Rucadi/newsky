[font=Verdana]Description[/font]
  
  
This is a non-beast lore-friendly female texture replacement with optional pubic hair, dirt, scars and muscle smoothness in different combinations.
  
Mod provides choice of nude, dressed and topless UNP bodies.
  
There are also different versions of faces: three age variants, two skin smoothness variants.
  
Additionally there are new complexion textures; old females, vampires and afflicted ones covered as well.
  
Textures are gameplay-seamless, but screenarchers might find seams in close-ups under certain light conditions.
  
  
[font=Verdana]About[/font]
  
  
Just as [Tempered Skins for Males](http://www.nexusmods.com/skyrimspecialedition/mods/7902/?) this mod started as an attempt to fix neck seam both in SSE versions of different texture replacements and in Oldrim ones. After finishing *TS for Males* I wanted to finally do something with female characters because in spite of way bigger modders attention to them I couldn't find a retexture I could be completely pleased with. Besides just like for *TS for Males* I wanted to offer a choice to select from nude, dressed and, more important, topless bodies.
  
  
[font=Verdana]Features[/font]
  
  
**Bodies:** UNP pinup body, two bikini versions, second bikini has topless variant.
  
**Skin**: naturally looking body skin with hand painted veins.
  
**Seams**: corrected for every kind of texture (neck, legs, hands for diffuse, normal and specular).
  
**Pubic hair options**: shaved, hairy, bushy.
  
**Muscle options**: 
  
• soft - body is tight, but muscles are not defined, 
  
• fit - more sporty type, muscles become more visible, 
  
• muscular - muscles much more visible, veins stand out too.
  
**Dirt options**: with and without.
  
**Faces**: younger, elder, hard life.
  
**Faces normal maps**: smoother, rougher.
  
• smoother - less wrinkles, soften traits, 
  
• rougher - vanilla-like.
  
**Complexions**: new and improved, parasite tint removed.
  
**Additional texture rexplacement:** for afflicted ones, vampires, old females. 
  
**Technical info**: diffuse maps - body and head 4K, body normal map 4K, head normal maps 2K. Textures saved in BC7 format which allows to have almost uncompressed quality (you can't find the difference without picture editor usage) with DXT5 file size. Specular maps saved in BC4 (single channel usage, grayscale mode).
  
  
[font=Verdana]Installation[/font]
  
  
FOMOD
  
1. Download.
  
2. Install via manager.
  
  
Manual
  
0. Uncheck any packed appearance mod (like WICO) and make a backup of the following folders (if you have loose files there):
  
• in '*textures/actors/character/*' all folders containing word "female" and '***character assets***' folder,
  
• in '*meshes/actors/character/*' - '***character assets'*** folder.
  
1. Download. Extract it in a temporary directory.
  
2. You will find several folders inside named from 0 to F. Choose one option from each directory, description and example pictures granted. 0 to E - choose one option. F - choose none or several.
  
3. Copy those you liked to 'Skyrim/Data' folder.
  
  
*For example, my personal choice is 0-A1-B4-C2-D2-E2-F1-F2.*

[font=Verdana]FAQ[/font]
  
  
*Q: CBBE please?*A: Personally - and obviously - I prefer UNP, but lots of people keep asking for it. So when I'm ready and have free time I'll do it. No obligations though.
  
  
*Q: I'd like to adjust textures for a follower, but can't open it.  What's wrong?*
  
A: I used BC7 format - it has almost uncompressed quality with DXT5 size, but this format is not widely supported by graphics editing software.
  
So:
  
• If you have Photoshop - download and install [Intel Texture Works plugin](https://software.intel.com/en-us/articles/intel-texture-works-plugin).
  
• If you don't have Photoshop, you can use AMD Compressonator to convert BC7 to another format:
  
  ▸ download it [here](https://github.com/GPUOpen-Tools/Compressonator/releases)- select one with GUI;
  
  ▸ do [this](https://compressonator.readthedocs.io/en/latest/gui_tool/user_guide/TextureCompression.html#converting-image-formats) - select uncompressed destination format like ARGB\_888 or whatever you like and work with it as usual. 
  
*Q: Head colour is different from body one!*
  
A: Make sure you've installed everything. Backup and delete at least '*Data/textures/actors/character/male'* folder and perform a fresh install. 
  
  
*Q: I've got neck/hands/legs seam!*
  
A: Same as above. But keep in mind that slight seams might be visible under certain light conditions. 
  
  
*Q: Could you add ...?*
  
A: Give a try and ask. Though no promises.
  
  
*Q: What ENB are you using for screenshots?*
  
A: See Mods Recommendations section.
  
  
*Q: Will there be a more perfomance friendly version or conversely, 4K or even 8K everything?*
  
A: No. Mod is designed to deliver both good quality and performance during gameplay at the same time. 
  
BC7 texture compression format allows to reduce VRAM usage without hurting quality so that there is no need in lighter version. Current relatively modern PC specs will handle this mod wihout any problems. 
  
As for bigger resolution, this mod uses 4K only where it is really necessary - body and hands diffuse maps. Further size increase of any textures will notice only screenarchers on close-up shots.
  
  
[font=Verdana]Compatibility[/font]
  
  
• Obviously this mod is not compatible with other mods that change female character textures and body meshes. 
  
• It is compatible with such character enhancements as scars, tintmasks (including warpaints ***except*** lips, read further), brows, beards, eyes, hair. Keep in mind that the mod offers scars and tintmasks in the package.
  
• It is compatible with mods that change NPCs appearance.
  
• If you do want to combine this mod with other texture mods that change characters appearance, please remember that you will most likely get a seam. 
  
  
This mod does have overlapping files with *TS for Males*:
  
**blankdetailmap.dds** - a blank complexion texture with fixed tint. This is because females use this files from males folder. Identical to TS for Males file. Ok to overwrite, ok not to overwrite, just make sure you are using one from either Tempered Skins.
  
  
**Do not overwrite these files with any other mods!**
  
*textures/actors/character/male/blankdetail.dds* - **very important**, because it's a corrected neutral gray blank complexion. Original Skyrim file is tinted because of its incorrectly chosen texture format and color shift due to it. Delete or swap this file for something else - and head colour will go wild.
  
*textures/actors/character/male/femaleheaddetail\*.dds -*not recommended, though possible. Besides remaking I also removed unwanted tint.
  
*textures/actors/character/character assets/tintmasks/femalehead\*\_lips.dds*-  these files were corrected specifically for this mod head textures. They show where lips should be coloured.
