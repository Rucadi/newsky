This mod retextures the steel armor, shield and weapons, Steel Plate armor, Blades armor and weapons, Nordic Carved armor and weapons, Tsun's armor, CC Steel Soldier, CC Fists of Randalgulf, CC Steel Staff, CC Steel Horse Armor, CC Nordic Jewelry and CC Brawler's Steel fists. Consistency patches for many mods﻿
  
  
  
The AMB-Style steel plate, AMB-Style Sentinel gilded steel plate, Keepers of hattu, Nord carved armor, blades armor and explorer's armors are based on [Cabal120's aMidianBorn Book of Silence](https://www.nexusmods.com/skyrimspecialedition/mods/35382)﻿. I used their textures as a base to create mine. All other textures are made using vanilla skyrim textures as a base (I used the normal maps as a starting point in substance painter, and then did my own texture work from there)
  
  
Fomod lets you pick and choose what you want. 
  
  
Textures are 2k except Chest armor textures, which are 4k.
  
  
Half-res version available in Optional files (2k chests, shields and weapons, 1k everything else)
  
Low-res version is also available in optional files (1k chest, shields and weapons, 512 everything else)

**CONSISTENCY PATCHES**
  
  
Consistency patches are optional retextures of modded armors, clothes and/or weapons I like. The goal of these is to make these items match my retextures. I do not ever claim that my versions are objectively better, all authors who's work I cover are extremely talented people! The original mods are always a hard requirement.
  
  

**NORDWAR UA / SENTINEL**
  
By [NordwarUA](https://next.nexusmods.com/profile/NordwarUA?gameId=1704)﻿
  
  
As of this moment, these textures are not yet included into sentinel. If you use Sentinel and wish to use my versions of these textures, install these options and load my mod after Sentinel. If/when these textures are integrated into a future update of Sentinel, I will update this message.
  
  
I have tested these with Realistic Armors and Sentinel - an equipment overhaul. Should work with any other mods using these assets, as long as the author did not change the file paths
