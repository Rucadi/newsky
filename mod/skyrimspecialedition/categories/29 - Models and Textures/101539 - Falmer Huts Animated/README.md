# Falmer Huts Animated
- Author: StoneSpiralGaming
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/101539


This Base Object Swapper made mod replaces the Large Falmer Tent, Hut, Chief Tent and Totem Meshes with animated ones (using a vanilla meshes that I've split and vertex painted) making it look like the loose cloth is gently rustling in the breeze.
