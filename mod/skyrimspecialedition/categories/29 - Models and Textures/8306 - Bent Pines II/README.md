# Bent Pines II
- Author: vurt
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/8306


***Bent Pines II*** places some very large bent pines on the tundra to make it look somewhat more interesting and epic.
  
  
**LOD's has to be generated using DynDoLOD!**
  
  
Changes from version 1:
  
  
Branch animations (thanks to MrNeverLost for doing these!)
  
Better leaf animations
  
Better normal maps
  
2 more placements on the map
  
New pine needle texture
  
Some tweaks and Improvements to the mesh and textures
  
  
**FAQ**
  
*Want less trees, better performance, or is a tree in the wrong location - interfering with a mod etc.?* 
  
  
Then go in to the console, click the tree, enter ***disable**.* This will remove the tree from that location. Save your game afterwards.This can be used on everything except for grasses/ground flora. This is also useful when you want to take that epic screenshot, but something is ruining the view for you ;)
