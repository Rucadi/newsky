# Tomato's Riften and Ratway
- Author: Tomatokillz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/157340


[font=Tahoma]**Tomato's Riften and Ratway**
  
[/font][font=Tahoma]PBR & Complex Parallax
  
[/font]A complete overhaul of Riften and the Ratway.
  
 I have also included re-textures for mesh mods like SMIM and Major Cities Mesh Overhaul.
  
**Vanilla and non-parallax users can simple install the mod and enjoy, no need to do any other steps.**
  
**Installation for PBR:**
  
[size=4]**1.** Install this mod after any PBR overhauls you use.
  
**2.** Load [GDOS](https://www.nexusmods.com/skyrimspecialedition/mods/150727) after if you rather It's doors.
  
**3.**Load [Faultier's Windows](https://www.nexusmods.com/skyrimspecialedition/mods/125308) after mine if you want cool windows.
  
**4.** Run ParallaxGen (PG) with TruePBR enabled.
  
**5**. Install the "edge fixer" optional file if you get ugly moss on walls/bad roof trims.
  
**Info:**
  
PBR only works with Community Shaders and you MUST run PG Patcher for it to work.
  
*No it doesnt work for ENB or vanilla use the **Complex Parallax version (its also hosted here!)***
  
-
  
[size=4]**Installation for COMPLEX PARALLAX:**
  
[/size]**1.** Install this mod after whatever baseline textures you use.
  
**2.** Load things like [GDOS](https://www.nexusmods.com/skyrimspecialedition/mods/32376) after if you rather It's doors.
  
**3.** Run ParallaxGen (PG Patcher) with Complex Material or Parallax enabled.
  
**4.** Enjoy.
  
**Info:**[size=3]
  
Both \_p for standard parallax and \_m for CM is provided.
