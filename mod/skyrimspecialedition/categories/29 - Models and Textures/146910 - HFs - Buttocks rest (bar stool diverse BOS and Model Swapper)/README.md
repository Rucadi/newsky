# HFs - Buttocks rest (bar stool diverse BOS and Model Swapper)
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/146910


My new models for**Wooden Bar Stool**(*woodenbarstool.nif,**woodenbarstool01.nif, woodenbarstool01static.nif*)*.* And, of course, my new HD hand-made textures for this models (**not** *vanilla **retexture***). 
  
There are 3 different models available, which will randomly replace the original vanilla model in the game using [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) or [Model Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/136461).
  
  

[font=Georgia]**Compatability:**[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the *[i]woodenbarstool.nif,**woodenbarstool01.nif, woodenbarstool01static.nif*[/i].
  
  
**Not compatible** with mods that make any changes to the the *woodenbarstool.nif*, *woodenbarstool01.nif*, *woodenbarstool01static.nif*, included in this mod, even if it is a simple name or folder structure change. Not compatible with any other meshes replacers and/or retextures for **Wooden Bar Stool**.
  
  
![](https://i.postimg.cc/nhCpXwq7/0-Trending-Mods.jpg)
  
  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
