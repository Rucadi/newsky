# Skyshards - ENB Light Addon
- Author: Deadmano - monkeyangie
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68443


![](https://pixelsafari.neocities.org/dividers/vine.gif)
  
**Description**
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)

  
Simple mod that adds ENB light to the [Skyshards](https://www.nexusmods.com/skyrimspecialedition/mods/60748), the light turns off when the Skyshard is absorbed
  

**Installation**
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)

  
Drop everything on your Data folder or use a Mod Manager, of preference use MO2
