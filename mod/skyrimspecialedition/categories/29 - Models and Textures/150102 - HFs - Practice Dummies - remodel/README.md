# HFs - Practice Dummies - remodel
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150102


My new models forvanilla Practice Dummies (practicedummy01, practicedummy02 and practicedummy03)*.* And of course my new HD BC7 hand-made textures for this models (**not** *vanilla **retexture***). 
  
  

[font=Georgia]**Compatability:**[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the practicedummy01.nif,*practicedummy02.nif*and*practicedummy03.nif*.
  
  
**Not compatible** with mods that make any changes to the ***practicedummy01.nif*,*****practicedummy02.nif***and***practicedummy03.nif***, included in this mod, even if it is a simple name or folder structure change. Not compatible with any other mesh replacers and/or retexture for vanilla Practice Dummies.
  
  
Meshes and textures only, no ESP, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
