# FrankBlack's Honey in a Jar
- Author: FrankBlack
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/104496


Hi everyone!
  
  
This is a mesh and texture replacer for Skyrim's honey pot. It can either be used as a complete replacer, or with  [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) by [powerofthree](https://www.nexusmods.com/skyrimspecialedition/users/2148728) to replace some honey pots (with a 50 percent chance of replacement).
  
  
Because it's a mesh and texture replacer, you can install or uninstall it anytime without any ill effects on your game. 
  
  
For a while, I've wanted a honey jar in the game that more resembled honey in The Witcher series of games, so here's my take on that. I hope you dig it :)
  
  
The jar's glass uses textures from [TMD Jars of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/84235) by [themilkdrinker](https://www.nexusmods.com/skyrimspecialedition/users/5487668). Thanks so much for making your work available to other modders!
  
The cloth on top of the jar uses a texture from [Folded Rag Diversification](https://www.nexusmods.com/skyrimspecialedition/mods/88484) by  [PRieST47](https://www.nexusmods.com/skyrimspecialedition/users/927130). Again, many thanks for allowing your work to be included in other mods!
  
I highly recommend both of these mods. Please give the authors some kudos for their excellent work!
  
  
Installation:
  
  
For the replacer version: Download, install, and overwrite if prompted. You'll be good to go.
  
For the BOS version: The same as above, but make sure you have the latest version of Base Object Swapper installed first.
