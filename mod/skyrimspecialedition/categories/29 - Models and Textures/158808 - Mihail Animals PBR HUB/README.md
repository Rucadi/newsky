# Mihail Animals PBR HUB
- Author: TyLuis
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/158808


Conversion of several Mihail's animal mods to PBR, let's make all of Skyrim in PBR.
  
  
**Installation**
  
  

  
1. Install the original matching mod﻿﻿﻿﻿
Install this mod
  
2. Install [PGPatcher (aka ParallaxGen)](https://www.nexusmods.com/skyrimspecialedition/mods/120946)﻿ and run, install output as a mod and activate it
Profit﻿
  

  
**Suggestions**
