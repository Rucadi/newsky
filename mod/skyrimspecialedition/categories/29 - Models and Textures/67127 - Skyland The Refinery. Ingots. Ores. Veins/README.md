# Skyland The Refinery. Ingots. Ores. Veins
- Author: Skyking2020
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67127


![](https://staticdelivery.nexusmods.com/mods/1704/images/67127/67127-1650987462-1411091056.jpeg)
  
  

The Refinery - offers realistic ingots, ores, & veins featuring brand new custom meshes for ingots paired with realistic 2k textures. This mod covers all the the base game materials.
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/67127/67127-1650988055-1385293965.jpeg)*[size=3]All* *the ores and ingots included*
  
[/size]
  
**Features:**
  
Custom meshes for ingots
  
2k textures for ingots, ore, and veins
  
Covers all base game materials
  
  
**Materials Included:**
  
Corundum
  
Dwarven
  
Ebony
  
Gold
  
Iron
  
Moonstone
  
Obsidian
  
Orichalcum
  
Quicksilver
  
Silver
  
Steel
  
Heartstone Ores and Veins
  
Stalhimore Ores and Veins
  
Clay Bricks and Ore Stone from Hearthfire
  
  
**INCLUDED IN SKYLAND AIO**
  
  
**V1.5 Update**
  
Added Heartstone ores and veins. Added Stalhrimore ores and veins. Added Clay Bricks. Added Ore Stone.
  
  
**Compatibility:**Not compatible with other mods that adds meshes for ingots or retextures veins, ores, and ingot. Place this mod below any mod that makes those changes
