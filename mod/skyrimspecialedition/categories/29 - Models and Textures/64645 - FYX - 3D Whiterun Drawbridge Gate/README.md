# FYX - 3D Whiterun Drawbridge Gate
- Author: Yuril
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64645


Simply replaces the 2D planks of the Drawbridge Gate with a 3D planks. SMIM and Parallax version. No ESP
