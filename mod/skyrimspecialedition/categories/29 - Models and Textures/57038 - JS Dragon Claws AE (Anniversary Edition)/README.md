**What this mod includes:** 
  
  
• New Meshes & Textures for all Vanilla Dragon Claws (with Multi-layer Parallax for Gem Claws)
  
• New Meshes & Textures for third-party claws added in Legacy of the Dragonborn, Wyrmstooth, Konahrik's Accoutrements, Helgen Reborn & Skyrim Sewers
  
• Compatibility patches (ESL Flagged) for Legacy of the Dragonborn, Wyrmstooth, Konahrik's Accoutrements, Helgen Reborn & Skyrim Sewers
  
  
**Note:** Not compatible with any other mod that changes Dragon Claws (including JS Dragon Claws SE)

**[b][color=#a4c2f4]FAQ:**
  
[/color][/b]
  
1) I have [JS Dragon Claws SE](https://www.nexusmods.com/skyrimspecialedition/mods/1394) installed, what do I do?
  
• Uninstall it prior to installing this mod or let the FOMOD overwrite any files when prompted.
  
  
2) I have patches from [JS Dragon Claws SE - Patches](https://www.nexusmods.com/skyrimspecialedition/mods/23833) installed, what do I do?
  
• These will need to be removed due to incompatibility.
  
  
3) I have a patch installed from [Legacy of the Dragonborn Patches](https://www.nexusmods.com/skyrimspecialedition/mods/30980)﻿, what do I do?
  
• Overwrite it with the patch from this mod. FormID's are unchanged, so there should be zero issues.
  
  
**Note:** If you uninstall this mod however, you will need to re-download the original official patch to restore nif paths & display placements
  
  
4) I have the Jade Claw patch from [JS Dragon Claws Legacy of the Dragonborn patch](https://www.nexusmods.com/skyrimspecialedition/mods/46447), what do I do?
  
• This is no longer needed.
