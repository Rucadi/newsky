# HFs - Flagons (BOS and Model Swapper)
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/145495


My new models for**Flagon***(flagon01.nif).* And, of course, my new HD hand-made textures for this models (**not** *vanilla **retexture***). 
  
There are 4 different models available, which will randomly replace the original vanilla model in the game using [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) or [Model Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/136461).

[font=Georgia]**Compatability:**[/font]

  
Should be **fully compatible** with anything except mods from other author's that modify the *flagon01.nif*.
  
  
**Not compatible** with mods that make any changes to the the flagon01.nif, included in this mod, even if it is a simple name or folder structure change. Not compatible with any other meshes replacers and/or retextures for Flagon (***flagon01.nif***).
  
  
ESP flagged ESL, no scripts. So it absolutely safe to install/uninstall at mid-game (at any time).
