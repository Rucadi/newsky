# HFs - Bruma braziers
- Author: Halffaces
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/133867


Bruma's ugly braziers mesh replacers. 
  
A very simple mod that replaces the ugly vanilla meshes ***cyrbrumabrazier01.nif*** (logs only) and ***impbrazier01.nif*** from the Bruma mod. I originally made this mod for myself, but then decided to share it with everyone. Maybe some of you will like it.
  
  
Only meshes & textures, no ESP and/or scripts. So it safe to install/uninstall at mid-game (at any time).
