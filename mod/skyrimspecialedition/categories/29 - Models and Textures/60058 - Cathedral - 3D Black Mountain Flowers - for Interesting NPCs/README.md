[left]Description:
  
 ﻿This mod simply replaces Black Mountain Flowers added by [Interesting NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/29194)﻿ which uses edited vanilla textures with an edited version of [Cathedral - 3D Mountain Flowers](https://www.nexusmods.com/skyrimspecialedition/mods/41312)﻿.
  
  
Requirement:
  
 [﻿Interesting NPCs SE (3DNPC)](https://www.nexusmods.com/skyrimspecialedition/mods/29194) [﻿Cathedral - 3D Mountain Flowers](https://www.nexusmods.com/skyrimspecialedition/mods/41312)Optional File:
  
 - ﻿A version with tweaked mesh similar to [Fixed Mesh Lighting](https://www.nexusmods.com/skyrimspecialedition/mods/53653) is available. Use only one file.
  
 ﻿- An optional plugin (esl-flagged) to revert filepath for users of [Snazzy Items for Interesting NPCs (3DNPC)](https://www.nexusmods.com/skyrimspecialedition/mods/58811)﻿ or [Legacy of the Dragonborn - Interesting NPCs Patch](https://www.nexusmods.com/skyrimspecialedition/mods/39834)﻿.[/left]
