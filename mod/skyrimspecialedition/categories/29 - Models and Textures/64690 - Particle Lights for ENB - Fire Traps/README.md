# Particle Lights for ENB - Fire Traps
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64690


If you use [ENB Light](https://www.nexusmods.com/skyrimspecialedition/mods/22574) and an interior lighting overhaul mod… you will notice that there are some fire spouts casting enough light.
  
But the trap mechanisms in this mod are separated from all of that... and their fire FX looks sad…
  
Enough of that!!!
  
  
  
  
**Attention!!** The “Nordic Fire Pressure Plate” works perfectly because the lights only appear when the fire comes out.
  
On the other hand, the “Dwemer Fire Pillar” isn’t perfect because the lights are always on.
  
However, every time you find one of these, it’s either hidden underground or already working.
  
So, chances are you won’t even notice… and if you do, there’s exactly 1 second difference from when the pillar comes out and when it starts spitting fire.
  
Still, if that bothers you… I included an optional download with the "Nordic Fire Pressure Plate" only :)
  
  
\*note: shh… when it’s working; the lights rotate with the pillar and it looks cool… but don’t tell anyone… xD

**[size=4][b]INSTALLATION:**
  
[/size][/b][size=4]1) Install the main file manually or using any mod manager.
  
[/size]
  
[size=4][size=3][size=4]2) Be sure that you have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
[size=3][size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
  
  
**COMPATIBILITY:**
  
-[the Dwemer 4K 2K](https://www.nexusmods.com/skyrimspecialedition/mods/49234) includes the same mesh and it's almost the same after I checked. I don't think a patch would be worth... so, just overwrite that mod with this.
