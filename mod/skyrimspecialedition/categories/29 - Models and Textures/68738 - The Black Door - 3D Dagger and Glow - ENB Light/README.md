# The Black Door - 3D Dagger and Glow - ENB Light
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68738


![](https://i.imgur.com/ZTjfxtd.jpg)
  
  
I think this artwork by Nuare Studio Inc. looks awesome! And while I can't create something exactly like it... I made something inspired by it :)
  
You will get AI Upscaled textures edited  to replace the 2D dagger with a 3D dagger, a new glowmap, ENB light and a material fix.
  
Previously you couldn't hit this door... now you can and you will find that it's made of stone xD
  
I hope you like it ^-^
  
  
**Check the images! :O**[size=4]**[size=3][size=4][b]INSTALLATION:**
  
[/size][/b][size=4]1) Choose one main file and install it manually or using any mod manager.
  
[/size]
  
[size=4][size=3][size=4]2) IF you chose one main file with ENB Light... Be sure to have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
[size=3][size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
