# Slim Fit Glass Armor for Men
- Author: GrowlingGuy41
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128329


This mod replaces the male glass armor, gauntlets and boots .nif meshes with slimmer versions. No more mister fat bastard sneaking around doing backflips! Light, slim, stealthy and healthy :)
  
  
Install:
  
Unzip the "meshes" folder from one of the archive folder options to the game's "Data" folder or install with mod manager that supports FOMODs.
