# Less glowy spider drool
- Author: CarpeNihil
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/102060


Having issues with bright, colorful spider drool glowing in the dark? This should fix that by reducing emissive multipliers, changing some alpha values and adding less saturated textures for both the drool itself, the projectiles and the impact. Hopefully I got it all, but as usual I don't know what I'm doing when fiddling with these things in Nifskope. The issue seems most pronounced with ENBs.
