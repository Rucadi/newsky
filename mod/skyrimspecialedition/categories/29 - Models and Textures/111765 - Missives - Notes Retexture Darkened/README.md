# Missives - Notes Retexture Darkened
- Author: Artisto
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111765
Makes the notes designed by [Jayserpa](https://www.nexusmods.com/skyrimspecialedition/users/5201727) and [FrankBlack](https://www.nexusmods.com/skyrimspecialedition/users/146320) look weathered, adds a couple of new drawings and small details, such as burned edges, water stains, fly specks, etc. Also includes The Witcher Board retexture, done in a similar tonal range with the same attention to detail for a complete look.
  
  
Required original mods:
  
[Missives](https://www.nexusmods.com/skyrimspecialedition/mods/17576)
  
[Missives - Notes Retexture](https://www.nexusmods.com/skyrimspecialedition/mods/46201)
  
  
Recommended to make use of all included files:
  
[Missives - The Witcher Board HD (4K-2K-ENB Light)](https://www.nexusmods.com/skyrimspecialedition/mods/47097)
  
[Diverse Witcher Missives Boards - Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/111770)
  
  
Compatible:
  
[Missives - Unique Missive Boards](https://www.nexusmods.com/skyrimspecialedition/mods/111375)Place my mod in the end to get the files showing
