# Wyrmstooth - PBR
- Author: Jonx0r Faultier and tmwd
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148883


[font=Arial][font=Arial][font=Arial]**[size=6]Description**[/size][/font] 
  
  
This mod is a PBR Conversion of the assets found in the x4 Generative Upscale optional file for Wyrmstooth. What this means is, it may not cover everything if that file doesn't. You will still need PBR textures for any vanilla texture & texture paths Wyrmstooth may use.
  
[/font][/font]
