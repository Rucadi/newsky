# Vigilant - Greymarch Retexture
- Author: xavbio
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/58390
**IF YOU USE VIGILANT SE 1.7 OR HIGHER, THIS MOD IS NO LONGER NECCESSARY. IT WILL NOT BREAK ANYTHING, BUT SIMILAR CHANGES TO THE MESHES ARE NOW INCLUDED IN VIGILANT SE, MAKING MY MOD IRRELEVANT.**
  
I am leaving it up for people who are still on pre 1.7 version of Vigilant
  
  
  
This is a quick mod I made to fix the noticeable tiling from the greymarch section of Vicn's [Vigilant SE](https://www.nexusmods.com/skyrimspecialedition/mods/11849), when using my other mod [Vigilant armors and weapons retexture SE](https://www.nexusmods.com/skyrimspecialedition/mods/45735). It was mostly meant as a patch to match the visuals from my retexture of the Laza of order and regular order weapons and armors (The armors and environment use a shared texture), but I made it available as a standalone version, if you like the look of this greymarch version but do not wish to use my armor retexture.
