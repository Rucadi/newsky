# TDN Vampire Armor Retexture SE Port
- Author: TheDNightshade and SeranaXx
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115570


Description as on the original page:
  
  
HD retexture of all Vampire Armors:
  
- Vampire Boots and Gauntlets
  
- Black, white, red versions - female and male
  
- Royal Vampire Armor – female and male
  
- Valerica’s Armor
  
- Serana’s Hood
  
  
There are also three optional files: 
  
- A plugin for crafting. All the items can be found under the Leather section (main file is required for upgraded textures)
  
- A file that removes the pauldrons from the male Royal Vampire Armor (main file is required)
  
- An optional texture pack that will remove the Vampire brooch from all the outfits.
  
  
If you use Rustic Clothing, load this mod **after** it to get the new robe appearances.
  
  
Link to the original mod: [TDN Vampire Armor Retexture](https://www.nexusmods.com/skyrim/mods/55640)
