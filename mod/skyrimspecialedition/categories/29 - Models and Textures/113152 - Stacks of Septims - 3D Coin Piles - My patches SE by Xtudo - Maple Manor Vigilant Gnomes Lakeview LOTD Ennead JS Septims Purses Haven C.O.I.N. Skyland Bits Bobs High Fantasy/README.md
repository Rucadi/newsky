**Stacks of Septims**
  
**3D Coin Piles**
  
**My patches by Xtudo**

  
**Description**
  

  
* **Septim HD**. It will match the color of the Septim HD textures. Requires the original 3D Coins Piles mod installed first.
**Skyrim Coin Replacer Redux or similar mods**. It will replace the **Harald** and **Dumac** piles. [Skyrim LE mod](https://www.nexusmods.com/skyrim/mods/48195)﻿, but a lot of people already ported it to their SE-AE game, so I'll upload it here just in case.
  
* **JS Purses and Septims**. It will match the color of the JS Septims mod. Requires the original 3D Coins Piles mod installed first.
**C.O.I.N.** **Default** and **Ancient Dwemer Metal Nchuak** versions available, install only one.
  
* **Ancient Imperial Septims - Silver and Classic Gold**. Requires the original 3D Coins Piles mod installed first.
**2K Coin Retexture.** Requires the original 3D Coins Piles mod installed first.
  
* **Alessian Coin - Septim Replacer SE**. Requires the original 3D Coins Piles mod installed first.
**Gold Septim - Coins Retex**. Requires the original 3D Coins Piles mod installed first.
  
* **Ruin Coins Default** and **Ancient Dwemer Metal Nchuak** versions available, install only one.
**New Golden Coin**. Requires the original 3D Coins Piles mod installed first.
  
* **Coins of Tamriel.** Standalone. It also perfectly works for **Medes in Your Pockets**.
**Smaller Septims.** 75% and 50% versions available. Requires the original 3D Coins Piles mod installed first.
  
* **Septim - a coin retexture.** Bronze, Gold and Silver versions available. Requires the original 3D Coins Piles mod installed first.
**Skyland Bits and Bobs - A Clutter Overhaul**. Requires the original 3D Coins Piles mod installed first.
  
* **LOTD - Big 3D Coin Piles - JS Purses and Septims.** Requires the original 3D Coins Piles mod installed first. The ESPFE file is required to adjust some items to the new big pile height. Install it before mods like Lux. **SMIM** version available. 100% compatible with LOTD v6.
**Haven Bag**. Big 3D coin piles and chest replaced. Requires Haven Bag, along with Haven Bag ESP Bug Fix and Haven Bag - Revamped installed first. Requires the original 3D Coins Piles mod installed first.
  
* **Lakeview. Manor - As It Should Be - 3D Coin Piles - JS Septims - SMIM**. Requires the original 3D Coins Piles mod installed first. The ESPFE file is required to adjust some items to the new piles. Install it before mods like Lux.
**Gnomes- Mihail Monsters and Animals - JS Septims**. **High Fantasy Pack** version available. Requires the original 3D Coins Piles mod and my JS Septims patch installed first.
  
* **Maple Manor SSE - Player house with Legacy Support - 3D Coin Piles - JS Septims - SMIM**. Requires the original 3D Coins Piles mod installed first. The ESPFE file is required to adjust some items to the new piles. Install it before mods like Lux.
**Ennead Septims**. Cooper, Gold, Rose-Gold and Silver versions available. Requires the original 3D Coins Piles mod installed first.
  
* **Gold Master**. Requires the original 3D Coins Piles installed first.
**Better Gold - Nordic Edition**. Requires the original 3D Coins Piles installed first.
  
* **Historical Revival - The Roman**.
**VIGILANT - JS Purses and Septims**. Standalone. Default and Lux (install it after Lux - Vigilant patch.esp) versions available. 
  
* **Gold Pile Spell.** Requires the original 3D Coins Piles mod installed first.
**Treasure of the Ancients - Coin**. Standalone. The patch will work with any version (Option) of this mod.
  
* **HFs - StrongBox Diversification.** Default Vanilla coins and JS Septims versions available. **These versions had sound included, don't use together with the HFs - StrongBox Diversification - sound original optional download, please remove it if you have it installed.**
It's safe to install and uninstall it at any time.
