# Lunar Guard Armor - PBR
- Author: Deydranos and tmwd
- Game: skyrimspecialedition

[font=Arial][font=Arial][font=Arial]**[size=6]Description**[/size][/font] 
  
  
This mod is a PBR version of [Lunar Guard Armor](https://www.nexusmods.com/skyrimspecialedition/mods/75349)by [Deydranos](https://www.nexusmods.com/skyrimspecialedition/users/146458538). This is made using PBR materials laid over the original mods textures. 
  
  
Recently there was an AI conversion of this mod posted, and I felt that this mod deserved to have a (in my opinion) much higher quality PBR version available.
  
  
There is also a version that has a more silver color to the metal - and a version available with both color options (for Sentinel users). Currently all options are full 4K Textures
  
  
As always, if you have any feedback or encounter any issues, please feel free to leave a comment! :)[/font][/font]
