# Fluffy Mihail's Sewer Rats
- Author: FrankBlack and Mihail
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/137658


Hi everyone!
  
  
This is a mesh patch for Mihail's [Sewer Rats - Mihail Monsters and Animals (SE-AE version)](https://www.nexusmods.com/skyrimspecialedition/mods/137591) that adds shell-textured fluff to the rats. A BIG thank you to Mihail for allowing patch mods like this one to be made. Check out the gallery to see how the shell texturing looks in-game.
  
  
Installation:
  
  
a) Install [Sewer Rats - Mihail Monsters and Animals (SE-AE version)](https://www.nexusmods.com/skyrimspecialedition/mods/137591).
  
b) Endorse that mod.
  
c) Install this mod and overwrite when prompted.
  
d) Enjoy!
