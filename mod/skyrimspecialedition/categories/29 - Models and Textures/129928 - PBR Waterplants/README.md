# PBR Waterplants
- Author: jepster
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129928


PBR (physically based rendering) approximates how real materials react to light and is used in most modern games. PBR is a part of Community Shaders that has several benefits which includes  more realistic lighting than in vanilla and complex material. To achieve this I have unbaked shadows from diffuse maps, created separate Ambient Occlusion maps, roughness maps, parallax maps and maps for subsurface scattering.
  
Also meshes needed to be updated to enable physically based rendering. I have not included meshes, so you need to rerun ParallaxGen. This has the advantage that the mod also works with Animated Water Lily mod.
  
  
Installation:
  

  
* Install like any other mod and make sure that you have the latest version of Community Shaders installed

  
* Overwrite Waterplants / Flora Additions - Waterplants with this mod
Rerun ParallaxGen
  

  

  
Seasonal patch for Flora Additions - Waterplants is available [here](https://www.nexusmods.com/skyrimspecialedition/mods/118601) (use PBR version)
  
  
Load order:
  
Make sure that nothing overwrites the meshes of this mod. Except ParallaxGen
