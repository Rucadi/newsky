# High Quality PBR Ivy
- Author: jepster
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129934


PBR (physically based rendering) approximates how real materials react to light and is used in most modern games. PBR is a part of Community Shaders that has several benefits which includes  more realistic lighting than in vanilla and complex material. To achieve this I have cleaned compression artifacts, unbaked shadows from diffuse maps, created separate Ambient Occlusion maps, roughness maps, parallax maps and maps for subsurface scattering. I have not included meshes, so you need the original mod and run ParallaxGen.
  
  
**It requires the [original mod](https://www.nexusmods.com/skyrimspecialedition/mods/113578) and is compatible with any of the install options (Nordic Stonewalls, animated or non-animated ivy etc.). So choose freely and use this mod here to upgrade the textures to PBR.**
  
  
Installation:
  

  
* Install like any other mod and make sure that you have version 1.0 or higher of Community Shaders installed
Load order does not matter
  

  
* Rerun ParallaxGen if you install the first time or upgrade the mod (v1.0 to 1.1)
    
  No need to rerun ParallaxGen if you reinstall the mod and chose different texture options

  
  
  
Load order:
  
Make sure that nothing overwrites the meshes of this mod. Except ParallaxGen
