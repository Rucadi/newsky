# Particle Lights for ENB - Staff Enchanter
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/51454
**Check the images and you will know what this is all about :D**
  
  
  
**INSTALLATION:**
  
1) Install manually or using any mod manager.
  
2) Be sure that you have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
**REQUIREMENTS:**
  
An ENB preset to enable the particle lights effect.
  
  
  
**COMPATIBILITY:**
  
-Compatible with [4K Dark Elf Furniture](https://www.nexusmods.com/skyrimspecialedition/mods/40026) by IWantATardis. Just replace 1 mesh with this mod. It will look different but... idk... I like it more this way... lol
  
-Compatible with [size=3][Solstheim Objects SMIMed - High Poly Dark Elf Furniture](https://www.nexusmods.com/skyrimspecialedition/mods/53779)﻿ (patch provided under optional downloads :)).
  
-Compatible with [Less Crappy (High Poly) Dark Elf Furniture](https://www.nexusmods.com/skyrimspecialedition/mods/51844) (see sticky posts).
