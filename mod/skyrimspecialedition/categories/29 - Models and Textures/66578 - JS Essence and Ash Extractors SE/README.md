**What this mod includes:**
  
  
• New Meshes & Textures for the Essence Extractor from the "Discerning the Transmundane" Quest & The Ash Extractor from the "Telvanni Research" Quest
  
• Essence Extractor has articulated levers
  
• Ash Extractor has various animated elements
  
  
Both models include textures for Complex Parallax
  
  

![](https://i.imgur.com/ZM71deY.png)

  
**Installation**
  
  
• Mod Manager / Manual Extraction to the Skyrim Data folder.
  
 • Overwrite any existing files. No plugin requirement.
