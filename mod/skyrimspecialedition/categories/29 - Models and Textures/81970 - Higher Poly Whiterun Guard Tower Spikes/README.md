# Higher Poly Whiterun Guard Tower Spikes
- Author: Pieter82
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/81970


The five sided wooden spikes fixed to some of the Whiterun guard towers seemed like they could do with some rounding out. 
  
Polygon count has been doubled and the spikes have been smoothed out.
  
  
Compatibility:
  
This mod replaces only the mesh file, no plugin/ESP included. No textures are included in the download, your own load order will determine the actual textures used.
  
  
Installation:
  
Use your preferred mod manager to install as per normal.
  
Overwrite SMIM if prompted / give my mod priority.
