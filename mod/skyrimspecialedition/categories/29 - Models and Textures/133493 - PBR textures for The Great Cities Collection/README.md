# PBR textures for The Great Cities Collection
- Author: jepster
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/133493


PBR (physically based rendering) approximates how real materials react to light and is used in most modern games. PBR is a part of Community Shaders that has several benefits which includes  more realistic lighting than in vanilla and complex material. To achieve this I have unbaked shadows from diffuse maps, upscaled them using an AI model,  created separate Ambient Occlusion maps, and roughness maps and also parallax maps.
  
  
  
Installation:
  

  
* Install like any other mod and make sure that you have version 1.0 or above of Community Shaders installed

  
* Overwrite The Great Cities - Resources with this mod. Make sure no other mod overwrites this one (except ParallaxGen output later).
Rerun ParallaxGen
