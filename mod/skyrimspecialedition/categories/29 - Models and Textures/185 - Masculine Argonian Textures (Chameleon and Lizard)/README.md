**EXTRA Package:** ﻿It's the additional content for more chest variations including:
  
**Diffuse Maps:**
  
1. Abs
  
2. Standard Nipples
  
3. No Nipples and Belly Button
  
**Normal Maps:**
  
1. Abs
  
2. No Belly Button
  
**Specular Maps:**
  
1. Abs
  
2. No Nipples
  
  
These options are mixed with the previous chest variations: Heavy Scales, Less Scales and Bare Chest (Standard Scales option is excluded).
  
You can choose whatever chest shape you want with any other option you like whether it's diffuse maps, normal maps or specular maps thus creating hundreds of possible outcomes.
  
Example: you can mix diffuse map (Bare Chest - Standard Nipples) with normal map (Heavy Scales - Abs) with specular map (No Nipples).
  
The EXTRA package is optional so you still need the main file to use it, I have excluded the options that already exist in the main files.*[/font]

[font=Courier New]*- Install normally via Vortex, NMM or Mod Manager.
  
- Vanilla textures option doesn't require any other mod, it's for base-game Argonian male.
  
- SOS Lite textures option requires SOS Lite installed first then overwrite with this mod.
  
- SOS Full textures option requires SOS Full (Smurf Average).
  
- Better Males textures option requires Better Males body (Cut, Uncut and Erect).
  
- SAM Light textures option requires SAM Light body (Cut, Uncut and Underwear).
  
Use the patch for (Uncut version) and load it after the main file.
  
- If installing manually, you need to extract to Skyrim inside "Data" folder.*[/font]
