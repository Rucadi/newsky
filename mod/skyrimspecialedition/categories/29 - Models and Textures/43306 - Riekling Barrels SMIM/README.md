**Description**
  
SMIM adds new barrels, but Riekling exploding barrel traps still use old vanilla barrels, which makes them distinct and defeats their purpose. This mod makes Riekling barrels use SMIM style barrels
  

  
* Replaces Riekling barrel traps with SMIM style barrels
Replaces exploded barrel debris with SMIM style debris
  
* Riekling traps now remain hidden
Mesh replacer only. No plugins
  
**Requirements**
