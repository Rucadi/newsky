# Grindstone Variants - Base Object Swapper
- Author: Pieter82
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106719
Grindstone Variants uses Base Object Swapper (BOS) to dynamically add three variants of the Grindstone (Blacksmith Sharpening Wheel) meshes. 
  
  
Main File: 1) Grindstone Variants
  
Each variant features refined vanilla geometry, animated water in the cup and some small added details.
  
Vanilla texture paths are used, so any vanilla retexture of the grindstone will be compatible. In my screenshots I am using [Grindstone HD - Texture Replacer](http://www.nexusmods.com/skyrimspecialedition/mods/57395).
  
Variants are dispersed randomly throughout the game world on installation of the mod and/or with a new game creation (variants remain persistent after initial swap). 
  
The plugin is flagged as ESL.
  
  
[u]Main File: 2) Grindstone Variants - ElSopa Variant
  
[/u]This version of the mod includes ElSopa's beautiful grindstone, from [ElSopa - HD Grindstone Redone SE](http://www.nexusmods.com/skyrimspecialedition/mods/58149?tab=description), as a replacer for the vanilla mesh, in addition to the variants found in Grindstone Variants. ElSopa's mesh is then included in the swap file for distribution throughout the game world, bringing the total number of distributed variants to four.
  
﻿Special thanks to [ElSopa](https://www.nexusmods.com/skyrimspecialedition/users/6960827) for creating and hosting the required texture files to make the inclusion of this fantastic mesh possible. Please remember to endorse ElSopa's mod should you choose to make use of this version.
  
 
  
Optional Files:
  
Various patches for [Improved Sparks](http://www.nexusmods.com/skyrimspecialedition/mods/19831)﻿ (IS) and 1) Grindstone Variants (GV) / 2) Grindstone Variants - ElSopa Variant (GV-EV). Please choose the appropriate optional file for your load order and ensure priority is given to the patch file / have it overwrite both the GV main file and Improved Sparks.
  
These patches are also compatible with [Really Seriously Improved Sparks](http://www.nexusmods.com/skyrimspecialedition/mods/78439). 
  
  
Requirements:
  
[Base Object Swapper](http://www.nexusmods.com/skyrimspecialedition/mods/60805) for SE and AE.
  
[Base Object Swapper VR](http://www.nexusmods.com/skyrimspecialedition/mods/61734) for Skyrim VR.
  
[ElSopa Grindstone Textures For Variants 2k](http://www.nexusmods.com/skyrimspecialedition/mods/58149?tab=files) - Required for the ElSopa Variant version, available under the "Optional Files" section.
  
[Improved Sparks](http://www.nexusmods.com/skyrimspecialedition/mods/19831) - Only required if making use of any of the Optional IS patches.
  
  
Installation:
  
Use only one main file.
  
Use your preferred mod manager to install as per normal.
  
Give Optional patch files priority.
  
  
If using the ElSopa Variant main file, please remember to install the optional file: [ElSopa Grindstone Textures For Variants 2k](http://www.nexusmods.com/skyrimspecialedition/mods/58149?tab=files), hosted on ElSopa's mod page. You require only the optional file, as my mod includes the required mesh. Load order does not matter.
