# Vanaheimr - Mountains - MM - Complex Material - PBR
- Author: Cl3mus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115471


[font=Trebuchet MS][/font]

[font=Comic Sans MS]﻿﻿A complete retexture of Mountains. Compatible with Majestic Mountains ( Complex Material ), ERM - Enhanced Rocks and Mountains ( Complex Parallax Material ) and True PBR.
  
  
  
------
  
**![](https://iili.io/2OWbDI2.png)﻿
  
  
*This mod includes :***
  
  
  
- A fomod that lets you choose the version you want.
  
- New textures for the Mountains in 8k and 4k.
  
- Textures compatible with [Majestic Mountains](https://www.nexusmods.com/skyrimspecialedition/mods/87547)﻿﻿. ( with Complex Parallax Support )
  
- Textures compatible with [ERM - Enhanced Rocks and Mountains](https://www.nexusmods.com/skyrimspecialedition/mods/85196) ( all versions ) and [ERM - Complex Parallax Materials](https://www.nexusmods.com/skyrimspecialedition/mods/121336).
  
- True PBR version.
  
-Grey textures if you prefer a desatured version.
  
﻿
  
  
------
  
  
  
The Majestic Mountains Version works with [Majestic Mountains](https://www.nexusmods.com/skyrimspecialedition/mods/11052) or [Majestic Mountains Complex Material](https://www.nexusmods.com/skyrimspecialedition/mods/87547)﻿.
  
  
  
The Normal Version works with [ERM - Enhanced Rocks and Mountains](https://www.nexusmods.com/skyrimspecialedition/mods/85196)﻿ ( All versions )
  
(  I include all the maps necessary for the parallax and Complex Parallax Material to work )
  
  
  
The True PBR version requires [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492)﻿ to function. It is recommended to use [ParallaxGen](https://www.nexusmods.com/skyrimspecialedition/mods/120946)﻿ for mesh generation. 
  
**The Vanilla mountains textures are in 4k.****A 2k version is available for potato pc.**
