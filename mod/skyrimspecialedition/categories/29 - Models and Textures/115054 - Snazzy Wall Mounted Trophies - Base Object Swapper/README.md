# Snazzy Wall Mounted Trophies - Base Object Swapper
- Author: gutrmaw
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115054


[font=Times New Roman]

***Snazzy Wall Mounted Trophies - Base Object Swapper﻿*****Using Base Object Swapper, replaces some wall mounted animal trophies with new models...Trolls, Vale Deer and Sabre Cats, Foxes, Rabbits, Chaurus Hunters, Werewolves, Huskies, Death hounds, Bristlebacks, Ash Hoppers, etc.**

[/font]
  
  
[font=Times New Roman]Mod Features:
  
~ ESL flagged main plugin...won't count against your max plugin count.
  
~ Base Object Swapper is used to distribute (5-10% chance) 9 new animal and monster wall mounted trophy models throughout Skyrim
  
~ Swaps in (35% chance) Werewolf and Husky wall mounted trophies in any Vampire locations.
  
~ Swaps in (35% chance) Death Hound wall mounted trophies in Dawnguard and Stendarr locations.
  
~ Swaps in (40%) either a Bristleback or Ash Hopper wall mounted trophy In Solstheim locations...2% chance in all non-Solstheim locations.
  
~ Easy to configure BOS Swap.ini...if you don't want a certain model, or want to increase the swap rate, you can easily change to your liking.
  
  
  
Compatibility:

  
* [font=Times New Roman]Requires [/font][Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) and it's requirements﻿
Compatible with [Dynamic Things Alternative - Base Object Swapper﻿﻿](https://www.nexusmods.com/skyrimspecialedition/mods/60741) with patch

﻿Optional File - Fluffy Wall Mounted Boars
  
* Optional File - Fluffy Wall Mounted Rabbits (fluffy textures from [Real Rabbits HD](https://www.nexusmods.com/skyrimspecialedition/mods/29223)﻿ by fadingsignal required)
