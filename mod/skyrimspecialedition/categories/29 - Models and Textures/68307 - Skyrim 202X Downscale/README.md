# Skyrim 202X Downscale
- Author: Pfuscher - Wartom94
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68307
The goal of this mod is to reduce VRAM (or video memory) usage from original mod by reducing the texture resolution size.
  
  
The textures of this mod are uploaded at 1K, 2K and 4K resolution size instead of 8K-16K (maximum size).Check [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492) with [Complex Parallax Materials](https://www.nexusmods.com/skyrimspecialedition/mods/95134) for non enb users who want to use complex parallax shader.
  
  
For parallax terrain use this mod: [Skyrim 202X Complex Terrain Parallax](https://www.nexusmods.com/skyrimspecialedition/mods/54860)
