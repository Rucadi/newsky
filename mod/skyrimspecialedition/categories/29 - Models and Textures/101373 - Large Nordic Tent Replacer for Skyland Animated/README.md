# Large Nordic Tent Replacer for Skyland Animated
- Author: Skyking2020-StoneSpiralGaming
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/101373
This simple mesh replacer replaces the static cloth on the canopy to one that blows in the wind. Created with the full support of [Skyking2020](https://www.nexusmods.com/skyrimspecialedition/users/841133)
  
This mod REQUIRES either [Skyland AIO](https://www.nexusmods.com/skyrimspecialedition/mods/34179) or [Skyland Imperial and Nordic Tents](https://www.nexusmods.com/skyrimspecialedition/mods/57002) as no textures are included in this mod
