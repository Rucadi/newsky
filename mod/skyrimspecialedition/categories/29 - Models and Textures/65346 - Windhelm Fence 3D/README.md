**Description**
  

  
* Replaces Windhelm's 2D fences with 3D fences
[2 texture options](https://staticdelivery.nexusmods.com/mods/1704/images/65346/65346-1648109134-761275501.png)﻿
  
[list]* [Custom textures](https://youtu.be/bjpUXlqr5m0)﻿ - available in 8k, 4k, 2k, and 1k options
[Vanilla textures](https://youtu.be/ur9hWlOL9aM)﻿ - works with texture mods that does not alter the fence's design (screenshots taken with [Northfire's Windhelm](https://www.nexusmods.com/skyrimspecialedition/mods/44099)﻿)
  

[\*]Fixes some [stretched vanilla UVs](https://staticdelivery.nexusmods.com/mods/1704/images/65346/65346-1647942640-66159473.png)﻿
  
[\*]Fixes [incorrect lighting due to double-sided shader](https://staticdelivery.nexusmods.com/mods/1704/images/65346/65346-1647942630-257721839.png)﻿: on double-sided faces, both sides are lit the same (they have the same normal)
  
[\*]Added specularity
  
[\*]Fixed some [bad vertex colors](https://staticdelivery.nexusmods.com/mods/1704/images/65346/65346-1674516225-1576390982.png)﻿
  
[/list]**Older videos showcasing the vanilla texture path option**
