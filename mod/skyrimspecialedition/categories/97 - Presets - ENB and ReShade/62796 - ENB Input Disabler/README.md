# ENB Input Disabler
- Author: doodlum
- Game: skyrimspecialedition

DESCRIPTION**
  
[/b]
  
This is a simple SKSE plugin which disables mouse and keyboard input for everything but the console, when the ENB editor is active.
  
  
Gamepads function as normal.
  
**![](https://staticdelivery.nexusmods.com/mods/1704/images/62796/62796-1643547068-114865313.gif)****[b][b]REQUIREMENTS**[/b][/b][Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444) 
  
[VR Address Library for SKSEVR](https://www.nexusmods.com/skyrimspecialedition/mods/58101)
