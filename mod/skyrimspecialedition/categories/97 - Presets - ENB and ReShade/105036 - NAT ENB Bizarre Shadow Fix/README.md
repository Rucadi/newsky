# NAT ENB Bizarre Shadow Fix
- Author: Ivanmand
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105036


With NAT.ENB ESP Weather Plugin there is a bizarre shadow / black line that follows the player character. This effect is more noticable with certain weathers. Using this mod fixes it (see before/after screenshots).
