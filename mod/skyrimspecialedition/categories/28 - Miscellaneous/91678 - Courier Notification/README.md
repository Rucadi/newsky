# Courier Notification
- Author: tarlazo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/91678


Adds an on-screen notification with the name of what you have just received, no action required.
  
If you always carry around your entire library, this might help to quickly find out what's new to read.
  
  
  
  
  
A note (undelivered)
  
The esp has only one form -- you could easily copy it anywhere and get rid of the plugin.
