# SPID - NFF - Add Ignore Token to CustomAI Followers
- Author: clickvd
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/82945


![](https://staticdelivery.nexusmods.com/mods/1704/images/82945/82945-1673978137-1079457890.png)
  
~ If an NPC can think outside the regular follower barriers, you need this! ~

  
**This mod aims to supply with a vetted SPID distribution-list for followermods and/or NPCs that are NOT to be imported into NFF.**
  
Blocking is done by means of using SPID to distribute the 'NFF Ignore Token' into the respective followers inventory, basically stopping the player
  
from importing it into the NFF framework (at least not without explicitly cleaning the follower inventory first).
  
  
More or less all 'vanilla' followers in Skyrim can be used with/attached to a follower framework like AFT, EFF, UFO, NFF (enhancing their functionality).
  
Some followers, however, come with their **own custom AI and separate follower framework**, and **MUST** be ignored by other follower frameworks.
  
And no, It's not even recommended to import them with 'custom settings' enabled.
  
  
***IMPORTANT:
  
This mod is not going to stop you from removing the token and importing the follower to NFF.
  
If you see the token in the NPC inventory you know that the NPC is not just a 'vanilla' follower, and are ment to be left alone framework-wise.
  
  
Again: Take heed if you remove the token and import the follower into NFF - the follower will in most cases not work as intended after that.***
