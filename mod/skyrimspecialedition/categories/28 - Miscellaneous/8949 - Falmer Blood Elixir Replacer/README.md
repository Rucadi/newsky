# Falmer Blood Elixir Replacer
- Author: Kelretu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/8949
This is a simple replacement for the Falmer Blood Elixir found in Riften.
  
   
You can use the resource for your own mods. You are not allowed to use it for mods that are being sold.
  
Please let me know when you use it, so I can enjoy whatever mod you made out of it :)
