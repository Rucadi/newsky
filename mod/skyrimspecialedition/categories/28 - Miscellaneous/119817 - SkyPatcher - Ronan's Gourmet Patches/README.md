# SkyPatcher - Ronan's Gourmet Patches
- Author: RonanTheRonin
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119817


[Vigilant SE](https://www.nexusmods.com/skyrimspecialedition/mods/11849)This mod uses [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659) to make patches for several mods for [Gourmet - A Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/96876).
  
  
Patches that add entirely new records such as Books or Recipes probably won't be covered unless they also have a EAS Patch and/or Gourmet - EAS patch. For example still use the Fishing CC patch from the Gourmet Patch Page. I will mark mods I added back the EAS - Patch Collection with [EAS] and mods I still recommend still using the mod from a patch hub with a [GPH].
  
  
I have now added Container patches marked with a [CONT]. Depending on your load order you may get duplicate items with these so if you don't want that you can go inside the ini file and remove the ";" to uncomment the clear container line. For example ;filterByContainers=MerchantCaravanCChest:clear=true  to filterByContainers=MerchantCaravanCChest:clear=true
