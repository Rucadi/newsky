# Thrones of Skyrim
- Author: Kelretu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/41198
Replaces the meshes for the generic thrones of the various holds. No .esp required.
  
  
The use vanilla textures, so texture mods will affect these thrones.
  
  
No idea as to compatibility.
