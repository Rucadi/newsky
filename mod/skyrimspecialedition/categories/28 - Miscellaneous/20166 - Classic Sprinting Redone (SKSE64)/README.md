# Classic Sprinting Redone (SKSE64)
- Author: Vermunds
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/20166
This mod will make the sprinting work the same way as it did in old Skyrim. Hold instead of a toggle. It works with keyboards and controllers too.
  
  
To install simply put the contents to your Skyrim's Data folder, or use your Mod Manager.
  
[SKSE64](https://skse.silverlock.org/) and [Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444) is required.
  
  
Recommended mod that is fully compatible: [Better Jumping SE](https://www.nexusmods.com/skyrimspecialedition/mods/18967) by meh321 and z65536 .
