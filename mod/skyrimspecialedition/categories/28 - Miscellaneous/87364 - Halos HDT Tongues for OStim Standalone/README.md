# Halos HDT Tongues for OStim Standalone
- Author: Halo---Shirou---VersuchDrei
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/87364


**Original required!**

  
Adds all ten of Halos Human HDT tongues to the MCM and random selection for OStim NGs tongues. Requires OStim NG 6.6.
  
  
**Warning**: I ESL-ified the plugin, so it is not compatible with mods requiring the original Halos Human HDT tongues, as the form IDs changed.
