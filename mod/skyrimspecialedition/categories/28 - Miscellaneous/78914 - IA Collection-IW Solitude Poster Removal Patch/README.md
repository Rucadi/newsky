# IA Collection-IW Solitude Poster Removal Patch
- Author: Geeknasty
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78914
This is a small ESPFE (light plugin) patch to increase immersion and compatibility with the I&A collection by reducing the visibility of the poster outside Solitude. This was made by request for a user of the I&A collection. It just hides the poster.
