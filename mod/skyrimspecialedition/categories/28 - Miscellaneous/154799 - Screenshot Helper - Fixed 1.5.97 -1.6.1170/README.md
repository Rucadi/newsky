# Screenshot Helper - Fixed 1.5.97 -1.6.1170
- Author: TwistedModding Doodlez Kruziikrel13
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/154799


[Kruziikrel13](https://next.nexusmods.com/profile/Kruziikrel13?gameId=1704)﻿ updated [Screenshot helper](https://www.nexusmods.com/skyrimspecialedition/mods/64124)﻿ for 1.6.640 [here](https://www.nexusmods.com/skyrimspecialedition/mods/126563), but both mods suffer the same issue of overwriting screenshots if they have the same name. This typically happens when you take a screenshot at the same in game minute. Now it will append a suffix (incrementing from 1) to the end of a screenshot if it shares the same name with another screenshot.
  
  
All credit to [doodlum](https://www.nexusmods.com/skyrimspecialedition/users/28038035) and [kruziikrel13](https://next.nexusmods.com/profile/Kruziikrel13?gameId=1704)﻿ for creating the mod and updating it to the latest game versions respectively.
