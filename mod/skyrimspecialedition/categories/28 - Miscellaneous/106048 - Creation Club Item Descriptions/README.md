**This mod simply provides fitting descriptions to most of miscellaneous items across Bethesda's Creation Club Addons.**
  
**Using the power of** [Description Framework](https://www.nexusmods.com/skyrimspecialedition/mods/105799)**and inspired by classic RPGs item descriptions.**
  
**The mod aims to enrich Creation Club's quest by giving each item unique an interesting flavor text.**
  
  
**Supports EVERY Creation Club mod currently available**
  
*It is highly encouraged that you use a mod to expand the item card size, since the longitude of some descriptions cannot fit on the vanilla space and could appear significantly small.*
