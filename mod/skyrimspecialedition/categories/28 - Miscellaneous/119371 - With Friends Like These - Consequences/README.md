# With Friends Like These - Consequences
- Author: Oldkottor
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119371


To express their gratitude each of survivors will send you a letter and point you to a small reward.
  
  
The logical flow could be broken if you are on a killing spree, but do you care for the logic in this case anyway?
  
  
The mod relies on the stages and properties of the vanilla quest DB02. So if another mod changes this quest there could be conflicts.
