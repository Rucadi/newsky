[size=6][font=Tahoma]**OInflation**[/font]

![](https://i.postimg.cc/HLWPz5HQ/Inflate.gif)﻿
  
To get the animation to look like the gif above enable Always fill to max and set Cum spurts to 3.
  
  
By default the actor will inflate by 20% full each time, in order to make the inflation amount based on character level disable "Manually set amount to inflate". Have fun!

                                  Current Features[font=Tahoma]                                   [/font]
  
Animated cum inflation.
  
Support for inflating up to 3 bodymorphs at a time.
  
Automatic MCM saving and loading.
  
Extensive customization options, including the ability to edit the inflation animation.
  
Support for negative morph values.
  
Futa and male support.
  
Supports the latest version of OCum Ascended and all of it's future updates.
