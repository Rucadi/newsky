**Description**
  

  
* Bethesda created landscapes explicitly labeled "No Grass" in order to paint landscape texture underneath objects without the grass clipping through
Many grass mods add grass to these landscapes.
  
* Some grass mods (Such as [Cathedral Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/21954)﻿) add short grass to the "No Grass" landscapes to prevent grass clipping.
[Seasons of Skyrim SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/62861)'s auto generated configuration swaps landscapes with grass to tall snow grass, no matter how short the original grass is
  
* This mod ensures landscapes labeled "No Grass" are swapped to snow landscapes without grass
Inspired by [No Grassias](https://www.nexusmods.com/skyrimspecialedition/mods/35639)﻿
