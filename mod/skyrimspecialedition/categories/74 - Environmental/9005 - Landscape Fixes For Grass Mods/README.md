# Landscape Fixes For Grass Mods
- Author: code1k
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/9005


Goal of this mod 
  
Primary[color=#ffffff]:[/color] Change landscape to reduce grass clipping through objects and restore lost details (details=pathways and small objects)
  
[color=#76a5af][color=#1e84cc]Secondary:[/color] [/color]Improve performance by removing grass where it cannot normally be seen without ~tcl (under Whiterun and inside Buildings of Falkreath for ex.)      & Reduce overgrown sections of some roads  
  
  
This mod adds support to the vanilla game for all expanded landscape type grass mods however some may require the 'generic grass limiter' from files here to remove grass from certain dirt/rock textures, check list below or do own comparison.
  
And yes it will fix some clipping/placement issues with vanilla landscape grass also.

***Load order information[color=#ffffff]:[/color]*****Keep this mod near the top of your load order and make sure you load it before any mods that change land height in the areas this mod edits.**
  
Load this mod after Verdant, Folkvanger or other grass mods [color=#ffffff]unless otherwise instructed. (Generally it is best to manually move grass mods above this mod if LOOT has no rules for new grass mods and places them very low in your load order.[/color])
  
  
**Mods that require terrain height changes must load after and disable this mod in the areas they edit**.
  
If you load this mod after mods that edit landscape height you could see things like buried or floating houses, objects, etc.., a tear/gap on the land between cells with different height or even NPCs not able to navigate due to the land height not matching new navmesh.
  
**Do not try to force the edits in this mod to override another mod** if you don't know how to test for the issues stated above.
  
  
\*There are exceptions to the rule above for example smooth shores, if you use that always move that mod before this mod. It has patches.
  
  
\*Hard texture seam conflicts i.e. non vanilla texture seams caused by this mod loading before another mod should be reported as a bug here. Great effort has been made to reduce the possibility of this mod causing hard texture seam conflicts. Some older edits or spots with unavoidable edits between cells still exist but most already have patches or have no known mods to conflict with.
  
  
Patches
  
Do not ask authors of the mods being patched here for support for any patches on this page. Any issues related to patches here such as terrain/grass should be directed to this page.
  
  
Too many patches? I recommend merging grass patches together into one file using the zMerge function of [zEdit](https://github.com/z-edit/zedit/releases).
  

  
* Do not merge the patches into the main file of this mod.

A guide on how to make a merge is available over on [Lexy's LOTD SE Mod guide Page](https://wiki.step-project.com/User:DarkladyLexy/Lexys_LOTD_SE_Merge_Page).
