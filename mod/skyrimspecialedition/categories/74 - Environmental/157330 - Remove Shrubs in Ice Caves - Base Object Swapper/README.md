# Remove Shrubs in Ice Caves - Base Object Swapper
- Author: roskospoko
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/157330


**Description**
  
  
This mod use [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805?tab=description) to remove dead shrubs only in ice caves.
  
This is [size=3]expecially useful to not break immersion if you use [Icy Cave Remaster](https://www.nexusmods.com/skyrimspecialedition/mods/90476?tab=description) or [Skyrim Remastered - Caves](https://www.nexusmods.com/skyrimspecialedition/mods/38220?tab=description)﻿.
  
  
If you are looking for an extensive solution try [No More Dungeon Greenery](https://www.nexusmods.com/skyrimspecialedition/mods/163164).
  
  
[/size]**Installation**
  
  
This mod use [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805?tab=description)﻿. [size=3]So make sure you have the latest version installed.
  
[/size]
