# Seasons of JK's Mods
- Author: RavenKZP
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105475


Mod Description
  
  
Small add-on for [jkrojmal](https://www.nexusmods.com/skyrimspecialedition/users/1305814)﻿ mods﻿ to make it work with [Seasons of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/62861).
  
In specific it's just for [JK's Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/6289)﻿, [JK's Sky Haven Temple](https://www.nexusmods.com/skyrimspecialedition/mods/63039) and [JK's Whiterun Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/78351)﻿, as they add new trees and plants.
  
You can use my mod with just one of them (for example only with JK's Skyrim but without Outskirts) , no difference.
  
Other JK's mods don't add new Trees or plants, that mean no changes needed for seasons.
  
Seasonal Alchemy integration for plants is provided, depends if you use [Seasonal Alchemy](https://www.nexusmods.com/skyrimspecialedition/mods/63969)﻿ and  [Seasonal Alchemy add-on](https://www.nexusmods.com/skyrimspecialedition/mods/83636)﻿.
  
  
[Unfrozen](https://www.nexusmods.com/skyrimspecialedition/mods/71967)﻿ support is available as optional files (All in one or separate files, your choice).
  
This (unfrozen) file cover, [JK's Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/6289), [JK's College of Winterhold](https://www.nexusmods.com/skyrimspecialedition/mods/65676)﻿, [JK's High Hrothgar](https://www.nexusmods.com/skyrimspecialedition/mods/62219)﻿, [JK's Windhelm Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/86975)﻿, [JK's Septimus Signus's Outpost,](https://www.nexusmods.com/skyrimspecialedition/mods/66915)﻿ [JK's Candlehearth Hall](https://www.nexusmods.com/skyrimspecialedition/mods/45617)﻿ and [JK's New Gnisis Cornerclub](https://www.nexusmods.com/skyrimspecialedition/mods/48293)﻿.
  
 JK's College of Winterhold﻿, JK's High Hrothgar﻿ and JK's Windhelm Outskirts have standalone versions as my mod need them as masters, so if you don't use all of them install just these patches you want.
  
JK's Skyrim, JK's Septimus Signus's Outpost,﻿ JK's Candlehearth Hall﻿ and JK's New Gnisis Cornerclub have one "Modular version", because I don't need specify them as masters in my .esp file, so no matter if you have just one or all of them, everything will work.
  
  
Nothing will break if you will use All in one with another unfrozen file, but it's just wasting of ESL slots ;)
  
Other JK's mods don't contain snow covered or frozen objects, so that mean no changes needed for unfrozen.
  
  
Note that Seasonal Landscapes - Unfrozen have build in compatibility patches for JK's Skyrim and JK's College of Winterhold.
  
My mod Is compatible with this by default, changes from SL - Unfrozen will take priority over my changes.
