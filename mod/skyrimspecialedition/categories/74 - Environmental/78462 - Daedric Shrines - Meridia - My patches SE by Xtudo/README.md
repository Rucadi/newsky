Daedric Shrines - MERIDIA****My patches by Xtudo**
  
**Description**
  
**Some little patches to the lovely statue made by Mandragorasprouts.**
  

  
* Carried on Unofficial Patch fixes. ESL.
**HD Pedestal.** 2K. Now it matches the statue color palette. Compatible with [Meridia's Luxon Beacon Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/34782) ﻿too.
  
* [No Snow Under The Roof](https://www.nexusmods.com/skyrimspecialedition/mods/518)﻿﻿ patch. ESL.
[VIGILANT SE](https://www.nexusmods.com/skyrimspecialedition/mods/11849)﻿ patch. Adjusts the Statue of Meridia to the new lovely model. Which is used three times by the Vigilant mod. ESL. Lux compatible version available.
  
* [VIGILANT SE](https://www.nexusmods.com/skyrimspecialedition/mods/11849)﻿ **Standalone** patch. This version is intended if you want to only see the new statue in VIGILANT. The vanilla statue won't be affected. It only requires the original 4K or 2K textures. ESL. Fully compatible with v1.8.0.
[Rayek's end the mORIS's end](https://www.nexusmods.com/skyrimspecialedition/mods/35130)﻿ patch. 6 statues patched.
  

  
**Requirements**
