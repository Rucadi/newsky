# TMD The Rift Leaves
- Author: TheMilkDrinker
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111461


The mod adds floating autumn leaves to The Rift waters. Plugin ESL flagged.
  
  
If using ENB, it requires complex material and dynamic cubemap on.
