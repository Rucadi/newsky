**Description**
  

  
* Uses ﻿[Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿ to distribute snowy mountain flowers
    
  [list]
Dynamically replaces regular mountain flowers with snowy mountain flowers
  
* 0 worldspace edits
Greatly reduces potential conflicts
  
* Contains thousands of manual swaps based on examining objects in game and user reports. More accurate than cell-based swaps as hundreds of cells are only partially covered by snow.
