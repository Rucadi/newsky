# Ruins of the Ideal - New Ruins in Soul Cairn
- Author: AgentW
- Game: skyrimspecialedition

[font=Georgia]**Ruins of the Ideal**[/font]

New Ruins in Soul Cairn

  
**Description:** ﻿This mod added new ruins using models from The Witcher 3 (taken from [Lor Modders' Resources](https://www.nexusmods.com/skyrim/mods/84433/?tab=description)﻿) edited to use vanilla Soul Cairn textures. The new ruins fill the void in the landscape of Soul Cairn where there isn't much going on in the vanilla game.
  
  
**Requirement**:
  
 ﻿None
