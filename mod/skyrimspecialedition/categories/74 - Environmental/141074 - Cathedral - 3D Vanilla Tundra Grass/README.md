# Cathedral - 3D Vanilla Tundra Grass
- Author: DrJacopo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141074


-Crafted from the vanilla grass textures.
  
-Efficient 3D grass models.
  
-BC3 optimized textures to improve BSA efficiency, yielding a 5.7 MB file size.
  
-Does not touch \_nograss landscape records, so if you are using other grass mods, you will probably get a blend of both.
  
-Requires imingrasssize of 50. If you decrease the density to 60, you will see ground cover tiles.
