# Complex Grass - The official Patch Compendium
- Author: Skurkbro
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67304


**-This mod page is currently under reconstruction-**
  
  
-**Important**-
  
-Make sure ENB is up to date (V0.473+)
  
-Set EnableComplexGrass=true in enbseries.ini
  
-**If you're running Cathedral 3D Mountain Flowers, let it overwrite this mod or you gonna run into ugly looking mountain flowers.** 
  
  
This mod also works with [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86502).
  
  
-**Can I use these textures/patches when i generate lod/textures?**, Yes, but please follow a [guide](https://stepmodifications.org/wiki/SkyrimSE:2.2.0#STEP_5) to get it properly setup. Not my area of expertise so I can't help you here.
