**Overview**
  

  
* A straightforward, no-hassle landscape improvement mod
Integrates under-utilised assets from the into the wider world of Skyrim
  
* Mod replaces some repetitive base game flora and repetitive objects in The Reach region with:
﻿(a) Gem Geodes
  
* (b) Rot Scale
(c) Indian Rice Grass 
  
* (d) Desert Grass
(e) Sword Ferns 
  
* (f) Thistle, Lavender and Nightshade
(g) Root Weeds
  

  
I have created separate .inis for each individual replacement if you prefer to mix-and-match. Please see optional mods
  
  
The main mod does **not** contain the .inis for my Reach Tree and Reach Clifftree replacers as seen in the images. You will need to download those mods separately (optional download)

**Installation and Uninstallation**
  
Use the mod manager of your choice to install. Or manually (in your Data folder). To uninstall, remove with mod manager. My understanding is that BOS will not impact save games in any way. **Safe to install or uninstall at anytime**.
