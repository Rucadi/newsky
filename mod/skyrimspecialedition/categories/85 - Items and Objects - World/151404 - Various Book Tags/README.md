Intended as a(nother) niche mod, originally meant to help keep track of [Skill Books](https://en.uesp.net/wiki/Skyrim:Skill_Books), and to help keep track of which mod a spell tome or book is associated with. Now, if you're not bothered by **book tags**, then these patches will let you easily see which mod is associated with *that* 'Note.'
  
  
VBT uses SkyHorizon's [Dynamic String Distributor (DSD)](https://www.nexusmods.com/skyrimspecialedition/mods/107676) to add tags to books from various mods. VBT functions like a DSD translation, using DSD to dynamically replace text without the risks associated with editing plugins/ESPs. This allows VBT to be **compatible with everything**.
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/63625/63625-1676996774-1699634888.png)

  
Inspired by Kevkas's [Better Skill and Quest Books Names](https://www.nexusmods.com/skyrimspecialedition/mods/5260)﻿ and olioster's [Better Skill and Quest Books Names - Consistency Patches](https://www.nexusmods.com/skyrimspecialedition/mods/32721), these patches similarly tag associated skill bonuses for both vanilla and mod-added books, **for all tagged mods**. The **Lite** version of VBT only tags vanilla/CC content, for similar functionality as the two above, but with greater compatibility due to [Dynamic String Distributor (DSD)](https://www.nexusmods.com/skyrimspecialedition/mods/107676).

[font=Arial]**Installation**[/font]
  
Install like any other mod. Requires [Dynamic String Distributor (DSD)](https://www.nexusmods.com/skyrimspecialedition/mods/107676) and [Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444).
  
  
**[font=Verdana]✧[/font]** **Can be placed anywhere in load order.**
  
  
  
[font=Arial]**Compatibility**[/font]
  
Compatible with other [Dynamic String Distributor (DSD)](https://www.nexusmods.com/skyrimspecialedition/mods/107676) mods. More information:SpoilerVBT has a low filename priority, so that other DSD mods take precedence (if conflicting). In cases of conflict, other DSD mods will successfully overwrite changes from VBT. At the time of writing, I'm unaware of any other English-based DSD mod that might possibly conflict.
  
  
VBT functions like a DSD translation. It may conflict with non-English DSD translations, in which case, those non-English DSD translations should 'win' the conflict, and their non-English changes should successfully be presented in-game (overwriting VBT's tags).
  
**[font=Verdana]✧[/font]** **Otherwise compatible with everything.** No performance impact.
  
  
[font=Tahoma]⧉[/font] **Updates** — *How do updates work?*
  
[font=Tahoma]✧[/font] With [DSD](https://www.nexusmods.com/skyrimspecialedition/mods/107676), older VBT patches are now **always safe** to use. [DSD](https://www.nexusmods.com/skyrimspecialedition/mods/107676) prevents game-breaking issues and other typical consequences, even if mods alter or update underlying book mechanics/scripts. However, in rare cases, mod updates might change or totally revise book titles, leading to cosmetic inconsistencies.
