- Author: AndrealletiusVIII

**Introduction**
  
  
In the past, I made [a mod](https://www.nexusmods.com/skyrim/mods/68205) that adressing woodchopping blocks to no longer be an infinite supply of firewood.
  
  
In the meantime, I found some better ways to actually adress this. That's where this mod comes in.
  
  
**What does mod does**
  
  
Woodchopping blocks are now crafting stations, which can be used to turn tree logs into firewood.
  
  
Tree logs can be gathered by "harvesting" a tree in the world, while having at least one woodchopping axe in your inventory.
  
  
This will encourage you to actually go out into world, rather than just "generating" an unlimited amount of firewood at a woodchopping block.
  
  
**Compatibility**
  
  
Any mod that makes changes to trees, or woodchopping blocks, will conflict with this mod. For this reason, I've added an optional file called "override patch" which can be used to be loaded near the end of the load order, to make sure this mod wins the conflict.
  
  
I've added a patch for [Skyrim Flora Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/2154).
  
  
As far as I know, some of the more popular mods that conflict might include [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667), [Frostfall](https://www.nexusmods.com/skyrimspecialedition/mods/671), [CCOR (Complete Crafting Overhaul Remastered)](https://www.nexusmods.com/skyrimspecialedition/mods/28608) and [CACO (Complete Alchemy and Cooking Overhaul)](https://www.nexusmods.com/skyrimspecialedition/mods/19924). There could be others as well though.
  
  
Mods like [Enhanced Dynamic Things](https://www.nexusmods.com/skyrimspecialedition/mods/19521) might need a compatibility/consistency patch.
  
  
**[Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667) Patch**
  
  
In the optional files, there's a patch for [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667). This patch always you to craft a portable wood chopping block, using the "Create Item" from campfire. It requires 4 pieces of Firewood and 1 Treelog. You can place it in the world, like other items from Campfire. To use it, you will need to have a Woodcutter's Axe in your inventory.
  
  
[Immersive Interactions - Animated Actions](https://www.nexusmods.com/skyrimspecialedition/mods/47670): "harvest ingredients" animation conflicts. Disable its "harvest ingredients" in MCM.Â 
  
  
If you're having issues still getting firewood automatically, even if the crafting menu has opened, [GiraPomba](https://www.nexusmods.com/users/20507499) has made an OAR patch for that. You can find it in the optional files.
