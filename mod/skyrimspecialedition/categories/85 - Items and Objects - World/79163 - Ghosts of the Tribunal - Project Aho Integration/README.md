# Ghosts of the Tribunal - Project Aho Integration
- Author: Decaroidea
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79163


**This mod simply distributes the clothing sets found in the creation to the population of Sadrith Kegran in a manner that makes sense to the npc's standing and economy to bring diversity among the Dunmer citizens, using SPID.**
  
  
-ESL-Flagged
  
-Spid to minimize compatibility issues
