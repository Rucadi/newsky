# Civil War Champions - Reduced Cut
- Author: Kamikaze
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/94999


**DESCRIPTION**
  
  
It doesn't make sense for armies to fight duels let alone do it with highly enchanted one-of-a-kind armor. Now you'll receive champion's armor once you have proven yourself and just in time to rub it in Tullius' or Ulfric's face.
  
  
You should receive a miscellaneous quest to pick up the champion's armor shortly after starting either quest "Battle for Solitude" or "Battle for Windhelm" depending on which side you are on.
  
  
If that does not happen, enter the following in the console to start the quest manually:
  
  

```
setstage cwcrquest 10
```

  
  
Do not install if the quest "Battle of Champions" is already running.
  
  
**COMPATIBILITY**
  
  
Should be compatible with anything. The stats of the armors and weapons are not changed.
