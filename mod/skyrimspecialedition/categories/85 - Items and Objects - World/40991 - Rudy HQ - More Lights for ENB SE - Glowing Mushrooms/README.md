**Straight from the Heart Quality - more light sources for the ENB users (sorry but only for them). This mod utilizes a new "ENB Particle and Fire Lights" feature. Requires 0.365 binaries or newer.**
  
  
  
**Adds a light sources to Glowing Mushrooms, HD mesh and new textures.**
