# Saint Jiub's Bookmarks
- Author: shazdeh2
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/165897


Adds visual indicators (swirling papers) to the locations of Saint Jiub's Opus, to help find them easier. They are located where the pages are, or at the entrance to their location.
  
These go away after you pick up the page.
  
  
  
**Compatibility**
  
Load this before environment / lighting mods that affect the Soul Cairn.
  
File is ESP-FE.
