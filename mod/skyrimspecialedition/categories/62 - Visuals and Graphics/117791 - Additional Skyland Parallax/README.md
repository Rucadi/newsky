# Additional Skyland Parallax
- Author: Leostevano - Yuril
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/117791


Some extra parallax textures for Skyland's architectures.
  
  
- Farmhouses
  
- High Hrothgar
  
- Markarth
  
- Riften
  
- Solitude
  
- Windhelm
  
- Winterhold
  
- Sky Haven Temple
  
- Nordic Ruins
  
- Dwemer Ruins
  
- Imperial Forts
  
- Dawnguard DLC
  
- Dragonborn DLC
  
  
I only provide \_p textures, so [Skyland AIO](https://www.nexusmods.com/skyrimspecialedition/mods/34179) is required.
  
I don't provide parallax mesh, I recommend to get parallax mesh from mesh fixes mods, like [FYX Series](https://next.nexusmods.com/profile/Yuril/mods?_gl=1*1irwsvp*_ga*MjAzMjM2Njc5OS4xNjc4MzQ1MDkw*_ga_N0TELNQ37M*MTcxNDYyNjgwNi4yOTc4LjEuMTcxNDYyNzEwMy4wLjAuMA..) and [Assorted Mesh Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/32117)Lighting mods like [Lux (interior)](https://www.nexusmods.com/skyrimspecialedition/mods/43158) and [Lux Orbis (exterior)](https://www.nexusmods.com/skyrimspecialedition/mods/56095) also provides parallax mesh.﻿
  
You can also use [Nif Batch Scripts](https://www.nexusmods.com/skyrimspecialedition/mods/33846) to make your own parallax mesh, or you can use [Auto Parallax](https://www.nexusmods.com/skyrimspecialedition/mods/79473) and set **bAutoEnableParallax = true** on the INI.
