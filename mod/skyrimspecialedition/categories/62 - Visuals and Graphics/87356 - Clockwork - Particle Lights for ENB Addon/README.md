Support me on [Patreon](http://patreon.com/AgentW)﻿or [Buy Me a Coffee](https://www.buymeacoffee.com/agentW)

**Description:**﻿ ﻿This mod adds ENB particle light to meshes added by [Clockwork](https://www.nexusmods.com/skyrimspecialedition/mods/4155) (chandeliers and lamps). Also comes with an esp to remove fake glows that are replaced by this mod's particle lights (This esp isn't needed if you use Lux - Clockwork patch).
  
  
**Requirement:**- [Clockwork](https://www.nexusmods.com/skyrimspecialedition/mods/4155)
  
- ENB with particle light enabled
  
  
**Compatibility:** ﻿Don't use the esp if you already use Lux - Clockwork patch as the changes are already included. Overwrite meshes from [Additional Clockwork](https://www.nexusmods.com/skyrimspecialedition/mods/47087).
