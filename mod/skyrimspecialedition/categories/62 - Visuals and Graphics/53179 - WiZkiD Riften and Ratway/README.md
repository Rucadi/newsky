﻿[font=Georgia]The main purpose of this mod is to give Riften the relistic look of a true medieval port town. Particular attention has been given to the consistency both as regards the various textures used and to the area to which the city itself belongs. Obviously, the Ratway which is an integral part of Riften has also been given the same attention, everything concerning these undergrounds has been taken care of in the smallest details, giving it the typically damp and worn appearance of an underground sewer. The meshes used have numerous improvements including a better UVmap and cubemaps on metallic surfaces. This mod comes in four different versions, 4k and 2k parallax and 4k and 2k non parallax.
  
  
Note: Windows aren't included, use [Rustic Windows](https://www.nexusmods.com/skyrimspecialedition/mods/1937), it's simply the best window mod on Nexus.﻿
  
If you are using [Seasons of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/62861)﻿ it would be better to chose the "non parallax version".
