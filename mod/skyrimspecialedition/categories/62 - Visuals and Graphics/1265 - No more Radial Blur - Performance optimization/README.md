# No more Radial Blur - Performance optimization
- Author: Gorgulla
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/1265


![](https://staticdelivery.nexusmods.com/mods/110/images/77791-0-1470837319.png)   
   
[font=Trebuchet MS] **This mod removes all radial blur, including from deathcam, killcam or anything. See this mod as an all in one radial blur remover.**[/font]
  
   
   
**Many have tried to remove the radial blur because of:**    

  
* Performance
Quality
  
* Personal views
Bugs
  
* Clean killcam without blur

  
   
By Removing it by changing ini settings, the radial blur is still there because the only options is to choose quality.
  
The bDoRadialBlur=0 doesn't work and by adding it to skyrim.ini the game ctd's.
  
   
This mod fixes it by completely erase it from the game by changing effects like the onhit effect.
  
   
This is the only mod that changes the effects from all dlc's and also contain fixes from USLEEP.
  
There is another mod that does partly the same thing but it also affects eye adaption and could therefore conflict with lightning and weather mods.
  
   
   
**This mod will also remove the bug:**    
Permanent radial blur screen some users have.
  
   
   
**What you will notice:**   

  
* Increased performance
Cleaner view
  
* Smoother combat with less stutters

  
**Savegame compatible and clean:**   
The mod is savegame compatible and is also safe to uninstall.
  
**This mod is completely script free.**   
   
**Compatibility:**   
This mod removes all the radial blur, that includes the blur from killcams, so if you have a mod that removes the blur from killcams, remove that mod or mine.
  
   
**Reported to be compatible so far:**   

  
* ELFX
ELE   
* Vivid Weathers
Dynavision    
* RLO
