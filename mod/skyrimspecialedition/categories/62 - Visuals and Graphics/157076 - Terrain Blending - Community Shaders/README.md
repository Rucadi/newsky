[font=Trebuchet MS]**OVERVIEW**[/font]
  
  
Community Shaders addon which makes terrain realistically blend into objects
  
  
[font=Trebuchet MS]**FEATURES**[/font]
  
  

  
* This feature is based on an effect from Deus Ex Mankind Divided and other games such as Breath Of The Wild, which instead of using more advanced methods of blending terrain, it is done in screenspace which maximises performance and is highly compatible.
This feature uses many optimisations to achieve the best possible performance of such a technique, such as z-prepass, stochastic blending and automatically disabling blending for far away terrain.
  
* Uses same fix as [ENB Terrain Blending Fix](https://www.nexusmods.com/skyrimspecialedition/mods/140041)

Terrain Blending seamlessly blends terrain objects with the terrain itself:
