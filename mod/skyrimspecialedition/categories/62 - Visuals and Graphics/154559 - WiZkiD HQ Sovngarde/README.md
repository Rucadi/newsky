[font=Tahoma]**WiZkiD HQ Sovngarde is my new [b]texture overhaul of all Sovngarde related models.**This mod contains a custom version of [**Sovngarde Meshes - FIXES**](https://www.nexusmods.com/skyrimspecialedition/mods/150539)﻿, so you don't need the aforementioned mod.
  
  
  
  
Installation:
  
  
Installation is very simple and happens via a convenient fomod. Choose the version you prefer, 4k or 2k, and make sure that all WiZkiD HQ Sovngarde assets aren't overwritten by anything else in your load order. I strongly recommend using the parallax version, it's more than enough to get the best results from my textures.
