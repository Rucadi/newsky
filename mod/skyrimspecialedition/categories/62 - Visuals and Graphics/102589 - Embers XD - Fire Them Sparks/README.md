- Author: Mindflux - WeldingManDE

**[b][size=5]**Embers XD - Fire Them Sparks
  
[/size][/b][Embers XD](https://www.nexusmods.com/skyrimspecialedition/mods/37085)﻿﻿**﻿** **is required!**
  
  
**Works great with** **[FYX - Sparks of Fire reacts to the Wind](https://www.nexusmods.com/skyrimspecialedition/mods/102099)﻿**
  
A simple mesh replacer for EmbersXD to give all fire meshes the ability to cast some awesome sparks.
  
There is might some clipping with cooking pots now, maybe I will add a BOS version later to swap them.
