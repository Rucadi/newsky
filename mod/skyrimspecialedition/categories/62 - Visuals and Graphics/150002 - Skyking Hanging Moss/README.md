# Skyking Hanging Moss
- Author: Skyking2020
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150002


**![](https://staticdelivery.nexusmods.com/mods/1704/images/150002/150002-1747159705-310946133.jpeg)
  
﻿--**
  
  
Replaces the textures for the hanging moss throughout Skyrim and Solstheim. These textures are offered in 1k, 2k, and 4k versions.

  
  
**Compatability** - Everything as long as the vanilla moss meshes are not changed
  
  
**Install** - Install with your mod manager and place below mods that retexture moss. If you have mods that replace the moss meshes and textures, this mod may not work for you.
  
  
**Thanks** to those on the discord who tested and thanks to *cozysienna* for the idea.
