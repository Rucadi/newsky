OVERVIEW[/size][/b][/color]
  
  
**SKSE plugin which makes local maps use the full screen resolution.**
  
  
This mod doesn't have any customisation, it just sets the maps to the correct resolution automatically.
  
This will impact performance, although the local map is much more simplified than normal rendering.
  
  
![](https://i.imgur.com/YlFbONu.gif)
