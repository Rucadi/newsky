# Particle Lights for ENB - Shellbug
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/88919
This creature has a glow map and a glow flag for its eyes.
  
So, I believe it deserves the ENB Particle Lights treatment :)
  
  
Check the images!! :O
  
  
[size=4]**INSTALLATION:**
  
1) Install the main file manually or using any mod manager. OR...
  
 ﻿ ﻿2) Install one of the patches depending on the mod you use.
  
  
3) Be sure that you have this settings in enbseries.ini file:
  
[EFFECT]
  
EnableComplexParticleLights=true
  
  
[COMPLEXPARTICLELIGHTS]
  
EnableBigRange=true
  
  
  
[size=4][size=4]**REQUIREMENTS:**
  
-An ENB preset to enable the particle lights effect.
  
  
**COMPATIBILITY:**
  
-Compatible with [HD Reworked Shellbug](https://www.nexusmods.com/skyrimspecialedition/mods/41191); patch provided under optional downloads.
  
-Compatible with [Shellbug Overhaul- Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/81529); patch provided under optional downloads.[/size][/size]
