# Ambient Templates for Lighting Mods
- Author: Dlizzio - isoprophlex - minizbot2012
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153425


[![](https://i.postimg.cc/prDW7GS0/THE-banner-mod-page.gif)](https://discord.gg/V6qa82Spxw)
  
  
*Neutralizes ambient light colors and reduces fog intensity in interiors, offering 8 brightness options for a balanced ambient lighting experience. Designed for a modular interior lighting setup, utilizing SkyPatcher for conflict-free cell modification.*
  
  
  
**Required Mods:**
  

  
* [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659)﻿
