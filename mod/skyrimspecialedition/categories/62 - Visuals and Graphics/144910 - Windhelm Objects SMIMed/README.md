This mod tries to improve lots models in the Windhelm city that are too sharp and make them smother. It also add tons of fixes about holes, wrong uv maps and bad normal map for the snow shader.[/font]

![](https://i.ibb.co/BjxGLtM/Wi-Zki-D-Features.png)

  

  
* [font=Arial]Reshape (and heavy fix) some objects to make them smoother;[/font]
[font=Arial]UV map fix;[/font]
  
* [font=Arial]Normal height fix to avoid incorrect application of snow shader;[/font]
[font=Arial]No esp;[/font]
  
* [font=Arial]No data or cell modify;[/font]
[font=Arial]No requirement;[/font]
  
* [font=Arial]No real incompatibility.[/font]
