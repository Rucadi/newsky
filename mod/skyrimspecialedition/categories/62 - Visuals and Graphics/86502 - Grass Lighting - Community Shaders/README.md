[font=Trebuchet MS]**OVERVIEW**[/font]
  
  
Community Shaders feature which replaces grass shaders to add more advanced shading including Complex Grass support.
  
  
[font=Trebuchet MS]**FEATURES**[/font]
  
  
[font=Trebuchet MS]***MORE ADVANCED GRASS SHADING***[/font]
  
  
Shades grass using vertex normals, adding directionality to lighting. Adds subsurface scattering to the backside of grass.
  
  
[font=Trebuchet MS]***COMPLEX GRASS***[/font]
  
  
Fixes UV-mapping of complex grass made for ENB. Includes normal and specular map support. ï»¿Based on reverse-engineered game object lighting code. ENB complex grass settings are not the same.
  
  
[font=Trebuchet MS]**INSTALLATION & REQUIREMENTS**[/font]
  
  

  
1. Install [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492) and all it's requirements.
Install this mod below it.
  
2. Confirm whether or not your grass mod supports complex grass. You may need to install a complex grass update from [Vanilla Complex Grasses for ENB](https://www.nexusmods.com/skyrimspecialedition/mods/68640) (also works for CS), [Complex Grass - The Official Patch Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/67304), or another source.
