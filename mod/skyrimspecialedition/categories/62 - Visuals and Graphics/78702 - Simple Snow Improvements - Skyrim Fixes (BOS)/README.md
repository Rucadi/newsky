- Author: WeldingManDE

**[b][size=5]﻿Simple Snow Improvement**s - Skyrim (BOS)
  
Base Object Swapper required![/size][/b]
  
  
A simple mod to improve the snow visuals on forgotten vanilla objects with Base Object Swapper support.
  
  
[color=#ffffff]**The mod will grow over time, feel free to report some more vanilla objects.**[/color]
  
When reporting a new object, please tell me the location and also post a screenshot with the EditorID by clicking on the relevant object
  
with [More Informative Console.](https://www.nexusmods.com/skyrimspecialedition/mods/19250)
  
  
"Vanilla / Simplicity of Snow / Better Dynamic Snow 2 & 3" version available.
