# Vampire Lord Retexture
- Author: Avike
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/8402


Vampire lord retextured into a more demonic form.
  
  
Texture dimensions are 2k(same as vanilla).
  
  
Note - this does removes the 'bra' worn by the vanilla female vampire lord.
  
  
**Compatibility
  
*-* [*Vampire Lord Armor of The King*](http://www.nexusmods.com/skyrimspecialedition/mods/8401?)
  
  
-** ***[Vampire Lord Royal Armor](http://www.nexusmods.com/skyrimspecialedition/mods/3602/?)***
  
**Install/Uninstall**
  
- Install via NMM, or extract data folder to Skyrim root directory and merge/overwrite if necessary.
  
To uninstall manually delete the 'vampirelord' folder from Data>textures>actors
