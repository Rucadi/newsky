# Skyking Imperial Forts and Dungeons
- Author: Skyking2020
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/164942


![](https://staticdelivery.nexusmods.com/mods/1704/images/164942/164942-1763575656-1982968223.jpg)
  
  

**A complete texture overhaul of Imperial forts and dungeons. Featuring 1k, 2k, and 4k options along with complex parallax materials.
  
  
*Don't forget to endorse if you enjoy this mod.***

  
  
**Features:**

  
* 1k, 2k, and 4k texture options
New debris meshes with new debris textures
  
* Complex parallax option
Covers all imperial forts and dungeons
  
* Includes new textures for stockades
Brown and grey wood options for forts and stockades
  

**Install:**
  
Download either manually or with mod manager and install as you would any other mod. Place in your load order where you put new textures and meshes and place after any mods that supply the same textures and meshes thus ensuring it is not overwritten by other mods. Run PG Patcher after installation if you are using the complex parallax option.
  
  
**Compatibility:**
  
Compatible with anything that doesn't change the texture file paths of imperial fort meshes.
  
  
**Parallax:**
  
When installing this mod, select the parallax option and then run Parallax Gen. Ensure your ENB is set up correctly to enable complex parallax. This mod and it's comment section is not a help section or guide on setting up parallax. This mod assumes you already have parallax set up or know how to do it.
