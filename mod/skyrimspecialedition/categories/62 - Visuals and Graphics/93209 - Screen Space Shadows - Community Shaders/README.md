[font=Trebuchet MS]**OVERVIEW**[/font]
  
  
Community Shaders feature which adds screen-space shadows, enabling grass, map, distant and self-shadows.
  
  
This is a custom implementation of [Inside Bend: Screen Space Shadows - Bend Studio](https://www.bendstudio.com/blog/inside-bend-screen-space-shadows/) with visual improvements designed for Skyrim's world.
  
  
**This can be relatively performance intensive compared to other features.**
  
  
[font=Trebuchet MS]**INSTALLATION & REQUIREMENTS**[/font]
  
  

  
1. Install [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492) and all it's requirements.
Install this mod below it.
