# Dynamic Cubemaps - Metals
- Author: doodlez
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/114119


**OVERVIEW****Cubemap replacer for vanilla and Modular Armory to add support for Dynamic Cubemaps.

All of the included textures were designed in-game with the Dynamic Cubemap Creator tool found in Community Shaders.
  
  
There are varying levels of blurryness for different metals. In the case of glass, it almost only has fresnel, so looks like a transparent surface, 
  
  
  
**REQUIREMENTS**
  
[Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492)
  
[Dynamic Cubemaps](https://www.nexusmods.com/skyrimspecialedition/mods/112763)
