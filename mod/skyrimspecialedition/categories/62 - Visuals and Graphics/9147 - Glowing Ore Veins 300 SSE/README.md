# Glowing Ore Veins 300 SSE
- Author: Freakenstein
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/9147
**ByblosHex**'s  original file here [Glowing Ore Veins 300](http://nexusmods.com/skyrim/mods/193/?) ported over to SSE, full credit goes to him for this awesome mod and I have his permission to upload this.
  
   
This has always been my go to mod for mining in Skyrim, I was disappointed none had ported it yet, so decided to try converting it for myself and figured I'd share it with others. I'm not supporting this as no one should have a problem with this file unless your using another ore texture, in that case just place this file after the file over writing it!
