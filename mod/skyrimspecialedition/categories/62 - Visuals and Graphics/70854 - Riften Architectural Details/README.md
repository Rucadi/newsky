Whenever I walk into Riften, I always wonder: why the hell are the log houses so low poly? I looked at the pole in front of the "Bee and The Barb" and noticed that it was an octagon shaped polygon. So I thought: what the hell is Bethesda doing? Why are there buildings in the city with an incredibly high number of polygons, yet these look like stop signs? Well, thanks to this mod, all that will change. [size=3][size=3]The goal is to change the current, horrible Riften meshes that show as low poly into circular meshes.[/size]
  
[/size]**But that's not all...** During my editing I noticed hundreds of errors, holes, incorrect UV maps, etc., so this a *high poly* mod as well as a bug fix compilation for Riften.
  
[/size]
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/50133/50133-1629449291-1489053992.png)

  
  
Being extremely lore friendly, this mod is compatible with just about anything. A patch for [Flickering Meshes Fix](https://www.nexusmods.com/skyrimspecialedition/mods/53957) can be found on its page (version 1.9 or newer). This mod has already incorporated the changes made by particle patch.
  
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/50133/50133-1629445917-1935787738.png)

  

The mod only includes meshes, so install it as you prefer (mod manager or manually).
