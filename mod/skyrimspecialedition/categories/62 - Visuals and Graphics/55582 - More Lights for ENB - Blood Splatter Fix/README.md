 **Description**
  

  
* **Rudy HQ - More Lights for ENB SE** mods add very nice looking ENB particle lights to glowing objects. However, the light meshes are not flagged correctly, so blood splatter decals are erroneously applied to them, and you get these floating blood splatters where shouldn't be any. This mod fixes that
As of v0.5+, I've removed the centering fix in the nif. [ENB Light Inventory Fix (ELIF)](https://www.nexusmods.com/skyrimspecialedition/mods/66411) is now a requirement.
