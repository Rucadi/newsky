**[font=Comic Sans MS]Adds footprints to the player, NPCs, and creatures to The Soul Cairn location.[/font]**

  

**[font=Comic Sans MS]Comes in two versions namely Default and No Grass. [/font]**

  

![](https://i.postimg.cc/cJK8fMBL/Skyrim-Special-Edition-8-1-2025-22-58-11.jpg)

  
  

**[font=Comic Sans MS]< Installation >[/font]**

  

**[font=Comic Sans MS]Install with favorite mod manager and place plugin right after Footprints.esp[/font]**

  

**[font=Comic Sans MS]Pick one or the other downloadable options. Do not use both.[/font]**

  

**[font=Comic Sans MS]SPID file available that disables footprints for ghostly humanoids and cows present in The Soul Cairn. [Ultimate fix - SPID for Footprints (and kids addon)](https://www.nexusmods.com/skyrimspecialedition/mods/96086) required. Let my SPID file overwrite the one from said mod.[/font]**
