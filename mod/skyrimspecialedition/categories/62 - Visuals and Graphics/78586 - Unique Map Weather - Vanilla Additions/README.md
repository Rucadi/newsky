**[b][b]**[/b][/b]
  
  
[Enhanced Worldspace Maps](https://www.nexusmods.com/skyrimspecialedition/mods/57176) is **recommended**.
  
  
The addon adds unique weather to Solsteim from the Dragonborn DLC, which makes it reflective of the ashy climate. 
  
Unique weather is also added to the Soul Cairn from the Dawnguard DLC, which fixes colours and time-of-day using mods such as:
  
  

  
* [Dawnguard Map Markers](https://www.afkmods.com/index.php?/files/file/1896-dawnguard-map-markers/)
[Atlas Map Markers](https://www.nexusmods.com/skyrimspecialedition/mods/24104).
  

  
Alternatively, you can use [A Clear Map of Skyrim and Other Worlds.](https://www.nexusmods.com/skyrimspecialedition/mods/56367?tab=description)
