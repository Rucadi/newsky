**Description**
  

  
* Adds Ayleid lock ([The Cause](https://en.uesp.net/wiki/Skyrim:The_Cause)﻿, [Beyond Skyrim - Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/10917)﻿)
Adds Nordic lock
  
* Adds Soul Cairn lock (a mod, such as [Soul-Cairn Objects Secured](https://www.nexusmods.com/skyrimspecialedition/mods/59741), is needed, as unmodded Soul Cairn has no locked doors or chests)
Adds Cold Harbor lock ([Vigilant](https://www.nexusmods.com/skyrimspecialedition/mods/11894)﻿)
  
* Adds [Tension Cable](https://en.uesp.net/wiki/Skyrim:Traps#Tension_Cable)﻿ lock
Adds Underwater default lock

**FAQ**
  
> Q: Can I install, uninstall, upgrade this mid playthrough?
>   
> A: Yes
>   
>   
> Q: What is required for this mod?
>   
> A: [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) if you're playing on Skyrim Special Edition ("Anniversay Edition" is still just SSE).
>   
>     [﻿Security Overhaul SKSEVR - Lock Variations VR](https://www.nexusmods.com/skyrimspecialedition/mods/58298) if you're playing on VR
>   
>   
> Q: Does this work with the "Anniversary Edition update" (v1.6)?
>   
> A: Yes, as of [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) version 2.0
>   
>   
> Q: Does this work with Special Edition (v1.5.97)?
>   
> A: Yes
>   
>   
> Q: Does this work for VR?
>   
> A: Yes, if you've installed the VR requirement: [Security Overhaul SKSEVR - Lock Variations VR](https://www.nexusmods.com/skyrimspecialedition/mods/58298)
>   
> ﻿﻿﻿Q: Does this work with <popular lock replacer mod> as well?
>   
> A: Yes
>   
>   
> Q: Mod isn't working even though I have installed all the requirements?
>   
> A: Uninstall [Wearable Lanterns MCM Fix](https://www.nexusmods.com/skyrimspecialedition/mods/25372?). Wearable Lanterns MCM Fix included an LE/Oldrim version of StorageUtil.dll which will completely prevent SKSE from loading any dll plugins.
