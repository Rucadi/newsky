> **This asset pack for the [Dismembering Framework](https://www.nexusmods.com/skyrimspecialedition/mods/126203) allows for dismemberment of all types of humanoid NPCs** (in other words, all playable races). It is one of the two main packs, aiming to eventually **cover all assets from the base game and its DLCs**. With this pack, **you can dismember arms, legs, heads, and even tails** for the beast races.
>   
>   
> If a dismembered NPC is wearing a piece of base game armor, the immersion will be complete, as you will see a perfect visual replica of the severed limb being flung into the air. If it's not a base game armor, no worries— a **generic asset** of the same class as the severed one will be generated, and the modded armor will still be cut at the point of impact.
>   
>   
> **To summarize, all armor is compatible with the framework once this pack is installed.**

This pack is **intended to be completed over time to cover the entirety of the game and its DLCs**. If you encounter a case where a generated asset does not match a base game armor part, please let me know. With over 700 variants currently, it is possible that there may be one or two discrepancies.
  
  
As for decapitations, the framework uses the game's original functionality, so they should theoretically be compatible with all types of armor. However, you might still encounter the usual visual glitches, especially if you're using mods for hairstyles or other facial assets.

![](https://www.seb263.fr/mods/skyrim/_shared/chip_01.svg?height=40)![](https://www.seb263.fr/mods/skyrim/_shared/installation.svg?height=40)![](https://www.seb263.fr/mods/skyrim/_shared/chip_01_r.svg?height=40)

  
  
The mod can be installed like any other mod via a mod manager or manually.
  
⚠ **Don't forget to install and activate the [Dismembering Framework](https://www.nexusmods.com/skyrimspecialedition/mods/126203).**
  
  
Manual Installation :
  
Spoiler1. Download the latest version of the mod.
  
2. Open the archive using 7zip or Winrar.
  
3. Place the entire contents of the archive into the "Data" folder of your game.
  
(Default: C:\Program Files\Steam\Steamapps\common\Skyrim Special Edition\Data)
  
4. Activate "DF - Official Humanoid Pack.esp" from the game's mod manager (or any other mod manager).
  
5. Launch Skyrim and envoy!
