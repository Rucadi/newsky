# VioLens Compatibility Extension - Mod Creatures KillMove Compatible - SkyPatcher
- Author: Mokeine
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/127117


[font=Comic Sans MS]Killmoves triggers depends on target race( except on NPCs ). So you're unable to trigger killmoves on mod-add race creatures.This mod make killmoves able to trigger on creatures in other mods when you use [Violens](https://www.nexusmods.com/skyrimspecialedition/mods/668).
  
  
This is an upgrade version of [VioLens Compatibility Extension](https://www.nexusmods.com/skyrimspecialedition/mods/67973). Use the power of SkyPatcher to automatically detect creature's skeleton path, so you don't need to patch for your every mod, and it's more accurate, reducing incorrect pairing animations due to manual spell errors.
  
  
The only exception is when a creature uses a custom skeleton path, for example mihail mods. I have built-in compatibility of some
  
mihail mods, Zombies from [Creepy Zombies](https://www.nexusmods.com/skyrimspecialedition/mods/6044)﻿ or [/font][font=Comic Sans MS][Skyrim Underground](https://www.nexusmods.com/skyrimspecialedition/mods/131), and [Skyrim Extended Cut - Saints and Seducers](https://www.nexusmods.com/skyrimspecialedition/mods/72772)﻿.
