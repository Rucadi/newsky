**Description**
  
**My patches for this amazing mod.** (All patches are fine to install/uninstall at any time.)
  

  
* **Mandragorasprouts Draugrs**. **Dry Blood/Fresh version** available, install only one.
**Mandragorasprouts Falmer Overhaul**.
  
* **Dry Blood/Flesh Texture Patch for the vanilla Draugr** included in the original DF - Creature Asset Pack. It will make the bone whiter, and remove the specular bright on the dried flesh/blood.
**Creation Plague of the Dead**. The patch will affect the 3 zombie types added by it. ESP flagged as ESL. **Do not use it together with the Mihail's CC Zombies Overhaul patch.**
  
* **Mandragorasprouts Hulking Draugr** from the Dragonborn DLC. **Dry Blood/Fresh version** available, install only one.
**Vanilla Hulking Draugr** from the Dragonborn DLC.
  
* **Mihail's CC Zombies Overhaul (SE's Creation Club- AE)**. Two versions available. **Without resurrection** and **with resurrection** (as intended in the original mod). In the No Resurrection version I decided to remove the auto-revive effect of the Zombies added by this mod, as it looks weird after they were dismembered. Install only one. ESP flagged as ESL. **Do not use it together with the Creation Plague of the Dead patch**.
**Creation Goblins - Blue God**. Fully compatible with Mihail's CC Goblins Overhaul (SE's Creation Club- AE).
  
* **CC Bone Wolf.** Featuring similar bone dust effects than dry Draugr.
**Simply Better Wolves**.
  
* **Wolves of Skyrim**.
**Trolls SE** and**My Armored Version of 4thUnknown's Trolls SE.**
  
* **Mihail's Wolves of Cyrodiil.** **Default** and **Fluffy** versions available. It also requires [Wolves of Cyrodiil - My optimized textures](https://www.nexusmods.com/skyrimspecialedition/mods/107913) ﻿and [Fluffy Wolves of Cyrodiil](https://www.nexusmods.com/skyrimspecialedition/mods/68835)﻿ (for the Fluffy version).
**Ogrims SE**. Only Feral Ogrims can be dismembered for now. Because they use the released Troll skeleton, other Ogrims use the Giant one. So I'll do the rest when the Giants are released on the Official Creature Asset Pack.
  
* **Daedra SE**. It includes **Feral Ogrims** so far. Do not use it together with the Ogrims SE patch.
**Fluffworks (Fluffy Animals)**. Trolls patched, so far.
  
* **Ancient Falmer Ruins (Snow Elf Dungeons)**. Requires my [Draugrs - SE by Xtudo - Mihail Undead Snow](https://www.nexusmods.com/skyrimspecialedition/mods/123225)﻿ installed first. Don't use together with the Undead Snow Elves patch, use the **COMBO** instead.
**Undead Snow Elves- Mihail Monsters and Animals (SE-AE version) (''falmer'')**. Requires my [Draugrs - SE by Xtudo - Mihail Undead Snow](https://www.nexusmods.com/skyrimspecialedition/mods/123225)﻿ installed first. Don't use together with the Ancient Falmer Ruins patch, use the **COMBO** instead.
  
* **Ancient Falmer Ruins - Undead Snow Elves- Mihail COMBO patch**. Install this version if you have the two mods installed. Requires my [Draugrs - SE by Xtudo - Mihail Undead Snow](https://www.nexusmods.com/skyrimspecialedition/mods/123225)﻿ installed first.
**[True Wolves Of Skyrim](https://www.nexusmods.com/skyrim/mods/59174) ﻿patch.** LE mod that works perfectly on SE-AE.
  
* **TNT - True Nordic Trolls - Revamped**.
**GRAHL - The Ice Troll** (Bloodmoon Creature Restoration Project).
  
* **Udefrykte. Uderfrykte. Udyrfrykte.** (Bloodmoon Creature Restoration Project).
