# Animation Ledge Block NG
- Author: MaskPlague
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148203


**Description**
  
This is a SKSE plugin that stops players (optionally NPCs) from attacking/dodging off of a cliff similar to Elden Ring.
  
Made with CommonLibSSE NG (tested on 1.5.97, 1.6.640, 1.6.1170).
