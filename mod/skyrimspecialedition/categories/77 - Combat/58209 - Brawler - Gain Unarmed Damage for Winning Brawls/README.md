# Brawler - Gain Unarmed Damage for Winning Brawls
- Author: Yeti
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/58209


Each time you win a brawl your unarmed damage goes up. It only goes up a little (2 points) but there's around 15 fights in the game so it adds up.
