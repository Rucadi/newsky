# Scriptless Stagger Limit - Prevent Staggerlock
- Author: MadAborModding
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148376


**Mod Overview**Adds a cooldown to stagger effects in order to prevent NPCs (and optionally the player) from becoming "stagger-locked".
  
There are other [mod(s)](https://www.nexusmods.com/skyrimspecialedition/mods/62554)﻿on the nexus that do this, but my implementation is scriptless.
  

TLDR: Once staggered, an actor cannot be staggered again (by default) for 10s.
  
(Options to change the stagger cooldown period are within FOMOD Installer).

  
  
**Compatibility**
  
Scriptless Stagger Limit has no vanilla record edits and is **compatible with all mods**. (Including other combat mods). 
  
  
**Requirements**
  
[Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)
  
This mod has two versions: A Scrambled Bugs Version and a **Non**-Scrambled Bugs Versions
  
  
The **Scrambled Bugs** version requires [Scrambled Bugs](https://www.nexusmods.com/skyrimspecialedition/mods/43532) with Apply Multiple Spell = true in ScrabledBugs.json.
  
  
The **Non-Scrambled Bugs** version removes the Scrambled Bugs dependency, requiring [Core Impact Framework (CIF)](https://www.nexusmods.com/skyrimspecialedition/mods/146873) instead. 
  
  
Which you use is up to you, the **performance and features of the two versions are identical**. The only reason to use the Non-Scrambled Bugs version would be if Scrambled Bugs is incompatible with one of your mods (i.e. Ordinator).
  
  
**Installation**
  
Simply install with your preferred mod manager. Position in Load Order is irrelevant. 
  
  
**Recommendations**Originally, I was going to add this as an option in my mod, [Engaging Combat](https://www.nexusmods.com/skyrimspecialedition/mods/132625), but felt it too subjective of an addition, hence why I am releasing it separately. Definitely also check out [Engaging Combat](https://www.nexusmods.com/skyrimspecialedition/mods/132625) if interested in increasing the difficulty of combat in the late-game.
