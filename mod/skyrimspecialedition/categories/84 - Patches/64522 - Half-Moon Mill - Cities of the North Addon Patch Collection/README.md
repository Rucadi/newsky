# Half-Moon Mill - Cities of the North Addon Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64522
Collection of patches for [Half-Moon Mill - Cities of the North Addon](https://www.nexusmods.com/skyrimspecialedition/mods/64360). All patches are ESP flagged as ESL and compatible with each other.
  
  
Details of patches:
  
[AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/21654) - shifts a few idle markers. Last checked against version 1.4
  
[At Your Own Pace: Thieves Guild](https://www.nexusmods.com/skyrimspecialedition/mods/52704) - shifts treasury box to match the new interior. Last checked against version 2.0.1TG
  
[Bowgasm SE](https://www.nexusmods.com/skyrimspecialedition/mods/42929) - moves a bow so it isn't floating, last checked against version 1.1
  
[Distinct Interiors](https://www.nexusmods.com/skyrimspecialedition/mods/6130) - applies mod additions to reworked interior, resolving conflicts and clipping, last checked against version 1.82
  
[Embers XD](https://www.nexusmods.com/skyrimspecialedition/mods/37085) - changes fireplace type and adjusts its position, last checked against version 2.5.4
  
[Enhanced Lights and FX](https://www.nexusmods.com/skyrimspecialedition/mods/2424) - adjusted lights changes to new interior layout, conflict resolution for ELFX - Exterior, last checked against version 3.06
  
[Farmhouse Chimneys](https://www.nexusmods.com/skyrimspecialedition/mods/8766) - adjusts chimney position to reworked house model, last checked against version 4.1.1
  
[Half Moon Creek](https://www.nexusmods.com/skyrimspecialedition/mods/42119) - conflict resolution on some records and navmesh, last checked against version 1.2
  
[Immersive Citizens - AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/173) - adjusts positions of Immersive Citizens markers correctly for a new interior, conflict resolution on persistency flags and navmesh, last checked against version 0.4
  
[Immersive Laundry](https://www.nexusmods.com/skyrimspecialedition/mods/2011) - shifts clothing lines to match the new exterior. Last checked against version 1.0d
  
[Lanterns of Skyrim II](https://www.nexusmods.com/skyrimspecialedition/mods/30817) - shifts two lanterns and lights to match the exterior changes, last checked against version 4.0
  
[Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095) - adjusts lantern and few lights position to match the exterior changes, last checked against version 2.5
  
[Medieval Lanterns of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/27622)﻿ - shifts one lantern and its light to match the new exterior. Last checked against version 1.0.5
  
[Radiance - Radiant Quest Enchancements - Locations](https://www.nexusmods.com/skyrimspecialedition/mods/45419) - conflict resolution, last checked against version 1.0.1.
  
[Red Flame](https://www.nexusmods.com/skyrimspecialedition/mods/11486) - shifts a placed book to match the new interior. Last checked against version 1.00
  
[Ryn's Lumber Mills](https://www.nexusmods.com/skyrimspecialedition/mods/115134) - resolves conflicts, adjusts navmesh. Last checked against version 1.0
  
[Skyrim Artwork Imports](https://www.nexusmods.com/skyrimspecialedition/mods/22472)﻿- moves the paintings and a tapestry to match the new interior. Last checked against version 1.0
  
[Skyrims Unique Treasures](https://www.nexusmods.com/skyrimspecialedition/mods/5261) - shifts a flute position to match the new interior, last checked against version 4.5
  
[Sounds of Skyrim Complete SE](https://www.nexusmods.com/skyrimspecialedition/mods/8286) - adjusts sound markers to new interior, forwards acoustic space changes, last checked against version 2.0.
  
[Tamrielic Culture](https://www.nexusmods.com/skyrimspecialedition/mods/11418) - shifts the placement of some clutter to match the new interior, last checked against version V2
  
  
Additional notes:
  
- patch for Sounds of Skyrim should be loaded as the last one and overwritten only by SoS compatibility patch for your light mod (f.e. Lux, ELFX)
  
- the following mods are checked and compatible without the need of a patch: Carriage and Ferry Travel Overhaul (CFTO), Cutting Room Floor, ELFX Fixes, Happy Little Trees, Interesting NPCs (3DNPC), Legacy of the Dragonborn BadGremlins Collection, Morning Fogs, Nature of the Wildlands, Skyrim Landscape and Water Fixes, Trees Addon and Creation Club - Fishing
  
- patch for Lux is now provided by [Lux - Patch Hub](https://www.nexusmods.com/skyrimspecialedition/mods/113002)
