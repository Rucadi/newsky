# SoulCairn - f(0)o_o(0)tprints patch
- Author: SiweX92
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/71222


It is simple edit to Soul Cairn landscape's textures to be recognize as a sand instead of dirt, so thanks to [Footprints sand patch](https://www.nexusmods.com/skyrimspecialedition/mods/50484) footprints will appear on those surface. You can copy those changes to whatever other plugin which edit danguard.esm, integrate it with mergePatch or just use as is. Plugin is flagged as ESL.
