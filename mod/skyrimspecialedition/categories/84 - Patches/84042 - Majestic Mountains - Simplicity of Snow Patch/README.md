# Majestic Mountains - Simplicity of Snow Patch
- Author: DarthBlender T4gtr34um3r wSkeever
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/84042
Changes the Static objects edited by MM to use Simplicity of Snow's snow shaders for consistency. Essentially the custom MM snow shaders are not used. Made for MM 4.02 and SOS 0.10.
  
  
  
**What this mod does NOT do:**
  

  
* Edit any other objects to have snow shader applied (use [this](https://www.nexusmods.com/skyrimspecialedition/mods/78702) ﻿mod)
Remove snow shader from objects
  
* Modify any snow shader (as of v1.1 vanilla version it reverts MM's LOD snow shader to vanilla values)
Edit anything other in MM esp except of Static
  
* Edit anything in SOS esp (as of 1.1 edits the original single pass shaders to brighter color for custom projecteddiffuse from [here](https://www.nexusmods.com/skyrimspecialedition/mods/57643)﻿)

**LOAD ORDER:**
  
Majestic Mountains.esp
  
Simplicity Of Snow.esp
  
SOS MM.esp
  
  
Does not affect any other esp/esm/esl in MM.
