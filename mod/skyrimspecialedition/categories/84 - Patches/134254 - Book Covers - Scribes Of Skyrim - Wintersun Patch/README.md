# Book Covers - Scribes Of Skyrim - Wintersun Patch
- Author: HermitaeusMora
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/134254


This patch enables [Wintersun](https://www.nexusmods.com/skyrimspecialedition/mods/22506) scripts that trigger faith changes when books are read for books modified by [Book Covers Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/901) and [Scribes of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/50008).
  
This patch merges Book Covers and fonts for these books.
  
  
Load in this order:
  
  
﻿Book Covers Skyrim.esp
  
ScribesofSkyrim.esp
  
﻿[book cover vs scribe patch.esp](https://www.nexusmods.com/skyrimspecialedition/mods/50244)Wintersun - Faiths of Skyrim﻿.esp
  
Book Covers + ScribesOfSkyrim + Wintersun.esp
  
  
  
Other Wintersun patches for Book Covers (such as [Wintersun - BCS Patch](https://www.nexusmods.com/skyrimspecialedition/mods/24228) or [Wintersun Book Covers](https://www.nexusmods.com/skyrimspecialedition/mods/70549)) are not needed.
