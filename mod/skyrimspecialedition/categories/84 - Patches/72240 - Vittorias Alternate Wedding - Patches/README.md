# Vittorias Alternate Wedding - Patches
- Author: Blackread
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72240


Patches for some mods to make them play nice with [Vittorias Alternate Wedding](https://www.nexusmods.com/skyrimspecialedition/mods/62466).
  
  
[size=4]**Patch Descriptions**:
  
  
[/size][AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/21654) - Updated the patch from the original mod page. Forwards packages on Aquillius Aeresius and resolves conflicts with vanilla packages both mods modify.
  
  
[At Your Own Pace - Thieves Guild](https://www.nexusmods.com/skyrimspecialedition/mods/52704) - Combines the edits both mods do to the quest ["The Dainty Sload"](https://elderscrolls.fandom.com/wiki/The_Dainty_Sload).
  
  
[Conversations Raise Speechcraft](https://www.nexusmods.com/skyrimspecialedition/mods/93435) - Conflict resolution.
  
  
[Dark Brotherhood Reformation](https://www.nexusmods.com/skyrimspecialedition/mods/56377) - Alternate wedding plays out after "Bound Until Death" if Vittoria is still alive.
  
  
[Guard Dialogue Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/22075) - [size=3]Conflict resolution on the conditions of one dialogue so changes from both mods go through.
  
  
[/size][Immersive World Encounters](https://www.nexusmods.com/skyrimspecialedition/mods/18330)Â  - Conflict resolution on two quests modified by both mods.
  
  
[Innocence Lost - Quest Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/80974) - Alternate wedding plays out if the player chooses not to kill Grelod.
  
  
[Open Cities Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/87707) - Moves the wedding to the open city space.
  
  
[Relationship Dialogue Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/1187) - Conflict resolution on one dialogue record so changes from both mods go through.
