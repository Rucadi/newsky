# Requiem - Unmarked Locations Pack (AIO)
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/131616
Balance and Compatibility patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Unmarked Locations Pack - All In One](https://www.nexusmods.com/skyrimspecialedition/mods/113660).
  
*- Rebalances the loot from some of the locations
  
- Reworked the Old Grave to have a blessing of Arkay instead of a new one to avoid compatibility issues
  
- Replaced some unique Skooma flasks with vanilla Skooma to avoid compatibility issues
  
- Rewrote some messages to better fit Requiem
  
- Rebalanced NPCs/Ingestibles/Misc Items/Spell Effects*
