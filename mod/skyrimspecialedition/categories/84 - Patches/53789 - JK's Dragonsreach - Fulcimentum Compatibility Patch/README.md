# JK's Dragonsreach - Fulcimentum Compatibility Patch
- Author: snipps98
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/53789


A very simple compatibility patch that moves the staff enchanter added by Fulcimentum to Farengar's lab in Dragonsreach to the other side of the room, so that it is compatible with JK's Dragonsreach Overhaul.
