# Requiem - Standalone Dungeon Mod Patches
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136700
Balance and Compatibility patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and various standalone dungeon mods.
  
  
  
*These patches can include several changes from compatibility to balance to other things such as text changes to fit seamlessly into Requiem.*
  
  
  
*List of Dungeon Mods:*
  
**- [Dragon's Teeth Prison](https://www.nexusmods.com/skyrimspecialedition/mods/111523)- [Ysmir's Tear II﻿](https://www.nexusmods.com/skyrimspecialedition/mods/133627)- ﻿[Taarengrav Burrow](https://www.nexusmods.com/skyrimspecialedition/mods/84371)**
