# JK's Windhelm Outskirts Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/87964
A collection of compatibility patches for [JK's Windhelm Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/86975). All patches are ESP flagged as ESL.
  
  
**Patches descriptions:**
  
  
[Birds of Skyrim SSE](https://www.dropbox.com/s/0zmfolkcxrpv5d0/Birds%20of%20Skyrim%20SSE%20Edition%202.7.7z?dl=0) - adjusts the placement of a few birds. Last checked against version 2.7 - the last one available for SE.[BUVARP SE RE](https://www.nexusmods.com/skyrimspecialedition/mods/56120) - adjusts landscape and navmesh near the Anska's home. Last checked against version 1.3.1
  
  
[CACO Farm Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/36283) - applies CFO additions to JK's farms' layout. Last changed against version 1.0.1.
  
  
[Capital Windhelm Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/42990) - fixes clipping issues, conflict resolution on navmesh and landscape. Last checked against version 1.3
  
  
[Carriage Drivers Are Alive](https://www.nexusmods.com/skyrimspecialedition/mods/51260) - removes the tent and adds a new place for a driver to stay, conflict resolution. Last checked against version 7.8
  
  
[City Entrances Overhaul: Windhelm](https://www.nexusmods.com/skyrimspecialedition/mods/19400) - shifts/disables some elements, CR on navmesh and landscape. Designed to work only with [Rob's Bugfixed version](https://www.nexusmods.com/skyrimspecialedition/mods/65092) of the plugin. Last checked against version 3.1 of Rob's Bug Fixes. Additional consistency patch for [Capital Windhelm Expansion - City Entrances Overhaul Patch](https://www.nexusmods.com/skyrimspecialedition/mods/47186) available.
  
  
[Civil War Refugees](https://www.nexusmods.com/skyrimspecialedition/mods/29595) - adjusts/disables some conflicting elements, moves the entire camp from the stables area closer to Hlaalu farm, conflict resolution on landscape. Last checked against V4.
  
  
[Convenient Carriages](https://www.nexusmods.com/skyrimspecialedition/mods/12693) - ﻿removes the tent and adds a new place for a driver to stay, conflict resolution. Last checked against version 5.9.8
  
  
[Creation Club Alternative Armors: Ebony Plate](https://en.uesp.net/wiki/Skyrim:Alternative_Armors_-_Ebony_Plate) - adjusts position of a few records.
  
  
[DK's Realistic Nord Ships](https://www.nexusmods.com/skyrimspecialedition/mods/43407) - resolves navmesh conflicts. Last checked against version 2.0
  
  
[Elder Scrolls Online Imports](https://www.nexusmods.com/skyrimspecialedition/mods/11729) - conflict resolution. Last checked against version 10.0.1
  
  
[Enhanced Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/18162) - conflict resolution on landscape records, switches/disables a few added elements. Last checked against version 2.0.1
  
  
[Illustrated Town Panels](https://www.nexusmods.com/skyrimspecialedition/mods/43603) - shifts town panel placement. Last checked against version 1.2.3
  
  
[Immersive Citizens - AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/173) - resolves landscape and navmesh conflicts forwarding and adjusting settings from both mods where necessary, shifts one lean marker. Last checked against version 0.4.0.2
  
  
[Immersive Laundry](https://www.nexusmods.com/skyrimspecialedition/mods/2011) - shifts a couple of clothing lines to match the modified exterior. Last checked against version 1.0d
  
  
[iNeed - Extended](https://www.nexusmods.com/skyrimspecialedition/mods/645) - conflict resolution on one snow drift. Last checked against version 1.83
  
  
[Interesting NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/29194) - navmesh conflict resolution. Last checked against version 4.53
  
  
[JUSTICE - City Exteriors](https://www.nexusmods.com/skyrimspecialedition/mods/680) - adapts added elements to a new bridge entrance layout. Last checked against version 1.15
  
  
[Khajiit Has Wares](https://www.nexusmods.com/skyrimspecialedition/mods/73500) - fixes issue with camp not appearing (moved xmarker) and some landscape clipping. Last checked against version 1.08.2. Patch by JPdreamer.
  
  
[Lanterns of Skyrim II](https://www.nexusmods.com/skyrimspecialedition/mods/30817) - adjusts lanterns positions. Last checked against version 4.1
  
  
[Lux Via](https://www.nexusmods.com/skyrimspecialedition/mods/63588) - shifts and disables a bunch of references from both mods (prioritizing JK's), adjusts navmesh and landscape. Last checked against version 2.1
  
  
[No Snow Under the Roof](https://www.nexusmods.com/skyrimspecialedition/mods/518) - adjusts and disables some NSUTR additions. Last checked against version 3.2
  
  
[Realistic Boat Travel](https://www.nexusmods.com/skyrimspecialedition/mods/16280) - adjusts the placement of the purchasable boat to the dock added in JK's Windhelm Outskirts. Last checked against version 1.0
  
  
[Routa - Stormcloak and Warrior Cabin](https://www.nexusmods.com/skyrimspecialedition/mods/1193) - moves house further up the hill, adjusts navmesh and landscape. Last checked against version 1.0
  
  
[Ryn's Farms](https://www.nexusmods.com/skyrimspecialedition/mods/72305) - a mix'n'match combo of both mods, shifts/restores/removes a bunch of elements, conflict resolution, landscape and navmesh adjustments. Last checked against version 2.0
  
  
[Scarecrows of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/39575) - adjusts the position of scarecrows to fit the modified farms' layout. Last checked against version 1.6 (it won't work with older versions due to FormIDs mismatch!)
  
  
[Skyrim 3D Signs](https://www.nexusmods.com/skyrimspecialedition/mods/21338) - adjusts signs' position. Last checked against version 1.2.5
  
  
[Skyrim Bridges](https://www.nexusmods.com/skyrimspecialedition/mods/21263) - combines the bridge changes from both mods. Last checked against version 3.2
  
  
[Skyrim Landscape and Water Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/26138) - conflict resolution, forwards ownership changes to crops. Last checked against version 8.4
  
  
[Skyrim Sewers 4](https://www.nexusmods.com/skyrimspecialedition/mods/9320) - CR on navmesh. Last checked against version 4.15
  
  
[Stairs of Skyrim - Windhelm Bridge Stairs](https://www.nexusmods.com/skyrimspecialedition/mods/78399)﻿ - removes the stairs added in JK's mod in favour of the new ones from SoS. Last checked against JK's version 1.0
  
  
[Touring Carriages](https://www.nexusmods.com/skyrimspecialedition/mods/15948) - conflict resolution, shifts a few markers. Last checked against version 3.5.2
  
  
[Traverse the Ulvenwald](https://www.nexusmods.com/skyrimspecialedition/mods/57874)﻿ - adjusts the placement of a few trees. Last checked against version 3.3.2
  
  
[Unofficial Skyrim Special Edition Patch](https://www.nexusmods.com/skyrimspecialedition/mods/266) - disables one snow pile. Last checked against version 4.2.8
  
  
[Veydogolt - Extra Trees](https://www.nexusmods.com/skyrimspecialedition/mods/86679) - moves/disables a few trees. Last checked against version 10.1
  
  
[Windhelm Bridge Revived](https://www.nexusmods.com/skyrimspecialedition/mods/84686) - conflict resolution, little adjustments on a few records, and navmesh finalisation. Last checked against version 1.4
  
  
[Windhelm Exterior Altered](https://www.nexusmods.com/skyrimspecialedition/mods/5824) - CR, navmesh adjustments. Last checked against version 2.0. Patch by Teabag86
  
  
[Windhelm Exterior Altered Altered](https://www.nexusmods.com/skyrimspecialedition/mods/62656) - CR, navmesh and landscape adjustments. Last checked against version 1.0
  
  
[Witcher Horse Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/72490) - places the horse in the stables added in JK's mod. Last checked against version 2.0
  
  
**Additional notes:**
  
- some of the fixes in question are made on persistent references and require a new save to work properly,
  
- there's an alternative version of [CACO Farm Overhaul / Ryn's Farms combo with JK's Windhelm Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/87262) made by Janosz - it might not be compatible with the rest of the patches here, but you should check it if you want a different approach to the farms' layout
  
- if you use DK's Realistic Nord Ships and Skyrim Sewers, load the JKs Windhelm Outskirts - Skyrim Sewers patch after Realistic Nord Ships - Skyrim Sewers patch (available on the Realistic Nord Ships page)
  
  
**Load order suggestion:**
  
1) most of the patches
  
2) CEO Windhelm / Capital Windhelm / Eli's Routa patches
  
3) CACO Farm Overhaul / Ryn's Farms / BUVARP patches
  
4) Lux Via patch
  
5) consistency patches
  
6) Northern Roads patches
