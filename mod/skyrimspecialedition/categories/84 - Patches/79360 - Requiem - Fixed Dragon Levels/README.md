# Requiem - Fixed Dragon Levels
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79360
Fixes an issue in Requiem with Dragons having completely arbitrary levels.
  
  
There are 7 tiers of Dragons in Requiem, each of which has an assigned level. The higher the Dragon tier the higher their health and shout level. The levels do not reflect that.
  
  
Includes a version for [Deadly Dragons](https://www.nexusmods.com/skyrimspecialedition/mods/23723). (requires the [Requiem patch](https://www.nexusmods.com/skyrimspecialedition/mods/64410))
