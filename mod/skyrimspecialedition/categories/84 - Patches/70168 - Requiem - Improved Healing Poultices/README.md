# Requiem - Improved Healing Poultices
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70168
**Increases the availability and duration of Healing Poultices in Requiem.**
  
  
*- Increased the amount of Healing Poultices on vendors by 2-4
  
- Increased the chance to find Healing Poultices on NPCs, in containers and in the world
  
[size=4][i]- Duration from 20 minutes to 30 minutes*
  
  
- Removed the restriction that it didn't regenerate while swimming and sprinting
  
- They now regenerate health twice as fast
  
- Small changes to the description to reflect the changes
  
  
If you use [Requiem - More Potions](https://www.nexusmods.com/skyrimspecialedition/mods/67235), load the Poultice mod after it.
