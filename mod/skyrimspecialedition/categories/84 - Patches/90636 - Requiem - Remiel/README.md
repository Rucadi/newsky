# Requiem - Remiel
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/90636
**Compatibility and Balance patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Remiel-Custom Voiced Dwemer Specialist and Companion](https://www.nexusmods.com/skyrimspecialedition/mods/51874).** 
  
  
- Rebalanced Remiel for Requiem (including perks, stats, and everything else)
  
- Rebalanced all new potions
  
- Rebalanced dwemer misc items, weapons, and ammunition
  
- Removed the lockpicking feature from Remiel in favor of Requiem system
  
[size=3]- Rebalanced all the new NPCs and Scrap
  
- Fixed some compatibility issues (e.g. the intro quest not working)
