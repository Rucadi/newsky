# Lost Races of Nirn - High Poly Head Addon
- Author: TateTaylor - KouLeifoh - AlbinoWalrus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/147619


Exactly what it says on the tin; **High Poly Head** patches updated for **Lost Races of Nirn**'s latest, modular version.
  
  
The installer includes patches for Dwemer, Maormer, Snow Elves, and Tsaesci.
  
*\*Ayleids don't require a patch since they reuse the default Altmer head.*
  
  
**Alternative Textures Plugin:**I've also included an optional patch that modifies Dwarves to use Dark Elf normal maps, for those who want to avoid texture mismatch.
