# Creation Club The Cause - BS Bruma Scamps Synergy
- Author: Midgetalien
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/59591


A simple mod that provides some synergy between Beyond Skyrim and The Cause Creation.
  
Adds Scamps from Beyond Skyrim:Bruma to the Deadlands from Creation Club content: The Cause.
  
I've also added the Dremora Keywords to the Scamps to prevent Dremora from attacking them.
  
  
Thanks to [Decaroidea](https://www.nexusmods.com/skyrimspecialedition/users/24539314) for the ESL flagged version.
  
  
**Requirements:**
  
Creation Club "The Cause" and Beyond Skyrim Bruma
  
  
  
**Recommended mods:** 
  
[Creation Club Curios and Zombies: Beyond Skyrim Synergy Patch](https://www.nexusmods.com/skyrimspecialedition/mods/28630?tab=description)
