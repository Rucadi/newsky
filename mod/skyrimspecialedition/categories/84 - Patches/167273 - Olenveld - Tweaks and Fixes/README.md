# Olenveld - Tweaks and Fixes
- Author: IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/167273


**Fixes some issues, inconsistencies and rewrote some parts of [Olenveld](https://www.nexusmods.com/skyrimspecialedition/mods/133579) for a smoother experience.
  
Olenveld\_Fix\_Pack is compatible and recommended!****Full Changelog:**
  
*- Capitalized the names of objects and areas (e.g. Forgotten keep -> Forgotten Keep, Alvor's Death note -> Alvor's Death Note)
  
- Improved the names of interactables and items (e.g. Mage Seal Unlock -> Scroll of Unbinding, Remove Scaffold -> Wooden Barricade)
  
- Some small improvements to quest journal entries
  
- Removed the ability to craft enchanted robes
  
- The recipe for smelting Mithril Ingots no longer shows without owning at least 1 Mithril Ore
  
- Lowered the power of Olenveld Spiders and Magic Anomalies
  
- Rewrote a lot of player dialogue and message boxes*
