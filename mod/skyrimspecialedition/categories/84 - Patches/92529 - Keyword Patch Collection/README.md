**Keyword Patch Collection**
  
**Overview**
  
**Keyword Patch Collection** (or **KPC**) is a collection of [Keyword Item Distributer](https://www.nexusmods.com/skyrimspecialedition/mods/55728) INIs (\_KID.ini files) that increase compatibility and synergy between mods by adding [keywords](https://www.creationkit.com/index.php?title=Keyword) to items and other [forms](https://www.creationkit.com/index.php?title=Base_Object#Base_Object), and replacing potentially problematic INIs with new INIs utilizing [Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469), new [KID](https://www.nexusmods.com/skyrimspecialedition/mods/55728) features, and/or improved syntax.

**Installation**
  
Install Keyword Patch Collection and its requirements with your choice of mod manager. Overwrite any other files if asked. Let me know if you have any problems.
  
  
**Hard Requirements**
  

  
* [Keyword Item Distributer](https://www.nexusmods.com/skyrimspecialedition/mods/55728) (v3.0.2+ required).
[Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469) (v4.0.0+ required).
  
* All of the above mods' requirements (obviously).

  
**Optional Requirements**
  

  
* [MergeMapper](https://www.nexusmods.com/skyrimspecialedition/mods/74689) (Strongly recommended *if*you use [zMerge](https://github.com/z-edit/zedit/releases). Not needed if you don't use that modding tool.)
Any of the supported mods that you want patched (obviously).
