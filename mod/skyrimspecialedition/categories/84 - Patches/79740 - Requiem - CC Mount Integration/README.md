# Requiem - CC Mount Integration
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79740
**A balance and consistency patch for all of the mounts added by the CC (AE) DLC.**
  
  
*[size=3]- Rebalanced all the mounts with unique upsides in-line with Requiem balancing (= Health, Stamina, Speed, Level, Confidence)
  
- Dwarven Horse/Reindeer can now be examined for horse inventory
  
- Increased the base value of the reindeer mount from 500 to 2000
