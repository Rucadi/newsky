# Sa'chil Body and Texture Patches (CBBE-Default-HIMBO-UNP)
- Author: JuzyJ
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106914


Adds [CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174), [HIMBO](https://www.nexusmods.com/skyrimspecialedition/mods/46311), [UNP](https://www.nexusmods.com/skyrimspecialedition/mods/31126) or default body Patches for [size=4][Sa'chil - Custom Voiced Khajiit Follower](https://www.nexusmods.com/skyrimspecialedition/mods/106892).[size=4][size=4]
  
Also includes new custom textures from [Feminine Khajiit Textures](https://www.nexusmods.com/skyrimspecialedition/mods/183) and [Masculine Khajiit Textures](https://www.nexusmods.com/skyrimspecialedition/mods/186).[/size]
  
Optional support for [New Beast Feet](https://www.nexusmods.com/skyrimspecialedition/mods/107689).
  
  
Safe to use with any version of CBBE or HIMBO.
  
  
Comes with ESL flagged ".esp". Prebuilt meshes are 0-ed. Ke'tud's armor comes with zaps for additional edit options in BodySlide.
