# Requiem - Heart of Ice Astrid
- Author: IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/147927


Balance and Compatibility patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Heart of Ice - Astrid](https://www.nexusmods.com/skyrimspecialedition/mods/132350).
  
  
*I'm aware that there is another Requiem patch for the mod but it's outdated and didn't meet my expectations.
  
  
  
- Rebalances armor, weapons, tempering recipe, container loot, food items to match Requiem
  
- Rebalanced Astrid with new perks, changes to base stats, and she now starts at level 20
  
- Removed 2 mudcrabs added by Requiem at the exit of the first dungeon
  
- One bandit in the first dungeon now starts dead so Astrid has an easier time dealing with them*
