# Requiem - Bruma Fixed
- Author: Requiem Community
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/66569
**Compatibility patch for Requiem and Beyond Skyrim - Bruma SE.** *This is an updated and improved version of [Requiem - Bruma Patch (Updated)](https://www.nexusmods.com/skyrimspecialedition/mods/35780)
  
  
**Balanced the content of Bruma for Requiem:**
  
- NPCs and enemies
  
- Weapons and Armor
  
- Crafting
  
- Ingredients
  
- Leveled Lists
  
- Food (+ added Survival Mode support)
  
- Replaced some Bruma items that were identical with skyrim ones (e.g. Glass Shield that was duplicated for Bruma and not balanced for Requiem now uses the standard Glass Shield)
  
  
**And much more!** (check out the changelog for more information)
