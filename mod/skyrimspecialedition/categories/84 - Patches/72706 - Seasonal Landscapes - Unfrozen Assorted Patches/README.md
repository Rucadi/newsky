# Seasonal Landscapes - Unfrozen Assorted Patches
- Author: Reign
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72706


A small collection of patches that I've made for my personal game that might be helpful to others. The following patches are available.
  
  
These patches are meant to accompany [Mefariah](https://www.nexusmods.com/skyrimspecialedition/users/2617981)'s [Seasonal Landscapes - Unfrozen](https://www.nexusmods.com/skyrimspecialedition/mods/71967), but none of the files technically require it as a master.
  
All of the patches are esl flagged and are available in the FOMOD to use with your mod manager of choice. A couple of the patches add some meshes that needed to be edited to remove snow from their models that weren't using snow shaders, so be aware if they conflict with any model replacers in your setup.
