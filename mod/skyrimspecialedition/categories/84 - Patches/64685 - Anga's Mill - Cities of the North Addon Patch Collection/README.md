# Anga's Mill - Cities of the North Addon Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64685


Collection of patches for [Anga's Mill - Cities of the North Addon](https://www.nexusmods.com/skyrimspecialedition/mods/64398). All patches are ESP flagged as ESL and (at least the interior ones) are compatible with each other.
  
  
Details of patches:
  
[At Your Own Pace - Thieves Guild](https://www.nexusmods.com/skyrimspecialedition/mods/52704) - adjusts the placement of the box, last checked against version 2.1.1TG
  
[Cheesemod for EVERYONE](https://www.nexusmods.com/skyrimspecialedition/mods/36506) - shifts added cheeses to match the new interiors, last checked against version 1.01
  
[Complete Alchemy and Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/19924) - shifts a few placed references, last checked against version 2.0.8
  
[Edmond's Nature Series - Birds](https://www.nexusmods.com/skyrimspecialedition/mods/123068) - shifts the placement of two crows, last checked against version 1.02
  
[Elder Scrolls Online Imports](https://www.nexusmods.com/skyrimspecialedition/mods/11729) - modifies some added elements positions in the interiors, provides exterior navmesh consistency, last checked against version 8.1
  
[Embers XD](https://www.nexusmods.com/skyrimspecialedition/mods/37085) - adjust fireplaces positions, last checked against version 2.5.4
  
[Enhanced Lights and FX](https://www.nexusmods.com/skyrimspecialedition/mods/2424) - adjusted lights changes to new interior layout, conflict resolution for ELFX - Exterior, last checked against version 3.06
  
[Farmhouse Chimneys](https://www.nexusmods.com/skyrimspecialedition/mods/8766) - adjusts chimney position to reworked house model, last checked against version 4.1.1
  
[Immersive Citizens - AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/173) - forwards persistency and ownership records, navmesh conflict resolution, last checked against version 0.4
  
[Landscape Fixes for Grass Mods](https://www.nexusmods.com/skyrimspecialedition/mods/9005) - conflict resolution on the landscape, last checked against version 5.5
  
[Lanterns of Skyrim II](https://www.nexusmods.com/skyrimspecialedition/mods/30817) - shifts two lanterns and lights to match the exterior changes, last checked against version 4.0
  
[Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095) - adjusts lights position to match the exterior changes, last checked against version 2.5
  
[Nature of the Wild Lands](https://www.nexusmods.com/skyrimspecialedition/mods/63604) - adjusts the trees placement, last checked against version 3.11
  
[No Snow Under the Roof](https://www.nexusmods.com/skyrimspecialedition/mods/518) - conflict resolution, adjust two snow drifts on the ground and disables the roof ones, last checked against version 3.2
  
[Ryn's Lumber Mills](https://www.nexusmods.com/skyrimspecialedition/mods/115134) - conflict resolution, landscape and navmesh adjustments. Last checked against version 1.0
  
[Skyrim Artwork Imports](https://www.nexusmods.com/skyrimspecialedition/mods/22472) – adjusts the wall decorations to match the modified interiors, last checked against version 1.0
  
[Skyrim Project Optimization](https://www.nexusmods.com/skyrimspecialedition/mods/14084) - modifies the position of a plane marker in one of the new interiors, last checked against version 1.5
  
[Snazzy Furniture and Clutter Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/2414) - adjusts the paintings to match the modified interiors, last checked against version 2.0
  
[Sounds of Skyrim Complete SE](https://www.nexusmods.com/skyrimspecialedition/mods/8286) - adjusts sound markers to new interior layout, forwards acoustic space changes, last checked against version 2.0
  
[Vanilla Landscapes Corrections](https://www.nexusmods.com/skyrimspecialedition/mods/104962) - conflict resolution on the landscape, last checked against version 1.1.2
  
  
Additional notes:
  
- the patch for Sounds of Skyrim should be loaded as the last one and overwritten only by other SoS compatibility patches for other mods affecting the area (f.e. Lux, Immersive Citizens)
  
- the following mods are checked and compatible without the need for a patch: Cutting Room Floor, Immersive Fallen Trees, Landscape and Water Fixes, Creation Club - Fishing
