**WHAT'S HERE**
  
  
Most of these are extremely straightforward, but Animated Armory requires a bit of a comment:
  
  
The animated armory patches were built off of [Animated Armoury - Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/47213) but do not require it. It also does not distribute whips (for compat with the DAR version of Animated Armoury).
  
  
The main editorial decision this patch makes is that it mixes the new weapon types in with the vanilla ones, such that:
  
  

  
* Daggers include Claws.

  
* Swords include Rapiers.

  
* Great Swords include Pikes.

  
* Battle Axes include Halberds.

  
* War Hammers include Quarter Staffs.

  
The other alternative wouldÂ  have been to introduce new weapon type lists for the new weapon types and then distribute THOSE everywhere, but the result would have been, I think, the same.
  
  
Imperials now get 2H handers distributed to them in addition to one handers if they didn't have a preference (the same as if you're using RFM).
