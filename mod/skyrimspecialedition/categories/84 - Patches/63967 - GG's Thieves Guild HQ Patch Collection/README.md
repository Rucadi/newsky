# GG's Thieves Guild HQ Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63967


Collection of patches for [GG's Thieves Guild Headquarters](https://www.nexusmods.com/skyrimspecialedition/mods/64018). **All patches are ESP flagged as ESL.****Details of patches:**
  
[Additional Hearthfire Dolls](https://www.nexusmods.com/skyrimspecialedition/mods/46930) - moves a doll to a new position
  
[Ars Metallica - Smithing Enhancement](https://www.nexusmods.com/skyrimspecialedition/mods/321)﻿ - moves a letter to a new position
  
[Brynjolf and the Riften Guild - Birthright](https://www.nexusmods.com/skyrimspecialedition/mods/81471) - adapts the Birthright's interior additions and changes to GG's interior
  
[Daedric Shrines - Nocturnal](https://www.nexusmods.com/skyrimspecialedition/mods/70696) - adjusts the shrine position in a modified interior
  
[Distinct Interiors](https://www.nexusmods.com/skyrimspecialedition/mods/6130), [Opulent Thieves Guild](https://www.nexusmods.com/skyrimspecialedition/mods/931)﻿ and [Distinct Opulent Thieves Guild](https://www.nexusmods.com/skyrimspecialedition/mods/63717) - apply mods additions to GG's Thieves Guild HQ, resolving conflicts and clipping, expanded treasury room in both OTG patches
  
[DX Faction Crossbows](https://www.nexusmods.com/skyrimspecialedition/mods/4047)- moves a book to a new position
  
[Enhanced Lights and FX](https://www.nexusmods.com/skyrimspecialedition/mods/2424) - disables ELFX light changes/additions and some overlapping effects, adjusts the position and adds new smoke meshes, patch available in two versions - for regular ELFX with GG's cell lighting settings and one with modified settings for ELFX Enhancer
  
[ESO Imports](https://www.nexusmods.com/skyrimspecialedition/mods/11729) - conflict resolution
  
[Environs - Abandoned Abodes](https://www.nexusmods.com/skyrimspecialedition/mods/82410) - resolves navmesh conflicts, disables the passage from cistern to Riftweald Manor after the interior change
  
[JK's Thieves Guild HQ](https://www.nexusmods.com/skyrimspecialedition/mods/133787) - mix of both mods, modifies the main hall to upgrade and change with the TG quest progression, expands the treasury, moves new JK's rooms to a different, non-conflicting spots, adapts navmesh, resolves conflicts
  
[JS Thieves Guild Training Chests](https://www.nexusmods.com/skyrimspecialedition/mods/57984) - conflict resolution
  
[Kozakowy's Falka Armor](https://www.nexusmods.com/skyrimspecialedition/mods/33939) - moves the armor to a different chest
  
[Kozakowy's Falka Armor CBBE 3BA Bodyslide](https://www.nexusmods.com/skyrimspecialedition/mods/52651) - moves the armor to a different chest  
  
[Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802) - moves LOTD objects and activators into new Mercer's room
  
[Leviathan Axe - Dwemer Artifacts SE](https://www.nexusmods.com/skyrimspecialedition/mods/40364) - moves a scroll to a new position
  
[Lux](https://www.nexusmods.com/skyrimspecialedition/mods/43158) - removes all Lux changes and forwards Lux' specific templates
  
[More Nocturnal Statues](https://www.nexusmods.com/skyrimspecialedition/mods/57081) - disables additional light to prevent flickering
  
[Mystic Condenser](https://www.nexusmods.com/skyrimspecialedition/mods/10176)﻿﻿ - moves condenser to a new position
  
[Oblivion Artifact Pack](https://www.nexusmods.com/skyrimspecialedition/mods/10644) - moves the artifact to a new position, conflict resolution
  
[Realistic Lighting Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/844) - disables all RLO changes/additions to the interior
  
[Riften Docks Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/40021) - CR on navmesh in docks
  
[Riftweald Is Mine](https://www.nexusmods.com/skyrimspecialedition/mods/79881) - resolves navmesh conflicts, adds a passage to a new Riftweald interior variant
  
[Riftweald Manor - A Player Home](https://www.nexusmods.com/skyrimspecialedition/mods/75066) - fixes the placement of three objects
  
[Skyrim Project Optimization](https://www.nexusmods.com/skyrimspecialedition/mods/14084)﻿ - disables all added occlusion planes
  
[Snazzy Furniture and Clutter Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/2414)﻿ - forwards SFCO changes to furniture and weapon racks, adjusts paintings position
  
[Sneak Tools SE](https://www.nexusmods.com/skyrimspecialedition/mods/1863) - shifts the position of a new book, moves the new door to a different spot, and changes some quest script properties so everything could run without issues
  
[Sounds of Skyrim Complete](https://www.nexusmods.com/skyrimspecialedition/mods/8286) - adjusts the position of sound markers
  
[Thieves Guild for Good Guys](https://www.nexusmods.com/skyrimspecialedition/mods/10745) - adjusts the book and note placement
  
[Unofficial Skyrim Special Edition Patch](https://www.nexusmods.com/skyrimspecialedition/mods/266)﻿ - moves added collision box to the proper position
  
[Weapons Armor Clothing and Clutter Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/18994) - conflict resolution
  
  
**Load order reminder:**
  
- users of Distinct Interiors / Opulent Thieves Guild / Distinct Opulent Thieves Guild should load all patches designed for those mods **BEFORE** GG's Thieves Guild Headquarters main file and all of its patches
  
- Lux/ELFX Enhancer patch for the GG's TGHQ should be loaded as the last﻿ file of the TG collection
