# Requiem - Fozars Dragonborn 5.2.0 Fixes
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70494
**Fixes issues while using Requiem 5.2.x with Fozars Dragonborn patch.***- Overwrites Fozars tweaks to armor with the ones made by Requiem.
  
- Dragonborn Enchanted Arrows are no longer retrievable
  
  
**Other fixes, not related to the new Requiem update:**
  
- Ash Shell displays the correct Skill Level, fixed mana cost
  
- Orcish Smithing is unlocked with Dwarven OR Morrowind Smithing (instead of needing both)
  
- Glass Smithing is no longer unlocked with Morrowind Smithing Level 2 (this was inconsistent, there was no indication that it did that, and it caused the tree to clip and look bad)
  
- Moved Morrowind Smithing in the tree so it no longer clips with any other perk
  
  
**Included a Patch for [Trad's Requiem CC patch](https://www.nexusmods.com/skyrimspecialedition/mods/62981)** (requires Main File)
  
- Included the smithing tree fixes and repositioned the added shivering smithing perk
