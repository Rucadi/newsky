# Unofficial Skyrim Modders Patch - USMP - Patch Emporium
- Author: sattyre
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/50813


I made a number of patches for USMP and some other mods and decided that USMP patches really need their own page.  I will be creating more patches for USMP as time goes by.  Requests will be considered.  However due to the sheer number of NPC overhauls, they don't have a high priority. and they will overwrite without issue.  Any patches I create alway have USSEP edits and USSEP is a master for USMP anyway.
  
  
The [Unofficial Skyrim Modders Patch](https://www.nexusmods.com/skyrimspecialedition/mods/49616) can be found here.
  
  
Patches so far created are for version 2.1 or later of USMP,  but most will work with any version of USMP 2 and later with the exception of CACO, which is only needed for 2.1 and any later USMP versions.  This info may change as patches are added/updated.
  
  
Just because a mod updates doesn't necessarily mean a patch is out of date.  It only needs to be updated if the conflicting information changes.
