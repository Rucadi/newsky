[font=Georgia][color=#d8d8d8][size=4]Being ELFX Shadows a lighting mod it is easy to find
  
conflicts with all those mods that modify internal cells. We have created this patch hub to resolve these conflicts and therefore to help users who want to install our mod along with the most used mods by the community

  
[/font]This patches hub comes with an easy to use FOMOD that automatically recognizes the mods you are using, we recommend that you read the instructions included for each patch.
  
there is also a section that cannot be automated so you have to choose what you prefer manually.
  
  
  
For now ther'are few patches already included in the FOMOD but this hub will be updated really often.
