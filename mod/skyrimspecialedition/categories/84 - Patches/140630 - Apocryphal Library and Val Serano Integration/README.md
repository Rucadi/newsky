# Apocryphal Library and Val Serano Integration
- Author: CAMGS11235
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140630


Simple SWAP.ini file to enable integration of [Jenasto](https://www.nexusmods.com/skyrimspecialedition/users/1048346)'s [Apocryphal Library](https://www.nexusmods.com/skyrimspecialedition/mods/121654) with [Axarien](https://www.nexusmods.com/skyrimspecialedition/users/72078788)'s [Val Serano - Pirate Custom Voiced Follower and Quest Adventure](https://www.nexusmods.com/skyrimspecialedition/mods/103669).
  
  
This makes it so that piles of books within the slice of Apocrypha accessible via the Black Book Sailor's Bargain will become containers holding the books added by Apocryphal Library. Now you can join Val in his arcane studies of forbidden texts!
