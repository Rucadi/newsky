# JS Instruments of Skyrim - Patch Collection
- Author: IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148330


**Compatibility patches for several mods to work with [JS Instruments of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/51959).****Adds compatibility for the following mods:****[Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/51959)** - Forwards the keyword and value changes to the new instruments
  
[At Your Own Pace Misc](https://www.nexusmods.com/skyrimspecialedition/mods/52704) - Forwards the changes to 3 quests so you receive the intended unique instruments while having the changes from AYOP
  
[Skyrim's Got Talent](https://www.nexusmods.com/skyrimspecialedition/mods/50357) - Enables SGT support to the new instruments and forwards a keyword 
  
[Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/51959) + [Skyrim's Got Talent](https://www.nexusmods.com/skyrimspecialedition/mods/50357) - Combines both patches because they touch the same entries
