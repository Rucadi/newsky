# Skyshards - Tweaks - Patches
- Author: WhereDeadAngelsLie
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115880


![](https://i.imgur.com/8jkV3Pk.png)

  
Patches for the mods
  

  
* [Ryn's Bleakwind Basin](https://www.nexusmods.com/skyrimspecialedition/mods/73141) - Move the Skyshard at the top of the tower. Now you will have a good reason to go there﻿
[Ryn's Karthspire](https://www.nexusmods.com/skyrimspecialedition/mods/77059) - Move the Skyshard very high above the Akaviri buildings, you will have to find a way to get it (or install [Skyclimb](https://www.nexusmods.com/skyrimspecialedition/mods/97253)﻿... but it's not necessary)﻿
  
* [RedBag's Rorikstead](https://www.nexusmods.com/skyrimspecialedition/mods/56114) - Move the Skyshard on the edge of a cliff instead of the middle of the plain
[The Great City Of Winterhold](https://www.nexusmods.com/skyrimspecialedition/mods/17127)﻿ - Move the Skyshard to have it 90% covered in snow, as if it was almost lost during the great collapse
  
* [Skyshards Full LOD](https://www.nexusmods.com/skyrimspecialedition/mods/73183)﻿ - To use only you select an option that keep the Beacons, otherwise you don't need this mod

  
If you see any other mod that needs a patch or a Skyshard moved, please let me know with a post
