# Blubbos Apple Tree Replacer for JK Mods
- Author: BlubboDE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/132893


I love the JK Skyrim mod and also Whiterun Outskirts. But the apple trees in it are not very nice. So today I created a new model for it.
  
Just install it and place it after the JK mods in the loading order. If you use JK Skyrim, or Whiterun Outskirts, you now have the new tree.
