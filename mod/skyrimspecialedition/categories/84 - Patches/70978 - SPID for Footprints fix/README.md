![](https://staticdelivery.nexusmods.com/mods/1704/images/70978/70978-1690270880-879554239.png)

  
  
[font=Georgia]

This is a fix for [SPID for Footprints](https://www.nexusmods.com/skyrimspecialedition/mods/54924)﻿ by [reza9892](https://www.nexusmods.com/skyrimspecialedition/users/35353440).

[/font][font=Georgia]

For version 4.0.0 new game or clean save required.

[/font]
  
  
[font=Georgia]Optimized almost all scripts.
  
Removed counting scheme.
  
DISTR file has incorrect FormID for Footprints spell distributed to Skeletons.
  
Removed useless footprintsAddFootstepsSpell spell from magic effects and scripts. Also removed all other useless variables from scritps.
  
Removed notifications about added footprints spell.
  
Fixed incorrect variable references and incorrect assignments to variables in magic effects and quests.
  
Small fix in KID file (barefoot footprint for footwraps).
  
  
Compatibility:
  
Compatible with most other footprint fix mods:  Footprints Sand patch, Footprints — Alternative Design, etc
  
Not compatible with other mods that have similar function, such as Ultimate Fix, etc. Use only one.[/font]
