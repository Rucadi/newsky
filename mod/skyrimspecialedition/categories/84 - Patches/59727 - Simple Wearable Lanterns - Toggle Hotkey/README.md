# Simple Wearable Lanterns - Toggle Hotkey
- Author: Natasha
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/59727
This mod adds a configurable hotkey to Qwib's [Simple Wearable Lanterns](https://www.nexusmods.com/skyrimspecialedition/mods/58464) mod, allowing you to easily turn the light on and off without having to unequip the lantern. Works for all 3 lantern variations.
