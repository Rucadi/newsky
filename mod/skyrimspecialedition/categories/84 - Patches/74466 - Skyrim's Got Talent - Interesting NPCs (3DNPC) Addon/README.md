# Skyrim's Got Talent - Interesting NPCs (3DNPC) Addon
- Author: TheCyclist
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/74466
**[![](https://i.imgur.com/BZarQov.png)](https://www.buymeacoffee.com/TheCyclist)
  
  
Description**
  
Some of the Followers from Interesting NPCs will react to your performance in a positive and negative way.Keep in mind that you're not gonna get comments everytime you perform. Otherwise It'll be very repetitive.ESL-Flagged ESP.
