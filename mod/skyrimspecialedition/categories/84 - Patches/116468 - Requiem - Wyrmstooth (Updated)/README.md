# Requiem - Wyrmstooth (Updated)
- Author: Jonx0r - NRM Team - Xanza - IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/116468
**Compatibility patch for Requiem and Wyrmstooth.** *This is an updated and improved version of [Requiem - Wyrmstooth Patch](https://www.nexusmods.com/skyrimspecialedition/mods/46608)
  
**Changes from the original mod:****While I don't go into details about the changes made, I can assure you that the quality of balancing and other changes will match Requiem much closer.**- Rebalanced the ingredients
  
- Completely rebalanced the spells "Call of the Wild" and "Call of the Dead"
  
- Rebalanced the 3 draugr conjuration spells slightly
  
- Rebalanced Ore/Ingot weight and value
  
- Rebalanced the Dragon Priest
  
- Rebalanced the Dragon, Dragon Priest, Werewolves to be more in-line with Requiem
  
- Rebalanced some armor and trinkets
  
- Rebalanced some weapons and weapon enchantments
  
- Improved the Wyrmstooth player shout slightly
  
  
- Player level requirement from 10 to 20
  
- Rewrote some descriptions and messages to better fit Requiem
  
- Updated several NPC entries to the newest version of Wyrmstooth
  
- Redone NPC naming to fit Requiem (e.g. "Marauder Berserker" to "Marauder")
  
- Fixed the package of Thelma being outdated
  
- Fixed some temper recipes not being set up properly for Requiem
  
- The new alcohol now supports Survival Mode
  
- Replaced many of the Wyrmstooth spells with similar Requiem spells
  
- Removed some Wyrmstooth spells that were not properly integrated or not unique enough
  
- Increased the variations of spellbooks found in Wyrmstooth
  
- Removed some placed spellbooks where there was an abundance of them
