# Project ja-Kha'jay - Patch Collection
- Author: Dudeling
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/110682


These are all small ESL-flagged patches that make the amazing mod [Project ja-Kha'jay](https://www.nexusmods.com/skyrimspecialedition/mods/57610?tab=description)﻿ compatible with several other popular weapons, armors and other mods, by forwarding the visual changes and merging conflicts. Let me know, if you encounter any issues. Thanks!
  
  
The mods currently covered are:
  
[Beyond Skyrim - Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/10917)[The Forgotten City](https://www.nexusmods.com/skyrimspecialedition/mods/1179)
  
[Immersive College of Winterhold](https://www.nexusmods.com/skyrimspecialedition/mods/17004)[Immersive Weapons](https://www.nexusmods.com/skyrimspecialedition/mods/16788)﻿[Morrowloot Ultimate](https://www.nexusmods.com/skyrimspecialedition/mods/3058)
  
[Lawless - A Bandit Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/88080)﻿[Minor Factions Armor](https://www.nexusmods.com/skyrimspecialedition/mods/48484)﻿[Unofficial Skyrim Modders Patch](https://www.nexusmods.com/skyrimspecialedition/mods/49616)+ several combo patches merging the changes of three mods
