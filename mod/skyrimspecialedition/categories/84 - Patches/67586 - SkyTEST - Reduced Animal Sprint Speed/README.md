# SkyTEST - Reduced Animal Sprint Speed
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67586
**- Reverts the sprint speed of Deers/Elks/Wolfs/Sabrecats to vanilla values while keeping the other changes of SkyTEST**This mod was made mostly for users of [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) or other mods that increase the movement speed of these animals.
