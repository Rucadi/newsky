Bashed Patch
  
  
The official Requiem advice regarding a [Bashed Patch](https://github.com/ProbablyManuel/requiem/wiki/Bashed-Patch) applies to the Requiem Patch Central too. The Import feature of a Bashed Patch is not supported for any types except those explicitly listed as safe in the aforementioned advice. The manner in which a Bashed Patch imports records is too limited to handle non-trivial interactions between multiple pugins even when the plugins have proper bash tags assigned.
  
  
A Bashed or Smashed Patch can replace compatibility patches for a number of mods with a narrow scope. A (non-exhaustive) list of such mods is given below. It's recommended to run LOOT before building a Bashed Patch to fetch all available bash tags.
