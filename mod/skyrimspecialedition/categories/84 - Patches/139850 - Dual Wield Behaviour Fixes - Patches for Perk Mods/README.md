# Dual Wield Behaviour Fixes - Patches for Perk Mods
- Author: Blackread
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/139850


These are perk fixes for the relevant dual wield enhancing perks from several popular perk mods using the new graph variable introduced by [Dual Wield Behavior Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/139775)﻿. These are standalone and you don't need the **optional** perk fixing files from Dual Wield Behavior Fixes (and using them with these perk mods will likely break some perks). The **main file** from Dual Wield Behavior Fixes is required and these patches won't work without it.
