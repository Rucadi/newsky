# Requiem - Magic Sneak Attacks
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78519
**Fixed an issue with the Requiem Perk "Anatomical Lore" granting x5 to magic sneak attacks.
  
This has been nerfed to x2, which is equivalent to the bow increase.
  
  
Requires both [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Magic Sneak Attacks](https://www.nexusmods.com/skyrimspecialedition/mods/67613).
