# Fortified Whiterun - Immersive Clutter Patch
- Author: gutmaw
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68386
[font=Times New Roman]**I love [Fortified Whiterun](http://www.nexusmods.com/skyrimspecialedition/mods/40094), but I thought it'd be more immersive if I...**
  
[/font]

  
* [font=Times New Roman]Swapped all static clutter to the appropriate dynamic clutter - static books to readable books, static shield to actual shield, etc. [/font]
[font=Times New Roman]Added every piece of dynamic clutter and container to the GuardFactionSolitude, so you can't just freely take things from the guard posts, you'll need to steal them if you want them.[/font]
  
* [font=Times New Roman]Removed any crossbows laying around - they don't make sense here (unless u have mods that add crossbows from the start I suppose)[/font]
[font=Times New Roman]Swapped a few non-lore friendly items (Dark Elf table for a common table, etc)[/font]
  
* [font=Times New Roman]Swapped to a better vanilla texture form the big bell[/font]
[font=Times New Roman]ESL flagged plugin[/font]
  

[font=Times New Roman]
  
I play a pretty heavily modded game and couldn't find any conflicts, let me know if you do.
  
  
Enjoy!
