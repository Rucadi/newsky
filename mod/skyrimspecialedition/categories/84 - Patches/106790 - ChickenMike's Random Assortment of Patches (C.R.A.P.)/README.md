# ChickenMike's Random Assortment of Patches (C.R.A.P.)
- Author: ChickenMike FelesNoctis
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106790


I needed a place to upload the random patches I make that I either can't fit into any of my other collections, or am too lazy to make a whole mod page for. So I made this. Please bother me if you have an issue with my patches not the original mod authors. All plugins are .ESL flagged .ESPs so they won't take up a place in your load order. If there is an issue with my FOMOD please politely point it out. 
  
  

These patches require [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441) to function properly on older game versions. 
  
  
IT IS RECOMMENDED THAT YOU ADD THESE PATCHES TO A NEW GAME. To quote [robertgk2017](https://www.nexusmods.com/skyrimspecialedition/users/5544506) "Modding is illegal post character start."﻿ You might be able to get away with it but this is neither recommended nor supported.
