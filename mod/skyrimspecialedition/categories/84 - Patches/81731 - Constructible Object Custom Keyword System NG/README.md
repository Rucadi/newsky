# Constructible Object Custom Keyword System NG
- Author: Parapets-Meridiano
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/81731
NG patch for [Constructible Object Custom Keyword System](https://www.nexusmods.com/skyrimspecialedition/mods/81409) mod by Parapets.
  
  
Original DLLs work for specific versions only:
  
-- SE/AE 1.6.1130+ (Steam)
  
-- VR 1.4.15 (Steam)
  
  
[1.5-Patched DLL](https://www.nexusmods.com/skyrimspecialedition/mods/81723) works for specific version only:
  
-- SE 1.5.97 (Steam).
  
  
This version brings the power of [CommonLibSSE-NG](https://github.com/CharmedBaryon/CommonLibSSE-NG) by CharmedBaryon making the DLL working for many SE and AE and VR versions including:
  
-- SE 1.5.97 (Steam).
  
-- SE/AE 1.6.317-1.6.1170 (Steam).
  
-- SE/AE 1.6.659 and 1.6.1179 (GOG).
  
-- VR 1.4.15 (Steam).
