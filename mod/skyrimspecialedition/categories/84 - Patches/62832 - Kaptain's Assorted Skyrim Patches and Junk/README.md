# Kaptain's Assorted Skyrim Patches and Junk
- Author: KaptainCnucklz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/62832


Random patches I made that address conflicts and issues of varying severity. Only doing mods I like or use. See the files tab for options.
  
Cross reference the date of upload with it's dependencies' dates of uploads if you're ever worried about "is this compatibility with version x, y, or z?"
  
  
**Disclaimer: Everything provide as-is.** I am indeed only a human, so quirks or oversights may happen. I fix 'em as I sees 'em. :)
  
  
Please remember that many incompatibilities, like edits to form lists and leveled lists, can be smoothed over through a quick bashed patch. Thus (~~look at me being all fancy and using "thus" in 2025 lmao~~), not every incompatibility I run across will get a dedicated plugin. Many should instead be fixed by running a bashed patch and double checking changes in SSEEdit. So make sure you get familiar with Wrye Bash bashed patches and SSEEdit! :)
  
  
This most likely will be a growing collection of patches as time goes on. Or it might not. Arkay doesn't grant me visions so I don't know.
