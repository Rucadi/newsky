# Mesh Patch for Various Mods
- Author: captainlei1993
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/45597


[font=Comic Sans MS]If you have [Skyrim Particle Patch for ENB](http://enbseries.enbdev.com/forum/viewtopic.php?t=1499)﻿, [SMIM](http://www.nexusmods.com/skyrimspecialedition/mods/659), [Blended Roads](http://www.nexusmods.com/skyrimspecialedition/mods/8834), [ELFX](http://www.nexusmods.com/skyrimspecialedition/mods/2424),﻿ [Assorted Mesh Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/32117)﻿, [Unofficial Material Fix](http://www.nexusmods.com/skyrimspecialedition/mods/21027), [Dlizzio's Mesh Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/39260)﻿ and [Skyrim Landscape and Water Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/26138)﻿ in your load order, you'll find a few mesh conflicts between each other that no one bothers to patch.  Now you have an OCD patch for you, hooray!
  
  
*What my patches actually do:*
  
My patches either combined changes from conflicting mesh fix mods or applied mesh fixes to [SMIM](http://www.nexusmods.com/skyrimspecialedition/mods/659), [Blended Roads](http://www.nexusmods.com/skyrimspecialedition/mods/8834) or [ELFX](http://www.nexusmods.com/skyrimspecialedition/mods/2424) models.
  
  
[/font]
  
[font=Comic Sans MS]*Load Order Instructions:*[/font]
  
[font=Comic Sans MS]The load order for assets(left panel in mo2) should be [Skyrim Particle Patch for ENB](http://enbseries.enbdev.com/forum/viewtopic.php?t=1499)->[SMIM](http://www.nexusmods.com/skyrimspecialedition/mods/659)->[ELFX](http://www.nexusmods.com/skyrimspecialedition/mods/2424)->[SMIM - Quality Addon](http://www.nexusmods.com/skyrimspecialedition/mods/44388)->[Skyrim Landscape and Water Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/26138)->[Unofficial Material Fix](http://www.nexusmods.com/skyrimspecialedition/mods/21027)->[Assorted Mesh Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/32117)->[Dlizzio's Mesh Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/39260), otherwise you'll need some additional patches to be conflict-free. [Blended Roads](http://www.nexusmods.com/skyrimspecialedition/mods/8834) shouldn't matter as long as it's placed below [SMIM](http://www.nexusmods.com/skyrimspecialedition/mods/659).
  
  
If you have [No Sunlight Through Mountains](http://www.nexusmods.com/skyrimspecialedition/mods/45401)﻿ installed, simply put it under [Skyrim Landscape and Water Fixes](http://www.nexusmods.com/skyrimspecialedition/mods/26138) then you're good to go.
  
  
Assorted Mesh Fixes - SMIM - Blended Roads Patch should be placed under SMIM- Assorted Mesh Fixes Patch.
  
  
SMIM - Quality Addon - Unofficial Material Fix Patch should be placed under Skyrim Particle Patch for ENB - ELFX - Unofficial Material Fix Patch.
