# Missives - Bigger Trigger Box
- Author: Missile
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67772
﻿[Missives](https://www.nexusmods.com/skyrimspecialedition/mods/17576)﻿

  
  
Makes the trigger box that starts the generation of missives in the board bigger to prevent having to wait at the board for a couple seconds for it to generate new missives. Was made with version 2.03 of Missives. (ESL Flagged)
  
  
NOTE: HAS TO BE LOADED **AFTER** ANY MISSIVES PATCHES THAT MOVE THE MISSIVES BOARD TO PREVENT THE TRIGGER BOXES FROM REVERTING BACK
