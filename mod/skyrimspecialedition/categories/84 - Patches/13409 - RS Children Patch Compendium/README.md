# RS Children Patch Compendium
- Author: GamerGirlBones
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/13409
This is a collection of RS Children patches for certain mods that are available for SE, and some that are not. Before listing the patches:
  
  
**DISCLAIMER:** 
  
  
  
- I am not the modder, just the caretaker of the page. I volunteered to handle it because I believe in sharing and helping between mod users.
  
  
  
- Some of the patches are for mods that have NOT been ported publicly. You will have to either find them or port them by yourselves.
  
  
We will NOT provide ported mods that haven't been approved/original modder is absent/mod is not available anymore. Find your way.
  
  
  
- Be nice, you're given this freely.
  
  
  
- Some of these mods already have a RS Children patch, choose whichever you prefer.
  
  
  
- Suggestions might be considered.
  
  
  
  
ALL OF THESE PATCHES NEED [RS CHILDREN OVERHAUL](https://www.nexusmods.com/skyrimspecialedition/mods/2650/?) AND THE ORIGINAL MODS TO WORK.
