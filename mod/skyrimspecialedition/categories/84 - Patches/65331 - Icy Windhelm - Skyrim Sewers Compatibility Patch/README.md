# Icy Windhelm - Skyrim Sewers Compatibility Patch
- Author: wizkid34 and viltuska and LordMithro
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65331
**SUMMARY:**
  
This mod resolves one incompatibility between Icy Windhelm and Skyrim Sewers.
  
  
WHAT THIS MOD DOES:
  

  
* Moves meshes to allow the mods to be compatible.

  
Recommended Mods:
  
  
 [Rallyeator](https://www.nexusmods.com/skyrimspecialedition/users/39335975)'s [Manhole texture](https://www.nexusmods.com/skyrimspecialedition/mods/28804)﻿
