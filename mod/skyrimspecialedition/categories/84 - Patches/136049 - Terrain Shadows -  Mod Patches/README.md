# Terrain Shadows -  Mod Patches
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136049


This mod page hosts several height map of worldspaces added by several popular island mods which is needed by terrain shadow mod. This mod's purpose is for people who don't want to generate and rename the height map with xlodgen themselves. You can save a few minutes per mod by just downloading from this mod page. Enjoy!
