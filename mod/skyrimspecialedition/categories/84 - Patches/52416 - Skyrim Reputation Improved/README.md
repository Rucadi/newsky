# Skyrim Reputation Improved
- Author: kieranjohn1
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/52416


Several bugfixes and new features for Skyrim Reputation.
  
  
Bugfixes
  
  

  
* Fix several statistics being incremented at the beginning of the game.

  
* Excluded several quest related NPCs from the mod.
Assault count randomly increasing.
  
* Thanes are now recognised anywhere in the city.
Several quest check fixes.
  
* Fix Cidhna Mine prisoners being hostile.

  
Features
  
  

  
* Black soul trap tracking (Requires papyrus extender).
Cannibalism tracking (Requires papyrus extender).
  
* Stat multipliers.
An option to disable load screens.
  
* Removed custom vampire race option and replaced all checks with HasKeyword(Vampire).
Remove duplicate dialogue if [Guard Dialogue Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/22075)﻿ is installed.
  
* Support for the Gray Cowl from [Sneak Tools](https://www.nexusmods.com/skyrimspecialedition/mods/1863)﻿, [The Gray Cowl of Nocturnal](https://www.nexusmods.com/skyrimspecialedition/mods/4509)﻿, and the Creation Club. Wearing the cowl will make you disliked and stop all influences on your reputation.
[Skyrim Unbound](https://www.nexusmods.com/skyrimspecialedition/mods/6532)﻿ / [Skyrim Unbound Reborn](https://www.nexusmods.com/skyrimspecialedition/mods/27962) ﻿support. You won't gain reputation or thane status from the quests that are skipped by Skyrim Unbound.
  
* An option to automatically load presets when starting a new game (preset must be saved using [JContainers](https://www.nexusmods.com/skyrimspecialedition/mods/16495)﻿).

  
Changes from [Skyrim Reputation - Fixed and Patched](https://www.nexusmods.com/skyrimspecialedition/mods/42538) are forwarded where appropriate. Load this mod later.
  
Load order:
  

  
* SkyrimReputation\_SSE.esp
Skyrim Reputation Patch.esp (optional)
  
* Skyrim Reputation Improved.esp

ESL flagged, so won't take up a slot in your load order.
