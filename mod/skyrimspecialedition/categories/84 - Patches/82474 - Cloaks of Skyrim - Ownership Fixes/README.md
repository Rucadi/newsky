[font=Times New Roman]ABOUT[/font]
  
Simple patch that removes player ownership from every cloak added by Cloaks of Skyrim﻿. By default, every cloak from CoS is owned by player, so you can take them from everywhere, no matter what (it doesn't make much sense). This patch fixes that problem.
