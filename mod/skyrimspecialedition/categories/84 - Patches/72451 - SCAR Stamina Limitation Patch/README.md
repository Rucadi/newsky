# SCAR Stamina Limitation Patch
- Author: Maxsu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72451


[font=Georgia]Just a simple esp plugin that set minimum stamina requirement for NPCs perform SCAR combos attacking, so NPCs will not continually attacking you like a crazy dog.
  
  
Normal attack require 5 point of stamina while power attack require 30 by default. Best when using together with Fenix Stamina Overhaul. 
  
  
**How to Install:**
  
  

  
* [font=Georgia]Install the mod files with mod manager.[/font]

  

  
* [font=Georgia]Load "scar\_adxp\_staminaLimit.esp" after "scar-adxp-patch.esp".[/font]

  
You can confiugre the minimum stamina by enter console commnad: 
  
  
[/font]

```
Set ADXP_NormalAttackStaminaMin to xxx
  
Set ADXP_PowerAttackStaminaMin to xxx
```
