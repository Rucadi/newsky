Skyrim on Skooma Changes:
  

  
1. Disabled Skyrim on Skooma's addiction system in favor of Gourmet's. Dialogue options will check for Gourmet's effects to account for this
Drugged screen effects won't play while a Skyrim on Skooma trip is active
  
2. You can disable Gourmet's visual effects completely by typing this in the console: set GourmetDisableVisualEffects to 1
Eversnow can take you on trips!
  
3. Recommended to have [FLM](https://www.nexusmods.com/skyrimspecialedition/mods/74037)﻿ installed!

  
If you do not like the "Restores Hunger" tooltip on food, install [Hide Hunger Amount](https://www.nexusmods.com/skyrimspecialedition/mods/65464)﻿!
