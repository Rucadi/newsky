# Requiem - Seasonal Plant Fixes
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119030
﻿Fixes issues with the seasonal plants added by [Seasonal Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/66903)﻿ and [Cathedral 3D Mountain Flowers](https://www.nexusmods.com/skyrimspecialedition/mods/41312)﻿.
  
  
*- Seasonal plants now yield the intended amount of ingredients (previously seasonal variants only provided the vanilla amount, which is 1)*
  
  
  
Compatible with mods such as [Cathedral 3D Mountain Flowers - Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60756), [Seasonal Alchemy](https://www.nexusmods.com/skyrimspecialedition/mods/63969), and [Requiem - Reduced Ingredients](https://www.nexusmods.com/skyrimspecialedition/mods/67234)﻿.
