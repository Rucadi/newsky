# No more glitchy textures. Witcher Horse Expansion compatibility patches (ENG and RUS)
- Author: PharaoN_Man
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/75512


Witcher Horse Expansion is a great mod, but unfortunately he has some compatibility issues with other mods that change or add new horses. So I tried to fix some of them that I found while playing on Skyrim.
  
All patches a simple and ESL-ified so don't worry about your load order slots.
