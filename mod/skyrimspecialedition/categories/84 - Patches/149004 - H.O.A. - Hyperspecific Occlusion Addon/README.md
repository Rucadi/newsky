**Hyperspecific Occlusion Addon**

  
This is a small but very focused addon that adds or fixes **[occlusion planes](https://ck.uesp.net/wiki/Bethesda_Tutorial_Optimization)** in cities, towns, and settlements that other mods have overhauled. Many city/town mods overhauls skip proper occlusion setup, which can hurt performance and make your game render stuff it shouldn't. This mod aims to fix that. Nothing fancy, just hand-placed occlusion planes where they’re needed most.
  
  
This mod is not compatible with eFPS. It will never be patched. Use this mod or that one. Not both. You will have repeatable crashes. 
  
  
**Requirement**
  
[Reading](https://en.m.wikipedia.org/wiki/Reading) \*Very Important
  
If you are using [Capital Whiterun Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/37982) all our patches are based on [Rob's Bug Fixes - Capital Whiterun Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/63355).
  
It is important to understand that these plugins add custom occlusion for the exact combination of underlying mods, If you do not have that exact combination you will have disappearing objects and/or repeatable crashes.

**FAQ**

  
* **Is this flagged ESL?**
    
  Yes.
**Can I install this mid-game?**
  
Yes.
  
  
* **Can you patch for eFPS?**
    
  We don't recommend using eFPS with this mod as it will likely lead to crashes.
**Can you patch X-Mod?**
  
It depends on the scope of the mod/work and whether ChickenMike or I happen to use that mod ourselves.
  
  

  
* **Are there any other mods like this that you recommend?**
