# Lively's Dwemer Gates Don't Reset Patches
- Author: Lively
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/84138


Patches for [Dwemer Gates Don't Reset](https://www.nexusmods.com/skyrimspecialedition/mods/26331) for the following mods:
  
  
[The Second Great War](https://www.nexusmods.com/skyrimspecialedition/mods/17204)
  
[Inigo](https://www.nexusmods.com/skyrimspecialedition/mods/1461)
  
[The Timelost Dwemer](https://www.nexusmods.com/skyrimspecialedition/mods/22499)
  
  
The FOMOD will autoselect patches based on your load order. All plugins are ESP flagged as ESL so they don't take a full plugin slot. Yay.
