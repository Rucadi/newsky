# Jesper the Guard - Lollygaggin' Patch SPID
- Author: Pixellz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/143472


**----------------------------
  
DESCRIPTION:**
  
  
A simple patch between [Lollygaggin' - Vanilla Guard Armor Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/117833) and [Jesper the Guard - A Custom Voiced Follower (From the Series Skyrim Guard Tales)](https://www.nexusmods.com/skyrimspecialedition/mods/132362)Jesper has an outfit that points to the vanilla whiterun guard outfit. So, if you use both mods in conjunction, you'll see some incompatibilites.
  
  
For instance:
  
  
- if you load Lollygaggin' before Jesper, the outfits for Whiterun Guards of Lollygaggin' gets replaced to the vanilla ones.
  
- if you load Jesper before Lollygaggin, Jasper will still uses the vanilla Whiterun Guard outfit.
  
  
This patch is a compatibility from both mods. It creates a outfit exclusively for Jesper, using the outfit from Lollygaggin' ﻿(since the .esp file) and then uses SPID to distribute this outfit to Jesper.
  
  
  
**----------------------------
  
LOAD ORDER:**
  
  
*- GuardTalesFollowerMod.esp
  
- Lollygaggin Fashions.esp
  
- Jesper - Lollygaggin Patch SPID.esp (this mod)*(the esp is flagged as esl, so does not count into your plugin limit)
  
  
  
**----------------------------
  
COMPATIBILITY﻿:**
  
  
Since it uses a SPID file to distribute the outfit, it will not conflict with any mod that edits/modifies Jesper (tweaks, replacers, etc)
