# General Stores  Cloud Storage and Ordinator - Dynamic Activation Key
- Author: Pete McPete
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/97041


This Patch add Dynamic Activation Key functionality to General Stores Cloud Storage so it does not conflict with Ordinator and possibly other Mods that add an event to crafting stations.
  
  
If you use **Dienes\_Crafting\_Storage(GS).esp** from General Stores, this automatically pulls your crafting materials from the cloud storage. However, the activator for this breaks functionality from other mods that use activators, like the 'Gem dust' perk from [Ordinator](https://www.nexusmods.com/skyrimspecialedition/mods/1137).
  
  
Using the fantastic [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273) this patch adds DAK fuctionality to [General Stores - Auto Sorting Cloud Storage Resource for packrats SE Convert](https://www.nexusmods.com/skyrimspecialedition/mods/4322), specifically the Crafting Bench Script update-Idontevenknow file: **Dienes\_Crafting\_Storage(GS).esp**
  
  
Requires:
  
[Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273)﻿ and all of its requirements
  
[Ordinator](https://www.nexusmods.com/skyrimspecialedition/mods/1137) (if using main version. Not required for the Optional file)
  
[General Stores - Auto Sorting Cloud Storage Resource forpackrats SE Convert](https://www.nexusmods.com/skyrimspecialedition/mods/4322)﻿,
  
Crafting Bench Script update-Idontevenknow (**Dienes\_Crafting\_Storage(GS).esp**) from the general stores page above.
  
  
Load order
  
**PMP\_GS\_CCS\_DAK.esp** must be below Dienes\_Crafting\_Storage(GS).esp
  
  
Is an ESL flagged ESP.
  
  
Usage
  
To use the your cloud storage, press your Dynamic Activation Key (default is shift) whilst activating the crafting bench.
  
To use your Ordinator Perk/other mod that adds activators, use bench as normal.
  
  
Known issues
  

  
* Using the bench as normal won't pull through your materials, so grab them first if you want to use Gem Dust etc.
If you upgrade your bench/table - you **must** retrieve your materials first and then use bench as normal without DAK pressed to benefit from the upgrade buffs.
