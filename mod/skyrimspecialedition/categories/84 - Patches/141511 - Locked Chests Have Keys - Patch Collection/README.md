# Locked Chests Have Keys - Patch Collection
- Author: ChickenMike
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/141511


[Locked Chests Have Keys](https://www.nexusmods.com/skyrimspecialedition/mods/74565) is a great mod. It just has one tiny problem. The conflict surface is MASSIVE on it. This page attempts to alleviate some of that. Please bother me if you have an issues with my patches not [jayserpa](https://www.nexusmods.com/skyrimspecialedition/users/5201727). All plugins are .ESL flagged .ESPs so they won't take up a place in your load order.
  
  

IT IS RECOMMENDED THAT YOU ADD THESE PATCHES TO A NEW GAME. To quote [robertgk2017](https://www.nexusmods.com/skyrimspecialedition/users/5544506) "Modding is illegal post character start."﻿ You might be able to get away with it if you haven't entered the cell yet but this is neither recommended nor supported. 
  
  
These patches require [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441) to function properly on older game versions. ﻿
  
  
  
All of the patches basically do some combination of four things, either they make sure the key is in an accessible position, ensure the chest is in the correct position, make the LCHK keys open chests added by other mods and ensure the key will actually unlock the chest.
