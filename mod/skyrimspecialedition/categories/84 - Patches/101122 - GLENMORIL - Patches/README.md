# GLENMORIL - Patches
- Author: Vicn - DarShonDo - ToosTruus - Foamimi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/101122


Hello! I recently started the deep dive into [GLENMORIL](https://www.nexusmods.com/skyrimspecialedition/mods/32998) and noticed a few things that conflicted with my load order. I threw these together for consistency's sake and thought it couldn't hurt to share them here. I may add more depending on what I stumble across.
  
  
Patches
  
  
[High Poly Sweet Rolls](https://www.nexusmods.com/skyrimspecialedition/mods/44077) - Changes the default pathing from each variation of sweet roll throughout the questline (costume, dessert plate, and golden artifact) and introduces a new path, utilizing textures from the mod [High Quality Food and Ingredients SE](https://www.nexusmods.com/skyrimspecialedition/mods/10897). Without this patch, the textures on the mentioned variants will be misaligned.**It's recommended to utilize alongside [Glenmoril - Sweetrolls Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/52319?tab=files) (Optional Files)!**
  
  
**Lalanoah Uses Your Default Textures** - Changes the diffuse, normal, subsurface, and specular pathing from the custom texture set introduced by Glenmoril to the set the player and NPCs utilize. This is beneficial if you use body/feet/hand replacers, such as [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/198), that change the vanilla foot into a higher poly one. **This is an ESPFE plugin.**
  
  
**Supreme Weathers Integration** - Courtesy of [ToosTruus](https://www.nexusmods.com/skyrimspecialedition/users/7389257), I've been allowed to implement standalone supreme weather types that will occur throughout select worldspaces via Glenmoril's storyline. The locations impacted include; **Cradle Forest** (supreme fog), **Quagmire** (supreme rain), and **Piper's Hill** (supreme snow). **Patches are available** for [Obsidian Weathers](https://www.nexusmods.com/skyrimspecialedition/mods/12125) and [Azurite Weathers III](https://www.nexusmods.com/skyrimspecialedition/mods/42731) (will NOT work for Azurite I or II). 
  
  
**No Grass in Sanctuary of Namira** - Simply removes the grass within the worldspace. Useful if running a dense or tall grass mod. Comparison image shows usage with [Freak's Flora Fields](https://www.nexusmods.com/skyrimspecialedition/mods/125349). Be sure to load this mod AFTER [Lux](https://www.nexusmods.com/skyrimspecialedition/mods/43158)﻿ if using it.
