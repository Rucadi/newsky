# Wintersun Description Fixes
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70254


Fixes an issue with descriptions while choosing a god.
  
  
Includes a patch for [Wintersun - Tribunal Integration](https://www.nexusmods.com/skyrimspecialedition/mods/63012) ﻿
  
  
**When choosing a god the message box has a lot of text and therefore not everything is readable in some cases.**
  
***This mod removes a somewhat redundant message which fixes this issue. (e.g. "Follow the Divine Stendarr?")
