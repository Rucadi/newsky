# Dwemer Gates Don't Reset - Patches
- Author: Blackread
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/113264


A few patches for [Dwemer Gates Don't Reset](https://www.nexusmods.com/skyrimspecialedition/mods/26331) that I didn't see covered elsewhere.
  
  
**Patches:**
  
These are compatibility patches that resolve conflicts between Dwemer Gates Don't Reset and other mods. They are for the complete .esl version only. Covered mods:
  
[Blood and Silver - Cidhna Mine Expanded](https://www.nexusmods.com/skyrimspecialedition/mods/20269)
  
[Navigator](https://www.nexusmods.com/skyrimspecialedition/mods/52641)
  
[Xelzaz](https://www.nexusmods.com/skyrimspecialedition/mods/62893)
  
  
**Integration:**
  
These patches expand the Dwemer Gates Don't Reset functionality to new locations added by mods. They don't depend on Dwemer Gates Don't Reset so can be used with any version (or even without it), but they follow the same pattern as the complete version. Covered mods:
  
[EasierRider's Dungeon Pack](https://www.nexusmods.com/skyrimspecialedition/mods/2218)
  
[The Elder Scrolls Online Imports](https://www.nexusmods.com/skyrimspecialedition/mods/11729)
  
[Forgotten Dungeons](https://www.nexusmods.com/skyrimspecialedition/mods/449)
  
[Hammet's Dungeon Pack 1](https://www.nexusmods.com/skyrimspecialedition/mods/12186)
  
[Hammet Dungeon Pack 2](https://www.nexusmods.com/skyrimspecialedition/mods/95339)
  
  
**Compatibility:**
  
[Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802) - Patch not needed, load [size=3][Dwemer Gates Don't Reset](https://www.nexusmods.com/skyrimspecialedition/mods/26331) later
