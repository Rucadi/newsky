# JK's Tel Mithryn Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/154109


**Description:**
  
A collection of compatibility patches for [JK's Tel Mithryn](https://www.nexusmods.com/skyrimspecialedition/mods/151512)﻿ to resolve various compatibility issues by:
  
- placement fixes for elements and/or markers, so they can match the modified exterior layout,
  
- simple conflict resolution preserving some changes from one mod over the other (usually in favour of JK's Tel Mithryn),
  
- navmesh and landscape adaptation.
  
  
**Additional notes:**
  
- All plugins are ESL-flagged and provided with a FOMOD installer.
  
- Patches are load order agnostic, so the mutual order of JK's Tel Mithryn and patched mod isn't relevant unless you load the patch after both mods.
  
- All plugins were made using the newest versions of Creation Kit and xEdit, therefore, they have a new 1.71 header. If you don't run Skyrim version 1.6.1130+, you need to install the mod Backported Extended ESL Support to use them.
