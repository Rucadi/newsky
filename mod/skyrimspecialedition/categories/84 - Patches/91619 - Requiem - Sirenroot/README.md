# Requiem - Sirenroot
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/91619
**Balance patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [SIRENROOT - Deluge of Deceit](https://www.nexusmods.com/skyrimspecialedition/mods/70917).** 
  
  
- Rebalanced all new NPCs and enemies, they now have a static level (level 25-40)
  
- Rewrote some messages to better fit Requiem
  
- Rebalanced an item and a container
  
  
- This mod doesn't rebalance many of the enemies since Sirenroot uses already existing ones *(meaning that your other mods and addons will determine how hard the dungeon will be)*
