# Requiem - Ryn's Mehrunes Dagon's Shrine
- Author: MadAborModding
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/118908
**Overview**This patch fixes incompatibilities between [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Ryn's Mehrunes Dagon's Shrine](https://www.nexusmods.com/skyrimspecialedition/mods/88451). Without this patch, items and NPCs added by Requiem will be either floating or out-of-bounds. This patch fixes that and patches new NPCs added by Ryn to use Requiem stats.
  
  
**Specific Changes:**-Requiem-added items and the dead wizard are now located near the enchantment table in the first room and are no longer floating mid-air. (See Photo).
  
-Slighted NPCs no longer spawn out-of-bounds. Two are found in the main chamber and the third is found in new chamber added by Ryn.
  
-NPCs added by Ryn now use Requiem Templates.
  
  
**Load Order:**
  
Ryn's Mehrunes Dagon Shrine.esp
  
Requiem.esp
  
Requiem - Ryn's Mehrunes Dagon Shrine.esp
