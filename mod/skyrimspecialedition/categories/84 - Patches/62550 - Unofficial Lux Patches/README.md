# Unofficial Lux Patches
- Author: GGUNIT and Skurkbro Nexus Community
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/62550
This mod only changes 2 variables in the references (**Lighting template** and **Image lighting**), it doesn't add anything new or change positions of existing lights. The results are not gonna be as pretty as an official patch by GGUNIT, but this works as a temp solution to remove the "flatness" of all interior lighting.
  
  
To be used with **[ENB Light](http://www.nexusmods.com/skyrimspecialedition/mods/22574)** (Stronlgy recommend patching your load order with **[this](http://www.nexusmods.com/skyrimspecialedition/mods/42180)**) and an **ENB Preset** with **PointLightingDesaturationInteriorDay=0.0** and **PointLightingDesaturationInteriorNight=0.0**
