# Amazing Lockpicks - Enhancements
- Author: Eudalus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109455


**Description**Enhances [Amazing Lockpicks](https://www.nexusmods.com/skyrimspecialedition/mods/108832), [Amazing Lockpicks - Additions](https://www.nexusmods.com/skyrimspecialedition/mods/109452), and [Amazing Lockpicks - Radiance](https://www.nexusmods.com/skyrimspecialedition/mods/109667) through usage of mods such as [Inventory Interface Information Injector](https://www.nexusmods.com/skyrimspecialedition/mods/85702), [Sound Record Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/77815), and [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805).
  
  
Includes unique models and textures for [Amazing Lockpicks](https://www.nexusmods.com/skyrimspecialedition/mods/108832) and [Amazing Lockpicks - Additions](https://www.nexusmods.com/skyrimspecialedition/mods/109452).
  
  
Includes .json files for [Inventory Interface Information Injector](https://www.nexusmods.com/skyrimspecialedition/mods/85702) to improve lockpick menu icons and text.
  
  
Includes .yaml files for [Sound Record Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/77815) to add [Immersive Sounds - Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/523) to lockpicks.
  
  
Includes \_SWAP.ini files for [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) to replace the hand-placed vanilla lockpicks lying around in game with unique lockpicks.
  
  
**﻿Installation**
  
Install following the fomod instructions.
