# JK's Solitude Outskirts Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/103927


A collection of compatibility patches for [JK's Solitude Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/103209). All patches are ESPs flagged as ESL.
  
  
**Patches descriptions:**

**Additional notes:**
  
- some of the fixes in question are made on persistent references and require a new save to work properly,
  
- patch for [Animated Ships](https://www.nexusmods.com/skyrimspecialedition/mods/110260) is available on [Animated Ships Patches](https://www.nexusmods.com/skyrimspecialedition/mods/111556) thanks to RavenKZP
  
  
**Load order suggestion:**
  
1) most of the patches
  
2) Enhanced Solitude patch
  
2) Lux Orbis / Via patch (their relative order isn't important if you use both mods)
  
3) Solitude Watchtower / CACO Farm Overhaul / Grand Solitude patch
  
4) Solitude Docks Updated patches
  
5) Northern Roads patch
  
6) Northern Roads + Solitude Docks Updated patch
  
7) Northern Roads + KHW patch
  
8) Northern Roads + Grand Solitude patch
