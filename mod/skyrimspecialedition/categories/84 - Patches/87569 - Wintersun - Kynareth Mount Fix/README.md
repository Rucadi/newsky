# Wintersun - Kynareth Mount Fix
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/87569
**Fixes an issue with the mount gained from Devotee of Kynareth being unable to turn.
  
  
This bug is not an issue with Wintersun itself but is caused because of other mods in combination. As far as I'm aware this issue arises if [TDM](https://www.nexusmods.com/skyrimspecialedition/mods/51614) is used. (not tested)
