# Undeath - Immersive Mountain Pass
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/84544
**Adds a proper way that leads to the Dragonclaw Mountains and a way back to Skyrim, instead of the need to fast travel there.**
  
Requires a save where the Undeath quest wasn't started yet.
  
  
Includes a version for base [Undeath](https://www.nexusmods.com/skyrimspecialedition/mods/6180) and [Undeath - Classical Lichdom](https://www.nexusmods.com/skyrimspecialedition/mods/40802)
  
  
*- Adds a cave that leads to the Dragonclaw Mountains and a way back to Skyrim
  
- There will be an undiscovered location marker added to your map as soon as you get the relevant quest
  
- The cave is only accessible when the quest leads you there
  
- Disabled the ability to fast travel to the Dragonclaw Mountains without discovering it
