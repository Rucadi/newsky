**Installation**
  
[left]**Step 1 - Core requirements**[/left]

  
* Install [Gourmet - A Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/96876)
Install [Eating Animations and Sounds SE](https://www.nexusmods.com/skyrimspecialedition/mods/42602)
  
* Install [AnimObject Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/75167)
Install [Keyword Item Distributor (KID)](https://www.nexusmods.com/skyrimspecialedition/mods/55728)
  

  
**Step 2 - [b]Requirements for c**ombo patches (optional)[/b]
  
  

  
* Install [Eating Animations - Fishing Patch](https://www.nexusmods.com/skyrimspecialedition/mods/63309)﻿ if you have CC Fishing

  

  
* Install [Eating Animations - Patch Collection](https://www.nexusmods.com/skyrimspecialedition/mods/63741) for all other combo patches

  

  
* Install Gourmet Patches from [it's main page](https://www.nexusmods.com/skyrimspecialedition/mods/96876?tab=files)﻿ for this mods:

﻿- CC Survival
  
 ﻿- CC Fishing
  
 ﻿- [USSEP](https://www.nexusmods.com/skyrimspecialedition/mods/266)
  

  
* Install Gourmet Patches from [it's patch page](https://www.nexusmods.com/skyrimspecialedition/mods/97050?tab=files)﻿ for this mods:

﻿- [Beyond Skyrim - Bruma SE](https://www.nexusmods.com/skyrimspecialedition/mods/10917)
  
 ﻿- [Wyrmstooth](https://www.nexusmods.com/skyrimspecialedition/mods/45565)
  
  
**Step 3 - Gourmet - EAS Patch**
  
  

  
* Install the main file (fomod) from this page. It will auto-detect all necessary patches if you installed all requirements properly
Optionally you can install all my patches separately, in case you don't want the fomod. If you choose this option then install the main patch from optional files, and then install necessary combo patches from misc files if you need them.
  

  
**Step 4 - Load order**
  
  

  
* Load patches after their respective mods

  
**Done!**
