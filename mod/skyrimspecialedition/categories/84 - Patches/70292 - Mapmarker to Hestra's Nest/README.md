# Mapmarker to Hestra's Nest
- Author: Arcane University and Betalille
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70292


The map marker can be found next to the cave added by the mod, only once you've been through the cave at least once. Also includes a mapmarker to go back to Skyrim when you are in Hestra's Nest.
  
  
ESPFE flagged.
