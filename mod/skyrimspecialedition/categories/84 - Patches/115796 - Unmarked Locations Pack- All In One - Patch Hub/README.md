**Welcome to the official patch hub for Unmarked Locations Pack - All In One. Here you can find compatibility list and patches for various mods.
  
  
Info:
  
Fomod installer has auto-detect feature
  
Fomod installer has two types of files:
  
ESL - ESP flagged as ESL
  
BOS - Base Object Swapper ini (made via [ESP to BOS'Inator](https://www.nexusmods.com/skyrimspecialedition/mods/111729))
