# Growl-Mortal Enemies Patch
- Author: lordkenyonkmp
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/31270
Makes sure Growl and Mortal Enemies play nice. Ensures all of Growl's changes to the werewolf race are present, while carrying over Mortal Enemies's changes to range and attack angle.
  
  
Compatible with all versions of Mortal Enemies.
  
Last checked for compatibility in xEdit with Growl 3.0
  
Must load below both Growl and Mortal Enemies!
