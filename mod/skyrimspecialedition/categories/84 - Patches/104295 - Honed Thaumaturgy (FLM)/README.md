# Honed Thaumaturgy (FLM)
- Author: daken72127
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/104295


A patch which allows [Honed Metal](https://www.nexusmods.com/skyrimspecialedition/mods/61015) to have access to enchantments added by [Thaumaturgy](https://www.nexusmods.com/skyrimspecialedition/mods/57138) using [FormList Manipulator](https://www.nexusmods.com/skyrimspecialedition/mods/74037). Optional plugin version for players who dont use FLM.
  
  
Requirements:
  
- [Honed Metal -NPC Crafting and Enchanting Services-](https://www.nexusmods.com/skyrimspecialedition/mods/61015)
  
- [Thaumaturgy - An Enchanting Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/57138)
  
- [FormList Manipulator - FLM](https://www.nexusmods.com/skyrimspecialedition/mods/74037) (Optional)
