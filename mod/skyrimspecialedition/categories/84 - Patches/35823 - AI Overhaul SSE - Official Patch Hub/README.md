**SUPPORTED PATCHES:**
  
[patches made by me or send to me by users]
  
[Cutting Room Floor](https://www.nexusmods.com/skyrimspecialedition/mods/276)﻿ v3.1.11 (last non-AE version) by Arthmoor - CRF should be loaded before AIO. Patch combines one of the AI Packages from both mods. All conflicts resolved.
  
[Distinct Interiors](https://www.nexusmods.com/skyrimspecialedition/mods/6130)﻿﻿ by Sokkvabekk with [Distinct Interiors - Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/23831)﻿ by mnikjom - load order between both of them doesn't matter. Patch moves some idle markers to not interfere with objects added by Distinct Interiors.
  
[Relationship Dialogue Overhaul - RDO SE](https://www.nexusmods.com/skyrimspecialedition/mods/1187)﻿ by cloudedtruth - RDO should be loaded before AIO. Combines AI Packages from both mods.
  
[Relationship Dialogue Overhaul - Update and MCM](https://www.nexusmods.com/skyrimspecialedition/mods/44601)﻿﻿ by cloudedtruth and mnikjom - RDO Updated should be loaded before AIO. Combines changes from both mods.
  
[Denizens of Morthal](https://www.nexusmods.com/skyrimspecialedition/mods/33965)﻿ by Lollia - AIO should be loaded before DoM. Combines AI Packages from both mods.
