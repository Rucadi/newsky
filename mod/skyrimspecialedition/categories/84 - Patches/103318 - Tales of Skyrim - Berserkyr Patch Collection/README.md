# Tales of Skyrim - Berserkyr Patch Collection
- Author: Czasior and Monops
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/103318


A collection of patches for JPSteel's [Tales of Skyrim - Berserkyr](https://www.nexusmods.com/skyrimspecialedition/mods/103559).
  
  
Some of the patches were made with the use of the newest versions of Creation Kit and xEdit, therefore they have a new 1.71 header - if you don't run Skyrim version 1.6.1130+, you need to install the mod [Backported Extended ESL Support](https://www.nexusmods.com/skyrimspecialedition/mods/106441)to use them.
  
  
These are mostly simple conflict resolution or consistency patches, except for the Growl Integration patch which is further detailed in [this article](https://www.nexusmods.com/skyrimspecialedition/articles/5932).
