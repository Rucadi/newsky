# Requiem - Better Blacksmiths
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67236
**- Increased the amount of weapons/armor/ammunition sold by blacksmiths** (by about 10-20%)
  
  
Includes a patch for [Heavy Armory](https://www.nexusmods.com/skyrimspecialedition/mods/6308). Also requires the [Requiem Patch for Heavy Armory](https://www.nexusmods.com/skyrimspecialedition/mods/64410).
