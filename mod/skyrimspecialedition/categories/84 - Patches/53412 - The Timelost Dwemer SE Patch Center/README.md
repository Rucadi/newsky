**The TLD Patch Center contains the following files:**
  
  
  
  
**\* Essentials Patcher [b]*[color=#ffff00](Required)***[/color][/b]
  
 ﻿- contains various Timelost Dwemer textures to replace the race's default textures
  
 *﻿- make sure to load the Skin Patcher AFTER the main mod*
  
 ﻿- contains scripts and meshes to make the Multiballista, Manuscorpio, and the Autoballista work
  
  
  
**\* BVLT Patcher**
  
 ﻿- allows female dwemer vampires to use their humanoid model when transforming into a vampire lord
  
  
  
  

![](https://i.imgur.com/JmOFDUv.png)﻿
  
Spoiler
  
If you can't install due to a **"File Path too long"** error in your mod manager, download the mod manually from the site, and rename the .zip file that you downloaded to something shorter, like**"TLD.zip"**. Then try installing it again, manually.
