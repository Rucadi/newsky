# Experience - Creation Club Main Quest Patchs
- Author: XIENANBIN
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/77408


tweak main quest type of all Creation Club,most Creation Club main quest do not include keywords for the quest type.  'None' type are NOT seen as side quests thereby not giving XP.
  
  
this mods changes the ‘None’ type task to ‘Side’ Quest Type In this way, you can correctly obtain XP value when completing the quest.
  
  
english is not my native language. I'm no sure I can express my meaning well, but this is actually a very simple tweak.
  
  
Install/uninstall safely at any time.
  
  
  
Experience模组在完成任务获取经验的前提是该任务带有正确的关键词，例如Side、Main等，然而大部分CC任务的关键词为'None'，这导致在完成绝大多数CC的任务时不会获得经验，这个MOD更改了所有创意俱乐部主线任务内容，让玩家在使用Experience模组进行CC任务时正确获得经验。
