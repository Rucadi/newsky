# JK's Markarth Outskirts Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/95750


A collection of compatibility patches for [JK's Markarth Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/93006). All patches are ESP flagged as ESL.
  
  
**Patches descriptions:**
  
  
[Beyond Reach](https://www.nexusmods.com/skyrimspecialedition/mods/3008) - modifies the placement of the carriage. Last checked against version 4.61
  
  
[Birds of Skyrim SSE](https://www.dropbox.com/s/0zmfolkcxrpv5d0/Birds%20of%20Skyrim%20SSE%20Edition%202.7.7z?dl=0) - adjusts the initial of a few pigeons and a rooster. Last checked against version 2.7 - the last one available for SE.
  
  
[CACO Farm Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/36283) - a mix of both mods' changes for Salvius Farm. Last checked against version 1.0.1
  
  
[Carriage Drivers Are Alive](https://www.nexusmods.com/skyrimspecialedition/mods/51260) - adapts carriage driver's stuff into a new building, conflict resolution, and landscape adjustments. Last checked against version 7.8
  
  
[Carriage Stops of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/10434) - adjusts some elements to match the modified area layout. Last checked against version 1.6
  
  
[Clockwork](https://www.nexusmods.com/skyrimspecialedition/mods/4155) - adjusts the entrance area, navmesh finalization. Last checked against version 1.0
  
[Convenient Carriages](https://www.nexusmods.com/skyrimspecialedition/mods/12693) - adapts carriage driver's stuff into a new building, conflict resolution, and landscape adjustments. Last checked against version 5.9.7
  
  
[Creation Club: Fishing](https://en.uesp.net/wiki/Skyrim:Fishing) -  shifts one critter spawn marker
  
  
[Elder Scrolls Online Imports](https://www.nexusmods.com/skyrimspecialedition/mods/11729) - conflict resolution. Last checked against version 10.0.1
  
  
[Enhanced Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/18162) - conflict resolution on landscape records, switches/disables a bunch of added elements. Last checked against version 2.0.1
  
  
[Immersive Citizens - AI Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/173) - resolves landscape and navmesh conflicts forwarding and adjusting settings from both mods where necessary, and adapts carriage driver's stuff into a new building. Last checked against version 0.4.0.2
  
  
[Immersive Laundry](https://www.nexusmods.com/skyrimspecialedition/mods/2011) - shifts a few elements to avoid clipping. Last checked against version 1.0d
  
  
[Immersive World Encounters](https://www.nexusmods.com/skyrimspecialedition/mods/18330) - shifts one marker. Last checked against version 3.6.1
  
  
[Khajiit Has Wares](https://www.nexusmods.com/skyrimspecialedition/mods/73500) - conflict resolution on landscape and shifts the entire Ri'Saad tent (along with the clutter) placement. Last checked against version 1.08.1
  
  
[Lanterns of Skyrim II](https://www.nexusmods.com/skyrimspecialedition/mods/30817) - adjusts the placement/size and disables some light sources. Last checked against version 4.1
  
  
[Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802) - shifts shipment crates placement. Last checked against version 6.8
  
  
[Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095) - adjusts the placement/size/base and disables some light sources. Last checked against version 3.3
  
  
[Lux Via](https://www.nexusmods.com/skyrimspecialedition/mods/63588) - shifts and disables some records, resolves landscape conflicts, adjusts navmesh, and changes the lighting for consistency with Lux Via. Last checked against version 1.7
  
  
[Markarth Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/70213) - the best-of-both-worlds version of JK's and GGUNIT's Markarth Outskirts. ATTENTION! It's very performance-heavy! Last checked against version 1.6.1
  
  
[Rigmor of Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/5271) - adjusts a marker and activator. Last checked against version 1.1c
  
  
[Road Signs Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/55764) - adjusts the position of some elements, conflict resolution. Last checked against version 1.0
  
  
[Sky City - Markarth Rising](https://www.nexusmods.com/skyrimspecialedition/mods/22482) - removes the new empty buildings, and modifies the statue's placement. Last checked against version 4.21
  
  
[Skyrim Bridges](https://www.nexusmods.com/skyrimspecialedition/mods/21263) - combines the bridge changes from both mods. Last checked against version 3.2
  
  
[The Second Great War](https://www.nexusmods.com/skyrimspecialedition/mods/17204) - shifts the placement of a bunch of soldiers and a few markers. Last checked against version 1.10
  
  
[Traverse the Ulvenwald](https://www.nexusmods.com/skyrimspecialedition/mods/57874)﻿ - adjusts the placement of a few trees. Last checked against version 3.2.1
  
  
[Trees Addon SE](https://www.nexusmods.com/skyrimspecialedition/mods/22053) - modifies placement of a few added trees. Last checked against version 1.2.8
  
  
[Witcher Horse Expansion](https://www.nexusmods.com/skyrimspecialedition/mods/72490) - conflict resolution, placement adjustments on a few records. Last checked against version 2.0
  
  
**Additional notes:**
  
- some of the fixes in question are made on persistent references and require a new save to work properly,
  
- patches for Lanterns of Skyrim II and Lux Via - just like that main mods, aren't compatible with each other,
  
  
**Load order suggestion:**
  
1) most of the patches
  
2) Lux Via patch
  
3) CACO Farm Overhaul patch
  
4) Northern Roads patches
