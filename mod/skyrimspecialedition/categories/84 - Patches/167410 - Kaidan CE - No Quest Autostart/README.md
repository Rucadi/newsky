# Kaidan CE - No Quest Autostart
- Author: IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/167410


**Disables the [Kaidan Community Edition](https://www.nexusmods.com/skyrimspecialedition/mods/135345) startup quest.**
  
  
*The quest will no longer be present in the journal when starting a new playthrough. Instead the player needs to find Kaidan in the Abandoned Prison and talk to him to start the quest.*Compatible with Kaidan appearance mods and all other mods that don't touch the quest entry.
