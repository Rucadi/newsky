# Vigil Enforcer - Pilgrim Tweaks
- Author: Slampire
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/82069


This is a very simple patch between Pilgrim by SimonMagus616 and the Vigil Enforcer Creation, making the Shrine of Molag Bal found in the Creation consistent with Pilgrim. Now you too can worship Molag Bal surrounded by the corpses of the Vigilant!
