# Trees patches for Woodcutting Tweaks
- Author: Lolilofol
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136527


**Tree patches for [Woodcutting Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/53538)﻿ by [AndrealphusVIII](https://www.nexusmods.com/skyrimspecialedition/users/5646623) and major Skyrim tree enhancement mods.**
  
  
  
With the mod created by [AndrealphusVIII](https://www.nexusmods.com/skyrimspecialedition/users/5646623), you need to get logs from the trees using a lumberjack's axe, which you must have in your inventory so that you can use the logs to cut them into firewood.
  
  
Patches are required for tree enhancement mods so that [Woodcutting Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/53538)﻿ can work.
  
There is already a patch for [Skyrim Flora Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/2154) available as an option on the mod page, and one for [Happy Little Trees](https://www.nexusmods.com/skyrimspecialedition/mods/50961) on [this page](https://www.nexusmods.com/skyrimspecialedition/mods/65332) ﻿accompanied by a patch to make the logs lighter.
  
  
**Mod patched** 
  
- [Aspens Ablaze](https://www.nexusmods.com/skyrimspecialedition/mods/39998) Load this pach after the patch for Happy Little Trees or Happy Little Trees - Ivy on Trees if you use Aspen Ablaze simultaneously with them
  
  
- [Aspens Ablaze - Animated Branches](https://www.nexusmods.com/skyrimspecialedition/mods/115719)  Remember that you'll need to activate Aspens Ablaze - Animated Branches and then this patch only after you've generated your LODs.
  
  
- [Blubbos Aspen Replacer 2022](https://www.nexusmods.com/skyrimspecialedition/mods/65565)
  
  
- [Happy Little Trees - Ivy on Trees](https://www.nexusmods.com/skyrimspecialedition/mods/82469) You don't need to install the patch for Happy Little Trees from [size=3][this page](https://www.nexusmods.com/skyrimspecialedition/mods/65332)
