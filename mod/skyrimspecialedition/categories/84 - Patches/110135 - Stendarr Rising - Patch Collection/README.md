# Stendarr Rising - Patch Collection
- Author: Dudeling
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/110135


These are all ESL-flagged patches created to make the amazing [Stendarr Rising](https://www.nexusmods.com/skyrimspecialedition/mods/49346?tab=description)﻿ mod compatible with several popular mods, by merging leveled lists.
  
  
The mods currently patched are:[CACO](https://www.nexusmods.com/skyrimspecialedition/mods/19924)[Undeath](https://www.nexusmods.com/skyrimspecialedition/mods/40802)
  
[Obscure Magic](https://www.nexusmods.com/skyrimspecialedition/mods/103805)
  
If there are other mods you would like to have a patch for, contact me and I will see if I can provide one! Happy Modding ;)
