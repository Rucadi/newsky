# Lux Orbis - Markarth Entrance Overhaul
- Author: IxionXVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153635


Changes the lighting of [City Entrances Overhaul - Markarth](https://www.nexusmods.com/skyrimspecialedition/mods/19538) to match the lights from [Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095).
  
Compatible with [Markarth Entrance Overhaul Fixed](https://www.nexusmods.com/skyrimspecialedition/mods/37375) (highly recommended)
  
  
  
- Simply changes the lights from vanilla ones to the ones used by Lux Orbis to match the appearance.﻿ (check out the screenshots for a comparison)
