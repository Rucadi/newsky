# Lord of Coldharbour Tweaks and Unofficial Universal Patch with Harkon's Sword Replacer Option
- Author: Ja1zinZamp
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/56890


**DESCRIPTION**
  
 ﻿My tweaked version of [Lord of Coldharbour](https://www.nexusmods.com/skyrimspecialedition/mods/48302?tab=description)﻿ mod.
  
 ﻿Comes in different tastes, one that doesn't replace Harkon's Sword and another that does, replace his outfit or don't, remove the container in the Soul Cairn or don't, it is up to you, select during installation process.
  
**Just a warning, if you select an option that removes the Soul Cairn Container, the file will be ESPFE (ESP flagged ESL) and won't, technically count towards the 254 esp files limit, but you must use it on a save that has never ever seen this mod before!**
  
 ﻿Requires the original mod, of course! This is a "patch".
  
 ﻿SPID is a requirement for my changes 2 and 7 [See below] - OPTIONAL in the FOMOD
  
  
**MAIN CHANGES FROM THE ORIGINAL MOD - Optional changes available in the FOMOD**
  
  

  
1. Removed NPC records on Harkon in xEdit.
Added an enchanted version of the Helmet record, and added it to Harkon's Inventory via [SPID](https://www.nexusmods.com/skyrimspecialedition/mods/36869?tab=description)﻿, and he should drop after the boss fight (not fully tested)
  
2. Increased the values of some items.
Changed the Harkon outfit armor pieces to the enchanted version.
  
3. Changed Harkon's sword enchantment magnitude from 15 to 20.
Replaced Harkon's sword with the Montane Greatsword (same enchantment mentioned on item 4) [Optional file without this change available]
  
4. Added the enchanted version of the Rapier to Valerica and give it a *unique* name, via [SPID](https://www.nexusmods.com/skyrimspecialedition/mods/36869?tab=description)﻿[See pictures]

**WHY?** ﻿I make this to avoid problems in load order and make this mod compatible with any mod that alters Harkon's appearance/NPC records, such as [Real Bosses](https://www.nexusmods.com/skyrimspecialedition/mods/18183) and/or [Pandorable's NPCs - Dawnguard](https://www.nexusmods.com/skyrimspecialedition/mods/24135)﻿﻿, and I also wanted some changes, such as his sword and drop, etc...
  
  
**INSTALLATION**
  
 ﻿Install via mod manager and let this overwrite the original mod.
  
  
**COMPATIBILITY**
  
 ﻿Incompatible with any mod that aims to change Harkon's outfit or any mod that aims to put a new outfit on Harkon and probably any mod that changes the  [Lord of Coldharbour](https://www.nexusmods.com/skyrimspecialedition/mods/48302?tab=description)﻿﻿ mod.

**WHAT IF?**
  

  
* *I merge [Lord of Coldharbour](https://www.nexusmods.com/skyrimspecialedition/mods/48302?tab=description)﻿﻿ with other weapons and armor mods because of my big ass modlist and want to save slots in my load order, will your tweaks continue working?*

Hmm, probably, except for the SPID options, in this case, you have to change the .ini file line from Lords of Coldharbour.esp to MyMergedModPluginName.esp, and probably have to check if the formIDs of the items are the same, if not, change them. [I am probably going this way soon].
  

  
* *I want Valerica or any other NPC to have another weapon of the mod.*

Change the item ID in .ini file accordingly:
  
  
Montante: 0x02F9C
  
Mace: 0x16C83
  
Unenchanted Rapier: 0x11D3C
  
  
And add the name of the NPC alongside or replacing Valerica's name, e.g. Valerica, Serana. I recommend you to go to the SPID mod page to learn more about it.
