# The Dragonborn's Bestiary and Know Your Enemy 2 Patch
- Author: tjhm4 and Kulharin
- Game: skyrimspecialedition

![](https://staticdelivery.nexusmods.com/mods/1704/images/124145/124145-1722189105-359452031.png)
  
  
**Notes**
  
  
There are two versions. The **Lite** version just adds KYE2's resistances to the bestiary. The **Full** version also rearranges the bestiary categories to match KYE2's categorization system (e.g. the golem category is removed, etc.). Do not use the full version with other bestiary patches as they will expect the default categorization. If using the full version you must delete or hide the "creatures" folder from the bestiary mod (it is inside the interface folder).
