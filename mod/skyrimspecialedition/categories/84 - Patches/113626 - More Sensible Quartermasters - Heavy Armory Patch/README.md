# More Sensible Quartermasters - Heavy Armory Patch
- Author: TheShiningAbsol
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/113626
Simple patch for [More Sensible Quartermasters](https://www.nexusmods.com/skyrimspecialedition/mods/73887) to add appropriate weapons from [Heavy Armory](https://www.nexusmods.com/skyrimspecialedition/mods/6308)﻿ to the quartermasters' stock.
  
  
Imper﻿ial Quartermasters will sell new Imperial weapons and Stormcloak quartermasters will sell Iron/Steel, set appropriately to respect whether or not you've disabled Halberds, Spears, and Tridents in Heavy Armory.
  
  
Additionally swaps steel daggers laying around in Imperial camps with Imperial daggers through [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805).
