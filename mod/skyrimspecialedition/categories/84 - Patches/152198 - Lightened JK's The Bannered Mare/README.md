[font=Arial]For an in-depth description you can see what a mod like this usually does--> [link](https://www.nexusmods.com/skyrimspecialedition/mods/111475). In this case it would be impossible to do the same thing that [Lightened Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/111475) does with invisible objects so what the mod does is reduce the amount of clutter and less visible objects in a way that has no visual impact at all. You will say may how many fps you gain ![](https://www.nexusmods.com/assets/images/emoticons/dark/wink.gif). This mod works for JK's Dragonsreach and requires the use of [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805).
