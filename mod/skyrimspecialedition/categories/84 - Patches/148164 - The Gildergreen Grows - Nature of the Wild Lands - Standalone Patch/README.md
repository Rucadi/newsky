# The Gildergreen Grows - Nature of the Wild Lands - Standalone Patch
- Author: Ivy - fxxkthisworld
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148164


This mod uses one of the OLD DRUID assets from  [Nature of the Wild Lands](https://www.nexusmods.com/skyrimspecialedition/mods/63604)﻿ to replace the Gildergreen in Whiterun. I have also edited the texture so the tree is now pink.
  
  
FAQ
  
Do I need the nature of the wild lands?
  
No, this is a standalone replacer.
