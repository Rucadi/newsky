# JK's Riften Outskirts Patch Collection
- Author: Czasior
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/91642


A collection of compatibility patches for [JK's Riften Outskirts](https://www.nexusmods.com/skyrimspecialedition/mods/90864). All patches are ESP flagged as ESL.
  
  
**Patches descriptions:**
  
  
[A Cat's Life](https://www.nexusmods.com/skyrimspecialedition/mods/37250) - shifts a few markers and changes the position of Cat's Store. Last checked against version 2.0
  
  
[AI Overhaul SSE](https://www.nexusmods.com/skyrimspecialedition/mods/21654) - shifts one marker. Last checked against version 1.8.5
  
  
[Aspens Ablaze](https://www.nexusmods.com/skyrimspecialedition/mods/39998) - adjusts the position of a few aspens to prevent clipping. Last checked against version 2.35
  
  
[Atlas Map Markers Updated](https://www.nexusmods.com/skyrimspecialedition/mods/24104) - conflict resolution on one farm marker. Last checked against version 2.7
  
  
[Birds of Skyrim SSE](https://www.dropbox.com/s/0zmfolkcxrpv5d0/Birds%20of%20Skyrim%20SSE%20Edition%202.7.7z?dl=0) - adjusts the placement of a rooster. Last checked against version 2.7 - the last one available for SE.
  
  
[Carriage Drivers Are Alive](https://www.nexusmods.com/skyrimspecialedition/mods/51260) - conflict resolution, landscape adjustments. Last checked against version 7.8
  
  
[Carriage Stops of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/10434) - adjusts some elements to match the modified area layout. Last checked against version 1.6
  
  
[Convenient Carriages](https://www.nexusmods.com/skyrimspecialedition/mods/12693) - conflict resolution, landscape and a few records adjustments. Last checked against version 5.9.8
  
  
[Creation Club: Wild Horses](https://en.uesp.net/wiki/Skyrim:Wild_Horses) -  shifts one marker
  
  
[Dawnguard Sentries Plus](https://www.nexusmods.com/skyrimspecialedition/mods/13547) - shifts a few markers. Last checked against version 1.1c
  
[DK's Realistic Nord Ships](https://www.nexusmods.com/skyrimspecialedition/mods/43407) - adjusts the placement of a boat and some clutter, conflict resolution on navmesh. Last checked against version 2.0
  
  
[Enhanced Landscapes](https://www.nexusmods.com/skyrimspecialedition/mods/18162) - conflict resolution on landscape records, switches/disables a bunch of added elements. Last checked against version 2.0.1
  
  
[Enhanced Lights and FX - Exteriors](https://www.nexusmods.com/skyrimspecialedition/mods/2424) - adjusts lanterns/lights/FX addons positions, removes duplicated elements. Last checked against version 3.06.
  
  
[Gallows of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/26542) - moves the gallows to a different spot. Last checked against version 1.1
  
  
[Haystacks](https://www.nexusmods.com/skyrimspecialedition/mods/125021) - shifts a few clipping elements. Last checked against version 1.
  
  
[Illustrated Town Panels](https://www.nexusmods.com/skyrimspecialedition/mods/43603) - shifts town panel placement. Last checked against version 1.2.3
  
  
[Immersive Laundry](https://www.nexusmods.com/skyrimspecialedition/mods/2011) - shifts a few elements to avoid clipping. Last checked against version 1.0d
  
  
[JUSTICE - City Exteriors](https://www.nexusmods.com/skyrimspecialedition/mods/680) - adjusts the position of clipping elements. Last checked against version 1.15
  
  
[Khajiit Has Wares](https://www.nexusmods.com/skyrimspecialedition/mods/73500) - adjusts landscape and shifts the entire Ri'Saad tent (along with the clutter) placement. Additional option patched with [Immersive Citizens](https://www.nexusmods.com/skyrimspecialedition/mods/173) for placement consistency. Last checked against version 1.07.6 (KHW) and 0.4.0.1 (ICAIO)
  
  
[Lanterns of Skyrim II](https://www.nexusmods.com/skyrimspecialedition/mods/30817) - adjusts the placement and disables some lanterns, switches JK's added lanterns to LoS models. Last checked against version 4.1
  
  
[Legacy of the Dragonborn](https://www.nexusmods.com/skyrimspecialedition/mods/11802) - shifts shipment crates placement. Last checked against version 5.6.5
  
  
[Lux Via](https://www.nexusmods.com/skyrimspecialedition/mods/63588) - shifts and disables some lanterns, switches JK's added lanterns to Via models, and changes the lighting for consistency with Lux Via. Last checked against version 2.2
  
  
[Riften Docks Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/40021) - shifts a few elements to avoid clipping, conflict resolution on navmesh. Last checked against version 2.5.7
  
  
[Riften Extensions - Southwoods District](https://www.nexusmods.com/skyrimspecialedition/mods/47074) - landscape and navmesh adjustments. Last checked against version 1.1 and [Rob's Bug Fixes - Riften Extension Southwoods District](https://www.nexusmods.com/skyrimspecialedition/mods/65416) version 3.0
  
  
[Ruska - Riften Player Home](https://www.nexusmods.com/skyrimspecialedition/mods/16177) - landscape and navmesh adjustments, conflict resolution. Provided also with a different version for Ryn's Farms, which moves the entire house and all its addons closer to the city. Last checked against version 1.0
  
  
[Ryn's Farms](https://www.nexusmods.com/skyrimspecialedition/mods/72305) - a mix'n'match combo of both mods, shifts/restores/removes a bunch of elements, conflict resolution, landscape and navmesh adjustments. Last checked against version 2.0
  
  
[Scarecrows of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/39575) - adjusts the position of scarecrows to fit the modified farms' layout. Last checked against version 1.6 (it won't work with older versions due to FormIDs mismatch!)
  
  
[Skyrim Creations: East Empire Expansion](https://creations.bethesda.net/en/skyrim/details/5294/East_Empire_Expansion) - adjusts the position of a few NPCs near the main city gate, so they won't stuck inside the building. Last checked against version 1.01
  
  
[Skyrim Landscape and Water Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/26138) - conflict resolution, forwards ownership changes to crops. Last checked against version 8.5.2
  
  
[The Second Great War](https://www.nexusmods.com/skyrimspecialedition/mods/17204) - shifts a few markers and the banner above the main gate. Last checked against version 1.10
  
  
[Touring Carriages](https://www.nexusmods.com/skyrimspecialedition/mods/15948) - conflict resolution, shifts a few markers. Last checked against version 3.5.2
  
  
[Traverse the Ulvenwald](https://www.nexusmods.com/skyrimspecialedition/mods/57874)﻿ - adjusts the placement of a few trees. Last checked against version 3.3.2
  
  
**Additional notes:**
  
- some of the fixes in question are made on persistent references and require a new save to work properly,
  
- patches for Lanterns of Skyrim II / ELFX Exteriors / Lux Via - just like that main mods, aren't compatible with each other.
  
  
**Load order suggestion:**
  
1) Enhanced Landscapes / Landscape and Water Fixes patches
  
2) other patches
  
3) Ryn's Farms / Eli's Ruska patches
  
4) Riften Docks Overhaul / Khajiit Has Wares / A Cat's Life patches
  
5) consistency patches
  
6) Northern Roads patches (in a similar order to above)
