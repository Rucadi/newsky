# Fish Anywhere - Streamlined Fishing (Patch)
- Author: wankingSkeever and Parapets
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/85070


**Description:**
  
Maintains compatibility between Fish Anywhere With Water and Streamlined Fishing.
  
  
  
**Requirements:**
  
[Fish Anywhere With Water](https://www.nexusmods.com/skyrimspecialedition/mods/60915)
  
[Streamlined Fishing](https://www.nexusmods.com/skyrimspecialedition/mods/80683)
  
  
  
**Update History:**
  
  
**v1.0.0:**
  

  
* Initial Release

  
  
**Installation:**
  
Install manually, or just use your favorite mod manager.
  
There is no plugin. Just one script.
  
  
  
**Load Order:**
  
Script should overwrite the one included in Fish Anywhere With Water.
  
Streamlined Fishing's scripts are archived, so this will always overwrite automatically.
  
  
  
**Compatibility:**
  
Will be incompatible with anything that also edits the *ccbgssse001\_fishingactscript* script (Nothing else that I know of).
  
Compatible with [Diziet's Auto Weapons Patch for Streamlined Fishing](https://www.nexusmods.com/skyrimspecialedition/mods/90501).
  
Compatible with [Simple Fishing Overhaul - Animations and Improved Quest Dialogue](https://www.nexusmods.com/skyrimspecialedition/mods/103440).
