Supports both the default and Lotd version of the mod.
  
  
Install with a mod manager as usual.
  
  
If you are aware of other mods that use vanilla staff models leave a comment! Also if you find that there are staves with double meshes or other issues let me know.
  
  
I don't use most of these mods myself, so I'm mostly unaware of any updates they receive. If there is an update that requires the corresponding patch to be updated give me a nudge!
  
  
CGO users can use this Synthesis patcher to patch their staves with CGO sounds: <https://github.com/minis-patchers/SynCGOStavesPatcher>﻿
  
  
Separate patches won't be provided for [Sorcerer](https://www.nexusmods.com/skyrimspecialedition/mods/95196), use the [Synthesis patcher](https://www.nexusmods.com/skyrimspecialedition/mods/95212?tab=files) instead.
