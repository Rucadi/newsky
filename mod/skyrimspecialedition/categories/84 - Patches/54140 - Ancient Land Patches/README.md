- Author: L0rkhaan

**[Ancient Land](https://www.nexusmods.com/skyrimspecialedition/mods/22900) Patches**

  
  
All the plugins are flagged as ESL and cleaned with SSEEdit. All the patches are compatible with both the standard and the No Whiterun doomstones version of Ancient Land. Contributions are welcome to expand the patch collection.
  
  
**Included Patches**
  
  
> From version 1.2.1 (FOMod installer), all patches are compatible with the standard version and the No Whiterun doomstones version of Ancient Land. Patches that moved the arch of doomstones have been replaced by patches that remove it. Moving the arch was incompatible with the Western tower dragon’s combat zone.
