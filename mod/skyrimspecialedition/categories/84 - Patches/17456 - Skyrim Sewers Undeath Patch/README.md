# Skyrim Sewers Undeath Patch
- Author: Viltuska modified by ManyStrongWords
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/17456

 It moves a sewer exit so it does not conflict with the Undeath sewer exit.
