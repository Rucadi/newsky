# Requiem - Simple Hunting Overhaul
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111718
Balance and Compatibility patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Simple Hunting Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/95943).
  
  
*- Trolls now drop meat
  
- Rabbits now drop a pelt and increased their value
  
- Crafting recipes balanced for Requiem
  
- Reduced the value of Hunting Permits slightly
  
- Improved compatibility between SHO and Requiem*
