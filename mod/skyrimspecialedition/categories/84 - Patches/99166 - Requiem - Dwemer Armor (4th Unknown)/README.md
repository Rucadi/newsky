# Requiem - Dwemer Armor (4th Unknown)
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/99166
**Balance patch for [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888) and [Dwemer Armor SE](https://www.nexusmods.com/skyrimspecialedition/mods/81043).**
  
  
  
- Rebalanced the armor set (same as common dwemer armor but slightly more expensive)
  
- Requiem temper recipes
  
- Added new crafting recipes (slightly different materials compared to the common dwarven armor)
  
- Integrated the new armor set to rarely appear on NPCs via Outfits (very compatible)
  
- Removed the test chest in Riverwood
