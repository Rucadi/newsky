# Silentroll Distribute For Ultimate NPC Dodging -  Skypatcher SPID
- Author: Headless Orc
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/120956


Distribute Silentroll Perk for Ultimate NPC Dodging in SPID/Skypatcher to some NPCs.
  
  
Choose either the skypatcher version, which is limited to certain Unique NPCs like Darkbrotherhood/Thivesguild Member with sneak skills, or the SPID version, which distributes perk to several classes with high sneak skills.
  
  
Please make your own additions/changes to the ini as needed.
  
  
Ultimate NPC Dodging向けに、サイレントロールのPerkをSkypatcherもしくはSPIDでベースゲームのNPCに配布します。
  
上記MOD導入環境ではサイレントロールを習得しているNPCはステップドッジのかわりにドッジロールすることができるようになります。
  
  
Skypatcherバージョンは闇の一党/盗賊ギルドなど特定のNPCに限定しており、SPIDバージョンはclassに基づいてperkを配布する仕様です。
  
SPIDバージョンはNPCにベースゲームのclassが設定されているクエストMOD、フォロワーMODなどの追加キャラクターにもperkが配布されます。
