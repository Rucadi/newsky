# The Knight of the Void - Armor Replacers
- Author: OlivePeel
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/102253


Armour replacers for the follower mod [Knight of the Void](https://www.nexusmods.com/skyrimspecialedition/mods/100373).
  
  
If updating mid-save, remove all items from follower inventory, open console, click follower, type resetinventory . It should reset what armour she is wearing.
  
Hopefully shouldn't conflict with any future patches made to the original mod.
  
  
Feel free to request additional patches
