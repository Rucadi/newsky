# A Few Patches For Some Sound and Effects Mods - AOS-ISC-PSI-DPI
- Author: BloomAndGloom
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/57465


**Deadly Spell Impacts, Audio Overhaul, and Immersive Sound Compendium Patch**
  
  
**DSI AOS ISC Patch** - Combines the sound, and impact edits from all three mods, which are Deadly Spell Impacts, Audio Overhaul, and Immersive Sounds Compendium. Designed to be used with Audio Overhaul - Immersive Sounds Integration (AOS - ISC Compatibility Patch).
  
  
  
**Parallax Spell Impact Patches** 
  
  
**PSI AOS Patch** - Patches PSI with Audio Overhaul.
  
  
**PSI AOS DPI Patch** - Patches PSI with Audio Overhaul, and Deadly Spell Impacts.
  
  
Only one of these patches needed. Install the one for the combination of mods you're using.
  
  
  
**Arctic - Frost Effects Redux Patches** 
  
  
**FrostMesh ISC Patch** - Patches Artic with Immersive Sounds Compendium.
  
  
**FrostMesh ISC DPI Patch** - Patches Artic with Immersive Sounds Compendium, and Deadly Spell Imapcts.
  
  
Only one of these patches needed. Install the one for the combination of mods you're using.
  
  
  
**Other Patches**
  
  
**IllusionNoPush ISC Patch** - Patches No Force for Illusion Spells with Immersive Sounds Compendium
  
.
  
  
**MLPsoulGems ISC Patch** - Patches MultiLayer Parallax Soul Gems SSE with Immersive Sounds Compendium.
  
  
**LethalTraps AOS Patch** - Patches Lethal Traps with Audio Overhaul.
  
  
  
**Load Order**
  
  
Load order shouldn't matter providing they are below the mod the patches are for, with the exception of:
  
  
If you use an Artic Frost Mesh patch, and/or a Parallax Spell Impact Patch they should be below the Deadly Spell Impacts, Audio Overhaul, and Immersive Sound Compendium Patch. In the order shown below.
  
  
DSI AOS ISC Patch
  
FrostMesh ISC DPI Patch
  
PSI AOS DPI Patch
  
  
You only need one of the Parallax Spell Impact Patches, and one of the Arctic - Frost Effects Redux Patches. Install the one for the combo you need.
