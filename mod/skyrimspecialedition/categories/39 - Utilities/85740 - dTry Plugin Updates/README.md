# dTry Plugin Updates
- Author: doodlum
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/85740
**[[size=4]Buy dTry a coffee](https://ko-fi.com/dtry79711)
  
  
OVERVIEW[/size]****dTry's SKSE plugins ported to 1.6.640/NG in a FOMOD installer. Includes Key Utils, Combat Music Fix, Stop On Slash, Shield Of Stamina, No Crime Teleport RE, Elden Parry, Elden Counter.**[﻿﻿](https://ko-fi.com/dtry79711)

  
**[dTry](https://www.nexusmods.com/skyrimspecialedition/users/77140323)** is too busy at the moment to update his plugins. Here, are updates of all of his plugins which support all game versions. VR may work on some but is untested.
  
  
  
**REQUIREMENTS**
  
[Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444)
  
**Install the original mod if you select it.**
  
Install any version (AE or SE) unless noted otherwise.
  
  
[dTry's Key Utils](https://www.nexusmods.com/skyrimspecialedition/mods/69944) 
  

  
* Requirement for some animation mods.

  
[Combat Music Fix SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/67015) 
  

  
* An awesome fix you should definitely be using.
**VR** compatible.
  

  
[No Crime Teleport RE](https://www.nexusmods.com/skyrimspecialedition/mods/71691)
  
[Elden Parry](https://www.nexusmods.com/skyrimspecialedition/mods/70240) 
  

  
* **Compatible** with [Valhalla Combat](https://www.nexusmods.com/skyrimspecialedition/mods/64741).
Required more work than the other plugins to update.
  

[Elden Counter](https://www.nexusmods.com/skyrimspecialedition/mods/65579)
  
[Stop On Slash](https://www.nexusmods.com/skyrimspecialedition/mods/66155)
  

  
* Don't use with [Precision](https://www.nexusmods.com/skyrimspecialedition/mods/72347)﻿.

  
[Shield Of Stamina](https://www.nexusmods.com/skyrimspecialedition/mods/62137) 
  

  
* Install the **AE** version.
**Incompatible** with [Valhalla Combat](https://www.nexusmods.com/skyrimspecialedition/mods/64741).
