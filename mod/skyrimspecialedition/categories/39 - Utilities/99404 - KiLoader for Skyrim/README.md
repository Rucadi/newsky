# KiLoader for Skyrim
- Author: Kitsuune
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/99404


**[left][color=#ff8844]Features[/color][/left]**

* Powerful communication facilities between plugins (including data sharing)
Interoperability with other plugin systems (SKSE and ENB)
  
* Centralized error handling and compatibility checking
Easy access to additional runtime information
  
* Multi DLL chain loading
Centrally managed default and dependency directories
  

  
  
**[left][color=#ff8844]Installation[/color][/left]**Download and install with a modmanager of your choice
  
  
  
**[left][color=#ff8844]Uninstallation[/color][/left]**Remove all files with your modmanager. If you use ENB, you might also have to remove the ENB satellite DLL that can be found inside of your enbseries folder.
  
Make sure to look into your overwrite folder if you're using MO2 with root builder.
  
  
  
**[left][color=#ff8844]Details[/color][/left]**The loader API will be open sourced shortly. The modpage will be updated accordingly.
