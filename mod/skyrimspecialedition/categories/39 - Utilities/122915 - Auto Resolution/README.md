# Auto Resolution
- Author: Quantumyilmaz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/122915


Makes SSE Display Tweaks set the game resolution to the Windows display resolution by automatically changing the resolution setting in its INI.
  
  
Optionally you can set the game resolution to a fraction of your display size by changing the ratio setting in the INI of this mod.
