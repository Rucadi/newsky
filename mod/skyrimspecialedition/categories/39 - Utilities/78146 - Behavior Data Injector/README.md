# Behavior Data Injector
- Author: Maxsu and Dropkicker
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78146


[font=Tahoma]**[![](https://raw.githubusercontent.com/doodlum/nexusmods-widgets/main/Ko-fi_40px_60fps.png)](https://ko-fi.com/maxsu2020?__cf_chl_rt_tk=tQRmDuRLsAotwyWDWJKUhZKoaQ9dXkCJWq3wIkSS07s-1667570103-0-gaNycGzNCX0)﻿
  
  
MOD Descriptions:**BDI - Behavior Data Injector is a SKSE plugin that allows mod authors to inject their custom [graph variables](https://www.creationkit.com/index.php?title=List_of_Animation_Variables) (a.k.a animation variables) and [animation events](https://www.creationkit.com/index.php?title=Animation_Events) into behavior files during game runtime, without the necessary to modify behavior file or create an nemesis patch.
  
  
[/font][font=Tahoma]**For Graph Variable:**
  
[/font][font=Tahoma]You can add three type of graph variable: Int, bool, float into behavior projects via BDI. The added variables are all the same as the vanilla existed one, they can be set & get via vanilla [papyrus functions](https://www.creationkit.com/index.php?title=GetAnimationVariableInt_-_ObjectReference) or [condition functions](https://www.creationkit.com/index.php?title=GetGraphVariableInt).
  
  
Mod Example of adding graph variable: [Stagger Direction Fix](https://www.nexusmods.com/skyrimspecialedition/mods/43339)
  
  
[/font]**[font=Tahoma]For Animation Event:
  
[/font]**[font=Tahoma]You can add new animation event into behavior projects via BDI, the added event then could be trigger by animation annotations.
  
  
Mod Example of adding animation event: [Payload Interpreter](https://www.nexusmods.com/skyrimspecialedition/mods/65089)
  
  
**How To Create Config Files:**
  
Check this [doc](https://github.com/max-su-2019/BehaviorDataInjector/blob/master/doc/How%20to%20create%20BDI%20config%20files.md) to learn more detail about how to create config files for BDI.
  
You can also join my [discord](https://discord.gg/qkfGYcqpfC) for further communication.
  
  
  
  
**Installation:****Requirement:** 
  
(1) SKSE for the correspond game version.
  
(2) [Address Library for SKSE Plugins﻿](https://www.nexusmods.com/skyrimspecialedition/mods/32444?tab=description)
  
Download and Install the mod files with mod manager.
