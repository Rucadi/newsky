# Experience for CCFE
- Author: Meridiano
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/90837


This addon will add 2 console commands to trigger [Experience](https://www.nexusmods.com/skyrimspecialedition/mods/17751) mod Papyrus functions using [CCFE](https://www.nexusmods.com/skyrimspecialedition/mods/78243).
  
  
**#1**
  
 ﻿ExpAE <int> [<bool>] : Triggers AddExperience function.
  
 Adds experience points in integer, shows meter if bool is specified and true.
  
 ﻿Example usage: "ExpAE 3000 1"
  
 ﻿^^^ Will add you 3000 XP and show level meter.
  
  
**#2**
  
 ﻿ExpSN <string>|<string>|<string> : Triggers ShowNotification function.
  
 ﻿Shows a notification with text, status and sound arguments.
  
 ﻿Example usage: "ExpSN Are u mad bro?|Meridiano asks|NPCGoatBleat"
  
 ﻿^^^ Will show you "Meridiano asks: Are u mad bro?" notification with a goat bleat sound (Skyrim.esm~016CAB).
