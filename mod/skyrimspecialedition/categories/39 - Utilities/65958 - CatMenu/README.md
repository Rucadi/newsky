# CatMenu
- Author: FiveLimbedCat
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65958


**2.0.0 is out. Installation is different so beware!**
  
  
CatMenu is an ImGui menu for dispalying and navigating through all sorts of menu from my other mods. **Default toggle button F6**.
  
  
**Limitation**
  

  
* No support for localization currently.
No gamepad navigation.
  

  
**Used By**

  
* [Post-hit Killmoves](https://www.nexusmods.com/skyrimspecialedition/mods/65117) (Old Version)﻿﻿
[Kaputt](https://www.nexusmods.com/skyrimspecialedition/mods/78063) (2.0+ Version)﻿
  

  
**Installation**
  

  
* Extract imgui.dll and imgui.pdb to the game folder (the same folder where Skyrim.exe is).
Install CatMenu.7z with your mod manager.
  

  
**Incompatibilities**
  

  
* [KreatE](https://www.nexusmods.com/skyrimspecialedition/mods/83757)﻿
