**Notes**
  
** Dareni's Spells - Merchants doesn't require you to install all of the spell mods.*
  
  
**Compatibility**
  
  
Dareni's Spells - Merchants is flagged as ESL and ESM. It needs to load before any of the supported mods.
  
None of the supported spell mods require Dareni's Spells - Merchants as a master.
