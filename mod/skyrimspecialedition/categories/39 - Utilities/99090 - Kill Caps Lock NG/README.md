# Kill Caps Lock NG
- Author: JollyGopher
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/99090
This is useful for anyone who gets annoyed when they mistype a console command thanks to capslock.
  
  
Only affects one line of assembly code and is compatible with every other mod.
  
  
  
Supports AE and SE thanks to [CommonLibSSE NG](https://github.com/CharmedBaryon/CommonLibSSE-NG). Incompatible with Skyrim VR.
  
  
Safe to install or uninstall at any time.
  
Tested on Skyrim version 1.5.97 (SE), version 1.6.640 (AE) and version 1.6.1170 (AE).
  
  
Source: <https://github.com/jolly-gopher/KillCapsLockNG>
