# Custom Window
- Author: doodlum
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/83945
**OVERVIEW****SKSE plugin which replaces the game window title and icon.**
  
**[b][b][size=3][b][b][size=3][![](https://github.com/doodlum/nexusmods-widgets/blob/main/Ko-fi_40px_60fps.png?raw=true)](https://ko-fi.com/doodlez)      [![](https://github.com/doodlum/nexusmods-widgets/blob/main/Patreon_40px.png?raw=true)](https://www.patreon.com/Doodlezoid)**[/b][/size][/b][/size][/b][/b]

  
By default the title and logo are **Skyrim Script Extender**. Pretty cool mod you should check out.
  
  
Partially inspired by and compatible with [Desktop Splash Screen](https://www.nexusmods.com/skyrimspecialedition/mods/83470).
  
  
The title can be configured in the INI, for example:
  
  
**![](https://i.imgur.com/hlA1u97.png)**
  
  
**REQUIREMENTS**
  
[Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444)
  
  
**COMPATIBILITY****[b]Compatible with all game versions including VR.**[/b]
