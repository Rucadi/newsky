# ClickLight - Highlight Objects Clicked in Console (SSE)
- Author: Fox_Tek
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/121555


[font=Lucida Sans Unicode]**![](https://i.postimg.cc/ydhhpDBh/Description.png)**[/font][font=Lucida Sans Unicode]Highlights objects clicked in console, say goodbye to the trial and error of making sure you've selected the right thing!
  
  
Simple command that will run via a "preset" for Console Commands Extender, assuring all objects clicked in the console are highlighted in orange.
  
  
  
**Note:** The method used in this version isn't as efficient as other Clicklight versions, for now it's just a preset for CCE, but I plan to provide a better method in a future update! (WIP)
