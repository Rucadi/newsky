**Description**
  

  
* Use MCM to assign hotkeys
Use hotkeys to drink health/stamina/magicka potions
  
* Will attempt to use the optimal potion to avoid or prevent over healing

**FAQ**
  
> **Q: Is this compatible with potion mod XYZ?**A: Mods that leave relevant potion mechanics the same and simply rebalances them are fine. The main thing being that the keywords must be the same and the relevant potions must take effect instantly. If a mod makes potions restore over time, then the optimal potion calculations will be off. If a mod removes the relevant keywords from potions, then this mod won't work.
