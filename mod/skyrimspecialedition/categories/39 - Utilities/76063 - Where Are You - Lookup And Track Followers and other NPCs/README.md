# Where Are You - Lookup And Track Followers and other NPCs
- Author: k0mp1ex
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/76063


[font=Trebuchet MS]

**﻿![](https://raw.githubusercontent.com/k0mp1ex/mods-resources/main/mods/where-are-you/mcm.png)﻿**

  

So, you've been wandering around Skyrim, found a few NPCs on your way but you're not sure where they went later?
  
Maybe you installed the best followers out there but don't know where they spawned?
  
Don't worry, we'll find them for you!

**Features**
  
  

*(video showing features from an older version 1.4.0, some changes have been made since then)*

[/font]

  
* [font=Trebuchet MS]Search for all **unique** NPCs by their display names[/font]
[font=Trebuchet MS]Support to search using [regular expressions](https://www.regular-expressions.info/)﻿﻿[/font]
  
* [font=Trebuchet MS]Customizable UI (result list type and format, icons, colors, etc.)[/font]
[font=Trebuchet MS]MCM[/font]
  
* [font=Trebuchet MS]Customizable hotkeys (with modifiers) for search, tracking and run commands for NPCs[/font]
[font=Trebuchet MS]Option to only allow followers to do favors[/font]
  
* [font=Trebuchet MS]Tracking markers up to 100 NPCs[/font]
[font=Trebuchet MS]Bidirectional teleport (teleport the NPC to you or teleport you to where the NPC is)[/font]
  
* [font=Trebuchet MS]Check NPC stats[/font]
[font=Trebuchet MS]Open NPC inventory[/font]
  
* [font=Trebuchet MS]Enable/Disable NPC[/font]
[font=Trebuchet MS]Result list up to 128 NPCs[/font]
  
* [font=Trebuchet MS]Option to auto select the NPC found on the console (via command [PickRefById, aka prid](https://falloutcheats.com/command/pickbyrefid)﻿) for quick access[/font]
**[font=Trebuchet MS]ESPFE[/font]**
  
* [font=Trebuchet MS]SKSE plugin powered by **[CommonLib-NG](https://github.com/CharmedBaryon/CommonLibSSE-NG)﻿** to support multiple Skyrim versions[/font]

**FAQ**
  
  
[/font]

  
* [font=Trebuchet MS]How to show the search bar?[/font]
[font=Trebuchet MS]The default key is F3. You can change this via MCM.[/font]
  

[font=Trebuchet MS]
  
[/font]

  
* [font=Trebuchet MS]Can I install it mid playthrough?[/font]
[font=Trebuchet MS]Yes. Always create a backup save, just in case.[/font]
  

[font=Trebuchet MS]
  
[/font]

  
* [font=Trebuchet MS]Can I uninstall it mid playthrough?[/font]
[font=Trebuchet MS]It's not recommended, but you can. If you wish to proceed what I would do:[/font]
  
[list]* [font=Trebuchet MS]1) Disable the **kxWhereAreYou.esp**[/font]
[font=Trebuchet MS]2) Make a new save[/font]
  
* [font=Trebuchet MS]3) Clean the new save on [Resaver](https://www.nexusmods.com/skyrimspecialedition/mods/5031)﻿[/font]
[font=Trebuchet MS]4) Load the cleaned save[/font]
  

[/list][font=Trebuchet MS]
  
[/font]

  
* [font=Trebuchet MS]I think I found a bug, what should I do?[/font]
[font=Trebuchet MS]You can report it here on Nexus or in the GitHub mod's page in the section below. Help me with as much info as you can, enabling "Debug Mode" on MCM > System and providing info as logs (from CrashLogger/.NET Script Framework, kxWhereAreYou.log, Papyrus.0.log), your game version, the version of the required mods (MCM Helper and UIExtensions) and what you did to trigger the problem. Additionally you can test the Debug version of the mod available on the [GitHub releases page](https://github.com/k0mp1ex/where-are-you/releases)﻿﻿ that outputs more helpful info.[/font]
  

[font=Trebuchet MS]
  
[/font]

  
* [font=Trebuchet MS]I can't find my NPC, what should I do?[/font]
[font=Trebuchet MS]This mod only tracks the **unique** NPCs. Make sure your NPC is **unique** and have spawned yet (has a reference ID). Some NPCs won't spawn until later when their cells are loaded. You might see this with Jenassa, Lydia, Jordis, etc. As a rule of thumb: if you can use the **prid** console command with a reference ID for an **unique** NPC you will be able to also find it using this mod.[/font]
  

[font=Trebuchet MS]
  
[/font]

  
* [font=Trebuchet MS]I have a very specific case where my NPC is not tracked by this mod at all. Can I change any settings or do some troubleshoot on my own?[/font]
[font=Trebuchet MS]You can enable "Debug Mode" on the mod MCM > System tab and then look at the logs. Remember that for Papyrus logging you need to set [font=Courier New]bEnableLogging[/font] and [font=Courier New]bEnableTrace[/font] to **1** on your **Skyrim.ini** file. The Papyrus logs will appear on your **Papyrus.0.log** file. The SKSE plugin logs will be stored in the file kxWhereAreYou.log with all your other SKSE plugin logs (folder **My Games\Skyrim Special Edition\SKSE**).[/font]
  

  

  
* [font=Trebuchet MS]It's not lore-friendly, you can break your quest disabling NPCs or taking something from their inventory. You know this, right?[/font]
[font=Trebuchet MS]I do. This mod is not intended to be lore friendly, just a quality of life mod. It's up to you to decide what commands you'll use to not break your quests or immersion.[/font]
  

[font=Trebuchet MS]
  

  
* [font=Trebuchet MS]What happens if I disable the mod via "Mod enabled" toggle on MCM > System tab?[/font]
[font=Trebuchet MS]All the hotkeys associated with this mod will be unregistered.[/font]
  

  

  
* I can't use MCM at all, is there any way to bind the search key to something other than F3?
﻿Sure, follow the steps:
  

  

  
1. Check for the File:
     
     
   * Look for *Data\MCM\Settings\kxWhereAreYou.ini*
       
     This file saves the changes you make via MCM in-game.
   If you don't have it, overwrite the original *MCM\Config\kxWhereAreYou\settings.ini* that comes with the mod.
Configure the Search Function:
  

  
* Open the ﻿﻿.ini file and navigate to the *[Hotkeys]* section.
Find the *uSearchKey* property.
  
* Update it using any DirectX Scan Code. Refer to these resources for the codes:
    
  [list]
[Input Script Documentation](https://ck.uesp.net/wiki/Input_Script)
  
* [DirectX Scancodes Guide](https://wiki.nexusmods.com/index.php/DirectX_Scancodes_And_How_To_Use_Them)

[/list][/font][font=Trebuchet MS]
