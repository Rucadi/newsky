**Description**
  

  
* Use broom to clean up physics mess
Just click on broom in inventory
  
* No: this mod doesn't give you a free broom. Go get a broom yourself.
Range of 10,000 units
  
* Plays animations
Won't mess up weapon racks
  
* Optional [Description Framework](https://www.nexusmods.com/skyrimspecialedition/mods/105799)﻿ support for item descriptions
ESL
  
* Doesn't clean trash you littered out yourself (via dropping from inventory)
Update 0.2: bookshelf support
  
* Update 0.2: multithreaded speed

  
  
  
  
  
**Upgrading**
  

  
1. Uninstall mod
Make a new save
  
2. Clean that save with [Fallrim Tools Resaver](https://www.nexusmods.com/skyrimspecialedition/mods/5031)
Install new version﻿
  

**Installation/Uninstallation**
  

  
* Install whenever
Uninstall whenever
  
* Clean that save with [Fallrim Tools Resaver](https://www.nexusmods.com/skyrimspecialedition/mods/5031)
