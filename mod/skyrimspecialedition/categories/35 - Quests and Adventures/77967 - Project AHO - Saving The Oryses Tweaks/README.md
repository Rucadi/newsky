# Project AHO - Saving The Oryses Tweaks
- Author: asdasfa
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/77967


**Requirements:**
  
**[Project AHO](https://www.nexusmods.com/skyrimspecialedition/mods/15996)**
  
  
**Description:**
  
Upon concluding the quest **"A Burden of the Past"** with the choice of letting the Oryses escape,
  
they can then be located in Windhelm, spending time around the city and inside the New Gnisis Cornerclub.
  
Step outside Sadrith Kegran's entrance to trigger these changes.
  
  
**Compatibility:**
  
This mod changes the AI Package of these two NPCs, so any mod that touches the same NPCs (e.g. appearance mods) might need a patch.
  
This mod adds two idle markers in Windhelm's docks area, mods that significantly alter that certain spot might need a patch.
