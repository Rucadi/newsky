# Blood on the Ice Redux - Re-Voiced
- Author: Nostalgic - Chef
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/87340


[font=Tahoma]**DESCRIPTION**[/font]
  
Thanks to [11labs.io](https://beta.elevenlabs.io/)﻿ and Lazy Voice Finder, I've replaced the spliced dialogue by Repzik in: [Blood on the Ice - Voiced ESP-FE](https://www.nexusmods.com/skyrimspecialedition/mods/54417) with ai cloned dialogue that should sound more natural and blend in better with vanilla voice lines.﻿
  
  
[font=Tahoma]**INSTALLATION**[/font]
  

  
1. Download and install **[Blood on the Ice Redux - Voiced ESP-FE](https://www.nexusmods.com/skyrimspecialedition/mods/54417)﻿.**
Install this mod below Repzik's mod.
  
2. You're done! Enjoy!
