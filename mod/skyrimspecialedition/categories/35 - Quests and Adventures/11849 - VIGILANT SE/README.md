- Author: Vicn
- Game: skyrimspecialedition

[font=Georgia]**Installation :**[/font][line]
  
1.Check that you have all required mods and DLCs installed and enabled
  
2.Install Vigilant.esm and Vigilant.bsa using your Mod Manager of choice
  
3.Wait or sleep several hours ingame. Save, close your game, and then reload.
  
  
  
[font=Georgia]**Updating :**[/font][line]
  
Number means version of  this mod, not skyrim.exe.
  
  
**If Version == 1.7.x**Updatable to 1.8.0 safely. 
  
  
**If Version <= 1.6.x**Outdated version.
  
You should start new game or use a save that has never had VIGILANT installed before.
  
  
**If ESP**
  
You cannot update the ESP version to the ESM version. You must start a new game or use a save that has never had VIGILANT installed before.

[font=Georgia]**The Quests:**[/font][line]

**Episode 1: The Summoner** (Tutorial)[font=Georgia]
  
[/font]The Player becomes a Vigilant of Stendarr, and chases down a Summoner of Molag Bal.
  
To start the quest, speak with Altano, who is located in the Windpeak Inn in Dawnstar.
  
  
**Episode 2: Bloody Matron** (Small Dungeon)[font=Georgia]
  
[/font]The Player investigates the disappearance of multiple prisoners from the Windhelm dungeons.
  
It automatically start after Act1. When you update from older version, you need to activate "Water Snakes"of debug room.
  
  
![](http://i.imgur.com/R6RYzib.jpg)![](http://i.imgur.com/46NJtSL.jpg)
  
**Episode 3: Child of Oblivion** (Horror, Hide and Seek)
  
The Player must rescue the owners of a noble mansion.
  
It automatically start after Act2. When you update from older version, you need to activate "Water Spirits" of debug room.
  
  
**Episode 4: Oblivion** (New World, Non-Journal)
  
The Player is trapped in Oblivion, and must wander the wasteland in order to find a way out.
  
The quest begins immediately after Episode 3.[font=Georgia]
  
[/font]
  
**Orlando the Knowledgeable** (NG+)
  
When player accomplish his quest, player can go to Ep4 directly. This story branch is for users who've ever already played story.
  
  
**Anatomancer** (1.6.0)
  
After Exile from oblivion,  you can hear rumor about Anatomancer from Librarian.
  
If you know where it is,  you can meet with Anatomancer in any time.

[font=Georgia]**Skipping Scene :**[/font][line]
  
[/font]![](https://i.imgur.com/nzfYQdF.png)
  
Default Key : Pause/Break (Look upper right of keyboard.  If there is not, you can change the key in MCM.)
  
  
When press and hold this key, you can skip the scene event (Conversation between NPCs, Following Me event).
  
Use this function when the scene is stuck or you don't want see the scene again.

[font=Georgia]**MCM :**[/font][line]
  
on the MCM, you can change the Health (Health Mult) and the Attack Power (Attack Power Mult) of bosses.
  
  
**Health**
  
Each time the value is increased by 1, Health of Bosses is increased by 10%.
  
  
**Attack Power**
  
Each time the value is increased by 1, attack power of Bosses is increased by 20%.
  
  
**Animation Stability**
  
If NPCs are shaking while attempting to use additional animations, activate this option. 
  
  
  
[font=Georgia]**NG+ :**[/font]
  
[line]
  
From version 1.8.0, Vigilant now saves cleared data to an external JSON file accross save files. These flags relate to dialogue changes and event unlocks.
  
  
**File location:**
  
skse\plugins\VIGILANT\ElderScroll\_Global.json
  
  
**Mod Organizer:**
  
overwrite\skse\plugins\VIGILANT\ElderScroll\_Global.json
  
  
Clear data is shared across all characters.
