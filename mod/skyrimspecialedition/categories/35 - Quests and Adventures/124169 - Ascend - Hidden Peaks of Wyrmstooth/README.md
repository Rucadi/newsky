# Ascend - Hidden Peaks of Wyrmstooth
- Author: Rotol
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/124169


Overview

﻿This mod introduces realistic climbing mechanics to the mountainous terrain of Wyrmstooth, enhancing exploration and adding new challenges. There is a hiking guide in the East Empire Company Tradehouse in Stone Hollow
  
  
[size=2]- Climb Valus Summit and get a view on Fort Valus and Stonehallow (new in Version 1.0.2.0)  SpoilerBeginning: Close to the entrance of Fort Valus. Console port for testing
  
purposes: cow wyrmstoothworld -3 -3.
  
- Climb Twin Peak and explore an ancient hiking trail, but others have been here before you, and not just recently (new in Version 1.0.1.0)  SpoilerBeginning: Above Cragwater Camp the start is on top of the entrance to the camp. Console port for testing purposes: cow wyrmstoothworld 5 -8.
  
[/size]- ﻿Reach Grimsfyr Peak and find out about the lost Imperial mission SpoilerBeginning: Near Wyrmstooth Barrow the start is on top of the ruin (you can either go through the barrow or climb up left from entrance). Console port for testing purposes: cow wyrmstoothworld 11 -2.
  
- Climb Frostwolf Crag and solve the mystery of the missing wanderers SpoilerBeginning: Frostwolf Crag Pass, near the tower where the tree fell on. On the side where tree stood you will find the start. Console port for testing purposes: cow wyrmstoothworld 0 2.
  
- Flagged as ESL
