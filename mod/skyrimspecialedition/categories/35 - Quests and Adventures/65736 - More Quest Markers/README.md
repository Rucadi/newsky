# More Quest Markers
- Author: mfvicli
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65736


Simple mod that adds in markers for:
  
(Chasing Echoes) The three seals found in the Volkihar Courtyard, Valerica's Journal, and the three ingredients needed to open the soul cairn.
  
(Lost to the Ages) All aetherium shards.
  
(Deathbrand) All chest locations.
  
(Reluctant Steward) Markers for Varona and Drovas.
  
(Tolfdir the Absent-Minded) Adds a marker for Tolfdir's Alembic.
  
(Destruction Ritual Spell) Adds markers for all pedestals.
