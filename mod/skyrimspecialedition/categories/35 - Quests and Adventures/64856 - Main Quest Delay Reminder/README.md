# Main Quest Delay Reminder
- Author: WarbirdShaman
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64856


'Main Quest Delay Reminder' sends a courier with a letter from the Jarl of Whiterun (letter adapts with civil war Jarl/Steward changes) reminding you to continue the main quest by speaking to Farengar once you reach level 14. To be used with 'Delay Bleak Falls Barrow' or similar mods.
  
  
I recommend using this [Patch](https://www.nexusmods.com/skyrimspecialedition/mods/149907) by - NaevysKrisvalur.
