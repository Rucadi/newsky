[font=Georgia]**Installation :**[/font][line]
  
1.Check that you have all required mods and DLCs installed and enabled
  
2.Install Glenmoril.esm and Glenmoril.bsa using your Mod Manager of choice
  
3.Wait or sleep several hours ingame. Save, close your game, and then reload.
  
  
  
**[font=Georgia]Updating[/font] :**[line]**If Version >= 0.8.0 ESM**
  
Updatable to 0.96.x
  
  
**If Version <  0.8.0 ESM**
  
All IDs were renumbered and lost compatiblity with older version perfectly.
  
So, you must
  
・start new game or use a save that has never had GLENMORIL installed before.
  
・remove/update sub plugIn/add-on which use older Glenmoril.esm as master files.
  
  
**If ESP**
  
You cannot update the ESP version to the ESM version. You must start a new game or use a save that has never had GLENMORIL installed before.

[font=Georgia]**Skipping Scene :**[/font][line]
  
![](https://i.imgur.com/nzfYQdF.png)
  
Default Key : Pause/Break (Look upper right of keyboard.  If there is not, you can change the key in MCM.)
  
  
When press and hold this key, you can skip the scene event (Conversation between NPCs, Following Me event).
  
Use this function when the scene is stuck or you don't want see the scene again.
  
  
  
[font=Georgia]**MCM :**[/font][line]on the MCM, you can change the Health (Health Mult) and the Attack Power (Attack Power Mult) of bosses.
  
  
**Health**
  
Each time the value is increased by 1, Health of Bosses is increased by 10%.
  
  
**Attack Power**
  
Each time the value is increased by 1, attack power of Bosses is increased by 20%.
  
  
**Animation Stability**
  
If NPCs are shaking while attempting to use additional animations, activate this option.
