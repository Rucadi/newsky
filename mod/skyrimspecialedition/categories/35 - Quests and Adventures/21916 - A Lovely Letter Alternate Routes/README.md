# A Lovely Letter Alternate Routes
- Author: Parapets
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/21916


Description
  
  
Changes for the quest "A Lovely Letter":
  

  
* The dialogue to start the quest won't appear if Camilla is your lover.
You can say no to Sven and Faendal when they ask you to deliver their letters to Camilla Valerius.
  
* If you speak to both of them and pickpocket back the first note, you can complete the quest by exposing both of them.

  
Compatibility
  
  

  
* Load after The Choice Is Yours
Load after Relationship Dialogue Overhaul
