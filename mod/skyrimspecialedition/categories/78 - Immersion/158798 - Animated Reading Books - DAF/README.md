# Animated Reading Books - DAF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/158798


A simple mod which adds reading book/note animation everytime you read any book/note from inventory using the recent Dynamic Animation Framework. Enjoy!
  
  
**Installation:**1. Install the requirement DAF and Keyword Item Distributor
  
2. Install this mod
