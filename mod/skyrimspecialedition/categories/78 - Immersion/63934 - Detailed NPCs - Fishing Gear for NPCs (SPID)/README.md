# Detailed NPCs - Fishing Gear for NPCs (SPID)
- Author: BecomingNplus1
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63934


**Description**
  

  
* Mod uses [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ to distribute and equip fishing gear to NPCs﻿﻿
Fishing Gear uses body slot 47
  
* File standalone
File ESL-flagged
  
* No records edited
Player character not impacted﻿
  
* \_DISTR.ini designed for maximum configurability

  
**Requirements**

  
* [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿

  
**Overview**
  
My mod utilizes a small script that runs once (at game initialization) to distribute and equip fishing gear to NPCs. Mod is currently configured to distribute and equip fishing gear to: 
  
  

  
* [Addvar](https://en.uesp.net/wiki/Skyrim:Addvar)
[Riften Fish Faction](https://en.uesp.net/wiki/Skyrim:Factions_R#Riften_Fishery_Faction)
  
* Any NPC who is a [Fish Vendor](https://en.uesp.net/wiki/Skyrim:Factions_F#Fish_vendor) (must be in this faction)﻿﻿﻿﻿
[Fisherman](https://en.uesp.net/wiki/Skyrim:Fisherman)﻿
  
* [FishermenFaction](https://en.uesp.net/wiki/Skyrim:Factions_F#Fisherman_Faction)﻿

  
**Mod can be re-configured to distribute the fishing gear to any NPC and / or any group of NPCs that you wish (e.g., ActorTypeNPC)!** Edit the \_DISTR.ini that comes with this mod, and add (or delete) any Form ID that you wish.
  
  
**Compatibility**
  
Highly compatible with most setups, and with most other mods. 
  

  
* [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ does not edit levelled lists, and does not modify or generate new records.
Body slots, however, are not compatible. Choose wisely, or edit the body slot. I will not alter body slots.
  
* Clipping will likely occur depending on setup. I will not resolve clipping issues.

  
**Installation and Uninstallation**
  
Use the mod manager of your choice to install. Or manually. To uninstall, remove with mod manager. My understanding is that SPID will not impact save games in any way. Safe to uninstall at anytime. 
  
  
**Configuration**
  
Huge number of configuration options available in the \_DISTRI.ini. Some knowledge of [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ is useful to achieve desired results.
