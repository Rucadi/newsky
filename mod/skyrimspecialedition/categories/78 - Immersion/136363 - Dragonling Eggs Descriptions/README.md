# Dragonling Eggs Descriptions
- Author: Ragnarok Raven
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136363


Gives short descriptions to each of the eggs from the Collectible Dragonling Eggs mod via Description Framework. Most of these were written by NebulonRanger, with edits by me to conform to the lore of these not being actual eggs.
