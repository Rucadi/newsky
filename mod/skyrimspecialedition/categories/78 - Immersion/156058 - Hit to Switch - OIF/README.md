# Hit to Switch - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/156058


Ever bored to always press E everytime to activate a lever, pullbar, or a button? If so, then this mod is for you. With this mod now you can hit a lever with a sword to lower a bridge, shoot an arrow to push a button from a far, and use telekinesis to pull a lever to stop a trap or to open a hidden door. Enjoy!
  
  

Hit to push a lever

![](https://images2.imgbox.com/6f/5d/3id6IN9f_o.gif)
  
  
Shoot an arrow/bolt to push a button from far
  
![](https://images2.imgbox.com/36/45/AvYBgTLV_o.gif)
  
  
Use telekinesis spell to push a lever, pull a pullchain, or remove a door bar
  
![](https://images2.imgbox.com/56/d0/jG0qwk7r_o.gif)

  
  
**Installation:**1. Install the requirement OIF minimum v.1.6.8, formlist manipulator and Use Telekinesis on Traps
  
2. Install this mod
