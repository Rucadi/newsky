【Overview】
  
This is a simple mod that adds **a more "ritual" like scene to Serana's vampire cure quest.** 
  
Serana actually receives the ritual from Falion, like experienced by the player.
  
  
Serana goes to Morthal for the cure and stays with Falion for about two days.
  
Then Serana heads to the summoning circle in the swamp and performs the ritual. Player will be in a position to observe the ritual.
  
The ritual is completed after a day, even if the player is not present.

【Q&A】
  
  
--------------------------------------------------------------------------
  
Q : How can I start the vampire cure quest for Serana?
  
A :  The conditions under which the quest begins remain vanilla. I have not made any changes.
  
When you are done with the Dawnguard story, please speak to Serana and ask, "**Have you thought about getting cured of vampirism?”.**
  
  
For more information, please refer to the following article on UESP.
  
<https://en.uesp.net/wiki/Skyrim:Serana#Curing_Serana.27s_Vampirism>
  
  
...Serana doesn't want to do the cure?　Oh, it must be something you have said so far that has put her in a bad mood. The conversation during the Dawnguard quest will dictate whether the cure is possible or not.
  
  
If you really want to start the cure, please start the cure vampirism quest with the following command.
  

```
setstage DLC1SeranaCureSelfQuest 10
```

  
--------------------------------------------------------------------------
  
Q : What mods are compatible?
  
A :  This mod heavily modifies DLC1SeranaCureSelfQuest. It is not compatible with any mods that alter this quest. Please feel free to make patches and other modifications.
  
--------------------------------------------------------------------------
  
Q : What should I do if the quest stops working?
  
A : If you have a problem such as Serana not going to Morthal ,If you can skip the time by an hour or so, it will often be fixed.

【概要】
  
このMODはセラーナの吸血鬼治療クエストに、より「儀式」らしいシーンを加えるシンプルなMODです。
  
プレイヤーが体験したように、セラーナがファリオンから儀式を実際に受けます。
  
  
セラーナは治療のためモーサルに行き、ファリオンの所で約2日間滞在します。
  
その後、沼地の召喚サークルへ向かい、儀式を行います。プレイヤーが介入することはできません。儀式を見守る立場となります。
  
プレイヤーが儀式に立ち会わない場合でも、一日経つと儀式は完了します。

Q :どうしたらセラーナの吸血鬼治療クエストが始まりますか？
  
A :クエストが始まる条件については、バニラのまま、特に変更は加えておりませんわ。
  
　ドーンガードの物語に決着がついたら、わたしに「吸血症を治すことは考えてみたか？」と話しかけてくださいませ。
  
  
詳しくは、UESPの以下の記事をご覧になって。
  
<https://en.uesp.net/wiki/Skyrim:Serana#Curing_Serana.27s_Vampirism>
  
  
…治療をしたがらない？　ああ、それはあなたがこれまでにわたしの機嫌を損ねるような発言をしたのでしょう。
  
ドーンガードのクエスト中の会話によって、治療の可否が分岐いたしますの。
  
どうしても治療を始めたいのなら、以下のコマンドで吸血鬼治療クエストを始めてくださいまし。
  

```
setstage DLC1SeranaCureSelfQuest 10
```
