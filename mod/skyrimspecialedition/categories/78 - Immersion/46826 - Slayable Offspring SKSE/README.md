# Slayable Offspring SKSE
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/46826
**SLAYABLE OFFSPRING SKSE**
  
**Requirements**
  
  
SKSE64
  
Visual C++ Redistributables 2019 
  
Address Library for SKSE Plugins
  
  
**Description**
  
  
 ﻿﻿SKSE plugin that makes all children damageable and killable, by patching out the child checks in the game's damage/kill code.
  
  

  
* Plugin works on all child NPCs with the Child race flag. [size=3]Unique child animations (and other game features that depend on IsChild) will work properly.
  
For a mod that allows children to fight and defend themselves, check out [Savage Offspring SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/49377/)﻿
