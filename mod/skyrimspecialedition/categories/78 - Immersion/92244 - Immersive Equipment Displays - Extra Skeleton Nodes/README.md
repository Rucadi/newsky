# Immersive Equipment Displays - Extra Skeleton Nodes
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/92244


Using the skeleton extender function from IED, this mod will add some extra nodes that you can assign weapons to, using the in-game UI.

  
[font=Georgia]

[size=6]**REQUIREMENTS**

[/font]
  

[Immersive Equipment Displays](https://www.nexusmods.com/skyrimspecialedition/mods/62001)(IED for short): and its requirements.

  
[font=Georgia]

**GETTING STARTED**

[/font]
  
After installing the mod and loading the game, open IED's UI and navigate to: View>Gear positioning(or gear nodes).
  
Choose where to apply, Global or Actor for example and toggle "Placement".
  
Toggle "Unrestricted placement" if you are not seeing the nodes.
  
  
Extra nodes available:
  
**One-handed sword, back hip:**
  
Spoiler![](https://staticdelivery.nexusmods.com/mods/1704/images/78447/78447-1759317389-830122670.png)
  
**Two-handed, back hip:**
  
Same node as above, but for Two-Handed weapons.
  
  
P.S.: With [Combat Gameplay Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/33767), you can dual wield two-handed weapons, if you want to use this node for the left weapons, you'll need to:
  
- Use [Simple Dual Sheath](https://www.nexusmods.com/skyrimspecialedition/mods/50049).
  
- Open SimpleDualSheath.ini and navigate to [2HSword] - [2HAxe].
  
- Add Player in Flags.
  
- Then change SheathNode to WeaponBackLeft - WeaponBackAxeMaceLeft.
  
  

```
[2HSword]
  
Flags=Player
  
SheathNode=WeaponBackLeft
  

  
[2HAxe]
  
Flags=Player
  
SheathNode=WeaponBackAxeMaceLeft
```

  
In IED's UI navigate to View>Gear Nodes>Placement, expand Two-Handed Left and select the corresponding node:
  
Spoiler![](https://staticdelivery.nexusmods.com/mods/1704/images/78447/78447-1759317444-1393640149.png)
