**COMPATIBILITY**
  

  
* **Compatible** with all mods that replace vanilla meshes.
**Incompatible** with mods that add NEW weapons (unless included, of course). Any new weapon will have to be added to the list inside of the IED mod.
  
* **Incompatible** with most saddles added by Convenient Horses and other mods where saddles add weapons to the horse. Your computer won’t catch on fire if you use them together but… why?

* A mod that allows you to access your horse's inventory is **REQUIRED** to use this mod.  [Immersive Horses](https://www.nexusmods.com/skyrimspecialedition/mods/13402) is an example of such a mod. Use [Quick Inventory](https://www.nexusmods.com/skyrimspecialedition/mods/47165) as a work around. As a bonus, Quick Inventory also allows you to add equipment to a follower's horse.

  
  
**KNOWN ISSUES/CONCERNS**
  

  
* Did my best to space out the equipment so they didn't rub all over each other, hit the rider, or make Carry My Carcasses unusable. Except for warhammers, greatswords, and battleaxes. Those are stored in the exact same spot. Had to make some sacrifices, folks.
Clipping... Yep. It's there.
  
* IED presets are designed to fit vanilla horses but fit most modded horse models. Verified compatibility for [Oblivion Horses](https://www.nexusmods.com/skyrimspecialedition/mods/32335)﻿ and [SC - Horses](https://www.patreon.com/posts/sc-horses-35289631)﻿.
IED presets were designed around vanilla saddles. May need to adjust placement if using modded saddles or horse armor.
  

  
  
  
[size=4]**INSTALLATION:** *(thanks! you're the real Slim Shady)*
  

  
* Install using your mod manager
Once in game, open[Immersive Equipment Displays](https://www.nexusmods.com/skyrimspecialedition/mods/62001)' menu, using the default **LeftShift**+**Backspace** keys.
  
* Click on the "View" tab at the top
Select "Custom", which should open a new window
  
* From there, go all the way over to the right most tab labeled "Race"
Select "Horse Race" from dropdown
  
* Make sure "Sync" is checked
Click on the Preset bar and select "Use Those Horses" from the dropdown
  
* Select "Merge."
