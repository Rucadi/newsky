# Misc Voice Consistency Fixes
- Author: Exalderan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/135051


![](https://willeshaben.de/skyrimmods/Projectenhancedvanillaexperience.png)﻿

**Misc Voice Consistency Fixes** addresses a subtle but immersion-breaking issue in Skyrim:
  
certain NPCs using the wrong voice actor for specific dialogue lines.
  
These mismatches disrupt the flow of the game and break character continuity.**Key Fixes:**
  

  
* **Astrid(Dark Brotherhood):** Resolved dialogue inconsistencies where lines like *“The anticipation is killing me”*  were spoken in a mismatched sultry voice type (Susan Eisenberg).
**Nirya (Winterhold College):** Fixed 4 sudden voice changes to a different VA when she discusses Ancano, restoring consistency to her character.
  
* **Niranye (Markarth):** Fixed 2 voice changes to a different VA related to the thieves guild quest
**Mallus (Thieves Guild):** Fixed 2 voice changes to a different VA related to the thieves guild quest
  
* **Tonilia (Thieves Guild):** Fixed 2 voice changes to a different VA related to the thieves guild quest

  
This mod ensures that key NPCs sound as they should, keeping your game immersive and polished
