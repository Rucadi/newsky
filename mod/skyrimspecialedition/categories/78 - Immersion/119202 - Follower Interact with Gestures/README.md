**【Overview】**This is a mod that allows followers to react to the followers or NPCs around them.
  
They will take animations to talk or dance with each other, bringing the characters to life.
  
If there are dog followers, they can stroke them.
  
  
With MCM, you can adjust the frequency of reactions. The default is every minute.
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/119202/119202-1716996570-492057953.gif)![](https://staticdelivery.nexusmods.com/mods/1704/images/119202/119202-1716995073-832373044.gif)
  
Vanilla animation and some new animation created in Blender are used.
  
  
  
**【Required MOD】**
  
**[Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)**, or [Dynamic Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/33746)
  
[SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604) 
  
  
**【Mod Compatibility】**
  
SE, AE, and VR compatible.
  
I think it will work with all follower mods.
  
( In the internal data, this corresponds to the NPC who is the "PlayerTeammate")
  
  
I checkd it works with vanilla and follower management mod: [FollowerLivePackage](https://www.nexusmods.com/skyrim/mods/33002).
  
  
  
**【Install, How to use】**
  
Install mod using a MOD management tool etc.
  
  
Works when there are 2 or more followers. Some of the gestures can be operated by one person.
  
If a player stands still, followers around them will react.
  
It does not work during battle or while moving player.
  
  
  
**[List of gestures]**
  

  
* Talk :　　　Animation of two followers talking.　(Follower: 2 or more)
Dog :　　　Followers pet the dog followers.　(Follower: 2 or more)
  
* Dance : 　　If you are in an inn, 2 followers will dance. ﻿﻿Use the default cicero dance motion for each user.　(Follower: 2 or more)
SitTalk : 　　Follower goes near the follower who is sitting on the chair.　(Follower: 2 or more)
  
* JobTalk : 　　Animation of a conversation between followers and the store owner at day time.　(Follower: 1 or more)

  
  
**【MCM Menu】
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/119202/119202-1717090050-1534811129.png)**

* **Follower Interact Interval**

**【Q&A】**------------------------------------------------------
  
Q: Not only in the inn, but everywhere, followers doing the Cicero dance.
  
  
A: If for some reason the mod does not work correctly, this may be the case.
  
Please check the following: 
  
  1: Check if OAR (or DAR) is installed. 
  
  2: Check if the animation playback priority of the OAR of this mod is not overwritten by other mods.
  
       If the OAR files **cicerodance1.hkx**, **cicerodance2.hkx** are overwritten, this mod will not work correctly.
  
  
If it still does not work, it is a problem that I am not aware of. As soon as we find the cause, I will take care of it.

**【概要】**
  
フォロワーが周囲のフォロワー、またはNPCに反応するMODです。
  
お互いに話したり、踊ったりするモーションを取ることで、キャラクターを生き生きとさせます。
  
フォロワーに犬がいる場合は犬を撫でたりします。
  
  
MCMで、相互反応の頻度を調節できます。デフォルトでは、1分おきとなっています。
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/119202/119202-1716996570-492057953.gif)![](https://staticdelivery.nexusmods.com/mods/1704/images/119202/119202-1716995073-832373044.gif)
  
バニラのモーションおよび、Blenderにて作成したいくつかの新規モーションを使用しています。
  
  
  
**【必須MOD】**
  
**[Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)**, または [Dynamic Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/33746)
  
[SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604)﻿ 
  
  
  
**【MODの互換性】**
  
SE、AE、VR対応。
  
フォロワー系MOD全般で動作するかと思います。
  
（正確には、内部データで、 “PlayerTeammate” とされたNPCに対応します）
  
  
バニラ及び、フォロワー管理MOD：[FollowerLivePackage](https://www.nexusmods.com/skyrim/mods/33002)導入環境で動作確認しています。
  
  
  
**【インストール、動作条件】**
  
MOD管理ツールなどを使い、インストールをしてください。
  
日本語版があります。これはMCMメニューを日本語化したものです。
  
  
2名以上のフォロワーがいる場合に動作します。一部1名でも動作するジャスチャーがあります。
  
プレイヤーが立ち止まっている場合、周囲にいるフォロワーが反応します。
  
戦闘中、移動中は動作しません。
  
  
  
**【ジェスチャー一覧】**
  

  
* Talk :  　　2人のフォロワーが会話するモーションをします。（フォロワー：2名以上）
Dog : 　　フォロワーが、犬のフォロワーに向かって撫でます。（フォロワー：2名以上）
  
* Dance : 　宿屋にいる場合、2人のフォロワーが踊ります。﻿各ユーザーのデフォルトのシセロのダンスモーションを用いています。（フォロワー：2名以上）
SitTalk : フォロワーが椅子に座っているフォロワーの近くに行きます。（フォロワー：2名以上）
  
* JobTalk : フォロワーと、店の店主が会話するモーションをします。 昼限定。　（フォロワー：1名以上））

**【Q&A】**﻿------------------------------------------------------
  
Q 宿屋だけではなく、あらゆる場所で、フォロワーがシセロのダンスをしてしまう！
  
  
A 何らかの原因でMODがうまく働かない場合、そのような現象になる場合がございますわ。
  
以下を確認していただけると幸いです
  
  
1： OAR（またはDAR）がインストールされているか確認する
  
2： このMODのOARのモーション再生優先順位が、他のMODで上書きされていないか確認する。　　
  
  具体的には、OARで、**cicerodance1.hkx**, **cicerodance2.hkx**のファイルが上書きされていると、このMODは正しく動かなくなりますの。
  
  
それでも動かない場合は、こちらで把握できていない不具合ですわ。原因が判明次第、対応をいたしますわね。﻿------------------------------------------------------
