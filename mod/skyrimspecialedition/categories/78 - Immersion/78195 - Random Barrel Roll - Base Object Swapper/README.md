**Description**
  

  
* Level designers sometimes leave barrels with the same rotation.
This mod randomly rotates all barrels in the game so they are never aligned.
  
* Uses [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿ 2.5.0
No ESP
  
* No meshes or textures
No worldspace edits
  
* [Dynamic Things Alternative - Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60741)﻿ automatically supported

  
**Requirements** 

  
* [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿ 2.5.0 minimum
