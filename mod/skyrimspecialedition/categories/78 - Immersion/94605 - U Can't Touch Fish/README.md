# U Can't Touch Fish
- Author: lilmetal
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/94605


**Description:**
  
This mod blocks the ability for the player to grab fish out the water.
  
Offered in two flavors:
  
  

  
* **Basic:**

Basic. U Can't Touch Fish.
  
(With an MCM to toggle the touching of fish)
  
  

  
* **Chance:**

Chance. U CAN Touch Fish if you're Argonian or Khajiit.
  
All other races have a 5% chance of touching fish.
  
With a 25% chance the fish runs away upon failure.
  
(No MCM or customization options)
  
Chance Edition also offers optional compatibility with Immersive Interactions.

**The Salmon Roe Problem:**
  
Spoiler
  
Bethesda added the [Salmon Roe](https://en.uesp.net/wiki/Skyrim:Salmon_Roe) alchemy ingredient in Hearthfires, but never properly distributed it in the game.
  
The only way they integrated this item was by catching jumping waterfall Salmon (which this mod will block), and only a few specific other locations.
  
  
Some mods that fix this:
  

  
* [Containers and Leveled Lists Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/26575) and [SkyRem Loot](https://www.nexusmods.com/skyrimspecialedition/mods/26732) (Adds Salmon Roe to all leveled lists where alchemy ingredients can be found or purchased)
[Producers of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/61123) and [Fishermen Sell Creation Club Fish](https://www.nexusmods.com/skyrimspecialedition/mods/89208)﻿ (Adds Salmon Roe to all fishermen)
  
* [Unofficial Skyrim Special Edition Patch](https://www.nexusmods.com/skyrimspecialedition/mods/266) and [Apothecary](https://www.nexusmods.com/skyrimspecialedition/mods/52130) (Make dead Salmon drop Salmon Roe)
[Merchant Wares Variety](https://www.nexusmods.com/skyrimspecialedition/mods/34151) (Makes Arcadia sell Salmon Roe in Whiterun)
  

All of these mods can be used together, if you wish.
  
  
Otherwise, you can edit the KID file in this mod, and remove the
  
**",0x10B035,0x10C3BA"** part from the end of the Skyrim line.
  
These are the waterfall salmon. U can touch if removed.
