# Missives - Notes and Announcements
- Author: Enneal
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/116028
**Description**A small addition to Missives to add notices (~90) during important events. Includes announcements, eg. during the Civil War, and other less important messages. The notes don't do anything. 
  
  
**Requirements**
  

  
* [Missives](https://www.nexusmods.com/skyrimspecialedition/mods/17576) ﻿(obviously)
[Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿
  

  
**Compatibility**
  

  
* Compatible with any patches that move the Missive boards
Compatible with [Missives - Worldspace Additions](https://www.nexusmods.com/skyrimspecialedition/mods/26788)﻿
  
* Should be compatible with any mod that doesn't edit the activators
There may be inconsistencies with mods that significantly alter certain quests
  

  
**F.A.Q
  
  
ESL Flagged?**
  
Yes
