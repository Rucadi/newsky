# Simple Campfire Additions
- Author: lonewolfsniper676
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/21956


[![](http://i.imgur.com/kPyl4AD.png)](http://www.nexusmods.com/skyrim/mods/64798)

  
  
This mod adds a variety of new items under the Create Item power to add to the camping experience.
  
  
Current Additions:
  
- Common Chair
  
- Common Table
  
- Archery Target
  
- Grindstone
  
- Workbench
  
- Anvil
  
- Practice Dummy
  
  
Will consider adding more based on personal preference and suggestions.
  
  
Known issues:
  
When first placing certain items you may become stuck in the center of an invisible outline of the placeable object and must jump to free yourself. Only really noticeable when placing large objects such as the table and workbench.
