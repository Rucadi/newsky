# Animated Carriage Additional Route
- Author: Vicn-kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/112548


Animated carriage by Vicn brings more immersion to Skyrim by enabling carriage travelling on the road. Unfortunately, the original mod only adds two routes near whiterun. This mod adds more route for the travelling carriage so that you can find it in other region. This is an ongoing project to hopefully cover all skyrim regions. As for now, i have added 42 more routes in Skyrim and Solstheim:
