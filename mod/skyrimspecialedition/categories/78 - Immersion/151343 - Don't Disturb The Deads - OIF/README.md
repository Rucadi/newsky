# Don't Disturb The Deads - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/151343


This mod will make everytime you loot corpses (draugr, burnt corpses, web woven corpse), there is 20% chance of you making the ghost of those corpses attack you. There is also 10% chance of you getting cursed when you loot the corpses. All chances are configurable in json if you feel it is too rare or too frequent. Enjoy!
