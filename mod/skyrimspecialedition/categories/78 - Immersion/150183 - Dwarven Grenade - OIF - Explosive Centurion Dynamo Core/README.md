# Dwarven Grenade - OIF - Explosive Centurion Dynamo Core
- Author: zhitsak
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150183


![](https://i.imgur.com/c7tRUmF.png)
  
  
Repurpose the mysterious Dwemer core into a grenade. Throw it or strike it to trigger an explosion dealing 50 points of damage.
  
  
Once a vital part of a Dwemer Centurion, this dynamo core now serves a deadlier purpose. When hurled or hit, it detonates in a burst of stored energy. Simple, destructive, and unmistakably Dwemer.
  
  

**Scriptless, ESPFE, compatible with everything.**
