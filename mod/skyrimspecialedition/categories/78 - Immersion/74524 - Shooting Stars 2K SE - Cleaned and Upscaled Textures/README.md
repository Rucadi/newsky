# Shooting Stars 2K SE - Cleaned and Upscaled Textures
- Author: Isoku - XilaMonstrr
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/74524


Effect textures and gradients are now much smoother. Original files were in 256 and 512 resolutionin mostly BC3 compression.
  
  
I have cleaned all files of compression artifacts, and then upscaled them all by 4x, to a maximum resolution of 2K in BC7 compression and generated full mipmaps. I cleaned and upscaled them using [Cupscale](https://github.com/n00mkrad/cupscale)/ESRGAN and then finalized with NVIDIA Texture Tools and [Octagon](https://www.nexusmods.com/skyrimspecialedition/mods/28773). 
  
  
Installation
  
Requires [the original mod](https://www.nexusmods.com/skyrimspecialedition/mods/73090)
  
  
Performance Option for Potato PCs
  
Mipmaps and BC7 compression reduce VRAM usage, especially when the files are cleaned of compression artifacts.
  
For Potato PCs, I've uploaded a downscaled version where the diffuse texture files are larger than normal and environment map files, which are less visible in game. This reduces VRAM usage considerably while maintaining higher visual quality, using the [Octagon config files posted on this mod](https://www.nexusmods.com/skyrimspecialedition/mods/68055), which I recommend using if you have a Potato PC.
  
  
Thank you so much to [Isoku](https://www.nexusmods.com/users/2463371)for creating the original mod and giving permission for me to release this mod.
