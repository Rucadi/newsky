# DAc0da - CoMAP Addon
- Author: Giuug and Sputnik
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/147698


We're working to bring hand-drawn map markers to all of Vicn's major mods, from Vigilant to Unslaad, so stay tuned!
  
  
If there are any bugs or problems feel free to tell me and I'll do my best to fix them.
  
  
Here's Giuug's [Instagram page](https://www.instagram.com/giuug_art?igsh=amN4cnVoMGs0eWZk), feel free to visit anytime.
  
  
Made with Vicn's blessing, of course!
  
  
WARNING TO MO2 USERS: The mod is NOT broken. If you get the warning: "content <data> does not look valid" simply select the "ignore"
  
option and the mod will still install properly without issues.
