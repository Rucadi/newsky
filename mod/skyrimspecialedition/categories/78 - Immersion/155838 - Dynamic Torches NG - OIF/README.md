# Dynamic Torches NG - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/155838


This is a remake and a better version of older mod, Dynamic Torces - BOS, which basically only swap all torches with removable version. There are actually two kind of torches in Skyrim, one is the permanent which is not removable and the other is removable one which you can take the torch from. Therefore swapping all the torches into removeable one i think that is not the best solution.
  
  
This mod handles the torches differently and will make interaction with torches more advanced than the previous mod. With this mod, now you can interact with torches, turn it off or on, fire spell at them to ignite or extinguish them, and also take the torches if they are removeable. This mod doesn't remove the light, so any mod such as light placer with placed light is recommended for maximum immersion.
