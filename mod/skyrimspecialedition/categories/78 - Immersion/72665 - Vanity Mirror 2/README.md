# Vanity Mirror 2
- Author: RGMage2
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72665


This version of the Vanity Mirror is now a CLUTTER item that will open the [Limited Race Menu](https://en.uesp.net/wiki/Skyrim:Surgery) to allow you to make changes to your appearance, anywhere, anytime, at no cost\*.
  
  
Read [here, the original's Mod Page](https://www.nexusmods.com/skyrim/mods/52712) for more info.
  
  
[Requirements]
  
[SKSE](https://skse.silverlock.org/)
  
  
[Features]
  

  
* **The mirror is now a clutter item -** It is categorized as a Misc Item, and will not unequip an armor in Slot 53.
    
   ﻿Also prevents a bug where it will somehow equip a random armor you have.
**A crafting recipe -** [size=4][size=4]\*[/size]Requires a Silver Ingot, a Firewood and a Leather Strip.[/size]
  
* **Has clutter related sounds -** dropping it will produce a sound as well.
**Mirror Icon** - If you have [T.H.I.C.C.](https://www.nexusmods.com/skyrimspecialedition/mods/90508)﻿, the inventory icon becomes a mirror.  
  
 ﻿Load the esp after the icon mods. Load it especially after [AIT](https://www.nexusmods.com/skyrimspecialedition/mods/68557)﻿ and [I4IconAddon](https://www.nexusmods.com/skyrimspecialedition/mods/85702)﻿.
