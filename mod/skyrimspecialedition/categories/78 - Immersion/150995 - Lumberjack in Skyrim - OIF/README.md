# Lumberjack in Skyrim - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150995


[left]﻿This mod will enable you to cut off tree in Skyrim forest, the tree will then fall off (kinda), after that you can chop them again into firewoods. The process of chopping off the tree is divided into four parts:
  
1. cutting the tree
  
2. cutting the fallen tree
  
3. cutting the half of the log
  
4. cutting the quarter of the log into firewoods
  
Every part need at least to be hit 5 or 6 times. In total, every tree will result in 48 firewoods, you can change all the configuration in json file to your liking. This mod uses vanilla mesh so any new tree mod may result in looking not consistent between the tree and the log. The aim of this mod is not realism but immersion and gameplay as close as possible to reality. [/left]
