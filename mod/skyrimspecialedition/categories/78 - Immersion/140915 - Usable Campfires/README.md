# Usable Campfires
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140915


**[font=Georgia]INTRODUCTION[/font]**

  
The mod replaces lit(or unlit) campfires in the world with interactible "furnitures" that you can cook on, utilizing Base Object Swapper.
  
  
Two versions, one for vanilla game or for [Camping CC](https://en.uesp.net/wiki/Skyrim:Camping) and another one for the [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667) mod.
  
  

**[font=Georgia]REQUIREMENTS[/font]**

  
- [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/96273)
  
Optional:
  
- [FormList Manipulator](https://www.nexusmods.com/skyrimspecialedition/mods/74037): Patches Sunhelm, Survival Mode, Warming Hands, Hunterborn, Campfire/Frostfall and Simple Portable Cooking.
  
- [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273): Required if using the Rest at the Campfire optional file.

**[font=Georgia]BONUS FEATURES[/font]**
  
  
Cooking animations will be added to the campfire if using [Campfire Animations](https://www.nexusmods.com/skyrimspecialedition/mods/112322) and [Keyword Item Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/55728).
  
**[font=Georgia]
  
COMPATIBILITY[/font]**

  
- This mod doesn't touch the vanilla model of the campfire, so it should be compatible with other mods that retexture, remodel or "fix" the campfires, such as [Embers XD](https://www.nexusmods.com/skyrimspecialedition/mods/37085), [Campfire Reacts to the Wind](https://www.nexusmods.com/skyrimspecialedition/mods/103807), etc...
  
  
- [Diverse Campfires](https://www.nexusmods.com/skyrimspecialedition/mods/109731): Yes. Install the optional files and let Usable Campfires win the conflicts.
  
- [Rain Extinguishes Fires](https://www.nexusmods.com/skyrimspecialedition/mods/80419): Yes. Included config file for compatibility, let Usable Campfires win the conflicts.
  
- [Respawn - Soulslike Edition](https://www.nexusmods.com/skyrimspecialedition/mods/69267): Yes. Resting at the campfire (with the optional file) will create a respawn point.
  
- [Camping Lite](https://www.nexusmods.com/skyrimspecialedition/mods/2370): Yes. Its custom cell will be excluded while swapping campfires.
  
- [Usable Unlit Campfires - A Campfire Addon](https://www.nexusmods.com/skyrimspecialedition/mods/108712): Yes. Compatible if you're not using the "Unlit Campfires" optional file.
  
- [Rest By Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/96202): Not compatible. A similar feature is provided via optional file (Requires [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273)).
  
- [Swap all Campfires with Campfires from Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/98144): No. Both mods will fight with each other in order to replace the campfires.
