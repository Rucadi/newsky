# Akaviri History - Lore-Friendly Additions and Tweaks
- Author: pleonasmo3ponto  BURBURINHO
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/76804


[font=Trebuchet MS]The idea for Akaviri History came from me mulling over
  
just how under utilized [Mysterious Akavir Journey to the East 3D Assets](https://www.nexusmods.com/skyrimspecialedition/mods/45342) 
  
and looking at Adam Adamowicz's fantastic unused concept art for Blades clutter.
  
So I set on a mission to try and bring some of that to life
  
I'm happy to present to you my first mod ever﻿
  
[/font]

[font=Trebuchet MS]![](https://i.imgur.com/rPoWfiK.png)﻿
  
**Plugins**
  
Main File :
  
An ancient Gate found in Karthspire camp
  
Ancient tombstones (Karthspire Redoubt and Sky Haven exterior)
  
Akaviri pots found in Sky Haven Temple (interior)
  
Wooden tables chairs replaced by Akaviri stone furniture
  
  
Unique Esbern Equipment :
  
Gives Esbern a new Blades Robes set upon completing Alduin's Wall
  
New unique dagger for Esbern replacing his generic Ebony Dagger
  
  
[size=3]Akaviri Armory :
  
New Blades Weapons found in Sky Haven and
  
Sleeping Giant Inn (Delphine's basement)
  
**Weapons include :**
  
Blades Bow
  
Blades Battleaxe
  
Blades Greatsword
  
[/size]**Optional Integration : [[b]More Craftable Equipment](https://www.nexusmods.com/skyrimspecialedition/mods/44666)**[/b][/font][font=Trebuchet MS]**[[b]](https://www.nexusmods.com/skyrimspecialedition/mods/44666)**[/b]This will allow you to craft a full set of Blades clothes
  
without the need of a patch.
  
  
Beyond Skyrim Bruma Patch :
  
Draconian Madstone amulet found in Castle Bruma
  
Ancient Akaviri pots in Castle Bruma and Cloud Ruler Temple
  
Akaviri woven baskets in Cloud Ruler Temple
  
  
Ryn's Karthspire Patch :
  
Moves the Akaviri gate so Akaviri History can be used alongside
  
[Ryn2g's](https://www.nexusmods.com/skyrimspecialedition/users/81880043)﻿ incredible mod [Ryn's Karthspire](https://www.nexusmods.com/skyrimspecialedition/mods/77059)﻿
  
  
COMAP Patch :
  
Makes it so there is only one map marker for the karthspire, "Karthspire Ruins",
  
rather than the two seperate markers in vanilla
  
If you have COMAP installed it will use a Akaviri marker
  
thank you [Jelidity](https://www.nexusmods.com/skyrimspecialedition/users/4569617) for providing the patch <3
