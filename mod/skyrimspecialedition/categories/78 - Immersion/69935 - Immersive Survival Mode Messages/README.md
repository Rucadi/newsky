# Immersive Survival Mode Messages
- Author: saton269
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/69935


This is just a very small mod that changes the messages displayed by Survival Mode from third person to first person. It also changes the help popups as well to help keep the immersion going. 
  
  
Also available for [Xbox](https://bethesda.net/en/mods/skyrim/mod-detail/4276790)
  
  
Requires: CC Survival Mode
  
  
Compatibility: Should be compatible with just about everything that does not touch the messages displayed by Survival Mode.
