# Skip Vampire Lord Tutorial
- Author: Kamikaze
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/44433


**DESCRIPTION**
  
Skip the Vampire Lord tutorial when you accept Harkon's gift. Harkon still gives his exposition (but you can skip that too).
  
You won't notice there ever was a tutorial. What else is there to say? Download, install, endorse!
  
  
The plugin is flagged as an ESL so it won't take up precious slots in your load order!
  
  
**COMPATIBILITY**
  
  
Compatible with everything.
