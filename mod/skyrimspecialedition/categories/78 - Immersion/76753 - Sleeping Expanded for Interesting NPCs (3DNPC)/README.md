# Sleeping Expanded for Interesting NPCs (3DNPC)
- Author: TheCyclist
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/76753
[![](https://i.imgur.com/BZarQov.png)](https://www.buymeacoffee.com/TheCyclist)
  
  
**Description**

  
* Interesting NPCs will react when you wake them up.
Currently 16 NPCs are supported.
  
* ESL-Flagged ESP.
Safe to install mid-playthrough.
  

  
**Requirements**
  

  
* [Interesting NPCs SE (3DNPC)](https://www.nexusmods.com/skyrimspecialedition/mods/29194)

  
* [Sleeping Expanded](https://www.nexusmods.com/skyrimspecialedition/mods/59250)

  
﻿**Compatible with**
  

  
* Nether's Follower Framework (NFF)

  
* And any other mods
