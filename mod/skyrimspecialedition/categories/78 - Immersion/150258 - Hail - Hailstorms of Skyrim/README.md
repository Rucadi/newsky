# Hail - Hailstorms of Skyrim
- Author: Truman-Wired1up-Clofas
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150258


[﻿﻿](https://github.com/TrumanGIT/Hail---Hailstorms-of-Skyrim)﻿[﻿](https://www.nexusmods.com/skyrimspecialedition/mods/1090?tab=description)﻿﻿

  
  
  
 ﻿    ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿
  
  
  
  
  
![](https://i.imgur.com/jxdum7Z.png)
  
  
  
Hailstorms can vary in size and intensity and can damage the player and NPCs (can change in the in game menu) 
  
  
NPCs will react to hail with vanilla repurposed dialogue and will seek shelter by running home or to nearest inn if their from out of town. 
  
  
Hail storms can be stopped at any time by using the 'Clear Skies' shout.
  
  
**Compatibility?**
  
[Dylbills Papyrus Functions](https://www.nexusmods.com/skyrimspecialedition/mods/65410) - Slight incompatibility.
  
(Set iMaxArrowsSavedPerReference to 0 in DbSkseFunctions.ini).
  
  
**Safe to install / remove mid game?**
  
Yes, everything is handled through SKSE. (No scripts)
  
  
**Performance Impact?**
  
You may notice some fps drop during hail depending on your setup.
