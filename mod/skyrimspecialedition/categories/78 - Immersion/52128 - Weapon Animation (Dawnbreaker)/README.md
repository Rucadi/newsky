# Weapon Animation (Dawnbreaker)
- Author: saltmaxwellpshk
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/52128


Add an animation to Dawnbreaker.
  
Works with anime events.
  
No plugins or scripts are used.
  
Only the mesh is changed.
  
  
anything that changes the mesh of the weapon is incompatible.
  
Retexture mods are compatible.
  
  
Works with Weapon Draw and Weapon Sheathe,Power Attack.
  
See the video for animation.
