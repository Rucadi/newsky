DESCRIPTION

This mod adds chimneys to farmhouse exteriors that have fireplaces inside.
  
  
**Features**
  
[\*] Smoke effects for the chimneys
  
[\*] Dynamic snow shaders that will use the installed snow textures, where applicable
  
[\*] Meshes will use the installed farmhouse textures
  
[\*]Optional clay chimney stacks meshes
  
[\*]MCM for toggling chimneys for vanilla villages, as well as, many mods. 
  
[\*] Includes support for other mods by using plugins.
  
  
[line]
  
  

COMPATIBILITY

  
  
GENERAL COMPATIBILITY:
  
The chimneys should work with any mod that doesn't move farmhouses or adds its own chimneys. Mods that add new farmhouses will need a compatibility update to support them.

INSTALLATION

  
  
The patches have been made with a LOOT sorted load order; therefore, allow LOOT to sort any installed plugins.
  
  
MO and NMM:
  
To install the mod, simply follow the directions in the included installer. The installer will automatically select recommended options based on your active plugins.
  
  
Wrye Bash:
  
The mod is fully BAIN compliant.
