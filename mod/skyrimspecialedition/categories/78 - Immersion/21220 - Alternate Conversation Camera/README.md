# Alternate Conversation Camera
- Author: NasiGorengCumi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/21220
[font=Trebuchet MS]**Makes conversation with NPCs more immersive by zooming in to their faces. Fully supports both 1st and 3rd person camera, also supports on sitting and lock-on states.  Includes "Witcher 3" camera style. Configurable with INI file.
  
  
Features:
  
- Adjustable and dynamic zoom based on the distance between player and npc
  
- Adjustable pov
  
- Adjustable camera speed
  
- Auto lock-on the npc
  
- Adjustable third person camera position
  
- Built-in conversation headtracking
  
- Dynamic letterboxing when in conversation
  
- Auto switch between player and npc pov
  
- ~~Auto freeze all npcs when in conversation~~
  
  
*Please check the INI file first, before you are going to use the mod.*
  
Have fun and enjoy. :D
  
  
Requirements:
  
- Skyrim SE or AE
  
- SKSE64
  
  
Installation:
  
- Either use mod manager or manually copy the file
  
  
Uninstallation:
  
- Either use mod manager or manually delete the file
