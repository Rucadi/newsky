# Hammet's Dungeon EVGAT Patch
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/73960


This mod adds more than a hundred EVGAT markers to almost all dungeons from [Hammet's Dungeon Mod](https://www.nexusmods.com/skyrimspecialedition/mods/12186) to give you a different experience when exploring dungeon and more immersive experience!
  
  
**Even when there is already Skyclimb mod, this mod is not at all replaced because this mod adds more than climbing and vaulting over marker which skyclimb already covers.**
  
  
The light markers are on by default to make it easier to find. I have also added some additional location inside a few dungeon accessable only with EVGAT markers.
