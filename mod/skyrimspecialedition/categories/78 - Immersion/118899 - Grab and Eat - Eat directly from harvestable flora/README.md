# Grab and Eat - Eat directly from harvestable flora
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/118899


**[font=Georgia]INTRODUCTION[/font]**

  

Press Shift + E to eat food or ingredients directly from harvestable flora

  

**[font=Georgia]REQUIREMENTS[/font]**

  
- [Dynamic Activation Key](https://www.nexusmods.com/skyrimspecialedition/mods/96273)
  
- [powerofthree's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854)

**[font=Georgia]Q&A[/font]**

  
  
**Q: ESL flagged?**
  
*A: Yes.*
  
  
**Q: Animations?**
  
*A: No animations or displays for this mod :O, you'll need to use another mod for that.*
  
  
**Q: Limitations?**
  
*A: Only food or ingredients will be consumed when using the Dynamic Activation Key. Some flora actually gives you the items via scripts (Nirnroot or similar), the mod won't work with them.*
  
  
**Q: This xxx flora shouldn't be consumed!**
  
*A: Add the flora to the EFlora\_IgnoreList so it can be ignored by the mod. Use [FormList Manipulator](https://www.nexusmods.com/skyrimspecialedition/mods/74037) and the file EatFromFlora\_FLM.ini or similar method.*
