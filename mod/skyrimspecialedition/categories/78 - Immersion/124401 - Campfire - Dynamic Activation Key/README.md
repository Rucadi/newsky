# Campfire - Dynamic Activation Key
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/124401
**[font=Georgia]INTRODUCTION[/font]**

  
Press Shift + E to interact with a Campfire:
  
- **Light Campfire**: When the campfire is empty or has fuel/tinder ready (and you have enough materials to do so).
  
- **Upgrade Campfire**: When the campfire is burning and can be upgraded (and you have enough materials to do so).
  
- **Access Skills**: When the campfire is burning and can't be upgraded anymore (or you don't have enough materials to do so).
  
  
The first materials available will be used during the alternative interaction when lighting or upgrading a campfire.
  
  
Alternative interactions for some Campfire objects:
  

**Tent:** [list]* Normal activation = Sleep.
Press Shift + E = Prompt menu.
  

**Cooking pot:**

* Normal activation = Cook.
Press Shift + E = Pick up.
  

**Enchanter's Circle:**

* Normal activation = Enchant items.
Press Shift + E = Pick up.
  

**Tanning rack:**

* Normal activation = Create items.
Press Shift + E = Pick up.
  

**Hearth Craft furnitures:**

* Normal activation = Use furniture.
Press Shift + E = Pick up.
  

[/list]

**[font=Georgia]OPTIONAL FEATURE[/font]**

  
When near a flickering or above campfire, your character can start warming their hands when they're feeling cold.
  
The feature is disabled by default, to enabled it, access Open Animation Replacer menu, search for the mod and enable the folder:
  
![](https://i.imgur.com/8pgeu58.png)
  
Use "User mode" and save.
  
  
The trigger box is a bit large, so the animation can start playing a bit far away from the campfire.
