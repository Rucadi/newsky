# Go to bed
- Author: Andrelo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/4224


[font=Trebuchet MS]

﻿Description

  

This mod is for those who don't like to sleep standing next to the bed.

  

Gameplay changes:
  
  
- your character lies down after bed activation
  
- to open sleep menu use Wait(T)\* button
  
- to serve time in jail use Wait(T)\* button

  

\* button can be changed in mod settings

  

Sleepwear

  

You can select sleepwear for your character in MCM menu.

  

Npcs will use sleepwear if they have one.
