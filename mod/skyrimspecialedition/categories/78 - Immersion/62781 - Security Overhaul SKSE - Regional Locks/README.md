**Description**
  

  
* Adds [Solitude](https://en.uesp.net/wiki/Skyrim:Solitude)﻿ lock
Adds [Whiterun](https://en.uesp.net/wiki/Skyrim:Whiterun)﻿ lock
  
* Adds [Windhelm](https://en.uesp.net/wiki/Skyrim:Windhelm)﻿ lock
Adds [Markarth](https://en.uesp.net/wiki/Skyrim:Markarth)﻿ lock (none of these are locked in vanilla. Requires [Unique Markarth Doors](https://www.nexusmods.com/skyrimspecialedition/mods/62969) to see)
  
* Adds [Riften](https://en.uesp.net/wiki/Skyrim:Riften)﻿ lock
Adds [Raven Rock](https://en.uesp.net/wiki/Skyrim:Raven_Rock)﻿ lock
  
* Adds [Tel Mithryn](https://en.uesp.net/wiki/Skyrim:Tel_Mithryn_(settlement))﻿ lock
Adds [College of Winterhold](https://en.uesp.net/wiki/Skyrim:College_of_Winterhold_(place))﻿ lock
  
* Adds cage door lock
Adds safe and strongbox lock

**Requirements**
  

  
* [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) if you're playing on Skyrim Special Edition ("Anniversay Edition" is still just SSE). Requires version 3.0 or higher.
[Security Overhaul SKSEVR - Lock Variations VR](https://www.nexusmods.com/skyrimspecialedition/mods/58298) if you're playing on VR. Requires version 3.0 or higher.
  

**FAQ**
  
> Q: Can I install, uninstall, upgrade this mid playthrough?
>   
> A: Yes
>   
>   
> Q: What is required for this mod?
>   
> A: [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) if you're playing on Skyrim Special Edition ("Anniversay Edition" is still just SSE).
>   
>     [﻿Security Overhaul SKSEVR - Lock Variations VR](https://www.nexusmods.com/skyrimspecialedition/mods/58298) if you're playing on VR
>   
>   
> Q: Does this work with the "Anniversary Edition update" (v1.6)?
>   
> A: Yes, as of [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) version 2.0
>   
>   
> Q: Does this work with Special Edition (v1.5.97)?
>   
> A: Yes
>   
>   
> Q: Does this work for VR?
>   
> A: Yes, if you've installed the VR requirement: [Security Overhaul SKSEVR - Lock Variations VR](https://www.nexusmods.com/skyrimspecialedition/mods/58298)
>   
> ﻿﻿﻿Q: Does this work with <popular lock replacer mod> as well?
>   
> A: Yes
>   
>   
> Q: Mod isn't working even though I have installed all the requirements?
>   
> A: Uninstall [Wearable Lanterns MCM Fix](https://www.nexusmods.com/skyrimspecialedition/mods/25372?). Wearable Lanterns MCM Fix included an LE/Oldrim version of StorageUtil.dll which will completely prevent SKSE from loading any dll plugins.
>   
>   
> Q: My download is corrupt.
>   
> A: Reload the file then.
>   
>   
> Q: Another Security Overhaul SKSE addon is adding a different lock to the door or container.
>   
> A: Read the [Security Overhaul SKSE - Lock Variations](https://www.nexusmods.com/skyrimspecialedition/mods/58224) mod description and edit the \_LID.ini file to your desired outcome.
