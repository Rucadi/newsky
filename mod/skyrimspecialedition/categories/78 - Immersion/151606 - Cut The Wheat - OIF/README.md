# Cut The Wheat - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/151606


This mod was inspired by the new released mod "[Real Wheat Fields](https://www.nexusmods.com/skyrimspecialedition/mods/151229?tab=files)". The new, vast look of wheat field made me think, why not take it further to make it possible to cut them like in Zelda BOTW? Yeah with this mod, you can now cut the wheat with sharp large weapon and it will produce three wheat ingredients. Enjoy!
  
  

![](https://images2.imgbox.com/50/3e/G1oLvqcM_o.gif)

  
  
**Installation:**1. Install [Wheat Replacer - Gluten-Free Grainless Crop](https://www.nexusmods.com/skyrimspecialedition/mods/112074)2. ﻿Install Real Wheat Fields (Optional actually, but recommended to look like in the pic)
  
3. Install this mod and let this mod overwrite other mods
