# Detailed NPCs - Blanket Scarves for NPCs (SPID)
- Author: BecomingNplus1
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63951


**Description**
  

  
* Mod uses [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ to distribute blanket scarves from [Blanket Scarf Earth Tones Edition - 20 styles](https://www.nexusmods.com/skyrimspecialedition/mods/5050)﻿ to NPCs﻿﻿
Scarves uses body slot 46
  
* File standalone
File ESL-flagged
  
* No records edited
Player character not impacted﻿
  
* \_DISTR.ini designed for maximum configurability

  
**Requirements**

  
* [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿

  
**Overview**
  
My mod utilizes a small script that runs once (at game initialization) to distribute and equip blanket scarves to [Elder](https://en.uesp.net/wiki/Skyrim:Elder) NPCs. Mod randomly distributes one (1) scarf of a possible twenty-one (21) texture variations. I have edited the distribution so that only about half of the [Elder](https://en.uesp.net/wiki/Skyrim:Elder)﻿NPCs are covered. See \_DISTRI.ini
  
  
**Mod can be re-configured to distribute scarves to any NPC and / or any group of NPCs that you wish!** Edit the \_DISTRI.ini that comes with this mod, and add (or delete) any Form ID that you wish.
  
  
**Compatibility**
  
Highly compatible with most setups, and with most other mods. 
  

  
* [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ does not edit levelled lists, and does not modify or generate new records.
Body slots, however, are not compatible. Choose wisely, or edit the body slot. I will not alter body slots.
  
* Clipping will likely occur depending on setup. I will not resolve clipping issues.

  
**Installation and Uninstallation**
  
Use the mod manager of your choice to install. Or manually. To uninstall, remove with mod manager. My understanding is that SPID will not impact save games in any way. Safe to uninstall at anytime. 
  
  
**Configuration**
  
Huge number of configuration options available in the \_DISTRI.ini. Some knowledge of [Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869)﻿ is useful to achieve desired results.
