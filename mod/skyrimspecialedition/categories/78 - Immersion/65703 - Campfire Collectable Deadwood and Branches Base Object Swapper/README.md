[font=Georgia]I created this mod because I always thought it was ridiculous that those dead branches scattered here and there couldn't be picked up to start a campfire. With this [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667)﻿ addon its now possible to collect coastdriftwood and reachbranchpile and use the so obtained deadwood and branches to light a campfire. This mod works with all coastdriftwood and reachbranchpile, even those added by other mods.
  
[/font]
