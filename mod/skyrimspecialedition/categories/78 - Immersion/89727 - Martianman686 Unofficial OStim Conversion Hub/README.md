# Martianman686 Unofficial OStim Conversion Hub
- Author: Martianman686 SimonPhill
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/89727


Requires the Original versions.
  
  
Override the scripts with the "Martianman686 Scripts" file.
  
Then download the plugin for the mod you want here and override the original.
  
  
These mods were made by [Martianman686](https://www.nexusmods.com/skyrimspecialedition/users/179850360) I converted the scripts and plugins to use the OStim animation framework.
  
  
These should be compatible with OStim, OStim NG, and OStim Standalone.
