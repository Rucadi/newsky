# Immersive Equipment Displays - unequip when nude
- Author: zhongjiedong
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/71625


Immersive Equipment Display preset intended for Ostim/Sexlab/Flowergirls but can be used by anyone really.
  
  
When no chest armour is equipped, the displays will also be disabled.
  
  
Also disables the displays for NPCs otherwise when you give your follower items to carry they display all of them.
  
  
Includes a node preset for a modified Chesko quiver placement also
