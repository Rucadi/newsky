# Weapon Animation (Miraak's Sword)
- Author: saltmaxwellpshk
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/52390
Add an animation to Miraak's Sword.
  
Works with anime events.
  
No plugins or scripts are used.
  
Only the mesh is changed.
  
  
anything that changes the mesh of the weapon is incompatible.
  
Retexture mods are compatible.
  
  
Works with Weapon Draw and Weapon Sheathe
