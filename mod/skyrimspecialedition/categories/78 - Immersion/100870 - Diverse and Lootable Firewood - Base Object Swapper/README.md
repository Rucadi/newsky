ABOUT THE MOD:
  
  
Skyrim's a cold place, isn't it? But everyone just seems to hoard their firewood. The piles you see around are ALWAYS the same. This mod aims to change that. I've created over forty different firewood stacks that you'll now encounter on your travels. The piles change on save loads, and so if you visit a location again, the firewood stack will have depleted further or been re-stocked as if it's actually being used. It's a small thing, but I think it adds a lot of immersion. 
  
  
The mod covers variants of all the firewood stacks (small, medium, large, and huge), excluding the sparse stacks. It covers piles with and without snow. For a better idea of what the mod does, take a look at the gallery.
  
  
This mod wouldn't be possible without the magic that is [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) by [powerofthree](https://www.nexusmods.com/skyrimspecialedition/users/2148728). Please make sure you have the latest version installed, you've endorsed the mod, and you've given [powerofthree](https://www.nexusmods.com/skyrimspecialedition/users/2148728) some well-deserved kudos.
  
  
For the harvestable firewood piles, you'll need [Dynamic Things Alternative - Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60741). Please make sure you have the latest version and you've endorsed wSkeever's excellent mod.
  
  
INSTALLATION:
  
  
1). [**High Poly Project**](https://www.nexusmods.com/skyrimspecialedition/mods/12029) is a requirement to use this mod. Without installing the HPP first, you won't have any textures for your firewood piles. My mod only contains meshes and the files required by BOS. Install  [**High Poly Project**](https://www.nexusmods.com/skyrimspecialedition/mods/12029) (you can pick any texture size) first. Endorse that mod (it's amazing!) and give [LucidAPs](https://www.nexusmods.com/users/3180451) some kudos.
  
  
2). Install my mod. It contains an esl flagged esp for the firewood pile records. A BOS SWAP file to initiate the swaps, and the meshes required for the different firewood piles.
  
  
3). Enjoy!
