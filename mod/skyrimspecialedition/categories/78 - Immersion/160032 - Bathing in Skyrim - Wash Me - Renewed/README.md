[left][/left][left][font=Arial][size=5]⚠️How to change animation duration:
  
[/font][/left][/size]

  
1. [font=Arial]Go to MCM - «Bathing in Skyrim» - «NPC Animations»[/font]
[font=Arial]Set «Primary Animation Style» to - Vanilla[/font]
  
2. [font=Arial]Change «Loops when 'Ditry'» option:[/font]
     
     
   * [font=Arial]By default, this animation consists of two parts, each 5 seconds long. So if you set «Loops when 'Ditry'» to 2, each part will last 10 seconds; if 4, it will last 20 seconds; and so on.[/font]
[font=Arial]Reset «Primary Animation Style» to your configuration;[/font]
  
3. [font=Arial][font=Arial]Done =)[/font][/font]
