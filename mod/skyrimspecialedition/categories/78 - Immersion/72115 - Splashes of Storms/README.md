[/size]**Splashes**
  
  

  
* Splashes hit uncovered surfaces (ground, terrain, static objects) and characters
Splashes will only spawn in places visible to the player camera, saving FPS
  
**Ripples**
  
  

  
* Disables fake ripple texture on water
Adds real displacement based ripples to water
  

  
**﻿Configuration**
  
  

  
* Settings can be configured in *po3\_SplashesOfStorms.toml* (in Data/SKSE/Plugins).

  
* These can be reloaded and reapplied in-game by using the console command *splashes*

  

  
1. Alt-tab out of the game
Change settings and save config
  
2. Alt tab back into the game and type *splashes* into console
Profit
