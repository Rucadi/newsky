# Dynamic Torches - Base Object Swapper
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136409


This mod changes all permanent, non interact-able torches into interact-able ones using the power of base object swapper. With this mod, you can turn off the torches found inside a dungeon, so you can sneak better in the dark.
  
  
This mod is a remake of old mod from Skyrim LE, [Dynamic Torches](https://www.nexusmods.com/skyrim/mods/8070?tab=posts), which i found when i checked the new released mod ["LightsBane - Destroy Lanterns and Hide in Darkness"](https://www.nexusmods.com/skyrimspecialedition/mods/136263). With the recent release of light placer and placed light, it is possible to remake the original mod by using only BOS file configuration, since all lights from torches will also be turned off when the torch is taken.
  
  
**Installation:**
  
1. Install [placed light](https://www.nexusmods.com/skyrimspecialedition/mods/135488) and all of its requirements
  
2. Install [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)
  
3. Install this mod
  
  
Enjoy!
