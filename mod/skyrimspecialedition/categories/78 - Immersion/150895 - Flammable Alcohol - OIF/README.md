# Flammable Alcohol - OIF
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150895


This mod will make all alcohol bottles explode slightly when hit by fire enchanted weapon and fire spells too. Use them to your advantage when facing enemies. 
  
  
**Installation:**1. Install OIF
  
2. Install this mod
