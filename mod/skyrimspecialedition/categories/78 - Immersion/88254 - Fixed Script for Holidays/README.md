# Fixed Script for Holidays
- Author: Borgut1337
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/88254


Fixed Script for Holidays
  
  
Version 1.0.0
  
1 April 2023
  
Borgut1337
  
  

*Fixed a script in Isoku's "Holidays" mod, which could get into an infinite recursion when the player was in an inn with fewer than 7 beds during the Heart's Day holiday. Requires the original mod.*

  
  
Requirements
  
- [The original Holidays mod](https://www.nexusmods.com/skyrimspecialedition/mods/1533). Specifically: version 2.20A1.
  
- [powerofthree's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854)
  
  
  
Changes made by this patch
  
[color=#ffffff][size=3]- Fixed a script which could previously get stuck in infinite recursion.
  
  
[/size][/color]Installation
  
[color=#ffffff][size=3]- Same as any other standard mod, recommend installing using a Mod Manager.
  
- Note that this is not a full mod, just a patch.
  
- Should be safe to install mid-game.
