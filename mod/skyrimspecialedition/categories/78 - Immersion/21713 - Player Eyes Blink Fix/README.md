# Player Eyes Blink Fix
- Author: NasiGorengSpesial
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/21713


Enables blinking eyes for the player excluding helmets, skse plugin.
  
It's ON by default but you can toggle it by holding down the jump button for more than 2 seconds.
  
  
Requires Skyrim SE or AE and its equivalent SKSE64
  
Installation is pretty simple, either use a mod manager or manually copy the DLL file to Data\SKSE\Plugins\
  
For uninstallation, either use a mod manager or manually delete the DLL file
  
  
Credits for kapaer and the SKSE team
  
  
Source code is available on Github along with build instructions
