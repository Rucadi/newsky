- Author: CodeGrizzly

[left][font=Arial]**﻿Equipable Vampire Claws - CG's Version**[/font][/left]This mod is an extension of [AdoresUnicorns123](https://www.nexusmods.com/skyrimspecialedition/users/86825423)'s [Equipable Vampire Claws - Player Version](https://www.nexusmods.com/skyrimspecialedition/mods/96482) mod. I created this mod for personal use. Since there was some demand for it, I decided to share it. AdoresUnicorns123 was gracious enough to give me permission to upload this version of his mod. 
  
  
**Note: Due to script changes, please uninstall 1.0.0 before upgrading to the current version.**
  
**What's Different?**
  
**Version 1.1.0**
  

  
* [color=#f4f4f4][size=3]Added optional OAR animations.
  
Added keybind option to toggle claws.
  
- Added extra damage similar to a Khajiit's claws.
Added FOMOD
  
[/size][/color]
  
**Version 1.0.0**
  

  
* Uses slot 61 for the claws -- full first person race compatibility. No need for separate patches.
RaceMenu slider - Vampire Claws under the Body section.
  
* Only equipable via RaceMenu. No crafting, no claws in inventory.
Load Order -- as long as it is after RaceMenu, it doesn't matter where goes. 
  
* MCM -- provides an option to have the claws unequip when wearing gloves or gauntlets.
Fixed broken texture paths in meshes. 
  

  
**Requirements**
  

  
* [powerofthree's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854)
[Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)﻿- If using the animations.
  
* Either of AdoresUnicorns123's vampire claw mods for the textures. You can disable the esp from those mods if you don't need them. If you are using the Player Version, disabling the esp is recommended. 
    
  [Equipable Vampire Claws - Player Version](https://www.nexusmods.com/skyrimspecialedition/mods/96482)[Vampires have Claws](https://www.nexusmods.com/skyrimspecialedition/mods/87746)
