# Daedric Armor SE
- Author: Dremora Blacksmith
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/58661
With this modification, Daedric armor will become a little more like its counterpart from TES3.
  
I also changed the design of the sword, two-handed sword and dagger to better match the armor (optional).
  
  
  
С этой модификацией даэдрические доспехи станут немного больше походить на свой аналог из TES3
  
Также я решил изменить внешний вид меча, двуручного меча и кинжала, чтобы они лучше соответствовали доспехам (опционально).
