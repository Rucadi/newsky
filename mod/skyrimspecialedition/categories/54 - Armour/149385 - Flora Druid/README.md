# Flora Druid
- Author: Andrew21022
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/149385


This mod adds a brand-new, lore-friendly armor set From Diablo 4 to Skyrim: the Flora Druid Armor. Designed to blend naturally into the world of Tamriel, this set is ideal for characters with a mystical or nature-bound background — especially frost mages, druids, and shamans. Chest with armor located in *Dragonsreach*
  
  
Features:
  
 •  Fully standalone armor set
  
 •  Summonable ent-protector
  
 •  Male and female support
  
 •  High-quality textures
  
 •  Temperable for improved stats
  
 •  Available through console or AddItemMenu as well
  
  
Installation:
  
Use your preferred mod manager, or install manually by placing the contents into your Data folder.
  
  
Compatibility:
  
No known conflicts. The armor is standalone and doesn’t overwrite any vanilla assets.
