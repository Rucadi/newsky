# Morokei Clothing Fix With Shout Cooldown For SE
- Author: Chinocchi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/114760


If this mod ever gets bricked you can edit the file yourself in Skyrim Creation Kit (SE) as follows:
  
![](https://i.imgur.com/PtpojTd.png)
  
  
If the image above doesn't work anymore:
  
  
1. In the Creation Kit open 'Skyrim.esm'.
  
2. Go to Armor>DragonPriestHelms.
  
3. Select 'ArmorDragonPriestMaskMoonstoneHelmet'.
  
4. Change these values:
  
  
Enchanting: EnchDragonPriestMaskMoonstoneÂ 
  
Weight: Light>None (Inventory purposes only)
  
Armor Rating: 0.00
  
Keywords: [remove] ArmorHelmet; ArmorLight; VendorItemArmor [add] ArmorClothing; ClothingHead; VendorItemClothing
  
  
  
After making the edit:
  
1. Click 'Ok'.
  
2. In the top right select 'File'.
  
3. Click 'Save'.
  
4. In the Windows pop-up window name your .esp file however you like. (You can only save .esp files in the 'Data' location of the game folder.)
