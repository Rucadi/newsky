**![](https://staticdelivery.nexusmods.com/mods/110/images/headers/120163_1748611698.jpg)﻿****FROSTWARDEN REGALIA****My patches by Xtudo**

**Description**
  
**My patches for the amazing mod made by Ashtoreth.
  
  
Original ESP Fixes**

  
* Added clothing version.
Proper 1st person models.
  
* Fixed inventory items.
Optimized 4K textures (original size).
  
* Some other tweaks and fixes.
It's fine to update mid-game. It will replace the original**[Ashtoreth] Frostwardens Regalia.esp** file.
  

  
**My version**
  

  
* **Non-HDT**, **High Heels** and **Non High Heels**.
**HDT-SMP** version available. No BodySlides.
  
* **Trousers version** available. Install only one of the Main files.
**It's fine to switch between My Versions mid-game.**
  
* 4 colors available in-game.
Vambraces are a separated item now.
  
* Added stockings to the main outfit.
Proper 1st person models.
  
* Improved neck area for better beast races support.
Optimized 4K textures (original size).
  
* It should fine to update mid-game. You will need to craft the Vambraces. It will replace the original **[Ashtoreth] Frostwardens Regalia.esp** file.

  
**[color=#cfe2f3]Optional files**[/color]

  
* **Optimized 4K-2K-1K textures** (original are 4K). 4K is already included in Fixes and My Version.
**Color patches**. They will replace the default blue color. Not required for My Version files.
  
* **Black Fur patch**. It will work with default blue and any color replacer.
**Non High Heels Boots**. Standalone version.
  
* **Jarl Elisif the Fair Redwort Outfit replacer**. Non High Heels and High Heels versions available, install only one. Requires one of the main My Version files installed first (but only their Meshes and Textures folders). Fully compatible with any Elisif's face replacer mods.
