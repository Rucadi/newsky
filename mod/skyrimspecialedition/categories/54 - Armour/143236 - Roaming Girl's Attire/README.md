# Roaming Girl's Attire
- Author: Suna203
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/143236


[font=Verdana]**Description:**
  
This mod adds a light armor set for female characters. The outfit is designed to evoke the image of a free-spirited, fearless girl roaming the world.
  
  
The set includes the following items:
  
[/font]

  
* [font=Verdana]Roaming Girl's Outfit (an all-in-one body piece featuring coat, shirt, shorts, and bag)[/font]
[font=Verdana]Roaming Girl's Gloves[/font]
  
* [font=Verdana]Roaming Girl's Boots (Uses the High Heels system, requiring [RaceMenu](https://www.nexusmods.com/skyrimspecialedition/mods/19080)﻿)[/font]
[font=Verdana]Roaming Girl's Choker[/font]
  

[font=Verdana]
  
**Supported Body Types:**
  
[/font]

  
* [font=Verdana]3BA (main body type)[/font]
[font=Verdana]CBBE (patch available in options)[/font]
  
* [font=Verdana]BodySlide (compatible with both 3BA and CBBE)[/font]

[font=Verdana]
  
**How to Obtain:**
  
The armor set can be crafted at any forge and can be upgraded at workbenches.
  
  
**Plugin Information:**
  
ESPFE format
