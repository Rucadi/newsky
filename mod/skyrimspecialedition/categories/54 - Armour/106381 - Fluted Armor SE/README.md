# Fluted Armor SE
- Author: Umbasa
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106381


Эта модификация добавит в вашу игру комплект тяжелых рифлёных доспехов.
  
Отпрыск одной из знатных семей Вэйреста нашел свой конец в заснеженных землях Белого Берега.
  
  
  
This modification will add a set of heavy fluted armor to your game.
  
The scion of one of the noble families of Wayrest found his end in the snowy lands of the Pale.
  
  
\*beast races are not supported
