# Armors of the Velothi Pt. II
- Author: Pulcharm
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/69996


This mod adds 8 dunmer themed sets and 7 weapons

**Requirements:**
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿
  
ENB with complex particle lights support (optional, for emission effects)
  
  
**Installation Guide:** Please read if you encounter a bug causing physics to stretch down
  
SpoilerStep 1: Download and install [SKSE](https://skse.silverlock.org/)
  
Step 2: Download and install [FSMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339?tab=files), pick the version according to your Skyrim version, XP32 and FSMPV are not required for this mod to work, you don't have to install them
  
**If you are experiencing the cloth stretching downwards bug then you most likely have installed FSMP or SKSE incorrectly**
  
Step 3: Download and install this mod
  
**Integration (v1.1):**
  
Can be crafted at the forge:
  
Steel Smithing required for Camonna Tong, Hearthguard, Dres Saltplains sets
  
Elven Smithing required for Morag Tong Seeker, Morag Tong Grandmaster, Ash Reader, Travelling Merchant, Ashlander Warrior sets 
  
Ebony Smithing required for Bonesaint set
  
**Optional World Integration (v1.1):**
  
Adds Redoran Hearthguard set to Captain Veleth, distributes Camonna Tong set to bandits at the Trader's Post, distributes Morag Tong set to Morag Tong enemies on Solstheim.
  
Ash Reader set can be acquired during the quest Distant Memories.
  
New Morag Tong Grandmaster boss encounter in Ashfallow Citadel, new Bonesaint boss encounter near Refugee's Rest.
