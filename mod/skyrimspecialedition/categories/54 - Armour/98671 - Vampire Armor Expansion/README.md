# Vampire Armor Expansion
- Author: Hornclonic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/98671


**DESCRIPTION**
  
  
This mod adds 8 new armor meshes all carefully crafted from existing vanilla assets,
  
equally divided between male and female characters, adding much needed diversity to your vampire characters.
  
  
There are a few ways to obtain these items in game:
  
  

  
* All Items can be crafted and require Ebony Smithing perk.
Through the optional levelled list plugin that adds the items to the vampire levelled list, used by enemies level 25+.
  
* Through the optional SPID file, that distributes the armors to unamed vampire npcs above level 25.
Alternatively you can use AdditemMenu or console commands to add the items to your character.
  

  
  
**COMPATIBILITY**
  
  
Bodyslide files are provided, feel free to build to fit custom bodies.
  
Prebuilt meshes are provided for HIMBO and CBBE Curvy.
  
My esp is compatible with everything and the load order doesn't matter.
