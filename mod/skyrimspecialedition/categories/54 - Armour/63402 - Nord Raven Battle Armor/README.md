# Nord Raven Battle Armor
- Author: Max
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63402


This mod adds lighghtweight and lore-friendly leather armor, which can be created at the forge from leather materials. The set includes a male and female (UNP) light armor, hood and additional lowered hood. The characteristics of the armor are very similar to the Scaled Armor Set.
  
  
All my mods ported to Skyrim LE by [jb93lonewolf](https://www.nexusmods.com/users/45716587?tab=user+files) , Xbox One verions ported by [basspainter](https://bethesda.net/en/mods/skyrim?author_username=basspainter).

Installation
  
  
Copy the contents of the archive folder to your Skyrim\Data folder. Activate
  
the .esp file in the Skyrim Launcher.  Alternatively, use your mod manager of choice to install the mod.
