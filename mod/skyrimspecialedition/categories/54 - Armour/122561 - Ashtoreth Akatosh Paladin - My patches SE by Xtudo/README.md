**Description**
  
**My patches and tweaks for the amazing mod made by Ashtoreth.**
  

  
* **Torn version added.** **Please rebuild your main armor BodySlide files again, so the torn version works as intended.**
Proper keywords for the Survival Mode added.
  
* 4K, 2K and 1K optimized textures versions available. The original mod is 4K.
New ground items.
  
* Improved crafting recipes.
Minor fixes here and there.
  
* It's fine to install mid-game, you won't loose any already crafted item.
It will replace the original "[Ashtoreth] Akatosh Paladin.esp" file.
  
* **Optional Non-HDT 3BA, CBBE-UNP versions**. Requires one of the Main files installed first.
