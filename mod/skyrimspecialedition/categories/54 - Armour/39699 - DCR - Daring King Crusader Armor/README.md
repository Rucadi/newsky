This is an alternative design of the original [DCR - King Crusader Armor](https://www.nexusmods.com/skyrimspecialedition/mods/8856) for the female body. [Angilla](https://www.nexusmods.com/skyrimspecialedition/users/1566738), the author of the original gave permission for remakes as long as authors make it awesome and do their best. This is not as awesome as the original but it is my best attempt!

How to obtain:[/size]
  
[/size][/font][font=Georgia][size=3][font=Georgia][size=4][size=3]Look for two chests in [/size][/font]Bloated Mans Grotto (South West of Whiterun) near the Talos statue half
  
 way through.[/size]
  
  
The armor is craftable under the "Ebony" tree. *Extracted* *Sanctified Light* is found at the smelter.
  
  
  
Installation:
  
- Install the original [font=Georgia][DCR - King Crusader Armor](https://www.nexusmods.com/skyrimspecialedition/mods/8856)[/font] (required).
  
- Install this mod and place it below the original in the load order (or overwrite)
  
- Build meshes in your preferred body in BodySlide if desired (remember the Zap-slider).
