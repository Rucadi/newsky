# Armor Variants Expansion
- Author: Darkgondul
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/34100


![](https://i.imgur.com/lPapjUC.jpg)
  
  
[font=Comic Sans MS][font=Times New Roman]**Description:**[/font][/font]
  
  
  
Armor Variants Expansion aims to adjust and increase variety of armors on Skyrim, by adding new variants of vanilla armors to the leveled lists trying to be extremely lore friendly. **This is not a[size=4][b][u] model**  replacer mod[/u][/b], but it enhances variety of the armors found in Skyrim.
  
  
All variants include models for both genders & all races, enchantments, crafting and upgrade recipes, world locations and leveled list entries so NPCs will have a good chance of using these.
  
  
  
More in details; at the moment following new variants are complete and integrated for:[/size]
  
  
-Iron (helmet, cuirass, a heavy tower shield and a wooden one in light category for the hide variant).
  
  
-Steel (two horned helmets, one simple, two armors with one new padded variant and a rounded shield).
  
  
-Dwemer (helmet and cuirass).
  
  
-Leather (helmet and cuirass).
  
**Version 1.5:**
  
  
-Hide (helmet and two cuirasses).
  
  
-Elven (one winged helmet with the same values of the gilded one, and one gilded cuirass).
  
  
-Orcish (one crested helmet for the moment).
  
  
-Studded (one cuirass).
  
  
[size=3]**Version 2.0:**
  
  
-Ebony (helmet, cuirass and a tower shield).
  
  
-Dragonscale (closed helmet and cuirass).
  
  
-Dragonbone (helmet and cuirass).
  
  
-Daedric (helmet, cuirass and a tower shield).
  
[/size]
  
  
Plus it adds two new almost endgame sets used by common high level hostile NPCs such as bandits. There stats are comparable to the Nordic Carved set and the Glass set:
  
  
-Improved Steel Plate set (one helmet, one cuirass and gauntlets). This set is equiparable/similar to the Ebony set, so it will be craftable and upgradable with the Ebony Smithing perk.
  
  
-Improved Scaled set (helmet and cuirass). This set is equiparable/similar to the Glass set, so it will be craftable and upgradable with the Glass Smithing perk.
  
[color=rgb(97, 97, 97)]
  
[color=#ffffff]In addition with the 1.5 release there is a new middle armor rate set, the Nordic Vanguard set, comparable in a level between the regular Scale set and the Elven Gilded set.
  
It's an alternative for those who don't like the often seen elven set.[/color]
  
  
-Nordic Vanguard set (helmet, cuirass and shield). As said before, it could be an alternative for the player who don't like the overuse of the elven set; craftable and upgradable with Elven Smithing perk.
  
It's uncommon, so don't expect to see it around the world, apart from some sporadic hostile npc.
  
[/color]
