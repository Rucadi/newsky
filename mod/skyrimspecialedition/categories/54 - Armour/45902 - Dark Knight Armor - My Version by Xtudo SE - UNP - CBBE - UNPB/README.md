**Dark Knight Armor
  
My Final Version by Xtudo

Description**
  
**My Final Version of the Dark Knight armor.**
  
  
It includes:
  

  
* **Fully standalone (36 items).**
**Male and female armors.**
  
* **UNP, UNPB and CBBE.**
**No high heels.**
  
* **Armored boots.**
**Metallic, Ebony and Crimson cubemaps.**
  
* **Three armor types: [b]No Trousers, Trousers, and Reinforced Trousers.** [/b]
**The Boots fix.**
  
* **2K, 4K and 8K.**
**The Dark Knight Shield.**
  
* **Auriel's Shield, Shield of Ysgramor and Spellbreaker replacers (smaller versions available).**
**Beautiful swords, one hand and two hands.**
  
* **BSA - ESL.**
**[3BA - BHUNP - CBBE - HIMBO - UUNP bodyslide conversion](https://www.nexusmods.com/skyrimspecialedition/mods/56914) by Nekomata17 here.**
  
* **Red fabric color version.**
[Cloak Addon](https://staticdelivery.nexusmods.com/mods/110/images/106691/106691-1738954758-1751307563.jpeg)﻿. **Default** and **Red Fabric**, **HDT-SMP** and **Non-HDT** versions available. Intended for the female version of the armor. An amazing **Dark Knight Legendary Cloak**is buried [somewhere southeast of Windhelm](https://staticdelivery.nexusmods.com/mods/110/images/106691/106691-1738954961-679287795.jpeg)﻿, happy adventure! Requires [Nirn Necessities - SMP Accessories](https://www.nexusmods.com/skyrimspecialedition/mods/112481) ﻿installed first.
  
* **Optional [Celestine](https://www.nexusmods.com/skyrimspecialedition/mods/138611)﻿ SPID patches**. Two versions available. **Celestine Only** and **Celestine and Hela** versions. Celestine will wear the Metallic version, Hela the Ebony one, with helmet. Install only one. Requires one of the Main files installed first, but only their meshes and textures.

  
You can craft all items at any forge, and you can temper them too.
  
The mod is fully standalone.
  
  
**I made it for my game and now I'm sharing it, I hope you like it.**
