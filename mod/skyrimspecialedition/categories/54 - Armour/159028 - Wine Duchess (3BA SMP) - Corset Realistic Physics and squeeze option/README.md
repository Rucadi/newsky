# Wine Duchess (3BA SMP) - Corset Realistic Physics and squeeze option
- Author: SmolSaltLemon for original mod edits by kahvipannu84
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/159028


**[SmolSaltLemon](https://www.nexusmods.com/skyrimspecialedition/users/202025696) beautiful base mod [Wine Duchess (3BA SMP)](https://www.nexusmods.com/skyrimspecialedition/mods/158632?tab=description) hard requirement.**Preview gifs are with *smp* on.(gif below might take a bit to load)**![](https://i.postimg.cc/2YGMMF33/The-Elder-Scrolls-VSkyrim-Special-Edition-2025-09-12-20-52-ezgif-com-optimize.gif)
  
  
Please read, just a few words.**This adds realistic physics (bone weights) to the corset, and squeeze sliders.
  
  
**This does not overwrite any files of the original, this adds new alternative Bodyslide file for the corset**, that you can build.
  
  
**Wine Duchess - Corset - kp84squeeze:**
  
  
- New Bone weighting for corset and breasts
  
- Removed belly bone weighting from corset
  
- Two new sliders for optional squeeze
  
- Still has all of the zaps and sliders of the original

**Installation.**
  
  
1. (Optional) adjust the squeeze sliders for both 0 and 100 weight (you can go above value 100 by typing it in) to fit your body preset.
  
  
2. Build in Bodyslide.
  
  
Done.
