Please use a mod manager, this mod comes with a FOMOD, meaning it has options you must select.
  

  
1. Download this mod via mod manager (recommended)
Select the options you want
  
2. Open bodyslide and select the correct group starting with [ELLE] or UBE [ELLE] or search ELLE in the top right filtering
Batch build or [build the outfit](https://www.youtube.com/watch?v=45E8qqnqJDQ&ab_channel=PorterhauseModReviews)﻿ to your bodyslide preset.
  
3. Confirm it has built the meshes correctly to data\meshes\armor\[ELLE] Mod Name. ) SpoilerIf there are none there, and your meshes are invisible, check bodyslide or your mod organiser settings to see where they are being built to.
     
     
   This is a common issue with new users or those using a modlist.
     
   ![](https://i.imgur.com/Q6X87am.png)
     
   Once you've found the \_0 and \_1 meshes similar to above, simply drag and drop into the correct folder.
     
   (Data\meshes\armor\[ELLE] Mod Name. NOT the GND folder)
Enable the esp and enjoy!
  

  
* You can spawn it via additem menu, or in console using Help Sena or Help 00EL.
    
  Otherwise, craft it at a forge once you've picked up the crafting book and have the correct smithing perk.
