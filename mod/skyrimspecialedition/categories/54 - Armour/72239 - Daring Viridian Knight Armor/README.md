# Daring Viridian Knight Armor
- Author: Nzymia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72239


[font=Georgia]This variation of the original [Viridian Knight Armor](https://www.nexusmods.com/skyrimspecialedition/mods/71568) by [Kreiste](https://www.nexusmods.com/skyrimspecialedition/users/5321695) and [wSkeever](https://www.nexusmods.com/skyrimspecialedition/users/7064860) presents this amazing outfit in a more daring design.[/font][font=Georgia]

[font=Georgia]
  
For detailed information regarding the armor please see the original mod page.
  
  
  
The original [Viridian Knight Armor](https://www.nexusmods.com/skyrimspecialedition/mods/71568) is required for this mod to work.
  
This mod includes only BodySlide files and a set of pre-built meshes in CBBE 3BA curvy or BHUNP UNPB with zeroed sliders.
  
If you wish to have a different body shape, build your meshes in BodySlide.
