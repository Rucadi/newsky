# HDT-SMP Vanilla Ebony Armor and Ebony Mail Patches
- Author: tatururutia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/84159
**This patch adds cloth physics to the Vanilla Ebony Armor and Ebony Mail.
  
Only meshes replacement, support any retexture.**
  
  
**Require HDT-SMP (Skinned Mesh Physics) OR FSMP - Faster HDT-SMP.**
  
  
Support weight 0 to weight 100 vanilla bodyshape change.
  
If you change the body weight, and the size of armor will also change.
