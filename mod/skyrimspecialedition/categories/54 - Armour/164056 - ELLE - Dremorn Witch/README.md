Please use a mod manager.
  

  
1. Download this mod via mod manager (recommended)
Open bodyslide and select the correct group starting with [ELLE] or UBE [ELLE] or search ELLE in the top right filtering
  
2. Batch build or [build the outfit](https://www.youtube.com/watch?v=45E8qqnqJDQ&ab_channel=PorterhauseModReviews)﻿ to your bodyslide preset.
Confirm it has built the meshes correctly to data\meshes\armor\[ELLE] Mod Name. ) SpoilerIf there are none there, and your meshes are invisible, check bodyslide or your mod organiser settings to see where they are being built to.
  
This is a common issue with new users or those using a modlist.
  
  
![](https://i.imgur.com/Q6X87am.png)
  
  
Once you've found the \_0 and \_1 meshes similar to above, simply drag and drop into the correct folder.
  
(Data\meshes\armor\[ELLE] Mod Name. NOT the GND folder)﻿ 
  
  
3. Enable the esp and enjoy!
     
     
   * You can spawn it via additem menu, or in console using Help Dremorn Witch or Help 00EL.
       
     Otherwise, craft it at a tanning rack once you've picked up the crafting book.
