**Update 1.3 installation:** Just make sure the .esp and the .bsa are directly in the data folder.
  

Installation is very simple. Just make sure that the .esp file in inside the Data folder. This Meshes folder should be merged with the Data meshes folder and this texture folder should be merged with the Data texture folder. Path for Meshes: Skyrim/Data/Meshes/Armor/BosmerArmorMATY743/(All mesh files and some folders). Path for Textures: Skyrim/Data/Textures/Armor/BosmerArmor/(All texture files).
  
Note. Some people are having the bug where the folders a extracted twice inside each other. So instead of Skyrim/Data/Meshes/Bosmer/[files here] the folders are being extracted as Skyrim/Data/Meshes/Bosmer/Data/Meshes/Bosmer/[files here]. This will cause the items to be invisible in game. The fix is just drag or cut the last deepest meshes folder and past it inside the data folder. Agree to merge and it should work. This task needs to be preformed on both the texure and mesh folders. I might change the file format for next update so that it use .bsa files. Hopefully that will solve the issue
