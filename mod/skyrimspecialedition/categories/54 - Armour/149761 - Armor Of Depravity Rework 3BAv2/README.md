# Armor Of Depravity Rework 3BAv2
- Author: Leocid2
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/149761


Rework of Armor Of Depravity of Leocid2 from Loverslab ( author is long gone don't even know if he still online or not ) with some fixes and add 1 2 things
  
  
**Change compare to original mod** :
  

  
* Convert to SSE 3BAv2 and fix clipping
Fixed the bone weighting, added heels offset to boots, and made some changes to the body slots so includes the esp
  
* Add missing specular textures
Add a copy of armor as pauldrons slot 57 ( for more combination with other armor and outfit if you like combine )
  
* Already ESL so you don't need to do it

  
**Installation :** 
  
Install normally with MO2 or any of your mod organizer
  
  
**Requirement :**
  

  
* XP32 Maximum Skeleton Extended - XPMSE
CBBE 3BA
  
* Faster SMP SE
BodySlide & Outfit Studio 
  
* Backported Extended ESL Support

  
**Some small issues :** 
  

  
* Boots part kinda janky it hard to fix and hide all the clipping
Specular map on kneelpad is handmade try to fix missing textures so it could look weird sometime
