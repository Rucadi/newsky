# Sturdy Sandals SE
- Author: tiwa44 xj3st3r kroenel
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/142566


![](https://staticdelivery.nexusmods.com/mods/1704/images/162063/162063-1760870120-1141173962.jpg)
  
  
  
  
This is based on the Oldrim mods ATS Sandals (All Terrain) by xj3st3r and UNP Minidresses Collection by tiwa44. I compiled a new mod featuring
  
  

  
* meshes [size=3]overhauled in Blender and Outfit Studio,[/size]
1K textures in brown and black,
  
* male versions,
versions with heels,
  
* versions with long leg straps,
new wedges variant (1.7),
  
* working and decorative ground models,
CBBE, 3BA, BHUNP, UUNP, TNG, SOS, HIMBO and Vanilla conversions,
  
* body slides for ~~most~~ all of those,
zaps for upper/thigh straps,
  
* slider to make wedge heels,
optional clothing variants for mages,
  
* ESP flagged as ESL.

  
  
Due to some bikini armor magic, these sandals are protective and count as light armor. If needed you can choose additional clothing versions during the installation process. The sandals can be crafted on a tanning rack and improved on a workbench. They are enchantable like other footwear.
  
  
  
  
**[font=Comic Sans MS]Installation:[/font]**
  
  
In the FOMOD installer you can choose between the female body types CBBE, 3BA, BHUNP, UUNP and the male body types Better Males, SOS, HIMBO and Vanilla. Pre-built meshes with standard presets will be installed by default. If you use custom presets or need zeroed sliders you'll have to fire up [BodySlide](https://www.nexusmods.com/skyrimspecialedition/mods/201) and build new meshes. With BodySlide you can also zap upper or thigh straps and create wedge heels as can be seen in the user screenshot by triptherift.
  
  
For the Better Males option your mod list should include one of those many mods providing textures for improved male body meshes like Better Males, Total Character Makeover, Tempered Skins for Males, Vitruvia[size=3],Â The New Gentleman (TNG) etc. If you use the old meshes of TNG (high and low poly) choose SOS sandals in the installation menu.
  
  
If you need unarmored clothing versions for your mage character you can choose an alternative plugin in the second step of the FOMOD installation progress. It will give you the ability to craft both types, light armor and clothing.
  
  
To make heels work correctly the installation of [RaceMenu High Heels](https://www.nexusmods.com/skyrimspecialedition/mods/18045) or [Heels Fix](https://www.nexusmods.com/skyrimspecialedition/mods/64442) is required.
