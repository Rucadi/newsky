A new craftable scaled light armor found under Steel category.
  
  
Available for UNP.
  
  
new features:
  
  
- Added new boots.
  
- All textures got improvements but are still 2K.
  
- Corrected clippings around the armpits.
  
- Now the breasts are more covered in larger body sizes.
  
- Removed the box from Whiterun.
  
- Added crafting and upgrading for all items.
  
- Added crafting manual.
