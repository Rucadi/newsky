# Kvetchi Mercenary Set
- Author: Pulcharmsolis
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79226


This mod adds an imperial themed set, a weapon & a shield

**Requirements:**
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿ or [HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/30872) (If you have Nemesis neither FNIS nor XP32 are required)
  
  
**Integration:**
  
Found on an NPC in a [bandit camp near Helgen](https://elderscrolls.fandom.com/wiki/Bandit_Camp:_Skybound_Underhang), can be crafted and tempered with same requirements as vanilla steel plate, the loading screen only plays in Falkreath Hold
