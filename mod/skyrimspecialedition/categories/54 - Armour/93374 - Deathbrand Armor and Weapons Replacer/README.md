# Deathbrand Armor and Weapons Replacer
- Author: MANHUNTER69
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/93374


Mesh and texture replacer for the Deathbrand armor set for vanilla bodies, with unique models for Bloodscythe and Soulrender. Haknir Death-Brand will wear the outfit too. Esl flagged plugin. Will use your installed Stalhrim armor textures, with optional meshes for compatibility with [Stalhrim Refrozen](https://www.nexusmods.com/skyrimspecialedition/mods/41348)[Updated beast race support here](https://www.nexusmods.com/skyrimspecialedition/mods/116527)
