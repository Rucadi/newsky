# Wine Duchess (3BA SMP)
- Author: SmolSaltLemon
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/158632


Adds new armor and cloth 
  
  
  
  
-
  
  
Contains 9 part
  
  
Corset
  
Gloves
  
Nails
  
Heels
  
Pants
  
Skirt
  
Belts
  
Neck Corset
  
Cloak
  
  
-
  
  
Models can be customized by zap sliders in BodySlide
  
  
-
  
  
How to obtain
  
  
On tanning rack just craft book then all parts will appear in a forge
  
  
or
  
  
just use AddItemMenu
  
  
-
  
  
For beastfolk support use sliders in bodyslide
  
  
-
  
  
The clothing version has the keyword ClothingRich, and is compatible with mods that allow you to show off this set at Elenwen's reception.
  
  
-
  
  
All .esp flagged ﻿as .esl
