# Bonemold Armor Replacer (SMP)
- Author: Xeraphime
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128811
**Bonemold Armor Replacer** replaces the vanilla Bonemold armor, based on the original *Armors of the Velothi* mod by Pulcharmsolis.
  
I've made the following adjustments:
  
[\*]Improved cloth physics
  
[\*]Edited meshes
  
[\*]Updated textures
  
[\*]Added new file paths for meshes and textures to ensure compatibility with the original mod.
