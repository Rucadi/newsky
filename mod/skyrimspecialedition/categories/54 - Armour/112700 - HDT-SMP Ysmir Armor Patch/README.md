This MOD added hdt-SMP cloth physics to the Ysmir Armor.
  
Require HDT-SMP (Skinned Mesh Physics) OR FSMP - Faster HDT-SMP.
  
Also require original mod.
  
Then use this patch to cover it.
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/112700/112700-1709044808-1936687767.gif)
  
  
  
**[Version .1]﻿ HDT-SMP Ysmir Armor Patch**
  
Includes only the most basic body meshes files and physical xml file.
  
This version is available for the original game players who only need a little physics effect.
  
  
  
**[Version .2]﻿****HDT-SMP Ysmir Armor Patch (Fix)**
  
﻿Includes some meshes files(fix some meshes name issues) and esp file(flagged as esl, compatible with old version of Skyrim).
  
This version is available for the 1.5.97 Skyrim players who only need a little physics effect.
  
  
  
**﻿[Version .3]﻿****HDT-SMP Ysmir Armor Patch (for CBBE 3BA)**﻿Includes some meshes files(fix some meshes name issues) and esp file(flagged as esl, it is necessary, because I changed the female armor addon for weightslide, compatible with old version of Skyrim) and CBBE 3BA bodyslider.
  
This version is available for the 1.5.97 Skyrim players who need to change their bodyshape for CBBE 3BA.
  
  
***Choose one of these three versions, don't download them all.***
