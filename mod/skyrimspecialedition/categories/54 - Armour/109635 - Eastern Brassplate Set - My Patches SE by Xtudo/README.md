**EASTERN BRASSPLATE**

**My patches by Xtudo**

**Description**
  
**My patches for this amazing mod.**
  
  
**My Fixes and Tweaks**
  

  
* Added "Warm" keyword for the Survival mode.
Added some races missing in the Armor Addon records.
  
* Added missing ArmorMaterialDwarven keyword to the Gauntlets.
ESL file that works with any Skyrim version. It will replace the original EasternBrassplateSet.esl.
  
* It's fine to install/update mid-game.

  
**Optional files**
  

  
* The Forgotten City - **Immaculate Dwarven Armor** replacer.
Immersive Armors - **Dwarven Mage Armor** replacer.
  
* CC Alternative Armors - **Dwarven Mail** replacer.
CC Alternative Armors - **Dwarven Plate** replacer.
  
* **Vanilla Dwarven Armor** replacer. **Shield** included.
**Female Helmet** replacer. It will only replace the female helmet, the male will continue to be the original one. Default and Ancient Dwemer Metal versions available.
  
* **Solstheim Dwarven Armor** replacer. It will replace the default Dwarven Armor in the Solstheim's Heavy Armors Leveled Lists, and in some hand-placed items too. It can be used with another files without any problem. Use SSEEdit or Wrye Bash Patch to merge the edited leveled list edits, if you have conflicting mods. Carried on USSEP fixes.
**Non-HDT patch**. Only meshes.
  
* **2K-1K reduced VRAM usage textures (original are 4K)**. Only textures. It's fine to install/uninstall at any moment.

  
**I made it for my game and now I'm sharing it, I hope you like it.**
