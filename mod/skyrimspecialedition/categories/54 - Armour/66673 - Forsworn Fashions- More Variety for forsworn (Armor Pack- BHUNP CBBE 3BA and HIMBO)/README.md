# Forsworn Fashions- More Variety for forsworn (Armor Pack- BHUNP CBBE 3BA and HIMBO)
- Author: GolinskiPRH
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/66673


Forsworns have so much denominations, there is the ravager, the pillager, the looter, etc... and all of then have the same armor, this mod changes that.
  
This is an armor pack, featuring one armor set for each class of forsworn noted in [UESP](https://en.uesp.net/wiki/Skyrim:Forsworn#Forsworn_Types).
  
  
Forsworn Forager- Light Armor, ornated with lots of bones
  
Forsworn Looter- Light Armor, ornated mostly with feathers
  
Forsworn Pillager- Light Armor aimed for stealthy units.
  
Forsworn Ravager- Heavy Armor, made with steel plate and lots of fur.
  
  
Are those armors skimpy? yes, they are. Forsworns already wear skimpy armor in the base game, its part of their style. the armors are skimpy for both men and woman, but still keeps what need to be covered, well, covered.
  
  
Bodyslides for CBBE,CBBE 3BA and HIMBO are available.
