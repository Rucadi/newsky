# Onyx Knight Armor
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/95243


This mod adds a standalone 4 piece heavy armor set to Skyrim! Very good for a paladin playthrough.
  
  
You can craft the armor at any forge under Ebony.
  
  
The armor has stats the same as vanilla Ebony armor.
  
  
You can temper with the Ebony smithing perk.
  
  
Plugin marked as ESL.
  
  
Vanilla male weight slider supported!
  
  
Male Only.
