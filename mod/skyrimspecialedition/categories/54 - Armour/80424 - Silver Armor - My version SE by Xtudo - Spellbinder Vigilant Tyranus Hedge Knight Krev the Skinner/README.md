**Description**
  
**My personal version for this wonderful armor made by 4thUnknown.**It's a very improved version of my [Silver Armor - My patches and fixes](https://www.nexusmods.com/skyrimspecialedition/mods/79090) mod. I recommend you to update to this one, and it's fine to do it mid-game.
  
  

  
* **Heavy (Ebony tier) and Light (Glass tier) versions.**
**Weight sliders for all items.**
  
* **Full beast races support.**
**Seven colors available ingame.**
  
* **Now it uses its own cubemaps.**
**Added closed face helmets.**
  
* **CC Silver Armor and CC Imperial Legion replacers.**
**All items can be forged, Ebony perk required for Heavy version, Advanced Armors for Light.**
  
* **All items can be tempered and enchanted.**
**Several fixes.**
  
* **ESL.**

  
- [HDT patch by MinhazMurks here!﻿](https://www.nexusmods.com/skyrimspecialedition/mods/81351)
  
  
**Fixes**
  

  
* ESP file flagged as ESL.
Carried on the Unofficial Skyrim fixes.
  
* Added the "Warm" keyword to all the armor pieces, for the Survival mode.
Fixed the material type of all the items.
  
* Fixed some typos.
Fixed some weight-sliders values.
  
* Fixed the recipes of two items.
Fixes object bounds values of two items.
  
* Several clipping fixed.
Several texture artifacts fixed.
  
* Unweighted vertices fixed on the female armors.

  
**Optional downloads**
  

  
* **Optimized 2K textures**.
**Shinny Metal patch. Default and Alternative.**Only textures.**Default** will give a bluish cubemap to Silver armor, monochrome to Obsidian and golden to Gold. **Alternative** will give a bluish cubemap to Silver, Obsidian an Gold armors. Install only one.
  
* **AE-CC: Civil War Champions - Imperial Dragon Armor replacer.** Adjusted the cape to avoid clipping. ESL.
**AE-CC:** **Alternative Armors - Silver replacer****.** Only meshes.
  
* **Vigilant Tyranus outfit replacer.** Requires one of the Main Files installed first. He will wear the Umber, closed face helmet version. If you already meet him, maybe you'll need to do a "resetinventory" on him. ESL.
**High level player version.** Dragonplate and Dragonscale stats for the armors. You will need the Dragon Armor perk to forge it. ESL and ESP versions.
  
* **Immersive Armors - Hedge Knight Armor replacer**. Requires one of the Main Files installed first. Replaced by the Obsidian version. ESL.
**Immersive Armors - Spellbinder Armors replacer.** Requires one of the Main Files installed first. Replaced by the Redwort and Teal versions. ESL.
  
* **Krev the Skinner - Silver Hand Armor replacer.** Requires one of the Main Files installed first, only Meshes and Textures folders. Replaced by the Obsidian version. Krev will be leveled with the player (same as vanilla), but not the armor, so be prepared. ESL.
**Vanilla Silver Metal patch.**Only textures.
  
* **Equipment Toggle 2 patch.** Only meshes
**Feminine Silver Armor patch.** Only meshes. Based on the [Feminine Silver Armor SE](https://www.nexusmods.com/skyrimspecialedition/mods/79478)﻿ mod by [archlyde123](https://www.nexusmods.com/skyrimspecialedition/users/36138055). Requires the HDT-SMP patch.
