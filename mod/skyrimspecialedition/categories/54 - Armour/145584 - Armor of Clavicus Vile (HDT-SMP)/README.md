# Armor of Clavicus Vile (HDT-SMP)
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/145584


[Patreon](https://www.patreon.com/chakrasse)﻿      [Ko-Fi](https://ko-fi.com/chakrasse)﻿      [Discord](https://discord.gg/sMxZ6chT8c)
  
  
This mod adds a standalone heavy armor set, shield and sword inspired by Dark Fantasy.
  
  
The armor all share almost the exact same stats and recipes as vanilla Orcish armor, which means you craft them at a forge under the Orcish section! (Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Clavicus Vile's [Armor Piece]''. Weapon is ''Clavicus Vile's Sword''.
  
  
Males and Females have custom meshes. Textures are all 4K/ Resolution! 
  
  
There is 1 main file for this mod.
  
  
There are multiple versions of some armor pieces for visual differences.
