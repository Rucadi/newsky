[font=Times New Roman]
  
what is the composition of this armor?[/font]

  
* [font=Times New Roman]Ebony Ingot[/font]
[font=Times New Roman]Steel Ingot[/font]
  
* [font=Times New Roman]Leather[/font]
[font=Times New Roman]Leather Strip[/font]
  
* [font=Times New Roman]Dye (if you want to change the color)[/font]

[font=Times New Roman]
  
to make Perfected Armor, you need to make separate armors first, then combine them.
  
  
[/font][font=Times New Roman]to make Dye, you need Silver Ingot, and Garnet for red, Diamond for white, Sapphire for blue.
  
  
You can also temper this armor, you only need Steel Ingot and Leather
