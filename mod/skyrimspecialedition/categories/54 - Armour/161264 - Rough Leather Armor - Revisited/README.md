# Rough Leather Armor - Revisited
- Author: atomec
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/161264


The Rough Leather Armor is honestly one of my favorite armor mods for Skyrim. It fits perfectly with the game's aesthetics while still having a unique and realistic look to it.
  
  
This is my take of the mod, providing both the dark and the brown version in your game at the same time. The "Default Stats" version has the same stats as the original mod, providing armor rating between scaled and glass armors. I felt this was quite high for this armor, so the "Balanced Stats" version brings the armor rating down to scaled tier, except for the hood which provides less armor but only weighs one unit.
  
  
The only way to get the armor is to craft it with the Advanced Armors perk, as I didn't want to clutter any merchant loot tables. Maybe someone can make a SPID file for it.
  
  
[Originally](https://www.nexusmods.com/skyrim/mods/54672) made for LE by atomec, and [ported](https://www.nexusmods.com/skyrimspecialedition/mods/23725) to SE by bobbyp456. Please go endorse them if you enjoy this mod.
