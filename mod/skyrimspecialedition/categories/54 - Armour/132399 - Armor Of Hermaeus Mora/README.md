# Armor Of Hermaeus Mora
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/132399


[Patreon](https://www.patreon.com/chakrasse)﻿      [Ko-Fi](https://ko-fi.com/chakrasse)﻿      [Discord](https://discord.gg/sMxZ6chT8c)
  
  
This mod adds a standalone heavy armor set and greatsword inspired by Dark Fantasy.
  
  
The armor all share almost the exact same stats and recipes as vanilla Ebony armor, which means you craft them at a forge under the Ebony section! (Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Hermaeus' [Armor Piece]''. Weapon is ''Hermaeus' Greatsword''.
  
  
No HDT SMP on this mod, Males and Females have custom meshes. Textures are all 4K Resolution! **If anyone wants there to be a HDT SMP version, message me and we can discuss!**
  
  
There is 1 main file for this mod.
  
**There are multiple versions of some armor pieces for visual differences.**
