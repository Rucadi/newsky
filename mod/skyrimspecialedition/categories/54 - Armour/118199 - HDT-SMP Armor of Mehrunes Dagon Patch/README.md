# HDT-SMP Armor of Mehrunes Dagon Patch
- Author: ChakraSSE tatururutia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/118199


Added cloth physics to the skirt and cloak of the armor.
  
Require HDT-SMP (Skinned Mesh Physics) OR FSMP - Faster HDT-SMP.
  
Also require original mod.
  
Then use this patch to cover it.
