# Roaming Girl's Attire - 3BA Modularized
- Author: JAMIEDOW99
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/144038


Modularizes the torso armor from [Roaming Girl's Attire](https://www.nexusmods.com/skyrimspecialedition/mods/143236) by [Suna203](https://www.nexusmods.com/users/74489063).
  
  
 ・ **Body Slot 32**: Inner (Default color, Mottled, Fur, Patchwork, Ryza)
  
  
 ・ **Body Slot 48**: Outer (Default color, Mottled, Fur, Patchwork, Khaki Yellow)
  
  
 ・ **Body Slot 46**: Bag
  
  
 ・ **Body Slot 60**: Belt with zappable pouches
  
  
 ・ **Body Slot 33**: Gloves (Light Brown, Dark Brown, Beige)
  
  
 ・ **Body Slot 37**: Boots (Light Brown, Dark Brown, Beige)
  
  
 Also adds a BodySlide slider group containing all these 3BA slider sets. The name of the BodySlide slider group is "**Roaming Girl's Attire 3BA Modularized**".
  
  
 [The original mod](https://www.nexusmods.com/skyrimspecialedition/mods/143236) is required.
