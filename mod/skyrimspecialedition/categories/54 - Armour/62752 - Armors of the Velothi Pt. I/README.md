# Armors of the Velothi Pt. I
- Author: Pulcharmsolis - VishVadeva50
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/62752


This mod adds 8 dunmer themed sets

**Requirements:**
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿
  
  
**Installation Guide:** Please read if you encounter a bug causing physics to stretch down
  
SpoilerStep 1: Download and install [SKSE](https://skse.silverlock.org/)
  
Step 2: Download and install [FSMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339?tab=files), pick the version according to your Skyrim version, XP32 and FSMPV are not required for this mod to work, you don't have to install them
  
**If you are experiencing the cloth stretching downwards bug then you most likely have installed FSMP or SKSE incorrectly**
  
Step 3: Download and install this mod
  
**Integration (v1.3):**
  
Can be crafted at the forge:
  
Steel Smithing required for Bonepriest, Hlaalu Remnant, Narsis Watchman, Indoril Wayfarer, Redoran Exile sets
  
Elven Smithing required for Rhino Beetle, Ashlander Seer sets
  
**Optional World Integration (v1.3):**
  
Adds Redoran Watchman set to guards at Raven Rock and Captain Veleth, Redoran Exile set to Falas Selvayn who now also sells the rest of the armors
