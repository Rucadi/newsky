**Description**
  
**My tweaked version of the beautiful armor made by Kreiste and Mura.**
  

  
* 8 colors available in-game. Including the Optional Concept Art version.
"Warm" keyword for the Survival mode added
  
* No Smithing Perk required.
No meshes included, it should work with any body shape.
  
* ESPFE that works with any Skyrim version.
It's fine to update/install mid-game. You won't loose any item.
  
* 100% compatible with [Elmlock Armor - Color Variants](https://www.nexusmods.com/skyrimspecialedition/mods/107603).

  
**Optional files**
  

  
* **2K and 1K textures.** Reduced textures.
**Desaturated textures.** 4K, 2K and 1K options available.0
  
* **Smithing Ebony Perk required.** You'll need the Smithing Ebony Perk to forge the items. You can switch versions mid-game without any problem.
