# TES Arena Bikini - CBBE 3BA
- Author: Sandaron
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/98886


[left][/left]CBBE 3BA Bodyslide conversion of the LE mod TES Arena Bikini.
  
  
I'm aware there is already 3BA conversion uploaded here but I made my own before I knew that one was available and never got around to sharing my own. Though mine has proper UniBoob slider and physics weights support which according to user comments the other conversion lacks. Plus I added a couple of zaps and a breast squeeze slider.
  
  
Requires the [original mod](https://www.nexusmods.com/skyrim/mods/106393) and optionally the LE>SE converted plugin file from [UrbanNomadNome's mod](https://www.nexusmods.com/skyrimspecialedition/mods/66066?tab=description).
  
  
**Compatible and fully recommended to pair with** [**TES Arena Bikini Variants﻿**](https://www.nexusmods.com/skyrim/mods/111478)!
  
  
Enjoy :)﻿
