# Daring ESO Altmer Armor
- Author: Nzymia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/42412


[font=Georgia]This variation of the [ESO Altmer Armor](https://www.nexusmods.com/skyrimspecialedition/mods/14506), done with permission of the original author [NewerMind43](https://www.nexusmods.com/skyrimspecialedition/users/2959649), separates the armor into several different pieces and presents them in a slightly more daring design.[/font][font=Georgia]
  
  
  
[/font][font=Georgia]The outfit now includes the following pieces:
  
[/font]
  

  
* [font=Georgia]**ESO Altmer Armor**, redesignwith a shortened skirt[/font]
[font=Georgia]**[font=Georgia][size=4][b]ESO Altmer**[/font] Boots [/b]now with moderate high heels[/size][/font]
  
* [font=Georgia]**[font=Georgia][size=4][b]ESO Altmer**[/font] Gloves[/b], now combined with bracers[/size][/font]
[font=Georgia]**[font=Georgia][size=4][b]ESO Altmer**[/font] Leggings[/b] (original design)[/size][/font]
  
* [font=Georgia]**[font=Georgia][size=4][b]ESO Altmer**[/font] Bra[/b][/size][/font]
[font=Georgia]**[font=Georgia][size=4][b]ESO Altmer**[/font] Shoulders[/b][/size][/font]
  

  
  
  
[font=Georgia][size=5]**How to obtain:**
  
  
[/size][/font][font=Georgia][size=3][font=Georgia][size=4][size=3][size=4]Items can be crafted at the forge and requires the elven smithing perk. A chest containing the pieces can be found at the Lady Stone (on an island in Lake Ilinalta)[/size][/size][/font][/size]
  
  
**Installation:**
  
- Install with any mod manager or manually. Esp is in ESPFE format and will not count towards the plugin limit!
