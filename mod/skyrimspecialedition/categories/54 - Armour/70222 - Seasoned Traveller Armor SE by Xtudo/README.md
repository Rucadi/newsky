SEASONED TRAVELLER
  
By Xtudo**
  
**Description**
  
**A beautiful protective standalone armor and accessories for explorers and adventurers.**
  

  
* Male and female versions.
Weight sliders.
  
* Five colors available.
One handed and two handed swords.
  
* Backpacks.
A Pickaxe.
  
* A Walking Stick.
You can craft all the items at any forge, and you can temper them too.
  
* 4K and 2K versions.
No HDT required.
  
* HDT-SMP optional patch.
Full support for beast races.
  
* Survival "warm" keyword added.
Optional low level player version.
  
* Optional high level player versions.
BSA. ESL.
  

  
**Optional patches**
  

  
* **HDT-SMP version.** Requires [FSMP - Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)﻿ and one of the Main downloads installed first.
**Slightly brighter weapons patch.** If you find the weapons a bit dark, this could help. Only meshes and textures. It's fine to install/uninstall at any time.
  
* **Low Level Player version.** Same stats than the vanilla Studded and Leather armors.
**High Level Player versions.** Two options available : Armor and weapons **Glass tier**. Armor **Dragonplate** and weapons **Daedric tier**. Same perks, crafting and temper items are required.
  
* **Unenchanted Walking Stick version.** Default Main file, Low Level Player version and High Level Player versions options (folders) available in the same file, install only one. It's fine to switch mid-game.
**Bedroll Shown patch.** It will shown the bedroll under the backpacks. Works with any version. Only textures.
  
* **Walking Stick Staff addon**. It will use the lovely Alternate Movement from the [Goetia Animations - Enchanted Staves](https://www.nexusmods.com/skyrimspecialedition/mods/71110)﻿ mod. Or similar mods.
**Optional No Bandolier and Satchel patch**.
  
* **Optional Seasoned Traveller Pickaxe animation patch.** It will replace the vanilla Ancient Nordic Pickaxe model in the mining animation when using the Seasoned Traveller Pickaxe. Beware the Seasoned Traveller Pickaxe model with used when mining with the Ancient Nordic Pickaxe as well.

  
- [LE version here](https://www.nexusmods.com/skyrim/mods/112322)﻿.
