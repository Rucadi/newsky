Dwemer Researcher Armor and Outfit**
  
**By Xtudo**

  
**Description**
  
**Craftable Dwemer based armor and outfit.**
  
  
**The mod includes**

  
* A light armor, an outfit, gauntlets, amulet and boots.
2K and 4K versions.
  
* UNP and CBBE based, but it will work with any body type.
All items are craftable (no requirements), and you can temper them too. Up to Legendary (tested).
  
* The stats are the same that the vanilla Scaled Armor.
Proper inventory items.
  
* All the pieces are affected by "Ancient Knowledge".

  
The mod is fully standalone.
  
  
- [LE version here.](https://www.nexusmods.com/skyrim/mods/109696)
  
  
**Optional files**
  

  
* ESP file with the "Dwarven Smithing Perk" requirement to craft the items.
Covered waist area of the Armored version.
  
* **Calcelmo sells it**. Calcelmo, the Scholar of Markarth will sell a full set. It respawns. ESL.
**Katria Outfit version.** It will add the armored version to [Katria](https://en.uesp.net/wiki/Skyrim:Katria)﻿, to her ghost and dead body. It will replace DwemerResearcherArmor.esp. Fully compatible with other patches. Fully compatible with Katria aesthetical mods. It's fine to install mid-game. Requires one of the Main files installed first. Dwemer Smithing Perk version available.
  
* **Remiel Outfit**. Standalone. 4K and 2K versions available.
