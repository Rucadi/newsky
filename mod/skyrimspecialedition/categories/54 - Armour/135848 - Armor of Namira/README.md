# Armor of Namira
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/135848


[Patreon](https://www.patreon.com/chakrasse)﻿      [Ko-Fi](https://ko-fi.com/chakrasse)﻿      [Discord](https://discord.gg/sMxZ6chT8c)
  
  
This mod adds a standalone light armor set and greatsword inspired by Dark Fantasy.
  
  
The armor all share almost the exact same stats as vanilla Dwarven and recipes as Ebony armor, which means you craft them at a forge under the Ebony section! (Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Namira's [Armor Piece]''. Weapon is ''Namira's Greatsword''.
  
  
No HDT SMP on this mod, Males and Females have custom meshes. Textures are all 4K/2K Resolution!**If anyone wants there to be a HDT SMP version, message me and we can discuss!**
  
  
There is 1 main file for this mod.
  
  
**There are multiple versions of some armor pieces for visual differences.
