# Northman - Armor Mashup - HDT SMP
- Author: Fuse00 - NordWarUA - Hjartulv
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/133313


﻿Overview
  
Mashup of several armors with northern aesthetics in mind.
  
  
All armor set pieces, but one, are basically the same as Dragonbone armor pieces.
  
The first armor piece, the most basic one, can be crafted with an Ebony Smithing Perk; everything else requires Dragone Armor Smithing.
  
All armor pieces are named starting with "Northman".
  
![](https://i.imgur.com/iZGBfNO.png)
  
Compatibility
  
Contains same textures with same texture path as original armor sets. 
  
Load before any retextures, otherwise doesn't matter.
  
  
I don't know anything about creating textures, so if anyone is willing to help or provide instructions, I am willing to learn how to do a better job.
  
  
[Immersive Sounds - Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/523) integration in optional files.
