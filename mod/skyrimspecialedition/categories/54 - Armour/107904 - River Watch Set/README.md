# River Watch Set
- Author: Pulcharm
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/107904


This mod adds a imperial themed light armor set & a weapon
  
**Requirements:**
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)ï»¿
  
  
**Installation Guide:**
  
SpoilerStep 1: Download and install [SKSE](https://skse.silverlock.org/)
  
Step 2: Download and install [FSMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339?tab=files), pick the version according to your Skyrim version, XP32 and FSMPV are
  
not required for this mod to work, you don't have to install them
  
**If you are experiencing the cloth stretching downwards bug then you most likely have installed FSMP or SKSE incorrectly**
  
Step 3: Download and install this mod
  
**Integration:**
  
Can be crafted, requires Advanced Armors smithing perk, loading screen appears randomly
