# Melodic Bikini Armor - CBBE-TBD-UUNP SE
- Author: Melodic - Daymarr
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/84029


[font=Comic Sans MS]**![](https://i.imgur.com/H4SMPaZ.png)﻿﻿**[/font]**A made from scratch outfit by****[Melodic](https://twitter.com/Melodic_RU)﻿****, featuring a unique light armor set inspired by Nisetanaka's Artwork. The outfit comes with CBBE 3BA, TBD SE, COCOBody CBBE/UUNP bodyslides support****. The Body and Helmet come with HDT-SMP physics.**
