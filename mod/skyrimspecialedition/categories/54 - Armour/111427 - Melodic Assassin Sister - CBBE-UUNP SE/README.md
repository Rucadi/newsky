# Melodic Assassin Sister - CBBE-UUNP SE
- Author: Melodic-Daymarr
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111427


[font=Comic Sans MS]**﻿![](https://i.imgur.com/H4SMPaZ.png)﻿**[/font]**A mixed of made from scratch and daz assets by** **[Melodic](https://twitter.com/Melodic_skyrim)﻿﻿****, featuring a unique light armor set in an assassin/ninja style design by HM. The mod comes with 2 different weapons and an armor set in multiple colors with CBBE 3BA & COCOBody CBBE & UUNP bodyslides support. The veil comes with HDT-SMP cloth physics.**
