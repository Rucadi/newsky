**Introduction**
  
  
  
This is a retexture of Telthalion's [Witchplate Armor](https://www.nexusmods.com/skyrimspecialedition/mods/17053) using [Cabal120's Steel armor retexture](https://www.nexusmods.com/skyrimspecialedition/mods/35382) as a base. This retexture works for both the standalone version and [Immersive Armors](https://www.nexusmods.com/skyrimspecialedition/mods/3479). I plan on retexturing most sets from Immersive armors. But since the witchplate armor is also available as a standalone mod, I decided to release it right away.
  
  
  
  

**Installation**

  
  
  
Simply download using your favourite mod manager. I modified the NIF to change shader settings, so a patch will need to be created if you use a mesh replacer.
  
  

**Compatibility**

  
  
  
Can be used with both the standalone version and Immersive armors

Thank you for giving this mod a chance! If you anyone has suggestions or requests, feel free to do so!
