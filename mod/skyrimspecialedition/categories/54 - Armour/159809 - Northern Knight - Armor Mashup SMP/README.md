# Northern Knight - Armor Mashup SMP
- Author: Hjartulv
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/159809


Overview
  
Mashup of several armors with northern aesthetics in mind. For male characters only. Designed for a travelling sellsword knight or a more grounded paladin.
  
  
All armor set pieces are set to be an Ebony tier and require the Ebony crafting perk.
  
No beast race support.
  
  
I'm an amateur; there will be clipping. I did my best.
  
Your screenshots would be welcomed.
