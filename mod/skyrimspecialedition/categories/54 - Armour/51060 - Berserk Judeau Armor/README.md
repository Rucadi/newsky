# Berserk Judeau Armor
- Author: Gimora
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/51060


Description:
  
This mod adds Judeau’s armor from Berserk.
  
The mod includes Boots, Cape, Cuirass, Gauntlets, Helmet (invisible) and Cutlass for male and female.
  
Cutlass cannot be used as a weapon.
  
  
Requirements:
  
[DIMONIZED UNP female body](http://www.nexusmods.com/skyrim/mods/6709/?/)
  
  
How to get:
  
The armor can be crafted under the steel category or the misc category.
