# Adventurer HDT-SMP Armor
- Author: Fuse00
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/91741


[![](https://i.imgur.com/36W5s3A.png)](https://www.patreon.com/fuse00)
  
  
[font=Trebuchet MS]**MOD INCLUDES:**[/font][font=Trebuchet MS]
  
  
Unisex HDT-SMP standalone light armor. Physics is enabled for skirts, sleeves, belt props, laces, hood and backpack.
  
  
Armor includes:
  
  
1) Body
  
2) Gauntlets 
  
3) Boots
  
4) Hood (not compatible with beast races)
  
5) Backpack in 4 variations: default, no straps, clean, clean with no straps.
  
[/font][font=Trebuchet MS]
  
Outfit is craftable and upgradable, plugin is flagged as ESL.
  
  
**INSTALLATION:**
  
  
Use preferred mod manager or extract the content of the archive to your Skyrim Special Edition/Data folder.
  
  
The armor can be crafted on forge, or use the console commands:
  
  
Type in console:
  
*help adventurer*
  
...find the ARMO items ID and add the outfit to inventory with console command:
  
*player.additem <ID> 1*
  
  
Or use AddItem/QUI mods.
  
  
**REQUIREMENTS:**
  
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339)[/font][font=Trebuchet MS] is required for physics to work (as far as I can see "classic" HDT-SMP plugin doesn't get updates anymore).
  
[/font]
