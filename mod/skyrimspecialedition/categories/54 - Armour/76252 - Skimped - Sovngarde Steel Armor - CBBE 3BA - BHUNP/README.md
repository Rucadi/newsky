# Skimped - Sovngarde Steel Armor - CBBE 3BA - BHUNP
- Author: lustrianna
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/76252


Skimpifies the already wonderfully skimpy Sovngarde Steel Armor by [lustrianna](https://www.nexusmods.com/skyrim/users/2535456)!
  
  
This was done at the request of [TheFallofApril](https://www.nexusmods.com/skyrimspecialedition/users/88258998), who has made an [up-to-date port](https://www.nexusmods.com/skyrimspecialedition/mods/76115) of [lustrianna's original LE mod](https://www.nexusmods.com/skyrim/mods/16096). Please give everyone their deserved Kudos and Endorsements :)
  
**This file contains:** 
  

  
* Bodyslide files with no breast physics, realistic breast physics, and an *extra* skimpy version!
Zaps for nearly all pieces of armor.
  
* Short Cloak slider.
Leg armor zap for the boots.
  
* Fixed split/broken meshes in some spots, the necklace had some missing vertices for example.
Everyone's favorite: ***Boob Squish slider!** not recommended to be used on small chested presets!*
  

  
**This file does not contain:**
  

  
* lustrianna's mod. [Install TheFallofApril's port here.](https://www.nexusmods.com/skyrimspecialedition/mods/76115)
Prebuilt meshes. You MUST use Bodyslide to build them yourself.
  
* Preset for my beautiful Orc or her body, you get her [here](https://www.nexusmods.com/skyrimspecialedition/mods/98835)﻿ :)
