# Gorm HDT-SMP Armor
- Author: Fuse00
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119257


[![](https://i.imgur.com/36W5s3A.png)](https://www.patreon.com/fuse00)
  
[font=Trebuchet MS]
  
**MOD INCLUDES:**[/font][font=Trebuchet MS]
  
  
Unisex HDT-SMP standalone light armor. 
  
  
Armor includes:
  
  
1) Body
  
2) Gauntlets 
  
3) Boots
  
4) Helmet (not compatible with beast races)
  
  
Outfit is craftable and upgradable, plugin is flagged as ESL.
  
  
**INSTALLATION:**
  
  
Use preferred mod manager or extract the content of the archive to your Skyrim Special Edition/Data folder.
  
  
The armor can be crafted on forge, or use the console commands:
  
  
Type in console:
  
*help gorm*
  
...find the ARMO items ID and add the outfit to inventory with console command:
  
*player.additem <ID> 1*
  
  
Or use AddItem/QUI mods.
  
  
**REQUIREMENTS:**
  
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339) is required for physics to work (as far as I can see "classic" HDT-SMP plugin doesn't get updates anymore).
  
  
**KNOWN ISSUES:**
  
  
Physics sometimes become messy (especially when you enter new location), to fix the collisions go to first person view and back, or (recommended) use [this](https://www.nexusmods.com/skyrimspecialedition/mods/86845)﻿ mod.[/font]
