**Dwemer Armor Left Gauntlet replacer****by Xtudo**

**Description**
  
**Replaces the vanilla left gauntlet with the lovely one from the Dwemer Armor mod by 4thUnknown.**
  

  
* Intended for the [Creation Sunder & Wraithguard](https://en.uesp.net/wiki/Skyrim:Sunder_%26_Wraithguard) (ccbgssse008).﻿
Some fixes to the original Wraithguard.
  
* Proper weight sliders (intended for my [Dwemer Armor SE - My patches and fixes](https://www.nexusmods.com/skyrimspecialedition/mods/81046)﻿ mod).
It will use any texture replacer you used for the Dwemer Armor mod.
  
* No ESP, so it's fine to install/uninstall it at any time.
**FULL VERSION**: Sunder and Wraithguard edited to better match the original Dwemer Armor color palette. Not compatible with Ancient Dwemer Metal.
  

  
**Optional downloads**
  

  
* **Ancient Dwemer Metal patch.** Only textures. It's safe to install/uninstall it at any time.
