# Girl's Travel Outfit
- Author: Suna203
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/125910


[font=Lucida Sans Unicode]Description:
  
This mod adds a cute travel outfit designed for young female characters in Skyrim. The outfit includes the following items:[/font]

  
* [font=Lucida Sans Unicode]Torso (Light Armor)[/font]
[font=Lucida Sans Unicode]Gloves (Light Armor)[/font]
  
* [font=Lucida Sans Unicode]Boots (Light Armor)[/font]
[font=Lucida Sans Unicode]Choker (Amulet)[/font]
  

[font=Lucida Sans Unicode]
  
Supported body types:[/font]

  
* [font=Lucida Sans Unicode]CBBE[/font]
[font=Lucida Sans Unicode]3BA[/font]
  
* [font=Lucida Sans Unicode]BodySlide (optional)[/font]

[font=Lucida Sans Unicode]
  
Acquisition:
  
The outfit can be crafted at any forge. Light armor pieces can be upgraded.
  
  
Plugin:
  
ESPFE format

This is my first publicly released armor mod. Created from scratch using Blender, it's the result of much trial and error.
  
 As a beginner, there might be some imperfections. Nevertheless, I hope this mod adds a touch of charm to your Skyrim adventures. Please give it a try!
  
  
  
  
[/font][font=Lucida Sans Unicode]**HDT-SMP Cloth Physics and RaceMenu High Heels Patch by [[b]Level23](https://next.nexusmods.com/profile/Level23/about-me?gameId=1704)**﻿[/b]<https://x.com/BreakSunny/status/1820023233300820099>[/font][font=Lucida Sans Unicode]I would like to express my heartfelt thanks to Level23 for creating these amazing patches that add HDT-SMP cloth physics and RaceMenu High Heels to this outfit!
  
With his permission, I have added these patches as optional files.
  
[/font][font=Lucida Sans Unicode]These are BodySlide files. After installation, please build the items under "Girl's Travel Outfit SMP [3BA]" and "Girl's Travel Outfit Boots High Heels" in BodySlide.
  
Since this patch includes XML files, make sure to keep it enabled after building in BodySlide. Disabling it will prevent SMP from functioning.
  
If you experience issues with chest physics not working, please try installing [HDT-SMP Slot 32 Fix](https://www.nexusmods.com/skyrimspecialedition/mods/119010?tab=description). It helps resolve problems where physics stop working until the outfit is re-equipped.
