# Nordic Brute Armor (CBBE HDT-SMP)
- Author: Chakra SSE
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/146704


[Patreon](https://www.patreon.com/chakrasse)﻿      [Ko-Fi](https://ko-fi.com/chakrasse)﻿      [Discord](https://discord.gg/sMxZ6chT8c)
  
  
This mod adds a standalone heavy armor set inspired by Fantasy.
  
  
The armor all share almost the exact same stats and recipes as vanilla Stormcloak armor, you can buy the armor from the Windhelm blacksmith! (Or use console commands/[AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563))
  
  
Armor in game is named ''Nordic Brute [Armor Piece]''. 
  
  
Males and Females have custom meshes. Textures are all 4K Resolution! Females use CBBE (Curvy) Preset.
  
  
There is 1 main file for this mod.
