![](https://i.imgur.com/TBbmfPy.png)
  
This plugin has been created with the most recent version of the creation kit, if you are on a game version earlier than 1.6.1130 I recommend you use [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441) otherwise your game can crash, wont start or the mod wont be loaded. If you dont wanna use that mod you have to set the version of the .esp in SSEdit from 1.71 to 1.70.
  
  
Install using your favourite mod manager.
  
  
Optional **Complex Material** support.
  
Requirements: ENB
  
Set EnableComplexMaterial=true and EnableDynamicCubemap=true.
  
If these are greyed out in your ingame ENB control panel you have to set them manually in your enbseries.ini located in your Skyrim folder.
  
  
Optional **PBR** support.
  
Requirements: [Community Shaders](https://www.nexusmods.com/skyrimspecialedition/mods/86492).
  
  
You have to build the outfit in Body Slide.
  
  
Safe to install mid-save.
  
  
Default texture resolution is 2k, higher resolutions are found under optional files.
  
  
![](https://i.imgur.com/FUJK8F3.png)
  
Modular heavy or light armor set.
  
Stats are based on Orcish tier.
  
Esl flagged .esp, load early.
  
Complex Material support.
  
Immersive physics, only butt thighs and belly have physics enabled.
  
Only the brooch, cuirass, gauntlets and sabatons are temperable to retain balance.
  
Keywords for inventory icons for [B.O.O.B.I.E.S](https://www.nexusmods.com/skyrimspecialedition/mods/89241)﻿ included.
  
Optional high or low heels, for high heels you need [Heels Fix](https://www.nexusmods.com/skyrimspecialedition/mods/64442)
  
Optoinal heel sounds. requires [Heels Sound 1.6 SSE - AE](https://www.nexusmods.com/skyrimspecialedition/mods/62502)
