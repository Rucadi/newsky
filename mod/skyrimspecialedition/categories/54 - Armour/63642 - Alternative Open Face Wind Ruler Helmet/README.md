# Alternative Open Face Wind Ruler Helmet
- Author: 4thUnknown
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63642


An alternative open face style of the helmet from [Wind Ruler Armor SE](https://www.nexusmods.com/skyrimspecialedition/mods/60842) by [4thUnknown](https://www.nexusmods.com/skyrimspecialedition/users/49002333).
  
Whole lower face part of the helmet is removed to show them beards and added a back neck plate with a custom texture.
  
  
  
\* Includes:
  
- a regular helm version and a circlet version
  
- Also optional no hair mesh files for male/ female for the helm version
  
  
\*\*\*
  
- circlet version probably clips through some hair styles, but it's there if you want to try
  
 ﻿ ﻿- (beast races are not/yet supported - will clip through beastfolk skulls)
  
 ﻿﻿- male helmet model is same as in the helm version but character hair is visible
  
 ﻿﻿- female helmet model - see images section
  
  
 ﻿ ﻿- opt. file added: male circlet meshes for add'l options and consistency
  
  
  
Update:
  
- Added Beastfolk and Orc helm meshes update files (with and without braided hair mesh versions)
  
  made by [Xtudo](https://www.nexusmods.com/skyrimspecialedition/users/452087).

 ﻿- Meshes only
  
 ﻿- Requires the update 1.1 for 1WR.esl users
  
 ﻿- Requires the update 1.1.5 for 1WR.esp users
  
  
  
Installation:
  
 ﻿1.) Choose your main file, overwrite the original mod
  
 ﻿2.) Choose either of the update files, overwrite main helm version
  
 ﻿3.) DL opt file if using the .esp of the original mod
