# Civil War Champions - Proper Male Weighting
- Author: DraugrOfRagnarok
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/159016


**A complete fix for the male Imperial Dragon and Stormbear Armors.**This mod addresses a common visual issue with the male **Imperial Dragon** and **Stormbear Armors** by applying proper **weight painting** to both sets.
  
  
**What does this mod do?**In the original game, these armors lack correct weight painting on male characters, This mod corrects those errors, ensuring the armor moves smoothly and realistically with your character's body.
  
  
**Why use this mod?**You may have seen other mods that attempt to fix these armors. However, many only fix one set or don't apply the weighting correctly, which can introduce new visual bugs. This mod provides a complete, polished solution for both armor sets in a single file.Enjoy a more immersive experience with your favorite armor sets, free from visual glitches
