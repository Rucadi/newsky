# Wardancer Armor set - 3BA - HDT SMP
- Author: Mykky
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128851


**Add a female, Bosmer themed, light armor set, consist of:**
  
  
- Full combined set, including gauntlets, boots, bodyarmor
  
- Necklade
  
- Mask
  
- Toolbelt
  
- Front skirt
  
  
**INSTALLATION:**
  
  
1. Use preferred mod manager or extract the content of the archive to your Skyrim Special Edition/Data folder.
  
2. Generate mesh in Bodyslide, search "wd" in the name filter.
  
  
![](https://i.imgur.com/WtuBKLB.png)
  
  
Ingame it can be crafted at the forge.
  
Or use [Additem menu](https://www.nexusmods.com/skyrimspecialedition/mods/17563)﻿.
  
  
**REQUIREMENTS:**
  
  
1. [CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174).
  
2. [Faster HDT SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339).
