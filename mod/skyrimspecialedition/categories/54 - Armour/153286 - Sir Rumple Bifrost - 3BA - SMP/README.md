# Sir Rumple Bifrost - 3BA - SMP
- Author: Sir Rumple
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153286


This mod adds 3 new colors to the outfit
  
Black - Makes it all black except for a few shiny red bits
  
Red - Black with brighter Red highlights
  
Wine - Black with darker Red highlights
  
  
Original mod is required
  
Outfit must be crafted. It is not found in the World
  
Comes with pre-generated CBBE Curvy mesh
  
Use the Bodyslide files if you wish to change the body shape
  
Plugin was flagged as esl for your convenience﻿
