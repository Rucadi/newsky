# Armor of Peryite for the Afflicted (SkyPatched)
- Author: Ennui
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129286


**DESCRIPTION**
  
- Armor pieces are added to the Afflicted items leveled lists
  
- You will start seeing the Afflicted wearing pieces of the armor set around **level 24**. 
  
- No plugins, just .ini file. 
  
  
**INSTALLATION / REQUIREMENTS**
  
- Install [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659) and its requirements 
  
- Install [Armor of Peryite](https://www.nexusmods.com/skyrimspecialedition/mods/129243) 
  
- Install this mod whenever and enjoy
  
  
**UNINSTALLATION**
  
- Uninstall the mod whenever and enjoy 
  
  
**INCOMPATIBILITIES / ISSUES**
  
- NONE, this is just a .ini file, no plugins - should be compatible with everything
