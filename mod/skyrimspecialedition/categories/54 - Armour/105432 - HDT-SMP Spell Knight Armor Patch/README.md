# HDT-SMP Spell Knight Armor Patch
- Author: tatururutia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105432
This MOD adds hdt-SMP cloth physics to the **Spell Knight Armor** from CC content.
  
Supporting three colors of the original armor and some variants.
  
  
Require HDT-SMP (Skinned Mesh Physics) OR FSMP - Faster HDT-SMP.
  
Require **ccedhsse002-splkntset.esl** and **ccedhsse002-splkntset.bsa** from CC content.
  
  
  
If you want to install variants, you also need **Spell Knight Armor Variants** [textures file](https://www.nexusmods.com/skyrimspecialedition/mods/84620?tab=files) from Nexus.
  
(You can **delete 'FFMSpellKnightVariants.esl'**, it is not needed, only need textures file.)
  
Then use this physical patch to completely cover it.
  
  
  
[Compared to the previous physical patch version, I increased the weight of the metal skirt to make it look more like metal.]
