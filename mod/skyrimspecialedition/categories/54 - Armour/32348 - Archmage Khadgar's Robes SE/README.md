**What this mod includes:** 
  
  
**♂️ Male Only**
  
  
• Khadgar's Hood (with room for Circlets & support for Argonians/Khajiit)
  
• Khadgar's Gloves
  
• Khadgar's Boots
  
• Khadgar's Robe (Blue / Red variants)
  
  
Comes in ? **Mage Friendly** (no armor rating / armor type) and ?️ **Armored (Elven/Light)** versions
  
  

![](https://i.imgur.com/ZM71deY.png)

  
  
**Obtaining:**
  
  
• Each Khadgar piece is crafted at the **Skyforge.** Only the Armored [A] version can be tempered.
  
  

![](https://i.imgur.com/ZM71deY.png)
