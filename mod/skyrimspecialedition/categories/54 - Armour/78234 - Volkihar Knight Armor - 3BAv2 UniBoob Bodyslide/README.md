# Volkihar Knight Armor - 3BAv2 UniBoob Bodyslide
- Author: Vichoriz0
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78234


A 3BAv2 conversion of this wonderful armor set made by yamanotaka. Only the bodyslide files are included so you'll need to have the original mod and build these files with bodyslide.
  
I tested these with several body preset and only found negligible clipping. If you found some, please comment the body preset you're using and in what position it appears and I MAY fix it.
