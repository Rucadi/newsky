# Lifesworn Vestiges - 3BA patch
- Author: Neumeria
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/154997


***Description:***
  
CBBE 3BA Bodyslide patch for [Lifesworn Vestiges -- Champion of Arkay armor (optional SMP)](https://www.nexusmods.com/skyrimspecialedition/mods/136837). SMP and non-SMP version are supported. Batch-build all the files and then pick and choose the ones you want from the conflicting files.
  
  
***Install Order:***
  

  
1. [Lifesworn Vestiges -- Champion of Arkay armor (optional SMP)](https://www.nexusmods.com/skyrimspecialedition/mods/136837)﻿.
This mod.
  

  
***Requirements:***
  

  
* [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/198)﻿.
[CBBE 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174)﻿.
  
* [Bodyslide and Outfit Studio](https://www.nexusmods.com/skyrimspecialedition/mods/201)﻿.
[Lifesworn Vestiges -- Champion of Arkay armor (optional SMP)](https://www.nexusmods.com/skyrimspecialedition/mods/136837)﻿. Install the SMP patch as well unless you are 100% sure you won't be using SMP.
  

  
***Recommended:***
  
[Helmet Toggle 2](https://www.nexusmods.com/skyrimspecialedition/mods/100617) - To toggle between open and closed helmet. Immersion++!!!
  
  
***Installation/Uninstallation:***
  
Simply use a mod manager of your choice.
  
(Don't forget to build the files in bodyslide. It has its own group, check images for reference.)
