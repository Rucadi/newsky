# Immersive Collections Resources
- Author: Canliberk
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/158971
This mod includes only some resources for the collection. It is not recommended to download for non-collection users or curators that are building their own stuff.
  
  
- MCM recorder files for 17 mods for quick start
  
- Animation outputs (Pandora output)
  
- Bodyslide files for Immersive & Adult collection's body and clothing files
  
- MCM helper settings
  
- Payload interpreter folder fix
  
- Compass mod ini files
  
- File access interface script fix (With EPinter's written permission. Not the original FISS file, just a patch that makes it work w. collection)
