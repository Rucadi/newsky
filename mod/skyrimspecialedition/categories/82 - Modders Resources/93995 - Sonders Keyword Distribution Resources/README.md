Provides Animation **Mod Developers** With **Custom Keywords**
  
Ensuring animations only play when these Keywords are detected
  
 For use with the [OAR](https://www.nexusmods.com/skyrimspecialedition/mods/92109)﻿ framework or in some future cases [DAR](https://www.nexusmods.com/skyrimspecialedition/mods/33746)﻿
  
  
This enables easy **NPC specific animations** like filtering for **Player**
  
only or **Elder NPC** only animation sets "negate to effect other npcs"
  
Also the main point originally, this allows detecting specific
  
**Furniture Types** for mods like my [Female sitting overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/85599)﻿

- **OAR** Keywords -
  
  
Furniture Section
  
SpoilerFurniture Keywords can only be read by OAR
  
as far as I am aware
  
  
Why add furniture keywords? 
  
Some sitting animations are applied to all furniture ingame
  
this a complete mess for animation mods as animations will clip on one furniture
  
or float on another. with these keywords specific furniture can receive correct animations
  
  
Previously "cloak spells" have been used by modders including myself to apply specific animations
  
Now with OAR for furniture at least spells and scripts are unnecessary
