# Random Mining
- Author: IsharaMeradin
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/4476


[font=Comic Sans MS]

Random Mining
  
by IsharaMeradin

[/font][font=Comic Sans MS]About[/font]
  
Default option allows for configurable quantities in ore randomization using normal strikes.
  
Lazy Man option allows for configurable quantities in ore randomization using one strike set.
  
Hard Worker option allows for configurable ranges of skill level, configurable quantities in ore randomization, and configurable strike sets.
  
Magic mining mods are supported by allowing the user to select the magic skill used by an installed magic mining mod.
  
Mining Makes Noise adapted from the optional file in Traps Make Noise by kryptopyr
  
  
[font=Comic Sans MS]Requirements[/font]
  
SKSE
  
SkyUI
  
  
[font=Comic Sans MS]Installation[/font]
  
Use the mod manager of your choice to install.
  
If installing manually please ensure the files follow the structure below.
  
  
[font=Comic Sans MS]Files & Folders[/font]
  
Data > Random Mining MCM.esp
  
Data > Scripts > abim\_RM\_MCM\_QuestScript.pex
  
Data > Scripts > MineOreScript.pex
  
Data > Scripts > Source - Random Mining MCM > abim\_RM\_MCM\_QuestScript.psc
  
Data > Scripts > Source - Random Mining MCM > MineOreScript.psc
  
Data > Interface > abim > abim\_RM\_Title.dds
  
Data > Interface > Translations > Random Mining MCM\_english.txt
  
Data > Docs > Random Mining MCM Info.txt
  
  
[font=Comic Sans MS]Compatibility[/font]
  
Not compatible with any mod that modifies the MineOreScript.pex file. 
  
In some cases this mod performs the same function and can be installed after the other mod. Example: SkyRe and ARS Metallica both modify the script to provide smithing experience.
  
In some cases this mod is in direct conflict.  Example: Advanced Mining
  
  
[font=Comic Sans MS]Known Issues[/font]
  
After uninstall there will be some complaints in the papyrus log due to added variables missing. This does not affect proper functioning of the script.
