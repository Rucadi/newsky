# Dynamic Cubemaps - Additions
- Author: imnotfamous
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129583


Some dynamic cubemaps to work with [Dynamic Cubemaps](https://www.nexusmods.com/skyrimspecialedition/mods/112763)﻿ and based on [Physically Based](https://physicallybased.info/) values. 
  
  
Currently include (RGBA):
  
  
Gold: 255,253,225,011 
  
Limestone (based on Concrete):  192,192,192,255
  
Marble: 235,230,225,179
  
  
More to come...
