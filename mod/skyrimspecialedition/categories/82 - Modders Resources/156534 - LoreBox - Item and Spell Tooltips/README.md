# LoreBox - Item and Spell Tooltips
- Author: shazdeh2
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/156534


The Description field in Skyrim is quite limited, even when you put mods on top to make the item descriptions bigger. This mod provide an additional way to have some descriptive text show up for desired items or spells. These will show when you hover over them in your inventory or while bartering.
  
  

  
* Supports localization
Multiple mods can add or remove text for same item/spell, without conflict.
  

  
There is a tutorial file available for download to see how it works.
  
  
**Compatibility**
  
Mod is ESP-FE and doesn't take up a slot in your load order. Compatible with all mods, including UI overhauls.
