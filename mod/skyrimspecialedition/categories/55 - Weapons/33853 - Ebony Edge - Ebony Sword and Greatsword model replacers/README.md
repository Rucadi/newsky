This mod replaces the appearance of the Ebony Sword and Greatsword models.
  
  
As these use vanilla textures, the overall appearance of each weapon will depend on your current Ebony weapon textures.
  
  
There's an optional plugin for the standalone option that makes each weapon have a unique name that corresponds to their variant name for easy identification in the inventory.
  
 ﻿
  
 *﻿For example: "Ebony Sword - Variant 1" and so on.*
