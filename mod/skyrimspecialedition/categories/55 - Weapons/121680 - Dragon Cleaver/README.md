A blade that saw great usage in the dragon wars.
  
Many dragons saw their demise facing the warriors that wielded this weapon.
  
  
![](https://i.imgur.com/TBbmfPy.png)
  
Install using your favourite mod manager.
  
  
Safe to install mid-save.
  
  
Default texture resolution is 2k, higher resolutions are found under optional files.
  
  
![](https://i.imgur.com/FUJK8F3.png)
  
Immersive greatsword and one handed sword.
  
Stats are based on Ebony tier.
  
Esl flagged .esp, load early.
  
  
![](https://i.imgur.com/HxwyDnw.png)
  
This weapon can be found before a major story fight.
  
Spoiler
  
In the Halls of Valor in Sovngarde on a table.
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/121680/121680-1718135758-1326448703.jpeg)
  
If you dont wanna wait you can use console commands.
  
help "dragon cleaver"
  
help "dragon toothpick"
  
player.additem (whatever ID the console tells you)
