# Silver Bow
- Author: CanissaLupu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79302


This Mod adds a craft-only version of InsanitySorrow's Silver Bow with optional Leveled Lists Addon.
  
  
If you want more of their weapons, some even with leveled list integration, check out [InsanitySorrow Weapons Pack](https://www.nexusmods.com/skyrimspecialedition/mods/8206) and [Lore Weapon Expansion SE](https://www.nexusmods.com/skyrimspecialedition/mods/9660).
