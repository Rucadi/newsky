This mod replaces the appearance of the Elven Sword and Greatsword weapons using LeanWolf's mesh as the base, combined with other Elven weapon parts.
  
  
As these use vanilla textures, the overall appearance of each weapon will depend on your current Elven weapon textures.
  
  
**The textures in the weapon previews use the [Elven Weapons for Silence](https://www.nexusmods.com/skyrimspecialedition/mods/27994)﻿ by [Erlionel](https://www.nexusmods.com/skyrim/users/5734721) and ported by [sumugi](https://www.nexusmods.com/skyrimspecialedition/users/7464912) ﻿to SSE with enhanced features!**
  
  
  
There are 5 models available for each weapon which can now be mixed and matched as you like as replacers in the AIO installer.
  
  
=====================================
  
**Standalone plug-ins info**
  
=====================================
  
  
  
**Caenraes** version = technically formatted for LE usage where every feature is included in a **single plugin** which are:
  
  
 ﻿- Forge availability / craftability for all variants
  
 ﻿- Leveled-list integration
  
  
  
**kitwitissues** version = offers **3** different plugins which are:
  
  
 1. ***Forge Fodder*** - only makes all variants craftable/temperable at the forge
  
  
 ﻿2. ***Leveled-list with world edits*** - same as no. 1 above with:
  
 
  
 ﻿﻿ ﻿-- additional leveled-list integration
  
  
 ﻿ ﻿-- Worldspace edits; wherein any Elven sword or greatsword that can be found in the world
  
 ﻿ ﻿(based on skyrim.esm or vanilla plugins I presume) are now randomized by the variants.
  
 
  
 ﻿﻿3. ***Immersive Weapons Integration*** = same as no. 2 but:
  
 ﻿ ﻿-- this needs the [Immersive Weapon Integration](https://www.nexusmods.com/skyrimspecialedition/mods/22420) mod made by kitwitissues
  
  
**So if you want a...**
  
  
 ﻿...standalone that only adds variants as craftable/temperable in the forge? = use kitwit's ***Forge Fodder***
  
  
 ﻿...standalone that adds variants in the forge and have leveled-list integration and nothing else? = use ***Caenraes***version
  
  
 ﻿...standalone that adds forge craftability, leveled-list feats with a bonus worldspace loot randomization for elven sword and GS? = kitwit's ***Leveled-list with world edits***
  
  
 ﻿...standalone and you have or want to try Immersive Weapon Integration and wanna use this mod as well with every feature as stated before this line? = use kitwit's ***Immersive Weapons Integration***=====================================
  
  
 **There's also a Mystic Elven weapons version for each model available at the Optional download section if you use the Standalone version of the said mod. The file is structured as a Modular pack.**
