# Spear of Skyrim - 1h Conversion
- Author: nukanekom3260 - Hjartulv
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/133069


Overview
  
This is converted to one handed version spears from [Spear of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/89785)﻿ by [nukanekom3260](https://www.nexusmods.com/skyrimspecialedition/users/43664312)
  
  
- Rebalanced all weapons. They are basically better swords.;
  
- Crafting recipes adjusted;
  
- Spears given keyword vanilla "WeapTypeSword" so they will scale with other mods without any patching ([SkyValor](https://www.nexusmods.com/skyrimspecialedition/mods/106240)﻿ for example);
  
- Spears given new keyword "WeapTypeSpear" and "WeapTypeSpear1h", so some animations and icons mods will recognize them as Spears;
  
- If [Keyword Item Distributor (KID)](https://www.nexusmods.com/skyrimspecialedition/mods/55728) and [Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469) installed, all Spears will get the "OCF\_WeapTypeSpear1H" keyword;
  
- If [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659)﻿ is installed, Spears will appear in blacksmith's stores, and some spears will appear as loot in chests;
  
- If [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659)﻿ is installed, some Spears will be wielded by bandits; 
  
- No distribution to enemies, because NPCs will certainly use Spear as a sword in dual wielding;
  
- ESL Plugin marked as ESL;
  
- Install anytime, delete anytime. 
  
  
Installation
  
1. Install original mod [Spear of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/89785)﻿ by [nukanekom3260](https://www.nexusmods.com/skyrimspecialedition/users/43664312);
  
2. **(Optional)** You can delete or hide original ESP if you don't need two-handed spears;
  
3. Endorse original mod [Spear of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/89785)﻿ by [nukanekom3260](https://www.nexusmods.com/skyrimspecialedition/users/43664312);
  
4. Install this mod;
  
5. **(Optional)** If you don't want some type of distributation.
  
You can delete or hide files 1h Spears of Skyrim.ini and\or 1h Spears of Skyrim - Bandits.ini located here:
  
*\SKSE\Plugins\SkyPatcher\leveledList\*

﻿Simple preset to move Sword in right hand to Sword Back node, if it is spear.
  
To install navigate to "View>Gear nodes", choose where to apply (Global tab best), then select the preset "SpearsOnBack" and click on "Merge".
