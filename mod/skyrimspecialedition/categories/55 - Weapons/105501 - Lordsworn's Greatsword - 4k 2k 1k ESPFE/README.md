# Lordsworn's Greatsword - 4k 2k 1k ESPFE
- Author: Karasumi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105501


**Lordsworn's Greatsword from Elden Ring**
  
Available in 4k, 2k and 1k resolutions, the plugin is an ESP flagged as an ESL (ESPFE).
  
All assets were ran through CAO, so you don't have to.
  
The sword can be crafted as well as tempered, it requires Steel Smithing perk.
  
Screenshots were taken using the 2k resolution version.
