**You need [PrivateEye](https://www.nexusmods.com/skyrim/users/887024)﻿'s mod [Royal Armory - New Artifacts](https://www.nexusmods.com/skyrimspecialedition/mods/6994)﻿ as these are just model and texture replacers.**
  
  
  
This mod replaces 13 weapons from PrivateEye's Royal Armory mod. These are:
  
  
- Black Crow Dagger
  
- Dagger of Shalidor
  
- Dragonfang
  
- Exquisite Elven Blade
  
- Hafyllbrand
  
- Harbinger's Axe
  
- Judgement
  
- Northern Honor
  
- Shard
  
- Silvertongue
  
- Smile
  
- Sword of Dragonsreach
  
- Stormbird
  
  
                                                                                                                                                                                                                                                                 
  
**9-19-2021 Update:
  
  
Fixed some blood effects not appearing on some weapons, Stormbird inventory fix, Silvertongue, Dragonfang, Judgement mesh fix (including blade seams), Exquisite Elven Blade overhaul (see 2nd image sample for reference).**
  
  
                                                                                                                                                                                                                                                                 
  
  
  
The main file which is the default model replacer, uses vanilla textures except for Judgement, Tullius' weapon, as it uses a modified Dawnbreaker texture along with some scabbards for some weapons.
  
  
The textures of the Elven weapon parts in the weapon previews use the [Elven Weapons for Silence](https://www.nexusmods.com/skyrimspecialedition/mods/27994)﻿ by [Erlionel](https://www.nexusmods.com/skyrim/users/5734721) and ported by**[sumugi](https://www.nexusmods.com/skyrimspecialedition/users/7464912)﻿**to SSE with enchanced features!
  
  
There are no plans to recreate the remaining weapons that are not included in this mod.
  
  
**DSR/Ecotone patch files are included for the main and optional files.
  
  
[b][b]The main Royal Armory mod [b][b][b]DSR patch files by thaiusmle**[/b][/b] is now ported to SSE thanks to DigitalCabbage!
  
  
[Click here for the Royal Armory DSR**/Ecotone** patch for SSE.](https://www.nexusmods.com/skyrimspecialedition/mods/28079)[/b][/b][/b] **(Install the main Royal Armory DSR patch files first before installing this mod.)**
  
  
  
There are 5 alternate replacers available:
  
  
 - ﻿**Dragonfang - Penitus logo**
  
          \*Modified vanilla texture; changes the vanilla Empire logo into the Penitus one.
  
  
 - **Shard - Custom Texture**
  
        \*Slightly modified color; if you don't want Shard affected by your current custom Dawnbreaker texture
  
  
 - **Judgement - Vanilla Texture Setup**
  
         \*If you want Judgement to use your current Dawnbreaker texture
  
  
 - **Hafyllbrand - Custom Texture & Sword of Dragonsreach - Custom Texture**
  
         \*Modified vanilla blade textures; If you use Runed Nord Hero Weapon retexture and don't want these 2 affected
