# Aetherium Weaponry ESL
- Author: Shaqar
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/121323


Adds Aetherial Dwarven themed Greatsword, Sword, Dagger, Mace, Warhammer, Axe, Greataxe, Shield, Bow and Arrows
  
  
  
The meshes utilize vanilla textures, so you will see the Dwarven textures that you have installed
  
  
  
You can **craft** these weapons at the **AETHERIUM FORGE**. You must have **Dwarven** **smithing** perk.
  
  
  
The plugin is flagged esl, so it won't count towards your plugin limit
  
  
  
Enjoy!
