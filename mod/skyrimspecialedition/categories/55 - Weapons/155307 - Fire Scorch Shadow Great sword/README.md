# Fire Scorch Shadow Great sword
- Author: volodavkaa
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/155307


> When the stone itself screams, the sword is ready.

  

  
* [font=Courier New]Scorch‑Shadow Greatsword (Static) – constant lava glow[/font]
[font=Courier New]Scorch‑Shadow Greatsword (Flicker) – animated emissive pulse[/font]
  
* [font=Courier New]ESL‑flagged ESP[/font]

  
Stats

  
* Type: Greatsword (two‑handed)
Base damage: 28
  
* Weight: 22

  
How to obtain

  
1. Forge – Static version
     
   Requires Daedric Smithing perk
     
   • 3 × Bone Meal
     
   • 2 × Daedra Heart
     
   • 2 × Fire Salts
     
   • 5 × Ingot Ebony
Infuse – Flicker upgrade (any forge)
  
• 1 × Scorch‑Shadow (Static)
  
• 2 × Glow Dust
  
  

![](https://i.ibb.co/MxsD1MgF/Infernus-3.png)﻿Installation

  
* Vortex / MO2: click “Install”.
Manual: drop the Meshes, Textures folders and the ESP file into Data.
  

Safe to add mid‑playthrough. To remove, unequip the sword and delete the files.
