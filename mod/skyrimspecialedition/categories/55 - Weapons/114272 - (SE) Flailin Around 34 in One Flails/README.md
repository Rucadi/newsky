DESCRIPTION - WHAT IT DOES

More weapons will come with time, only 34 atm (32 1h, 2 2h @4/8/24)

• Adds 34 new weapons. It's flails. You need physics for it. 
  
• If physics don't work, re-equip the item. Occupies armor slot 59 for the physics. Only one flail's physics will work at a time, can not dual wield.
  
• Will work in VR if you have a physics mod. WILL NOT work in First person.
  
• Created at forge (requires various smithing perks), with various materials.
  
• Tempered at Sharpening Wheel.
  
• No Enchantments applied by default, so you can enchant it as you see fit. Tempered with Arcane Blacksmith if enchanted.
  
• IT IS NOT ESL'd BECAUSE IT IS NOT FINISHED. WHEN IT IS FINISHED, I WILL ESL IT.
