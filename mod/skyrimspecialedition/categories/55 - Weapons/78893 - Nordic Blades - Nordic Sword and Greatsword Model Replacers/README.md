# Nordic Blades - Nordic Sword and Greatsword Model Replacers
- Author: Vilidran
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78893


---------------------------------------------------------------------------------------------------------------------------------
  
  
- This mod replaces the appearance of the Nordic Sword and Greatsword that comes with 5 variants each.
  
  
  
- Standalone file adds all variants of the Nordic Sword and Greatsword to the forge, along with leveled-list integration.
  
  
  
- Actual look is dependent on your Nordic sword, greatsword, and Steel scabbard textures.
