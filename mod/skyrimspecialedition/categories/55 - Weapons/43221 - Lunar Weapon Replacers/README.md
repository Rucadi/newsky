# Lunar Weapon Replacers
- Author: Clipsymoon - Casualkhajiit
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/43221


**Introduction:**
  
I was bored of the vanilla lunar weapons being just steel/iron weapons with a new name and enchantment so using the great models/textures from ClipsyMoon's Iron Things mod, I created this.
  
  
**Features:**
  

ESL flagged

Standalone version avaliabe

Compatible with LOTD, with a patch from the [official patch page](https://www.nexusmods.com/skyrimspecialedition/mods/30980/)

Meshes Included for Ecotone's dual sheath
  
  
**Changes:**
  
Spoiler
  
Lunar Steel Mace = Lunar Bow
  
  
Lunar Steel War Axe = Lunar Battleaxe
  
  
Lunar Steel Sword = Lunar Sword
  
  
Lunar Iron Mace = Lunar Dagger
  
  
Lunar Iron sword = Lunar Claymore (Greatsword)
  
  
Lunar Iron Axe = Replaced with a custom variant, using meshes from jg1 and textures from JonnyWang13
  
  
  
**Installation:**
  
Download as normal with a mod manager of your choice.
  
Manual: Drop into the data folder, not recommended.
