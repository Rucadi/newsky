﻿ABOUT

[font=Verdana] ﻿Attack Animations And Movesets is a personal continuous project that aims to fill Skyrim with as many attack  animations as possible that creates pseudo-combos or movesets per weapon type, left handed attacks and Special Movesets to create a certain uniqueness to regular weapon types and especially to special/ legendary weapons found in the game. 
  
  
 ﻿Attack Animations and Movesets doesn't have a specific style or a specific direction that it aims to focus on. It is not a mod that attempts to cater to a specific audience. There will be versions that release more realistic and grounded style and there will be versions that will be released that will be in the vain of more action game oriented styles. It attempts to give as many animations as possible to the user.

Requirements:
  
  
1. SkySA(1.2 Version)[SkySA LE requirements on its mod page is no longer needed]
  
2. Nemesis Engine
  
3. Dynamic Animation Replacer
  
  
These animations were designed to adapt to SkySA(1.2 Specifically)behaviors. Technically, It will work without it but it won't look as intended. SkySA modifies attack movement to have directional lock and slight forward movement, the animations were made to adapt to this. Not having SkySA will probably not make this mod function to its full potential. As  other animations would overlap with this(moving standard attack animations that will play if SkySA is not installled)
  
  
1.2 is highly recommended because in later versions of SkySA, the behavior was altered where in the 3rd attack is the standing power attack. 1.3+ is installed, then the 3rd standing power attack would always be thesame which makes the moveset somewhat predictable and have less depth. version 1.7 specifically requires these animations to be patched. it is still up to you if you like to use those versions however 1.2 remains the favored to open up more combos and movesets.
  
  
\*SKYSA REQUIREMENTS THAT REQUIRED PORTING ARE NO LONGER NEEDED, THIS MOD HAS ENOUGH ANIMATIONS AND THERE IS NO NEED TO DOWNLOAD THOSE REQUIREMENTS.\*
  
  
  
  
  
[font=Times New Roman]INSTALLATION 
  
[/font] 
  
 1. Install DAR
  
 2. install nemesis
  
 3. Install SkySA( just download 1.2, no need to port it's requirements, download 1.2 and patch in nemesis don't be reluctant to port mods, it is no longer necessary with this installed)
  
 4. Download this mod and install through mod manager or manually if you like
  
  
Compatibility
  
  
 ﻿ ﻿these animations were made to adapt to behavior of skysa, if that is not used then the animations may not fully work as intended. The animations will can still technically work but most of it will be overlapped with mods like CGO because of how that mod modifies behaviors and prioritizes its animations. Skysa is very modular and is highly compatible as well. only cgo is known to be incompatible
