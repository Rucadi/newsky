# Flower Girls NPC Relationships
- Author: Zhalroth
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/31732
**REQUIREMENTS**
  
  

  
* [Unofficial Skyrim Special Edition Patch](https://www.nexusmods.com/skyrimspecialedition/mods/266)
[PapyrusUtil SE - Modders Scripting Utility Functions](https://www.nexusmods.com/skyrimspecialedition/mods/13048)
  
* [Flower Girls SE and VR](https://www.nexusmods.com/skyrimspecialedition/mods/5941)

  
**WHAT DOES THIS MOD DO?**
  
  
This mod creates automatic flower girls scenes between NPCs (that don't involve the player).
  
The NPCs have a predefined list of partners, locations, and times when the scenes will play.
  
80% of this mod is married couples in their own home at night. There are also a few scenes between single NPCs and the flower girls.
  
Many of the scenes will require the player to pick locks and go tresspassing in order to see them. Almost everything happens after midnight.
  
As far a sex mods go, this is relatively mild. The one exception is Haelga's Bunkhouse. I won't spoil any surprises in the description.
  
  
**LOAD ORDER**
  
  
This mod makes some edits to furniture. Most of these changes are subtle.
  
I recommend the following load order:
  

  
* 1) any mods than changes interiors
2) this mod
  
* 3) lighting mods
