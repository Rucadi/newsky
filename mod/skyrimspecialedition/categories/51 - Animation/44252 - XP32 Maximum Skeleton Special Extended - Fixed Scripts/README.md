**XPMSSE - FIXED SCRIPTS**

  
  
The [XP32 Maximum Skeleton Special Extended](https://www.nexusmods.com/skyrimspecialedition/mods/1988) unfortunately had some issues in its scripts that were uncovered by [cskriffin](https://www.nexusmods.com/skyrim/users/21205994), the author of [All Geared Up Derivative](https://www.nexusmods.com/skyrimspecialedition/mods/28833). They wrote detailed instructions on how to fix the XPMSSE scripts in an [article on the AllGUD mod page](https://www.nexusmods.com/skyrimspecialedition/articles/1372) as sharing of the scripts was prohibited at the time.
  
  
To simplify the process for users, the wonderful [Umgak](https://www.nexusmods.com/skyrimspecialedition/users/6877023) created a BAT file which does all the work for you. It was previously available on our [joint patch hub](https://www.nexusmods.com/skyrimspecialedition/mods/26092) and is still available here in the miscellaneous file section.
  
  
However, groovtama of the XPMSE team has recently announced they are leaving the modding scene and opened permissions of XPMSE. This enables me to directly upload the fixed scripts for maximum convenience. You're welcome.
