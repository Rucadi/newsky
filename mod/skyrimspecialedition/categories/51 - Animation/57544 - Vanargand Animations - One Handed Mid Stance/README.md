# Vanargand Animations - One Handed Mid Stance
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/57544
This mod contains the following one-handed animations:
  
-Equip.
  
-Idle.
  
-Turn.
  
-Walk - all directions.
  
-Run/strafe - all directions.
  
-Sprint.
  
-Unequip.
  
-Dual wield idle and sprint.
  
  
This will serve as a base stance for the upcoming one-handed attack animations both for the vanilla and the SkySA behaviors (previews at the bottom of the page of their current state), currently in early access, and should still blend fairly well with the rest of the vanilla animations.
  
  
The mod can be used as a vanilla replacer, or placed into your folder structure of choice, like [Stances - Dynamic Animation Sets](https://www.nexusmods.com/skyrimspecialedition/mods/40484?tab=description)﻿ and [Combo Animation Framework](https://www.nexusmods.com/skyrimspecialedition/mods/40541?tab=description) by OmecaOne, which originally sparked the idea for this mod.
