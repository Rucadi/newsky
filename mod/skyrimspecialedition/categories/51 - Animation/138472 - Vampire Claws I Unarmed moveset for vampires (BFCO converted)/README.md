# Vampire Claws I Unarmed moveset for vampires (BFCO converted)
- Author: MathieuTheDunmer
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/138472


Converted [Vampire Claws I Unarmed moveset for vampires (MCO I SCAR)](https://www.nexusmods.com/skyrimspecialedition/mods/112078) animations to BFCO with [converter](https://www.nexusmods.com/skyrimspecialedition/mods/119926 ). All thanks to [MathieuTheDunmer](https://www.nexusmods.com/skyrimspecialedition/users/119540343) and other authors of used mods.
  
  
To run this mod on your vampire character you only need the latest version of BFCO. This mods works on 1.5.97~1.6.xx Skyrim versions. If you want this mod for NPCs vampires then you need to check the original mod and its requirements.
