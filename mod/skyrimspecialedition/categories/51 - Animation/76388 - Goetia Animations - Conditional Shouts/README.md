# Goetia Animations - Conditional Shouts
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/76388
The character (PC/NPC) will now perform different Shout animations depending on the shout's effect, the Shouts are categorized as seen in
  
the following table:
  
  
Spoiler![](https://i.imgur.com/XEeZymh.png)
  
Note that this also includes all shouts added by EnaiSiaion's [Thunderchild - Epic Shouts and Immersion](https://www.nexusmods.com/skyrimspecialedition/mods/1460)﻿ and Keysmash2c's [Thoom](https://www.nexusmods.com/skyrimspecialedition/mods/62770?tab=description)﻿.
