# Vanargand Animations - Sneak Crossbows
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67282
Using DAR, these animations will replace, while sneaking and holding a crossbow, the idle, walks, runs, draw/reload bolt, aimed idle, aimed bolt turn, bolt release, equip and unequip.
  
  
[size=3]In addition to the animations, and similarly to Vanargand Sneak Archery, this will ensure that the character and crossbow keep facing the crosshair at all time while aiming and moving/straffing or turning in any direction (see comparison to vanilla at 0:16 in the preview below) for a much more consistent aiming experience.[/size]
