# New Creature Animation - Werewolf
- Author: TongDing
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/83806


The animations have been upgraded, replacing unnatural-looking ones and incorporating a dodge feature. New animations for the Bloodlust have also been added, making it especially friendly for werewolf players.
  
  
Requirements
  
DAR
  
AMR
  
Payload Interpreter, 
  
Behavior Data Injector,
  
Iframe Generator
  
  
*These modifications primarily affect the animations and should be compatible with other werewolf mods.*
