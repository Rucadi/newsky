# Random parry and block idle
- Author: 5poiler
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/75440


This will add randomized parry animation and block idle for 1hm one handed sword, only for 3rd person view.
  
For other weapon parry animation - 2HM greatswords, 2hm battleaxe, DW dualwield, or first person parry check here [Blocking Animation Pack](https://www.nexusmods.com/skyrimspecialedition/mods/4352) . Install with nmm or manual, requirements [Dynamic Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/33746)
