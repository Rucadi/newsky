**Description**
  
This mod adds sound to dodge animation. It is script free.
  
  
Roll dodge sound record from Code Vein.
  
Step dodge SFX from [here](https://www.youtube.com/watch?v=OsSXJXk-whc)
  
If you don't like it [this mod](https://www.nexusmods.com/skyrimspecialedition/mods/28567?tab=description) has other roll dodge sounds,feel free to replace my sound file.
  
  
~~Highly recommend merge my esp ﻿plugin with Tk dodge esp plugin.~~(new)0.2 version added new Sound Marker : TKRollDodge
  
                                                                            TKStepDodge
  
Both Sound Markers are use for [Soundplay.] annotation in hkx file. For example: Soundplay.TKRollDodge
  
  
**Installation****[b]Required [tk dodge](https://www.nexusmods.com/skyrimspecialedition/mods/15309)or [tk dodge re](https://www.nexusmods.com/skyrimspecialedition/mods/56956)**[/b]
  
  
**Uninstallation**
  
This mod is script free.
  
The ESPplugin only have sound records.
  
Just delete my files in your mod manager.
