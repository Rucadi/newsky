# Dynamic Random Spell Idle
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/77025


  
 
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/77025/77025-1665635523-1581801213.gif)![](https://staticdelivery.nexusmods.com/mods/1704/images/77025/77025-1665635541-856862190.gif)
  
  
  
When the spell is equipped, the idle animations are 6 each for the right and left hand.
  
idle : 6
  
equip : 3
  
unequip : 4
  
swap : 3 
  
  
  
Much improved animation than the existing "[Smooth Random Magic idle Animation](https://www.nexusmods.com/skyrimspecialedition/mods/49853)" mod
  
  
Added random equip, unequip, swap animations
  
  
Do not use with "[Smooth Random Magic idle Animation](https://www.nexusmods.com/skyrimspecialedition/mods/49853)" Even if used together, the "Dynamic Random Spell Idle" mod will overwrite everything.
  
  
Condition
  
- Does not apply to stealth.
  
- Does not apply when wearing a weapon or shield in the right or left hand.
  
  
Mods to use with weapons
  
  
[SpellSword Moveset](https://www.nexusmods.com/skyrimspecialedition/mods/69736)
  
  
Version
  
- Version available only for players and teammates
  
- A version that can be used by both players and NPCs
