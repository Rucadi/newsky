**Mod Overview**
  
  
This mod distributes the movesets from the original *NPC Variety Guide* in a FOMOD installer, with all animations converted to the ***[[b]Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)**[/b] **(OAR)*** format. Conditions have been optimized to dynamically adapt to each user's load order through the ***[[b]Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469)**[/b] **(OCF)***.
