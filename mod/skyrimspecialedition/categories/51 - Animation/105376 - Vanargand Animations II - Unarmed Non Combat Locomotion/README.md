# Vanargand Animations II - Unarmed Non Combat Locomotion
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105376


[size=3]This covers the idle, turns, walks, runs and strafes in all directions, sprints, equip and unequip animations with fists out/unsheathed with no weapons equipped (Unarmed) and while out of combat/during exploration, which means that your existing unarmed stance/movement animations will still play while in combat.
  
  
The animations apply to both the player character and NPCs.
