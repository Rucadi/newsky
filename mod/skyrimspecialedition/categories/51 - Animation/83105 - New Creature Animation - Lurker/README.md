# New Creature Animation - Lurker
- Author: TongDing
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/83105


This mod adds new attack animations for lurker. Also apply to monster mods using lurker skeleton (such as Mihail Bone Colossus).
  
  
  
  
  
Lurker still do vanilla attacks, but when health is low, he will charge with rage.
  
  
  
For modders, you can use these animations if you want to enhance monsters  with lurker skeleton.
