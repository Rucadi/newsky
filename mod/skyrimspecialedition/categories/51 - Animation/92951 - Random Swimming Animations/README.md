# Random Swimming Animations
- Author: Gabs
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/92951


This mod uses Open Animation Replacer (OAR) to choose between 3\* different animations when swimming. Animations are configured to be randomly selected at a 33% chance.
  
  

**Animations Used**[left][/left]

Stronger Swimming Animation SE by Artsick
  
Stronger Swimming Animation SE - No Kick by Artsick
  
Swimming Animations. by Gunslicer

**Installation**
  
Install [Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109) and it's requirements.﻿
  
Install this mod with your desired mod manager
  
It should work straight away, jump in some water to test it,
  
every time you come to a rest and move it should cycle again.
