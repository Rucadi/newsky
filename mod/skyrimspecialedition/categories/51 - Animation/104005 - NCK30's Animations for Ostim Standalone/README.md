# NCK30's Animations for Ostim Standalone
- Author: NCK30 Magink
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/104005


31 sets of High Quality and Creative sex animations made by NCK30, with each set containing 5+ animations. Now ported to Ostim Standalone. 
  
  
[NCK30's Blog](https://www.loverslab.com/files/file/6878-nck30-animations/) with video previews of these amazing animations.
  
  
  
**Installation:**1. Install through your mod manager.
  
2. Refresh your Nemesis.
  
  
  
**Navigation:**You can find NCK30's animations in -> Go to Straight Idle -> NCK30fmHub.
  
  
  
**Suggestions:**Some misalignment is to be expected considering everyone is using different bodies and shapes. Use alignment tool to re-align instead of complaining.
  
DO NOT harass NCK30 for any problems in this conversion. Ask questions in the comments. 
  
It is my first time converting Sexlab animations to Ostim Animations. Let me know if there is any problems and I hope you enjoy OStim.
