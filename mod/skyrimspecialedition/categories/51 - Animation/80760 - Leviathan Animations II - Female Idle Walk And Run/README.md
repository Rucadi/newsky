# Leviathan Animations II - Female Idle Walk And Run
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/80760
Remake from scratch of my first animation mod [size=3][Leviathan Animations - Female Idle Walk And Run](https://www.nexusmods.com/skyrimspecialedition/mods/42204)﻿ released just over 2 years ago using a more powerful [size=3]rig made by [/size][Kars](https://www.nexusmods.com/skyrimspecialedition/users/59603001) and different animating techniques I've learned since then, to better reflect an intimidating and heavily armored character locomotion, while also preventing the long standing issues with camera transitions from occurring.
  
  
[/size]This can be installed either as a simple replacer with no requirements, or as a "Conditional Armor Type Animations" addon which can be used alongside other CATA addons to change the actors' animations depending on their armor type, b[size=3]oth versions will affect both the player character and NPCs, [size=4][size=3]the male version is ~~coming [size=4][size=3]soon~~[/size] [available here](https://www.nexusmods.com/skyrimspecialedition/mods/81463).[/size][/size][/size]
