# Igniting Animation for Campfire
- Author: Elsawirr and GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/83142


A mod that adds an animation for lightning the fire when using the "strike stone" option from [Campfire - Complete Camping System.](https://www.nexusmods.com/skyrimspecialedition/mods/667)
