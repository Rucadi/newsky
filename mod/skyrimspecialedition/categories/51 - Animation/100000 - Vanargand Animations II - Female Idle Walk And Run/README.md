# Vanargand Animations II - Female Idle Walk And Run
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/100000
Complete remake of the old [Vanargand Animations - Female Idle Walk and Run](https://www.nexusmods.com/skyrimspecialedition/mods/51681/) on Kars' rig, this should more accurately represent the feel I was trying to achieve when I first released the original mod over 2 years ago.
  
  
Can be installed either as a simple replacer with no requirements (make sure no OAR/DAR based mod replaces your locomotion animation if you choose this option), or as a "Conditional Armor Type Animations" addon which can be used alongside other CATA addons to change the actors' animations depending on their armor type, b[size=3]oth versions will affect both the player character and NPCs, this mod is now also included in my [All-In-One Collection](https://next.nexusmods.com/skyrimspecialedition/collections/oxtos9?_gl=1*1lhbvwj*_ga*OTc3NDEwMjMuMTY1NzUzMTYyNA..*_ga_N0TELNQ37M*MTY5MzkzNDc2Ni4xOTM4LjEuMTY5MzkzNDkzNC4wLjAuMA..) to replace the Vanargand female animations.
