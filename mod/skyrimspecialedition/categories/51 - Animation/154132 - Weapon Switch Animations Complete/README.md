# Weapon Switch Animations Complete
- Author: RavenKZP
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/154132


Mod Description
  
  
This mod aims to fill the gaps left by the base game and Weapon Styles - Draw-Sheathe animations, by adding complete and functional draw/sheathe animations for a wide variety of weapon and hand combinations.

Immersive Weapon Switch Support:
  
This mod includes built-in contextual animation support.
  
Smart behaviour: When changing the weapon in your left hand, the right hand won’t play unnecessary sheathe/draw animations — making it look cleaner and more realistic.

Installation
  
  
Install this mod.
  
Run [Nemesis](https://www.nexusmods.com/skyrimspecialedition/mods/31667) or [Pandora](https://www.nexusmods.com/skyrimspecialedition/mods/133232) 
  
Make sure that **Weapon Switch Animations Complete** is checked.
  
Done.
  
  
Compatibility.
  
  
It's compatible with everything.
