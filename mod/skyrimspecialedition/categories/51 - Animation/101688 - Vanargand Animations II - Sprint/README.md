# Vanargand Animations II - Sprint
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/101688


Complete remake of the old [Vanargand Animations - Sprint](https://www.nexusmods.com/skyrimspecialedition/mods/53381) on Kars' rig.
  
  
Can be installed either as a simple replacer with no requirements (make sure no OAR/DAR based mod replaces your locomotion animation if you do choose this option), or as a "[Conditional Armor Type Animations](https://www.nexusmods.com/skyrimspecialedition/mods/51507)" addon which can be used alongside other CATA addons to change the actors' animations depending on their armor type, b[size=3]oth versions will affect both the player character and NPCs, this mod is now also included in my [All-In-One Collection](https://next.nexusmods.com/skyrimspecialedition/collections/oxtos9?_gl=1*1lu1cfl*_ga*OTc3NDEwMjMuMTY1NzUzMTYyNA..*_ga_N0TELNQ37M*MTY5NjE4MDMwNS4yMDUzLjEuMTY5NjE4MDUyNS4wLjAuMA..) to replace the Vanargand Sprint animations when wearing **light armor**.
