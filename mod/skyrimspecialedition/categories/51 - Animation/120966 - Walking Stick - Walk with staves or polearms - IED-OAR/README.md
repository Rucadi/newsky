# Walking Stick - Walk with staves or polearms - IED-OAR
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/120966


**[font=Georgia]INTRODUCTION[/font]**

  
Utilizing Immersive Equipment Displays, this mod will add new nodes to carry staves or two-handed polearms on the right or left hands, with some animations included.
  
  

**[font=Georgia]REQUIREMENTS[/font]**

  
  
- [Immersive Equipment Displays](https://www.nexusmods.com/skyrimspecialedition/mods/62001)
  
- [Simple Dual Sheath](https://www.nexusmods.com/skyrimspecialedition/mods/50049)
  
- [Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)
  
- [Open Animation Replacer - IED Conditions](https://www.nexusmods.com/skyrimspecialedition/mods/98308)
  
**Optional:**
  
- [Immersive Equipment Displays - Extra Skeleton Nodes](https://www.nexusmods.com/skyrimspecialedition/mods/92244): To use the Walking Stick from [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667).
