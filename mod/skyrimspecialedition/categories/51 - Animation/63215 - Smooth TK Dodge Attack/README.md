# Smooth TK Dodge Attack
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63215


  
  
  
  
  
  
Normally, normal dodge is used, but if you dodge during an attack, it will attack at the same time as dodging.
  
  
  
  
  
The DAR condition folder number is 70120.
  
  
If you use another evasion related mod with a higher number than that number, it won't work.
  
  
  
  
Available in [TK Dodge SE](https://www.nexusmods.com/skyrimspecialedition/mods/15309) or [TK Dodge RE - Script Free](https://www.nexusmods.com/skyrimspecialedition/mods/56956).
  
  
If you are using [DAR Combat Dodge Animation](https://smooths.tistory.com/24) mode, arrange it below that mod.
