# Vanargand Animations - One Handed Power Attacks
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/58997
This is a vanilla replacer version, to be used as is, or to be placed in your folder structure of choice (Nemesis PCEA, Combo/Stances Animation Framework, etc...) and contains the following One-Handed animations:
  
  
-Standing power attack.
  
-Forward power attack.
  
-Backward power attack.
  
-Left power attack.
  
-Right power attack.
  
  
These animations have been adjusted throughout the past weeks to account for and avoid clipping with most types of equipment (or the lack thereof) on the left hand without compromising the animation quality.[size=3]
  
My next animation project will mainly be focusing melee dual wielding.[/size]

* [Animation Motion Revolution](https://www.nexusmods.com/skyrimspecialedition/mods/50258)﻿: Allows each animation's movement to be driven by it's own custom motion data, without this, you will experience sliding while attacking. ~~For Anniversary Edition users: Until AMR is updated to support AE, these animations will sadly not be able to use the custom motion data~~. Fully compatible with Anniversary Edition.
