# No Spinning Death Animation SE
- Author: dDefinder
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/1432
Removed certain creature death pause animations which cause them to lose all momentum, animation snap upon dying, and have no physical push based on your attacks. Includes dragons, falmer, riekling, wolf, spider, giant, horker, bear and spriggan. The generic standing non-killmove dragon death animation is removed.
