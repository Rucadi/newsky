[/font][/b]This mod includes **3** new Dynamic **Ledge Sitting Idles. [LE Version](https://www.nexusmods.com/skyrim/mods/115412)﻿﻿**﻿﻿
  
That will play on female npcs depending on outfits(or lack thereof) worn
  
when Idling at npc **Idle Markers**. **Idle Markers** are normally only accessible to npc's
  
in vanilla skyrim. To use these animations as **player** you will need the mod
  
﻿[**Dovahkiin can lean**](https://www.nexusmods.com/skyrimspecialedition/mods/32748) which allows players access to **npc idle** markers.
  
  
Looking for somewhere to try out the animations?
  
Try out my location mod [Volkihar's Descent EVGAT](https://www.nexusmods.com/skyrimspecialedition/mods/90529)﻿
  
*or the optional* 
  
COTN addon for [Cities of the North Winterhold](https://www.nexusmods.com/skyrimspecialedition/mods/40088)﻿
