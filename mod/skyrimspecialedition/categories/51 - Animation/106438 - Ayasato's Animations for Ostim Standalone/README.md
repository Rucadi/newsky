# Ayasato's Animations for Ostim Standalone
- Author: Ayasato Magink
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106438


5 sets of High Quality sex animations and 3 sets of furniture sex animations made by Ayasato. Each set has 5+ animations.
  
[Ayasato's Loverslab page](https://www.loverslab.com/files/file/6249-slal-ayasato-animations-28022018/) for more preview.
  
  
  
**Installation:**
  
1. Install through your mod manager.
  
2. Remove your nemesis cache, then refresh.
  
  
  
**Navigation:**
  
You can find Ayasato's animations in -> Go to Straight Idle -> AyasatofmHub.
  
Furnitures are Chair and Table.
  
  
  
**Suggestions:**
  
Some misalignment is to be expected considering everyone is using different bodies and shapes. Use alignment tool to re-align instead of complaining.
