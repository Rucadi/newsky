# Disable Turn Animation SE I AE
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/78106


  
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/78106/78106-1740027575-2068772556.gif)
  
  
Removes character turn animation when rotating the screen.
  
  
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/78106/78106-1740027567-865175855.gif)
  
  
Idle and movement-related turn animations used by NPCs are not removed.
  
  
  
  
  
  
**Version 1**

  
* **Requirements :** None
Most turn animations are removed, but there are three cases where a brief turn animation may still appear:[list]
  
* When rotating the character while drawing a weapon.
When rotating the character right after stopping movement.
  
* When lock-on an enemy, TDM ([True Directional Movement](https://www.nexusmods.com/skyrimspecialedition/mods/51614)﻿) users will see a brief turn animation for their character depending on the enemy's position.

  
[/list]
  
  
[line]
  
  
**Version 2**

  
* **Requirements:**[list]
[Behavior Data Injector](https://www.nexusmods.com/skyrimspecialedition/mods/78146)﻿ (SE/AE)
  
* [Behavior Data Injector Universal Support](https://www.nexusmods.com/skyrimspecialedition/mods/78159)﻿ (AE) /
[Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)﻿ (SE/AE)
  

  
[\*]All turn animations are completely removed.
  
[\*]The three cases in **Version 1** where a brief turn animation could still occur are also eliminated.
  
[/list]
  
  
  
  
AE users will need to install both [Behavior Data Injector](https://www.nexusmods.com/skyrimspecialedition/mods/78146) and [Behavior Data Injector Universal Support](https://www.nexusmods.com/skyrimspecialedition/mods/78159)﻿.
