# Stand still in RaceMenu (OAR)
- Author: CHIMgarden
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/96970


Quality of life mod for everyone who uses my [Male Player Animations](https://www.nexusmods.com/skyrimspecialedition/mods/89225) and [Female Player Animations](https://www.nexusmods.com/skyrimspecialedition/mods/85073) idle animations or something similar :)
  
This OAR mod simply activates still pose while your character is in RaceMenu and deactivates when you exit. Choose between 4 versions: with completely motionless mt\_idle animation, hands on hips, A-pose and with vanilla idle (imho too boring for actual gameplay, but good for making cosmetic adjustments and not so mannequin-like compared to the first variant).
  
This mod is compatible with all other mods!
  
  
If you use [Show Player in Menus](https://www.nexusmods.com/skyrimspecialedition/mods/81291) and want player character to stand still while inventory is open, add optional file. Download, install with mod manager, place after any version of main download or merge with main download to overwrite standard config file it has.
  
  
  
  
To have this pose activated instantaneously your idle mods in Open Animation Replacer have to have Interruptible box checked in submod settings. My updated [Male Player Animations](https://www.nexusmods.com/skyrimspecialedition/mods/89225) and [Female Player Animations](https://www.nexusmods.com/skyrimspecialedition/mods/85073) idles have it configured. But if you are using other idle mods that may not have this check mark in place, you will have to wait untill any of those animations is played until the end to see still pose to be played after. Simply make Interruptible box checked in OAR for all other idle mods you have and this mod will work as I intended.
