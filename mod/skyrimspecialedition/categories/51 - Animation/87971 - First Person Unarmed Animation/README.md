# First Person Unarmed Animation
- Author: 5poiler
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/87971


First person unarmed animation in vanilla is generalize by using 1hm  sword animation file as default in certain area. This wll change that  for
  
1. Idle, run walk
  
2. Random blocking
  
3. Dodge
  
  
  
  
Video show idle, walk, run and 5  block stance, for combat video show dodge, bash and parry here <https://m.youtube.com/watch?v=NYpk0jl_0MM>
  
Note for first person dodge, all other weapon is adjusted and visible now while dodge in 1st person view
  
Other first person animation mod use in video
  
[Random first person magic animation SSE with DAR](https://www.nexusmods.com/skyrimspecialedition/mods/61188)
  
[Enhanced 1stperson Parry Animation for SSE](https://www.nexusmods.com/skyrimspecialedition/mods/20374)
  
Requirements:
  
[Dynamic Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/33746)
  
[TK Dodge RE - Script Free](https://www.nexusmods.com/skyrimspecialedition/mods/56956)
