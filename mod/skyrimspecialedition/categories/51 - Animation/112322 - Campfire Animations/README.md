# Campfire Animations
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/112322


**[font=Georgia]INTRODUCTION[/font]**

  

I always thought it was a bit boring to stare at the campfire while crafting the items, so I came up with some small animations for it.

  

**[font=Georgia]REQUIREMENTS[/font]**

  

- [Campfire](https://www.nexusmods.com/skyrimspecialedition/mods/667) (can work without it)
  
- [Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)
  
- [FormList Manipulator](https://www.nexusmods.com/skyrimspecialedition/mods/74037)
  
**Optional:**
  
- [Immersive Equipment Displays](https://www.nexusmods.com/skyrimspecialedition/mods/62001): To see the items during the animations.
  
- [PapyrusUtil](https://www.nexusmods.com/skyrimspecialedition/mods/13048): To get better placements for the items during the animations.
  
- [MCM Helper](https://www.nexusmods.com/skyrimspecialedition/mods/53000): To use the MCM.

**[font=Georgia]Q&A[/font]**

**Q: ESL flagged?**
  
*A: Yes.*
  
  
**Q: How to add the animations to other campfire mods?**
  
*A: If the mod uses the **isKneeling** keyword for the furniture (makes character kneel down when interacting) you can distribute the keyword **CFAnim\_Keyword** to it, via plugin or [KID](https://www.nexusmods.com/skyrimspecialedition/mods/55728) config file.
  
Then you can check the FormLists from this mod and add the items to play the animations when necessary.*
  
  
**Q: Dynamic Animation Replacer?**
  
*A: This mod uses a feature that is exclusive to OAR. I'm sorry, but it won't work with DAR.*
  
  
**Q: Compatibility?**
  
*A: If using [Hunterborn Animations](https://www.nexusmods.com/skyrimspecialedition/mods/80677) or [Immersive Hunting Animations](https://www.nexusmods.com/skyrimspecialedition/mods/96961), disable the cooking animation for Campfire in their respective MCMs.
  
Otherwise, should be compatible with everything else. Leave a comment if you found an incompatibility.*
  
  
**Q: I don't like x feature?**
  
*A: Use the MCM to customize the mod.*
  
  
**Q: I have camera issues with the animations?**
  
*A: Use the MCM and toggle "No camera" option or use this console command: Set CFAnim\_EnableNoCamera to 1.*
