**Subtle Feminine Carriage Sitting Animation**
  
**By DenSixSeven and****Xtudo**

  
**Description**
  
**05 new lovely female carriage sitting animations.**
  

  
* They're a very subtle feminine versions.
01 new animation to get on the carriage. 03 new randomized sitting animations. 01 new animation to get out the carriage.
  
* The intention is to give more variation to the game.
DAR and OAR versions available, install only one.
  
* Optional**Both Genders version** available. It includes the new animations and the vanilla version as well, so it's not lost.
No ESP.
