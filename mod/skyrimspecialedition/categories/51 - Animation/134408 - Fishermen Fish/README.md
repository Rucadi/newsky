# Fishermen Fish
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/134408


**[font=Georgia]INTRODUCTION[/font]**

New fishing animations for some Fishermen NPCs in the game.

  

**[font=Georgia]REQUIREMENTS[/font]**

  
- [Fishing](https://en.uesp.net/wiki/Skyrim:Fishing)
  
- [Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)
  
- [AnimObject Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/75167): Optional, to replace some fish and fishing rods during the animations

**[font=Georgia]IMPORTANT[/font]**

  
 This mod will add new fishing animations to some fishermen around Skyrim:
  
- Two fishermen at Lake Ilinata.
  
- One fisherman near the Standing Stones (starting area).
  
[size=4]- One fishing spot at Riften's Docks, for the Fishery Faction.
  
- Brutius in Morthal.
  
- Klimmek in Ivarstead.
  
- Lynea at Markarth's Docks.
  
- Jolf at Solitude's Docks.
  
- Slay-Many-Beasts from [Whispers of the Depths - Quest Mod](https://www.nexusmods.com/skyrimspecialedition/mods/127087), if using the optional file.
  
  
The mod will add a furniture that NPCs can use. The furniture will be used at random times.
  
If you spawn the furniture in the world, other NPCs can use it and fish. **FF\_FishingFurniture [FURN:XXXXX804]**
  
  
There's an [IED](https://www.nexusmods.com/skyrimspecialedition/mods/62001) preset to hide the fishing rod weapon while the NPCs are fishing, Merge it via Gear nodes window:
