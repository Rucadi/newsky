# Dynamic Block Hit
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/100570


  
  
  
This mod adds animations for each weapon to [Maxsu Block Overhaul](https://github.com/max-su-2019/MaxuBlockOverhaul/releases) and separates normal attacks and power attacks to operate block hit animations.
  
  
and random block hit animations for each weapon group, and additional special block hit animations.
  
  
  
  
***Updates since video***
  
1.5
  
- The groggy block hit activation condition has been changed to health below 30% or stamina below 30%.
  
- Change condition folder name
  
- Same applies to humanoid enemies (normal attack block hit, power attack block hit, groggy block hit)
  
- converted to ESL
  
- OAR support
  
  
1.6
  
- Add block hit sound
