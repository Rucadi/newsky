# Followers Salute Casually (OAR)
- Author: CHIMgarden
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/95093


7 new much less formal idlesalute.hkx gesture animations for your followers to use when your character addresses them while using [Immersive Interactions - Animated Actions](https://www.nexusmods.com/skyrimspecialedition/mods/47670). Without said mod my animations from this mod won't be showning in your game!
  
  
I made this small mod for myself to fix mismatch between informal dialogue lines followers use so often and too formal standard idlesalute used by Immersive Interactions. Now they will have a variety of head nodding to signify attentive listening. Housecarls will have old fist to chest animation though, but not too often. 
  
  
Females now have animations fitting their skeleton, instead of using bulky universal idlesalute.
  
  
  
How to install
  
At any time in your playthrough add main mod archive to your mod manager and follow it's standard installation process. Don't forget to make files active :)
  
  
How to uninstall
  
At any time in your playthrough remove mod files in your mod manager.
  
  
How to update
  
At any time in your playthrough download new main file version and replace all files with a new version files.
  
  
Compatibility
  
Must be compatible with everything because there are simply no other mods to replace this animation :)
