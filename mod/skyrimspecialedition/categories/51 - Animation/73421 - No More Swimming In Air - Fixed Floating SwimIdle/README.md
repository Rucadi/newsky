# No More Swimming In Air - Fixed Floating SwimIdle
- Author: Exalderan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/73421


![](https://willeshaben.de/skyrimmods/Projectenhancedvanillaexperience.png)﻿

**Details**
  
This mod fixes the SwimIdle Animation by fixing the angle of the bones as well as relocating the whole animation lower on the z-axis.
  
It really was an eyesore far too long.
  
  
  
  
  
![](https://willeshaben.de/skyrimmods/Compatibility.png)
