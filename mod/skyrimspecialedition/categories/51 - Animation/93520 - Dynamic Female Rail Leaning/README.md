This mod includes **3** new Dynamic **Rail Leaning Idles.**﻿﻿ [LE﻿ Verson](https://www.nexusmods.com/skyrim/mods/115745)﻿﻿﻿﻿
  
That will play on female npcs depending on outfits(or lack thereof) worn
  
when Idling at npc **Idle Markers. Idle Markers** are normally only accessible to npc's
  
in vanilla skyrim. To use these animations as **player** you will need the mod
  
﻿[Dovahkiin can lean](https://www.nexusmods.com/skyrimspecialedition/mods/32748) which allows players access to **npc idle** markers.
  
  
**Integrated support for my**
  
[Sonders Keyword Distribution Resources](https://www.nexusmods.com/skyrimspecialedition/mods/93995)
  
"Above Detects Heels and restricts Elder NPCS via Keywords﻿"
  
  
**Alternatively includes support for**
  
[High Heel Condition Spell](https://www.nexusmods.com/skyrimspecialedition/mods/76899)
  
SpoilerBoth Heel solutions work best with about 2inch tall heels
