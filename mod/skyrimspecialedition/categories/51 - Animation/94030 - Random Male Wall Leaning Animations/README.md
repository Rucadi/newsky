[b]Description**
  

  
* Adds 4 different leaning animations for male player/NPCs.
Inspired by Sonderbain's [Dynamic Female Leaning](https://www.nexusmods.com/skyrimspecialedition/mods/87536?tab=description)﻿ and RDR2.
