# Thundertrot Horse Animations
- Author: b13nxx
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140941


Yes, you heard me right. This is THE MOD with brand new Walk, Trot and Gallop animations of the horse without trying to fix the existing animations in the game. It will directly replace their vanilla counterparts.

**Installation**
  
You can install this mod via MO2 (which is recommended) or manual. If you want to go with manual option instructions are:
  

  
* First download the mod manually.
Extract rar file anywhere you want.
  
* Open Skyrim main folder.
Move the folder you just extracted to game's main folder.
  

  
  
**Uninstallation**
  
Remove the mod via MO2 or manually from the game's main folder.
