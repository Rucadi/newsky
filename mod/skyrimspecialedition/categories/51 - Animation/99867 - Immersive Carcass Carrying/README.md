# Immersive Carcass Carrying
- Author: Elsawirr
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/99867
  
  
[font=Arial]Basically it makes the same thing with my [/font][font=Arial][old mod](https://www.nexusmods.com/skyrimspecialedition/mods/65501), but this time it has running animations so no need for esp. file or carrying weight penalty. Also it uses [OAR](https://www.nexusmods.com/skyrimspecialedition/mods/92109) so it has less compatibility issues now thanks to [GiraPomba](https://www.nexusmods.com/skyrimspecialedition/users/20507499).
  
  
[/font]
  
Installation
  
  

  
* [size=4]Install [OAR](https://www.nexusmods.com/skyrimspecialedition/mods/92109), won't work without it
  
Install [IED](https://www.nexusmods.com/skyrimspecialedition/mods/62001)﻿, won't work without it
  
- As I edited IED presets you need to decide which version are you going to use:
  
If you are using [CYC](https://www.nexusmods.com/skyrimspecialedition/mods/62628) ([Hunterborn](https://www.nexusmods.com/skyrimspecialedition/mods/7900)), go with **ICC - Hunterborn**
  
  
If you are using ***only*** ﻿[Immersive Hunting Animations](https://www.nexusmods.com/skyrimspecialedition/mods/96961), go with ICC - ﻿Immersive Hunting Animations. This has no preset because "﻿Immersive Hunting Animations" mod already include the preset needed.
  
  
If you are using [Simple Hunting Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/95943), go with **ICC - Simple Hunting Overhaul**
  
  
  
If you are using [SHO](https://www.nexusmods.com/skyrimspecialedition/mods/95943)﻿ and [Immersive Hunting Animations](https://www.nexusmods.com/skyrimspecialedition/mods/96961) together, you can still go with **ICC - Simple Hunting Overhaul**
  
  

  
* Download and install the version you choose through your mod manager.

  

  
* In-game open IED tab(backspace by default), click "Display manager - Custom" and find the preset according to which version you are using.

  

```
﻿![](https://i.imgur.com/Nz2ZBsc.png)
```

  
* Apply the preset and that's all, enjoy!!

[/size]
