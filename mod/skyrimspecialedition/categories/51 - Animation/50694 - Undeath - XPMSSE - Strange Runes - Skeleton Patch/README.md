# Undeath - XPMSSE - Strange Runes - Skeleton Patch
- Author: Maskroz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/50694
**What this mod does**Patches the skeleton from [XPMSSE](https://www.nexusmods.com/skyrimspecialedition/mods/1988) to work with [Undeath](https://www.nexusmods.com/skyrimspecialedition/mods/6180) and [Strange Runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456)Without this patch the camera and the runes will end up in the ground.
  
  
**Recommended mods**
  
[Undeath - Classical Lichdom](https://www.nexusmods.com/skyrimspecialedition/mods/40802): Fixes sevaral things from Undeath and greatly enhances what was already there.
  
[Strange Runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456): Makes your spellcasting fancier. 
  
  
﻿**Compatibility**
  
﻿Compatible with everything unless it overwrites the dragon priest skeleton
  
Meshes/Actors/dragonpriest/character assets/skeleton.nif
  
  
﻿**Installing**
  
1. Download and install using your preferred mod manager
  
2. Make sure to load Undeath and/or Strange Runes after XPMSSE
  
3. Let this patch overwrite what it wants
