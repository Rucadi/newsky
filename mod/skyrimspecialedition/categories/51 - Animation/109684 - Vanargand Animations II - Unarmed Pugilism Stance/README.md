# Vanargand Animations II - Unarmed Pugilism Stance
- Author: Verolevi
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109684


Hand-to-hand stance animations inspired by "[Classical Pugilism](https://en.wikipedia.org/wiki/Bare-knuckle_boxing)", covers the idle, turns, walks, strafes, runs and sprints while unarmed,
  
 the character will keep facing the crosshair no matter their movement direction, if used alongside "[Vanargand Animations II - Unarmed Non Combat Locomotion](https://www.nexusmods.com/skyrimspecialedition/mods/105376)", the Pugilism stance will only play during combat.
  
  
[size=3]The animations apply to both the player character and NPCs.
