# Conditional tavern cheering (OAR)
- Author: CHIMgarden
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63029


[left]This mod adds variety to animations of cheering and applause![/left]

  
* Works with NPC added by mods
You can install/delete/reinstall this mod at any time
  
* Install like any other OAR or DAR mod :)
Compatible with [(DAR)More Noemic drink animations SE](https://www.nexusmods.com/skyrimspecialedition/mods/61212). Use patch for it from optional files
  

  
My other DAR mods: [Immersive folded hands](https://www.nexusmods.com/skyrimspecialedition/mods/62849?tab=description) | [NPC Animation Remix](https://www.nexusmods.com/skyrimspecialedition/mods/63471) | [Gesture Animation Remix](https://www.nexusmods.com/skyrimspecialedition/mods/64420)
  
  
*If during/after installation some sort of error pops up or animations in your game can`t play and you see static pose instead:*
  
Mod Organizer 2 users - [color=#d9ead3][size=3]rename mod folder in your mod manager to "CTC OAR" and it will fix this
  
Other mod manager users - extract the zip file and rename the "Conditional tavern cheering (OAR)" named folder to "Data", repack and install[/size][/color]

-1001015 -- 0.12 applause very happy variant from npc\_applaud4
  
-1001016 -- 0.42 applause variant from npc\_applaud2
  
-1001017 -- 0.80 applause variant from npc\_applaud5
  
-1001018 -- replacer of basic sarcastic applause for females
  
-1001019 -- 0.5 cheering drinking animation variant (swaying) for all NPC
