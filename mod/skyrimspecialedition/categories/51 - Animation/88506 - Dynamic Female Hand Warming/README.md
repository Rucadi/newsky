[/font]**This mod includes **6** new Dynamic **Hand Warming Idles.** [**LE Version**﻿](https://www.nexusmods.com/skyrim/mods/115200)﻿
  
That will play on female npcs depending on outfits(or lack thereof) worn
  
when Idling at npc **Idle Markers**. **Idle Markers** are normally only accessible to npc's
  
in vanilla skyrim. To use these animations as **player** you will need the mod
  
[Warming Hands](https://www.nexusmods.com/skyrimspecialedition/mods/73581)﻿which allows players to idle at any **heat source**Their mod also provides a **warmth effect** for survival mods like [Sunhelm](https://www.nexusmods.com/skyrimspecialedition/mods/39414)﻿
  
  
"**Update 1.3** *added 3 new animations. Armor type animation conditions also returned*
  
**This mod now has 3 Standing and 3 Crouched Idle animations**
  
*To use crouched Idles as player simply crouch before Idling*"
