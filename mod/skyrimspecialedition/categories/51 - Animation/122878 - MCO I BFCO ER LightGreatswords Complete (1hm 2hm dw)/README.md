# MCO I BFCO ER LightGreatswords Complete (1hm 2hm dw)
- Author: black
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/122878


Players only, NPCs can't use this, if you want NPCs to use it, download [the SCAR version of the light greatsword
  
  
Usage:
  
Install and use directly, play different animations according to different conditions, if the animations don't work, open your OAR UI and increase the folder number priority to bigger, make sure that there are no other animations to cover them up!](https://www.nexusmods.com/skyrimspecialedition/mods/122801)
  

Affects the animation of all greatsword weapons
