# Anub Big Guy Pack for OStim Standalone
- Author: Anub VersuchDrei
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/120512


A port of Anubs big guy animations to OStim Standalone. Load order doesn't matter since there should be no file conflicts.
  
  
I know these are already included in OStimTools version of Anubs pack, but this was done at the request of one of my patrons since he prefers the manually created navigation.
  
  
Only works with Nemesis. There is no FNIS version.
