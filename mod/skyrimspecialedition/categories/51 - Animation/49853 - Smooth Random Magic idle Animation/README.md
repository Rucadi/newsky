# Smooth Random Magic idle Animation
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/49853
  
  
  
  
**In the meantime, all of Skyrim's NPCs looked the same without finding their personality.
  
  
You may have learned the same because there is only one college of magic, but to get rid of the monotony,
  
  
we have given life to all NPCs with natural and varied animations.**
  
  
  
  
Both players and NPCs apply.
  
  
There is no need to use FNIS or Nemesis.
  
  
Instead, it can crash if you are using magic IDLE animations with PCEA.
  
  
  
  
**2.0 Update Supports the use of weapons and magic.**
  
  
**- Added male-only animations**
  
  
**- Added individual animations for combat and non-combat situations when equipping a weapon + spell.**
  
  
**- Removed some female spell idle animations**
