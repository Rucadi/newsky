- [The XPMSSE - Fixed Scripts mod](https://www.nexusmods.com/skyrimspecialedition/mods/44252).
  
- [The XPMSSE - Nemesis - Papyrus Stack Fix mod](https://www.nexusmods.com/skyrimspecialedition/mods/60363). Specifically: version 4.8.3 OR version 4.8.4.
  
- [Nemesis](https://www.nexusmods.com/skyrimspecialedition/mods/60033)
  
- SKSE
  
- [powerofthree's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854)

Installation
  
- Same as any other standard mod, recommend installing using a Mod Manager.
  
- Note that this is not a full mod, just a patch.
  
- Clean saving should not be required, though to be 100% safe, try to create the saved game for which you first install this file in a quiet place (minimising the chances of any major scripted events being currently active).Â  
  
  
  
(In)compatibilities
  
There are many other mods out there also including various XPMSSE scripts, with various fixes in them. I built on these files, and you should install these before mine, letting mine overwrite at the end. [color=#ffff00]**Please make sure that you have these mods installed, in exactly this order** (such that later ones always overwrite earlier ones):
  
  
1) Firstly, install XPMSSE itself: <https://www.nexusmods.com/skyrimspecialedition/mods/1988>
  
2) Secondly, install XPMSSE - Fixed Scripts: <https://www.nexusmods.com/skyrimspecialedition/mods/44252>
  
3) Thirdly, install XPMSSE - Nemesis - Papyrus Stack Fix: <https://www.nexusmods.com/skyrimspecialedition/mods/60363> (**version 4.8.3 OR version 4.8.4**)
  
4) Finally, install my mod.[/color]
