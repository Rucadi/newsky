# First Person Animations - For some mods
- Author: GiraPomba
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148289


**[font=Georgia]INTRODUCTION[/font]**

  
This mod will add some new first person animations to mods that usually force third person perspective. Useful (and immersive) if you're a first person exclusive player and don't like forced camera transitions.
  
  

**[font=Georgia]REQUIREMENTS[/font]**

  
- [Offset Movement Animations](https://www.nexusmods.com/skyrimspecialedition/mods/110408)
  
- [Open Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/92109)
  
- **And the mod of your choice**.
  
  
  

**[font=Georgia]MODS SUPPORTED[/font]**

  
**Equipping armor**:
  
 - [Immersive Equipping Animations](https://www.nexusmods.com/skyrimspecialedition/mods/77187)
  
  
**Religion**:
  
 - [Pilgrim - A Religion Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/54099)
  
  
**Call mount**:

 - [Press H to Horse](https://www.nexusmods.com/skyrimspecialedition/mods/81195)
  
  
**Crafting**:

**[font=Georgia]Q&A[/font]**

**Q: Improved Camera?**
  
*A: Compatible with upcoming 2.0 version.*
  
  
**Q: What if I don't want to use Nemesis or Offset Movement Animations?**
  
*A: The animations won't play, so you can still use the mod to avoid forced third person perspective I guess.*
  
  
**Q: What about this mod and that mod?**
  
*A: If you have a mod that adds animations and they could work in first person, leave a comment and I'll take a look.*
