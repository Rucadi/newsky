# Dynamic Sprint
- Author: lSmoothl
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/95561


Dynamic Sprint 1.0
  
  
  
 
  
 
  
  
  
 
  
Dynamic Sprint 1.1
  
  
  
  
  
  
Added a pre-sprint animation when sprinting starts.
  
 
  
**At the start of a sprint, it will switch to sprint after a dash.**
  
Added sprint animations for combat and non-combat situations for each weapon
  
  
Dynamic sprint pursues a different direction from [Random Sprint Animation](https://www.nexusmods.com/skyrimspecialedition/mods/46193).
  
  
While the [Random Sprint Animation](https://www.nexusmods.com/skyrimspecialedition/mods/46193) mod only pursued diversity, the dynamic sprint pursues animations suitable for each situation for each weapon.
  
  
  
Even if used with [Random Sprint Animation](https://www.nexusmods.com/skyrimspecialedition/mods/46193), **dynamic sprint** overwrites everything.
  
  
  
  
***I made it possible to use it even if you don't use FNIS or Nemesis.***
  
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/95561/95561-1689008089-1466431420.png)
  
Based on dynamic sprint 1.0
  
  
  
  
This is a sprint animation for each weapon combination supported by dynamic sprint.
  
  
Because it is made in the structure of [Open Animation Replacer](https://www.patreon.com/posts/open-animation-0-83043829),
  
  
the user can activate and deactivate each animation directly in the in-game UI, and modify the condition.
