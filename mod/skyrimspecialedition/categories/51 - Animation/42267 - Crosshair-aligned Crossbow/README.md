# Crosshair-aligned Crossbow
- Author: Pragasette
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/42267


[font=bebas\_neuebook][Classic Edition](https://www.nexusmods.com/skyrim/mods/105457) • Special Edition﻿ • [GitHub﻿](https://github.com/pragasette/crosshair-aligned-crossbow)﻿[/font]

  
  
  
This is an animation replacer especially intended for [Immersive HUD](https://www.nexusmods.com/skyrimspecialedition/mods/12440) and hidden crosshair.
  
  
[heading]Features[/heading]
  

  
* Replace first person crossbow animations to line up with the target.
No plugin, safe to remove at any time.
  

  
  
[heading]Installation[/heading]
  
Use your mod manager of choice, [Mod Organizer 2](https://www.nexusmods.com/skyrimspecialedition/mods/6194) is recommended.
