# OSTIM - Rest and Respite
- Author: Temryuu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109507


**Information**
  
  
Includes a few new NSFW solo animations for OStim SA, all NSFW animations are intended and designed for female only, however the SFW poses that are included from my other work are included and may be used by males too however there may be misalignment or clipping if you choose to do so.
  
  
The pack is a small project of animations both NSFW and SFW as I learn animating and wished to continue learning a wider variety of skills and complex movements. The SFW pack and NSFW pack will be updated separately, however the NSFW pack will include all the content of the SFW pack so you *do not* need to download both mods if you only want the content of one.
  
  
Hold the UP arrow key for 5 seconds whilst not looking at any actors/NPCs to access the animations.
  
  
See my other mod [OSTEM - Animations](https://www.nexusmods.com/skyrimspecialedition/mods/105849) for more information and previews of the SFW animations.﻿
  
  
Requires OStim SA, please refer to [OStim SA installation guide](https://www.nexusmods.com/skyrimspecialedition/articles/5678) for a well-written guide on how to install OStim.
