# Eating Animations - Patch Collection
- Author: Mefariah
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63741


As of V2.0, this mod requires AnimObjectSwapper.
  
  
This page is intended as a living patch compendium between the beautiful Eating Animations and Sounds and other mods that either add additional foods or modify food records thus requiring patching.
