**Features:**
  
  

  
* All new hand-made jump animations based on real life references
Jump animation variations and adjustments based on worn equipment
  
* Unique animations for back jump separate from forward jump
Realistic motion adjustment to forward jump when sprinting (with [Better Jumping SE](https://www.nexusmods.com/skyrimspecialedition/mods/18967)ï»¿)
  
* Different jump landing animations depending on fall height and/or perks
Slight values adjustments to improve jump behaviors
  
* Removes sliding for certain jump landing
Recommended .ini edits for realistic fall speed
  
* Compatibility with other behavior mods through Nemesis patch
Available for both SE and LE versions
  

  
**Install/Uninstall:**
  
  

  
* Install with your preferred mod manager and follow instructions in the FOMOD.
Uninstall with your mod manager.
  

  
**Compatibility:**
  
  

  
* [Combat Gameplay Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/33767) (Full Compatibility - patch available)
[360 Movement Behavior](https://www.nexusmods.com/skyrimspecialedition/mods/33139) (Full Compatibility - patch available)
  

  
*\*Most other behavior mods should be fully compatible once patched with Nemesis Engine*
