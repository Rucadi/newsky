# Tools not weapons (pickaxe and woodcutter axe) DAR animations
- Author: samuelga24
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70117
This animation mod dynamically changes the animations when wiedling a pickaxe, a woodcuter axe and fishing rods to a more relaxed/non combat posture as well as the swinging animation whenever you are not in combat.
  
  
  
  
  
Is not a replacer all done through DAR, does not have a plugin or scripts, should be compatible with everything, now compatible with MCO thanks to [Roxy785](https://www.nexusmods.com/users/155729863); safe to install mid playthrough and safe to uninstall at any moment.
