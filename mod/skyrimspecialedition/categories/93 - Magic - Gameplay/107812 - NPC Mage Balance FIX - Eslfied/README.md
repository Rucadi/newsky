**About**

  
  
[NPC mage balance fix](https://www.nexusmods.com/skyrimspecialedition/mods/17041) eslfied + one extra option.
  
Small mod which edits 2 game settings:
  
  
**fMagicCasterSkillCostMult**: from 0.5 to 2.0(or 1.0)
  
**fMagicSkillCostScale**: from 0.5 to 0.65
