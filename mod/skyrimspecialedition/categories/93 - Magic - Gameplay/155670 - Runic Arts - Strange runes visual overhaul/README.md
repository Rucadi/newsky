- Game: skyrimspecialedition

**Description**
  
Changes the runes from strange runes from palm based circular ones to wrist based cylindrical ones. Something to note is that the design [size=4]has some mismatch for different spells and the ritual rune effects have been removed so the wrist runes dont show up for those.
  
  
[/size][size=5][size=6]**Requirements**
  
[size=3][Strange runes](https://www.nexusmods.com/skyrimspecialedition/mods/19456?tab=description) and all its requirements[/size][/size]
  
  
[size=6]**Installation**
  
Install via mod manager. Load after strange runes and alternative runes if you use it to replace wards/ritual.
  
  
[size=4][size=5][size=6]**Compatibility**
  
Not compatible with [strange runes scale options](https://www.nexusmods.com/skyrimspecialedition/mods/147832)﻿ and [size=3][SkyrimVR Strange Runes - Spellcasting Enhanced VR Edition](https://www.nexusmods.com/skyrimspecialedition/mods/30558)[/size][/size][/size][/size][/size]
