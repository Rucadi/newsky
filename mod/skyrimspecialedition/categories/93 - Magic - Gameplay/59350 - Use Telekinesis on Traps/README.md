**Description**
  
The main goal of this mod is to allow you to disarm [soul gem traps](https://en.uesp.net/wiki/Skyrim:Traps#Spellcaster_Trap) with [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿. This is something that is intuitive but was impossible in the vanilla game. In the process of fixing this issue, [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ became effective against other traps too.
  

  
* Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to disarm [soul gem traps](https://en.uesp.net/wiki/Skyrim:Traps#Spellcaster_Trap)﻿
Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to trigger [tripwires](https://en.uesp.net/wiki/Skyrim:Traps#Tripwire)﻿
  
* Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to trigger [pressure plates](https://en.uesp.net/wiki/Skyrim:Traps#Pressure_Plate)﻿
Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to trigger [hinge traps](https://en.uesp.net/wiki/Skyrim:Traps#Tension_Cable)
  
* Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to drop [oil lanterns﻿](https://en.uesp.net/wiki/Skyrim:Traps#Oil_Slicks.2FOil_Lanterns)
Use [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ to catch fish
  
* ESL﻿

* **Spell edit version** is incompatible with mods that edit the [Telekinesis](https://en.uesp.net/wiki/Skyrim:Telekinesis)﻿ spell record (0001a4cc).
**po3's Papyrus Extender version** is compatible with every spell that uses the correct **MagicTelekinesis** keyword and sensible delivery and casting types.
  

**Installation**

  
* Install whenever.

**Uninstallation/Upgrade**
  

  
1. Save in an interior location without soul gem traps (save1).
Uninstall/Upgrade mod.
  
2. Load save1 and make a new save (save2).
Clean save2 with [FallrimTools Resaver](https://www.nexusmods.com/skyrimspecialedition/mods/5031).﻿
