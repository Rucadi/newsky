**Requirements**
  
  
SKSE64
  
Visual C++ Redistributables 2019 
  
Address Library for SKSE Plugins
  
  
**Description**
  
  

  
* Simple SKSE plugin that allows you to read or take books, and auto learn spells from spell tomes
﻿For other items, see [Use or Take SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/70868)﻿
  

  
* Default action is to read books, take already read books, and learn spells from spell tomes
Hold hotkey (Left Shift by default) to toggle between read and take.
  
* Books will be flagged as stolen if owned, same behavior as other activated items
Features dynamic Read/Take prompt
  

  

  
* [size=3]Settings and hotkey can be configured using *Data/SKSE/Plugins/po3\_TakeOrReadBooks.ini*
  
[/size]
  
**Installation**
  
  

  
* Install as normal with a mod manager.
