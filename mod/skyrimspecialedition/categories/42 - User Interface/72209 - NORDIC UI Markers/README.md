My mod fixed, restore and redraw **ALL** markers of Skyrim for Nordic UI.
  
Mod have TWO versions! Standard version is designed for the default Nordic UI,
  
there is also a separate version that returns the icons of undiscovered locations,
  
 ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿  as in the original Skyrim.
  
  
  
**-------------------------------------------------------------------------------------------------------**
  
I redrawn icons of undiscovered locations like Nordic UI (on compass and world map).
  
That is, instead of question marks, there will be location icons, as was case in **vanilla game**.
  
To see this changes in game, you need install [NORDIC UI - Compass Markers Restored](https://www.nexusmods.com/skyrimspecialedition/mods/64249).
  
**-------------------------------------------------------------------------------------------------------**
  
Markers compatible with [Compass Navigation Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/74484) by [alexsylex](https://www.nexusmods.com/skyrimspecialedition/users/29450205)!
  
Just install 'Nordic UI Skin' in Optional Files on his page!
