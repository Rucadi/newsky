# Simple Hunting Overhaul MCM Helper
- Author: MaskedRPGFan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115638
[font=Georgia]MCM for [Simple Hunting Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/95943)﻿ ﻿by [JaySerpa](https://www.nexusmods.com/users/5201727)﻿﻿.[/font]

  
  
[font=Georgia]Features:
  
- let you see your current Hunting XP and change it (Cheat).
  
- let you change effects during looting the animal, like fade to black and time pass.
  
- enable/disable Hunter's Permit.
  
- enable/disable changes in skinning animals look.
  
- enable/disable skinning your own horse.
  
- enable/disable debug information for base mod.
  
- adjust rewards for carcasses.
  
- adjust minimal time for skinning from time reduction that Hunting XP will give you.
  
- adjust minimal time used for skinning.
  
- adjust maximal time used for skinning.
  
- adjust skinning time for specific animal.
  
- loader included to auto save/load setting.
  
- light plugin (ESP-FE), localization friendly.[/font]

> **FAQ**
>   
>   
> **Can I install/update/uninstall MCM midgame?**
>   
> Yes.
>   
>   
> **INI Settings file not changed.**
>   
> You need to install [MCM Helper](https://www.nexusmods.com/skyrimspecialedition/mods/53000).
>   
>   
> **MCM is duplicated**
>   
> Remove [Simple Hunting Overhaul - MCM](https://www.nexusmods.com/skyrimspecialedition/mods/103439).
>   
>   
> **Can I merge MCM plugin with any other plugin into one?**
>   
> No.
>   
>   
> **MCM doesn't show up.**
>   
>
>   
> * This can often be resolved by opening the command console and typing: **setstage SKI\_ConfigManagerInstance 1**
> You can also install [Jaxonz MCM Kicker SE](https://www.nexusmods.com/skyrimspecialedition/mods/36801).
>   
> * Or just use [Menu Maid 2 - MCM manager](https://www.nexusmods.com/skyrimspecialedition/mods/67556) (this manager automatically finds and adds all MCMs).
