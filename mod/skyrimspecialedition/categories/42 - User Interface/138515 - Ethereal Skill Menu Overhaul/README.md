# Ethereal Skill Menu Overhaul
- Author: Iconic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/138515


**Ethereal Skill Menu Overhaul**

![](https://i.imgur.com/0khl1Zr.png)﻿

  
Overhauls and retextures the skill menu interface to a darker fantasy style.
  
Support for many custom skills.
  
Fun fact: the thumbnail was created entirely using assets in this mod.
  
  
Replaces:
  

  
* Constellation clouds
Constellation stars
  
* Constellation images
Background stars
  
* Perk stars
Perk star lines
  
* Vampire clouds
Werewolf moon
  

  
Edits Skill Tree Mesh Domes:
  

  
* Removed extra effects that cluttered up the interface
Changed the background for standard skill trees, vampire skill tree, and werewolf perk tree
  

  
  
Compatibility
  
Compatible with everything except other skill menu texture mods.
  
Compatible with perk menu overhauls like Ordinator, Vokrii. and Adamant.
