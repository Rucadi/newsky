# Baka World Map Filter
- Author: shad0wshayd3
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148887


This is a small mod that disables the map in specific worldspaces when their names are added to the blocklist in BakaWorldMapFilter.json.
  
  
[GitHub Source](https://github.com/shad0wshayd3-TES5/BakaWorldMapFilter).
