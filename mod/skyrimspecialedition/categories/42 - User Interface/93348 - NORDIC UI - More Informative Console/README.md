# NORDIC UI - More Informative Console
- Author: Wuerfelhusten
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/93348


Description
  
This is a patch to combine the artstyle of NORDIC UI with the great functionality of More Informative Console. I was always bothered by the divergence of the Console UI from the rest of the UI. So I fixed that. Enjoy it!
  
  
[![](https://i.imgur.com/ROFYznU.png)](https://ko-fi.com/wuerfelhusten) [![](https://i.imgur.com/sIjqMOg.png)](https://trello.com/b/dlRCTHzW/wuerfels-roadmap) [![](https://i.imgur.com/VMdA0q7.png)](https://discord.gg/pqEHdWDf8z)﻿﻿
  
  
  
Usage
  
1. Uninstall every instance of Nexus Mod Manager.
  
2. Install MO2 or Vortex.
  
3. Install using one of them and let my mod overwrite both mods.
  
4. Enjoy!
