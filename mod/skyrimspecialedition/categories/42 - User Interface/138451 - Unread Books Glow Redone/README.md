# Unread Books Glow Redone
- Author: STB team
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/138451


![](https://i.imgur.com/PZzZGif.png)
  
**[General]**
  
 **﻿This mod allows you to enable a glow on the unread books and set configuration based on category and glow brightness. ﻿ ﻿Everything is done on SKSE, meaning it won't overload Papyrus.
  
  
 ﻿Mod comes with an INI file, where you can configure the main settings and also configure the provided rules under the [Visual].
  
  
 ﻿Optionally, you can select an MCM version of the mod during the installation, which will allow you to configure [Main] setting  ﻿ ﻿directly in the game. 
  
  

[Mod Authors]
  
[b] ﻿** ﻿You can add glow to the new quest books you make by creating a new INI file in SKSE - Plugins - UnreadQuestBooks folder.
  
 ﻿In order to avoid conflicts with other mods, it is recommended to give your configs unique names.[/b]
