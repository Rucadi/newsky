It only has the same concept.
  
This has been redone from scratch.
  
  
The font used in the screenshot is the [Enderal font](http://www.nexusmods.com/skyrimspecialedition/mods/2004/?).
  
  
  
**ABOUT**This mod replaces the mesh for the loading fog to be blank. This is mainly for people with ENB/ReShade presets installed, custom main menu wallpapers, or just don't like the fog.
  
  
  
**INSTALLATION**Simple. Download, extract, and move the meshes folder into the Skyrim SE data folder.
  
  
  
**UNINSTALLATION**Delete Meshes\Interface\intmenufogparticles.nif
