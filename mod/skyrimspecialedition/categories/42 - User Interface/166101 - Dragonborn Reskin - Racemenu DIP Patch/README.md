# Dragonborn Reskin - Racemenu DIP Patch
- Author: borokoshow
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/166101


This design complements the **[Dragonborn UI](https://www.nexusmods.com/skyrimspecialedition/mods/163074)﻿**[size=4]**[b][b][b][b][b][b][b] ![](https://i.imgur.com/1ioRGcy.png)**[/b][/b][/b][/b][/b][/b][/b]**Dragonborn UI style for the Racemenu mod****. Needs to be applied using [DIP](https://www.nexusmods.com/skyrimspecialedition/mods/96891)﻿.
  
  
This file does not contain any assets of the [RaceMenu](https://www.nexusmods.com/skyrimspecialedition/mods/19080) creator [expired6978](https://www.nexusmods.com/skyrimspecialedition/users/2950481)!**[/size]
  
[size=4] ![](https://i.imgur.com/uoCZiGd.png)
  
 **To patch RaceMenu with this patch:****1.** Disable other patches in MO2 for Racemenu if you have them
  
**2.** Delete interface folder in overwrite of MO2 if you used DIP patch before for this or any other UI
  
**3.** Install DIP from [here](https://www.nexusmods.com/skyrimspecialedition/mods/96891) as a regular mod in Vortex or MO2
  
**4.** Add the DIP executable as a tool
  
**5.** Install this DIP patch using MO2 or Vortex.
  
[size=4][size=4]**6.** [/size]Before you start DIP.exe only active racemenu mods in MO2 should be Racemenu original mod and this DIP patch
  
**7.** Start DIP.exe and click on "Patch!".
  
**8.** After success, a message box appears and DIP can be closed.
  
[size=4][color=#b6d7a8]**9.** Leave Dragonborn Reskin - Racemenu DIP Patch enabled (it contains button art for racemenu).[/size]
