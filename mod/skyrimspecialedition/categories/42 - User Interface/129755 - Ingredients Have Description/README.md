# Ingredients Have Description
- Author: shazdeh2-kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/129755


All alchemical ingredients now have description for more immersion. It shows the description when you hover over the ingredient. It will fade in and fade out, changing between the vanilla appearance and the new added description.

![](https://i.ibb.co.com/B6gdwtz/ex6.gif)               ![](https://i.ibb.co.com/BG5hWRv/ex3.gif)﻿ ﻿ ﻿

  
All vanilla and some ingredients added by anniversary edition are all covered. Ingredients from some mods are also covered:

Wyrmstooth
  
VIGILANT
  
 various mihail monster mods
  
  
If you don't have any mods listed above, this mod will still work.
  
  
**Installation:**1. Install all of the requirements (skyui, [powerofthree's Papyrus Extender](https://www.nexusmods.com/skyrimspecialedition/mods/22854) v5.8 or above) 
  
2. Install this mod with your mod manager
  
  
Enjoy!
