# Contextual Crosshair
- Author: doodlum
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/63980


**[left][![](https://github.com/doodlum/nexusmods-widgets/blob/main/Ko-fi_40px_60fps.png?raw=true)](https://ko-fi.com/doodlez)      [![](https://github.com/doodlum/nexusmods-widgets/blob/main/Patreon_40px.png?raw=true)](https://www.patreon.com/Doodlezoid)[/left]****[b]OVERVIEW**[/b]
  
  
**Contextual Crosshair smartly hides HUD elements when they are unneeded** with smooth fading transitions, numerous options, and deep integration with other mods.
  
  
For a less intrusive compass, [SkyHUD](https://www.nexusmods.com/skyrimspecialedition/mods/463) is recommended. 
  
  
  
**[color=#a4c2f4][b]DEFAULT OPTIONS**[/b][/color]
  
![](https://i.imgur.com/1foLEUO.png)
  
The crosshair is only enabled when:

  
* **Targeting a non-actor****reference**
**Aiming a bow or crossbow**
  
* **Aiming a targeted, non-concentration spell**        ﻿

  
  
  
             ﻿![](https://i.imgur.com/siMumL3.png)
  
  
  
The sneak meter changes in opacity depending on **detection level**, and is stacked with **crosshair opacity.****[b]SMOOTHCAM SUPPORT**[/b]When [SmoothCam](https://www.nexusmods.com/skyrimspecialedition/mods/41252) is enabled, the crosshair will be **replaced with the 3D Crosshair in third person**, and the sneak meter can become only half opaque.
  
  
  
**[b]TRUE DIRECTIONAL MOVEMENT SUPPORT**[/b]When [True Directional Movement](https://www.nexusmods.com/skyrimspecialedition/mods/51614)﻿ is enabled, the crosshair will be **disabled when locked on a target**, with transitions between crosshair types synced.
  
**[b]
  
  
DETECTION METER SUPPORT**[/b]When [Detection Meter](https://www.nexusmods.com/skyrimspecialedition/mods/63057)﻿ is enabled, the sneak meter will be **disabled unless aiming**.
  
  
  
**[b]BETTER THIRD PERSON SELECTION SUPPORT**[/b]When [Better Third Person Selection](https://www.nexusmods.com/skyrimspecialedition/mods/64339)﻿ is enabled, the **crosshair will not be affected by object activation.**
  
  
  
**REQUIREMENTS****Skyrim Special Edition**[Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444) 
  
  
  
**COMPATIBILITY**

* **Not compatible with Immersive HUD**
**Not compatible with Less Intrusive HUD II**
  
* **Compatible with 1.5.97 (pre-AE), 1.6.x (post-AE)**
