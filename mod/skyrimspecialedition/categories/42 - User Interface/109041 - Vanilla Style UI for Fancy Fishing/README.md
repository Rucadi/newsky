# Vanilla Style UI for Fancy Fishing
- Author: Jelidity
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109041


This mod is simply an overhaul for the UI elements in Fancy Fishing's Stardew Valley style fishing minigame to be consistent with the vanilla UI style.
  
  
Additional optional files available for a bigger and smaller fish icons.
