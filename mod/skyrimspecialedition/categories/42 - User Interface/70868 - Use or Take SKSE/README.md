# Use or Take SKSE
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70868


**Use Or Take SKSE**
  
  
[![](https://i.imgur.com/PMbnf6v.png)](https://www.patreon.com/bePatron?u=8408266)
  
  
**Requirements**
  
  
SKSE64
  
Visual C++ Redistributables 2019 
  
Address Library for SKSE Plugins
  
  
**Description**
  
  

  
* Simple SKSE plugin that allows you to use or take items by activating them
Press hotkey (Left Shift by default) + Activate to perform alternate action
  
* Keep holding hotkey to perform secondary action, if any (eg. Take scroll -> Equip scroll -> Read scroll)

  

  
* Items will be flagged as stolen if owned, same behavior as other activated items
Features dynamic activate text hover prompt, activate labels can be configured or changed to your language in settings
  

  

  
* [size=3]Armor
  
 ﻿Take | Equip
  
[/size]

  
* Weapons

 ﻿ ﻿Take | Equip | Equip and draw immediately
  

  
* Potions

 ﻿ ﻿Take | Eat food/drink potions/apply poisons
  

  
* Ingredients

﻿ ﻿Take | Eat
  

  
* Scrolls

﻿ ﻿ ﻿Take | Equip | Read (and cast scroll)
  
  
 ﻿For books, see [Read or Take SKSE](https://www.nexusmods.com/skyrimspecialedition/mods/69588)
  
  

  
* [size=3]Settings and hotkey can be configured using *Data/SKSE/Plugins/po3\_UseOrTake.ini*
  
[/size]
  
**Installation**
  
  

  
* Install as normal with a mod manager.
