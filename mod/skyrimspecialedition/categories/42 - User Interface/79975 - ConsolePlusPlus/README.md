# ConsolePlusPlus
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/79975


**CONSOLE++**
  
  
**Requirements**
  
  
Latest version of SKSE64
  
Visual C++ Redistributables 2022
  
Address Library for SKSE Plugins
  
  
**Description**Copy/Paste
  
  

  
* Copy paste text from clipboard to console, using Ctrl+C/Ctrl-V (for game version 1.597-1.6.640).
Copied text will be pasted at cursor position, just like standard copy/paste
  
* Keycodes and other options can be configured in *Data/SKSE/Plugins/po3\_ConsolePlusPlus.ini*

  
Cache Console Commands
  
  

  
* [size=3]Inputted console commands are saved to file and restored upon starting a new game instance.
  
Last 50 console commands are stored in My Documents/Skyrim Special Edition/SkyrimConsoleHistory.txt
  
- Limit can be raised/decreased
  

  
* Compatible with More Informative Console, and other mods that edit console.swf

[/size]
