# Dragonborn Reskin - Tween Menu Overhaul
- Author: borokoshow
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/144457


![](https://i.imgur.com/1ioRGcy.png)[font=Trebuchet MS]
  
A complete visual overhaul for the Tween menu (or Tab menu)!
  
Two icon styles to choose for Alternate version!
  
New custom made icons!
  
New dragonborn level up bar design!
  
NEW! UI navigation improved a lot in comparison to base TMO!
  
  
[size=4][size=5]**New UI navigation in v. 2.5 for Alternate version!**[/size][size=5]
  
[/size][size=5]-Go from first to last option of the menu using left key, and from last to first using right key!
  
-Select an option using up or down keys too!
  
Works with keyboard or gamepad![/size]

Download with your mod manager, load after Tween Menu Overhaul
  
  
Alternate version: To use any icon bellow with any mod add-on, simply open the respective .json file and change
  
icon keyword to one of the new keywords.
  
e.g. for the [Character Sheet](https://www.nexusmods.com/skyrimspecialedition/mods/143688)﻿ add-on, change icon keyword from "human" to "stats" in tweenoptions/stats.json
