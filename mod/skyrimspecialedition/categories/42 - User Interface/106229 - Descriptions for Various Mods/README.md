[left]**Installation**[/left]

  
* Requires [Description Framework](https://www.nexusmods.com/skyrimspecialedition/mods/105799)﻿ and [powerofthree's Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/51073)﻿ (for EditorID support)
Install with any mod manager, the FOMOD will auto-select any necessary files
  

**Mods Included So Far**
  

* Click the "Requirements" drop down above to see the list of mods included

**Notes**
  

* If you would like me to add descriptions to any other mods, please let me know and I will see about adding them in.
I am also open to receiving suggestions about better descriptions if you believe one to be lackluster or doesn't fit. Please feel free to share in the comments section!
