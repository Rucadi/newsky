Troubleshooting:
  
  
[font=Trebuchet MS][size=5]**Can't see the ? icons**[/font]Spoiler
  
In [size=5]skyhud.txt, set like this:
  
  
bHideEnemyHealth=0
  
bHideEnemyMarker=0
  
bHideLocations=0
  
bHideUndiscoveredLocations=0
  
bHideQuest=0
  
  
And in CompassNavigationOverhaul.ini, set like this:[size=5][size=4]
  
[/size]
  
bShowUndiscoveredLocationMarkers=1
  
bUndiscoveredMeansUnknownMarkers=1
  
bUndiscoveredMeansUnknownInfo=1[/size]
  
[/size]
