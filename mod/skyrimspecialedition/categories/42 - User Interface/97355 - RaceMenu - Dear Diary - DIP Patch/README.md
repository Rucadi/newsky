# RaceMenu - Dear Diary - DIP Patch
- Author: Wuerfelhusten
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/97355
Description
  
This is a Dear Diary patch for RaceMenu to be used with the [Dynamic Interface Patcher (DIP)](https://www.nexusmods.com/skyrimspecialedition/mods/96891) by [Cutleast](https://www.nexusmods.com/users/65733731). This file does not contain any assets of the [RaceMenu](https://www.nexusmods.com/skyrimspecialedition/mods/19080) creator [expired6978](https://www.nexusmods.com/skyrimspecialedition/users/2950481)!
  
DRIP was just a proof of concept for dynamic SWF patching, so Cutleast used what he learned from it to create a universal interface patcher. Just like DRIP, this Patcher performs the patching, a normal UI-modder would do, automatically based on steps written to a json file.

Usage
  
To patch RaceMenu with this patch:
  
1. Install DIP from [here](https://www.nexusmods.com/skyrimspecialedition/mods/96891) as a regular mod in Vortex or MO2
  
2. Add the DIP executable as a tool
  
3. Install this patch using MO2 or Vortex.
  
4. Start DIP.exe and choose paths (first one should be auto-detected and second one is only necessary if the mod is not at the same location).
  
5. Click on "Patch!".
  
6. After success, a message box appears and DIP can be deactivated.
