# Skyrim 10th Anniversary Menu
- Author: Guitarninja
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/86184


A custom main menu texture using a cleaned version of the 10th Anniversary Skyrim wallpaper. If you use a widescreen monitor, make sure you use the widescreen version.
  
  
Highly recommend also using [this](https://www.nexusmods.com/skyrimspecialedition/mods/30810) main menu design replacer with it also.
