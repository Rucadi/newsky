# Armors Have Description
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/117818


Using the power of description framework mod, all vanilla armors and creation club armors from anniversary edition now can have description!
  
The description is written some by using by generating text ai, some by using description taken from skyrim info website (uesp.net and elderscrolls.fandom.com). 
  
  
**Installation:**1. Install Description Framework mod at least  v.2.1 with mod manager
  
2. Install this mod
  
That's it!
  
  
Notes: 
  
**~~Unenchanted armor will show up as enchanted, it is a known bug from description framework and it is harmless. Don't worry, you can still enchant it without problem!~~> This has been fixed with description framework v2.0**
