# Custom Skill Mod JSON Config Conversions for Custom Skills Framework 3
- Author: Xelphos
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/127448


Decided it would be better to just have them all in one place instead of having individual nexus pages just for JSON config files.
  
  
**What this is:**
  
A small collection of converted Custom Skills Framework config files to the new JSON format to replace the old configs shipped with these mods. This "should" also solve the issue of the skill menu not opening in Anniversary Edition when using these skill mods.
  
  
**Why?**
  
Because I wanted to.

**Install:**
  
- Install the original mod and it's requirements, make sure to use Custom Skills Framework 3.0.0+ (Do not install NetScriptFramework).
  
- Install the desired skill menu config from here as you would any other mod.
  
- Remove the config TXT file from the original mod, it should in the NetScriptFramework folder.
