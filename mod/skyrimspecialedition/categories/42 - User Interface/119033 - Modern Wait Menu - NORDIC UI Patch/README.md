# Modern Wait Menu - NORDIC UI Patch
- Author: Wuerfelhusten
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119033
Description
  
An interface patch for Modern Wait Menu by [Fallen01135](https://www.nexusmods.com/skyrimspecialedition/users/3086217) to match NORDIC UI‘s artstyle better. You will require the original Mod and this patch.
  
This patch also includes support for whichever cursor mod you like. The cursor has been exported to a different swf so that it can easily be modified. (Beware that this mod only includes the original NORDIC UI Cursor)
  
  
[![](https://i.imgur.com/ROFYznU.png)](https://ko-fi.com/wuerfelhusten) [![](https://i.imgur.com/sIjqMOg.png)](https://trello.com/b/dlRCTHzW/wuerfels-roadmap) [![](https://i.imgur.com/VMdA0q7.png)](https://discord.gg/pqEHdWDf8z)﻿﻿﻿﻿
  
Usage
  
1. Uninstall every instance of Nexus Mod Manager.
  
2. Install MO2 or Vortex.
  
3. Install using one of them and let my mod overwrite Modern Wait Menu
  
4. Enjoy!
