# Character Suggestor
- Author: shazdeh2
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/144854


Do you suffer from Restartitis and have to create new characters so often? This mod adds a button in the character creation screen (called Inspire), clicking it reveals a panel and you can hit Generate button to generate a random character. Press tab or click the button again to hide the panel.

**Compatibility**
  
Compatible with all mods, including mods that change the racemenu, like **RaceMenu**.
  
Also compatible with mods that add new playable races, they will show up in the list.
