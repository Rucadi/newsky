# NORDIC UI - Alternate loading screen and start menu
- Author: Dopesan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/54153
This is an alternative version for the loading screen and start menu of NORDIC UI. **Does not change the background**, so it's compatible with vanilla or modded backgrounds. No plugins.
  
  
I don't know if it works with AE.
  
  
**THINGS CHANGED**
  
1. Reduced Start menu options to continue, new, load and quit.
  
2. Removed Skyrim special edition logo.
  
3. Moved swirly dragon closer to the vanilla loading icon position.
  
  
**INSTALLATION**Requires - [NORDIC UI - Interface Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/49881)
  
Simply install it after "NORDIC UI - Interface Overhaul", it's just some .swf files
