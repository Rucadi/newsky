# Favorite Misc Items
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/42750


**FAVORITE MISC ITEMS**
  
  
**[![](https://i.imgur.com/PMbnf6v.png)](https://www.patreon.com/bePatron?u=8408266)**
  
  
**Requirements**
  
  
SKSE64
  
Visual C++ Redistributables 2019 
  
Address Library for SKSE Plugins
  
  
**Description**
  
  
SKSE plugin that allows miscellaneous items to be favorited. Useful if you have mods that depend on misc objects to provide functionality, such as AddItemMenu.
  
  
You may also be interested in [Essential Favorites](https://www.nexusmods.com/skyrimspecialedition/mods/42997), which protects favorited items from being sold, disarmed, or dropped.
  
  
**Installation**
  
  

  
* Install using a mod manager.

  
[size=3]**Uninstallation**
  
  

  
* Favorited misc items cannot be unfavorited if the mod is uninstalled.
