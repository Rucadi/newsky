# Sneak Vignette
- Author: Stentorious
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/145046


**![](https://i.imgur.com/swP6NiD.png)
  
![](https://i.imgur.com/fFwFy5m.gif)
  
  
![](https://i.imgur.com/4BCgyVl.png)
  
Adds a [vignette effect](https://docs.unrealengine.com/4.27/en-US/RenderingAndGraphics/PostProcessEffects/Vignette/) to the screen while the player is sneaking.
  
Vignette will smoothly transition between sneak states.
  
Widescreen support included.
  
  
![](https://i.imgur.com/aHE0SIt.png)﻿
  
Configurable via the MCM:
  
> Set vignette intensity
  
> Set vignette color
  
> Set vignette behavior mode
  
> Toggle vanilla sneak indicator visibility
  
  
BEHAVIOR MODES
  
> Sneak State - Vignette intensity is based on player sneak state (Standing -> Sneaking)
  
> Detection Level - Vignette intensity is based on player detection level (Hidden -> Detected)
  
  
![](https://i.imgur.com/xBofrnm.png)﻿
  
Compatible with any HUD mod.
  
This mod works best with a 60 FPS hudmenu.swf file.
  
Using a 24 or 30 FPS menu file will reduce the smoothness of the vignette transitions.
  
A 60 FPS hudmenu.swf for vanilla HUD users is included under optional files.
