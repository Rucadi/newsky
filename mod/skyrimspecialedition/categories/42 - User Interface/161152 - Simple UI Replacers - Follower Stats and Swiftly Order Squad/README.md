# Simple UI Replacers - Follower Stats and Swiftly Order Squad
- Author: monkeyangie
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/161152


![](https://pixelsafari.neocities.org/dividers/vine.gif)

**Description

![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)**
  
Simple re-placer of the icons of Swiftly Order Squad and Follower Stats, made to be used with UIs like Dear Diary/Dark mode, but is compatible with any UI. I used AI to generate the drawings because i can't draw and i have no talent.

**Installation
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)**

? Install [Swiftly Order Squad - Follower Commands UI](https://www.nexusmods.com/skyrimspecialedition/mods/63259) for this version
  
? Install [Follower Stats](https://www.nexusmods.com/skyrimspecialedition/mods/159406) for this version
  
? Install this mod
