# Dynamic Location Pop-ups
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153122


DYNAMIC LOCATION POPUPS
  
  
[![](https://i.imgur.com/PMbnf6v.png)](https://www.patreon.com/bePatron?u=8408266)
  
Requirements
  
  
SKSE64
  
Visual C++ Redistributables 2022
  
Address Library for SKSE Plugins
  
  
Description
  
  

  
* Display location name with discovery music jingle when entering already discovered locations.
Compatible with all vanilla and modded locations.
  

* Compatible with all UI related mods (except for mods that remove the location discovery pop-up message) and Map Marker Framework.

  

  
* See *SKSE/Plugins/po3\_LocationPopup.ini* for mod configuration.
