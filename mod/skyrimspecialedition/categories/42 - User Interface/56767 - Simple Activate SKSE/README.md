meh's Address Library for SKSE Plugins
  
  
**Description**
  
  
SKSE plugin that offers several activate prompt customization options, such as:
  
  

  
1. Hide activate button ( [E] )
Hide activate text (Take, Harvest, Sleep)
  
2. Streamlined indicator with customizable colors for Steal/Pickpocket/Owned/Locked/Empty
Hide or create custom tags for Locked/Empty
  
3. Removing ALL activate prompts for NPCs/activators/containers/doors/furniture/items/flora/projectiles (this will override above options)

  
All of these can be configured in *po3\_SimpleActivateSKSE* configin Data/SKSE/Plugins*.* It will generate itself if missing once you start the game. MO2 users can find this in the overwrite folder.
  
  
Default settings:
  

  
* Hide activate button  = true
Hide activate text = true
  
* Steal indicator (red)
Owned indicator (yellow)
  
* Custom Locked tag for all locked items
