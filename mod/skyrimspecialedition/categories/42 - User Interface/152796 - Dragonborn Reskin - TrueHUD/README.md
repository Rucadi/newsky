[/font][/size][font=Trebuchet MS]Download and install with your mod manager, **load after** True[font=Trebuchet MS]HUD[/font] - [font=Trebuchet MS]HUD[/font] Additions.
  
If you want the layout to be the same as the images, install the "settings" option. Check if you have TrueHUD.ini in your **Overwrite** **folder**, if so delete it, or you won't see the changes.
  
  
[font=Trebuchet MS][size=5]To change colors and position, access Truehud MCM or truehud.ini file.
  
Color code need to be in [hex](https://color-hex.org/color/9d002c) ﻿format![/font]
  
  
[/size]**Compatibility**
  
Not compatible with anything that overwrites my .swf files
  
Truehud - inventory injector Patch - load **before** my mod
  
Any TrueHUD bars with numbers mods- Not needed, bars with numbers option is included.
  
 TrueHUD - No Enemy Healthbar Fade Delay - Not needed, similar tweak included.
