# Oxygen Meter 2 - Nordic UI like Skin
- Author: kuroishitam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/69567
**Update 17/5/24**
  
For two optional version, more whitish and bluish version: 
  
for bluish version change Widget Bar Colour = in oxygenmeter2.ini to 3308797
  
for whitish version change Widget Bar Colour = in oxygenmeter2.ini to 15921906
  
Optionally change also Widget Flash Colour = in oxygenmeter2.ini to same number above
  
  
**Update 13/06/2022**
  
A little touch up on the edge of the box so that it is more obtuse and more matching nordic UI
  
  
I use both [Oxygen Meter 2](https://www.nexusmods.com/skyrimspecialedition/mods/64532) and Nordic Ui, but  it is just unfortunate that the Oxygen Meter 2 use vanilla box HUD and really doesn't match Nordic UI.
  
Since nobody make the Nordic UI skin for Oxygen Meter 2, i decided to make it myself.
  
  
I made this using [Nordic...ish - A TrueHUD preset based on Nordic UI](https://www.nexusmods.com/skyrimspecialedition/mods/63212) as a base to match the color and the design, since i also use that mod to make trueHUD bar match Nordic UI.
  
  
INSTALLATION/UNINSTALLATION
  
Simply install or uninstall using mod manager, it won't do anything to your savefile
  
  
**RECOMMENDED MODS FOR COMPLETE NORDICISH UI:**[NORDIC UI - Interface Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/49881)
  
[TrueHUD - HUD Additions](https://www.nexusmods.com/skyrimspecialedition/mods/62775)
  
[Nordic...ish - A TrueHUD preset based on Nordic UI
  
NORDIC UI - Miscellaneous Patches](https://www.nexusmods.com/skyrimspecialedition/mods/63212)[NORDIC UI - Configurable Sleep Wait Menu](https://www.nexusmods.com/skyrimspecialedition/mods/69056)[NORDIC UI - Better Scroll Bar](https://www.nexusmods.com/skyrimspecialedition/mods/66039)
