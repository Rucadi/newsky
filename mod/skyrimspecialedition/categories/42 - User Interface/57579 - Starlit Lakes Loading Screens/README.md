# Starlit Lakes Loading Screens
- Author: marblemist
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/57579


![](https://staticdelivery.nexusmods.com/mods/110/images/109588/109588-1635379563-1835414791.png)

**Overview**
  
  
A collection of 100 new loading screens, accompanied by various snippets of Tamrielic and Skyrim-oriented lore. The mod replaces the vanilla loading screens.
  
  
The plugin is ESL-flagged, so it won't contribute to your plugin total.
  
  
*Note: The mod contains 37 images, each of which is represented approximately three times, each with a different lore entry.*
