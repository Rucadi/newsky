# Lorebox - Requiem armor bonus addon
- Author: Patman023
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/158069


Requiem introduces hidden armor bonuses and resistances tied to each armour type, affecting gameplay in subtle but meaningful ways.
  
  
This Lorebox patch enhances transparency by adding tooltips that reveal these bonuses directly in the interface. Players can now see exactly what benefits each armour type provides, making it easier to make informed equipment choices without relying on external guides.
  
  
Works for both Requiem V5 & V6.
  
  
![](https://i.imgur.com/xDc7gSF.jpeg)
  
  
  
**Installation guide:**
  
  
1. First install [LoreBox - Item and Spell Tooltips](https://www.nexusmods.com/skyrimspecialedition/mods/156534) and it's requirements
  
2. Install my patch, load after main Lorebox file
  
3. (optional) install a UI patch to suit your UI: 
  
[LoreBox - Untarnished UI Reskin](https://www.nexusmods.com/skyrimspecialedition/mods/157224) (The screenshots are taken using Untarnished UI and this patch)
  
[LoreBox Nordic UI Patch](https://www.nexusmods.com/skyrimspecialedition/mods/157165)4. Enjoy!﻿
