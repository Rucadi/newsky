# I4 Weapon Icons Overhaul
- Author: komegaki
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/106432


Tired of your weapon icons using generic weapon icons? No more! This mod adds over 30 custom made icons to over 30 different weapon types. (With Wheeler Compatibility!)

(This mod categorizes many things, including weapons, without it, maybe only 10-15% of this mod will work)
