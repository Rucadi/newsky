# Menu Zoom
- Author: powerofthree
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/117402


**MENU ZOOM
  
  
[![](https://i.imgur.com/YwGFWI0.gif)](https://www.patreon.com/bePatron?u=8408266)**
  
**Requirements**
  
Latest version of SKSE64
  
Visual C++ Redistributables 2022
  
Address Library for SKSE Plugins
  
  
  

![](https://i.imgur.com/uGIx7Po.gif)

  
**Description**
  

  
* Enables inventory zoom feature for crafting and magic menus
Supports keyboard and controllers
  
* Press C or scroll wheel over item to zoom in/out
Press Escape to...escape zoom
  
* Also adds mouse rotation for spell/shout visuals (some spell objects may rotate or move incorrectly)
