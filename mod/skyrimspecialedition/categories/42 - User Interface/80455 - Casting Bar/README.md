# Casting Bar
- Author: zax
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/80455


[heading]Description[/heading]Make sure you will never cancel the spell in middle of casting. Especially useful with ritual type vanilla spells or spell mods with very long casting times, such as Apocalypse etc.
  
  
  
[heading]Features[/heading]

  
* Displays a progress bar for spell casting, shouting, and drawing the bow
Custom scaleform widgetÂ 
  
* Configurable meter placement

  
  
[heading]Details[/heading]Casting bar prioritizes whichever hand started the casting, its progress will be shown till the spell is released.
  
Shouting progress is shown for second and third word (first word is always instant).
  
Changes to fShoutTime1 and fShoutTime2 game settings are fully respected.
