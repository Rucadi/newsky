# The Dragonborn's Bestiary - Draugrs
- Author: DanielChicken
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/125052


**This mod adds various mod support for the amazing Dragonborn Bestiary mod.**

You do not need every supported mod. It'll work out the box with whatever supported mods you do have installed.

  
  
  
Mods currently supported are:
  
**[Know Your Enemy 2](https://www.nexusmods.com/skyrimspecialedition/mods/93258)**thanks to [Kulharin](https://next.nexusmods.com/profile/Kulharin/about-me)
  
  
[The Restless Dead](https://www.nexusmods.com/skyrimspecialedition/mods/94100) (tested v1.2.5) (Skeleton support still needs doing)
  
[Infant Draugrs- Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/121527) (tested v1)
  
[Torch-wielder Draugrs- Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/120990) (tested v1)
  
[Skyrim Immersive Creatures](https://www.nexusmods.com/skyrimspecialedition/mods/12680) (tested v7.0.2) (Basic compatibility - no new typings)
  
[Cannibal Draugr on Solstheim](https://www.nexusmods.com/skyrimspecialedition/mods/21238) (tested v2.9)
  
[The Dragon Cult - A Draugr Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/81422) (1.4.3)
