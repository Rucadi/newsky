# I4 Shout Icons Overhaul
- Author: komegaki
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/107334
Tired of every single shout icon having the same icon? No more! This mod adds immersive icons to shouts in vanilla and modded skyrim! (With Wheeler Compatibility!)
