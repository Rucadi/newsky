# The Dragonborn's Fishiary - Bestiary Addon
- Author: Missile
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/134739


[**The Dragonborn's Fishiary - Bestiary Addon**](https://www.nexusmods.com/skyrimspecialedition/mods/123521)
  
  
You can now unlock new entries in the bestiary by fishing up fish. 
  
  
When you fish up a new fish, you will unlock that fish's bestiary entry. The entry will give you some information about the fish and what types of water you can catch it in, along with any weather requirement.
  
  
Theres a total of 30 entries you can unlock. They are listed below:
