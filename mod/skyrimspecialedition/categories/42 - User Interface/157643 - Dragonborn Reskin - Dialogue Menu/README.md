# Dragonborn Reskin - Dialogue Menu
- Author: borokoshow
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/157643


My dialogue menu reskin based on Convenient Dialogue UI
  
New visual style + even more customizable
  
This is a **standalone** version (with more design options) of the dialogue menu that is part of [font=Trebuchet MS]the **[Dragonborn Reskin - SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/163074)﻿**[/font][font=Trebuchet MS][size=5][size=5][size=5][size=4][size=5][size=6][/size][/size][/size][/size][/size][/font]
  
[font=Trebuchet MS]**[b][b][b][b][b][b][b][color=#ffe599]![](https://i.imgur.com/1ioRGcy.png)**[/b][/color][/b][/b][/b][/b][/b][/b][/font]
  
**Former Convenient Dialogue UI features (all included):**
  
- Better Dialogue Controls integrated.
  
- Left sided or right sided topics list.
  
- Topics selectable with numbers keys as shortcuts.
  
- More dialogue topics displayed.
  
- Adjustable placement for topics list and NPC name.
  
- NPC name and exit button can be hidden.
  
- Configurable text size and color.
  
  
**Added Features:**
  
-New Dragonborn UI style background, selector and buttons.
  
-Fixed screen size for 16:9 displays for all Skyrim versions.
  
-NEW in v1.1 Can change icon color and toggle visibility.
  
-NEW in v1.1 Can change background color and toggle visibility.
  
-Can change NPC names' font color.
  
-Can change NPC subtitles' font color\*.
  
-NPC name only appears while npc is talking.
  
-Can change the highlighted topic’s font size independently.
  
-Longer text can be displayed in a single line.
  
-Added number 0 as aditional 10th keyboard shortcut.
  
  
[/size]\*beware that the subtitles **during dialogue menu** are different from the other subtitles (these are part of hudmenu and don't change!)
  
  
[size=6]
  
[font=Trebuchet MS]![](https://i.imgur.com/uoCZiGd.png)[/font]
  
No requirements!
  
  
[/size]Use **dialoguemenu.txt** to configure it as you like!

Pro tip: no need to exit the game to see the changes, just exit and reopen dialogue menu.
