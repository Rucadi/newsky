# Dragonborn Reskin - Loading Menu Overhaul
- Author: borokoshow
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150050


This is a reskin for Loading Menu Overhaul, it contains new UI elements like level up bar, buttons and such.
  
The wallpapers are from another mod. Use whatever wallpaper mod you want.
  
  
**Wallpaper mod in the pics is [The Elder Scrolls Legends - Loading Screens](https://www.nexusmods.com/skyrimspecialedition/mods/37929)﻿**﻿![](https://i.imgur.com/1ioRGcy.png)
  
[font=Trebuchet MS]Dragon themed level up bar design
  
New button art
  
New lore text background
  
Two layouts (level up bar at the bottom or upper left)
  
Versions with hidden buttons and no level up bar

Optional file with my new button art for your SkyUI menus too!
