# NPCs Names Distributor - Patches for Various Mods
- Author: Preyka
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/119090


[font=Georgia][NPCs Names Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/73081) is an awesome mod for immersion. It includes an option for Obscuring Names. However, some modded NPCs with unique names are not flagged as such. My small mod fixes this issue by distributing keywords.
