**Description**
  
  

  
* Adds dialogue history feature.
Logs all conversations between the player and NPCs/ambient NPC conversations.
  

  

  
* Supports mouse/keyboard. Gamepad is untested.
Keys and other settings can be modified using MCM.
  
  
**Local History**

  
* View transcript of your current conversation.
Default key is **Ctrl**.
  

  
**Global History**
  
  

  
* Switch between Dialogue History (PC - NPC)/Conversation History (NPC)
Can be sorted chronologically or by location. 
  
* Filter by NPC name to find out how many times you've spoken to Lydia.
Click on NPC line to hear audio playback.
  

  

  
* Default key is **Shift-D**

**[b]Conversation History**
  
[/b]
  

  
* Filter out scene/combat/detection/misc dialogue with this mod's MCM

[size=4]**Styles/Fonts**
  
  

  
* Requires [ImGui Icons](http://nexusmods.com/skyrimspecialedition/mods/114790)﻿ mod for button art/fonts.

  
* Adjust background color, line colors etc by editing **Data/Interface/DialogueHistory/styles.ini**.
Color values should be hex (#FFFFFFFF) or RGBA values (255,255,255,255)
  

  

  
* Adjust fonts/font scale by editing **Data/Interface/DialogueHistory/fonts.ini**.
If you're using large fonts and experience freezing when opening menus, set bLoadFontsOnce to true.
  

  

  
* Changes will be automatically applied by reopening either menu in game.
