# B.O.O.B.I.E.S  (aka Immersive Icons)
- Author: WhereDeadAngelsLie and Discrepancy
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/89241


**﻿What's the most immersion-breaking experience you can face in Skyrim?**

  

*Imagine slaying a troll, looting its skull, and then opening your inventory only to find... a skull with just two eye sockets? Wtf is this? Where is my real troll skull?*

  
  

**Have you ever tried to buy some spell books from a vendor in a list of 2000+ spells all displayed with the same boring white book icon?**

  
  

**Have you ever installed complex armors with 20 different body parts, all displaying the same dull icon?**

  
  

**Has it ever left you feeling disheartened when you discover a mighty artifact crafted by a Daedric prince, only to find that it's represented by a plain white icon?**

  
  

**Have you ever been triggered because you created an LOTD replica and put the real item in the chest, while keeping the replica in your inventory?**

  
  

**Finally! A proper answer to all those unbearable situations which plagued our beloved Skyrim for years!**

  
  
  
  

**You can now enjoy a beautiful display of various icons and colors, designed mainly by yours truly, and based on the work of other talented mod authors.**

Install the mod in that order
  
1. Install [KID](https://www.nexusmods.com/skyrimspecialedition/mods/55728) 3.0+
  
2. Install [I4](https://www.nexusmods.com/skyrimspecialedition/mods/85702)
  
3. Install [I4 Patch for 1.5.97](https://www.nexusmods.com/skyrimspecialedition/mods/87002) if needed
  
4. Install BOOBIES
  
  
If you don't want the addons you can stop here... but why would you do that? Those addons are fantastics ?
  
  
5. Install [OCF](https://www.nexusmods.com/skyrimspecialedition/mods/81469) ﻿and select **Light Version** in the FOMOD (this mod contains the rules for Ingredients and food items)
  
6. Install [THICC](https://www.nexusmods.com/skyrimspecialedition/mods/90508) (this mod contains all the icons used by [GroundAura](https://www.nexusmods.com/skyrimspecialedition/users/97658973) mods that are not in BOOBIES)
  
7. Install [ASS](https://www.nexusmods.com/skyrimspecialedition/mods/89823) (addon to manage food items)
  
8. Install [PENIS](https://www.nexusmods.com/skyrimspecialedition/mods/90526) (addon to manage alchemical ingredients)
