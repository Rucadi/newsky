**Install instructions:**
  
1. Install the required mods:
  
 ﻿~ [Custom Skills Framework](https://www.nexusmods.com/skyrimspecialedition/mods/41780)
  
 ﻿﻿~ [Custom Skills Menu - A Custom Skills Framework](https://www.nexusmods.com/skyrimspecialedition/mods/62423)
  
 ﻿~ Any Custom Skill mods of your choice
  
2. Install this mod with your favorite mod manager
  
3. Let it replace any existing files
