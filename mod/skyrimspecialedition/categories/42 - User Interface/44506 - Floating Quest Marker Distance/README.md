# Floating Quest Marker Distance
- Author: all00700
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/44506


This simple mod Increases the distance Floating Quest Marker is visible
  
  
by adding 1 Game setting**fFloatQuestMarkerMaxDistance**
  
  
There are 3 options, 2x, 3x, and 4x of Vanilla.   (+optional 6x, 8x)
  
  
  
**As you already know, it can interfere with the immersion. your choice**
  
  
Everything compatible
