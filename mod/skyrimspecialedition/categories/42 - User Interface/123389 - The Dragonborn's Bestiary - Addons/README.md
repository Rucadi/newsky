**Description**
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)

  
This modification changes the Bestiary images of existing creatures with modded versions and introduces several new ones from popular modifications. Some of them are added in the main file and some are separated by author, mainly because it was easier for me.

**Installation**
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)

  
? Install the main file and the Addons you want from [The Dragonborn's Bestiary](https://www.nexusmods.com/skyrimspecialedition/mods/123521?tab=files)﻿, and then install this one
  
? Drop everything on your Data folder or use a Mod Manager, of preference use MO2
  
  
Now if for whatever reason you don't want a creature, just delete the undesired creature or overwrite it, these are just textures and .json files, very easy to change
  
  
**FAQ**
  
  
Q: What if I need just 1(one) of the creatures you have in a pack?
  
A: Well, [actually](https://i.kym-cdn.com/photos/images/newsfeed/001/191/035/135.png), you can install the whole pack and the mod will only use the creatures you have installed, the not installed creatures will never show up in your bestiary
