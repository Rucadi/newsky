# No Numpad parity patch for Fix any Pre-update Gamepad
- Author: Slamminslug and kryten397
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/164882


Thanks to kryten397 for their original mod:
  
https://www.nexusmods.com/skyrimspecialedition/mods/107898
  
  
This mod makes it so your number row keys and numpad keys are no longer linked, meaning the numpad can be used for new binds.
