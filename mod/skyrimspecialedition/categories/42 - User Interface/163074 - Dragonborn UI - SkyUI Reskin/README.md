**UNIQUE**
  
Not every box or gauge bar needs to look the same. Each menu has its own distinctive shape. The level progress bar looks different than the life bar, charge bar or shout bar.
  
  
  
**MINIMALISTIC**
  
Unnecessary separators and frames removed, plus intentional transparency for a lighter, cleaner look. The dragon theme is present but not overused.
  
  
  
**HAND DRAWN ICONS**
  
Many hand-drawn icons replace text across the UI, adding a touch of artistry and giving the menus a more crafted, Elder Scrolls–inspired feel.
  
  
**IMPROVED**
  
SkyUI experience improved with customizable dialogue menu, [/font]search bar for the MCM menu, new Sleep/Wait Menu, no loading icon outside of loading screens, smaller and vertical charging bar and more!

[font=Trebuchet MS]

**Some Story:**
  
  
Spoiler
  
Since releasing my very first mod, the [color=#ffe599]*Dragonborn Wheeler* reskin, people were always asking the same thing: “Can you redesign the whole Skyrim UI?” For a long time, my honest answer was no—I didn’t feel I had the Flash animation skills for something that big.
  
  
*Flash CS* was a clumsy, unintuitive piece of software, and good tutorials felt as rare as actual Dwemer schematics. But after countless hours tinkering with *JPEXS* *Flash Decompiler*, creating many reskins and slowly understanding how Skyrim’s UI ticks, I finally felt confident enough to take on the challenge—armed with nothing but *JPEXS* and my loyal partner, *Inkscape*.
  
  
At first, I was just aiming for a rugged version of Untarnished UI, or maybe some kind of Untarnished-meets-Nordic UI hybrid. But after playing Oblivion Remastered and seeing its beautiful paper-style loot menu, the direction shifted—and that inspiration shaped what the project eventually became. And now… **here we are!**

>**SkyUI** (required)
  
>**mods that change interface**, like SkyHUD, moreHUD, Better Third Person Selection, Survival Mode Improved, ImGUI, CoMap, Quickloot IE etc
  
>**Dragonborn UI SkyUI Reskin**
  
>Other Dragonborn Reskin mods

**I want to install this over my modlist, how??**
  
  
It's simple — generally you need to disable the UI skin in your modlist and all its patches. Also disable Dear Diary.
  
  
If a crash happens, usually the culprit is a leftover UI patch asking for buttonartDD.swf. Find it and disable it.
  
  
Example: If you're using Untarnished UI, use the search function in your mod manager and disable all Untarnished mods and Dear Diary.
