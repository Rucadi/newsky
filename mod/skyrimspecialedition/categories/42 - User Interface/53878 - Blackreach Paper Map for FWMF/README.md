# Blackreach Paper Map for FWMF
- Author: Caites
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/53878
**Blackreach Paper Map for FWMF****[b]**[/b]Here is something new for you - first Blackreach map available from Map Menu. Nothing exceptionally sophisticated, but it will help you to navigate here and add little bit of immersion. Mod continues series of paper maps for a [Flat World Map Framework (FWMF)](https://www.nexusmods.com/skyrimspecialedition/mods/29932).
  
  

  
* Two map texture versions available in FOMOD - with and without crimson nirnroot markers.
4 theme backgrounds added.
  
![](https://i.imgur.com/IK9kApK.png)Flat World Map Framework uses mainly vertical look at map, but allows limited inclination and rotation with held right button as well.
  
INI file with comments included in FWMF allows to tune map control (keyboard, mouse and gamepad), possible angles and blur.
  
  
**Requirements**

  
* [FWMF](https://www.nexusmods.com/skyrimspecialedition/mods/29932)﻿

  
**Installation**
  
The archive is mod manager compatible.
  
  
Install [size=3][FWMF](https://www.nexusmods.com/skyrimspecialedition/mods/29932) FOMOD first (Fantasy Paper Maps package preferable). Make sure FMWF is the last in plugins list (only weather fixes or FWMF various patches may be lower), then install this mod normally and place it last in mod order.
  
  
[/size]Make sure Flat Map Markers config file (FlatMapMarkersSSE.json) has this part:
  
  
> },
>   
>  {
>   
>  "name": "Blackreach",
>   
>  "editorID": "Blackreach",
>   
>  "markerHeight": 180000,
>   
>  "enabled": true
>   
>  }

  
**Compatibility**
  
[size=3]Compatible - SE/AE/GOG (1.6.640 including), but be sure to pick your exact Skyrim version during FWMF FOMOD installation.
  
Compatible - [The Blackest Reaches](https://www.nexusmods.com/skyrimspecialedition/mods/21280).
  
Not compatible - any map mods except [FMWF](https://www.nexusmods.com/skyrimspecialedition/mods/29932).
  
  
[/size]**Recommended Mods**
  
[Atlas Map Markers - Updated with MCM](https://www.nexusmods.com/skyrimspecialedition/mods/24104)﻿ works perfect and smaller markers make map more readable. Just be sure to put Atlas' .esp before FWMF in load order.
  
  
Get along with most UI overhauls. [Dear Diary UI](https://www.nexusmods.com/skyrimspecialedition/mods/23010)﻿ and its skins are highly recommended.
