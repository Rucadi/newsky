# Easy Wheelmenu for Wintersun - Faiths of Skyrim
- Author: replica102
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/27535
An esl flagged plugin that adds Pray from [Wintersun - Faiths of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/22506) to [Easy Wheelmenu's](https://www.nexusmods.com/skyrimspecialedition/mods/22934) MCM. Now rebuilt and tested on [Easy WheelMenu II SSE](https://www.nexusmods.com/skyrimspecialedition/mods/59799).If the option to remove spells is enabled in Easy Wheelmenu's MCM, Pray will be removed your spell list.
