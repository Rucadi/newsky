# Aura's Scrumptious Supplement (A.S.S. for B.O.O.B.I.E.S.)
- Author: GroundAura
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/89823


**Aura's Scrumptious Supplement**

**A Food and Potions Addon for B.O.O.B.I.E.S.**
  
**Overview**
  

Tired of not knowing what you're putting in your mouth? Try A.S.S. I'm sure you'll enjoy it!

  
  
**Features**
  
Adds types, as well as icons and icon colors for some ingestible items. Overhauls food and touches on potions. Alchemy ingredients coming soon...
  
  
![](https://i.imgur.com/dmafy8g.jpg)﻿![](https://i.imgur.com/vpiqJzz.jpg)
  
  
Includes built in support for food from a number of mods including all the [Creation Club](https://en.uesp.net/wiki/Skyrim:Creation_Club) addons and AE content, [Complete Alchemy & Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/19924), [Beyond Skyrim Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/10917), [Hunterborn](https://www.nexusmods.com/skyrimspecialedition/mods/7900), [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888), and more!
  
  
  
  
**Installation**
  
Install A.S.S. and its requirements with your choice of mod manager. Overwrite any other files if asked. Let me know if you have any problems.
  
  

  
1. Install [SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604), [Keyword Item Distributer](https://www.nexusmods.com/skyrimspecialedition/mods/55728) (v3.0.4+), [Inventory Interface Information Injector](https://www.nexusmods.com/skyrimspecialedition/mods/85702), [The Handy Icon Collection Collective](https://www.nexusmods.com/skyrimspecialedition/mods/90508) (v1.4.0+) and their requirements[﻿](https://www.nexusmods.com/skyrimspecialedition/mods/12604)﻿.
If your Skyrim executable is v1.5.97, install [Inventory Interface Information Injector for Skyrim 1.5](https://www.nexusmods.com/skyrimspecialedition/mods/87002).
  
2. Install [Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469) (v4.3.0+)﻿ and select either the full or light options in its FOMOD.
Install [B.O.O.B.I.E.S. (aka Immersive Icons)](https://www.nexusmods.com/skyrimspecialedition/mods/89241) (v2.1.0+)﻿.
  
3. Install Aura's Scrumptious Supplement.
(Recommended) Install [P.E.N.I.S. (Phenomenally Enriched & Nuanced Ingredients for SkyUI)](https://www.nexusmods.com/skyrimspecialedition/mods/90526)﻿ for ingredient icons.
  

  
  
  
**Compatibility**
  
A.S.S. (along with [P.E.N.I.S.](https://www.nexusmods.com/skyrimspecialedition/mods/90526)) was made specifically as a lite-weight supplementary addon for [B.O.O.B.I.E.S. (aka Immersive Icons)](https://www.nexusmods.com/skyrimspecialedition/mods/89241)﻿, per [WhereDeadAngelsLie](https://www.nexusmods.com/skyrimspecialedition/users/9170915)'s request. I've taken a few elements from my other mod, [Aura's Inventory Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/68557), and packaged them into a tight little A.S.S.
  
  
  
There are 2 main options you can go with for inventory icon overhauls:
  
  
1) The **Bundle for Aesthetic Loot Lists with SkyUI (B.A.L.L.S.)**, which includes:
  

  
* [O.C.F. (Object Categorization Framework)](https://www.nexusmods.com/skyrimspecialedition/mods/81469)﻿ [Light or Full]
[B.O.O.B.I.E.S. (Beautifully Overhauled & Objectively Better Icons Enriched for SkyUI)](https://www.nexusmods.com/skyrimspecialedition/mods/89241)﻿﻿﻿
  
* [A.S.S. (Aura's Scrumptious Supplement)](https://www.nexusmods.com/skyrimspecialedition/mods/89823)﻿﻿﻿
[P.E.N.I.S. (Phenomenally Enriched & Nuanced Ingredients for SkyUI)](https://www.nexusmods.com/skyrimspecialedition/mods/90526)
  

and their requirements. This is [WhereDeadAngelsLie](https://www.nexusmods.com/skyrimspecialedition/users/9170915)'s taste, where he has the final say on how things are implemented.
  
  
2) **Aura's Inventory Tweaks (A.I.T.)**, which includes:
  

  
* [O.C.F. (Object Categorization Framework)](https://www.nexusmods.com/skyrimspecialedition/mods/81469)﻿ [Full only]

  
* [A.I.T. (Aura's Inventory Tweaks)](https://www.nexusmods.com/skyrimspecialedition/mods/68557)

and their requirements.﻿ This is my ([GroundAura](https://www.nexusmods.com/skyrimspecialedition/users/97658973)'s) taste, where I have the final say on how things are implemented.
  
You should not use either [A.S.S.](https://www.nexusmods.com/skyrimspecialedition/mods/89823) or [P.E.N.I.S.](https://www.nexusmods.com/skyrimspecialedition/mods/90526)﻿ if you also use [Aura's Inventory Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/68557), as these mods are already integrated into A.I.T.
