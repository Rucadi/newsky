**Loading Menu Overhaul - Nordic UI**
  
  

This is a reskin of JPSteel's Loading Menu Overhaul to match the style of Nordic UI. Works with loading screen replacers such asÂ [Starlit Lakes Loading Screens](https://www.nexusmods.com/skyrimspecialedition/mods/57579) and my own [Lore-Friendly Load Screen Compendium](https://www.nexusmods.com/skyrimspecialedition/mods/138294).
