# Deadly Dragons - Legacy Settings Loader
- Author: MaskedRPGFan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/57009
This is QoL type mod adds the ability to save and load settings in MCM for [Deadly Dragons](https://www.nexusmods.com/skyrimspecialedition/mods/23723) by [moose109](https://www.nexusmods.com/skyrimspecialedition/users/4251412) using [PapyrusUtil](https://www.nexusmods.com/skyrim/mods/58705). Automatically loads saved settings at startup, so you don't have to spend time in MCM again with each new game. NO PLUGIN. NO NEW GAME NEEDED.
  
  
  
USAGE:
  
1. Install mod.
  
2. Configure "Deadly Dragons" mod in MCM.
  
3. Go to Maintenance->User Settings->Save.
  
4. Every new game will automatically load saved user settings.
  
  
**FAQ**
  
  
**Failed to save user settings.**
  
You are using wrong PapyrusUtil version. For SE is 3.9, for AE 4.+.
