# Phenomenally Enriched and Nuanced Ingredients for SkyUI (P.E.N.I.S. for B.O.O.B.I.E.S.)
- Author: GroundAura
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/90526


**Phenomenally Enriched & Nuanced Ingredients for SkyUI****An Alchemy Ingredients Addon for B.O.O.B.I.E.S.**

  
**Overview**
  

Tired of not knowing what you're putting in your mouth? Try P.E.N.I.S. I'm sure you'll enjoy it!

  
  
  
**Features**
  

Adds types, as well as icons and icon colors for ingredients.

  
![](https://i.imgur.com/OtfEtxP.jpg)
  
![](https://i.imgur.com/WjvYjRI.jpg)
  
  
Includes built in support for ingredients from a number of mods including all the [Creation Club](https://en.uesp.net/wiki/Skyrim:Creation_Club) addons and AE content, [Complete Alchemy & Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/19924), [Beyond Skyrim Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/10917), [Hunterborn](https://www.nexusmods.com/skyrimspecialedition/mods/7900), [Requiem](https://www.nexusmods.com/skyrimspecialedition/mods/60888), and more!
  
  
Keep [Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469) up to date, as additional mod support will come from that. If there any mods with ingredients that I have not covered yet (and they don't have the proper icons/colors), feel free to suggest them either here or on OCF's modpage.
  
  
  

**Installation**

  
Install P.E.N.I.S. and its requirements with your choice of mod manager. Overwrite any other files if asked. Let me know if you have any problems.
  
  

  
1. Install [SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604), [Keyword Item Distributer](https://www.nexusmods.com/skyrimspecialedition/mods/55728) (**v3.0.4+**), [Inventory Interface Information Injector](https://www.nexusmods.com/skyrimspecialedition/mods/85702), ﻿[The Handy Icon Collection Collective](https://www.nexusmods.com/skyrimspecialedition/mods/90508) (v1.1.0+), and their requirements[﻿](https://www.nexusmods.com/skyrimspecialedition/mods/12604)﻿.
If your Skyrim executable is SE/AE ﻿1.6.318-1.6.1179 or VR, install [Inventory Interface Information Injector - Alchemy Fix](https://www.nexusmods.com/skyrimspecialedition/mods/141883)﻿.
  
2. If your Skyrim executable is SE v1.5.97, install [Inventory Interface Information Injector for Skyrim 1.5](https://www.nexusmods.com/skyrimspecialedition/mods/87002).﻿
Install [Object Categorization Framework](https://www.nexusmods.com/skyrimspecialedition/mods/81469) (v4.3.0+)﻿﻿ and select either the full or light options in its FOMOD.
  
3. Install [B.O.O.B.I.E.S. (aka Immersive Icons)](https://www.nexusmods.com/skyrimspecialedition/mods/89241) (v2.1.0+)﻿.
(Recommended) Install [Aura's Scrumptious Supplement](https://www.nexusmods.com/skyrimspecialedition/mods/89823)﻿ for food/potion icons.
  
4. Install Phenomenally Enriched & Nuanced Ingredients for SkyUI.

  
  
  
**Compatibility**
  
P.E.N.I.S. (along with [A.S.S.](https://www.nexusmods.com/skyrimspecialedition/mods/89823)) was made specifically as a lite-weight supplementary addon for [B.O.O.B.I.E.S. (aka Immersive Icons)](https://www.nexusmods.com/skyrimspecialedition/mods/89241)﻿, per [WhereDeadAngelsLie](https://www.nexusmods.com/skyrimspecialedition/users/9170915)'s request. I've taken a few elements from my other mod, [Aura's Inventory Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/68557), and made them into a perfectly average size P.E.N.I.S.
  
  
  
There are 2 main options you can go with for inventory icon overhauls:
  
  
1) The **Bundle for Aesthetic Loot Lists with SkyUI (B.A.L.L.S.)**, which includes:
  

  
* [O.C.F. (Object Categorization Framework)](https://www.nexusmods.com/skyrimspecialedition/mods/81469)﻿ [Light or Full]
[B.O.O.B.I.E.S. (Beautifully Overhauled & Objectively Better Icons Enriched for SkyUI)](https://www.nexusmods.com/skyrimspecialedition/mods/89241)﻿﻿﻿
  
* [A.S.S. (Aura's Scrumptious Supplement)](https://www.nexusmods.com/skyrimspecialedition/mods/89823)﻿﻿﻿
[P.E.N.I.S. (Phenomenally Enriched & Nuanced Ingredients for SkyUI)](https://www.nexusmods.com/skyrimspecialedition/mods/90526)﻿
  

and their requirements. This is [WhereDeadAngelsLie](https://www.nexusmods.com/skyrimspecialedition/users/9170915)'s taste, where he has the final say on how things are implemented.
  
  
2) **Aura's Inventory Tweaks (A.I.T.)**, which includes:
  

  
* [O.C.F. (Object Categorization Framework)](https://www.nexusmods.com/skyrimspecialedition/mods/81469)﻿ [Full only]

  
* [A.I.T. (Aura's Inventory Tweaks)](https://www.nexusmods.com/skyrimspecialedition/mods/68557)

and their requirements.﻿ This is my ([GroundAura](https://www.nexusmods.com/skyrimspecialedition/users/97658973)'s) taste, where I have the final say on how things are implemented.
  
You should not use either [A.S.S.](https://www.nexusmods.com/skyrimspecialedition/mods/89823) or [P.E.N.I.S.](https://www.nexusmods.com/skyrimspecialedition/mods/90526) if you also use [Aura's Inventory Tweaks](https://www.nexusmods.com/skyrimspecialedition/mods/68557), as these mods are already integrated into A.I.T.
  
  
  
Also yes, P.E.N.I.S. is compatible with [C.O.C.K.S.](https://www.nexusmods.com/skyrimspecialedition/mods/81409) It's even compatible with this [other P.E.N.I.S.](https://www.nexusmods.com/skyrimspecialedition/mods/91971) if you're feeling brave.
