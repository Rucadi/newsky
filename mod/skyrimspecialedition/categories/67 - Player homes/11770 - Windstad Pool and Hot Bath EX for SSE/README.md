Mod includes the following:
  
  

  
* Heated pool and bathtub with autostrip and reequip only for followers and other NPCs
Idle markers for NPCs
  
* Outdoor kitchen and dining area
Two greenhouses with 30 plantable plots
  
* Apiary
Shore-level shelter for fishing and resting
  
* Archery range
Alchemy and enchanting tables inside the northern greenhouse
  
* Proper ambient lighting
Frostfall compatible
  
* Checked and cleaned with TES5edit and tested
(NEW) Toggable perimeter wall, lighting, and archery range.
  

  
Installation: 
  
  

  
* [size=2]Enter any interior space then make a savegame
  
Install this mod using any mod manager or manually extract and copy
  
  
Uninstallation: 
  
  

  
* Retrive all items from storage chests
Enter any interior space then make a savegame
  
* Exit Skyrim and Uninstall this mod

  
Compatibility Notes:
  
  

  
* Other mods that modify Windstad, especially the navmesh or adds more clutter to cause clipping, may not work with this mod.
Compatible with Frostfall
  
* Compatible with Windstad Mine
