# Severin Manor Has A price
- Author: AndrealletiusVIII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/62165
**Description**
  
  
As I always ended up with way too much gold at higher level, I wanted to add some more goldsinks. For this reason, I added a cost of 10k septims to purchase Severin Manor. Originally, this was just a personal edit, but people convinced me to release it as an actual mod.
  
  
After you completed the [Served Cold](https://en.uesp.net/wiki/Skyrim:Served_Cold) quest, a notice will show up at the entrance of Severin Manor. If you read it, you can talk to Adril Arano's wife,Â [Cindiri Arano](https://en.uesp.net/wiki/Skyrim:Cindiri_Arano), to buy the manor for 10,000 septims. If you do, she'll hand you the deed and a key to the house.
  
  
You can change the price to whatever you would like, using the console command:
  
  

```
set ANDR_SeverinManorPrice to X
```

Whereas X is the price. Keep in mind that the amount displayed in dialogue may not update, but the actual price will.
