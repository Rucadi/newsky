# Campfire Cabin
- Author: Cynndal
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/13372


[![](http://i.imgur.com/kPyl4AD.png)](http://www.nexusmods.com/skyrim/mods/64798)

  
  
This mod adds a small placeable cabin to the survival crafting menu of Campfire.  The cabin has see-through windows and a skylight, and is elevated on posts so you should be able to place it anywhere without grass clipping through.  It contains a bed, table with chair, 3 storage containers, a stove and an alchemy workstation.  Clicking on the tanning rack outside will allow you to pick up the cabin and move it to another location.
