# Solstice Castle SE
- Author: Lazz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/2570


Solstice Castle
  
  
![](https://staticdelivery.nexusmods.com/mods/1704/images/2570-2-1479607970.jpg)
  
  
Located in The Reach near Rebels Cairn just above Broken Tower Redoubt (see image section for map)
  
  
Compatable with [Hearthfire multiple adoptions](http://www.nexusmods.com/skyrimspecialedition/mods/3862/?) so you can move your family in (Cast the "Bless Home" spell)
  
  
The Castle must be purchased for 40k (check the sign near the door)
  
  
![](http://staticdelivery.nexusmods.com/mods/110/images/60095-2-1416392581.png)
  
  
Features
  
Main Hall - Dining, Master Bedroom, Kids Room (6 beds)
  
Archives - Library, crafting, dorms (14 beds)
  
vault - Display / Recreation area (6 beds)
  
Outdoor Smithing, Greenhouse, Pools
  
Main Castle and Archives are separate buildings but connected underground via the Vault.
  
NPC's and Auto undressing can be disabled by the buttons in the master bedroom
