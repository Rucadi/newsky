*As a big fan of the wonderful [Mirele Bismath Reborn](https://www.nexusmods.com/skyrimspecialedition/mods/93817)﻿ mod
  
I felt it was lacking a few things to add to the experience, so I got to work!
  
Thanks to**[SoftRodvt](https://www.nexusmods.com/users/2333291?tab=about+me) the original creator for permission to share*﻿
  
  
This mod adds my new **extended bath** to the Mirele Bismath Vault
  
Located at the **bottom** of the **stairs** on the **right side** is the **entrance**
  
It is fully navmeshed for followers and is contained in the same cell
  
Which made it possible for me to **ESLify** the plugin.
  
  
**The bath features several** 
  
- **Shadow casting lights**so as to give better lighting for screenshots
  
- **Decorations** including food, drinks, chairs and more
  
- **Idle markers** for npcs and also player if using [Dovahkiin can lean](https://www.nexusmods.com/skyrimspecialedition/mods/32748)**Other changes﻿**- **EVGAT Ladder** to access Mirele vault balcony
  
- **Seating on balcony** "Great to relax on for a bit :3"
  
- **More Idle markers** for npc and player
