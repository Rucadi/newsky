Crafting Pulls Resources from Containers is a phenomenal way to marry the convenience of both Craftloot and The Curator's Companion without nasty script issues. They have been tested together and work great! However they do require a bit of mucking with the ini and finding the correct references to get up and running.
  
  
This ini file contains the following containers (if I've missed any, please drop a comment!):

Limitations
  
Crafting Pulls Resources from Containers will not activate any recipes that require an ingredient to be in your inventory. The easiest example of this is the vanilla recipes for Leather. This is something the main mod would have to address if possible, but is worth noting here.
