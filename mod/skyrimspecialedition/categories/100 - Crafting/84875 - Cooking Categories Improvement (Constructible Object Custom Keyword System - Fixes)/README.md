**DESCRIPTION**
  
  
Adds sub-categories to the **cooking pot** and **baking** stations using [Constructible Object Custom Keyword System](https://www.nexusmods.com/skyrimspecialedition/mods/81409).
  
Distributes keywords to food items with [Keyword Item Distributor](https://www.nexusmods.com/skyrimspecialedition/mods/55728), while addressing some oversights.
  
Includes support for [Survival Mode](https://en.uesp.net/wiki/Skyrim:Survival_Mode), [Fishing](https://en.uesp.net/wiki/Skyrim:Fishing), [Beyond Skyrim: Bruma](https://www.nexusmods.com/skyrimspecialedition/mods/10917), [Apothecary Food and Drink](https://www.nexusmods.com/skyrimspecialedition/mods/52130), and more.
  
  
  
**INSTALLATION**
