# Exquisite NPC
- Author: AizenV
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153754


![](https://i.ibb.co/bRDkpKYY/01-Main.jpg)

  
  
  
**SkyPatcher**, this mod use SkyPatcher to apply the new visual for the NPCs, it's not directly touching the NPC data at all, meaning it should be compatible with anything, except mods that do exactly the same thing of course.
  
  
**Modular**, you can select which NPC to install in the FOMOD installation.
  
  
**Modest and simple**, I don't intend to turn everyone into supermodel here, they look good for sure, but I do only vanilla-ish modifications, about the only fancy stuff here probably is just KS Hairdos, but I chose hairstyles that at least still make sense with the rest of the game.
  
  
**ESLified**, this mod won't count toward your plugin limit.
  
  
**Use your installed skin and body type**.
  
  
**Looks their age**, young people look young, daddies and mommies look mature, and elders look old.
  
  
**Family trait**, NPCs that come from the same family tend to have a similar appearance.
  
  
 **Female Muscularity**, some female NPC have muscular body. Four level of muscularity are distributed to some suitable NPCs, ranging from sporty thieves to hardcore warriors. 
  
  
  
**NPC Covered**
  
  
Basically everyone in the game, except children and the generic NPC like guards, soldiers, bandits, etc.
  
To be precise, the unique NPCs from:
  

  
* Skyrim
Dawnguard
  
* Hearthfires
Dragonborn
  
* All Creation Club DLC
JK Skyrim
  
* Arthmoor Town mods
Alternate Perspective
  

  
  
**Additional Features**
  
  
**Remove Opposite Gender Animation**
  
Remove Opposite Gender Animation flag from all humanoid female NPCs in the game, regardless they are in this mod or not. This flag make women act like men, which is kinda repulsive for some people.
  
Done dynamically and conflict-free via SkyPatcher.
  
*Note: require [Save Unbaker](https://www.nexusmods.com/skyrimspecialedition/mods/85565) if installed mid-game.* 
  
Spoiler[![](https://i.ibb.co/fJ19Kh8/01-Remove-OGA.jpg)](https://ibb.co/XH2Ddhy)
  
**Remove Headwear**
  
Remove hats and hoods from vanilla outfits, will affect every NPC that using the outfits, regardless they are in this mod or not. Hat tends to make NPC appear bald.
  
Done dynamically and conflict-free via SkyPatcher.
  
Spoiler[![](https://i.ibb.co/ch2Bm11f/01-Remove-Headwear.jpg)](https://ibb.co/5hs0766p)
  
**Outfit Addon**
  
Give some notable NPCs outfits from some popular armor mods, completely modular and optional. You need to install the armor mods.
  
Done dynamically and conflict-free via SkyPatcher.
  
Spoiler[![](https://i.ibb.co/B5rxrZ1V/02-01-Whiterun-Lydia.jpg)](https://ibb.co/Q3CRCN5F)
  
  
  
**Install Instruction**
  
  

  
* Use a mod manager! installing manually is highly discouraged.
This mod contain literal thousands of options, so it's highly recommended to have [FOMOD Plus](https://www.nexusmods.com/skyrimspecialedition/mods/141001)﻿ or [Remember Installation Choices](https://www.nexusmods.com/skyrimspecialedition/mods/140678)﻿ for MO2 users.  
  
* Let this mod override your skin mods.
This mod can be installed anytime, but I changed some of NPC's outfit, meaning if you install this mid-game, sometimes you will find those NPC are naked. This can be easily solved by [ResetInventory](https://www.google.com/search?q=skyrim+how+to+resetinventory+console+command)﻿ command. If you install this mod in a new game, this shouldn't be a problem.
