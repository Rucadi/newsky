# Draugr Warhounds - Deadly Draugr Companions
- Author: Guitarninja
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/82350
A lightweight enemy mod that adds a new Draugr enemy type Vargr which have a chance to spawn in any place where a normal melee Draugr might. Similar to Death hounds they slow you when you are bitten.
  
  
Based on [this](https://www.nexusmods.com/skyrimspecialedition/mods/13257) mod but spawns are re-done and added to the Draugr leveled lists instead. Also adds leveled variants which will scale with the player level.
