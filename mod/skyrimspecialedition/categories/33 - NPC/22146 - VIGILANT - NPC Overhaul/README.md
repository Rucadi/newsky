# VIGILANT - NPC Overhaul
- Author: nerjaveika
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/22146
**[font=Verdana]FEATURES[/font]**
  
  
- Overhauls appearance of all NPCs in the mod "VIGILANT".
  
(~120+ NPCs - important ones have unique look, others just hair)
  
- NO ESP, only mesh changes.
  
  
I use base of NPCs from another mods like [Bijin NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/11287) (here mostly are Bijin based), [Pandorable NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/19012) (later act3 additions), and edit them look closer to original character from Vigilant.
  
Children NPCs are based on [presets by](https://www.nexusmods.com/skyrim/mods/83916)[trzcinaj](https://www.nexusmods.com/skyrim/mods/83916) from Skyrim LE.
  
  
**[font=Verdana]REQUIREMENTS[/font]**
  
  
[VIGILANT](https://www.nexusmods.com/skyrimspecialedition/mods/11849) (any language), since I changed only meshes.
  
  
For **Glenmoril** - go [here](https://www.nexusmods.com/skyrimspecialedition/mods/52319).
  
For **Unslaad** - go [here](https://www.nexusmods.com/skyrimspecialedition/mods/153179).
  
  
**[font=Verdana]INSTALLATION[/font]**
  
  
Install with preferred Mod Manager or manually drop files into Data folder.
  
  
**[font=Verdana]UNINSTALL[/font]**
  
  
It could be uninstalled safely. 
  
It won't break anything.
  
  
[font=Verdana]**INCOMPATIBILITY**[/font]
  
  
Mods that add hairlines to vanilla hair or change vanilla hair.
  
Since this mod only changes meshes, there's no esp here that adds hair, so if you use mods like Shiva Hair Replacer or Vanilla Hair Replacer
  
it may result in black face bug. Try and see, it won't break anything. 
  
  
Mods that change Vigilant NPCs. You can still use some from here, some from other mods your prefer. Just find Base ID of the NPC.
