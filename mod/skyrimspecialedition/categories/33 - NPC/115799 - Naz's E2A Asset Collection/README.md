# Naz's E2A Asset Collection
- Author: nazarethblood
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115799


**What is this?**
  
  
A carefully curated set of 4K resolution skin textures pulled from BnP - Female Skin 2.0 by [Swefrida](https://www.nexusmods.com/skyrimspecialedition/users/84577193?tab=user+files)﻿ and [Khisartin](https://www.ddsworkshop.net/)﻿. I have also included a number of bodyslide presets which I use in the screenshots of all of my followers. All of the textures and presets are being used with permission and I encourage you to show your support by going and endorsing these amazing mods!
  
  
**Why would I use an asset collection?**
  
  
Simple. The benefits to the user are two fold: 
  
  
1) Greatly reduces the amount of physical space needed when using a number of my followers as all of the diffuse, normal, subsurface and specular maps for my chosen skin are included here (there are 8 types of diffuse textures and 4 types of normal maps included).  
  
  
2) In theory, if the same assets are being referenced by multiple followers instead of being referenced individually then it should lighten the load a little for your graphics card. We are all modders here and love to mod our games to the hilt so every little bit helps in my opinion. 
  
  
**Which collection do I download?**There are two options: One is for CBBE based bodies like 3BA and the other is for UNP based bodies like BHUNP. Each has the corresponding skin textures available for that body type. Please download the asset collection that is right for you. If you use any other body type or just want to save even more space I still offer a default body option on all of my followers. 
  
  
**Why not include a body or use bodyslides?**
  
  
Honestly, I can't advocate enough for [OBody Next Generation](https://www.nexusmods.com/skyrimspecialedition/mods/77016) which allows you to have as many bodyslide presets as you want while being able to change them on the fly with the click of a button. This is how I play my game and I was inspired to do this by [Coldsun's](https://www.nexusmods.com/skyrimspecialedition/mods/109528) courage to do the same. You should really check it out if you haven't done so already!
  
  
**No plugin?????**
  
  
Correct. Everything needed other than the skin is housed with the individual follower. Yay!
  
  
**But what if I also like your original asset collection?**
  
  
I'm glad you asked. Because the folder structure of each collection is entirely different they can be used together!
