# Cuyima Interesting NPCs SE
- Author: cuyima
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/27330
[font=Georgia]DESCRIPTION[/font]
  
  
This mod overhauls a large amount of characters from Kris Takahashi's Interesting NPCs. It includes all humanoid followers, spouses as well as many quest-related characters.
  
Apart from Interesting NPCs no mod is required to use this overhaul as all necessary assets are already included.
  
The NPCs are not using a custom skin texture, so you can use whatever body and skin mods you want.
  
A full list of overhauled NPCs can be found [here](https://docs.google.com/spreadsheets/d/1GSIg3zLkUGBkkrZKbgYbJCOTbh-uLdlHw9z9d65KzKs/edit?usp=sharing).

[font=Georgia]REQUIREMENTS[/font]
  
  
The only hard requirement to use this mod is [Interesting NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/29194).
  
While not technically required, my sculpts are tailored for use with Expressive Facial Animation [Male](https://www.nexusmods.com/skyrimspecialedition/mods/19532) and [Female](https://www.nexusmods.com/skyrimspecialedition/mods/19181) Edition.

[font=Georgia]COMPATIBILITY[/font]
  
  
This mod conflicts with vanilla hair replacers such as VHR, HMB and Shiva's and you will get blackfaces regardless of load order unless you create a patch. For more information on this, see [here](https://www.nexusmods.com/skyrimspecialedition/articles/1796).
  
  
As children aren't covered by my mod, it's compatible with all overhauls out there. Load order does not matter in this case.
  
  
[size=3]IMPORTANT: If you want to mix NPC overhauls, you absolutely must make sure your load order is correct. NPC overhauls are prone to breaking if you don't know what you're doing. I am not responsible for any bugs caused by combining overhauls without taking care of proper load order and compatibility.
  
  
My mod already covers a large amount of NPCs. In case you want to cover even more NPCs, I recommend installing [Hott Interesting NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/18818) and let my mod overwrite it, so you get a mix of both.
  
  
You can also use smaller overhauls like [Interesting NPCs - 3DNPC - Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/9638), [Interesting NPCs Visual Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/40046) (not really small), [Zora Customizer](https://www.nexusmods.com/skyrimspecialedition/mods/18698) or [Amalee Customizer](https://www.nexusmods.com/skyrimspecialedition/mods/19025). These mods should be loaded after mine.
  
  
[/size]
