# GLENMORIL - NPC Overhaul
- Author: nerjaveika
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/52319


**[font=Verdana]FEATURES[/font]**
  
  
Overhauls appearance of NPCs in the mod "[GLENMORIL](https://www.nexusmods.com/skyrimspecialedition/mods/32998)". No esp.
  
  
The models will use the skin that you have installed.
  
I'm using (on screenshots) - [Bijin Skin](https://www.nexusmods.com/skyrimspecialedition/mods/20078) for females and [Tempered Skins](https://www.nexusmods.com/skyrimspecialedition/mods/7902) for Males.
  
  
The things that are in the mod: mesh geometry, hair, textures for eyes, brows, tintmasks.
  
I use base of NPCs from another mods like [Pandorable NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/19012), [Bijin NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/11287) and edit them look closer to original character from Glenmoril.
  
Children NPCs are based on [presets by](https://www.nexusmods.com/skyrim/mods/83916)[trzcinaj](https://www.nexusmods.com/skyrim/mods/83916) from Skyrim LE.
  
  
**[font=Verdana]REQUIREMENTS[/font]**
  
  
[GLENMORIL](https://www.nexusmods.com/skyrimspecialedition/mods/32998)
  
  
For Vigilant NPC overhaul - go [here](https://www.nexusmods.com/skyrimspecialedition/mods/22146).
  
For Unslaad NPC Overhaul - go [here](https://www.nexusmods.com/skyrimspecialedition/mods/153179)﻿.
  
  
**[font=Verdana]INSTALLATION[/font]**
  
  
Install with preferred Mod Manager or manually drop files into Data folder.
  
  
**[font=Verdana]UNINSTALL[/font]**
  
  
No esp. It could be uninstalled safely. 
  
It won't break anything.
  
  
[font=Verdana]**INCOMPATIBILITY**[/font]
  
  
Mods that add hairlines to **vanilla hair** or **change vanilla hair** may result in a black face.
  
Mods like [Shiva Hair Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/7922) or [Vanilla Hair Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/25574)﻿.
  
The rule here is - if they come with esp, they probably add hairlines. 
  
  
[High Poly Vanilla Hair](https://www.nexusmods.com/skyrimspecialedition/mods/41863?tab=files) or [Vanilla Hair Remake](https://www.nexusmods.com/skyrimspecialedition/mods/63979) - don't add hairlines (they change meshes and textures) and **fully compatible**.
  
  
Mods that change Glenmoril NPCs.
