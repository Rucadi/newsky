# The Red-Eyed Warrior - A Kaidan NPC Replacer
- Author: Llewellyn
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/125951


[font=Courier New]![](https://imgur.com/qmDfaLf.png)
  
[/font][font=Times New Roman]◈ ***The Red-Eyed Warrior*****is an extensive overhaul of the NPCs, items, and weapons added from the Kaidan 2 and Kaidan - Immersive Features mods OR the Kaidan Revoiced - Community Expansion (KRCE) mod. Rebuilt from scratch, this mod is a fresh update from my**[previous overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/70022)**. With more experience under my belt in creating replacers, each NPC has been hand-crafted to better fit a more modern aesthetic. For more customization, this mod has been split into separate modules that can be installed individually or all together:
  
  
KAIDAN REPLACER:**
  
  
 ﻿◈ **My own Kaidan replacer, including custom body textures.
  
 ﻿ ﻿- SOS Full, SOS Light, SAM, SAM Light, The New Gentleman, or Vanilla**
  
**[b]NPC REPLACERS:**
  
  
﻿ ﻿[/b]◈**A complete overhaul of the NPCs added in Kaidan 2 (+ Immersive Features) OR the Kaidan Revoiced - Community Expansion (KRCE) ﻿mod.**
  
 **﻿**◈**Custom body textures for all NPCs, with multiple body type support.**
  
 **﻿ ﻿- WOMEN: CBBE or UNP**
  
 **﻿ ﻿- MEN: SOS Full, SOS Light, SAM, SAM Light, The New Gentleman, or Vanilla
  
  
[b][b]KAIDAN ARMOR REPLACER:**[/b]
  
[/b]
  
 **﻿**◈ **Replaces Kaidan's armor with NordwarUA's Legacy of Ysgramor Armor.**
  
 **﻿ ﻿﻿- Meshes are prebuilt for use with OBody**
  
  
**KAIDAN NODACHI REPLACER:**
  
 ﻿﻿◈**Replaces Kaidan's nodachi with billyro's Bloodskal Blade 4. It's a separate module from the "items" category to allow people who ﻿﻿don't want to use the sword replacer to enjoy the other items.**
  
  
**[b][b]ITEMS & OTHER WEAPONS REPLACER:**[/b][/b]
  
  
 **﻿**◈**Replaces all of the items/weapons\* added in the Kaidan 2/Kaidan Revoiced - Community Expansion mod(s).
  
 ﻿﻿*\*Small exception to the Carved Talisman.***

[line]
  
![](https://imgur.com/crP63sc.png)
  
◈**As a general rule, any mods that alter the NPC head meshes (.nifs) will NOT be compatible with this mod. This means that you can only use ONE replacer at a time.**
  
  
[/font][font=Times New Roman]◈ **Make sure that you load this mod** **AFTER****any other mods that alter NPC records/FaceGen files.**
  
  
◈ **If you do not like a particular weapon/item replacer, any mod that is loaded** **AFTER****this one will take priority. So, for example, if there is a mod that replaces Kaidan's bow that you like more than the one included in this mod, make sure to load it after this mod in order for the other mod's changes to take precedence.**
  
  
  
**RACIAL SKIN VARIANCE (RSV):**
  
  
◈**All of the NPCs in this mod utilize custom textures that may break when the SPID for Racial Skin Variance is present. In order to resolve this, please download the optional patch which excludes all NPCs in both Kaidan 2 and Kaidan - Immersive Features mods from the Racial Skin Variance SPID distribution.**[/font][font=Times New Roman]**[line]
