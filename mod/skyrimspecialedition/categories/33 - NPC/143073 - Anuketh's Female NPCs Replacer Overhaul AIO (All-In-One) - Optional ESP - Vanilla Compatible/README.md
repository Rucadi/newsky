**F.A.Q.**
  
  
+ What is the difference between ESP and NO ESP?
  
- ESP is made so you can force the game to load this mod last. Any previous replacer of this NPC will be overwritten. No ESP is for those who haven't modded this NPC yet and are using the vanilla version. I personally use the NO ESP version.
  
  
+ There is a conflict between your replacers.
  
- No more. This AIO version aims to avoid that warning. If you keep experiencing the warning, please read the description above.
  
  
+ Will this affect quests/dialogs/behaviors/etc?
  
- No, it only changes the appearance of this NPC.
  
  
+ This NPC does not look like your screenshots.
  
- Not all textures and ENB look the same. I use BnP skin + PI-CHO ENB for SSE. I no longer play LE. Beware that if you don't install at least a skin mod, all these overhauled NPCs will use Skyrim's vanilla textures.
  
  
+ Have you modified the vanilla weight or skin tone?
  
- No, they keep the same values. Otherwise, I wouldn't be able to make the No ESP version without a neck seam/gap. If you experience any of these problems, something in your loading order is interfering with the vanilla values.
  
  
+ Why is not ESL / ESPFE?
  
- This is a work in progress and will have several updates. I have not converted the plugin to ESL format, because the records will be constantly changing with each update and it is dangerous for already saved games. You can flag them as ESL on your own, but I won't be able to provide support if you break your game.
  
  
+ Any known incompatibilities?
  
- None except for UUNP users. Apparently one user reported a gap between the heads and their UUNP body and this cannot be fixed unless I manually sculpt all heads making the neck longer. And I don't have extra time for that.
