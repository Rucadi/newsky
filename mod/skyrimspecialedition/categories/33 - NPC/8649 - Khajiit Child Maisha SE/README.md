# Khajiit Child Maisha SE
- Author: tktk
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/8649


日本語の解説は[ブログ記事](https://tktk1.net/skyrim/mymod/prettykhajiit/)にて
  
This mod adds a Khajiit child whose name is Ma'isha.
  
You can adopt her such as other urchins.
  
  
**She can be found in Crabber's Shanty.**
