# Bosmer NPCs have antlers
- Author: wankingSkeever
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/33349
[![](https://i.imgur.com/iBvSVjB.png)](https://www.patreon.com/wSkeever)﻿﻿﻿

**Description**
  

  
* This mod gives Bosmer (Wood Elf) NPCs random antlers like Bosmers in ESO.
These antlers are not armors and are not tied to any slots. They are [Visual Effects](https://ck.uesp.net/w/index.php?title=Visual_Effect&mobileaction=toggle_view_desktop).
  
* [Supports decapitation](https://staticdelivery.nexusmods.com/mods/1704/images/33349/33349-1730175629-1192498504.png)﻿.
There are 4 types of antlers they can have. (Optional files to make only 50% or 80% of Bosmers have antlers)
  
* No records edited.
Compatible with all race mods
  
* Not compatible with mod-added Bosmer NPCs that already have antlers.
ESL flagged ESP
  
* Player character is not affected. If you want the player character to have this, use [cuyima](https://www.nexusmods.com/skyrimspecialedition/users/12436273) and [Chiinami](https://www.nexusmods.com/skyrimspecialedition/users/57236032)'s [Cuyi's Bosmeri Antlers - SSE](https://www.nexusmods.com/skyrimspecialedition/mods/26173)﻿
In the extremely rare event that the player character got bosmer antlers stuck on head, use [Universal Unwanted Effects Clearer](https://www.nexusmods.com/skyrimspecialedition/mods/93977) (this is an engine bug with [Visual Effects](https://ck.uesp.net/w/index.php?title=Visual_Effect&mobileaction=toggle_view_desktop), not unlike how Draugr eyes or Dwemer steam can get stuck on the player).
