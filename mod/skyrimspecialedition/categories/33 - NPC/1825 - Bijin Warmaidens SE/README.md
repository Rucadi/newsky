**[Summary]**
  
This Mod changes the appearances of Lydia, Jordis, Iona, Aela, Mjoll, Uthgerd, Ria, Frea, Rayya, Njada,
  
Annekke, Adelaisa, Jenassa, Illia, Delphine, Ingjard, Aranea, Eola, Borgakh, Brelyna, Beleval and Ugor. 
  
This Mod is **STANDALONE** which means no other Mods will be required.
  
You can install them with no problem whatever body types[color=#cc0000](UNP, CBBE or Vanilla etc...) you are using.
  
[/color]This mod is made as Fomod installer so please install this mod with a mod manager.
  
The skin textures included in this mod are compressed and scaled down for all users.
  
Face, body diffuse maps and body normal map => 2k
  
Hand diffuse map and face normal map => 1k
  
(All diffuse maps are compressed, all normal maps aren't compressed.)
  
  
Install the high-res textures option if you want to give them the better skin.
  
  
  
All characters are compatible with Werewolf, Vampire Lord and Werebear forms with mods such as AFT.
  
  
Ingjard and Beleval

Compatible with Werewolf and Vampire Lord forms.
  
  
Frea

However I only tested the Werewolf form in my game.

Install Bijin AIO SE or Bijin AIO SE for USSEP in the OPTIONAL FILES.

[Solutions to problems]
  
  
-- If the character's face gets black or dark --
  
・Check the load order(Use LOOT or move their esp to the bottom).
  
・Reinstall by overwriting all.
  
・Make sure the Mods related to this mod's characters.
  
・Load Bijin Warmaidens.esp after EEO's if you use it and this mod's elves have any problems.
  
  
  
-- If there is a neck gap on the character --
  
  
[1]
  
Open console and click the character precisely
  
[2]
  
 Type setnpcweight xx (xx has to be the default weight of that character. See the following)
  
[3]
  
  Neck seam will be fixed but the character face will be broken, so type enable and disable to reset once.

**[size=4][How to change their body types]**
  
  
[1]
  
Copy the following 2 body meshes from the body mod which you want to use. 
  
    If you want to use the same body as player character's, copy from Data/meshes/actors/character/character assets/here
  
  
femalebody\_0
  
    femalebody\_1
  
  
[/size][2]
  
Overwrite character's mesh files where Data/meshes/actors/character/Bijin Warmaidens/here by [1]'s files.
  
  
Copy and overwrite the following files as well if the hands and feet have any seams on the characters.
  
femalefeet\_0
  
    femalefeet\_1
  
    femalehands\_0
  
    femalehands\_1
  
  
[3]
  
Done.
