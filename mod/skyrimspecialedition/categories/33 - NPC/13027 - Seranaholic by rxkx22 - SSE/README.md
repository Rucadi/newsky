# Seranaholic by rxkx22 - SSE
- Author: rxkx22 - Ported by bchick3 and elrizzo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/13027
This is a port to SSE of the original Skyrim mod [Seranaholic](https://www.nexusmods.com/skyrim/mods/38326) with permission from the original authors.
  
  
**Replaces the look of NPC Serana and\or Valerica**
  
  
 You may not redistribute, upload or otherwise use the assets of this mod in any way without express permission from the original authors.
  
Mod cannot be uploaded anywhere else other than nexus without express permission from the authors.
  
  
**Ignore the requirements listed in the original description (other than Dawnguard)**
  
  
DO NOT attempt to install the oldrim mods mentioned in the original description as it probably wont work at best and at worst it will destroy the universe.
  
  
**Body Type:** CBBE, UNP and in some Versions 7base
  
**Body textures:** UNP or CBBE - **Select in the fomod**
  
**Default Armor:** Vanilla

[How to change her body type to your liking]
  
  
Spoiler
  
  
  
[1]
  
Copy the following 2 body mesh which you want to use from body mod.
  
    If you want to use the same body as player character, please copy from
  
Data/meshes/actors/character/character assets/here
  
  
    femalebody\_0
  
    femalebody\_1
  
    
  
[2]
  
Overwrite her body mesh files where Data/meshes/actors/character/Serana/here by [1]'s files.
  
  
[3]
  
Done.
  
  
\*Copy and overwrite the following files too if her hands and feet have any seams
  
  
femalefeet\_0
  
femalefeet\_1
  
femalehands\_0
  
femalehands\_1 
  
  
\*You have to change textures as well by yourself if you want to use the
  
special body that doesn't correspond to UNP or CBBE textures.
  
All characters have own textures where
  
Data/textures/actors/character/Serana/here, so you just replace these files if you want to change.
  
You can use these ways for all the standalone characters in skyrim.
  
  
  
[How to apply "No More Ugly Bronze Shine" to her]
  
  
Spoiler
  
  
  
Overwrite her following files by that mod's files.
  
femalebody\_1\_sk.dds
  
femalehands\_1\_sk.dds
  
femalehead\_sk.dds
  
  
[1] Download that mod manually
  
[2] Choose 01 - UNP A or 02 - CBBE A (I think whatever is fine as long as
  
it's written UNP or CBBE), whether UNP or CBBE depends on Serana body you use.
  
[3] Pick up 3 files written above and overwrite Data/textures/actors/character/Serana/here
