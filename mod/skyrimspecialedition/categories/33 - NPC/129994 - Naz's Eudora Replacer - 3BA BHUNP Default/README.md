![](https://staticdelivery.nexusmods.com/mods/1704/images/129994/129994-1727385897-1844368775.png)﻿﻿

This is a replacer for Eudora, The Dibellan Shieldmaiden, by ThatFernGuy and requires SkyPatcher. Why SkyPatcher you might ask? Because it makes updating the replacer unnecessary. Even if the follower mod gets updated it will not require an update of the replacer mod because of how SkyPatcher works. So how does it work exactly? Standard replacers use the intended follower mod as a master and reference the actor record directly. SkyPatcher only requires a generic actor record to be created that has the appearance of the replacer and then clones the appearance and applies it to the intended follower at game start. Even if you have another standard replacer installed for the same follower the SkyPatcher replacer will "win" so you don't need to worry about the dark face bug. A big thank you to [coldsun1187](https://www.nexusmods.com/skyrimspecialedition/users/9762372) for this awesome idea!
