# Stendarr Rising - Carcette Returns
- Author: asdasfa
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/77968


**Requirements:**
  
**[Stendarr Rising - The Hall of the Vigilant Rebuild](https://www.nexusmods.com/skyrimspecialedition/mods/49346)**
  
  
**Description:**
  
After rebuilding the Hall of the Vigilant and getting all the recruits, Carcette returns and becomes available as a follower.
  
To trigger the changes, **after meeting the requirements, just walk towards the exit door of the hall** and Carcette should appear just outside the hall.
  
**This mod uses the original Carcette NPC instead of creating a new clone**.
  
  
**Compatibility:**
  
This mod changes Keeper Carcette's stats and AI Package, so any mod that touches Carcette (e.g. appearance mods) will need a patch.
  
**A patch for [Kalilies NPCs](https://www.nexusmods.com/skyrimspecialedition/mods/30247) is available in the downloads**.
  
**A patch for [Dibella's Blessing](https://www.nexusmods.com/skyrimspecialedition/mods/82606) is available in the downloads.**
  
**A patch for [Valiant Vigilants](https://www.nexusmods.com/skyrimspecialedition/mods/126627) is available in the downloads.**
  
**Keeper Carcette Survives** - Creates a new Keeper Carcette NPC, so you're gonna have double Carcettes in the world if you have this installed too.
