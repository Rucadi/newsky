# HDT-SMP Traveler's Backpack All Colors Crafting and Rebalancing
- Author: Fuse00 - SonderBain - Miandru - VishVadeva50
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/111906


**HDT-SMP Traveler's Backpack - All colors, Crafting and Rebalancing**Implemented fixes from [VishVadeva50's](https://www.nexusmods.com/skyrimspecialedition/mods/67445) version and [SonderBain's](https://www.nexusmods.com/skyrimspecialedition/mods/70424) all colors patch.
  
Added immersive enchants on all backpack types.
  
Created ground/world models for each variant (No more dovahkiin stick figure in inventory)
  
  
**5 Variants in all colors.**

  
* **[b]Traveler's Backpack**: [/b]100 Carry weight, 25% Stamina Regeneration Reduction and Bedroll Rested All skills improve 5% faster.
**Adventurer's Backpack:** 75 Carry weight, 20 Stamina Increase, 25% Stamina Regeneration Reduction and Bedroll Rested All skills improve 5% faster.
  
* **Mage's Backpack:** 75 Carry weight, 20 Magicka, 25% Stamina Regeneration Reduction and Bedroll Rested All skills improve 5% faster.
**Thief's Backpack:** 75 Carry weight, 10% Fortify Lockpicking, 25% Stamina Regeneration Reduction and Bedroll Rested All skills improve 5% faster.
  
* **Hunter's Backpack:** 75 Carry weight, 10% Fortify Archery, 25% Stamina Regeneration Reduction and Bedroll Rested All skills improve 5% faster.

  
All variants are craftable with the following materials at a smithing forge.

  
* **1** Steel Ingot
**4** Leather Strips
  
* **3** Leather

  
**Requirements:**
  
[Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339) is required for physics to work.
  
  
**Installation:**
  
Download main file and install in MO2 or Vortex like any other mod.
  
If you don't want the 25% Stamina Regeneration Debuff also install the optional file and let it overwrite the main mod.
