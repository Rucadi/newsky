**Conflicts**
  
- [eFPS - Exterior FPS boost](https://www.nexusmods.com/skyrimspecialedition/mods/54907) - you will get flickering and  disapearing objects as you move around the Stronghold in some minor areas, i have made compatibility patch to fix this, you can find it in the optional files tab.
  
  
- [Spice of Life - Orc Strongholds SE](https://www.nexusmods.com/skyrimspecialedition/mods/22178) - There are floating leaves and a clipping floor near my newly placed walls when this mod is running alongside my mod, a Patch is provided in the optional files tab.
  
  
- [Lux Via](https://www.nexusmods.com/skyrimspecialedition/mods/63588) - You will see floating objects near the farm area if you use Lux Via with my mod, a Patch is provided in the optional files tab
  
  
- [Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095) - If you have Lux Orbis and my mod the campfire will have been moved back to vanilla location, there is a patch for this in the optional files tab.
  
 
  
- Any mod that alters the exterior cell of Narzulbur will be incompatible.
