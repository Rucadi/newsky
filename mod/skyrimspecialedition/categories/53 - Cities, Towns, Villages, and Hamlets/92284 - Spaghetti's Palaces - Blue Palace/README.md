# Spaghetti's Palaces - Blue Palace
- Author: RubberSpaghetti
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/92284


Have you ever wanted the Palace interiors to look nicer, but don't like the associated FPS loss or incompatibilities that come with major overhauls? Then this mod is for you!
  
  
This mod adds some light clutter around the Blue Palace. It doesn't go overboard adding thousands of things that you will never see or notice, which causes FPS loss, and it also doesn't completely change the layout of the palace, causing massive compatibility issues. Instead, I focused on adding easily noticed details in the major sections of the palace, providing and updated look to the Blue Palace, while still maintaining that vanilla feel!
  
  
This overhaul doesn't edit any vanilla records, it only adds new ones. This means it's great for compatibility! You can combine this mod with other mods that affect the Blue Palace without worrying about which navmesh edits will win, or this mod removing an object that another mod relies on, etc! If there are some conflicts, you can request a patch from me to see about addressing the problems. Then you get to use both mods, and nothing breaks!
  
  
The file is an .esp flagged as an .esl, so it won't take up any space in your load order!
