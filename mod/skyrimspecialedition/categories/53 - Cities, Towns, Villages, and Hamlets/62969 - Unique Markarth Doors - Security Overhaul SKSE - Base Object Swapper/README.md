**Description**
  

  
* Replaces some Dwemer doors in Markarth with new unique doors with [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿
    
  [list]
No cell edits
  
* No worldspace edits

[\*]Based on Markarth specific door from the original game that was only used twice: [Silver-blood Inn](https://en.uesp.net/wiki/Skyrim:Silver-Blood_Inn)﻿ and [Treasury house](https://en.uesp.net/wiki/Skyrim:The_Treasury_House)﻿
  

* Trivia: in the original game, Silver-blood Inn's exterior uses a Markarth door for the entrance while its interior uses a Dwemer door for the same door

[\*]Contains 4 new doors:
  

* Beeg door
Beeg load door
  
* Smol door
Smol load door
  

[\*]Doors to actual Dwemer dungeons like [Markarth Ruins](https://en.m.uesp.net/wiki/Skyrim:Cidhna_Mine#Markarth_Ruins)﻿, [Dwemer Museum](https://en.uesp.net/wiki/Skyrim:Understone_Keep#Dwemer_Museum)﻿, and [Nchuand-Zel](https://en.uesp.net/wiki/Skyrim:Nchuand-Zel)﻿ etc are not changed
  
[\*]Uses your own textures. Screenshots use [Xtudo's Ancient Dwemer Metal Update](https://www.nexusmods.com/skyrimspecialedition/mods/38845) for the irondoor2 texture, specifically [this file](https://www.nexusmods.com/Core/Libs/Common/Widgets/DownloadPopUp?id=257721&nmm=1&game_id=1704)
  
[\*]Works great with [Security Overhaul SKSE - Regional Locks](https://www.nexusmods.com/skyrimspecialedition/mods/62781)﻿﻿. The new doors will use the [Markarth regional lock](https://staticdelivery.nexusmods.com/mods/1704/images/62781/62781-1643543281-1426965276.png)
  
[\*]Edits some door dimensions so it no longer clips with the door frame
  
[\*]ESL
  
[\*]Fomod installer with open cities/no open cities option
