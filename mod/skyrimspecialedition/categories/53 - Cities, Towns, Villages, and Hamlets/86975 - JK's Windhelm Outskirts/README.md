﻿**- Mod Description -**[font=Trebuchet MS][/font]
  
A lore-friendly overhaul of the Windhelm outskirts, including the Talos Shrine, the Stables and the 3 farmhouses ( Brandy Mug, Hlaalu and Holy Frost)
  
  
[font=Trebuchet MS]**- Performance -**
  
  
My Rig: Asus P8B75-M/CSM Motherboard, Intel I7 2600 @ 3.4 GHz
  
16 Gb DDR3 1333 Mhz RAM, NVidia GTX 980TI 6 Gb, running alongside Aequinoctium ENB + 2048 Textures and I get 60 FPS average in interiors, 40 to 50 in exteriors.
  
[/font]
  
[font=Trebuchet MS]**- Installation -**
  
  
[/font][font=Trebuchet MS]Install with your preferred mod manager.
  
  
[/font]**- Requirements -**
  
  
- Skyrim up to date, will work with both SE and AE.

[/font]
  
[JK's Windhelm Outskirts Patch Collection](https://www.nexusmods.com/skyrimspecialedition/mods/87964/) by Czasior.
