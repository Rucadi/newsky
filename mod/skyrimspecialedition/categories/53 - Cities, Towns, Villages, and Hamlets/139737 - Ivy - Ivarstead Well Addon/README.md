# Ivy - Ivarstead Well Addon
- Author: Ivy
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/139737


**Mod Description**
  
Add a simple, lore-friendly well to Ivarstead that fits naturally into the village. It blends in with the environment and provides a small but meaningful addition to the settlement. The well is fully compatible with Sunhelm, making it a practical addition for survival-focused playthroughs.
  
  
**Features**
  
**Custom Mesh:** A unique well mesh created by [jet4571](https://www.nexusmods.com/skyrimspecialedition/users/2811323).
  
**Seasonal Support:** Changes appearance with the seasons, blending naturally with the environment year-round.
  
  
﻿**Installation**
  

  
* Download and install the main mod file using your preferred mod manager.
During installation, select the appropriate patches from the FOMOD installer for your load order.
  
* Ensure the patches are loaded after the main mod in your load order.
