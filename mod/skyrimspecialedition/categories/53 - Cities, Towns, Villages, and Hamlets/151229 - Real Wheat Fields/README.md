**Description**
  
  
I always disliked the sad forlorn wheat plants standing there in an open field. Whoever was responsible for the farm design at Bethesda either had no time whatsoever or didnt even touch grass once in his life. So let me fix that issue with this mod!

[/color]**Manual installation:
  
  
1. Download the mod file from the Nexus Mods website and extract it to a folder.
  
2. Copy the contents of the "Data" folder from the mod folder and paste them into the "Data" folder of your Skyrim installation directory.
  
3. Activate the mod by enabling the plugin in the game launcher
  
  
Mod manager installation:
  
1. Download and install a mod manager, such as Vortex or Mod Organizer
  
2. Download the mod file from the Nexus Mods website and add it to the mod manager's "Mods" folder.
  
3. Install the mod using the mod manager's interface.
  
4. Activate the mod in the mod manager's load order list.
