# Alduin Bane Fix
- Author: kirito
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/80927


I got tired of using the console to skip tasks, so I made this
  
I will try to fix the following issues.
  
  
  
1: Alduin does not land on the wall
  
2: Hakon does not draw his weapon (leading to 1)
  
3: Felldir gets stuck in Alduin's body (resulting in inability to read scrolls)
  
  
Keep 'Felldir' away from 'Alduin' in battle
  
Will not get stuck in Alduin's body and unable to open the scroll due to joining the battle
  
So that 'Felldir' can open the scroll at the right time
  
  
'Hakon' to correctly see 'Alduin' instead of staying in place
  
  
Allow 'Alduin' to enter the plot without any plot bugs caused by not flying over
  
  
  
  
Installation
  
  

  
* Can be installed mid-game

  
Uninstall
  
  
  .    Anytime
