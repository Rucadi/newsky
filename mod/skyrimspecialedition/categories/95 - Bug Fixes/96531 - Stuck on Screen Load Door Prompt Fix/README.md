**Description**
  

  
* If an auto load door prompt is displayed and the player loads an earlier save or is teleported away, then the auto load door prompt is stuck on the HUD.
The only way to clear this from the HUD is to restart the game or to find another auto load door to display its prompt. Loading an earlier save won't help.
  
* This mod fixes it. Whenever you go to a new location or load a save, the auto load door prompt is cleared from the HUD.
ESL.
