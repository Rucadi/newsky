# Assorted mesh fixes
- Author: wankingSkeever
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/32117


[![](https://i.imgur.com/iBvSVjB.png)](https://www.patreon.com/wSkeever)﻿﻿﻿

Things addressed in this mod:
  

  
* Incorrectly flagged meshes not casting shadows, receiving shadows
Incorrectly flagged meshes not contributing to depth buffer, and thus has incorrect depth of field
  
* Incorrectly flagged meshes not being reflected in water
Missing/inaccurate collision
  
* Bad normals and tangents
Bad vertex colors
  
* Strange lighting flags
Bad geometry causing Z-fighting
  
* Broken UVs
Broken animations
  

**FAQ:**
  
> ﻿Q: This conflicts with some other mod
>   
> A: Just pick whatever mod's mesh you want to win. SMIM changes are already included, so put this after that.
>   
>   
> Q: The LE version is broken.
>   
> A: I am not responsible for the LE version. I am not the person who ported it.
>   
>   
> Q: I don't like some of the fixes.
>   
> A: You can find the file you dislike and hide them in Mod Organizer, or simply delete them.
>   
>   
> Q: What are parallax meshes?
>   
> A: Don't install parallax meshes unless you know what you are doing. You have 2 options:
>   
>
> 1. Ensure you have the following requirements or your game will look bugged:[list]* Parallax meshes requires parallax textures
> Parallax meshes requires multipass snow/ash shader
>   
> * Parallax meshes requires ENB or parallax shader fix
> If any of this is meaningless to you, don't install it
>   
> * I am not responsible for your incompetence if you run into issues with this
>
> 2. Ensure you have [Parallax Fixer](https://www.nexusmods.com/skyrimspecialedition/mods/79473), which will prevent bugged graphics if you don't have the above requirements.[/list]
