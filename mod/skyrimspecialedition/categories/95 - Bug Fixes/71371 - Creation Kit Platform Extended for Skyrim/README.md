 
Various patches and bug fixes for the **Creation Kit SE** to make life easier.
  
A collection of modifications, enhancements, and reverse engineered resources for **SSE CK** version **1.5** and **1.6**.

  
  
[font=Bebas\_Neuebook]About[/font]
  
The mod has been rewritten under a different name **"Creation Kit Platform Extended"** and continues the work of the mod [SSE Creation Kit Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/20061). Supports all version **CK**.
  
The powerful open-source platform makes numerous fixes and improvements to the **Creation Kit** editor from **Bethesda**. This is a continuation of two previously archived projects, as I am already confused in the code. The new implementation is based on the principle of modularity, which will allow for better control of development.
  
  
[font=Bebas\_Neuebook]Issue[/font]
  

  
* This mod is incompatible with **Unicode patch**, if you want to use it, the *bUnicode* option must be **false**. *(this is relevant only for v1.5.73.0, patch was no longer developed and is outdated)*
**CK**does not work well with **[b]ENBSeries** [/b]*([v0.338](http://enbdev.com/mod_tesskyrimse_v0338.htm)﻿ or newer this fixed and disabled when Creation Kit is running)*. Use **ckpe\_loader.exe**.
  

  
[font=Bebas\_Neuebook]Installation: MANUAL[/font]
  

  
* Delete all previous versions of the mod and its libraries.
Extract the zip contents to your **Skyrim SE/AE** directory containing **CreationKit.exe**.
  
* Run the **Creation Kit**.
Done.
  

  
[font=Bebas\_Neuebook]Installation: Vortex[/font]
  
v0.5 and older, you need to specify that the mod type is **Engine Injector**.
  
v0.6 installation as **Engine Injector** automate on Vortex [v1.14](https://github.com/Nexus-Mods/Vortex/wiki/File-Based-Override-Instructions).
