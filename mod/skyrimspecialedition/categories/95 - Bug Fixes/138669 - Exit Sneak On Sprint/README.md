# Exit Sneak On Sprint
- Author: Quantumyilmaz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/138669


In Vanilla Skyrim, trying to sprint while sneaking does nothing (except triggering a sneak roll if you have the perk).
  
This mod fixes that.
  
  
Press your sprint key while moving and start sprinting - no need to manually exit sneak mode first.
  
You will need to hold the sprint key slightly (0.25 sec).
