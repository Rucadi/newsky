# DALC Fix KreatE Preset
- Author: Leo
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/103052


**Description**
  
  
Skyrim has a buggy Directional Ambient Lighting Color system, and [KitsuuneNivis](https://www.nexusmods.com/skyrimspecialedition/users/49767351) amazing mod [KreatE](https://www.nexusmods.com/skyrimspecialedition/mods/83757) has a fix for that. See the screenshots tab to see the difference that it makes. The fix is a great addition for Community Shaders user and ENB user.
  
  
**Requirements** 
  
  
[Native EditorID Fix](https://www.nexusmods.com/skyrimspecialedition/mods/85260)
  
[KreatE](https://www.nexusmods.com/skyrimspecialedition/mods/83757) 
  
  
**Instalation** 
  
  
Install KreatE and its requirements
  
Install this preset
  
Press "END" key to open KreatE UI
