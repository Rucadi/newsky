# Unofficial Material Fix
- Author: Elzee
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/21027
[font=Arial]There're countless material bugs in vanilla game.
  
It will be a bad feeling when you run on a stone road but can't hear any footstep sounds.
  
Even bad when you're slashing a wooden chest by a sword but can't feel any collision sound and sparks.  
  
This mod is a project tries to fix wrong Havok material marks on statics and landscapes in vanila game.
  
  
No script.
  
No esp/esm/esl.[/font]
  
  
  
[font=Arial]**REQUIRED / [b]RECOMMEND**[/b][/font]
  

  
* [font=Arial][SMIM](https://www.nexusmods.com/skyrimspecialedition/mods/659) - Required. This mod is based on vanilla game and SMIM meshes[/font]

* [font=Arial][USSEP](https://www.nexusmods.com/skyrimspecialedition/mods/266) - Not required but strongly recommend. USSEP also fix many material bugs.[/font]
