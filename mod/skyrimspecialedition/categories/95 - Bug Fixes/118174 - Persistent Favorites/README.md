# Persistent Favorites
- Author: Quantumyilmaz
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/118174


When you favorite/hotkey an item/spell, it will be tracked by this mod: Any time it enters your inventory, it will be automatically favorited/hotkeyed again, if it was not already.
  
When you unfavorite an item/spell, it will be untracked if it was previously tracked.
  
  
**Works in SE (1.5.97), AE (1.6.1170) and VR.**
