# Dragon Mound Dirt FIX
- Author: RemmyKun
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/69119
The dragon mounds' dirt mesh has some transparency (or holes) which I cannot remove without messing out the blending around the borders.
  
In vanilla this problem is alleviated because many dragon mounds will have another layer of dirt underneath so you can't see the void. But as you can see in the images, this isn't always the case (**e. g. Labyrinthian**).
  
  
v2.0: and even when they have another layer of dirt underneath, it may look bad because it uses different textures, because you can see the grass moving under the main layer, etc... Now it will look much better everywhere, not only at Labyrinthian.
  
  
**I modified the dragon mound meshes to ensure there is always another layer of dirt underneath which will always use the same textures and will block the view of any objects you are not supposed to be able to see.**
  
Now you won't see the void ever again! :)
