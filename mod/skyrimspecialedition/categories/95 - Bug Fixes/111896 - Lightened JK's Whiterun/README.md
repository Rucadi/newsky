For an in-depth description you can see what a mod like this usually does--> [font=Arial][size=3][font=Arial][size=3][font=Arial][size=3] [link](https://www.nexusmods.com/skyrimspecialedition/mods/111475).[/font][/size][/font][/size][/font] In this case it would be impossible to do the same thing that [Lightened Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/111475) does with invisible objects so what the mod does is reduce the amount of clutter and less visible objects in a way that has no visual impact at all. You will say may how many fps you gain ;). This mod works for both [JK's Whiterun](https://www.nexusmods.com/skyrimspecialedition/mods/2259) and [JK's Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/6289) (only for the portion of Whiterun city). This mod use [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805).
