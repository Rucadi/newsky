# Reliquary of Myth - Fishing Patch
- Author: Kriana
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/58543


If you have Reliquary of Myth installed into your game as well as the CC addon "Fishing" the Warlock Ring will no longer be able to work. This patch aims to fix that.
