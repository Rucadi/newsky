# Cure Poison Potion Fix
- Author: me
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/146226


Fixes Cure Poison potions not curing poisons from some effects
  
With this mod Cure Poison potions will now dispel any hostile effect that is resisted by poison resistance
  
  
Now these effects are dispelled by Cure Poison potions:
  
Frostbite Spider Spit
  
Chaurus Spit
  
Spriggan Claws
  
Afflicted Spray
  
Poison Gas
  
Poisoned Arrows
  
Poison Spells/Enchantments
  
  
Compatibility:
  
This mod edits the AlchCurePoison effect and should be loaded as late as possible to override other mods
  
Mods that add new Cure Poison effects will not be affected
  
Mods that add new Poison effects will be affected
