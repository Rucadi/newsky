# Goldenhills Plantation (Farming CC) Reset Patch
- Author: Dovahzard
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/105134
Simply sets the interior of the main house in the Farming CC (Goldenhills Plantation) to never reset.  Additionally the two barrels that was not safe storage was changed to never respawn so it is safe to store any items in any of the containers in the main bunk.
  
  
Requires Farming Creation Club Content.  
  
Credit given to Creation Club Content Authors and Bethesda.
