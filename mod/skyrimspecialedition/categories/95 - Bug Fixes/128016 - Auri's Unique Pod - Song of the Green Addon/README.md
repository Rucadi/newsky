# Auri's Unique Pod - Song of the Green Addon
- Author: biggieboss
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128016


This is a very simple mod that just changes Auri's pod to use the same bark on the outside as it does inside. Useful if you have a mod such as Rally's Solstheim which changes the default Telvanni wood textures. (in my case it was resulting in a blue pod)
