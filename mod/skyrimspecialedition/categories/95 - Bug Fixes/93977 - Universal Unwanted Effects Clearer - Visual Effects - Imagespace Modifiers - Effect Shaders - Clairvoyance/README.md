**Description**
  

  
* For various reasons, unwanted effects can get stuck on the player:
    
  [list]
[Visual Effects](https://ck.uesp.net/wiki/VisualEffect_Script)﻿
  
[list]
  
* Draugr eyes
Dwarven Centurion and Sphere steam and sparks
  
* Wispmother fog
Spriggan bugs
  
* Atronach effects
Etc...
  

[\*][Imagespace modifiers](https://ck.uesp.net/wiki/ImageSpace_Modifier)﻿
  

  
* Such as the annoying blur effect from [Boethiah's Calling](https://en.uesp.net/wiki/Skyrim:Boethiah%27s_Calling)
    
  > Upon talking with Boethiah your vision field may be stuck in the distorted oval it creates when talking to the daedric prince.
Etc...
  

[\*][Effect shaders](https://ck.uesp.net/wiki/EffectShader)﻿
  

* Like from magic and shit.

[\*][Clairvoyance](https://en.uesp.net/wiki/Skyrim:Clairvoyance_(effect))﻿
  

> Saving the game while Clairvoyance is active may make the shown path remain permanently.

* This mod provides functionality to clear any Clairvoyance trails in the **currently loaded area only**.

[\*][Actor Alpha](https://ck.uesp.net/w/index.php?title=SetAlpha_-_Actor)﻿
  

* Changed by ghosts effects, may get stuck if not set to 1 after the effect is finished.

[/list][\*]This mod works with mod-added [visual effects](https://ck.uesp.net/wiki/VisualEffect_Script)﻿, [effect shaders](https://ck.uesp.net/wiki/EffectShader)﻿, and [imagespace modifiers](https://ck.uesp.net/wiki/ImageSpace_Modifier)﻿.
  
[\*][USSEP](https://www.nexusmods.com/skyrimspecialedition/mods/266)﻿ blocks a lot of these from happening, but doesn't block mod-added effects.
  

* This mod also adds support for mod added effects (and imagespace modifiers and effect shaders) to the [USSEP](https://www.nexusmods.com/skyrimspecialedition/mods/266)﻿ book in the [Temple of Kynareth](https://en.uesp.net/wiki/Skyrim:Temple_of_Kynareth)﻿.

[\*]Optional ESL
  
[\*]Optional spells to remove unwanted visual effects from player or other NPCs.
  
[/list]**Installation/Uninstallation/Update**

  
* Whenever doesn't matter. As long as you're not currently running the clearing process.

**Note**

  
* This mod will clear out all [visual effects](https://ck.uesp.net/wiki/VisualEffect_Script)﻿, [effect shaders](https://ck.uesp.net/wiki/EffectShader)﻿, and [imagespace modifiers](https://ck.uesp.net/wiki/ImageSpace_Modifier), whether or not you want them there.
If you want some effect to remain, you might need to reapply them.
  

**Usage**

  
* If you have USSEP, simply use the book in the [Temple of Kynareth](https://en.uesp.net/wiki/Skyrim:Temple_of_Kynareth)﻿
If you picked the spell option in the FOMOD, you can use the spells under restoration.[list]
  
* To remove the spells, use console command:

  ```
  stopquest ClearShadersQuest
  ```
To add the spells back, use console command:

```
startquest ClearShadersQuest
```
