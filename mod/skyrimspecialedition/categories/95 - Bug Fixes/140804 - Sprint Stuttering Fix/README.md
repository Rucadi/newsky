# Sprint Stuttering Fix
- Author: Antonix35
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140804


![](https://i.imgur.com/ziWaxV4.png)﻿

[font=Georgia]Description[/font]

  
[font=Arial]This SKSE plugin fixes annoying twitching/stuttering when running with the first person camera, especially on uneven surfaces. This was achieved by increasing the hardcoded speed limit value after which motion smoothing is disabled.
  
  

![](https://i.imgur.com/YyIDCVP.gif)﻿

[/font]
  
  

[font=Georgia]Requirements[/font]

  
[font=Arial][Skyrim Script Extender (SKSE64)](https://www.nexusmods.com/skyrimspecialedition/mods/30379)﻿ - Read the installation instructions carefully and always launch the game through SKSE.
  
[Address Library for SKSE Plugins](https://www.nexusmods.com/skyrimspecialedition/mods/32444)﻿﻿ - Select the file for your version of the game.
  
[/font][[font=Arial]Latest Visual C++ Redistributables (x86/x64)[/font]](https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads)
  

[font=Georgia]Installation[/font]

  
[font=Arial]Use any mod manager or unpack the archive manually into the game folder (Skyrim Special Edition\Data).
  
[/font]
  
  

[font=Georgia]Compatibility[/font]

  
[font=Arial]For all game versions (SE, AE)
  
Should be compatible with any mods.
  
Safe to install/update/uninstall.[/font]
