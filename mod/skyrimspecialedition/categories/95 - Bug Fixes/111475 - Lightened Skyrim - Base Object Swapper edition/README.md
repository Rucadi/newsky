[font=Arial]
  
This mod born after the great success of [Lightened Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/50755). For the deep description you can see the [original mod](https://www.nexusmods.com/skyrimspecialedition/mods/50755).  The main difference between the 2 versions is that this one uses [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) to achieve the same effect and does not have an esp (there in only a .INI file).[/font]
  

![](https://i.ibb.co/BjxGLtM/Wi-Zki-D-Features.png)

  

  
* [font=Arial]Same benefits made by [original mod](https://www.nexusmods.com/skyrimspecialedition/mods/50755).[/font]
[font=Arial]No esp.[/font]
  
* [font=Arial]No data or cell modify.[/font]

  

![](https://i.ibb.co/z8b5ZPY/Wi-Zki-D-Compatibility.png)

[font=Arial]
  
It may be strange to hear but the mod could be both more and less compatible. In this sense I consider it experimental although I don't think it presents any critical issues. The question is the follow:
  
  
[/font]

  
* [font=Arial]The [original mod](https://www.nexusmods.com/skyrimspecialedition/mods/50755), being an esm, was loaded very high in the load order and possibly a mod that was loaded later and that wanted to reactivate an object could do so. It must be said that until now I don't remember any mod that reactivates those objects.[/font]
    
  [font=Arial]
    
  [/font]
[font=Arial]The new mod does not allow subsequent mods to use those items that will remain permanently inconsistent in game. So no patch is necessary and possible. It also has the advantage that by deleting the ini file, the objects return to where they were, without having to worry about the load order. It can be uninstalled at any time without affecting the game.[/font]
  
[font=Arial]
  
[/font]* [font=Arial]Compatible with VR.[/font]
    
  [font=Arial]
    
  [/font]
[font=Arial]Safe to install/remove midgame.[/font]
  

![](https://i.ibb.co/9v1hwVS/Wi-Zki-D-Installation.png)[font=Arial]﻿[/font][font=Arial]
  
[/font]

[font=Arial]Install this like a plugin into YourSkyrimFolder/data. [/font]
