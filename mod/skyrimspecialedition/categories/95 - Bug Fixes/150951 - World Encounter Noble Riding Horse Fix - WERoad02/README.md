**Description**

  
* Problem: 
    
  [list]
Noble will sometimes separated from the horse. When this happens, the noble abandons the horse forever.
  
* The horse remains in the world, alone, abandoned, mysterious.

[\*]Fix:
  

  
* The horse will now run back to the noble, allowing the noble to get on the horse.
Prevents random abandoned horses in the world.
  

[\*]Fix is retroactive: will fix existing WERoad02 bug as soon as it is installed.
  
[\*]ESL.
  
[\*]No vanilla records edited. No conflicts.
  
[\*]Compatible with cutting room floor and relationship dialog overhaul.
