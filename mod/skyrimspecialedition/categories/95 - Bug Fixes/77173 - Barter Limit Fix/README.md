**BARTER LIMIT FIX**
  
  
**Requirements**
  
Latest version of SKSE64
  
Visual C++ Redistributables 2019Â 
  
Address Library for SKSE Plugins
  
  
**Description**
  
  
Fixes a bug that can prevent the player from receiving gold when selling to a merchant if the merchant has more than 32,767 gold. This bug can similarly prevent the player from receiving items when buying from a merchant if the merchant has more than 32,767 of said item.
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/77173/77173-1666183956-949987551.gif)

  
  
**Logs**
  
  
The log file will report whether the Barter Limit Fix has been installed. It will also report any warnings and errors that have been silently handled, such as conflicts with other mods.
  
  
The log file can be found in the following directory, which can be opened in any text editor:
  

* Skyrim Special Edition/Data/SKSE/Plugins/BarterLimitFix.log
