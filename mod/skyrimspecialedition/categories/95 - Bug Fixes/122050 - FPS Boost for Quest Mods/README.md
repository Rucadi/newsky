# FPS Boost for Quest Mods
- Author: ChickenMike
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/122050


Please bother me if you have an issue with my plugins and not the original mod authors. These plugins were made by me, ChickenMike and not the original mod authors. All plugins are .ESL flagged .ESPs so they won't take up a place in your load order.
  
  
This mod utilizes the same methodology of [Skyrim Project Optimization](https://www.nexusmods.com/skyrimspecialedition/mods/14084) and [eFPS - Exterior FPS boost](https://www.nexusmods.com/skyrimspecialedition/mods/58727) of adding occlusion planes in order to prevent your computer from loading things the player cannot see. My plugins are compatible with, but do not require either mod, nor [eFPS - Anniversary Edition](https://www.nexusmods.com/skyrimspecialedition/mods/58727) so you will not have to worry about my planes overlapping with theirs and causing crashes. These plugins will likely be more helpful to those on low end systems. 
  
  
In the event you run into problematic plane please provide the location and a [more informative console](https://www.nexusmods.com/skyrimspecialedition/mods/19250) screenshot of a nearby object so I can easily fix it. 
  

IT IS RECOMMENDED THAT YOU ADD THESE PATCHES TO A NEW GAME. To quote [robertgk2017](https://www.nexusmods.com/skyrimspecialedition/users/5544506) "Modding is illegal post character start."﻿ You might be able to get away with it if you haven't entered the cell yet but this is neither recommended nor supported.

FAQ﻿

  
Q:Why don't you have frame benchmarks?
  
A:I'm lazy
  
  
Q:But seriously how much FPS will this give me?
  
A:Honestly not a ton. The planes only really help if the angle in which the player is standing is correct. It is however better than not having it. 
  
  
Q:Didn't this mod used to have a different name?
  
A:It used to be called "Occlusion for Quest Mods" but it occurred to me that a lot of people don't know what that means so I changed it. 
  
  
Q:I'm a mod author, can you please do occlusion on my mod
  
A:Absolutely. Send me a DM and we can work something out. 
  
  
Q:Isn't doing this like mind numbingly boring?
  
A:Yes but I figured out I can make these while listening to audiobooks. 
  
  
Q:Isn't your anniversary earlier than this because you had stuff published on other pages?
  
A:Yes but July 25th is for my first page. Also I do what I want. You're not my dad. 
  
  
Q:Do you recommend any other mods to give similar performance improvements?  
  
A:Yes! [H.O.A. - Hyperspecific Occlusion Addon](https://www.nexusmods.com/skyrimspecialedition/mods/149004)
  
  
Q:Why does the Undeath plugin require Undeath Classical Lichdom?
  
A:I use it and didn't want it to not work with my load order. If you remove the masters in xedit it might work?
  
  
Q:Do these plugins work with my game version?
  
A:They should, but best practice is to use [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441).
  
  
Q:Do you take requests?
  
A:My primary reason for making tweaks like these is personal use but if I can make it quickly or you can convince me I need it for myself, I'll consider it. 
  
  
Q:You haven't had any recent activity on Nexus and I need to get a hold of you, how can I do that?
  
A:@ChickenMike in either Ryn, or Janquel's discords and I should see it. I'm also in a bunch of other discords, but I check those two the most regularly.
