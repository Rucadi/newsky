HAGRAVEN**

**My patches by Xtudo**

  
**Description**
  
**My patches for the amazing mod made by Mandragorasprouts.**
  

  
* **Skyrim Immersive Creatures Special Edition** patch. 1 model updated.
**Monster Mod SE** and **Monster Mod AE**. 3 models updated.
  
* **No Top patch.** Vanilla, SIC and Monster Mod included. Only meshes. For the Monster SE-AE mods, their patch is still required, for the new textures. But there's no problem if you don't have them installed. Set **No Top** last of all. Please don't see it as a sexual mod, it is not.
**Unique Glenmoril patch.** Gives an unique appearance to the Glenmoril Witches, based on the Vanilla style. ESPFE.
  
* **Wormpie's Hagravens patch.** 50 models updated. Don't use together with **Unique Glenmoril patch**, it's already included.
**Hagraven Claw ingredient patch**.
  
* **Unique Matriarch patch**. 1 model updated.
**Slightly Better Hagraven Feathers patch**. Replaces the Hagraven's feathers for better consistency with those of this mod.
  
* **VIGILANT.** Replaces [Reyda](https://tes-mods.fandom.com/wiki/Reyda)﻿.
**Hagraven Variance**.
  
* It's fine to install/uninstall the patches at any time.
