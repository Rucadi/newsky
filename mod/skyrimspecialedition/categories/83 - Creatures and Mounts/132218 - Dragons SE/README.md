# Dragons SE
- Author: Akatosh
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/132218


Новые модели и текстуры для наших любимых драконов.
  
  
Были заменены модели для:
  
- Алдуина
  
- Партурнакса
  
- Одавинга
  
- Дюрневира 
  
- Коричневого дракона
  
- Кровавого дракона
  
- Морозного дракона
  
- Старейшего дракона
  
- Древнего дракона
  
- Благородного дракона
  
- Легендарного дракона
  
  
Не заменены модели:
  
- Змеевидный дракон (слишком отличающаяся модель)
  
- Драконий скелет (я всё ещё хочу увидеть солнце)
  
- Загрузочные экраны (некоторые технические проблемы)
  
  
На переработку этих ребят ушло почти три месяца.
  
  
Чтож... Надеюсь, вам понравится. Желаю удачи (:
