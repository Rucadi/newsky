**~~~Installation~~~**
  
**Step One:** head over [here](https://www.nexusmods.com/skyrimspecialedition/mods/56361)﻿ and install fluffworks, preferably the**﻿** quality version, as its what these patches are based on
  
**Step Two:** check out all the cool mods listed above, and install any of them, being sure there aren't any conflicts
  
**Step Three:** from this mod download the main auto patcher, or any individual patches for the mods listed above
