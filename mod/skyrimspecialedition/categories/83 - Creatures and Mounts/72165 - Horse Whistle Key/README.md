# Horse Whistle Key
- Author: Edryu
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/72165


**Horse Whistle Key**

**Features**

  
* Use the **H key (Configurable)** to access this mod features.

  
* Press to call your horse. If the horse is visible, it will try to get near you. If it is not visible, it will be summoned behind you.
Press while pointing to your horse or follower to open their inventory.
  
* When riding, the key will open the inventory of your current mount.
Sneak and press the key to rename the horse you are pointing.
  
* Equip saddles when adding them to your horse inventory.
Multiples ways to call your horse: whistle sound or War Horn call.
  
* Set your horses as protected *(Only horses that you have accessed their inventories).*
Holding the key will show you all owned horses and let you call one.
  
  
* Notification when your horse is killed.
Horses will have new names and their confidence will be set to Cowardly.
  
* **(Optional)** CC Armor patch to disable original dialogues, new recipes for CC Armors and tweaked meshes.

**Requirements**
  

  
* [MCM Helper](https://www.nexusmods.com/skyrimspecialedition/mods/53000): Used for the MCM menu.
[Spell Perk Item Distributor (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/36869): Distributes exclusion keywords to some NPCs.
  
* [Keyword Item Distributor (KID)](https://www.nexusmods.com/skyrimspecialedition/mods/55728): Used to add compatibility with saddles mods.
[Animated Whistling](https://www.nexusmods.com/skyrimspecialedition/mods/76733)﻿: Required if using animated whistle feature.
  
* [Dynamic Animation Replacer](https://www.nexusmods.com/skyrimspecialedition/mods/33746): Required when using unlocked war horn call.
[AnimObject Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/75167?tab=files): Required when using unlocked war horn call.
  
* [FormList Manipulator](https://www.nexusmods.com/skyrimspecialedition/mods/74037)﻿: Expands the worldspaces where you can summon your horse.

  
**Compatibility**
  
Compatible with saddles mods that use the new keywords. Currently, compatibility added for:
  

  
* Equestrian
Craftable Horse Barding
  
* Craftable Saddles from Witcher

Others mod that add new saddles will need to use this mod keywords using KID. This mod is not compatible with mods that alters horse records (If this mod is being overwritten, the horses will not get their new names or new behavior).
