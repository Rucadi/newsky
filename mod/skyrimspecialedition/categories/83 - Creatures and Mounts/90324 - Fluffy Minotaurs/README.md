# Fluffy Minotaurs
- Author: WillOhTheWisp
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/90324


[font=Georgia]A fluff overhaul for Mihail's Minotaurs! The minotaurs now use new shell meshes and textures to give them a fluffier, fuzzier appearance. Now you can be ripped limb from limb by a brutal but fluffy beastman!

Requirements
  
This mod only includes meshes and textures and requires [Minotaurs - Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/72454), [High Minotaurs - Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/34392), or [Cyrodilic Minotaurs - Mihail Monsters and Animals](https://www.nexusmods.com/skyrimspecialedition/mods/27364). It's up to you which of those mods you'd like to use. Just download the mod you want and pick the corresponding fluffy file from this page. Overwrite the original(s) with the file(s) from this mod.
