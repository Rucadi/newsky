﻿
  
***!! Mod Replaces Only: Giant01.nif**
  
(in Skyrim: 3 Giant versions: Giant, Giant01 & Giant02)
  
  
  
  
**You can combine different versions of the mod ([i]Nude + Topless + Dressed[size=4])***:[/size]
  
Rename one version--->Giant.nif or Giant02.nif------>and there will be 2 versions in the game at once))
  
Example: 
  
Dressed version ---> Giant.nif
  
Nude version ---> Giant01.nif  
  
Rename Topless version ---> Giant02.nif
  
[/i]
