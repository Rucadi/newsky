# Nimhe - Unique Web Mother
- Author: Missile
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/107115
﻿**Nimhe Web Mother**

  
This mod gives Nimhe a unique look and more difficulty by changing him to a Web Mother with a unique texture. Nimhe will be level 16 when you fight him and will have the same abilities as a web mother. (ESL Flagged)
  
  
Also includes a small patch for [Webmother Brutality](https://www.nexusmods.com/skyrimspecialedition/mods/106278) 
  
  
Requires [The Contest CC](https://en.uesp.net/wiki/Skyrim:The_Contest) 
  
  
﻿Thanks [Vektile](https://www.twitch.tv/vektile) and [Xynerorias](https://www.nexusmods.com/skyrimspecialedition/users/6546516) for testing!
  
  
If you have [Dynamic Animal Variants SPID](https://www.nexusmods.com/skyrimspecialedition/mods/72978?tab=description) and run into Nimhe not having the web mother skin, edit the DAV SPID ini like this: Spoiler
  
> This replaces the skin Web Mother skin. I had to change this line
>   
>
> ```
> ﻿Spell = 0xE3F~DynamicAnimalVariantsPlus.esp|ActorTypeCreature|0x4E507~Skyrim.esm|NONE|NONE|NONE|100
> ```
>
>   
> to this to prevent it.
>   
>
> ```
> ﻿Spell = 0xE3F~DynamicAnimalVariantsPlus.esp|ActorTypeCreature|0x4E507~Skyrim.esm,-ccBGSSSE069_FrostbiteSpiderRaceGiant|NONE|NONE|NONE|100
> ```
