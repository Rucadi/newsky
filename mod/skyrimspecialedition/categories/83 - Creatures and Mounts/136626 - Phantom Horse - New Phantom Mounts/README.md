# Phantom Horse - New Phantom Mounts
- Author: WillOhTheWisp
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/136626


[font=Georgia]New additional mounts for Phantom Horse. You can now use Phantom Horse to conjure up mounts from various different mods! For Shadowmere and Arvak replacers, see [Phantom Horse - Horse Replacer Patches](https://www.nexusmods.com/skyrimspecialedition/mods/136258). Container Item Distributor is required to add the rings and spell tomes to the Riverwood Trader. Otherwise you will need a mod like AddItemMenu or console commands to access the new items.

[font=Georgia]Please note that the original mods are still required! This mod is intended to make phantom mounts out of existing mounts in your game. However, if you want just the phantom mounts, you can delete everything except the textures from the original mods. The textures from Lion Horse are included on this page and do not need to be downloaded separately, as the original mod is hosted on Steam Workshop.
  
  
There is an [article](https://www.nexusmods.com/skyrimspecialedition/articles/9501) included on this page that serves as a tutorial if you want to create your own phantom mount. There is an optional file to get you started.
  
  
There is a patch for [Arvak Burning Hooves Restored - Cut Content Restoration](https://www.nexusmods.com/skyrimspecialedition/mods/51517), which can be used with any Arvak replacer. I you are using any of the replacers available on the Phantom Horse - New Phantom Mounts page, use only the patch from this page.
