Currently only wolves are covered here, but Xtudo's got trolls patched [over here](https://www.nexusmods.com/skyrimspecialedition/mods/126608)﻿ (and more).
  
The framework only covers wolves and trolls for the moment, but I may patch more as they are released.
  
The meshes use your installed re textures, and shell textures/patches
