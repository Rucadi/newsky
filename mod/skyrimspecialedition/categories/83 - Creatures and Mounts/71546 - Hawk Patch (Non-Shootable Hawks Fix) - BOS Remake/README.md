# Hawk Patch (Non-Shootable Hawks Fix) - BOS Remake
- Author: lilebonymace
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/71546
**Description**
  
A [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805) remake of [Hawk Patch](https://www.nexusmods.com/skyrimspecialedition/mods/33530). Some hawks can't be shot because they use a different, non-interactive base object. This BOS config replaces these hawks with normal shootable hawks.
  
  
**Requirements**
  

  
* [Base Object Swapper](https://www.nexusmods.com/skyrimspecialedition/mods/60805)﻿
