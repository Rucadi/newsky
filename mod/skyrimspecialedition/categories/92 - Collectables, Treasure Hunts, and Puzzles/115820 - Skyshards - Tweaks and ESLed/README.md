# Skyshards - Tweaks and ESLed
- Author: WhereDeadAngelsLie
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/115820


![](https://i.imgur.com/8jkV3Pk.png)

  
  
Various tweaks as individual files to download one by one
  

  
* ESL Patch
Start in Lite mode, to have one quest instead of 108
  
* Start in Lite mode
Start with the setting 1 perk point per shard
  
* Droppable config book or No configuration book at all

  
The patches compatibles with this ESL version can be found [here](https://www.nexusmods.com/skyrimspecialedition/mods/115880)
  
  
My personal choice is:
  

  
* 1 perk point per shards - No Beacons
No configuration book
  
* Droppable configuration book (just in case I add it with AddItemMenu)

  
  

![](https://i.imgur.com/Z1KoBJC.png)

  

  
* Install Skyshards
Install the ESL version
  
* Install one optional file
Install the miscellenous files
