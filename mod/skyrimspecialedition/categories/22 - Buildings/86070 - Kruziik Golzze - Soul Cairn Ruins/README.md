Kruzzik Golzze (Dovahzul for "Ancient Stones") adds several ancient ruins to the Soul Cairn.
  
  
The newly added ruins have been worked into the vanilla content and tied into the existing architecture. 
  
  
Updated meshes and textures to fit the soul cairn, including optimised textures, and adding missing mipmaps. 
  
  
Edited navmesh so followers and enemies can make their way around the new buildings.
  
  
LOD meshes for DynDOLOD
  
  
The plugin is ESP-FE.
