[font=Trebuchet MS]**- Mod Description -**[/font]An lore-friendly Overhaul of The Drunken Huntsman shop in Whiterun.
  
The ESP has been cleaned with the SSEEdit auto clean feature and flagged as an ESL, so it doesn't count towards the ESP limit.
  
  
[font=Trebuchet MS]**- Performance -**
  
  
My Rig: Asus P8B75-M/CSM Motherboard, Intel I7 2600 @ 3.4 GHz
  
8 Gb RAM, NVidia GTX 980TI 6 Gb, running alongside Aequinoctium ENB + 2048 Textures and I get 60 FPS average.
  
[/font]
  
[font=Trebuchet MS]**- Installation -**
  
  
[/font][font=Trebuchet MS]Install with your prefered mod manager.
  
  
[/font]**- Requirements -**
  
  
- Skyrim up to date
  
  
[font=Trebuchet MS]**- Compatibility :**[/font]- [JK's Interiors Patch Collection by Janquel﻿](https://www.nexusmods.com/skyrimspecialedition/mods/35910)﻿: A collection of small patches/tweaks to help various mods play well with JK's Interiors series.
  
- Not compatible with mods that overhaul the interior of The Drunken Huntsman shop (with the exception of texture mods)
