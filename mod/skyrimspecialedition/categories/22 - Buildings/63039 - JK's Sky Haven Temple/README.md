﻿**- Mod Description -**[font=Trebuchet MS][/font]
  
A lore-friendly overhaul of The Sky Haven Temple.
  
The ESP has been cleaned with the SSEEdit auto clean feature and flagged as an ESL, so it doesn't count towards the ESP limit.
  
  
[font=Trebuchet MS]**- Performance -**
  
  
My Rig: Asus P8B75-M/CSM Motherboard, Intel I7 2600 @ 3.4 GHz
  
8 Gb RAM, NVidia GTX 980TI 6 Gb, running alongside Aequinoctium ENB + 2048 Textures and I get 60 FPS average.
  
[/font]
  
[font=Trebuchet MS]**- Installation -**
  
  
[/font][font=Trebuchet MS]Install with your preferred mod manager.
  
  
[/font]**- Requirements -**
  
  
- Skyrim up to date, will work with both SE and AE.
  
  
[font=Trebuchet MS]- Compatibility :
  
  
- My[JK's Sky Haven Forge](https://www.nexusmods.com/skyrimspecialedition/mods/8706)﻿standalone mod is included and no longer needed, please uninstall if used.
  
- [**JK's Skyrim**](https://www.nexusmods.com/skyrimspecialedition/mods/6289)﻿also includes the forge, if you use My JK's Skyrim mod, a patch that disables the duplicated references can be found in the link bellow.
  
[/font]- [JK's Guild HQ Interiors Patch Collection by Janquel, Czasior, Teabag86, and DrMonops](https://www.nexusmods.com/skyrimspecialedition/mods/61416)﻿﻿: A collection of small patches/tweaks to help various mods play well with JK's Interiors series.
  
- Not compatible with other mods that overhaul the interior and exterior of The Sky Haven Temple. (with the exception of texture mods)
