﻿**- Mod Description -**[font=Trebuchet MS][/font]
  
An lore-friendly Overhaul of The Sleeping Giant Inn in Riverwood, featuring a new basement level
  
with a room for Orgnar, two extra rooms for travelers and a bathtub, plus new decorations all over the inn.
  
The ESP has been cleaned with the SSEEdit auto clean feature and flagged as an ESL, so it doesn't count towards the ESP limit.
  
  
[font=Trebuchet MS]**- Performance -**
  
  
My Rig: Asus P8B75-M/CSM Motherboard, Intel I7 2600 @ 3.4 GHz
  
8 Gb RAM, NVidia GTX 980TI 6 Gb, running alongside Aequinoctium ENB + 2048 Textures and I get 60 FPS average.
  
[/font]

[font=Trebuchet MS]**- Compatibility :**[/font]- Lighting: upstairs lighting has not been altered which should make it compatible with all interior lighting mods.
  
- [Enhanced Lights and FX (ELFX)](https://www.nexusmods.com/skyrimspecialedition/mods/2424): Compatible with patch to fix flooring and tweak lighting.
  
- [Distinct Interiors](https://www.nexusmods.com/skyrimspecialedition/mods/6130): Compatible with [The Distinct JK's Sleeping Giant Inn (Patch)](https://www.nexusmods.com/skyrimspecialedition/mods/35813)
  
-[Keep it Clean](https://www.nexusmods.com/skyrimspecialedition/mods/17846): Compatible with [JK's Sleeping Giant Inn - Keep It Clean Patch](https://www.nexusmods.com/skyrimspecialedition/mods/35828)
  
- Not compatible with other mods that overhaul the interior of The Sleeping Giant Inn (with the exception of texture mods)
