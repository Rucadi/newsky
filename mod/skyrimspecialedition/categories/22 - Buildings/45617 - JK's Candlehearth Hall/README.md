﻿**- Mod Description -**[font=Trebuchet MS][/font]
  
A lore-friendly overhaul of the Candlehearth Hall in Windhelm, featuring a new basement with three extra rooms for travelers a bathtub and a storage room for the Inn produce, tons of new decorations, and a balcony overlooking the city.
  
The ESP has been cleaned with the SSEEdit auto clean feature and flagged as an ESL, so it doesn't count towards the ESP limit.
  
  
[font=Trebuchet MS]**- Performance -**
  
  
My Rig: Asus P8B75-M/CSM Motherboard, Intel I7 2600 @ 3.4 GHz
  
8 Gb RAM, NVidia GTX 980TI 6 Gb, running alongside Aequinoctium ENB + 2048 Textures and I get 60 FPS average.
  
[/font]
