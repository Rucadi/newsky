# Visualized Critical Hits - MIF
- Author: ap05
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/95742


  
  
Description
  
Applies a new particle effect when hitting with a Critical Hit.
  
There are two types of critical hits in Skyrim.
  
1. CritChance is triggered by a Perk.
  
2. sneak attack successfully hits.
  
  
**Features**
  
New critical hit visual effects
  
Replaced critical hit sound
  
  
> **Requirements**
>   
> [MIF - Mu Impact Framework](https://www.nexusmods.com/skyrimspecialedition/mods/95624) / **This is a Core REQuirements, it won't work without it.**﻿
>   
> [[Textures Folder Only]Dynamic Impact - Slash Effects X](https://www.nexusmods.com/skyrimspecialedition/mods/86071) / You will need the **textures folder** included with that mod. **If you don't install it, all effects will show up as broken squares.**﻿
