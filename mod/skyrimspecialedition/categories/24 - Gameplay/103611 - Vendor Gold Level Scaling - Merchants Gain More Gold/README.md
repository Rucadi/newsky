# Vendor Gold Level Scaling - Merchants Gain More Gold
- Author: Dutchj
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/103611


**Vendor Gold Scaling**
  
Merchants Gain More Gold
  
  
Vendors will gain 25% of their starting gold for every 5 player levels, up to level 50.
  
  
**[b][size=4]●** Example (Blacksmith)[/b] **●**
  
[/size]

  
* Level 1: 1000 gold
Level 5: 1250 gold
  
* Level 10: 1500 gold
Level 15: 1750 gold
  
* Level 20: 2000 gold
(..............................)
  
* Level 50: 3500 gold

**Compatibility**
  
This mod only changes the various "VendorGold" leveled lists, so it should be compatible with most things. This mod is ESL flagged.
