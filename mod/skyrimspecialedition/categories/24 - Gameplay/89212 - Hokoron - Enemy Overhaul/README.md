# Hokoron - Enemy Overhaul
- Author: aelix
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/89212


Hokoron is intended to adjust several enemy types such that they continue to challenge the player throughout the later stages of the game. This mod accomplishes this by adding new, higher level variants for several enemy types and tweaking specific enemies.
  
  
**Features**
  

  
* Added high level variants for Hagravens, Wispmothers, Falmer, Dwarven Automatons, Rieklings, Ash Spawn, Gargoyles, Seekers, and Lurkers.
Hagravens now have Frost and Shock variants in addition to their vanilla, fire-wielding counterpart. They will also use more spells such as Expert Destruction spells, cloak spells, and atronach summons.
  
* Several fixes to bizarre scaling present in a few enemy types.
Fixed a vanilla bug where higher level Seekers weren't summoning clones.\*
  
* Lurkers' spit attack is now deadlier. They can also no longer be paralyzed.
Frost Giants can no longer be paralyzed.
  
* Restored an unused Gargoyle model for the new variant.
Rieklings deal bleed damage.
  
* Optional **Difficulty Addon** that ensures only the strongest available enemies are spawned according to your level.

**Installation**
  
  
Download and install through your favorite mod manager. **Uninstalling this or any other mod mid-save is not recommended.**
  
  
**Compatibility**
  
  
This mod will likely not be compatible with any mod that edits the same enemy types. I found no conflicts in my own load order of hundreds of mods, so you will need to let me know if there are any serious conflicts.
  
  
Perk overhauls present some minor conflicts with regard to the Perks added to Hagravens. A patch has been provided for [Adamant.](https://www.nexusmods.com/skyrimspecialedition/mods/30191?tab=description)
  
A patch has been provided for [Coven](https://www.nexusmods.com/skyrimspecialedition/mods/51596)ï»¿ that will allow you to use my Hagravens with Coven's witches without conflict.
