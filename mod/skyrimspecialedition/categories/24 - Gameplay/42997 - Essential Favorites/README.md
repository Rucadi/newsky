**ESSENTIAL FAVORITES**

Requirements [/color][/b]
  
  
SKSE64
  
Visual C++ Redistributables 2019 
  
Address Library for SKSE Plugins
  
  
[/size]**Description**
  
  
SKSE plugin that prevents favorited items from being crafted, sold, disarmed, disenchanted or dropped. These can be toggled separately in*po3\_EssentialFavorites.ini.*
  
  
Favorited items will no longer show when selling/crafting/disenchanting/gifting items.
  
  
You may also be interested in [Favorite Misc Items](https://www.nexusmods.com/skyrimspecialedition/mods/42750), which allows misc items to be favorited.
  
  
**Installation**
  
  

  
* Install using a mod manager.
