# Precision Trail Replacer - Simple
- Author: ap05
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/83544


**Requirements
  
[Precision](https://www.nexusmods.com/skyrimspecialedition/mods/72347)
  
  
Replaces original Precision Trail mesh.**
  
  

  
* Weapon trails are on the edge and faded throughout.
Refraction effect is stronger than original.
  

  
  
  
**How to install**
  

  
* Overwrite the original files or install via the mod installer.
