# Die When Killed - A Dead-Simple Permadeath Mod
- Author: BrainBlasted
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/46680
**Die When Killed**
  
The player character in Skyrim does not die when they are killed. This goes against one of the fundemental
  
laws of the universe, according to Shirou Emiya. This mod serves to correct this by implementing a permadeath system,
  
and *only* a permadeath system.
  
  
**How Does It Work?**When the player character dies, their name will be saved to "Data/SKSE/Die When Killed/HallOfTheDead.json". A dialog will show and you will be sent to the main menu. When loading a saved game that had Die When Killed installed, the mod will check HallOfTheDead.json. If the player character's name is in the list, the same dialog shown on death will show again, and you will be sent back to the main menu.
  
  
**Requirements**Die When Killed requires the latest versions of [SKSE](https://skse.silverlock.org/) and [PapyrusUtil](https://www.nexusmods.com/skyrimspecialedition/mods/13048).**[size=4][b]
