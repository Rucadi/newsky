# Simple Werewolf Looting
- Author: Hydraaawr
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150717


  
  
**[b][color=#ffd966][size=5] [﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿ ﻿![](https://i.imgur.com/R16Khlc.jpg) ﻿ ﻿ ﻿ ﻿](https://ko-fi.com/hydraaawr)**[/size][/color][/b]
  
  
**Features**
  

  
* Default interaction with containers is the "Search" (loot) action while on werewolf form

  
* Press Shift + E to feed from corpses in the vanilla way

  
* Prevents player from opening inventory/equipping anything while on werewolf form

  
  
**Installation**
  

  
* Safe to install at any point

  
* ESL Flagged

  
  
**Compatibility**
  
  

  
* **Doesn't overwrite any vanilla scripts**

  
* **Compatible** with anything that doesn't overwrite PlayerWerewolfFeed Perk
