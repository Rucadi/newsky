# Campfire - Reskin Skill Tree
- Author: Fernvud
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/160163


[font=Lucida Sans Unicode]My interpretation of the skill tree from the wonderful Campfire Complete Camping System mod. The original design, in my opinion, is a bit awkward. I think the Dovahkiin should be sketching out useful skills or writing them down in some kind of camping journal. I based it on that idea.
