# Horses Simply Turn Better
- Author: 0x4Ch0x4Fh0x4Bh0x49h
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/57790
REQUIREMENTS:
  
Skyrim Special Edition
  
  
COMPATIBILITY:
  
Literally everything that doesn't change horses' movement variables. Horse\_Default\_MT and Horse\_Sprint\_MT.
  
If it messes with a mod you have then a simple trip to xEdit/zEdit will solve it.
