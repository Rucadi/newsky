# Modern Controlmap (Gamepad and Keyboard)
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/89649
**This controlmap modernizes and improves the controls of Skyrim. It features additional functions and includes unbound keys to bind for dodging, blocking, etc.
  
  
Compatible with all controllers that work in vanilla Skyrim.**Includes a version for Quick Loot.
  
  
**Controls:** *(everything else remains [vanilla](https://en.uesp.net/wiki/Skyrim:Controls))*
  
  
- Back Button: Player Menu *(inventory/map/magic/stats)*
  
- D-Pad Up: Favorite Menu
  
- D-Pad Down: Ready/Sheathe Weapons
  
- D-Pad Left: Hotkey 1 *(Favorites)*
  
- D-Pad Right: Hotkey 2 *(Favorites)*
  
  
- LB+Back: Wait
  
- LB+Start: Quicksave
  
- LB+D-Pad Left: Hotkey 3 *(Favorites)*
  
- LB+D-Pad Up: Hotkey 4 *(Favorites)*
  
- LB+D-Pad Right: Hotkey 5 *(Favorites)*
  
- LB+D-Pad Down: Hotkey 6 *(Favorites)*
  
  
- A+D-Pad Left: Quick Magic
  
- A+D-Pad Up: Quick Skills
  
- A+D-Pad Right: Quick Inventory
  
- A+D-Pad Down: Quick Map
  
  
- R3 (Hold): Switch Perspective
  
- R3 (Hold) + Left Stick: Change Camera Zoom in 3rd Person
  
  
- B: Unbound *(recommended for Dodge)*
  
- X: Unbound *(recommended for Dual Wield Block)*
  
  
With True Directional Movement:
  
- R3: Lock-On
  
- Right Stick: Switch Target
  
  
Keyboard Changes:
  
- Shift: Running
  
- Alt: Unbound *(recommended for Dodge)*
  
- Space (Inventory): View Mode *(mouse wheel also works)*
