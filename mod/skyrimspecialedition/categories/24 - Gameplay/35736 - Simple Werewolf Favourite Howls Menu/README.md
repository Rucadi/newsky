# Simple Werewolf Favourite Howls Menu
- Author: NymicRazor
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/35736


Allows the use of the favourites menu to select unlocked werewolf howls, similar to the vampire lord form.
  
  
**Requirements**
  

  
* [SKSE](http://skse.silverlock.org/)﻿.
While not strictly required, [Bug Fixes SSE](https://www.nexusmods.com/skyrimspecialedition/mods/33261)﻿ will prevent the mod from breaking after a large amount of time has passed.
  
* [SkyUI](https://www.nexusmods.com/skyrimspecialedition/mods/12604)﻿ is required to use the Mod Configuration Menu.

  
**Compatibility**Is compatible with werewolf overhauls that don't change Howls. still might be compatible with some that do.
  
Compatible with Growl - Werebeasts of skyrim.
  
Should be compatible with everything else.
  
  
**Installing**
  
Use a mod manager.
  
for a manual install, just extract the ZIP file and drop the contents in your Data folder.
  
  
**Uninstalling**
  
Save the game while you are not in beast form. If you have installed the mod manually, remove the WerewolfQuickMenu.esp file from your data folder
