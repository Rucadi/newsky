# Werewolf Kill Moves Fixed
- Author: Ixion XVII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65803
Werewolf kill moves are only performed while killing the last enemy in combat.**The werewolf kill move animation takes a long time. Having it play several times in combat is not only unimmersive since enemies still hit you while you're invincible, it also gets annoying very quickly.**
