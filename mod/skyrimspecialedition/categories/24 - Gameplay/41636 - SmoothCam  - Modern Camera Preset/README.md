Requirement
  
  

  
* [SmoothCam](https://www.nexusmods.com/skyrimspecialedition/mods/41252).

  
Installation
  
  

  
* Use any mod manager or manually copy SKSE from this mod into your Skyrim SE 'Data' folder.

  
Usage
  
  

  
* Open the SmoothCam MCM, go to Presets, and load "ModernPreset" which uses the 4th preset slot.
