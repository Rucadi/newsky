# Biggie Traits - Thumbnail Overhaul
- Author: Patman023
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/148018


**V1.03 is a Massive update - Replaced every single image with much better quality versions, changed a few images completely
  
  
Now includes support for [Additional Traits for Biggie Traits](https://www.nexusmods.com/skyrimspecialedition/mods/149770)
  
  
![](https://i.imgur.com/wzn06d8.png)**
  
  
This overhaul replaces the original trait icons with high quality thumbnails that better represent each trait’s theme and purpose.
  
  
Each image has been carefully chosen to improve visual clarity, enhance immersion, and give the trait menu a more cohesive and polished look. 
  
  
[size=4]**Installation**
  
Install [Biggie Traits](https://www.nexusmods.com/skyrimspecialedition/mods/136384) then overwrite with this mod to get the new images
