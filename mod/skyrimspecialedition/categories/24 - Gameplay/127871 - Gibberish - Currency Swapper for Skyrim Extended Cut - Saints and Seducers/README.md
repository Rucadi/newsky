# Gibberish - Currency Swapper for Skyrim Extended Cut - Saints and Seducers
- Author: Tate Taylor
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/127871

[/size]

> *﻿"**You haven’t got the coin. You wouldn't be trying to... cheat me, would you?"*
>   
> *— Theodor Gorlash*

  
The Shivering Isles are a plane of madness and whimsy, distinct from anything found in Tamriel. Here, the locals use an appropriately quirky currency known as the gibber.
  
  
To make transactions in this realm, you'll need to venture forth and scrounge up some gibbers first!
  

![](https://raw.githubusercontent.com/PierreDespereaux/PierreDespereaux/master/assets/images/banners/Features.png)

  

  
* Adds a new fully functional currency to the Shivering Isles: the **gibber**!
Both Theodor Gorlash and Sees-the-Moon will now only transact with gibbers.
  
* Replaces all instances of septims in the Shivering Isles with gibbers.
Implements a dynamic value where the gibber is strong in Mania and weak in Dementia.
  
* Adjusts prices to simulate the gibber's value relative to the septim.

This mod also includes built-in compatibility with [C.O.I.N.](https://www.nexusmods.com/skyrimspecialedition/mods/51439)﻿'s auto exchange feature so that gibbers aren't automatically converted to septims. **[font=Courier New]ShiveringIslesUsesGibbers.esp[/font] will need to be loaded *after*[font=Courier New]C.O.I.N.esp[/font].**
