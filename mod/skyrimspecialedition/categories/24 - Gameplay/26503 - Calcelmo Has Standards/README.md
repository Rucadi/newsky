# Calcelmo Has Standards
- Author: AndrealletiusVIII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/26503


**Description**
  
  
This mod makes changes to [this subquest:](https://en.uesp.net/wiki/Skyrim:Buy_Dwarven_artifact)
  
  

  
* Removes the buying requirement. The dwarven item, that Calcelmo is after, can now be obtained though other ways.
Adds a 100 gold value requirement to the item, to prevent the item from being a dwarven arrow or other cheap items.
  
* Adds a restriction to exclude items looted in a player home.

  
You can change the default 100 gold value to any value you'd like by using the following console command:
  
  

```
set CHS_ItemValue to X
```

  
  
X is the minimum value the item has to have, in order to trigger the quest. (default value is 100.)
  
  
This mod is made as an answer to [this thread](https://www.reddit.com/r/skyrimmods/comments/byk4x7/request_calcelmo_has_standards/) on reddit.
  
  
This mod is flagged as an ESL, and as such doesn't take up a slot in your load order.
