**Description**
  
  
 ﻿﻿SKSE plugin that fixes bugs and tweaks Reanimation archetype.
  
  
 **﻿Bug Fixes**
  
  

  
* [size=3]Dead thralls/zombies will no longer be lost when fast traveling or moving across load screens.
  
Partial fix for paralyzed reanimated corpses. Bug will still occur (walking in place) but they should be back to normal on fast travel.
  
  
 **Enhancements**
  
  

  
* Decapitated corpses can be reanimated.
Reanimated necromancers will cast reanimate spells
  
* Ridable reanimated mounts (excluding dragons).

  
All of these can be toggled in the included *po3\_EnhancedReanimation.ini.***Installation**
  
  
[/size]

  
* Install using a mod manager.
