# Know Your Enemy 2 - Orca's Patches
- Author: Orca255
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/124247


These are patches for [Know Your Enemy 2](https://www.nexusmods.com/skyrimspecialedition/mods/93258) for several creature mods that I am using in my own game. Additionally, with Ironeyes permission using their mihail  patches I have created patches for some of mihail's AIO versions such as [Skeletal Revenants](https://www.nexusmods.com/skyrimspecialedition/mods/102415) and will be updating some of their patches for newer updates. I highly recommend to also check out their page [Know Your Enemy 2 - Unofficial Monster Patches](https://www.nexusmods.com/skyrimspecialedition/mods/96276)﻿ for additional KYE2 Patches.
  
  
Kulharin has asked me to host their patches for which I have uploaded as an additional main file. They contain different patches from my own so there should be no overlap. You will most likely want to download both main files.
  
  
**Mods patched listed below:**All patches should work with the most current version of each mod uploaded by 15 Jul 2024. It should work with later updates for the most part, but it all depends if new races are added.

Installation:
  
Download and install through your mod manager of choice. Make sure that each KYE2-Patch.esp loads after the original mods esp. It is safe to install on an existing save. The plugins are ESL-flagged.
  
  
Compatibility:
  
It should be safe with other patches. I haven't add any issues.
  
  
Uninstallation:
  
Patches should be safe to uninstall since they are just editing resistances. The only exception is the Elemental Golems KYE2 Patch because the patch actually fixes CTD issues between KYE2 and Elemental Golems. Here is the [Explanation](https://www.nexusmods.com/skyrimspecialedition/mods/115383).
