# Frost Resist provides Warmth (Survival Mode)
- Author: Zdenko7
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/61272
Makes your resistance to frost provide warmth for Survival Mode. Every 5 points of frost resist provides 2 warmth, up to a maximum of 30 warmth
  
at 75 frost resist. Alternate versions are available with different upper limits.
  
  
Install Instructions
  

  
1. Install the main file, or use one of the alternate maximum files in the Optional Files section.
**Use only one plugin**.
  

  
Upgrade / Change version Instructions
  

  
1. Uninstall your current version as detailed below.
Install the new version.
  
  

  
Uninstall Instructions
  

  
1. run the console command "player.removespell XX00aa02", replacing XX with the Mod Index
disable plugin
