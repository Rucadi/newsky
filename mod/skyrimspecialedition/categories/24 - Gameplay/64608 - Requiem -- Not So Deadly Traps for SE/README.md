# Requiem -- Not So Deadly Traps for SE
- Author: vram1974
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/64608
**What does this mod do?**
  
It makes [Requiem's](https://www.nexusmods.com/skyrimspecialedition/mods/60888) traps less deadly.
  
  
But, why?
  
  
I don't like being one-shotted or insta-gibbed. Requiem is a challenging mod but being killed outright isn't challenging. It's dumb.
  
  
So, now I can't die?
  
  
This is Requiem. You're still going to die. A lot.
