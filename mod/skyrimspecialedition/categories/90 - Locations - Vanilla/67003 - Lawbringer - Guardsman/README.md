\*Note : If you do not want one of the modified location, you can deactivate the corresponding plug-in. These are independant from each other, but they require the Skyrim Main Files, the Update and the LCO\_Framework introduced by Lawbringer.
  
You are free to modify/expand their content, but be cautious.
  
Also, a JSON file named LocationList is joint to the archive. The original version of this file is present in the original Lawbringer file, and was used to exclude the claimed locations from the radiant quests. In order to add the new locations, you will need to override this original file, and load L-G AFTER the original Lawbringer.
