FEATURES
  
  
  
Rebuilt the ruin from the ground up, expanding It's size while keeping It In a ruined state.
  
  
More orc bandits have been placed behind the walls.
  
  
Added more Orc huts for the Increased population.
  
  
  
![](https://imgur.com/VuvGkSO.png)![](https://imgur.com/8dKVgZF.png)﻿﻿﻿﻿﻿
  
Added Immersive clutter around the location.
  
  
The fort walls now have multiple entry points for you sneaky types.
  
  
The navmesh has been remade in the entire area from scratch.
  
  
Added some new small meshes I made.

**Conflicts and Patches**
  
  
This mod won't be compatible with other mods that make changes to the same area.
  
  
[Traverse the Ulvenwald - 3.3 - Trees and Forests of Skyrim](https://www.nexusmods.com/skyrimspecialedition/mods/57874)
  
- [size=6]Incompatible[/size]
  
There will be trees clipping through objects, so a patch will be needed.
  
  
Any other tree mod that adds trees to the area will need patching.
  
  
[Cracked Tusk Keep](https://www.nexusmods.com/skyrimspecialedition/mods/114566)﻿ by [Schlitzohr](https://next.nexusmods.com/profile/Schlitzohr?gameId=1704)﻿
  
- Incompatible
  
A Combo patch will be needed If people want to combine my mod with this.
  
  
  
**Provided Patches**I have made patches for the following mods;
  
  
[Lux Orbis](https://www.nexusmods.com/skyrimspecialedition/mods/56095)﻿ -
  
Moved some light markers and static objects, and changed the vanilla light markers to match Lux Orbis lights.
  
Load this patch after Lux Orbis.
  
  
[Nature of the Wild Lands - forest and trees improvement mod](https://www.nexusmods.com/skyrimspecialedition/mods/63604)﻿
  
- A patch to disable and move some trees to avoid clipping and have better placement around the area.

[Imperial Castles of Skyrim - Fort Battlements v2](https://www.nexusmods.com/skyrimspecialedition/mods/129327)﻿
