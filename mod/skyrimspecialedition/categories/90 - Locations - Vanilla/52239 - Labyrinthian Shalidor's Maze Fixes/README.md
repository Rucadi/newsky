**Description**
  

  
* Fixes level layout to address many gaps
Fixes Labyrinth underside and platform (also included in [Assorted Mesh Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/32117)﻿)
  
* Adds sunlight to worldspace (also included in [Shadows Of Sunlight - In Small Exterior World Spaces](https://www.nexusmods.com/skyrimspecialedition/mods/41368)﻿)
ESL
  
**Installation**
  

  
* Install whenever.
Included script properly resets persistent references.
  
**Uninstallation**

  
* Uninstall mod.
Click on elevator stairs in console and use "

```
recycleactor
```

" command.
  
**Compatibility**
  

  
* Load [Assorted Mesh Fixes](https://www.nexusmods.com/skyrimspecialedition/mods/32117)﻿ nif file assets after this, they are newer.
Load order with [Shadows Of Sunlight - In Small Exterior World Spaces](https://www.nexusmods.com/skyrimspecialedition/mods/41368) doesn't matter
  
* Load plugin later to overwrite any mod that might undo worldspace changes
