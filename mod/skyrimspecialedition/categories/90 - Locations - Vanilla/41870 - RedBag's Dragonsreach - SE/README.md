# RedBag's Dragonsreach - SE
- Author: RedBag
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/41870


**RedBag's Dragonsreach****Art and craftsmanship for the main hall****[Oldrim version available](https://www.nexusmods.com/skyrim/mods/105297)﻿**

**Features**

  
* Carved nordic portal behind Jarl's throne
A grill for the fireplace
  
* 3D carvings on 6 pillars around the fireplace

  
**Compatibility**
  

  
* [Patch for JK's](http://www.nexusmods.com/skyrimspecialedition/mods/45686)﻿﻿ by [WillOhTheWisp](https://www.nexusmods.com/skyrimspecialedition/users/102078168)
Navmeshes and lights aren't changed, all objects that were removed were done so by disabling, so lighting mods should be compatible.
  

  
**Installation**
  

  
* Extract the files to your Skyrim Special Edition/Data folder, and activate the .esp, preferably with a mod manager.
