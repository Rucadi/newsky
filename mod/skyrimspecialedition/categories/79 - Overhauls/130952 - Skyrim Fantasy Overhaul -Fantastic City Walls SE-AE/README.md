**Install** 
  
[size=5]**~~~~~~~~~~~~~~~~~~~~**
  
  
Any Mod Manager or Manual
  
  
1. Make sure you download the resource in [/size]**00 Core Essentials**[size=3]
  
2. Download your choice(AIO or Modular) in the **main files**
  
2. (optional. Run Dyndolod)
  
3. Download any interior of your choice in the **optional files**.
  
4. Activate all esps.
  
  
Simple. Easy. Literally just download and activate.
  
  
------
  
Dyndolod (for Lod users)
  
For Dyndolod users, it is on you to read the file descriptions and what order to download, generate, and activate plugins.
  
Download the optional files **ONLY** when you have finished running Dyndolod.
  
**NEVER**, run Dyndolod with any of my files that say -Interiors. They DO NOT need Dyndolod.
  
Users that fail to read this instruction will end up generating double LODs. 
  
Remove dyndolod gens, and restart if you messed up.[/size]
  
  
  
[Dyndolod beginners step guide here](https://www.nexusmods.com/skyrimspecialedition/articles/9408)﻿.
