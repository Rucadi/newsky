# Clockwork Shadow Redone
- Author: Fridam
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70363
[font=Georgia]Overhauls the Clockwork shadow with a scarier look. If you were afraid before, you will certainly be now!
  
I gave her a new skin (HD quality) with BC7 compression for maximum quality, new facetint and added blood on her dress.
  
  
**Requirement**
  
- Clockwork ofc.
  
  
**Installation**
  
- Install with a modmanager, or drag and drop the files into the data folder and overwrite the Clockwork files.
