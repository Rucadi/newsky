# Experience - MCM
- Author: zax
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65880


[heading]DESCRIPTION[/heading]Mod Configuration Menu (MCM) for [Experience](https://www.nexusmods.com/skyrimspecialedition/mods/17751) mod. Changes made via this menu are reflected in Experience.ini file (and other way around). Settings are not player profile specific, they are shared among other characters just like INI files. Supports translations.
  
  
  
[heading]SETTINGS[/heading]Some settings (6) are excluded from MCM for technical reasons. Changing them still requires editing of Experience.ini file.
