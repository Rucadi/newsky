# Simple Extra Summons Custom Skill Tree
- Author: Megablackk1d
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140296


Simple Extra Summons is a new Custom Skill Tree is a perk mod that allows you to summon more minions. Each perk adds the ability to add two more summons with a total of 5 perks. This stacks with Ordinator, or any perk, or enchantment that allows you to do the same. For example with all the perks in this plus the perk from my Knowledge and Wisdom New Custom Skill Tree mod https://www.nexusmods.com/skyrimspecialedition/mods/134411?tab=posts, and Twin Souls that is 12 extra minions that can be summoned. The description here, and the description of the perks in game will tell you what to do in order to unlock them.
  
  
Novice Summoner- 2 extra minions can be summoned. Requires 20 Conjuration.
  
Apprentice Summoner- 4 extra minions can be summoned. Requires 40 Conjuration.
  
Adept Summoner- 6 extra minions can be summoned. Requires 60 Conjuration.
  
Expert Summoner- 8 extra minions can be summoned. Requires 80 Conjuration.
  
Master Summoner- 10 extra minions can be summoned. Requires 100 Conjuration.
  
  
This is a simple necromancy mod that only allows you to summon extra minions that is it. There are no scripts or conflicts. It requires vanilla perks to unlock as well.
