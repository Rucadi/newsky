**Definition of Unarmoured:**[line]
  
Unarmoured is defined as not wearing equipment which has Keyword - ArmorLight, ArmorHeavy.
  
  
So, Unamoured means Naked or Clothed, Robed.
  
  
You are still unarmored even if using the following.
  
-Protection spell like a stone flesh
  
-Ward Spell
  
-Shield (does not have keyword of Heavy/Light.)
  
  
  
**Manual for Unarmoured Defense:**[line]
  
1.Touch "**the Statue of the Unarmoured One**", or use **[Perk] Unarmoured Defense** to open perk tree menu.
  
  
2.Unarmoured Defense skill activates when you are not equipped with light/heavy armor.
  
  
3.The level of the Unarmoured Defense skill is increased by being attacked when you are unarmoured. Defending against your opponent's attacks will make it easier to advance. When Naked , you can gain triple XP.
  
  
4.One skill point of Unarmoured Defense is granted when the skill level reaches a multiple of four.
  
  
5.You can earn Unarmoured Defense skill points by repeatedly Legendaryizing.
  
  
6.Unneeded skill points can be converted to regular perk points at **the bonfire** near the Statue of the Unarmoured One.

**Balance**[line]
  
When Unarmoured Defense is Rank 5, Armor rating is 300.
  
You probably can gain enough armor rating for not instant killed.
  
  
400 = 300 + 100 (Ebony Flesh)
  
600 = 300 + 100 (Ebony Flesh) \* 3.0 ( Mage Armor(Rank 3))
  
  
However, clothe and robe does not have hidden armor rating, it is still weaker than light/heavy armoured.
  
  
  
**Talisman**[line]
  
![](https://imgur.com/9MKssQE.png)
  
Talisman is invisible shield. You can block / bash by bare hand.
  
You can buy from merchant at Shrine of Unarmoured One.
  
  
Talisman of Warrior : Invisible Heavy Shield.
  
Talisman of Thief : Invisible Light Shield
  
Talisman of Mage : Invisible No Armor rating Shield. 
  
  
You can temper/enchant talisman. (Talisman of Mage can not be tempered.)

**FAQ**[line]
  
**Q. Compatible with other perk mod?**
  
A. Perfectly. Custom Skills Framework give us independent perk tree.
  
  
**Q. Compatible with other unarmed mods ?**
  
A. Yes. 
  
  
**Q. This mod use perk point?**
  
A. No. This mod add new skill point for Unarmoured Defense.
  
  
**Q.Can I gain Player Level XP when Unarmoured Skill Level up ?**
  
A. Yes. You can gain xp as same as Vanilla Skill. Formula is same as vanilla.
  
When fXPPerSkillRank is set to 0 by other mod, You can not gain Player Level XP.
  
  
**Q. Item/object does not works.**
  
A. Check installation/Requirement of .NET Script Framework  , Custom Skills Framework.
  
  
**Q. Version for Classic/Legendary?**
  
A. No. .NET Script Framework and Custom Skills Framework does not support older skyrim. It's only for Skyrim SE.
  
  
  
**Skill Experience**[line]
  
XP for next level : XpBase + SkillLevel \* XpMult \* (1+Legenday\*0.1)
  
  
Global Variables ID
  
zUDXPBase : XPBase (Default = 10)
  
zUDXpMult : XPMult (Default = 1.2)
  
  
  
**Cheat**[line]
  
If needed,  use console command "Set (Global Variable ID) to x", x is value.
  
  
Global Variables ID
  
zUDSkillLevel : Current Skill Level of Unarmoured Defense. (It must be 15~100).
  
zUDPerkPoint : Skill points of Unarmoured Defense
  
zUDLegendaryCount : The count of how many times this skill has been legendaried
