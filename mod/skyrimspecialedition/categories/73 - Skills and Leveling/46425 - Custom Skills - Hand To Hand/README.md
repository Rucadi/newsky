**Manual for Pugilist**[line]
  
1.You can open perk tree of Hand-To-Hand by touching the **Grand Champion's Diamond** or by using the **[Perk] Hand-To-Hand**.
  
  
2.Hand-To-Hand skill applies to bare hands or weapons that use the HandToHand animation.
  
  
3.The power of Hand-To-Hand weapons is defined by the power of bare hands.
  
  
4.Hand-To-Hand skill level can be increased by defeating enemies with Hand-To-Hand. The higher the level you defeated, the more experience you will gain.
  
  
5.One skill point is awarded when Hand-To-Hand skill level reaches a multiple of **4**.
  
  
6.Some perks, such as Body Blow, do not work with bare hands. You need to equip Hand-To-Hand weapon.
  
  
7.You need to Legendarize Hand-To-Hand skill to get powerful perks.
  
  
8.You can earn skill points by repeatedly Legendarizing Hand-To-Hand skills.
  
  
9.Unused skill points can be converted into regular perk points with **Sword of the Grand Champion**.
  
  
  
**Perk List**[line]
  
[ ] : require legendarize handtohand skills
  
  
**Grasping Power** : HandToHand strikes do 5/10/15/[20/25] additional damage.
  
**Artificial Claw** : Increase HandToHand power same as Khajiit race. If Khajiit, increase a little more power.
  
**Concentration** : HandToHand strike do 5/10/20% chance of a critical hit that does 5pt more critical damage.
  
**Body Blow** : Attack with HandToHand weapon do 5/10/20pt stamina damage.
  
**Guardbreak** : Power attacking with HandToHand weapon stagger target when target is blocking.
  
**Meridian Channel** : Attack with HandToHand weapon do magicka damage as same as stamina damage by Perk:Body Blow.
  
**Unreleting Blow** : Power attack with HandToHand weapon push away target when target is exhausted.
  
**Counter Punch** : HandToHand strikes do 5/10/15/[20/25]% more damage when target is attacking.
  
**Doubly Fist** : HandToHand weapon do more stagger when target is attacking.
  
**Spear Hand** : HandToHand Attacks ignore 25/50/75% of armor.
  
**KnifeHand** : Light attack with HandToHand do extra damage. + (one-handed skill)/10 .
  
**Pugilist Stance** : Power attacks with HandToHand cost 30/35/40% less stamina.
  
**Black Hand** : Sneak attacks with HandToHand do 6/10/16 times damage.
  
**Headlong rush** : Stagger Resistance increase 25% while HandToHand attacking, 50% while Power attacking.
  
**Hard Fist** : HandToHand strikes do 3/6/10/[15/20] additional damage while protection spells like Stoneflesh is active.
  
**Double Sledge Hammer** : Heavy attack with HandToHand do extra damage. + (two-handed skill)/10 .
  
**Thankful Fists** : HandToHand weapon attacks are 25% faster.
  
**Faith is Muscle** : HandToHand strikes do 3/6/10/[15/20] additional damage when player doesn't wear heavy/light armors.
  
**Evasion** : 10% chance of avoiding all damage from a melee attack while not wearing heavy / light armors and Striking with HandToHand.
  
**Lucky Punch** : 10% chance to Power-attaking with HandToHand do 100% more damage.
  
  
  
**FAQ**[line]
  
**Q. Compatible with other perk mod?**
  
A. Perfectly. Custom Skills Framework give us independent perk tree.
  
  
**Q. Compatible with other unarmed mods ?**
  
A. Yes. 
  
  
**Q.Perk of this mod apply to hand-To-Hand weapon of other mods?**
  
A. Perks apply to all weapon that use hand-To-Hand/Unarmed animation.
  
  
**Q. This mod use perk point?**
  
A. No. This mod add new skill point for Hand-To-Hand.
  
  
**Q. Item/object does not works.**
  
A. Check installation/Requirement of .NET Script Framework  , Custom Skills Framework.
  
  
**Q. Version for Classic/Legendary?**
  
A. No. .NET Script Framework and Custom Skills Framework does not support older skyrim. It's only for Skyrim SE.

**Skill Experience**[line]
  
XP for next level : **XpBase + SkillLevel \* XpMult \* (1+Legenday\*0.1)**
  
XP : Level of npc who player defeated with hand-To-hand.
  
  
Global Variables ID
  
zh2hXPBase : XPBase (Default = 5)
  
zh2hXpMult : XPMult (Default = 1.1)
  
  
  
**Cheat**[line]
  
If needed,  use console command "Set (Global Variable ID) to x", x is value.
  
  
Global Variables ID
  
zh2hSkillLevel : Current Skill Level of Hand-To-Hand. (It must be 15~100).
  
zh2hPerkPoint : Skill points of Hand-To-Hand
  
zh2hLegendaryCount : The count of how many times this skill has been legendaried
