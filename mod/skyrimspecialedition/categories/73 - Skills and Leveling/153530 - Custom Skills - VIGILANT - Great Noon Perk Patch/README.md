# Custom Skills - VIGILANT - Great Noon Perk Patch
- Author: Viconia1
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/153530


This is a simple patch I created that changes the "Great Noon" perk in the Vigilant Custom Skill tree to work with any sun spell, as long as it uses the appropriate sun damage keyword.
