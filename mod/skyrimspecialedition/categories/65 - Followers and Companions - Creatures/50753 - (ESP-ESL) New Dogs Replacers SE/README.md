# (ESP-ESL) New Dogs Replacers SE
- Author: gg77
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/50753


This mod replaces all  [Dogs of Skyrim](http://en.uesp.net/wiki/Skyrim:Dog#:~:text=Skyrim%3ADog.%20Dogs%20are%20found%20throughout%20Skyrim.%20A%20dog,although%20Death%20Hounds%20only%20ally%20themselves%20with%20vampires.)  (except the Spectral Warhound and the Armored Huskies) with new breeds**\***
  
  
![](https://i.imgur.com/PSExaTq.gif)
  
  
No Barbas replacer in the versions 1.0, 2.0, 2.5, because the statue of Clavicus Vile with barbas uses Irish Wolf hound mesh
  
  
This mod ONLY changes the breed and size of dogs
  
  
  
***You will have to choose between this mod and one of my [Dog Replacer SE](https://www.nexusmods.com/skyrimspecialedition/mods/50500) mods !***
  
  
[![](https://nsm09.casimages.com/img/2021/06/04//21060407103816616017448439.jpg)](https://www.casimages.com/i/21060407103816616017448439.jpg.html)
