# True Meeko SE
- Author: Tahira
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/12193


[size=4][Skyrim Version](https://www.nexusmods.com/skyrim/mods/70779/?)
  
  
This Mod changes the appearance of the companion dog Meeko.
  
There are 9 different colours available.
  
It also makes him a bit tougher, essential and stealthier.
  
He has no longer a level cap and he will level with you.[font=Trebuchet MS]
  
[/font][font=Trebuchet MS]
  
[font=Lucida Sans Unicode]**Requirements**[/font][/font]
  
Skyrim SE
  
  
[font=Lucida Sans Unicode]**Compatibility**[/font]
  
Place this mod after anything that makes changes to [font=Trebuchet MS]Meeko. [/font]
  
No conflicts for texture or mesh replacer because Meeko got his own unique model.
  
Works fine with Hearthfire. Your children can adopt him without a problem.
  
  
**[font=Lucida Sans Unicode]Known Issues[/font]**
  
If you found him before you installed the mod, you have to manually set him to essential.
  
Type > **setessential 000D95E9 1** < in the console.
