# Fluffers - Winterish - Christmas Outfit 3BA
- Author: wammy
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/81068


Hello and welcome to my modpage!
  
  
This one has been a ride. I present you my first outfit ever.
  
 Made from scratch with the help of blender, gimp and outfitstudio.
  
Many things were learned over the course of many moons.
  
Hope you like it! The ground files are a Christmas ball.
  
So it can double as decor.
  
  
  
Contains the following:
  
1x Onesie. or Shortsy. 
  
1x Mufflers. Ear muffs. 
  
1x Mittens. Warm hands.
  
1x Socks. Long Socks.
  
1x Choker. harder daddy
  
  
Requirements:
  
[Bodyslide](https://www.nexusmods.com/skyrimspecialedition/mods/201)﻿[Acro's 3BA](https://www.nexusmods.com/skyrimspecialedition/mods/30174)﻿
  
  
How to get:
  
1. Build in bodyslide first.
  
2. [AddItemMenu](https://www.nexusmods.com/skyrimspecialedition/mods/17563) or console. search for fluffy
  
Conflicts:
  
None
