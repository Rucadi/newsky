# Wanderer's Backpacks
- Author: Remiros
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/68965


Description:
  
This mod adds two new backpacks to the game. They can be crafted at any forge and are balanced to be consistent with the Adventurer's Backpack creation, but it's not a requirement. The textures are 2048x2048.
  
=================================================================================================
  
Installation:
  
Install the mod using your mod manager of choice and activate the esp.
  
=================================================================================================
  
Uninstallation:
  
Deactivate and remove the esp and BSAs.
  
=================================================================================================
  
Conflicts:
  
This mod should be compatible with everything
  
=================================================================================================
  
Requirements:
  
Only the base game
