# COCO LaceBody - CBBE-TBD SE
- Author: COCO
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/30449


[font=Comic Sans MS]**--﻿﻿- Description ---**[/font]**A clothing mod featuring 13 unique lace clothing designs made by COCO. The outfits come with CBPC/HDT-SMP with CBBE SE support.**
