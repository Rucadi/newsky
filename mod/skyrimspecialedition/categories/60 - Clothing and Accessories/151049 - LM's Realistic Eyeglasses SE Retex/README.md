[right]﻿﻿![](https://i.imgur.com/MUEgtt6.png)[/right][right][/right]**Info:**
  
I made this for myself and thought people might appreciate me sharing.
  
  

  
* textures made by me
adapted mesh values + added vertex paint shading
  
* ESP patch (is light)
fixed 6 ESP records who had the wrong glasses[list]
  
* Humanoids silver frame / black glasses for women
all Argonian frames / purple glasses for women
  

  
[\*]textures are 2k, BC7/1
  
[\*]makes use of ENB's Dynamic Cubemaps
  
[\*]covers all existing assets for all races, men and women
  
[\*]optional UBE patch
  
[\*]optional standard cubemap
  
[/list]
  
**Installation:**
  

  
* Enable Dynamic Cubemaps in your ENB ini.
Install ported original.
  
* Install my mod, load it after the original mod.
If you use UBE, install optional UBE Realistic Eyeglasses Patch.
  

  
If you don't use dynamic cubemapping install the optional file, let it overwrite my MAIN mod. 
  
  
**Compatibility:**
  
Works with Unofficial Patch for MannGT's Realistic Eyeglasses - No Glasses for NPCs.
