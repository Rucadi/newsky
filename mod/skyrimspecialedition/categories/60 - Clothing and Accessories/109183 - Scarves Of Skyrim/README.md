# Scarves Of Skyrim
- Author: QwibQwib
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109183


Hello!
  
  
 ﻿This mod adds a new set of craftable scarves to the game with brand new model's and textures. 
  
They can be crafted at the Tanning Rack using one Linen Wrap.
  
  
NPC distribution is here thanks to LuciusPortanius24
