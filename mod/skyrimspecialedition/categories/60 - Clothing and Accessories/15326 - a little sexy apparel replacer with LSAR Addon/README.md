Install:
  
 Please do either the following.
  
 1. Use NMM.
  
 You can install this mod with fomm-script-base installer.
  
  
 2. Use BAIN ( experimental )
  
  
 And, if you use LSAR Addon Blow up, You Need to install FNIS, and run GenerateFNISForUsers.exe.
  
  
Configuration:
  
 1. Please open
  
 YOUR\_SKYRIM\_FOLDERDataSKSEpluginszzLSARSetting.ini
  
 You can modify base settings of this mod.
  
  
 2. In game, please open MCM menu for a little sexy apparel replacer.
  
 You can modify some common settings of this mod.
