**HANDHELD LANTERNS - HDT-SMP**
  
  
This mod adds handheld, HDT-SMP versions of three popular lantern mods by [QwibQwibMods](https://next.nexusmods.com/profile/QwibQwibMods?gameId=1704) and ﻿[TalesOfStar](https://next.nexusmods.com/profile/TalesOfStar?gameId=1704).
  
Included are optional Non-SMP versions, performance textures, ENB glow maps, and more!
  
 ﻿ ﻿ ﻿ ﻿
  
  
**USAGE**
  
  
The main version of the mod requires [FSMP - Faster HDT-SMP](https://www.nexusmods.com/skyrimspecialedition/mods/57339) or [HDT-SMP Flex](https://www.nexusmods.com/skyrimspecialedition/mods/101564).
  
These mods are **not** required, but please **download** and **endorse** them to show some support!
  
  
﻿﻿ ﻿ ﻿? [Elden Lantern](https://www.nexusmods.com/skyrimspecialedition/mods/103985) ﻿
  
﻿ ﻿ ﻿? [Simple Wearable Lanterns - Legacy](https://www.nexusmods.com/skyrimspecialedition/mods/58464)﻿
  
﻿ ﻿ ﻿? [Simple Wearable Lanterns - Remastered﻿](https://www.nexusmods.com/skyrimspecialedition/mods/132196) ﻿
  
  
If using [Constructible Object Custom Keyword System](https://www.nexusmods.com/skyrimspecialedition/mods/81409) the lanterns will appear under "**Misc**".
  
Lanterns will not appear in the forge unless you have **1 x Lantern (000318EC)** present in your inventory.﻿
  
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/99892/99892-1736632256-120407214.gif)﻿

Install [Favorite Misc Items](https://www.nexusmods.com/skyrimspecialedition/mods/42750) to allow using from favorite menu along with hotkeys.
  
  
Check out [Handheld Lanterns Animations](https://www.nexusmods.com/skyrimspecialedition/mods/139652) for some great new animations.

**COMPATIBILITY and LIMITATIONS**
  
  
﻿**⚠️ NPCs DO NOT WORK**
  
 ﻿
  
 ﻿ ﻿Physics don't engage on NPCs. I'm open to solutions!
  
 ﻿
  
﻿**⚠️ ENABLING 1ST PERSON SUPPORT**
  
  
 ﻿ ﻿HDT-SMP physics only work in 3rd person by default.
  
 ﻿ ﻿To enable 1st person physics, do the following:
  
 ﻿ ﻿
  
 ﻿ 1. Make sure **"No SMP for your PC when in 1st person view"** is **unticked** in the FSMP Mod Configuration Menu:
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/99892/99892-1736145295-1509018808.png)

  
  
 ﻿ ﻿﻿2. Enable "**Third Person Torch**" using [Improved Camera SE](https://www.nexusmods.com/skyrimspecialedition/mods/93962):
  
 ﻿ ﻿ ﻿ 2A. Open the menu with **LEFT SHIFT + HOME** keys.
  
 ﻿ ﻿ ﻿2B. Choose **"General"** from the **"View"** dropdown.
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/99892/99892-1735953427-1135960038.gif)﻿

  
**⚠️ CHANGING ARMOR SLOT**
  
  
 ﻿ ﻿The HDT-SMP versions require an equipment slot in order to engage the physics. **SLOT 49** is used.
  
 ﻿ ﻿ This can be modified using [xEdit](https://www.nexusmods.com/skyrimspecialedition/mods/164) by navigating towards the Armor and Armor Addon records and changing the biped slot.
  
  

![](https://staticdelivery.nexusmods.com/mods/1704/images/99892/99892-1734486319-1658992055.png)﻿
