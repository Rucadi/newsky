# Common Clothing Expansion
- Author: Hornophobic aka Hornclonic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/134088


**DESCRIPTION**
  
This mod adds 4 lore friendly outfits to the game (+ one hood) , designed to fit Skyrim's Vanilla aesthetic.
  
The textures are hand painted in 2k, and meshes were modelled in blender.
  
Meshes are kept similar to vanilla poly, male trousers are same poly as Himbo.
  
  
  
**COMPATIBILITY**
  
  
Bodyslide files are provided, feel free to build to fit custom bodies.
  
Prebuilt meshes are provided for HIMBO and CBBE
  
My esp is compatible with everything and the load order doesn't matter.
