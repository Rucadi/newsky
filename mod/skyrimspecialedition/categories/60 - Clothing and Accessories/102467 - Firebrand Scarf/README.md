# Firebrand Scarf
- Author: Edryu - Sfordinza
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/102467


**Firebrand Scarf**I tweaked the meshes of the [Firebrand armor by Sfordinza](https://sforzmods.tumblr.com/sse_armours) to keep only the scarfs, making it usable with other outfits. The scarf uses the slot 45 and all the included colors can be crafted at the tanning rack with linen wrap.
