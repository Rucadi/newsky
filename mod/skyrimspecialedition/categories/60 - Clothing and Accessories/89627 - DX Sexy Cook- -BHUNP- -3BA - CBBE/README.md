*[font=Times New Roman]Installation[/font]
  
[font=Times New Roman]
  
[/font]*[font=Times New Roman]*If you already have all the requirements, install this mod using Vortex or Mod Organizer 2, then, depending on which Mod Manger you are using, deploy or activate the mod in your load order. (MO2 will read the "CalienteTools" folder automatically.)*[/font]*![](https://cdn.discordapp.com/attachments/930678149988098098/1099259053504475176/Screenshot_809.png)﻿*[font=Times New Roman]*To get the outfits to show up in-game, open BodySlide, the DX etc will be* *under* *"Unassigned", navigate to "Batch Build" and tick all of them (don't forget to* [/font]*[font=Times New Roman]check "Build Morphs).
  
[/font]**[font=Times New Roman]UnInstallation[/font]*[font=Times New Roman]*If you installed it via mod manager, just delete the mod from your load order.**If you**installed it manually, just delete the files you copied*.[/font]*[font=Times New Roman]
