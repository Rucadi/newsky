# Cultural Bards - CBBE 3BA
- Author: masimeiro
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/133265


CBBE 3BA Bodyslides for the awesome [Cultural Bards](https://www.nexusmods.com/skyrimspecialedition/mods/133000)﻿ clothing mod. I added body physics and UniBoob and UniLowerCurtain sliders where it makes sense.
