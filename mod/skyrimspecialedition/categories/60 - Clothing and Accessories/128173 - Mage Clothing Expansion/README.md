# Mage Clothing Expansion
- Author: Hornophobic aka Hornclonic
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128173


[font=Tahoma]**DESCRIPTION**[/font]
  
  
This mod adds 18 (9 male/9 female) new robes all carefully crafted from existing vanilla assets,
  
equally divided between male and female characters, adding much needed diversity to your mages.
  
Each robe has an alternative darker colour variant similar to vanilla. (total of 25  robes + One Hood)
  
  
The items can be crafted at a tanning bench, you need 3 x Leather.
  
Alternatively you can use the SPID patch which will distribute all Modest outfits to College NPCs.
  
  
[font=Tahoma]
  
[/font]**[font=Tahoma]COMPATIBILITY[/font]**
  
  
Bodyslide files are provided, feel free to build to fit custom bodies.
  
Prebuilt meshes are provided for HIMBO and CBBE Curvy.
  
My esp is compatible with everything and the load order doesn't matter.
