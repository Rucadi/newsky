# Shas' Chains and Silks (SMP-3BA)
- Author: Shas_Sa-al_L-Dred
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/143720


![](https://i.imgur.com/qfvKi5B.png)

  
  
  

![](https://i.imgur.com/Q72hTIV.png)

  
  
Outfit includes: Braces (with or without the chains), Body Silks, Veil, Top & "Skirt"
  
Base meshes are not included, outfit must be built in bodyslide to use
  
Included bodyslides are made for 3BA body
  
Acquirable only trough the console, additem menu or a similar mod
  
  

![](https://i.imgur.com/sS1GlrS.png)

  
  
  

![](https://i.imgur.com/7CCri6T.png)
