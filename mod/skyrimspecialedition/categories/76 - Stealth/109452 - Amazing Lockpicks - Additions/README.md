# Amazing Lockpicks - Additions
- Author: Eudalus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/109452


**Description**
  
[left]Adds additional lockpicks to [Amazing Lockpicks](https://www.nexusmods.com/skyrimspecialedition/mods/108832) using the textures from the Saints & Seducers Creation and the mods [Wyrmstooth](https://www.nexusmods.com/skyrimspecialedition/mods/45565) and [Beyond Skyrim - Wares of Tamriel SE](https://www.nexusmods.com/skyrimspecialedition/mods/31519). You can choose from the standalone or merged versions during the fomod install.
  
  
From weakest to strongest, the added lockpicks are:
  
  
Brimstone, mithril, wyrmstone, amber, and madness.
  
  
Recommend using [Amazing Lockpicks - Enhancements](https://www.nexusmods.com/skyrimspecialedition/mods/109455) as well!
  
Add even more lockpicks with [Amazing Lockpicks Radiance](https://www.nexusmods.com/skyrimspecialedition/mods/109667).[/left]
  
**Installation**
  
Install following the fomod instructions.
  
**[left][/left]****Requirements**
  
[left][Amazing Lockpicks](https://www.nexusmods.com/skyrimspecialedition/mods/108832)
  
[Wyrmstooth](https://www.nexusmods.com/skyrimspecialedition/mods/45565) for brimstone and wyrmstone lockpicks.
  
[Saints & Seducers Creation](https://bethesda.net/en/article/fxoBorhkvKlhvAjkiMBuU/golden-saints-and-dark-seducers-arrive-in-skyrim) for amber and madness lockpicks.
  
[Beyond Skyrim - Wares of Tamriel SE](https://www.nexusmods.com/skyrimspecialedition/mods/31519) for mithril lockpicks.[/left]
