**DESCRIPTION:**
  
  
I've always loved [Colorful Magic](https://www.nexusmods.com/skyrimspecialedition/mods/13261), it's a fantastic, expansive mod with lots of spells from every school of magic. Unfortunately, the original version had quite a few bugs, some of which were fixed by 184Gesu, but a large number of these bugs remained and that mod was never updated past 2018. I took it upon myself to remove most of these bugs, and keep only the "immersive", lore-friendly spells while removing the bosses and NPCs added by Colorful Magic.
