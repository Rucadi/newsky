# Call of Valor Tweaks
- Author: AndrealletiusVIII
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/113754


**Description**
  
  
I found the [Call of Valor](https://en.uesp.net/wiki/Skyrim:Call_of_Valor) shout was very underwhelming. It's supposed to be a reward for completing the main quest. Just one summonable hero isn't a lot and I ended up never using it. As such, I changed the Call of Valor shout, in an attempt to make it more useful:
  
  

  
* Using one word summons [Gormlaith Golden-Hilt](https://en.uesp.net/wiki/Skyrim:Gormlaith_Golden-Hilt).
Using two words summons [Gormlaith Golden-Hilt](https://en.uesp.net/wiki/Skyrim:Gormlaith_Golden-Hilt) and [Felldir the Old](https://en.uesp.net/wiki/Skyrim:Felldir_the_Old).
  
* Using three words summons [Gormlaith Golden-Hilt](https://en.uesp.net/wiki/Skyrim:Gormlaith_Golden-Hilt), [Felldir the Old](https://en.uesp.net/wiki/Skyrim:Felldir_the_Old). and [Hakon One-Eye](https://en.uesp.net/wiki/Skyrim:Hakon_One-Eye).

  
I increased the duration to 120 seconds. (up from 60)
  
  
In addition, I've buffed the 3 of them, so they can actually hold their own in battle. I also gave them new and better outfits, weapons, perks, shouts, classes and combat styles.
  
  
I have included an optional which doesn't have completing the main quest automatically unlock the words of Call of Valor. You will need to spend Dragonsouls to unlock them. (see optional file "Dragonsouls Reqs".)
  
  
I also added a patch which changes their outfits back to Vanilla. Note that they will be significantly weaker if you choose this.
