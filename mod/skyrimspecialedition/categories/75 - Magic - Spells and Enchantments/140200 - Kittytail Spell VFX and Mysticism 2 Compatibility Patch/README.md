# Kittytail Spell VFX and Mysticism 2 Compatibility Patch
- Author: Kittytail and SimonMagus
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/140200


Kittytail's mods are of exceptional quality. However, they are only compatible with spells that use vanilla hand and projectile spell effects. This patches Destruction master spells added by mysticism to use Kittytail's work.
