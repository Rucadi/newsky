# Spellforge libraries for some mods (KittyTail Darenii and More)
- Author: Enhab
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/94243


Some [Spellforge](https://www.nexusmods.com/skyrimspecialedition/mods/46482?tab=description) libraries for some popular spell packs as well as smaller spell mods I like to use. No bigger spell mods because those can already be found in Spellforge's installer.
  
These aren't base libraries, which means you can install all of these and Spellforge will import all of them back-to-back with no issues.
