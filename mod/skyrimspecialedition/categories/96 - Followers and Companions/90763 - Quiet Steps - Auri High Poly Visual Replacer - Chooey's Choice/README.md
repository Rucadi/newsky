**Installation steps:**
  
  
1. Install [Chooey's Choice Requirements](https://www.nexusmods.com/skyrimspecialedition/mods/90689) & [SkyPatcher](https://www.nexusmods.com/skyrimspecialedition/mods/106659)
  
2. Install [Auri](https://www.nexusmods.com/skyrimspecialedition/mods/11278)
  
3. Install this mod
  
4. Choose the options you want in the FOMOD.
  
5. That's it, have fun!
