# Maux Follower
- Author: Canliberk
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/65571
Basic but beatiful female follower mod that only uses vanilla assets. Since it uses vanilla assets, it is so light that it is only 1 KB size.
  
Pictures you see on this mod page are taken with different clothing mods.

This mod has been created to add follower variations to the [Immersive & Adult collection](https://next.nexusmods.com/skyrimspecialedition/collections/xxsqm4?tab=about) for Skyrim AE.
  
You do not have to have the collection to be able to use this mod, but you may not get the exact same look you see on images.
  
  
  
This follower is located in Dragonsbridge, at the inn, four shields tavern.
  
  
Maux is a 2-handed Redguard sword master. Her main skills are:
  
2 handed
  
Heavy armor
  
Archery
  
Sneak
  
Restoration
