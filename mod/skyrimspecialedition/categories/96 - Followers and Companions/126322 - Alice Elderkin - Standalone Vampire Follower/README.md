[font=Georgia]**INSTALLATION﻿\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_\_**[/font]﻿
  
[font=Georgia]Download the main file located in the files tab and install it using your Mod Manager[/font][font=Georgia]
  
  
In the FOMOD, you will see two options to choose from. One is for vanilla clothing, and the other is to replace it with the amazing Dark Witch mod by author OperatorCactus. To use the mod's clothing, you will need to install the original mod [here](https://www.nexusmods.com/skyrimspecialedition/mods/112260)﻿
  
  
**Alice will be located near the boat at Icewater Jetty by default
  
  
If you use a version of the game other than the latest, you need [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441)﻿, it is mandatory to mod everything**[/font]
