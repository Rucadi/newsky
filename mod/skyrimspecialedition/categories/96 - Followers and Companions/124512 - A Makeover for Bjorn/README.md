# A Makeover for Bjorn
- Author: Yennethan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/124512


A Bjorn replacer using the race mod COtR and the High Poly patch for COtR. You DO NOT need to have the COtR race mod installed. This replacer is mostly standalone (he uses whatever body mesh you have installed). He has a few options and patches, check the photos for reference.
