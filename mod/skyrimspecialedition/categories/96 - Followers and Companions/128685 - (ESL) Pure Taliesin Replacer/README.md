# (ESL) Pure Taliesin Replacer
- Author: tatururutia
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/128685


This mod will change the apperence of Taliesin.
  
  
Three different hair color versions are available: **black**, **blonde**, and **white**.
  
High Poly Head and ESL flagged.
  
  
*I really like Taliesin's voice.*
  
*He is not a typical Thalmor, a bit like Rumarin in 3DNPC, but a bit more arrogant and narcissistic than Rumarin.*
  
*These interesting personality traits make Taliesin even more attractive.*
  
*Thanks to Vanessa M and Pat Mahoney for making such a wonderful follower.*
  
  
[size=4]----------------------------------------------------------------------------------------------------
  
  
Update for 1.01: Fixed an issue that the forehead and eyebrows were a bit low.
  
  
*![](https://i.imgur.com/t8fhqPf.jpeg)*
  
  
----------------------------------------------------------------------------------------------------
  
  
[/size]**His weight has been changed from 10 to 40, so if you have Neck gap problem:**
  
  
Open the console, **click Taliesin**, then type "**setnpcweight 40**", enter, then you will see Taliesin have a wired face, 
  
type "**disable**" , enter, then type "**enable**", enter, he will return to normal.
  
  
Or starting a new game, then Taliesin won't have neck gap.
  
  
Or using [save unbaker](https://www.nexusmods.com/skyrimspecialedition/mods/85565), it is a great option that will prevent the weight bug.
