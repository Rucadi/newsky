Replacer for the follower mod [Jesper](https://www.nexusmods.com/skyrimspecialedition/mods/132362)
  
  
? Standalone
  
? .esl Flagged
  
? Requires Skypatcher (check requirements)
  
? Optional SPID file that makes Jesper wear Sons of Skyrim armour without helmet, like in the pics. You need this file [Housecarls - Sons of Skyrim (SPID)](https://www.nexusmods.com/skyrimspecialedition/mods/82845)? Optional [Skyrim Party Sheet](https://www.nexusmods.com/skyrimspecialedition/mods/167538) Portrait, see images
  
  

**Installation**
  
![](https://pixelsafari.neocities.org/dividers/3pinkflowers.png)

  
Use a mod manager/organizer to install or drop the loose files into your Skyrim data folder.
  
  
Your Skypatcher.ini located in SkyPatcher\SKSE\Plugins should have these options like this:
  
  
[color=#d9ead3][size=3][Patcher]
  
iEnableNPCPatching=1
  
  
[Features]
  
iUpdateNPCVisualsOnLoad=1
  
  
If using RSV mod, please add Jesper to the exclusion list[/size][/color]
