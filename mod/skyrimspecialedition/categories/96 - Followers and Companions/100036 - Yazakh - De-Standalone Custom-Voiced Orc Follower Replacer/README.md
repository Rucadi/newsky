# Yazakh - De-Standalone Custom-Voiced Orc Follower Replacer
- Author: nazarethblood
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/100036


The original mod by [chob0318](https://www.nexusmods.com/skyrimspecialedition/users/42072405) had a custom body and textures. This mod will allow you to use your own body and textures when installed over the original mod. It is also flagged ESPFE. In my testing, you will get good results if you use BnP - Female Skin. 
  
  
Please consider endorsing the original mod.
  
  
**Original Mod Required:**
  
Yazakh - Custom-Voiced Orc Follower Replacer - Standalone Follower
  
<https://www.nexusmods.com/skyrimspecialedition/mods/98398>
  
  
**Installation Instructions:**
  
  
Step 1: Check under Files tab of this page if a particular version of the original follower mod is required.
  
  
Step 2: Download and install mod from original follower page (found under requirements).
  
  
Step 3: Download and install desired version of my conversion mod. 
  
  
Step 4: Profit.
