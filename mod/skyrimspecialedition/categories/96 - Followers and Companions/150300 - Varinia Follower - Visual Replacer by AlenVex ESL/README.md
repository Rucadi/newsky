# Varinia Follower - Visual Replacer by AlenVex ESL
- Author: AlenVex
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/150300


This is what I kind of envisioned Varinia in my head - she appears young and innocent, but in reality she is the scariest thing in the room.
  
No requirements other than the original Follower [Varinia](https://www.nexusmods.com/skyrimspecialedition/mods/148853) mod.﻿
  
  
  
  
She will use *your Own Textures and Body mods*. I used BnP skin for the screenshots.
  
  
  
  
Please let me know if there are any issues so I can fix them asap :)
  
*If you like the replacer - please endorse and thank you!*

﻿﻿
  

**[font=Georgia][Compatibility and Issues][/font]**
  
  
  
1. Load this after the original [Varinia](https://www.nexusmods.com/skyrimspecialedition/mods/148853) mod. Or use XEdit if there's issues or conflicts.
  
  
2. Her weight matches the original, so no neck gap should occur.
  
  
3. If she Has a Dark Face - make sure this mod isn't being overwritten by another replacer. XEdit or MO2 can help with that. Remove any other replacer you're using to be safe.
  
  
4. If she looks weird or if her face didn't change - make sure the .esp is actually enabled in your Load Order.
