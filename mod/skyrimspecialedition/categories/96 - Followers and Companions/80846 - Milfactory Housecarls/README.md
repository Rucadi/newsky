This mod contains total makeover for all vanilla female housecarls: Lydia, Iona, Jordis The Sword-Maiden and Rayya. 
  
They are intended by design to be more thoughtful, mature yet still attractive and have a practical look. Milfactory walked that thin line between tough-looking, lore-fitting Nordic warriors and sensually mature beauty to deliver to you this overhaul pack. 
  
  
As the first Milfactory NPC pack, this mod requires [Milfactory Asset Hub](https://www.nexusmods.com/skyrimspecialedition/mods/80819) to work.
  
  
  
  
**Some additional info:**
  

  
* They are all set to use correct female animation.
All housecarls are set to essential.
  
* All housecarls are set to 100 weight.
Rayya has her hood removed.
  
* No stats alternated, no combat styles or AI package modified.
All screenshots were taken on BHUNP version.
  
* This pack is compatible with both BHUNP and 3BA version of the [Asset Hub](https://www.nexusmods.com/skyrimspecialedition/mods/80819)﻿.

Q: I got CTD / Mod not working / Game not starting
  
A:[Install the Milfactory Asset Hub first.](https://www.nexusmods.com/skyrimspecialedition/mods/80819) 
  
  
Q: I got texture mismatch on their body. How to fix that?
  
A: Since the asset hub uses whatever body you have installed in your game, you need to pick the correct version of the Asset Hub
  
  
Q: I got neck gap. How to fix that?
  
A: Remove all the valuable from their inventory.
  
Open console command, click on them and type: "setnpcweight 100" without quote then press enter
  
Then type "disable" enter. Then type "enable" enter
  
Alternatively you can try "resurrect" enter.
