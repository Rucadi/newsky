# Misty Skye - 3BA Preset Pack
- Author: Granhadd
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/146208


[font=Trebuchet MS]**THE MOD:**
  
  
This mod features a pack of 3 custom Bodyslide presets for CBBE 3BA bodies, inspired by characters from *[The Erotic Adventures of Misty Skye](http://www.nexusmods.com/skyrimspecialedition/mods/16374)*. The presets are designed to closely resemble the original appearances of the NPCs Misty, Summer, and Kara, who were based on UNP bodies in the original mod.
  
  
*The Erotic Adventures of Misty Skye* is not required for this pack, but if you wish to apply these presets to the 3 NPCs, you will need to also download *[Misty Skye CBBE 3BA Patch](http://www.nexusmods.com/skyrimspecialedition/mods/115025)* along with this mod.
  
  
For users of my other mod, *[OBodies - 3BA Preset Pack](https://www.nexusmods.com/skyrimspecialedition/mods/163386)*, I’ve already included support for distributing these 3 presets to the aforementioned NPCs.
