# Milfactory Asset Hub
- Author: Silver aka Silver Wolf
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/80819


![](https://i.imgur.com/MiDLrJP.png)

  
![](https://i.imgur.com/fIIStUK.png)
  
  
This mod contains assets and resource needed for upcoming Milfactory mods, which include but not limited to: Followers, NPC Replacers and future collaboration. The current number of planned NPCs is around 20 so an Asset Hub is absolutely needed in order to streamline Milfactory production as well as to reduce/optimize resources consumption for users during normal gameplay, should they decide to use a large amount of Milfactory characters.
  
  
This Asset Hub does not contain any body mesh. It will instead use whatever body you installed in your game, so make sure to pick the correct version to avoid texture mismatching issue. As of right now, both BHUNP and 3BA will be supported. 
  
  
  
**From version 2.0 the plugin has been updated to ESPFE instead of ESM
  
Please redownload all the Milfactory companion mods prior to this update.
  
And Thank you for your patience.**
