[/font]**[font=Comic Sans MS]How to use[/font]**[font=Arial]
  
[/font]

[font=Arial]You can change color of scars in Racemenu to best fit your skin tone. I recommend using some shade of red or yellow for most skin tones and opacity of 50% or less.
  
All scars have **dark** and **light** version, from my testing the dark ones look more fresh while the light ones look more healed up, but it depends a lot on your skin textures and skintone.
  
  
  
In Racemenu these scars have prefix **ISO** (name of the mod) followed by either **Dark** or **Light**, part of body they appear on and number.
  
For example **ISO Dark Right Arm Scars 01** or **ISO Light Full Body Scars 02.**
  
UBE overlays have additional prefix **UBE**, so it looks like **UBE ISO Dark Left Leg Scars 03.**[/font]

[font=Comic Sans MS]**Installation**[/font][font=Arial]
  
Plugin was made with most recent CK version so if you're not using latest skyrim version, you might need to use [BEES](https://www.nexusmods.com/skyrimspecialedition/mods/106441)﻿.
  
  
Use mod manager of your choice to install.
  
  
Both main files are standalone and can be used together but most users will probably need only one.
  
  
Both files use light plugin (ESP flagged as ESL) so it won't count towards your 255 plugin limit. 
  
All textures are 4K and packed into BSA to reduce file sized while installed.[/font]
