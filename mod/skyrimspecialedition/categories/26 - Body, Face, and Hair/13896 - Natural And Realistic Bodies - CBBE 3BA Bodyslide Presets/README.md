[font=Georgia]*- **I was starting to get tired of my own character and all female npc having a pinup-body which even by todays standards is unrealistic, let alone Skyrim.** -
  
- **With the harsh climate and ruggid conditions in Skyrim, aint nobody got time for fitness and diets. That's not lore friendly!** -
  
-**I started using Real Women of Skyrim Texture Overhaul some time ago. And with the insanely realistic look that brought.**- **That just didnt fit togheter with a pinup-looking bodyshape. Which lead me to the idea to make these.** -*[/font]
  
  
*[font=Georgia]- **So i started searching for a more realistic preset here on the Nexus and the other page(which shall not be named)** - 
  
- **But sadly I couldnt find anything besides unrealistic and unatural looking bodyslide preset** -[/font]**[font=Georgia]
  
[/font]*
  
  
[font=Georgia]*- **So I started to make my own preset to try and make the female body look more natural and realistic.** - 
  
- **I found myself pretty satisfied with the result and thought i'd share it.** -*[/font]

[font=Georgia]***All bodypresets has a working weightslider un-clothed. Making outfit versions very difficult due the vast amount of different outfits you can choose from. Making any outfits versions that fits well to all outfits/armors without any clippings is time consuming and difficult. I will maybe consider making them if i get enough support for it. Otherwise its a pretty simple fix for yourselves. Just default nipple sliders and set gravity to 0. And then twiddle with the sliders to remove any possible clippings if theres any.
  
  
  
![](https://i.imgur.com/JGengsF.png)***[/font]

**[font=Georgia]*NMM Installation*[/font]**
  
**[font=Georgia]*1. Download to Vortx
  
2. Install
  
3. Enable Mod
  
4. Open bodyslide studio
  
5. Choose the preset of your liking
  
6. Enter game
  
7. Profit!*[/font]**
  
  
[font=Georgia]***\*I would highly recommend you to install via Vortx,
  
but if you still want to install manually follow these steps\****[/font]
  
**[font=Georgia]*Manual installation*[/font]
  
  
[font=Georgia]*For the bodies:*[/font]****[font=Georgia]*1. Select manual download
  
2. Extract the folder
  
3. Select which bodies you want
  
4. Copy them to calientetools/bodyslide/sliderpresets
  
5. Open bodyslide studio
  
6. Choose the preset of your liking
  
7. Enter game
  
8. Profit!*[/font]**
