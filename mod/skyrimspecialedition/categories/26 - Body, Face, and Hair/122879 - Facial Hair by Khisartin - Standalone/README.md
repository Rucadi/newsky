**Features:**
  
  
[size=3][size=3]✧ [/size][/size]**STANDALONE**[size=3][size=3] Changes [Khisartin's Facial Hair](https://www.ddsworkshop.net/facialhair)﻿ to be standalone to improve compatibility[/size]. 11 (+22 for different versions) with different lightness brows for women. 11 [size=3] (+22 for different versions) [/size]brows & 43 beards for men.
  
  
[size=3][size=3]✧ [/size][/size][/size]**REPLACER**[size=3][size=3][size=3] In the FOMOD, you have the option to use the brows and beards as standalone and as a replacer[/size][/size] for vanilla textures
  
  
[size=3][size=3][size=3]✧ [/size][/size][/size][/size]**CONVENIENCE**[size=3][size=3][size=3][size=3] [Khisartin's Facial Hair](https://www.ddsworkshop.net/facialhair)﻿ are now properly packed in a FOMOD, making installation easy[/size][/size][/size]
  
  
[size=3][size=3][size=3][size=3]✧ [/size][/size][/size][/size]**ESL**[size=3][size=3][size=3][size=3] All .esp's in this mod are ESL-flagged, meaning they won't take up normal plugin slots in your load order[/size][/size][/size][/size]
  
[size=3]
  
[size=3][size=3][size=3][size=3]✧ [/size][/size][/size][/size]**NO REQUIREMENTS**[size=3][size=3][size=3][size=3] Thanks to [Khisartin](https://www.ddsworkshop.net)﻿ for permission, all files are in this mod page. Just install this mod and you are good-to-go![/size][/size][/size][/size][/size][/size]
