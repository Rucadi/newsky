# BHUNP BodySlide files for Half Dragon race
- Author: MauxRose
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/67072
**Requirements :**
  
[Half Dragon Race for SSE](https://www.nexusmods.com/skyrimspecialedition/mods/18268)
  
[BHUNP (UUNP Next Generation) SSE](https://www.nexusmods.com/skyrimspecialedition/mods/31126)
  
﻿
  
**Installation :**Look for "BHUNP HalfDragon" in bodyslide, there is four files. Body, hands, and feets (heelsready and normal). Batch build with whatever preset you want, or zeroed if you use autobody.
  
If you want to use optional pubic hair mod 
  
You can get the half dragon preset from screenshots [here](https://www.nexusmods.com/skyrimspecialedition/mods/66928) !
