# Derp's Presets
- Author: Me
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/157444


This is a collection of all my characters presets I've made in Skyrim, and most of the body-slide presets. Made for/using the collection [DOMAIN](https://www.nexusmods.com/games/skyrimspecialedition/collections/gnfjwh) by [Dae](https://next.nexusmods.com/profile/daedaddy?gameId=1704).
  
  
Include a modifies Skee64 file for the Damma Doden preset. All it does is change the limit of every overlay slot to 100.
  
  
Also some presets may require overlay mods not included with DOMAIN, I don't remember what they all are; so, download em all I guess.
