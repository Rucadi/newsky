- Game: skyrimspecialedition

REQUIREMENTS**
  
  
[Bodyslide and Outfit Studio](http://www.nexusmods.com/skyrimspecialedition/mods/201)[CBBE](http://www.nexusmods.com/skyrimspecialedition/mods/198)
  
  
**INSTALLATION**
  
  
After installing the requirements all you have to do is:
  
  
- Install it with your favorite Mod Manager (Vortex, MO2, etc...)      (Recommended)
  
  
[size=3]- Or if you prefer to do it manually then you need to drag and drop the contents of this mod into your Skyrim Folder
  
  
[/size]- Once you have done this you need to open Bodyslide and select the Preset in the Presets tabs, the last step is building the body, you have to select the Build button and then you are done.
  
  
Congratulations!! You have succesfully installed this mod.
