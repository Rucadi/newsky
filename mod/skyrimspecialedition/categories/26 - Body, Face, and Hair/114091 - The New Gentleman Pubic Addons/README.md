# The New Gentleman Pubic Addons
- Author: JuzyJ
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/114091


3D Pubic hair meshes for males and females. Able to match character hair colors!
  
Comes with ESL flaged .esp, zeroed prebuilt meshes, and BodySlides.
  
  
  
[size=5]**Requirements:**
  
  

  
* [The New Gentleman](https://www.nexusmods.com/skyrimspecialedition/mods/104215)﻿ (for distribution)
[Racemenu](https://www.nexusmods.com/skyrimspecialedition/mods/19080)﻿ (for coloring)
  
* [HIMBO](https://www.nexusmods.com/skyrimspecialedition/mods/46311)﻿ and [CBBE](https://www.nexusmods.com/skyrimspecialedition/mods/30174)﻿ (optional, if using custom presets in BodySlide)

  
  
[/size]**Installation:**
  
  

  
* Load behind "The New Gentleman" main mod

**How to use:**
  
  

  
* In In-game mod menu (MCMenu) enable addons
Edit MCMenu settings for NPCs
  
  
* For player character, use assigned button to change addons
(You might want to use "reset skin option" and choosing addon again if facing some problems)
  
* (It does work with body preset distribution mods, as long as meshes are 0-ed, and have .tri files)
Color changes with your character hair color
