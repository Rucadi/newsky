# The Eyes Of Beauty - Vampire Eyes SSE
- Author: LogRaam and JRICH604
- Game: skyrimspecialedition

Thank you [LogRaam](https://www.nexusmods.com/skyrim/users/308756) for the amazing eye textures, and thank you [JRICH604](https://www.nexusmods.com/skyrimspecialedition/users/16885489) for giving me permission to host and support this mod.
  
Thank you [humannature66](https://www.nexusmods.com/users/1341839) for giving me permission to post [HN66s Long Eye Lashes](https://www.nexusmods.com/skyrimspecialedition/mods/21837?) version in this mod.
  
  
**Description**
  
This mod acts as a **replacer of vanilla vampire eyes** (yes, that ugly yellow iris and sclera! or whatever you want to call it...) for **player and all NPC vampires**. Comes in 4 variations (red, red glowing, blue, blue glowing).
  
  
**Glowing effect only works on player**. For vampire NPC the way the game works is they will use the same eye texture as player but they do not use the same mesh. This is because NPC use their own eye mesh inside their exported facegen.
  
  
If you want the NPC vampires to have glowing eyes as well, install [Cosmetic Vampire Overhaul SSE](https://www.nexusmods.com/skyrimspecialedition/mods/8695) version C (it's a must mod! a solution to get rid of goblin vampires and dark face bug on vampires). If you use normal version of this mod, install CVO version B as a compliment.
  
  
It is not a requirement to use CVO but if a different vampire NPC mod is used or if they are still using vanilla facegen then a user should expect the eyes to not look the same on NPC. The texture itself will be the same for player and NPC but how that same texture looks in game will be different if the mesh is different.
  
  
**Recommendation**
  
This mod will go along well with [More Vampire Eyes SSE](https://www.nexusmods.com/skyrimspecialedition/mods/580) and [Cosmetic Vampire Overhaul SSE](https://www.nexusmods.com/skyrimspecialedition/mods/8695)﻿
