# THAE - CBBE 3BA PRESET (0 - 100)
- Author: Thaedrax
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/125866


**EN:** It is a body preset that I used in my gameplay when I was at the peak of my mod list. One day, I gave up on dealing with mods because I wasn't enjoying the game after adding them, so this preset remained as a legacy from a time when I enjoyed adding mods to my game. It is a 0-100 preset. I tried to create a body that ranged from a simpler and more 'youthful' look to a more muscular and defined 'strong' body. It is a simple creation where I used some other Nexus presets as inspiration for the settings. I recommend the 'realistic' gravity setting. If you like the preset, please send screenshots, and if you have any questions, feel free to ask me and I will try to answer your questions. Please endorse if you liked the preset!
