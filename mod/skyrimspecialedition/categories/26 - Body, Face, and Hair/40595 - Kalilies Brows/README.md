# Kalilies Brows
- Author: Kalilies
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/40595


This mod adds 17 standalone eyebrows for female characters, including 3 asymmetrical ones.
  
Texture size is 1k, and for the asymmetrical ones 2k.
  
  
Thank you [Artsick](https://www.nexusmods.com/skyrim/users/41861740) for the mesh!
