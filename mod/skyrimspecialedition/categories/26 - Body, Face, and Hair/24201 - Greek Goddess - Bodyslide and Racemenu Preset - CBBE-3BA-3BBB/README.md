# Greek Goddess - Bodyslide and Racemenu Preset - CBBE-3BA-3BBB
- Author: Vrayth
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/24201


**INSTALLATION**
  
Install with your favorite Mod Manager. For manual installation, simply drag and drop the contents of this mod into your Data Folder. Once everything is installed, open Bodyslide and select this preset and click build. You can use Obody NG if you want to use multiple bodyslide presets at once.
