# Big Beastly Bodies - 3BA BodySlide Presets for Beast Race Follower Mods
- Author: Westonini
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/71547


Bigger proportioned 3BA presets created for beast races in mind, specifically for follower mods that I've created or enjoy using. Highly recommend for use with OBody ([NG](https://www.nexusmods.com/skyrimspecialedition/mods/77016)/[Original](https://www.nexusmods.com/skyrimspecialedition/mods/51084)) or AutoBody ([AE](https://www.nexusmods.com/skyrimspecialedition/mods/61321)/[SE](https://www.nexusmods.com/skyrimspecialedition/mods/64860)) to diversify your beast race bodies! The main file only contains Nude variants since mods like Autobody & OBody have automatic refit algorithms to fit clothes onto the presets. However, an optional file containing Outfit variants is also provided for those who don't use those types of mods.
  
  
Note that some of these presets are **thicc** as hell. If some of them aren't your style, feel free to delete em' from your files. Also, without physics mods such as [CBPC](https://www.nexusmods.com/skyrimspecialedition/mods/21224) to apply gravity to the bits, certain presets will look unnaturally stiff. I'm personally using AutoBody, CBPC, and Faster HDT-SMP.
