[font=Comic Sans MS]Nails added by these mod are implemented through a new racemenu slider, these way they don't occupy a body slot. So they can freely be used along any outfit item[/font]
  

![](https://i.imgur.com/pPRlrpE.png)

  
[font=Comic Sans MS]Nails come in 5 base shapes (Short, Long Squared, Long Almond, Longer Squared, Longer Almond & Longer Stiletto), a few patterns and many colours, making for a total of 297 options to chose from : )[/font]

[font=Comic Sans MS]The colour of the glow effect can be changed in nifscope for all the glowing nails:[/font]
  

![](https://i.imgur.com/vZKBl7K.png)

  

![](https://i.imgur.com/v1D6PhX.png)

  
[font=Comic Sans MS]1. Open the mesh in nifscope
  
2. Expand the corresponding mesh block and select the shader properties block
  
3. Go to the emissive color line and click on the color wheel symbol
  
4. Pick the color of your choice
  
5. Click "ok" and save the file[/font]
  
  

![](https://i.imgur.com/cIDQrp4.png)

  
[font=Comic Sans MS]By default the new slider only appears on vanilla races
  
To enable the nails slider on your non-beast custom race, you need to open the races.ini file and add the lines:
  
YourCustomRaceEditorID = sliders\human.ini
  
YourCustomRaceEditorIDVampire = sliders\human.ini[/font]
  

![](https://i.imgur.com/rQZtvjf.png)

  
[font=Comic Sans MS]The file needs to be saved with UFT-8 encoding, so don't change that
  
  
Also, the valid races list may or may not need to be patched in the esp[/font]
