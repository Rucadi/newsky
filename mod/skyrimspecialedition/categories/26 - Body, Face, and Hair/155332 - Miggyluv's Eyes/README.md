# Miggyluv's Eyes
- Author: Miggyluv
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/155332


![](https://i.imgur.com/EiUZezG.png)﻿

  
A handcrafted set of fantasy-realistic eye textures for human races. Whether your character is a brooding vampire or a wide-eyed wanderer, these eyes offer striking detail without breaking immersion.
  
  
  
![](https://i.imgur.com/IPtg4vv.png)
  
  
  
WHAT'S INCLUDED
  
  
hundreds of unique eye colours with:
  
    ⤷ light versions
  
    ⤷ dark versions
  
[size=3]    ⤷ glow versions (for selected colours)
  
    ⤷ Available across two different eye styles for variety and expression
  
    ⤷ Available for all human, elven characters and Orc, Argonian and Khajiit characters (including vampires).
  
    ⤷ Works with both male and female characters
  
  
  
[/size]COMPATIBILITY
  
  
This mod should be compatible with everything and sit alongside any other eye texture mods you have
  
  
  
INSTALLATION
  
  
Download and install with a mod manager or manually download and drop into your Skyrim data folder.
