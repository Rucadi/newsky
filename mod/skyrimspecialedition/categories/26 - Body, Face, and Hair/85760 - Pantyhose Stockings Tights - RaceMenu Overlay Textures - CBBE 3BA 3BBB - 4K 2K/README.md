# Pantyhose Stockings Tights - RaceMenu Overlay Textures - CBBE 3BA 3BBB - 4K 2K
- Author: Eastbranch
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/85760


[![](https://i.imgur.com/Yxxoq1W.png)](https://ko-fi.com/eastbranch)

  
  
This mod adds a total of **126** texture overlays for stockings, pantyhose and tights with **4** different types of material for RaceMenu.
  
  
These textures are made to be layered together and blended to create almost any kind of pantyhose, stockings or tights imaginable in any color. The amount of combinations are basically infinite.
  
  
Since these textures are applied directly to the body there is no clipping, fits perfectly and looks amazing. Also, your character will always be able to wear it under armor and clothing without taking up any body slot.

**INSTALLATION**
  
**Install with a mod manager.**
  
*or*
  
**Manually**: extract mod contents into the "*Skyrim Special Edition\Data"*  folder.
  
  
**Optional Step (Recommended): Add extra texture slots for Body Paint and Foot Paint in RaceMenu**
  

  
1. Open *Skyrim Special Edition\Data\SKSE\Plugins\**skee64.ini***
Find the ***[Overlays/Body]*** section and edit ***iNumOverlays*** to whatever number you need *(I have [i]iNumOverlays=12* which is more than enough)[/i]
  
2. Do the same for ***[Overlays/Feet]*** if you need more slots for the feet textures as well.
  
  
  
**HOW TO USE**
  
**Choosing Texture and Color**
  
When it is installed properly; in the character creation menu *(console: showracemenu)* - go to the **Body Paint** section for leg textures and **Foot Paint** section for foot textures, select and press **T** on one of the **default** textures to bring up the **Texture List** *(the default texture is just transparent).*In the **Texture List** you should see a list of all the textures, for example:
  
  

```
﻿1: Type 1 - Basic
  
1: Type 1 - Crotchless
  
...
  
1: Type 2 - Stockings - 1
  
1: Type 2 - Stockings - 2
  
...
  
2: Type 1 - Crotchless
  
2: Type 1 - Crotchless & Hipless
  
...
  
3: Rim - Hips
```

  
The textures are structured in a way to make it easy and straightforward to filter textures and create the desired look. Simply press the texture and it should be visible on your character *(if it is not visible check if the color tint alpha is set to 0)*.
  
  
The textures are layered from top to bottom in the Body Paint list, meaning that the texture in the **top of the list is the lowest layer** and the texture in the **bottom of the list is the highest layer**.
  
  
To change color simply press one of the colored squares next to the texture in the Body Paint list and the select color and alpha of the **Tint** (for a natural look leave the **Glow** transparent i.e. alpha 0), switch between **Tint** and **Glow** by pressing **T** in the color menu.
