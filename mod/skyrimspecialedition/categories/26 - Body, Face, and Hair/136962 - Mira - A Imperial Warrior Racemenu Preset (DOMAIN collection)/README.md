Don't know much if anything about modding so i won't be able to list the required mods to use this preset outside of dae's collection.
  
  
Downloading via vortex should be fine, but, if you prefer to download manually to prevent possible issues, then download the file, unpack and drop the Data folder from the download into your Skyrim Special Edition install folder.
  
  
More detailed instructions: 
  
  
Open the downloaded file, go to presets to find the JSLOT file, then drop it in your preset folder usually found in: Skyrim Special Edition/Data/SKSE/Plugins/Chargen/Presets, then go back to the chargen folder from the downloaded file and drop the files into Skyrim Special Edition/Data/SKSE/Plugins/Chargen.
