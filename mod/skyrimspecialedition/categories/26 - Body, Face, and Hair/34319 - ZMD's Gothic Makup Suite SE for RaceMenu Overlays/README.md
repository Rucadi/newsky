# ZMD's Gothic Makup Suite SE for RaceMenu Overlays

- Game: skyrimspecialedition

[font=Comic Sans MS]**[size=5]ZMD's Gothic Makup Suite SE**
  
**Gothic Make Up and Face Overlays for RaceMeu 4k**
  
NOTE : THIS MOD NEEDS ADDITIONAL WORK WITH THE DESIGNS AND IS FAR FROM PERFECT VISUALLY !
  
  
A full makeup suite for RM with 70 Original Makeup and face overlays in 4k.
  
Gothic/Gothika in mind but can be used for pretty makeup too
  
Designed for Blending.
  
Standard Vanilla makeup or standard makeup replacer mods not required and limit this mod.
  
Designed to work with disabling the standard makeup in 'Colours' via racemenu otherwise your limited for overlays.
  
set the standard 'colours' of the following to blank:
  
 Eyeliner colour / Eye shadow / Eye tint / Forehead/ Neck / Nose & anything else you do not require.
  
it will unlock the full potential of any racemenu makeup mods.73 4K Original overlays.
  
Overlays Can be **COLOUR**  changed.
  
Full **Black** as standard.(Only using ENB)
  
18  Gothic makeup for eyes.
  
8 Devastated makeup runs / tears.
  
Smudged lips / 6 Lipsticks 2 lipliners.
  
Various Face overlays for head/face/eyes/other places.
  
Stiches/cracks/necklaces/other random designs.
  
Blusher is not really required becuase you can simply use the cheek colour silders.
  
Works nicely with Havok Lipstick overlay mod if you want more lip designs.
