Installation
  
Install with any mod manager and load the ESP after [Complete Vanilla Eye Overhaul (CVEO)](https://www.nexusmods.com/skyrimspecialedition/mods/160705)﻿ by [LevisageDeDior](https://www.nexusmods.com/skyrimspecialedition/users/187840056). **For all patched eye mods, you only need the textures from that mod**. You need to hide or delete the ESP that comes with the patched eye mod. For example, for Kala's Eyes you need to delete or hide Kala\_Eyes.esp.
  
  
Description
  
This is a collection of patches for the wonderful mod [Complete Vanilla Eye Overhaul (CVEO)](https://www.nexusmods.com/skyrimspecialedition/mods/160705)﻿ by [LevisageDeDior](https://www.nexusmods.com/skyrimspecialedition/users/187840056) (LDD). All patched eye colors can be found in the 'Eyes', 'Eyeball Left', and 'Eyeball Right' sliders in Racemenu. All of the patches provided can be used alongside each other.
