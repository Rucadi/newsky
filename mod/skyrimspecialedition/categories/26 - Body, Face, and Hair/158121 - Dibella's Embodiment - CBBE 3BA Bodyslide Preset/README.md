It's bodyslide preset for use with CBBE family bodies.
  
  
Make sure you have all requirements, otherwise it won't work.
  
  
  
INSTALLATION:
  
Via any mod manager or drop inside DATA folder.
  
  
HOW TO USE: 
  
pick from Bodyslide and Outfit studio:
  
any CBBE body (CBBE, CBBE physics, CBBE amazing etc.)
  
  
Example:
  
CBBE 3bbb Body Amazing - then (Dibella's Embodiment) preset and build your body.
  
  
If you want this body shape for armors as well, you need also build armors (use batch build in bodyslide or do it manually).
