 [/color][size=5]Installation:
  
 ﻿1. Prerequisites
  
[size=4]  Install both the*** [CBBE Body](https://www.nexusmods.com/skyrimspecialedition/mods/198) and [Bodyslide](https://www.nexusmods.com/skyrimspecialedition/mods/201) and do their requirements for each mod manager
  
  
 ﻿2. Installing
  
  Nexus Mod Manager/Mod Organizer: just use the Download With Manager option and install like any other mod. 
  
No need to worry if your manager says this mod won't work. It will if the instructions were followed properly.
  
  
  Manual:[/size][/size][/size][/size][/b][/i][/size]  ***drag the Aphrodite 1.xml and Aphrodite 1 Push.xml files to Bodyslide's SliderPresets folder, usually found in
  
﻿Skyrim Special Edition\Data\Caliente Tools\BodySlide***
  
  
 ***3. Building Body
  
[size=4]  Open BodySlide x64 through your mod manager. Choose a CBBE Body in the Outfit/[/size]Body section[size=4] and select Aphrodite 1 in the Preset section. Click Build.
  
  
 ﻿[size=5]3.1 Building Armor [color=#ffd966](Optional)[/color]
  
[size=4]  If you have any BodySlide armors/armor replacers, select Aphrodite 1 Push in the Preset section and click Batch Build. In the pop-up window, select all the armors you want to bodyslide and click OK. Depending on the amount of armor mods you have, it may take a while to finish.
  
  
[size=3]Note:[size=2] [size=3]Remember to select the preset before clicking Batch Build, as the option uses the currently selected preset and will not ask you which one to use before starting the proccess.
