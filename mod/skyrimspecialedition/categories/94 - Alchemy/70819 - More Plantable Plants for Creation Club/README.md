# More Plantable Plants for Creation Club
- Author: MaskedRPGFan
- Game: skyrimspecialedition
- Mod Page: https://www.nexusmods.com/skyrimspecialedition/mods/70819
This mod adds additional plants to plant from ingredients added by Creation Club Content (Bittercup, Goblins, Rare Curios, Saints and Seducers, The Cause), using FormList Manipulator - without scripts or conflicts with other mods. There is a separate add-on for each of the Creation Club extensions, which can be activated depending on the selected configuration. All add-ons are ESPFE.

**Recommended**
  
[More plants and recipes](https://www.nexusmods.com/skyrimspecialedition/mods/69955) - adds plants from the core game and expansions.
  
  
**Compability**
  
This mod is compatible with all mods. It dynamically updates the list of plantable plants using FormList Manipulator - without scripts or conflicts with other mods.
  
  
**Expansion patches - allow planting additional plants (in fomod)**[Complete Alchemy and Cooking Overhaul](https://www.nexusmods.com/skyrimspecialedition/mods/19924) and Bittercup - Allows planting Ironwood Nuts from CACO.
  
  
**Performance**
  
This mod uses the SKSE plugin, plants are added before the game starts-so it can't get any better.
