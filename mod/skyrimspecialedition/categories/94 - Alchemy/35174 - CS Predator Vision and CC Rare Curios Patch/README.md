**CS Predator Vision and CC Rare Curios Patch**
  
=Description=
  
This makes changes to five ingredients from Rare Curios; The night-eye effect used in alchemy creation now uses Predator Visions version.
  
This way, only one night-eye effect shows up in the alchemy crafting menu. 
  
  
No other changes were made.  Leveled lists were not touched.
  
  
=Installing=
  
Mod manager of your choice and check to make sure the esp is ticked on and of higher priority.
  
It should look like this:
  
ccbgssse037-curios.esl
  
Predator Vision.esp
  
cs\_curiopredator\_patch.esp
  
  
=Uninstalling=
  
Mod manager of your choice
  
  
=Compatibility & Issues=
  
Any mod that alters the five ingredients from Rare Curios
  
SpoilerBlister Pod Cap
  
Daedra Silk
  
Kagouti Hide
  
Watcher's Eye
  
Worm's Head Cap
